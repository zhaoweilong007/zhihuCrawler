# 30岁左右的人，来谈谈你犯了哪些错误？
- 点赞数：5032
- 更新时间：2018年08月27日12时12分50秒
- 回答url：https://www.zhihu.com/question/279387872/answer/479111962
<body>
 <p data-pid="v6ohPzEX">再过四个月，就33岁了，说忠告不敢，说点儿建议好了。以下均为亲身尝试，未尝试过的不说！</p>
 <ol>
  <li data-pid="bECDUTko">对聪明人要么不说话，要么说实话；对愚蠢的人要么不说话，要么说假话。</li>
  <li data-pid="MlUoVLfO">无论选择合作伙伴还是爱人伴侣、知己好友，如果从一开始感觉有不对，一定要及时止损，断交远离，在这方面，感觉比理性思考的准确率更高。</li>
  <li data-pid="XJk4NArQ">身边可学习的知识远比你想象的要多，而这些知识往往更接地气，更具备可操作性，只在于你是否善于发现。</li>
  <li data-pid="eZuw8xJZ">任何不合理且长期存在的东西一定有他存在的理由，且这个理由独一无二，不可或缺，存在即合理，但想要弄清他背后的原因，恐怕需要相当的知识、人脉和他人无可获取的信息源。</li>
  <li data-pid="WOmXzVcZ">如果有点儿闲钱，还有点时间，但又找不到商机，最好的办法就是去闹市、电梯、小区人流量最高的那个大门口，端杯茶，静静的听人们抱怨。</li>
  <li data-pid="wNsE4WmN">看老板靠不靠谱要看小承诺的兑现率，即使他给你画饼。</li>
  <li data-pid="0Lbq1FHI">技多不压身是真的，多会一门技术少求一次人也是真的，但距离成为一个好的管理者越来越远同样是真的。</li>
  <li data-pid="qjAee0Yr">别做救世主。</li>
  <li data-pid="YitisIaf">这个世界上，除了创新者生，守旧者亡以外！几乎没有一定的事儿，只有不一定才是一定的！</li>
  <li data-pid="kHkoDYvy">任何一件事情都没有看上去这么简单，即使是扫地擦桌子。</li>
  <li data-pid="mMxf65TC">如果你经常看的书是《如何修理汽车》、《如何制作蛋糕》、《如何制作表格》等书，那你比那些看《如何成功》、《如何一年内赚到100万》等书的人成功的概率至少高5倍以上。</li>
  <li data-pid="ELKr0Yst">美剧中主人公经常一生气沮丧或者歇斯底里时旁人问他是不是不舒服并不是刻意设置的桥段。</li>
  <li data-pid="nL9IbRyW">人在成长中总要犯错，或多或少，或大或小，或长或短，不可避免！</li>
  <li data-pid="Ny6g90KV">你之所以感觉能在知乎学到经验和知识，是因为答者大多会用简洁通俗的语言直接告诉你答案，你瞬感醍醐灌顶，但不甚了解运算过程，所以对你的帮助仅限于醍醐灌顶。</li>
 </ol>
