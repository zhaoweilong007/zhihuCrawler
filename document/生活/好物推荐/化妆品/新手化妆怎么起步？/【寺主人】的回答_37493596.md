# 新手化妆怎么起步？

- 更新时间：2015年06月06日13时09分45秒

<body>
 <p data-pid="EnCX89OJ">买买买是新手最头疼的问题，又花时间又花钱。</p>
 <p data-pid="NZ1PAhSH">买来不适合或者不会用就更惨了，然后还不敢再买，想化妆却迟迟下不去手。</p>
 <p data-pid="oRyVc8ny">你可能碰到过如下问题：</p>
 <ul>
  <li data-pid="pQndJek4">想买一整套化妆品，该有哪些必不可少的？<br></li>
  <li data-pid="cvAySIwG">粉底液粉饼散粉，唇釉唇膏唇彩，眉粉眉笔染眉膏，眼线液眼线笔眼线膏。。。我到底该选什么啊？<br></li>
  <li data-pid="KWhQcpI1">怎么才能挑到适合自己的颜色的粉底液、眼影、唇膏和腮红呢？</li>
 </ul>
 <p data-pid="QQ0gacYb">有很多彩妆教程都会告诉你，必须去一个一个试过来，或者买之前要做好多功课，这对很多有耐心又追求完美妆感的女生来说是非常必要的。</p>
 <br>
 <p data-pid="t4MgyOZX">但并不是每个女生都有这么多闲功夫和好耐心的。大部分女生因为怕麻烦，或者因为地理位置的原因没有办法去试妆，一旦买错了，再也不敢再下手买别的。</p>
 <br>
 <p data-pid="Cws4TTKA">所以为了拯救大部分懒妹子的这个问题，为你们量身订做了一个新手购物指南，只要照着步骤做就可以。</p>
 <ul>
  <li data-pid="bsvzA-fp">不用动脑子<br></li>
  <li data-pid="lxwDgyVP">不用去店里试<br></li>
  <li data-pid="5da5ZBCb">花钱炒鸡少</li>
  <li data-pid="ldIny8wb">效果炒鸡好</li>
 </ul>
 <p data-pid="MHyE9EvA">请按照一步一步来，不要囫囵吞枣。</p>
 <br>
 <p data-pid="HG5zNN5B">*已经买了化妆品的同学可以直接查看第三步检查表，检查自己买的产品是否有问题并根据第一步和第四步补买。</p>

 <p data-pid="fF5lA_t0">▋第三步：查</p>
 <p data-pid="qdvSYl3O">检查颜色：</p>
 <ul>
  <li data-pid="GYnL2eNG">比较4种色号的口红，你更适合哪种？这就是你以后可以长期坚持买的安全颜色类型了。</li>
  <li data-pid="TAAp89jq">比较2种眼影，是粉色的还是大地色你涂上去看着眼神更干净？这就是你以后的安全眼影参考色了。</li>
  <li data-pid="aNjfNlnz">比较2种腮红，是橘色还是粉色让你看上去气色更好？这就是你以后的安全腮红参考色了。</li>
 </ul>
 <p data-pid="HJuA1TY7">检查问题：</p>
 <p data-pid="5cunNseW">如果回答是“是”请看第四步对应数字的内容。</p>
 <ul>
  <li data-pid="NhvbgaqQ">粉底用后脸变黑了或者感觉掉进面缸子里（请根据色号图提示换色号）</li>
  <li data-pid="bJccpNWA">粉底涂不均匀而脏脏的？（-&gt;1）</li>
  <li data-pid="NJGZIZY7">眼影试过的两种颜色都脏脏的？（-&gt;2）</li>
  <li data-pid="hnj4aViE">唇膏涂布均匀，尤其是唇线处感觉不整齐？（-&gt;3）</li>
  <li data-pid="3txXh_bM">按照教程涂完底妆以后有没有感觉脸像一张大白饼特别平和大？（-&gt;4）</li>
  <li data-pid="okcUi4rs">粉底涂够量了但仍没有盖住脸上的红血丝，感觉红扑扑？（-&gt;5）</li>
  <li data-pid="06wTFvrN">粉底色号和皮肤一样，但感觉脸色蜡黄？（-&gt;6）</li>
  <li data-pid="M9Z7NBEo">粉底色号和皮肤一样，但感觉脸色苍白？（-&gt;7）</li>
  <li data-pid="OaL0N1zM">是否仍然有明显的痘坑、黑眼圈、雀斑、嘴角暗沉？（-&gt;8）<br></li>
  <li data-pid="-4YvZ7DL">是否有明显毛孔？(-&gt;9)</li>
  <li data-pid="8Dxg5Rz4">4小时后再检查你的妆容，是否有起皮或者干纹？（-&gt;10）</li>
  <li data-pid="aZxFJ-lb">4小时后再检查你的妆容，是否有脱妆或者暗沉？（-&gt;11）</li>
  <li data-pid="23-tRpOU">4小时后再检查你的妆容，眼妆是否有晕妆？（-&gt;12）</li>
 </ul>

 <p data-pid="vQ7b6stu">▋第五步：换</p>
 <p data-pid="HV6b3NOH">把不合适的化妆品都和别人换掉吧（眼影、腮红、口红），或者二手卖掉好了。</p>
 <br>
 <p data-pid="gpEZx3ZC">▋Tips</p>
 <ul>
  <li data-pid="BLdeebbO">为什么推荐这些产品？（眼影、口红都是挑的便宜颜色正质量也还行的，主要目的是试色，试完色以后可以根据自己的喜好去买贵一些的口红。 ）</li>
  <li data-pid="OT4Klevn">粉底如果有专柜仍然需要去专柜挑选最合适，由于地理原因没有专柜的再用这种网购方法挑选。<a href="http://zhuanlan.zhihu.com/nvshenjinhualun/19947387" class="internal">3步挑选完美粉底液 </a><br></li>
  <li data-pid="3Ma7MRYk">为什么推荐水粉霜？因为水粉霜非常自然，遮盖力最小，即使买错了色号影响并不会像别的粉底液那么大。适合干皮用，但据油皮反应脱妆和闷痘也很少发生。</li>
  <li data-pid="CSv9DVW-">关于粉底液色号：20#色是选用最多的，如果你的肤色一般人的话就买20#基本没有问题。（寺主人已经给肤色深浅不同的妹子都试过这个号，大家涂上以后都感觉不错。）</li>
  <li data-pid="oFy0UN0H">为什么推荐中样？1.便宜，出错了也损失不大 2.新手很多都不频繁化妆用不掉。</li>
  <li data-pid="QtPNvd_P">如果是学生党，强烈推荐眼影、腮红唇膏宿舍一起买一套然后都试试，瓜分一下。</li>
  <li data-pid="Jm9l8VDX">试色一定要和朋友一起来尝试，他人眼中的建议很重要。<br></li>
  <li data-pid="lFG8378E">
   <p data-pid="bjPPxL-C">已经买了化妆品的同学可以直接查看第三步检查表，检查自己买的产品是否有问题并根据第一步和第四步补买。<br></p></li>
 </ul>
