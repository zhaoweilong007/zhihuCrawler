# 如何正确地喷香水？
- 点赞数：25182
- 更新时间：2023年04月26日10时56分56秒
- 回答url：https://www.zhihu.com/question/50185449/answer/906091840

 <p data-pid="i3qd6jdh">最科学的喷香水方式：打开外套喷在衣服内侧，如果有内口袋的话直接喷在内口袋里最佳。这样你会持续散发淡淡的香味，这种方式最大的缺点有3个：</p>
 <ol>
  <li data-pid="KdoIjKTB"><b>喷一下就足够，减少了香水浪费，影响香水品牌赚钱。</b></li>
  <li data-pid="ZzIsbXJA"><b>留香时间长达3～5天，影响香水品牌赚钱。</b></li>
  <li data-pid="SdBsu6fV"><b>香水隔着衣服散发出来的味道会非常匀称而克制，效果太好不利于其它喷香水方式的推广。</b></li>
 </ol>
 <p data-pid="oN3uHxn6">所以香水品牌发明了一种非常蠢的喷香水“正确方式”：在空气中喷洒，然后往前走一步转个圈。这种方法有3大好处：</p>
 <ol>
  <li data-pid="SfPqatuf"><b>95％的香水被浪费，你喷了一下以后会发现好像不太够，需要再喷一下，香水品牌多赚钱。</b></li>
  <li data-pid="zD9p22ni"><b>留香时间极短，需要每天随身携带香水多次补喷，香水品牌多赚钱。</b></li>
  <li data-pid="Y2sJiUUC"><b>形成鄙视链，当你朋友看到你把香水喷到衣服里的时候会非常震惊地瞪大眼睛问你为什么不会正确喷香水。</b></li>
 </ol>
 <p data-pid="OdMp_Y9_">还有另外一种匪夷所思的喷香水方式就是把香水涂在耳后，不仅直接接触皮肤，还离自己鼻子太近熏死自己，当你跟恋人约会时，对方亲吻你的耳根，猝不及防一股恶心反胃直接涌上来，体会过这种恶心的人不少吧？</p>
