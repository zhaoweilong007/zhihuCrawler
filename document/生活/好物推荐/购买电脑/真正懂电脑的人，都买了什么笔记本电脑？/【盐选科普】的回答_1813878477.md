 <h2><strong>流畅度</strong></h2>
 <p>很多人买笔记本电脑总是会强调希望笔记本电脑可以流畅使用<span><span>，</span></span>实际上在固态普及以后<span><span>，</span></span>大部分线上购买的笔记本电脑都可以满足<span><span>「</span></span>流畅使用<span><span>」</span></span><span><span>。</span></span></p>
 <p>性能确实是影响笔记本电脑流畅度的一个因素<span><span>，</span></span>但对于大部分人要求的流畅其实并不需要担心<span><span>，</span></span>目前无论是轻薄本还是游戏本都可以满足日常使用的流畅<span><span>，</span></span>Office 办公软件或者看个视频<span><span>、</span></span>浏览网页之类的需求<span><span>。</span></span></p>
 <p>真正需要性能的人群<span><span>，</span></span>主要是重度视频处理工作者<span><span>、</span></span>大型单机游戏爱好者<span><span>、</span></span>建筑/环艺类/机械等使用 3D 类绘图的人群<span><span>，</span></span>这些人会很清楚自己对于性能的要求<span><span>。</span></span></p>
 <p><strong>总结<span><span>：</span></span></strong></p>
 <p>只要是线上正规渠道<span><span>，</span></span>搭载固态硬盘的机型均无需担心日常使用的流畅度<span><span>。</span></span></p>
 <h2><strong>CPU<span><span>（</span></span>中央处理器<span><span>）</span></span></strong></h2>
 <p>CPU 是一台笔记本电脑的核心<span><span>，</span></span>其性能的强弱会直接决定到这台笔记本电脑的极限算力<span><span>，</span></span>当然对于日常办公目前主流的 CPU 都是够用的<span><span>，</span></span>但是对于渲染/3D 制图<span><span>，</span></span>视频剪辑<span><span>，</span></span>仿真运算拥有一个性能较强的 CPU 是很重要的<span><span>。</span></span></p>
 <p>市面上做笔记本电脑 CPU 的一共有英特尔和 AMD 两家<span><span>。</span></span>其中英特尔的市场占有率比较高<span><span>，</span></span>能够涵盖三四千到五六万几乎全部价位段<span><span>，</span></span>且在高端轻薄本领域一家独大<span><span>，</span></span>不过近几年由于制程方面的进步比较迟缓<span><span>，</span></span>在主流笔记本电脑市场开始被 AMD 步步蚕食<span><span>。</span></span></p>
 <p>AMD 市场占有率相对较小<span><span>，</span></span>但是由于 AMD 近几年发展的不错<span><span>，</span></span>低价位轻薄本市场在性能和功耗都跟上的前提下做到了较低的价格<span><span>，</span></span>对于三四千预算的办公用户来说是首选<span><span>，</span></span>购买游戏本且追求性价比的用户也可以优先考虑<span><span>。</span></span></p>
 <p>处理器的稳定性是一个非常玄学的东西<span><span>，</span></span>不过在我看来<span><span>，</span></span>稳定是给预算充足的人群准备的<span><span>，</span></span>对于 90% 以上的消费者而言<span><span>，</span></span>便宜比稳定重要<span><span>。</span></span>因为那可怜的百分之零点几的稳定性多花几千块显然是不合算的<span><span>。</span></span></p>

 <h2><strong>GPU<span><span>（</span></span>显卡<span><span>）</span></span></strong></h2>

 <p>屏幕是电脑和用户的交互载体<span><span>，</span></span>因此<span><span>，</span></span>一块好的屏幕对日常使用的体验会有立竿见影的提升<span><span>。</span></span>IPS 屏幕比 TN 屏幕的可视角度更好一些<span><span>，</span></span>所以目前的推荐基本都是都是 IPS 屏幕<span><span>，</span></span>默认分辨率均为 1080P 或者更高<span><span>。</span></span></p>
 <p>若预算很低<span><span>，</span></span>无修图需求<span><span>，</span></span>可以选择厂家标称 45%NTSC 色域或者不进行标注色域的 IPS 屏幕<span><span>，</span></span>屏幕素质较差<span><span>；</span></span></p>
 <p>预算较充足或有修图需求<span><span>，</span></span>可以选择 72%NTSC 和 100%sRGB 色域的 IPS 屏幕<span><span>，</span></span>能够获得素质好的屏幕<span><span>；</span></span></p>
 <p>如果要追求更优秀的屏幕素质<span><span>，</span></span>则在预算充足或对色彩准确要求较高的情况下<span><span>，</span></span>选择 100%Adobe RGB 和 100%DCI P3 色域的屏幕<span><span>（</span></span>需要校色<span><span>）</span></span><span><span>。</span></span></p>
 <p>有 FPS 类游戏需求或者预算充足<span><span>，</span></span>追求游戏体验<span><span>，</span></span>可以优先考虑 144Hz<span><span>、</span></span>240Hz<span><span>、</span></span>300Hz 等高刷新率电竞屏幕<span><span>。</span></span></p>
 <p>购买笔记本电脑时<span><span>，</span></span>商家详情介绍里一般会标注屏幕的好坏<span><span>，</span></span>若不提屏幕色彩显示的数值<span><span>，</span></span>一般默认为最差的屏幕<span><span>，</span></span>只说高刷新率不提色域的默认低色域<span><span>。</span></span></p>
 <p>总结<span><span>：</span></span></p>
 <p>预算低<span><span>，</span></span>可以选择约 60%sRGB<span><span>（</span></span>45%NTSC<span><span>）</span></span>色域覆盖的 IPS 屏幕<span><span>；</span></span></p>

 <p>预算富裕或者对色彩要求较高的用户<span><span>，</span></span>购买时可以选择 100%DCI-P3 或 100%Adobe RGB 色域的屏幕<span><span>。</span></span></p>
 <p>上面是关于常规用户的选购屏幕色彩上的建议<span><span>，</span></span>屏幕素质方面影响比较大的还有另外两项<span><span>，</span></span>分别是<strong>刷新率</strong>和<strong>最高亮度<span><span>。</span></span></strong></p>
