 <h2>步骤一：洁面-洗面奶</h2>
 <p data-pid="bp3oOVHm">首先我们需要正确地清洁，过度地清洁会导致面部皮肤受损，<b>不会让黑头消失，但是会让脸摸一下都痛，变得敏感，</b>在选择护肤的时候我们一般选择更加温和的氨基酸洗面奶</p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-e63de77d18f7cdbcff8f21e78bc7a386_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="361" data-rawheight="459" data-original-token="v2-eb27b81c4b0099e291589ff0b7cacf88" data-default-watermark-src="https://picx.zhimg.com/50/v2-086e0c352b1b5c244977b6d8e8ff9957_720w.jpg?source=1940ef5c" class="content_image" width="361">
 </figure>
 <p data-pid="Om7KvNm8"><b>氨基酸洗面奶</b>：相对更加温和，清洁力是足够的，好的产品用完不紧绷，不假滑.</p>
 <p data-pid="YhmXDL-s"><b>皂基洗面奶</b>：清洁力爆强，甚至某雷敦等还有磨砂类/火山泥等<b>男士专用智商税产品</b>，大多数皂基洗面奶清洁力会过分强</p>
 <p data-pid="ihmISOHN"><b>并不是指所有的皂基洗面奶都清洁力过强，也有十分温和的皂基洗面奶，也有清洁力爆高瞎高的氨基酸洗面奶，不过这些都是少数，一般选择氨基酸洁面会更加安全。</b></p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="gDlpS0V7">无论是干皮油皮中性皮敏感肌肤质，<b>首选氨基酸洁面</b>，清洁力是足够的！</p>
 <h3>怎么洗？</h3>
 <ol>
  <li data-pid="28GMSuvT">使用温水弄湿脸部，避免水太冷或者水太热刺激到皮肤.</li>
  <li data-pid="i6buBxt2">先用打泡网或者打泡瓶（超好用，一按出来都是泡泡）把洗面奶打成泡泡，让洁面更加干净.</li>
  <li data-pid="QEWrNT6C"><b>手法：闭上眼睛和嘴巴涂满就完事了，T区（额头和鼻子）可以重点照顾一下.</b></li>
  <li data-pid="K2mYVDXG">确保全脸都洗干净了，鼻子和额头可以多照顾一下.</li>
  <li data-pid="5W8LQzYj">用温水冲干净.</li>
  <li data-pid="tnYHW7zk">洗完用纸巾或者毛巾按在脸上吸干水就可以了.</li>
 </ol>
 <h3>重要程度：高</h3>
 <p data-pid="8z_R44jl">清洁的重要性不言而喻，做好正确的清洁可以<b> 减少黑头和痘痘的产生几率</b>，皮肤也会更加好。</p>
 <h3>错误方法</h3>
 <ol>
  <li data-pid="fKepkJMi">没把洗面奶打成泡泡直接上脸，这样的后果是，清洁力不到位，而且会浪费洗面奶，<b>打成泡泡的洗面奶可以更加贴合皮肤，清洗到更加深入的毛孔.</b></li>
  <li data-pid="ns6UILB1"><b>长期使用皂基洗面奶，</b>油皮（<b>后面会说怎么判断肤质</b>）一周使用一次皂基是不反对的<b>，</b>但是如果天天使用，比如某秀雷敦的火山泥男士洗面奶，<b>长期使用的后果是：过度清洁导致皮肤发干脱皮，</b>皮肤受损，很快就发红过敏.</li>
  <li data-pid="q6t3DcB_"><b>长期使用磨砂洗面奶，</b>依旧是油皮可以一周去一次，干皮敏感肌绝对不能使用，磨砂洗面奶可以去角质，但是<b>长期使用会导致皮肤受损，变成敏感肌.</b></li>
 </ol>

 <h2>步骤二：乳液/面霜</h2>
 <p class="ztext-empty-paragraph"><br></p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-41315be72a00f450eefdded04541b6b2_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="615" data-rawheight="459" data-original-token="v2-38580cb0c1dc32129855f50ee2733b1b" data-default-watermark-src="https://picx.zhimg.com/50/v2-f8850b1b3be7ecd72276b58c31de458d_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="615" data-original="https://picx.zhimg.com/v2-41315be72a00f450eefdded04541b6b2_r.jpg?source=1940ef5c">
 </figure>
 <p data-pid="tzmrfP2B"><b>乳液</b>：乳液可以理解为液态的面霜，但是乳液更加轻薄更加合适油皮，避免因为太过厚重导致<b>闷痘.</b></p>
 <p data-pid="S5ZagjAz"><b>面霜</b>：相比起乳液更加厚重，常用于补充因干皮等油脂分泌过少的补充，来让皮肤得到油脂的保护.</p>
 <h3>洁面后为什么需要使用乳液/面霜</h3>
 <ul>
  <li data-pid="pIWwyI8Y">洁面后洗面奶会洗去脸上的脏东西和油脂，皮肤失去油脂的保护后，我们需要对皮肤进行保湿的补充，防止因为皮肤失去保护而受损.</li>
 </ul>
 <h3>选择乳液还是面霜的时候就要分肤质了（后面教怎么判断肤质）</h3>
 <ul>
  <li data-pid="8wSY6H7I"><b>油皮：</b>如果脸颊会感觉略干的时候，可以选择补充点乳液（真油皮绝对不可能用面霜，分分钟<b>爆豆</b>）</li>
  <li data-pid="aTALUTXV"><b>干皮中性皮：</b>乳液或者面霜按需求选择.</li>
 </ul>
 <h3>如何判断自己需要乳液或者面霜</h3>
 <ul>
  <li data-pid="6NDgpdTX"><b>油皮：</b>人体自己的皮脂本来就是最好的保湿剂，已经很油了还用更多保湿剂，只会让黑头和痘痘越来越多，所以油皮觉得两颊不用乳液还是有点干的，还是选择一些轻薄的乳液补充.</li>
  <li data-pid="0HgN0PT4"><b>干皮中性皮敏感肌：</b>出油多点的季节（夏天）推荐用乳液，觉得还是干就上面霜，冬天这种干燥的季节面霜基本没错的了.</li>
 </ul>
 <h3>怎么用？</h3>
 <ol>
  <li data-pid="HgoBJioQ">洗完脸后吸干水分</li>
  <li data-pid="2eoY2dsY">直接抹上完事（不需要按摩，按摩只是增加点仪式感）</li>
 </ol>
 <h3>重要程度：高</h3>
 <p data-pid="wcpsj8ik">保湿对油皮来说重要程度其实并不高，但是对干皮却是重中之重，因为油皮有自身分泌的油脂进行保湿，而干皮却缺油，需要自行补充油脂进行保湿。</p>
 <h3>错误方法</h3>
 <ol>
  <li data-pid="5objGYQs">油皮使用面霜，油皮分泌的油脂已经大多足够，上面说过最多选择轻薄的乳液，如果选择了面霜，<b>对油皮来说是火上浇油，</b>过于封闭的油脂<b>会导致闭口黑头痘痘的大量产生</b></li>
  <li data-pid="UUqCehLe"><b>干皮不进行保湿，</b>干皮本来就缺乏油脂对皮肤进行保护，如果不进行保湿，会导致皮肤受损，分分钟就会变成敏感肌。</li>
 </ol>

 <p data-pid="Pa9TyOn0">打伞真的一点都不娘，在遮阳伞下真的太舒服了，就算不是为了护肤，舒服一点不好嘛？</p>
 <p data-pid="u4RTWqSq"><b>防晒霜：</b>分物理防晒和化学防晒，一般防晒霜会标明SPF，日常使用一般选择15上下，太阳强烈的情况建议直接30以上.</p>
 <p data-pid="SeRti5XC">这里简单介绍一下一般防晒霜的2个参数是什么（虽然我知道直男不会看hhh）</p>
 <blockquote data-pid="943k3crI">
  <b>SPF 和 PA</b> <b>SPF</b> 是<b>防晒伤</b>； 防晒品对阳光中紫外线UVB的防御能力的检测指数，SPF越高，防晒效能越好。 <b>PA</b> 是<b>防晒黑。</b> 防护UVA的防晒指数，PA+表示低效防护；PA++是中效防护；PA+++是高效防护
 </blockquote>
 <h3><b>硬核防晒比防晒霜效果更好.</b></h3>
 <ul>
  <li data-pid="tJgRBBFJ">硬核防晒：打伞帽子口罩等，不过口罩可能会在脸上形成一个缺氧的环境，让丙酸杆菌（主要导致痘痘产生的厌氧菌）有了一个合适的生存环境-爆豆.</li>
  <li data-pid="CeudDH_r">防晒不仅仅是为了晒黑，还能抗衰老.</li>
  <li data-pid="w2PYD2-b">大部分防晒霜都使用好的洁面都可以其实洗去.</li>
 </ul>
 <h3>怎么用？</h3>
 <p data-pid="FI113p4s">防晒霜用量不够也是不行，一般大概是一个硬币大小</p>

 <h2>如何分辨自己的肤质</h2>
 <p data-pid="RRR_bGfz">一般只分为<b>油皮/干皮/完美的中性皮.</b></p>
 <p data-pid="2n0oXk9x">而每种皮都会因为<b>过度清洁/没做好保湿/乱用护肤品等不正确的护肤方式</b>导致的<b>皮肤受损变成的敏感肌</b></p>
 <h3><b>如何分辨？</b></h3>
 <p data-pid="npD7kaSL"><b>敏感肌：</b>经常泛红，稍微一点点刺激就泛红，<b>发痒，</b>如果分不清自己的肤质，直接按敏感肌的标准来护肤，敏感肌是最需要小心保养的肤质，产品以温和安全为上！</p>
 <p data-pid="jxQy7pXE"><b>敏感肌护肤重点：</b>一切以温和安全为重，避免再次伤害。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <h3><b>油皮：</b></h3>
 <p data-pid="hW3jig1O">脸上就算不做任何护肤品，都会<b>脸上很油，</b>手指一抹可以炒菜，一般没有保养会造成毛孔粗大，黑头痘痘很多，男生大多是这种肤质.</p>
 <p data-pid="j-NAEzno"><b>油皮护肤重点：</b>注意疏通油脂，防止毛孔堵塞产生黑头和痘痘。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="Imqi6UNz"><b>干皮：</b>如果在洗面奶之后不擦乳液或者面霜就会<b>脱皮，干燥</b>，基本就是干皮了.</p>

 <h2>-------这是一个手动分割线，下面是一些知友提出问题的汇总------</h2>
 <h2>问题汇总</h2>
 <h3>爽肤水一定要用吗？</h3>
 <ul>
  <li data-pid="DMPz5cwS">90%的爽肤水都是没卵用的（SK2神仙水这种我管它叫精华，<b>能当水用的和我们这类熬夜上班苦逼没法比</b>），洁面后是不需要补水的，但是如果长期在空调房，暖气房等干燥的地方可以喷一下补补水。（<b>空气加湿器也能解决环境干燥的问题</b>）</li>
 </ul>
 <h3>洗完脸后脸上都是水了，要擦掉吗？能马上擦水乳/面霜吗？</h3>
 <ul>
  <li data-pid="yMtGH9_p">建议用纸巾轻轻印上去吸干就可以了</li>
  <li data-pid="IyM21lij">有些人<b>喜欢风干水后再擦水乳/面霜，其实是不正确的</b>，不同的部位水分挥发速度不一样，等脸上干的时候有些地方可能已经快脱皮了，所以一般吸干后<b>直接抹上水乳/面霜保湿</b>即可</li>
 </ul>
 <h3>用手拍打能促进吸收吗？</h3>
 <ul>
  <li data-pid="p_oWLx1f">不能，手连细菌都拍不死，还能把更小的成分拍进去？（不存在的）所以不用打得脸啪啪响~护肤品抹上去自然干就可以了QAQ</li>
 </ul>
 <h3>防晒霜需要卸妆吗？</h3>
 <ul>
  <li data-pid="1uE-JScQ">不大需要，现在的洗面奶都洗得很干净的了。</li>
 </ul>
 <h3>冷热水交替有用吗？</h3>
 <ul>
  <li data-pid="v6HOoei-"><b>没用</b>，而且可能对敏感肌带来刺激，热水打开毛孔是没错，但是过热的水会带走更多的油脂，损伤皮肤，用温水就可以了，至于冷水收缩毛孔也是暂时的，坚持不了几分钟就会恢复原样，所以温水洗脸就完事了</li>
 </ul>
 <h3>护肤流程都要早晚用吗？</h3>
 <ul>
  <li data-pid="zup8IZC_">最好早晚使用，毕竟大城市空气污染这么严重。不过早上起床不用洗面奶还是可以的，在家里也不会有什么污染（保湿和防晒还是需要的），晚上就必须用啦（在家的当我没说），在玩出门一天，脸上超脏的，洗完保湿加精华睡觉~（不用防晒了哈哈哈）</li>
 </ul>
 <h3>早晨抹完乳霜之后，晚上睡前洗脸的话需要用卸妆水之类的吗？</h3>
 <ul>
  <li data-pid="oxSSuGEs">不用。。。乳霜是为了保湿。大部分成分是油脂，不需要专门卸妆，各种化妆品（乳霜是护肤品）才需要</li>
 </ul>
 <h3>男生一定要用男士护肤品吗？</h3>
 <ul>
  <li data-pid="PHUPHClP">不需要，这只是商家营销的概念，男生和女生都一样，只有肤质分类的不同，没有性别不同</li>
 </ul>
