# 有哪些女主不是傻白甜的小说?
- 点赞数：20365
- 更新时间：2024年03月11日18时38分39秒
- 回答url：https://www.zhihu.com/question/382066291/answer/2847846119
<body>
 <p>公考还剩十天的时候。</p>
 <p>我穿书成霸总文里的小炮灰，一个十八线女团爱豆。</p>
 <p>刚被经纪人哄骗签下天价违约金合同，塑料姐妹为了抢通告把她扔进游泳池高烧不退。</p>
 <p>当时她别无选择，只能加入恶毒女配的阵营，一起针对女主，最后下场惨烈。</p>
 <p>我一把撕了这本无脑小甜文。</p>
 <p>雌竞什么？自己走爽文路线不香吗？</p>
 <p>反手给大家表演一个炮灰的考公上岸。</p>
 <p>1.</p>
 <p>我穿书了。</p>
 <p>成了书里一个糊团的小爱豆。</p>
 <p>原书里，我是个没有存在感的小炮灰，因为受恶毒女配挑唆一起针对打压女主，最后下场惨烈。</p>
 <p>这一天里，我梳理了人物关系，自己的人设背景，看着卡里仅有的五千块，长叹一口气。</p>
 <p>是真的糊啊！</p>
 <p>「宋意枝，你有没有在听！」</p>
 <p>面前的女人一巴掌拍在桌上，厉声道：「今晚你必须和她们一起去刘总家里，给他庆生！」</p>
 <p>我环顾周围的五六个女生，他们看起来也就才二十岁的样子，低着头，不敢说话。</p>
 <p>我抬头问：「姜笙姐呢？她怎么不去？」</p>
 <p>女人从鼻腔里冷哼一声：「你别管她，她去参加 xc 盛典了。」</p>
 <p>姜笙是这本书的女二，也是我们团人气最高的摇钱树。</p>
 <p>我们团一共九个人，除开女二姜笙和女主梁麦麦有过出圈经历，剩下的都是查无此人。</p>
 <p>我这个角色更是炮灰中的炮灰，在书里就是姜笙的狗腿，替她干些见不得人的勾当。</p>
 <p>「悠悠姐，我今天可能不行。」我歉意一笑：「我还有点发烧。」</p>
 <p>林悠是带我们团的经纪人，闻言皱了皱眉：「还没好吗？」</p>
 <p>「当然了。」我笑起来，慢条斯理地说：「姜笙姐在直播里不小心把我撞进游泳池的事，大家不是都知道吗？还上了热搜。」</p>
 <p>这个小炮灰也是个可怜小女生，在团里容貌不差，只是家境一般，都是普通人背景，没什么关系，才处处被女二针对打压，最后为了生存不得不加入女二孤立女主的阵营。</p>
 <p>「今天还有营销号说是姜笙姐故意把我撞下水的。」我把玩着指甲，叹气：「说节目组那边定的嘉宾本来没有姜笙姐，但是……」</p>
 <p>「好了！」经纪人冷着脸：「那你不用去了，她们几个去就行，你自己好好休息！」</p>
 <p>我满脸笑容，甜甜一笑：「那就谢谢悠悠姐啦。」</p>
 <p>等出了会议室，其他五六个人走在我身后，各个唉声叹气。</p>
 <p>我不用回头都知道他们在嘀咕什么。</p>
 <p>今晚庆生的刘总是个娱乐公司的老板，平时手脚就不干净。小糊团没地位，公司让做什么就做什么，今天去庆生，免不了要被吃豆腐，还只能满脸堆笑。</p>
 <p>「意枝，真羡慕你不用去。」其中一个女生酸溜溜地说：「我们团就只有姜笙姐和梁麦麦还有你不用去了。」</p>
 <p>姜笙这个恶毒女二是公司摇钱树，公司当然不会招惹她。</p>
 <p>梁麦麦作为全书女主角，可是有三个大佬为她保驾护航。</p>
 <p>我这个小炮灰能和他们相提并论，属实高攀了。</p>
 <p>「但没办法，我身体不舒服，悠悠姐也不想我一直对着刘总咳嗽吧。」我笑眯眯地说。</p>
 <p>接管了宋意枝的身体，也继承了她的记忆。</p>
 <p>当时我被姜笙霸凌推进游泳池的时候，秋季瑟瑟冷水，无人站出来拉我一把。</p>
 <p>我自然也会永远记得他们的好意。</p>
 <p>回到宿舍，其他几人在挑选着晚上庆生的衣服，都是刚到大腿的短裙，为了能够露出身体的曲线，到时能拿几个资源。</p>
 <p>我蹲在行李箱旁，翻箱倒柜地找着东西。</p>
 <p>「意枝，在找什么？」</p>
 <p>「不管她，估计在找感冒药吧，你看我这条裙子怎么样？」</p>
 <p>「颜色太深了吧，我记得刘总喜欢浅色的。」</p>
 <p>……</p>
 <p>翻到箱底，我终于找到了证件。</p>
 <p>大学的学位证和毕业证双证齐全。</p>
 <p>翻开一看，竟然不是艺术类专业？</p>
 <p>虽然不明白宋意枝是怎么被哄骗加入女团的，但只要不是艺术类专业就好说。</p>
 <p>不错。</p>
 <p>专业还是考公的热门专业。</p>
 <p>有岗位可以报了。</p>
 <p>2.</p>
 <p>看了眼日历，我长叹一口气。</p>
 <p>放在现实世界里，还有十天我就能进考场了。</p>
 <p>准备了整整一年，结果因为吃梨噎着晕厥了。</p>
 <p>醒来就成了这个糊团小爱豆。</p>
 <p>我回忆了一下宋意枝的爱豆生涯。</p>
 <p>发现竟然没有任何快乐的回忆，除了暗无天日的小商演，就是经纪人嘲讽的嘴脸。</p>
 <p>宋意枝在文里就是个完完全全的背景板。</p>
 <p>女二姜笙苦恋男主无果，把怨气撒在女主梁麦麦身上，而她为了自保，只能教唆宋意枝这个小炮灰替她出头。</p>
 <p>「这是明天的综艺台本哦，大家熟悉一下。」staff 推开门进来，眼眶红红的，她把资料放在桌上：「不过大家不用怎么准备，节目组定的 c 是梁麦麦和姜笙，到时候她们发言的时候你们注意表情管理就好了。」</p>
 <p>其他正在试衣服的队友们闻言，有几个不服气地立马开口：「既然是团综，为什么只有姜笙姐和麦麦说话。」</p>
 <p>「就是，那我们都是背景板么？」</p>
 <p>这个 staff 是个新人，闻言慌乱解释：「不是的，只是这次节目很特殊，是省电视台和地方宣传部统一策划的直播，要上卫视的，所以节目里的发言很重要……」</p>
 <p>她话一说完，其他人果然偃旗息鼓。</p>
 <p>梁麦麦自出道以来，就是戏剧学院的艺考第一。</p>
 <p>而姜笙从小在外留学，家境优渥，谈吐举止自信不怯场。</p>
 <p>这种国家扶持的综艺，能发言就代表根红苗正，是一种殊荣。</p>
 <p>怎么会轮到我们呢。</p>
 <p>等 staff 走后，我起身去上厕所，在走廊处把她拦住：「薇薇，你怎么哭啦？」</p>
 <p>staff 还有些吃惊，连忙擦擦眼角，不好意思地说：「你记得我吗？」</p>
 <p>「是啊。」我笑眯眯地说：「之前看见你跟着悠悠姐学习。」</p>
 <p>薇薇叹口气：「没事啦，是因为工资的事，我试用期没过，悠悠姐让我走人还不给我工资。」</p>
 <p>「如果你没有犯错，她是没有理由辞退你的。首先不要签署离职书面协议，另外《劳动法》第 20 条规定，试用期工资不能低于工资 80%，他们提出辞退需要举证，你可以搜集证据要求公司经济补偿。」</p>
 <p>我安抚她：「不必担心法务的问题，直接去劳动局提交材料就行，会有专人指导你填写资料。」</p>
 <p>「法律保护每个公民的合法权益不受侵害。」</p>
 <p>薇薇愣住了，半晌才回过神来，满脸通红：「谢谢……谢谢意枝！」</p>
 <p>我朝她笑笑，转身准备离开：「那没有什么事我就先走啦！」</p>
 <p>「等一下——」</p>
 <p>她突然小声说：「我反正也要走了，偷偷给你透露一点明天综艺采访的内容吧。也不是什么机密，只是给麦麦他们看的而已……」</p>
 <p>……</p>
 <p>等我回到宿舍，队友们已经准备离开去刘总家庆生了。</p>
 <p>还在低声抱怨。</p>
 <p>「这日子什么时候才到头啊！」</p>
 <p>「如果我是姜笙姐就好了，家里资源多，什么都不愁。」</p>
 <p>「我也真羡慕梁麦麦，两个大佬争着宠她！」</p>
 <p>「我们既没有资源，也付不起违约金，只能熬着——是吧，意枝？」</p>
 <p>我正在看着笔记本电脑，突然被点名。</p>
 <p>抬头，看着一脸不满的队友，我笑笑：「也不是，也有其他的办法。」</p>
 <p>「什么办法？」队友嗤笑一声：「你能给得起八位数的违约金？」</p>
 <p>「你比我们露脸的时间还少，哈哈哈。」</p>
 <p>我没有参与他们的聊天，漫不经心地点开了国家公务员考试录用的网站。</p>
 <p>还好，离报名结束还剩一天。</p>
 <p>轻车熟路地完成注册填好资料，扫了一眼岗位表选了个国考岗位，发现宋意枝在的省份还有个下半年的各省份公务员考试，又顺手报了个省考。</p>
 <p>我当然付不起八位数的违约金。</p>
 <p>但是国家付得起。</p>
 <p>3.</p>
 <p>第二天，大家坐在商务车里开往录制现场。</p>
 <p>昨晚，和我同宿舍的三个女生，半夜四五点才回来，今早都在车里补瞌睡。</p>
 <p>和神色恹恹的众人不同，姜笙红唇娇艳欲滴，正用贴满水钻的指甲拨弄着睫毛：「这破地方也太远了，我都要晕车了——」</p>
 <p>我装作没听见，撑着头看平板。</p>
 <p>平板中是人民日报 APP，我每天习惯看短评和时政积累申论素材和金句。</p>
 <p>从昨天报名到最后考试，也不过短短一个月。</p>
 <p>我虽然现实世界中复习了整整一年，但这最后的三十天也不能放弃，还是得保持做题的手感，训练记忆。</p>
 <p>「姜笙姐，背一下今天的台词。」助理把台本交给她：「等会儿要单独采访你和麦麦的，一定要背熟，这个是现场直播呢。」</p>
 <p>「知道了。」姜笙不耐烦地说：「梁麦麦呢？她怎么不背？」</p>
 <p>「沈总打了电话，说今天麦麦有点感冒，他一会儿亲自送麦麦过来。」</p>
 <p>姜笙冷笑一声，「她才真的是金丝雀呢。」</p>
 <p>她扫了一眼车厢，见我撑着头，不禁笑笑：「意枝，跟你商量个事儿。」</p>
 <p>我懵懵地放下平板：「怎么啦，姜笙姐？」</p>
 <p>「等会儿麦麦估计来的比较晚，站位的时候你就站我旁边吧。」</p>
 <p>我连忙为难摆手：「不行的吧，这个站位有要求的，我不是 c 位。」</p>
 <p>姜笙翻了个白眼，估计在心里骂我上不得台面：「那这样，你站我后面，到时候如果我有记不住的台词你提醒我。」</p>
 <p>她把台词本扔给我，催促我：「你快点儿熟悉一下。」</p>
 <p>等到了现场，果然没见到梁麦麦的身影，姜笙嘴角勾起笑容：「梁麦麦也太不把咱节目放在眼里了，这不是丢的她自己的脸面，是我们团的面子啊。」</p>
 <p>其他的队友或多或少都有些怕姜笙，闻言连忙附和。</p>
 <p>「是啊，这马上就要开始了，还是直播。」</p>
 <p>「宣传物料都放出去了，总不可能等她一个人吧。」</p>
 <p>节目组本来就焦头烂额，一听这话更是火上浇油。</p>
 <p>「算了，先拍着，等会儿梁麦麦来了再继续后面的内容，到时候就跟粉丝朋友们解释一下，她今天身体不好。」</p>
 <p>节目导演皱了皱眉：「但是还缺一个人……」</p>
 <p>姜笙嘴角的笑容渐渐变大，她把我推到导演面前。</p>
 <p>「这儿还有个人，意枝代替一下麦麦吧。」</p>
 <p>「她今天早上看了一下台本的，能记住问题答案。」</p>
 <p>她用只有我们两人听得见的声音低声说：「怎么样，就当是给上次不小心把你推进游泳池的道歉了。」</p>
 <p>姜笙想让我感恩戴德，想用梁麦麦的站位卖我个人情。</p>
 <p>但我知道。</p>
 <p>她当然不会这么好心。</p>
 <p>连个小爱豆的通告都要抢的人，是不会放掉任何一个表现自己的机会的。</p>
 <p>表面上，姜笙是大度让我露面。</p>
 <p>实际上，她不过是想用我来衬托她的谈吐不凡。</p>
 <p>短短一个上午，我怎么可能记得住问题的答案。</p>
 <p>我记不住的地方，她就可以正大光明的抢过我的台词，表现出自己临危不惧的控场能力。</p>
 <p>而我背的断断续续的台词既能当作她背词的提醒，又可以用我衬托她的工作态度。</p>
 <p>比起和她平分秋色的梁麦麦，我这种低一个 level 的队友，姜笙完全吊打。</p>
 <p>好一个一箭三雕。</p>
 <p>其他队友自然不会考虑这么多，还以为是姜笙大发慈悲，纷纷朝我投来羡慕的目光。</p>
 <p>「你可以吗？」导演半信半疑地问：「我们这个是现场直播哈，你快再熟悉一下台本。」</p>
 <p>「既然姜笙姐这样说，那我也可以试试。」</p>
 <p>我礼貌笑笑。</p>
 <p>「我没问题的。」</p>
 <p>4.</p>
 <p>这档节目是助农增收的现场直播。</p>
 <p>内容是让我们团带着大家参观当地的一处特色乡村，采摘水果，直播互动做游戏，依托明星效益，带动当地特色农产品销售推广。</p>
 <p>「大家好，我们是 U-HANA。」队长梁麦麦不在，姜笙自然而然成了 c 位，她大方地对着镜头笑着介绍：「今天我们来到了一处世外桃源……」</p>
 <p>我不用看弹幕，都知道肯定是清一水的夸赞。</p>
 <p>今天虽然我们的演出服都是白色的齐胸襦裙，但是姜笙披了一件绣满金丝花团的大袖衫，满头插满珠翠。</p>
 <p>站在桃花林中，美艳动人。</p>
 <p>而我因为沾了姜笙出境的光，能破例多一些发饰绢花，多给我配了一条披帛。</p>
 <p>身后几人窃窃私语。</p>
 <p>「姜笙姐好漂亮啊……」</p>
 <p>「人家是 c 位，你就别想了……」</p>
 <p>我低头整理着自己的衣衫，未置一词。</p>
 <p>因为我知道，很快，这衣服就没什么用了。</p>
 <p>果然，后续的游戏环节，姜笙吃尽苦头。</p>
 <p>桃花树枝不时勾住了她的大袖衫，金丝脱线，树叶上的碎泥沾在衣衫上东一块西一块。</p>
 <p>「姜笙，作为我们新时代青年标杆的 U-HANA 女团，对于这次的助农增收活动。你有什么想说的呢？」</p>
 <p>主持人把话筒递给姜笙，姜笙刚整理完衣衫，看见镜头连忙换上微笑开口：「我们作为新时代的领航青年……啊——」</p>
 <p>她突然发出一声尖叫，原来是身后的桃花枝勾住了她琳琅满目的珠钗，主持人连忙扶了她一把，但由于刚刚她的挣扎，发髻松垮，满是狼狈。</p>
 <p>「嗯……作为新时代的领航青年……我们应该做好带头作用……」因为刚刚的小插曲打断了姜笙，她顿时卡壳，说不出一句完整的话，满脸通红。</p>
 <p>姜笙很快调整好状态，扔包袱似地丢给我：「你觉得呢，意枝？」</p>
 <p>十多个摄像机同时对准我，在主持人忐忑不安的神色中，我笑了笑，接过话筒。</p>
 <p>「我也是这样认为的。」</p>
 <p>「我们不该只有城市的霓虹，也该记住乡村的月光。」</p>
 <p>「陆游曾说：农为四民之本，食居八政之先。民族要复兴，乡村必振兴。」</p>
 <p>「十四五规划将全面振兴乡村作为新的布局规划，而作为新时代的青年，我们艺人也要发挥我们的榜样作用，鼓励大家一起来支持我们的乡村产业建设，用拳拳之劲绘就乡村的斑斓图景。」</p>
 <p>……</p>
 <p>中途休息的时候我随手打开微博，就看见热搜前三挂着我和姜笙的名字。</p>
 <p>而第一条赫然写着#宋意枝申论#</p>
 <p>我还没点进去看，姜笙已经满脸怒火地冲到我的面前。</p>
 <p>「宋意枝！你是不是故意的！」</p>
