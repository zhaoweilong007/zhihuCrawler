# 有哪些好看的重生爽文？
- 点赞数：7984
- 更新时间：2024年03月11日18时58分02秒

<body>
 <p>嫡姐恋慕的太子流连于青楼，她竟想扮作妓子，得到太子的青睐。</p>
 <p>是我强行拦住了她。</p>
 <p>后来，嫡姐嫁给三皇子，最后又成了皇后。</p>
 <p>我为她欢喜，封后大典那日，嫡姐却将我千刀凌迟，折磨致死。</p>
 <p>她的面孔扭曲淬毒：「这是你的报应。」</p>
 <p>再次醒来，回到了少年时，嫡姐和青楼老鸨密谋出走的时候。</p>
 <p>我微笑鼓励：「太子见姐姐如此痴心，定感动万分，与姐姐白头偕老。」</p>
 <p>这一世，便如她所愿，做一名青楼贵女。</p>
 <p>1</p>
 <p>我死的时候，尸体几乎成了一团碎肉，臭不可闻，连乌鸦都不啄食。</p>
 <p>那日是嫡姐云姝的封后大典，我欢欢喜喜地穿上了最漂亮的衣裳，进宫为她庆贺。</p>
 <p>我却万万没想到，云姝会在她成为皇后这一日，将我绑在铁架上，一刀一刀地划开了我血肉。</p>
 <p>皮肤、筋脉、骨骼、内脏。</p>
 <p>让我痛苦至极，又不会流血过多而亡，清醒地感受到每一秒的痛楚。</p>
 <p>为什么？</p>
 <p>我已经痛得无法发出声音，只用一双眼睛茫然地看着她。</p>
 <p>她是我最爱重的姐姐。</p>
 <p>每次，云姝身着云锦金绣裙，头戴百工珠翠簪，婷婷袅袅地走过，我都又敬又慕地仰望她。</p>
 <p>为什么要这么对我？</p>
 <p>云姝的面孔扭曲淬毒：</p>
 <p>「这是你的报应。」</p>
 <p>「若不是你，我早已经嫁给了太子殿下。」</p>
 <p>新皇不是太子，而是韬光养晦的三皇子。</p>
 <p>曾经的太子行事荒唐，他自诩风流才子，终日流连于青楼楚馆之间，甚至一夜会叫上七个妓子进他的房中。</p>
 <p>我没想到，云姝已经成了这世上最尊贵的女人，竟然还痴恋着那个花心滥情的太子殿下！</p>
 <p>我的喉管已经被云姝割断，只能发出「嘶嗬」的低笑声，甚为可怖。</p>
 <p>云姝面容微变：「你在笑什么？」</p>
 <p>我在笑你！</p>
 <p>云姝，你可真是蠢不可言！</p>
 <p>云姝割断了我的四肢，一刀捅进了我的心脏。</p>
 <p>若有来世。</p>
 <p>我的眼睛睁大，死不瞑目。</p>
 <p>我定要看着云姝，如何毁掉她的人生。</p>
 <p>2</p>
 <p>鹅梨香袅袅氤氲，我躺在贵妃榻上，昏昏沉沉间，听到外间传来一个妇人和一名妙龄少女的对话。</p>
 <p>「红妈妈，你说的可是真的？」</p>
 <p>妇人的声音谄媚：</p>
 <p>「大小姐，您赏面子来我们红袖楼，鸨母龟公，哪个敢欺辱到您头上？」</p>
 <p>少女的声音有些犹豫：「那京中的人，会知晓吗？」</p>
 <p>「定然不会。」鸨母斩钉截铁。</p>
 <p>「我们红袖楼有个传统，头牌姑娘都蒙面的，只有房中客人能看到姑娘的面貌。」</p>
 <p>「再说了，您不也是为了太子殿下吗？」</p>
 <p>红妈妈嘿嘿一笑：</p>
 <p>「到时候，我引您入太子殿下的房中，太子殿下定会醉心于您的美貌才情，温柔小意，这事儿便成了！」</p>
 <p>我蓦然惊醒，冷汗湿透了衣衫。</p>
 <p>上一世，我便是在里间小憩，云姝和老鸨以为房中无人，密谋出走。</p>
 <p>云姝是京城第一美人，求娶之人从京城排到了洛阳。</p>
 <p>更何况云家簪缨世家，祖父是镇国将军，父亲是内阁首辅，云家的嫡长女，便是皇后也做得。</p>
 <p>云姝偏偏痴心恋慕着那个花柳丛里的太子。</p>
 <p>她曾经连续一个月去东宫求见太子，全部被侍卫拒之门外。</p>
 <p>太子被青楼娼女养刁了胃口，对云姝这种名门贵女毫无兴趣。</p>
 <p>云姝的一生，本该是金尊玉贵、平安顺遂的。</p>
 <p>我如何能让我最敬爱的嫡姐，去那等虎狼之地！</p>
 <p>于是我冲出里间，苦劝不得，只能告诉父亲，让父亲拦住了云姝。</p>
 <p>父亲盛怒之下，要用马鞭抽死云姝：</p>
 <p>「我云家出将军，出良相，出皇后，还从来没出过娼女！」</p>
 <p>我替云姝挨了几鞭，死死抱住父亲的腿，哀求道：</p>
 <p>「爹爹，姐姐身子孱弱，千金贵女，万万不能鞭笞责打！」</p>
 <p>「姐姐已经知道错了，您罚她抄写佛经静心可好？」</p>
 <p>当时的我一心只想保全云姝，没有注意到云姝看向我的，怨毒的眼神。</p>
 <p>云姝是父亲的掌上明珠，佛经有婢女帮她抄写，只关了半个月禁闭，小惩大诫，这事儿便轻轻揭过了。</p>
 <p>我却被马鞭抽的皮开肉绽，养了大半年才好。</p>
 <p>现在回想起来，父亲哪里舍得让云姝吃苦。</p>
 <p>我明明已经拦住了他，最后那几鞭子，分明冲着我泄愤的。</p>
 <p>人影闪过，云姝的声音在外间骤然响起：「谁在里面？」</p>
 <p>「出来！」</p>
 <p>于是，我掀开帐纱，大大方方走了出来：「是我。」</p>
 <p>云姝的脸颊变得苍白：「云宁，你怎么在这里？」</p>
 <p>我笑道：「刚刚睡醒，便听到姐姐和这位鸨母的谈话。」</p>
 <p>云姝的嘴唇动了动：「你会...告诉爹爹吗？」</p>
 <p>「当然不会。」</p>
 <p>我用最真诚的语气说：</p>
 <p>「入青楼，扮妓子，有什么不好的？」</p>
 <p>「太子见姐姐如此痴心，定感动万分，与姐姐白头偕老。」</p>
 <p>3</p>
 <p>云姝讶然：「你竟会支持我。」</p>
 <p>我微笑：</p>
 <p>「戏本子不都是这么写的吗，千金小姐与贵族公子隐藏身份，相识于微，最后结为连理，百年好合。」</p>
 <p>不仅如此，我还要再给她加一把火：</p>
 <p>「姐姐，太子殿下阅人无数，你想要得到他的青睐，总要有些傍身的本事。」</p>
 <p>「红妈妈掌管红袖楼这么多年，必然有调教姑娘的秘法。姐姐不如跟红妈妈学一学，到时候到太子殿下身边，定能把他迷得神魂颠倒。」</p>
 <p>云姝恍然大悟：</p>
 <p>「红妈妈，就用你们调教姑娘的手法教我。」</p>
 <p>「哎！」红妈妈谄笑，「老奴定然竭尽全力，倾囊相授。」</p>
 <p>我垂下眼眸，掩饰中眼中的兴味。</p>
 <p>戏台上，小生和花旦说学逗唱，粉墨登场，满堂喝彩。</p>
 <p>但他们所说的每一句话，所做的每一个动作，皆是按照台本演的。</p>
 <p>戏本上让他们哭，他们便哭；让他们笑，他们便笑。</p>
 <p>上一世，云姝划在我身上的每一刀，我没齿难忘。</p>
 <p>一刀杀了她，未免让云姝死得太痛快了。</p>
 <p>所以，我给云姝也写了一出戏。</p>
 <p>让她体会到极乐，再堕入深渊，岂不是更有趣？</p>
 <p>比起一刀毙命，我更喜欢布局谋划。</p>
 <p>如同扑向蛛网的飞蛾，被蛛丝一寸寸紧紧缠缚，直到尸骨无存。</p>
 <p>云姝以为她是天命凰女，殊不知，这出好戏，才刚刚上演。</p>
 <p>翌日，我叫来了婢女墨画，递给她二十两银子。</p>
 <p>「听说你母亲得了重症，这些银子给她治病。不够的话，再来找我要。」</p>
 <p>墨画是云姝的贴身婢女。</p>
 <p>去戏台子看戏，总要买个好座位，才能分毫毕现地看清角色的演出，不是吗？</p>
 <p>墨画跪在地上，将头磕得咚咚作响：</p>
 <p>「谢谢二小姐，谢谢二小姐！」</p>
 <p>我扶起来她：</p>
 <p>「不必如此，我也不过是随手而为。」</p>
 <p>墨画哽咽：</p>
 <p>「但是我去求大小姐，大小姐却斥责我卑贱之躯，死何足惜。」</p>
 <p>重活一世，我也对云姝多了些认知。</p>
 <p>云姝每月都会给平民施粥，京城里的高官贵人，谁不夸一句云大小姐菩萨心肠。</p>
 <p>却对近在耳边的哭声充耳不闻。</p>
 <p>在上一世，她曾在冬日里「不小心」地把我推下水池，让我大病了半年；</p>
 <p>也曾在贵女们的赏花宴上，「不经意」地说出我娘只是个洗脚婢，放任对我的嘲讽嗤笑。</p>
 <p>而我，始终相信了她的说辞，一直真心把她当成我的好姐姐。</p>
 <p>我摇头，对墨画说：</p>
 <p>「千金小姐，平民丫鬟，又有什么分别。」</p>
 <p>出身仅是人生路的起点。</p>
 <p>接下来的每一步，都会决定下一道分叉路口通向哪里。</p>
 <p>「小姐可能落到尘埃里，丫鬟也能成为人中龙凤。」</p>
 <p>「更重要的是，人活一世，该如何抉择。」</p>
 <p>4</p>
 <p>自从云姝开始学青楼妓子的手段，她使唤我越发顺手，把功课全扔给了我。</p>
 <p>我坐在内堂，帮云姝写经史子集、数术珠算的功课。</p>
 <p>一道屏风之外，云姝却在跟老鸨去学勾栏里的媚术。</p>
 <p>「坐冥纸。」</p>
 <p>红妈妈将几个生鸡蛋放在床榻上，又在生鸡蛋上放了一层薄薄的软垫，接着将一叠洁白纤薄的冥纸放在软垫上。</p>
 <p>「你坐在冥纸上，将冥纸捻成一把扇子的形状，而软垫下的生鸡蛋完好无损，便算功夫成了。」</p>
 <p>红姑姑附耳过去，低声说了几句。</p>
 <p>云姝听得红晕满脸：「那处...那处如何使力。」</p>
 <p>红姑姑嘿嘿笑道：</p>
 <p>「太子殿下来我们红袖楼挑姑娘，就一条要求，要媚。」</p>
 <p>「大小姐还是太端庄了。青楼姑娘不骚不媚，如何勾住男人的心？」</p>
 <p>云姝练习月余，从最开始将生鸡蛋坐得稀碎，到鸡蛋不破不散，而冥纸铺成一面扇子。</p>
 <p>红妈妈极为满意，与她谋划下一步：</p>
 <p>「四月初一，是我们红袖楼大日子，名为『红袖招』。」</p>
 <p>「鲜嫩年轻的雏儿都会在这场亮相，出价最高的客人，可以得到姑娘的第一夜。」</p>
 <p>「到时候，太子殿下见到大小姐这般绝色，定然眼睛都直了！」</p>
 <p>云姝美目流盼，胜券在握：「红妈妈说得极是。」</p>
 <p>「青楼妓子，哪里有我好。」</p>
 <p>「容貌、家室、身段、才情，就连着床上功夫，谁能比得上我？」</p>
 <p>「我要太子再也不去找那些娼女，一颗心，只牢牢在我这里。」</p>
 <p>5</p>
 <p>四月初一，桃花灼灼，正是好时节。</p>
 <p>经过红妈妈这一个月的调教，云姝大有变化。</p>
 <p>她天天用桃花香脂擦身，周身萦绕着浅淡勾人的幽香；</p>
 <p>又服了红袖楼调理的药物，浑身幼嫩雪白，没有一丝毛发。</p>
 <p>美人腮粉面桃腮，肤如凝脂，杨柳细腰，走起路来一摇一扭，浑身透出来了一股媚气。</p>
 <p>清晨，云姝描秀眉、点朱唇、擦香粉、着罗衣。</p>
 <p>除了没有凤冠霞帔，一切都比着新妇嫁人的标准。</p>
 <p>云姝拉着我的手，柔柔地说道：</p>
 <p>「阿宁，这些日子多谢你帮我遮掩。」</p>
 <p>「别人家新妇嫁人，都有姐妹送嫁。」</p>
 <p>「我情况特殊，爹娘那边都不能说，我希望你能来红袖楼陪我，好不好？」</p>
 <p>我回握，情真意切地说：「当然。」</p>
 <p>「能陪姐姐出嫁，是我的福气。」</p>
 <p>云姝温柔一笑：「我也有一份礼物，要送给你。」</p>
 <p>6</p>
 <p>这夺命的馈赠，我可不敢消受。</p>
 <p>我救了墨画的母亲，作为报答，墨画悄悄带我去婢女住的耳房，偷听到了云姝和老鸨的计划。</p>
 <p>云姝声音阴狠：</p>
 <p>「云宁知晓我太多秘密。这人，不能留了。」</p>
 <p>红妈妈搓着手：</p>
 <p>「大小姐放心，我已经按照您说的，找了五六个最会折磨人的嫖客。」</p>
 <p>「待到红袖招的那一日，送到二小姐的房中，把他们锁在一起。」</p>
 <p>「都是极暴戾粗鲁的汉子，这一夜下来，啧啧，二小姐全身只怕留不下一块好肉，人都废了。」</p>
 <p>云姝满意地颔首，细细叮嘱：「你记得，不论云宁如何喊叫求救，绝不可放人出来。」</p>
 <p>墨画噗通一声跪下来：</p>
 <p>「大小姐竟然有如此歹毒的心肠！您可千万不要去红袖楼啊！」</p>
 <p>「不。」我笑道，「我当然要去。」</p>
 <p>云姝粉墨登场，主动站到了戏台上。</p>
 <p>我写的戏本子即将开场。</p>
 <p>这等好戏，我怎么能缺席？</p>
