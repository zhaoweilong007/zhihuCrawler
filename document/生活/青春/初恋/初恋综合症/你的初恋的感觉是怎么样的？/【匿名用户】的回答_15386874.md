# 你的初恋的感觉是怎么样的？
- 点赞数：7069
- 更新时间：2012年10月13日01时42分09秒
- 回答url：https://www.zhihu.com/question/19615248/answer/15386874
<body>
 <p data-pid="w20ilW8E">前几天和朋友去吃酸辣粉，朋友问我好吃吗？我说好吃，初恋的味道啊！</p>
 <ol>
  <li data-pid="7WJLceGN">初恋是个富二代，高中都没有念完就去美国念大学。</li>
  <li data-pid="Wrsw8QS9">曾经一次去费城的king of prussia消费了1.7万刀。</li>
  <li data-pid="0vV1cw6I">一次预定过一百条Tiffany水滴的项链回国送朋友。</li>
  <li data-pid="1WlY2szz">唯一看过的书是课本，还没看完过。</li>
  <li data-pid="ejmkz-RW">坚毅的追了我很久，在我各种反抗下还是同意了。</li>
  <li data-pid="MINvl9tP">其实长的不错，有点婴儿肥但是个美女。</li>
  <li data-pid="_-8mmGxT">生活习惯一塌糊涂，半点家务都不会。</li>
  <li data-pid="IXwu8K8H">频繁更换男友，我是唯一一个超过三个月的。</li>
  <li data-pid="FuWlSYS6">在我之前是处女，在我之后也是。</li>
  <li data-pid="yyRb5fL2">每天早起给我一块巧克力。</li>
  <li data-pid="YoZVhnpt">完全不懂车，但是要买车。</li>
  <li data-pid="9hHUUGA5">我教她开车时经常把她说哭。</li>
  <li data-pid="pEdLL3RZ">在她过生日的时候我用银丝做了一个她名字的项链。</li>
  <li data-pid="2YM0J4Ws">在我过生日的时候她送了我一块手表，表盘是自己画的。</li>
  <li data-pid="ndIdYSIj">每天出门的时候都必须拉着她。</li>
  <li data-pid="04fhn26K">讨厌我抽烟，所以我戒烟了，一直到现在。</li>
  <li data-pid="1IiBJV3T">我是唯一帮她剪过头发的男生。</li>
  <li data-pid="vxx-uUlr">她是我唯一剪过头发的女生。</li>
  <li data-pid="SF3NtuU3">她总丢课本，所以所有书上都有我用中文、英文、日文、韩文和西班牙文写的名字。</li>
  <li data-pid="3c2LZk6k">每天晚上下课后她不回家，一直等我到晚上9点下课一起走回去。</li>
  <li data-pid="guVvHsNY">她问我最喜欢什么车，我说mini cooper。</li>
  <li data-pid="OtMvK_aw">因为我喜欢mini cooper，喜欢玩具熊，所以她说我是最娘的男人。</li>
  <li data-pid="rJthGrmn">我喜欢Moleskine的本子，她就把市面上所有版本的全买给了我。</li>
  <li data-pid="F1-ak1WJ">我一共给她做过121顿饭。</li>
  <li data-pid="c9aaT3cr">她喜欢吃红烧肉，但是从来都是就吃两三块。</li>
  <li data-pid="qGs2sP-k">她说她要减肥，所以拉我跑步，结果我一直跑到现在，她从来没跑过。</li>
  <li data-pid="VOu5CEAc">她一直用Aveda迷迭香味道的洗发水，所以我对这个味道很有好感。</li>
  <li data-pid="V_Nw0A8I">那段时间她天天单曲循环B.o.B 的Airplanes。</li>
  <li data-pid="4SkWkVpQ">分手一年后我在肯尼迪机场听到边上有人在听这首歌之后就哭了。</li>
  <li data-pid="_WXyD2zP">我帮她写了很多作业，因为每次求我都会像小猫一样蹲在写字台边上看着我。</li>
  <li data-pid="05jt_HlY">唯一一个看过我日记的女生。</li>
  <li data-pid="QNOhFwxB">因为我说喜欢丝袜，所以她穿了一个冬天。</li>
  <li data-pid="j3hb7cwd">曾经把我锁到过厕所里。</li>
  <li data-pid="BwH6CDHQ">因为吵架，所以把用过的卫生巾放在我书架上。</li>
  <li data-pid="fkrKcC-Y">她自己烧水忘了关火，所以惊动了火警。</li>
  <li data-pid="7mwklF4d">她跟某香港明星同名，从此我不敢看港剧。</li>
  <li data-pid="OOq-y5Ot">她说最喜欢干的事情，就是趴在窗台上看我在门前颠球。</li>
  <li data-pid="XVfo779I">搬家的时候我扭了腰，她给我一直按摩了一个晚上。</li>
  <li data-pid="4-5LDyrI">我扭腰之后每次都是她踮脚亲我，说怕我腰疼。</li>
  <li data-pid="YEHdYY9_">她曾经把一个洋葱扔到冷冻室里，我还硬生生给炒了。</li>
  <li data-pid="Uj3pCE8f">有一天早起她坐在门口看野猫，我以为她离家出走了。</li>
  <li data-pid="CksQOwm5">生活了两年多的地方依然能迷路。</li>
  <li data-pid="zmR0cmOW">迷路是她最大的借口让我拉着她走。</li>
  <li data-pid="37bnxEG5">特别喜欢和我逛二手店，但是从来不买东西。</li>
  <li data-pid="jq2XQFpk">我问她原因，她说就是喜欢这种老夫老妻的感觉。</li>
  <li data-pid="3Mir5mHY">不吃香菜。</li>
  <li data-pid="yzSx1jQo">一次我放了香菜她把桌子掀了。</li>
  <li data-pid="vegSsnqK">我患了香菜恐惧症。</li>
  <li data-pid="odtplekD">我教她下了很久的国际象棋。</li>
  <li data-pid="xjDVGJ1v">我们两下棋都是在床上，穿着睡衣。</li>
  <li data-pid="gON0j3Q-">在我离开学校国际象棋社团后，她是主力。</li>
  <li data-pid="KBBPT3g-">每天她洗完澡我都要进去好好收拾一遍。</li>
  <li data-pid="pdx19dkT">她经常忘了冲厕所，包括大便。</li>
  <li data-pid="mYXAt97x">她国内住别墅，所以经常抱怨我们在那租的公寓小。</li>
  <li data-pid="9bC6B8e1">每次我喝可乐她都说对身体不好。</li>
  <li data-pid="9o1Pi5_p">但是她和可乐的时候就是对身体好。</li>
  <li data-pid="dERNEeVS">出去吃饭她给我点什么我都必须吃完。</li>
  <li data-pid="79HFF7V2">我给她点的她都不爱吃。</li>
  <li data-pid="iYg1x2XF">她最喜欢吃cheese cake。</li>
  <li data-pid="4UCqeXBO">每次买cheese cake都是一整个，但是她只吃一角。</li>
  <li data-pid="sLTdL0I7">她有大约二十款墨镜，但是从来没有带过。</li>
  <li data-pid="5TjKwiTb">她有40多个Coach包。</li>
  <li data-pid="jzUY2CLB">我问她为什么不买Lv的，她说土气。</li>
  <li data-pid="fcoj0yOp">我问她为什么不买gucci的，她说土气。</li>
  <li data-pid="_HHVT9LE">我问她为什么不买Prada的，她说你烦不烦啊！</li>
  <li data-pid="rg3VOnY7">但是她有两个BIRKIN。</li>
  <li data-pid="TPkTzjJx">有一天她问我为什么爱马仕的电动车那么便宜，我说那个是爱马，没仕。</li>
  <li data-pid="L5dmwG5s">她买东西从来都是信用卡。</li>
  <li data-pid="A7PlXI3F">终于在信用卡透支极限后，就给店员一把钱，让他自己选要哪张。</li>
  <li data-pid="PdTCLgo0">在我人人网转发一个朋友红斑狼疮需要捐款后，她捐了1万刀。</li>
  <li data-pid="FUKWs_px">我第一次陪她看车，我看上mini cooper clubman了。</li>
  <li data-pid="_qkBqI0b">但是她喜欢911。</li>
  <li data-pid="AQKDwY0e">mini店员说很快就有countryrman了，我看了图说超级喜欢，但她又说我娘。</li>
  <li data-pid="_8ArMrMX">发现回程没有班车，我们走了一个下午。</li>
  <li data-pid="bn3cQxX5">她中间笑着说让我抱她。</li>
  <li data-pid="9aTeDTYK">我抱了起来，她说不舒服。</li>
  <li data-pid="NfuPnmaD">我放下来问她怎么不舒服，她说你不舒服。</li>
  <li data-pid="R1tdCK2P">黄昏映衬着娃娃脸美极了。</li>
  <li data-pid="5_3cVeeH">我们之后经常在河边散步，但是蚊子总是咬她。</li>
  <li data-pid="VYXJiYLH">她和朋友去纽约的时候曾经给我买过一件play，是我们唯一的情侣装。</li>
  <li data-pid="EXqZuHCg">之后她带我去过一次，说里面一个日本女服务员太漂亮了。</li>
  <li data-pid="6H3RXaw7">我和她去华盛顿看过樱花。</li>
  <li data-pid="K0SEtk_8">头像就是在那个时候拍的，一直用到现在。</li>
  <li data-pid="1LHZr1lC">中午两个人在白宫门口看着轮子发传单吃了四个热狗。</li>
  <li data-pid="rm5m45ym">一次约好晚上吃饭她失约了。</li>
  <li data-pid="kAvbpLT4">回去知道她是跟自己的前男友去吃饭，因为那人准备回国了。</li>
  <li data-pid="DE_M5gvx">我第一次骂她。</li>
  <li data-pid="Uoq2bNgb">她在另一个屋子里哭了一晚上。</li>
  <li data-pid="qm7Aw4u8">第二天早上她给我写了一封非常长的道歉信，和一桌子早餐。</li>
  <li data-pid="1phl6oKJ">这是她唯一一次做饭。</li>
  <li data-pid="IEKannzm">分手后我们很久都没有过联系，互相躲着。</li>
  <li data-pid="YhMPR-o1">第二个学期两个人一门课是同学，而且被老师分到了一个小组。</li>
  <li data-pid="D1grq_eP">我们两个都去选择换课，之后发现又还到了一个班。</li>
  <li data-pid="6R80BS6M">她买车的时候给我打电话，我说“哦”。</li>
  <li data-pid="vQLN6NZp">第二天晚上她在教学楼门口等我说送我回家。</li>
  <li data-pid="tCxXwV_o">车是mini cooper countryman，红色的。</li>
  <li data-pid="D6YR9iJv">我做了三打茶叶蛋给她，她说喜欢吃。</li>
  <li data-pid="-r6S5ioW">我唯一能背过的一个手机号就是她的，我甚至背不过自己的。</li>
  <li data-pid="4slzXcLL">她说她想我了。</li>
  <li data-pid="7Whtfj6f">我和她分手时候在北京西客站边上吃酸辣粉。</li>
