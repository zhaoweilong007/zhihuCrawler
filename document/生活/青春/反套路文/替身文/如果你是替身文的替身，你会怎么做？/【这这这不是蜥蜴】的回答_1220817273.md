# 如果你是替身文的替身，你会怎么做？
- 点赞数：5035
- 更新时间：2024年03月11日15时42分21秒
- 回答url：https://www.zhihu.com/question/391074156/answer/1220817273
<body>
 <p>我看上了美强惨男配，但是他不中意我，我拿他当心上人，他当我是替身，这个白毛狗，哼！</p>
 <p>我叫陈金凤，江湖上知晓我的名字也常常是坠在风无痕后边的。</p>
 <p>风无痕曾是世上第一的剑客，而我是他唯一的废柴徒弟。</p>
 <p>我也算是个沾了师父光环的无名小卒。</p>
 <p>只可惜我剑法不精，师父那老头子怎么努力，我都没法参破他的飞花剑法。</p>
 <p>我没什么，看得挺开，还劝他广收门徒，老头子却有点儿看不开，整日愁眉不展，见我就唉声叹气。</p>
 <p>我不靠名声吃饭，虽说武艺不强，乱七八糟的奇技淫巧、旁门左道倒是学得挺通透，我靠着这些手艺整点儿钱，能养活我俩。</p>
 <p>这日，我在路边茶社等一位客人，还没把新做的弓弩交给客人，远处的大路上突现一股烟尘，随后听见了杂乱的马蹄声。我好奇站在凳子上看了半晌，才远远见一匹拉着车的马惊慌失措向茶社奔来。</p>
 <p>众人慌乱躲避，我却举起了弓弩架在手臂上，对那位客人说道：「我先给您试试，瞧好了。」</p>
 <p>我掰动了机关，只听见嗖一声，连残影都来不及看见，就只听见那匹马轰然倒下，马车侧翻，滚落出一个人来。</p>
 <p>那客人夸赞了我几句，十分满意地买下了弓弩。</p>
 <p>我凑到那马车旁边，先看那马，浑身黑亮四蹄雪白，一个血洞穿破脑袋。</p>
 <p>随后我又去看那车厢，雪蚕丝的车帘，镂花的装饰，四角挂着铃铛。</p>
 <p>又去看那人，一个浑身血迹的白衣男人趴在地上，头重重地磕在石头上。</p>
 <p>心里长叹一声，完喽，竟是醉花楼的人。</p>
 <p>假装好心扶起那人，嚯，这长相绝美……貌似知道他是谁了。</p>
 <p>醉花楼的主人，花轻尘。</p>
 <p>除了他，没有别人再这么骚包了。</p>
 <p>假装焦急地叫他，拍打他的脸，他终于睁开那双漂亮的凤眼，呆呆地望着我，嘴唇嚅动只说了一句：「阿燕。」</p>
 <p>又昏死过去了，只是那双手却死死抓住了我的衣袖，掰也掰不开。</p>
 <p>得了，脑子也撞坏了。</p>
 <p>别看我师父名气大，过得却十分节俭，我师徒二人向来省吃俭用，从来不敢乱花一文钱。</p>
 <p>只是这次我把花轻尘的爱驹一箭射死了，又可能让他破了相，讲理的倒还好说，偏偏这是个不讲理又十分难缠的主儿。</p>
 <p>若是不主动承认错误，等醉花楼查到了我，哪怕十个师父都救不回我一条小命。</p>
 <p>人在江湖，真是身不由己。</p>
 <p>我花重金给他找了间客房，又给他找了个大夫看，等到傍晚他才悠悠转醒。</p>
 <p>我故作殷勤坐在他身边，手里还端着药，只听他叫道：「阿燕，我知道你没走。」</p>
 <p>「公子，我不叫阿燕。」</p>
 <p>他露出些委屈的神色，说道：「你总是骗我。」</p>
 <p>他双眼无神，跟他鸡同鸭讲半天，我才察觉出不对来。</p>
 <p>这一磕，竟然给他磕失明了！</p>
 <p>完喽，花轻尘眼睛给我磕瞎了，这下一百个师父都救不了我了。</p>
 <p>我试探着问他：「你知道我是谁吗？」</p>
 <p>他肯定地说道：「你是东方燕。」</p>
 <p>他一直叫着的阿燕竟是东方家的大小姐，是了，听闻东方燕与醉花楼的花轻尘自小就有婚约。</p>
 <p>他不晓得我是谁便好说，我悄悄放下了药碗，大义凌然说道，「对，没错我就是阿燕，你既然口口声声念着我，想必也不会同我计较这些……」</p>
 <p>他伸出手，手中竟握着一个折断的箭头，「多少年没见，阿燕的箭法竟有些生疏了，这箭只伤了我的肩膀，却没伤到骨头。」</p>
 <p>只知道他头磕到了，没注意肩上竟然也有伤，他竟自己徒手给拔出来了。</p>
 <p>我下意识去夺，他却飞快地收了手：「你从不愿给我什么念想，这个我就收下了。」</p>
 <p>我叹口气，又恭恭敬敬端起药碗，「公子，您喝药吧。」</p>
 <p>江湖中八卦传闻我倒是略知一二，这花轻尘就像是话本里的深情男二，东方燕喜欢的其实是北宫雪，抛弃东方家大小姐的身份悄悄去北宫家做了北宫雪的贴身丫鬟。</p>
 <p>只是其中还牵扯出一个花轻尘，他一手建立起醉花楼，手握几千精英暗卫。</p>
 <p>听闻曾为了东方燕，一夜之间，将江湖中十几个门派给灭了，连条狗都没剩下。</p>
 <p>他是个疯子，是个不讲常理的疯子。</p>
 <p>他口口声声说我是东方燕，估计是真磕坏了脑子。</p>
 <p>大夫看了半天，才说道：「按理说是不会失明的，不过磕到了头也有这种可能，不过不用担心，这种症状只是暂时的，不超过一个月，就能好。」</p>
 <p>我数了数手里的银子，又十分心疼地让大夫给他做个全身检查。</p>
 <p>除了些肩膀和额头的伤，他身上倒是一点儿伤没有，那一身的血迹果然都是别人的。</p>
 <p>我问他：「我见你一身血迹，难道是被什么人追杀吗？」</p>
 <p>花轻尘轻轻笑道：「阿燕莫要担心，一些个无名小卒罢了，都让我杀了，只是中了些毒，让我身体麻痹，暂时无法控制，如今也已好了七七八八。」</p>
 <p>「我见你那马车疯跑，再不停下就撞了茶社了……」</p>
 <p>「哦，月影受惊发疯，那是我故意为之。」</p>
 <p>我突然回想起茶社后便是竖直的峭壁，他的马车往茶社撞去是他故意为之，哪怕是撞死几个人，他的马车至少还能安稳停下，若是撞向山壁，他就有可能受伤。</p>
 <p>啧啧，人命在他眼里就是草芥，真是个疯子。</p>
 <p>这种人就是江湖中人人得而诛之的大魔头，他一天不死，江湖一天不安生。</p>
 <p>但是我这人单力薄的，还是先把小命保下来。</p>
 <p>他对我的身份有了些错误的认知，我琢磨半晌，觉得不能立即跟他说，万一觉得我不是他心心念念的阿燕，回头一剑刺死了我，我就亏大了。</p>
 <p>「咳咳，花轻尘，我帮你联系你的下属，带你回醉花楼吧。」</p>
 <p>他没立即回我，我怕他起疑，只见他呆滞半晌，才略带些羞涩地笑起来：「阿燕，你从没叫过我的名字。」</p>
 <p>「……那我从前叫你什么？」</p>
 <p>他偏过头朝向我得方向，似是在思考，看得我是心都提了起来，但见他嘴角强扯出笑说道：「别管以前你叫我什么，现在你能叫我的名字，我很开心。」</p>
 <p>太卑微了，大佬！</p>
 <p>他有些累了，回头摸到了枕头，自己靠在床头，思索着说道：「这次刺杀就是醉花楼中出了内鬼所致，我需迟些回去，等他们露出马脚，才能揪住他们。」</p>
 <p>我心道不好，他下一句说的却是：「那这段时间，就麻烦阿燕照顾我……」</p>
 <p>我想着箭头他拿着就拿着了，我那箭头虽为了打些广告刻了特殊的符号，但是我一个无名小卒，他也不一定识得，今晚我就趁他熟睡跑路，带着师父去西北躲一阵再说，他忙着清理内鬼，不一定能想得到我。</p>
 <p>跟他牵扯的人物都太大了，一个东方一个北宫，还有一个醉花楼，我真是伺候不起。</p>
 <p>再跟这魔头待在一起，我的脑袋怕是要被拴在裤腰带上了。</p>
 <p>我见他有些累了，帮他把枕头放平，跟他说去楼下叫些饭菜上来。</p>
 <p>还没等我轻手轻脚地离开，他白袖一扬，我手腕竟被系上了一条极细的锁链，那锁链的另一头就被他握在手中。</p>
 <p>我愣愣不知他什么意思，他却依然和煦地笑着说：「阿燕，我怕你再离开我。你放心，这链子够长，足够你到这楼里的任何地方，不管什么东西，我需自己握在手中才能安睡。」</p>
 <p>不是长不长的问题，问题是我是东西吗？</p>
 <p>我才不是什么东西。</p>
 <p>我悄悄关上了门，背着门板从兜里掏出小锯小刀，这链子也不知用什么做成的，怎么砍怎么磨都弄不开，细细一条贴在我的手腕上，撸也撸不下来。</p>
 <p>我现在十分想说脏话。</p>
 <p>趁着夜色，我躲在门外，用我那小锯慢慢磨着铁链，内心十分后悔年前得到一本缩骨功的心法没好好练，又考虑着实在磨不下来我就将大拇指关节掰了回头再给装上……</p>
 <p>我咬着自己的衣服，悄无声息地掰了我的大拇指，疼得一身冷汗，抖着手将链子撸下来，又咔吧一声把大拇指给装上。</p>
 <p>赶紧溜！</p>
 <p>我不敢再从楼梯下去，走廊上有个窗户，我就从窗户跳下去，窗子刚打开，我就跃上了窗沿。</p>
 <p>忽然一条白布裹住了我的脖子，扯着我撞进了门里，这一下，摔得我不轻，我抬头一看，月色照映下，花轻尘一身沾了血迹的白衣站在那里，面无表情地看着我，就像是索命的白无常。</p>
 <p>他扯了扯嘴角轻声说道：「阿燕，你怎么不听话呢。」</p>
 <p>他的眉眼神情没了傍晚时的温柔，变得冷漠而又可怖。</p>
 <p>我扯开脖子上的白布，咳了几声才找到声音，「你认错人了，我不是……」</p>
 <p>他蹲下身，阴影笼罩了我，伸手摸索着我的脸，「我说你是阿燕，你就是阿燕，乖乖留在我身边，我就留你一条命。」</p>
 <p>我心中一惊，他竟早就知道了！</p>
 <p>回想起他说过的话。</p>
 <p>合着就是耍着我玩罢了，给我气笑了。</p>
 <p>「的确是我杀了你的马，害你跌倒，是我理亏，我不也没跑吗，我还给你花钱请大夫呢。我有罪，你把我杀了我都认自己倒霉，只是不至于耍着我玩吧。」</p>
 <p>他抬起头来，眼睛虽说看不到了，只是他凑得极近，仿佛真的在细细看我一样。</p>
 <p>他看不见，我却是看得见的，他凑得那样近，我能看到他的睫毛，能看到他暗沉的眼底，能闻到他身上的熏香。</p>
 <p>江湖上总有些个闲人排些榜单，第一剑客第一轻功都不算什么，曾有一个榜单很长时间都被人津津乐道，江湖美人榜。</p>
 <p>花轻尘排名第四。</p>
 <p>前三皆是女人，花轻尘便是江湖上样貌第一好看的男人。</p>
 <p>北宫雪本是第四，花轻尘原是第五，只是那排榜之人被醉花楼追杀一个月才哭哭啼啼地将两人调换了位置。</p>
 <p>他突然出声，拉回了我的注意。</p>
 <p>「非我戏耍你，你跟她长得太像了。」</p>
 <p>我，跟东方燕长得像？</p>
 <p>我这话要敢跟师父说，师父能笑话我一年。</p>
 <p>东方燕是何许人也，东方家剑术第一的继承人，美人榜排名第二的奇女子。</p>
 <p>我陈金凤只配跟着啰唆的糟老头子风餐露宿彻夜赶工。</p>
 <p>我配吗，我由衷地不配。</p>
 <p>我干笑两声，这魔头，弄死我之前还要猫逗老鼠一般。</p>
 <p>我更不配质疑他，他说什么就是什么，保住我一条命最为重要，找机会给师父发消息救我。</p>
 <p>师父，你再不来找我，你的晚年保障和唯一饭票就没了。</p>
 <p>2</p>
 <p>就当是一份没有酬劳的工作罢了。</p>
 <p>他让我扮演谁我就扮演谁，给他喂药喂饭，每时每刻都在他的监视之下。</p>
 <p>听他阿燕阿燕地叫我，我耳朵都痒痒。</p>
 <p>我曾小心翼翼地问他，这份工作究竟截止到什么时候。</p>
 <p>他状似深情地望着我，说道：「直到她回到我身边。」</p>
 <p>说实话，我觉得玄。</p>
 <p>人家东方燕抛弃自己千金小姐的身份，跟着北宫雪当一个贴身丫鬟。</p>
 <p>他还跟人家有婚约，结果啥也不是。</p>
 <p>呸，舔狗。</p>
 <p>祝他亲眼看见东方燕和北宫雪的婚礼现场，还得给人送礼。</p>
 <p>我心里平衡点儿了。</p>
 <p>我打开窗看外边的车水马龙，怀念以前的自由日子。</p>
 <p>「想出去？」</p>
 <p>他换了一身白衣，「走吧，我陪你。」</p>
 <p>大街上，他穿着一身扎眼的白衣神态自若，我跟在他身后捂着脸，实在无法面对别人惊讶又好奇的目光。</p>
 <p>据我观察，他能看见些了，但是他从不跟我透露这些。</p>
 <p>果然这些罪大恶极的人生命力都顽强得很。</p>
 <p>他给我买了根糖葫芦，我看着这根糖葫芦竟然犹豫了一下，我都这么大了，又不是什么小孩子。</p>
 <p>他脸上带着轻松的笑，将糖葫芦强递到我手中。</p>
 <p>「拿着吧，我记得我还小的时候，跟阿燕出去玩，她就总缠着我给她买糖葫芦。」</p>
 <p>我小心翼翼地接过那根糖葫芦，低头轻轻咬了一口。</p>
 <p>又甜，又酸。</p>
 <p>关于幼时的记忆只有跟着师父四处奔波，饥一顿饱一顿地过，每天天不亮就起来练功，再给人打些小工挣钱。</p>
 <p>糖葫芦我也曾肖想过，只是等到自己能买了，也长大了，没必要了。</p>
 <p>如今尝来，酸酸甜甜的确好吃，只是不如从前的想象中好吃。</p>
 <p>我只吃了一口，就再吃不下去。</p>
 <p>他似是沉浸在自己的回忆里，走到哪儿就买到哪儿，他说买，却让我掏钱。</p>
 <p>我也曾试图反抗，只是他斜眼威胁地一撇，我就立即闭了嘴。</p>
 <p>钱财乃身外之物，我用袖口拭了拭眼角，乖乖给他掏钱。</p>
 <p>直到把做弓弩挣的钱花得所剩无几，我手里拎满了，他才说要回去。</p>
 <p>这些吃食放不了几天就坏，他买只凭自己高兴，吃了又嫌弃不好吃。</p>
 <p>全都推给了我，还让我全都吃完。他手托着下巴一边微笑一边看着我吃。</p>
 <p>「你知道吗？阿燕就不会像你这般吃东西。」</p>
 <p>我艰难咽下豌豆糕，「怎么，吃东西也得学她吗？」</p>
 <p>他嗤笑一声：「你学不来。」</p>
 <p>当天夜晚，我一边叼着火烧，一边拿出了自己的家伙事儿，开始继续赶工。</p>
 <p>不行啊，按他这个造法，他不弄死我，我就得自己饿死。</p>
 <p>他穿的新衣，吃的鱼肉，住的客栈，买的药钱，都是我掏的。</p>
 <p>也不知道他回去了，还能不能把钱再还给我了。</p>
 <p>3</p>
 <p>我们中间已经换了两个地方，我手中仅剩的银钱都不够我俩人住店，我们两人只得假扮夫妻住进一间房内。</p>
 <p>我说我这身份应该给他扮老妈子，他凤眼一瞪说道：「与我假扮夫妻你还有什么怨言？」</p>
 <p>合着我还得感谢你呗。</p>
 <p>其实我知道，他这样做是为了躲过醉花楼的追查，他的性格习惯属下必然清楚，他专情那么些年，总不能短时间就跟一个姑娘牵扯在一起。</p>
 <p>但是不得不说，这厮是真狗。</p>
 <p>每日都要穿新衣，我没了银钱，买不来他就掉脸，我只能每日风雨无阻地给他洗衣服。</p>
 <p>还全都是白的。</p>
 <p>我问他就不能换个颜色的衣服穿吗，就你这么扎眼还躲个什么劲？</p>
 <p>他只是略带忧伤地说道，只有他穿白衣的时候，阿燕称赞过他好看。</p>
 <p>我一边默默骂他，一边给他狠狠地搓。</p>
 <p>他发现我在骂他，悄悄走到我身后，带着微笑问我：「你在说什么呢？」</p>
 <p>我动作一滞，继续说道：「公子真是风神朗俊清雅俊秀才貌双绝品貌非凡。」</p>
 <p>他哼笑一声，斜倚在墙边，闲闲问我：「那你说，我穿什么颜色的衣服好看？」</p>
 <p>我想了一会儿，这种大魔头通常不穿黑衣就穿红衣。</p>
 <p>就说道：「你穿红衣好看，公子这般样貌若是穿红衣必然是艳惊四方的。」</p>
 <p>他哼一声就走开了。</p>
 <p>我松了一口气，继续给他洗衣服。</p>
 <p>到了夜里，他躺在床上睡觉，我就借着烛火继续做我的机关。</p>
 <p>磨的声音大了些，他还是个耳聪目明的，一个枕头就打过来，被我灵巧躲过。</p>
 <p>「睡觉。」</p>
 <p>我叹了口气，不当家不知道柴米油盐贵。</p>
 <p>「我小声点，你睡吧，我出去磨。」</p>
 <p>「回来！」他坐起来，「你带着这磨具刀具去哪儿？」他觉得我的东西太显眼了，我不吭声，他似是气了一会儿说道：「你就在这儿，小点儿声。」</p>
 <p>我闭上眼深吸一口气，这是个贵公子，没过过苦日子，我的小命还在他手里。</p>
 <p>我上哪儿受过这委屈？</p>
 <p>待我心平气和了，又开始继续做工。</p>
 <p>这个弓弩做了有个五六天，本是半个月才能完工的，奈何养的这个白毛狗太贵重了，让我夜以继日地赶工。</p>
 <p>我的弓弩差了个箭头，其余的机关全让我做好了，待我哪天去了铁匠铺做些箭头去。</p>
 <p>已经到了深夜，我满意地欣赏我手中的弓弩，这次做的小些，能藏在袖子里做暗器。</p>
 <p>这回寻个好买家，能供我俩一月花销的。</p>
 <p>「你的东西拿来我看看。」</p>
 <p>他竟没睡。</p>
 <p>我的弓弩没装箭头，只有光秃秃的木杆，一只飞蛾缠缠绕绕地围着烛火飞舞，我将弓弩放在手臂上，屏息凝神，机关猛然一发动，木杆竟深深扎进墙壁中，我上前拔下了木杆，墙里还钉死了那只飞蛾。</p>
 <p>花轻尘走过来，拿起我的弓弩仔细查看，脸上没了那惯用的阴阳怪气的表情，反而变得认真，双眼都亮着光。</p>
 <p>「真是好东西。」</p>
 <p>那可不，我做的东西，想寻买家还是不缺的。</p>
 <p>他从袖子里拿出那只箭头，安在木杆上，然后打开了窗子，窗外不远就有一棵粗壮的树。</p>
 <p>他站在我身后，双手环着我，握着我的手渐渐瞄准了那棵树，树后的窗子里有一个女人抱着孩子哄睡的影子。</p>
 <p>他贴在我耳边轻声问道：「你这箭，能不能把树射穿。」</p>
 <p>我其实没怎么听到他说什么，只觉得一股热气贴着后颈，吹过我的耳边，酥麻与战栗从后背传到全身。</p>
 <p>我察觉他扣动了机关，心内一惊，手腕往上一扬，箭斜着穿透了树，射进对面的窗子里，瞬间，对面女人的惊叫声响起，孩子也哭闹起来，那家的男人也起来，打开了窗子骂人，并没有伤到人，只是扎碎了那家的花瓶。</p>
 <p>我松了口气，回头猛然将他一推，气愤道：「你有病吗？」</p>
