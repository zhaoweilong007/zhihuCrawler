# 有没有好看的沙雕小说？

<body>
 <p>我向四个男人告过白，四次都告白失败。</p>
 <p>他们分别是宠物店老板，大学校草，酒吧驻唱，风景摄影师。</p>
 <p>朋友问我向四个大帅哥告白失败后还能和他们当朋友的秘诀是什么？</p>
 <p>我自信地回答她：「天赋。」</p>
 <p>朋友不明觉厉。</p>
 <p>我粲然一笑。</p>
 <p>连续四次告白，都被统一回复，「集美，对不起，我不是直男。」</p>
 <p>这何尝不是一种天赋呢？</p>
 <p>封心锁爱一段时间后，我回老家祖坟上磕了三个头，求老祖宗保佑赐予我一个直男吧，再这样下去我都要变成通讯录了。</p>
 <p>回来后不久，我恋爱了。</p>
 <p>这回是对方先告的白。</p>
 <p>老祖宗在上，我这个异性恋终于碰到了一个直男！</p>
 <p>直到我闺蜜拉我去看隔壁学校的女装大佬。</p>
 <p>我与女装大佬四目相对。</p>
 <p>芜湖，是我男朋友。</p>
 <p>不顾身后喊着误会的男朋友，我飞快地跑出了隔壁学校。</p>
 <p>西湖的水我的泪。</p>
 <p>为何我身边没有直男！</p>
 <p>1.</p>
 <p>「怎么喝了这么多酒？」酒吧驻唱让别人替班后走到吧台问我。</p>
 <p>「我——呕——我他爹的又失恋了！」我一边干呕一边醉醺醺地喊道。</p>
 <p>酒吧驻唱拍了拍我的背，接过我手里的酒杯，递了杯酸梅汤给我。</p>
 <p>「告白失败了？这回那个风景摄影师总应该是直男了吧？」酒吧驻唱喂了我一口酸梅汤。</p>
 <p>「啊呜呜呜我好惨啊……」本来我还憋着一口气，酒吧驻唱一问我直接破防了。</p>
 <p>「我他爹的为什么会连续爱上的四个都不是直男啊！异性恋难道要灭绝了吗？这世道直男怎么比我脸上的痘痘还难找啊！呜呜呜……」我哭着大喊。</p>
 <p>这是个清吧。</p>
 <p>这还是个非直男聚集地。</p>
 <p>于是——周围的人都向我投来了目光。</p>
 <p>酒吧驻唱忍着笑意站起来替我向周围的人道歉。旁边一个穿短裤白袜的清秀男生怜悯地看着我，对着酒吧驻唱唏嘘地说道：</p>
 <p>「没想到，直女也不容易。」</p>
 <p>我头晕目眩，继续发疯：「谁说不是呢！谁他爹的想得到身高腿长剑眉星目有八块腹肌的大帅哥内心是小娇娘啊，还他爹的让我碰见四个！男同的门槛已经这么高了吗？」</p>
 <p>清秀男生看我的眼神更怜悯了。</p>
 <p>酒吧驻唱听完我的话惊讶地问我：「你不是说，那个风景摄影师身高一米九，小麦色皮肤，常年健身，一拳能打死三个家暴男吗，怎么还是受？」</p>
 <p>我恨声答道：「所以我才坚定不移地以为他是直男。他说他是钙的那一刻我内心还诡异地感到了平静，我心想，那他应该是 1 吧，也不枉我判断错性取向。」</p>
 <p>「可他为什么还是 0 啊？我怎么会爱上四个 0 啊！这他爹的比杀了我还难受！」我又哭又喊。</p>
 <p>酒吧驻唱见我的情绪不太稳定，就把我送回了家。</p>
 <p>「乖乖睡觉，别乱跑啊。」临走之前酒吧驻唱叮嘱我。</p>
 <p>我拍了拍胸口向他保证——实际上我拍到了头上：「你放心，我清醒得很。」</p>
 <p>说完我打了个酒嗝，把被子蒙在头上就睡。</p>
 <p>没过几分钟，我就被闹钟吵醒了。</p>
 <p>我划开手机一看，隐约想起来是我之前关注的电台节目要开播了。</p>
 <p>我点了好几次才点进直播间。</p>
 <p>听着男主播温柔又富有磁性的声音，我又想起了我那四段无疾而终的暗恋。</p>
 <p>「欢迎大家来分享自己的故事，听众朋友们可以和我连线了。」</p>
 <p>我脑子里捕捉到了「连线」二字，便开始不停地点连线。</p>
 <p>这个直播间人数并不多，我很轻易便连上了。</p>
 <p>「您好，欢迎您来分享您的故事。」男主播照常说道。</p>
 <p>这一刻，男主播并不清楚他的直播间即将迎来一个多么精彩的故事。也不会知道，从此以后他的温暖正能量电台节目会在后来变成奇葩故事会。</p>
 <p>故事？故事！</p>
 <p>我昏昏欲睡的大脑一下子精神了，我正好有一大堆故事要说，于是我自信地开麦了：</p>
 <p>「今天我们大家之所以欢聚在这里，是为我这个找不到直男的直女，庆祝我的第四次失恋。今天，我要敬我的四个男闺蜜，感谢他们，见证我的悲惨人生。我也发自内心地祝愿他们，从此以后，和我的人生一样，开始发鲜，发香！」</p>
