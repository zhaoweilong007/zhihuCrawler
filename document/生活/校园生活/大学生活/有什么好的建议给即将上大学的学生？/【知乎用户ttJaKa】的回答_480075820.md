# 有什么好的建议给即将上大学的学生？
- 点赞数：12302
- 更新时间：2018年08月28日15时37分28秒
- 回答url：https://www.zhihu.com/question/281895917/answer/480075820
<body>
 <p data-pid="ejZrpqV7">所有的建议归结为一句话就是：<b>趁早</b>。学习要趁早，交友要趁早，社团要趁早，所有大一能做的事情不要拖到大四做。因为别人还沉浸在大学世界的花花绿绿里，迷茫找不到方向的时候，就是你们能够拉开差距的时候。</p>
 <h2><b>学习要趁早</b></h2>
 <ol>
  <li data-pid="v00F6Kl-">不要因为上了大学，就看不起那些整天泡自习室图书馆的人。大学里学习依然重要，<b>拿奖学金、保研、出国留学</b>，到处都要拿成绩来做通行证，所以学习要趁早，大三大四再想奋起直追来拼绩点，几乎是不可能的。</li>
  <li data-pid="3aM1ONUZ"><b>挂科可大可小</b>，挂一两门的后果是重修+失去保研资格，挂三四门的后果是辅导员找你谈心，再多挂几门，可能就要回家高复了。</li>
  <li data-pid="7XAfZCSp"><b>翘课有“滚雪球”效应</b>，就好比上午翘了课，往往下午也不想去；今天翘了课，往往明天也不想去，所以翘课的“头”不要轻易开。</li>
  <li data-pid="mbz12B3_">自制力不好的同学千万不要轻易尝试在寝室备考的滋味。特别浮躁的时候可以去自习室、图书馆看一天书，感受安静好学的氛围，真的能够让人静下心来，找找自己的方向。</li>
  <li data-pid="IZc4-bTr">对于不喜欢的专业，大学是有再选择余地的，比如<b>辅修、转专业</b>。不过辅修的话课程量很大，转专业有失败的概率，所以要头脑清醒做决定。</li>
  <li data-pid="pzuxlxLS"><b>学英语</b>看似是件回报率很低的事，不过当你求职遇到英语面试的时候，可能就不会这样想了。所以基本的听说读写能力还是要掌握，四六级好好考，在没有其他证书的情况下，这是唯一能证明你英语水平的方式。</li>
 </ol>
 <h2>社团活动要趁早</h2>
 <ol>
  <li data-pid="vNp2htCy"><b>不要听别人说学生会很官僚，就不参加学生社团</b>。社团有许多的弊病，但真的可以锻炼一个人的能力，比如项目策划能力、沟通谈判能力、管理能力，也是除了寝室以外你能交到朋友的最佳方式。不过在社团中得到提升的<b>前提是在社团中担任部长及以上的领导职位</b>，没有做一年以上小干事的必要，那些经验是重叠的，时间也被浪费了，所以当你发觉社团没有提升意义的时候要<b>及时止损</b>。</li>
  <li data-pid="gzHlizzh"><b>社团宜精不宜多</b>。真正适合自己的，能得到锻炼的社团，1-2个就足够了。鱼与熊掌可以兼得，但做得好的人非常少，所以在精力有限的情况下，<b>做全才不如做通才</b>，早早想好自己的大学重心在哪。</li>
  <li data-pid="5nihkWhO">大学里的<b>信息是开放的也是封闭的</b>，如果你不去主动关注，就会错过很多信息。所以多查看学校的官网（本科生网、创新创业网、院网、就业网等等）。</li>
  <li data-pid="wHXc9bVj">参加符合你兴趣爱好或者未来发展方向，并且含金量高的比赛。在一个项目从0到1的过程中会收获到很多东西，也是未来你简历上闪闪发光的点。参赛要趁早，这样即使失败了还有重来的机会。</li>
 </ol>
 <h2>交友要趁早</h2>
 <ol>
  <li data-pid="FMvoava-">好的大学是一个好的平台，交友要趁早，建立起自己的人脉资源网，在这个过程中也能收获真心诚意的朋友。这并不是放大交际的作用，而是要提醒大家，<b>大学切忌做肥宅</b>，除了刷剧打游戏就什么也不干，两耳不闻窗外事是非常致命的。</li>
  <li data-pid="r_uaRaRg">大学是个小社会，所以多<b>锻炼情商</b>，考虑别人的感受，做到知世故而不世故。比如在寝室里不外放音乐，室友睡觉时轻手轻脚，对不了解的事情缄口不言等等。</li>
  <li data-pid="gdZSge7-">室友可能是你大学关系最好的朋友，也可能是关系最差的路人，谨慎处理寝室关系。如果关系不好，那就多出去走走，在其他社团里找朋友，别闷在寝室默默生气。</li>
  <li data-pid="4siJ7IwO">至于恋爱，大家<b>开心就好</b>。</li>
 </ol>
 <h2>实习兼职要趁早</h2>
 <ol>
  <li data-pid="hA5nHsgs">就业形势越来越严峻，就会倒逼大学生更早出去实习锻炼。大三不算晚，大二挺适宜，大一拼运气，总之<b>不要做一张白纸</b>。</li>
  <li data-pid="dGVFCijS">实习兼职在精不在多，挑选的原则很简单——<b>看看这段经历能否扩充你的简历，成为你的某种能力</b>，让面试官对你产生兴趣。所以多去大厂，尽量少做发传单、淘宝客服、跑腿这类功利但没有提升意义的兼职，你失去的时间和精力远比你得到的那几百块钱要多得多。</li>
 </ol>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="myeDU7WS">大学是最容易让人出现错觉的一个人生阶段，你总觉得自己有大把的时间可以去浪费与挥霍。但<b>其实四年很快，未来很长</b>。一年又一年的新鲜血液注入大学校园，意气风发，朝气蓬勃，希望你们能够收获一个有意义的大学！</p>
