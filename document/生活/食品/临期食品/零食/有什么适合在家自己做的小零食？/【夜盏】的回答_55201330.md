 <p data-pid="7x3TiSEy">今天更新<u><b>草莓蛋挞</b></u>！顺带芝心蛋挞&amp;麻薯蛋挞。</p>
 <figure>
  <img data-rawheight="655" data-rawwidth="891" src="https://picx.zhimg.com/50/8d8da25604d5711cf093b7b39a1af7e1_720w.jpg?source=1940ef5c" data-original-token="8d8da25604d5711cf093b7b39a1af7e1" class="origin_image zh-lightbox-thumb" width="891" data-original="https://pica.zhimg.com/8d8da25604d5711cf093b7b39a1af7e1_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="Tx6q8d12"><u><b>原料：</b></u>（9个量，刚好够30L烤箱一盘）</li>
 </ul>
 <p data-pid="n7y92hAs">蛋挞皮 9个（我用的是自己做的，但开酥特别难，暂时不写，MINA还是先用买的吧~）</p>
 <p data-pid="ezXNQ0wQ">挞水：淡奶油 120ml</p>
 <p data-pid="u1awQL9z">牛奶 50ml</p>
 <p data-pid="pXvFDQ5Q">炼乳 1小勺</p>
 <p data-pid="xCRmGl1n">糖 20g</p>
 <p data-pid="fTaFqSNu">蛋黄 2个</p>
 <p data-pid="_6EiF-9m">夹心材料（可选）：草莓酱/马苏里拉奶酪（咸） /Q心馅（甜） 【买不到请求助万能的某宝】</p>
 <ul>
  <li data-pid="r4ZG1LGv"><u><b>做法：</b></u></li>
 </ul>
 <ol>
  <li data-pid="HbZ3Ev3h">混合淡奶油、牛奶、炼乳、糖，加热搅拌至糖融化，等到温度不烫手了再加入打散的蛋黄搅匀，过滤一遍，即成蛋挞水。</li>
  <li data-pid="sZ2wlYGP">烤箱预热220°C。取出冷冻的蛋挞皮（不要提前取，会化冻），捏一下挞皮底部，放置夹心，再倒入挞水，七成满。放进预热好的烤箱下层（一共四层，最底下那层），烤制20分钟。</li>
 </ol>
 <figure>
  <img data-rawheight="342" data-rawwidth="607" src="https://pica.zhimg.com/50/07f544ff811b5258a601cc2cf0ac7cdb_720w.jpg?source=1940ef5c" data-original-token="07f544ff811b5258a601cc2cf0ac7cdb" class="origin_image zh-lightbox-thumb" width="607" data-original="https://pic1.zhimg.com/07f544ff811b5258a601cc2cf0ac7cdb_r.jpg?source=1940ef5c">
 </figure>
 <br>
 <ul>
  <li data-pid="nuSwhJsB"><u><b>夹心：</b></u></li>
 </ul>
 <ol>
  <li data-pid="WrZiuB7E">草莓蛋挞：草莓酱放一勺即可，草莓蛋挞切开会有流心效果。酸甜口味。果酱我是春天做好冻在冰箱里备用的，市售草莓果酱太甜，不合适，建议自己做。</li>
  <li data-pid="pf1zji6u">芝心蛋挞：马苏里拉芝士切成小块，拇指盖那么大，烤好后【咬开】会有芝士的拉丝效果，视觉效果比较赞，味道不如卖相好。</li>
  <li data-pid="KSAoYf2G">麻薯蛋挞：Q心馅对半切， 烤好之后趁热吃会吃到化开的麻薯馅，论味道这个是最好的。<br></li>
 </ol>
 <br>
 <ul>
  <li data-pid="jD9AnoLw"><b><u>TIPS：</u></b></li>
 </ul>
 <ol>
  <li data-pid="K-k4TU61">蛋挞皮不要提前取出，化冻了分层会不明显。</li>
  <li data-pid="DVWhyMDJ">捏一下挞皮底部是为了把它捏薄一点，市售挞皮质量层次不齐，如果太厚烤不透底部湿湿的会很难吃。</li>
  <li data-pid="VtpA4f5a">放在最下层烤也是为了把底部烤透，保留做法。<br></li>
  <li data-pid="biElg4Ut">挞水不要超过七成满，多了在烤制过程中会溢出来盘子很难洗。</li>
  <li data-pid="Uvj_p3rk">夹心不要贪多，适量就好。</li>
  <li data-pid="wYWVarfK">各家烤箱脾气不一样，可能会偏温，以蛋挞表面出现焦斑为准，时间不够就多烤两分钟，反之就尽快拿出来。</li>
  <li data-pid="waASRZJU">我能想到的问题都写上了还有啥你们再问吧……<br></li>
 </ol>

 <ul>
  <li data-pid="lTCVi5fr"><u><b>原料</b></u>：</li>
 </ul>
 <ol>
  <li data-pid="qRrbCPRZ">木薯淀粉：
   <figure>
    <img data-rawheight="168" data-rawwidth="150" src="https://pica.zhimg.com/50/b9bc2f707ebacda165733495e4b3b861_720w.jpg?source=1940ef5c" data-original-token="b9bc2f707ebacda165733495e4b3b861" class="content_image" width="150">

  <li data-pid="X5XkXhJL">芋头or红薯or紫薯or南瓜（随你喜欢，外面糖水店最常见的是红薯和芋头的）</li>
  <li data-pid="haH7l4by">糖 <br></li>
 </ol>
 <ul>
  <li data-pid="1I_LuLt2"><u><b>做法</b></u>(以芋头为例)</li>
 </ul>
 <ol>
  <li data-pid="BsNTIAzp">木薯淀粉，芋头和糖的<b>比例为1：3：0.3 </b>，如果芋头有300g，木薯淀粉就用100g，糖用30g ，糖量可以根据自己的口味增减。因为芋头含水量不一样，木薯粉可以在100~120g灵活调节。</li>
  <li data-pid="D1tqPPwi">芋头去皮切小块上锅蒸熟，加入适量糖，用勺子压成泥，热度会把糖融化~芋泥也会变稀一点。</li>
  <li data-pid="0eNTrZJc">将木薯淀粉倒入芋泥里，混合均匀，加入一勺开水，用手揉成面团（小心烫手），这个步骤中，觉得面团干了就加一点点水，太湿了就加一点点粉，就一点点别加太多！最后面团应该是较为光滑且不黏手的。
   <figure>
    <img data-rawheight="263" data-rawwidth="376" src="https://picx.zhimg.com/50/0589b8b76d87cb402bc968afad6fdabd_720w.jpg?source=1940ef5c" data-original-token="0589b8b76d87cb402bc968afad6fdabd" class="content_image" width="376">
   </figure></li>
  <li data-pid="kLJB4O8X">把面团搓成长条，切成小粒，见上图。在配方以外准备一些淀粉(就是做菜用的生粉)，把做好的芋圆扔进去滚一圈，用面粉筛捞起来，有一定间隔的平铺在烤盘上，放进冰箱冷冻室冷冻至硬，然后就可以装袋扔冷冻室备用了~想吃的时候再拿出来煮。</li>

 <p data-pid="dcfK-abp"><b>椰汁雪花糕(=85°C椰丝小方)~</b>：</p>
 <p data-pid="m8jLszZS">成品实物图：不打水印请勿盗图……</p>
 <figure>
  <img data-rawheight="530" data-rawwidth="544" src="https://picx.zhimg.com/50/263600dc730fa35fa4d9b0711415ef0d_720w.jpg?source=1940ef5c" data-original-token="263600dc730fa35fa4d9b0711415ef0d" class="origin_image zh-lightbox-thumb" width="544" data-original="https://pica.zhimg.com/263600dc730fa35fa4d9b0711415ef0d_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="mACyvTzC"><b><u>材料：</u></b></li>
 </ul>
 <ol>
  <li data-pid="nIMkqnrt">椰浆 200ML</li>
  <li data-pid="Tu8tDsY4">淡奶油 250ML</li>
  <li data-pid="418HLFMx">牛奶 50ML（没牛奶可以用水代替但是前面两种材料不可以替换！）</li>
  <li data-pid="K4NezDgV">糖 40G（一般牛奶勺子大概三勺，可以根据自己口味改变甜度）</li>
  <li data-pid="wxrzEXqD">吉利丁片 4片（20g）</li>
  <li data-pid="cAuInxo-">椰蓉 适量</li>
 </ol>
 <p data-pid="DeJRb6o0">以上均可在某宝买到~椰浆和淡奶油我给出的都是独立包装一盒的量，方便没有量杯的同学。</p>
 <ul>
  <li data-pid="W-5Am2G5"><u><b>做法:</b></u></li>
 </ul>
 <p data-pid="9ld0JETh">简单来说就是三步，<u><b>【加热搅拌——冷藏——切块】<br></b></u></p>
 <ol>
  <li data-pid="uMmiUuPd">吉利丁片用冷开水泡软，会变成类似果冻的一团。</li>
  <li data-pid="Y5DdBGaC">混合椰浆、淡奶油、牛奶、糖，微波炉加热or炉灶加热至糖化。趁热加入泡软的吉利丁片，搅拌至融化。</li>
  <li data-pid="qy3oWnB1">倒入方形盒子(我用的乐扣保鲜盒)，冷藏一晚。</li>
  <li data-pid="HEquxrB3">第二天用刀沿着盒子内壁四周划一圈，摇一下使奶冻脱离模具，倒扣切块，在椰蓉里滚一圈。就做好啦，做好之后再冷藏一下口感更好！夏天吃特别凉爽~</li>
 </ol>
 <p data-pid="6kOAkf8K">TIPS：吉利丁片加的很足，果冻成型好，脱模挺简单的，有必要的可以用上手辅助，实在不行可以在盒子里垫一层保鲜膜……</p>
