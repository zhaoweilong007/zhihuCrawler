# 东鹏特饮外面那层透明的盖子有什么用？
- 点赞数：6109
- 更新时间：2023年06月03日12时10分36秒
- 回答url：https://www.zhihu.com/question/39324406/answer/3029328311
<body>
 <p data-pid="amvNAdri">很多司机就是因为东鹏特饮有这个盖子，才买了这个饮料的。</p>
 <p data-pid="D4XWbOHT">主要作用有∶</p>
 <ol>
  <li data-pid="4n5LvTOM">弹烟灰</li>
  <li data-pid="xYCnzY43">喂狗子喝水</li>
  <li data-pid="Q3HgEmOq">保温瓶的茶倒出来喝没那么烫</li>
  <li data-pid="bpf0VIR-">验尿</li>
  <li data-pid="ImPyc3Bv">有个盖子，瓶口干净很多</li>
  <li data-pid="raozS99h">市面上唯一带杯子的抗疲劳饮料，长途车必备</li>
  <li data-pid="K25GzLK2">钓鱼人调制饵料，发挥了量杯的作用（评论区新增）</li>
 </ol>
 <p data-pid="QP1cXw4A">所以东鹏特饮卖得那么好，一半功劳就是这个盖子。</p>
</body>
