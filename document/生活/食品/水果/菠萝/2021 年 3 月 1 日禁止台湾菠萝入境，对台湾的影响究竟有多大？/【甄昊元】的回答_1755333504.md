# 2021 年 3 月 1 日禁止台湾菠萝入境，对台湾的影响究竟有多大？
- 点赞数：9683
- 更新时间：2021年03月02日15时31分51秒
- 回答url：https://www.zhihu.com/question/446790267/answer/1755333504
<body>
 <p data-pid="1pAVwrcb">涉及到的金额并不多，但是影响很大，很恐慌。后续变数也非常多——两岸的变数都多。</p>
 <p data-pid="GwmztIz5">大陆官方的说法可不是“禁止进口”，而是“暂停入境”。台湾不是外国，不能滥用进口。可以说“入关”，是海关不是山海关。也可以说入境。但是不适合用进口。</p>
 <h2>先说一下事情是怎么发生的：</h2><a href="https://www.zhihu.com/zvideo/1349519269294608384" data-draft-node="block" data-draft-type="link-card"></a>
 <ol>
  <li data-pid="kFZmB6bd">台湾菠萝10%左右用于出外销，外销量并不算很大。外销的菠萝中97%是销往大陆。相当于大陆是唯一的市场。如果大陆停掉它，那就相当于直接弄死台湾菠萝外销之路了。</li>
  <li data-pid="6V-VmJUa">菠萝是10年生作物，一般要两年左右开始结果，然后可以连续采收多年。记住这一点很重要。</li>
  <li data-pid="SQ2XE1ON">菠萝可以说是<b>台毒果</b>。菠萝产地集中在台湾民进党票仓。释迦的产地则以国民党区域为主。菠萝的总量其实不怎么大，但是涉及到的农民却多，算是打击小、影响大的类型。所以禁菠萝确实是吓到民进党了。</li>
  <li data-pid="1ODtFLjz">菠萝入境，主要采取小三通模式。小三通是介于国内和境外之间的一种操作，通关简单，放松。因为一直放的比较松，所以台湾果农和经销商就开始放肆。这些年台湾菠萝、香蕉都出现过严重品质问题。菠萝2015年被检出农残超标7倍。2018年前后香蕉因为品质和病害问题被拒，导致台湾香蕉市场大崩盘。这两年连续发生香蕉烂在地头的事件。</li>
  <li data-pid="dP3Je-jq">去年台湾菠萝检出蚧壳虫，也是这种品控下降的现象。检出蚧壳虫集中在去年三到五月。所以海关在今年同期要求禁入台湾菠萝，这个很合理。</li>
  <li data-pid="0MnYxeY3">菠萝采摘旺季是4-8月。所以海关发的这个消息，真是挺要命的。</li>
  <li data-pid="-aS8elKU">但是我目前判断海关这个操作基本上是技术性而不是政治性的。大陆领导没那么无聊搞什么小动作。要是想搞事，直接封掉台湾海上航路之类的，效果更好。这次操作完全是从生物安全角度考虑的。</li>
  <li data-pid="rAW1-4PR">去年三到五月发现蚧壳虫，海关应该就已经知会台湾了。但是我估计台湾就没当回事。反正去年我们疫情，加上懂王贸易战，台湾怕不是以为中国就要崩溃了。从这次台湾农委会发布会上出来的消息，是台湾直到去年10月19日以后才没再向大陆发“<b>含有蚧壳虫”</b>的菠萝。这说明台湾至少5个月时间是没处理蚧壳虫的问题的。这种态度当然海关总署会很不爽。</li>
  <li data-pid="zk4z7oks">所以在这种情况下，实际上最迟是2月25日（周四）下班之前，海关就给台湾方面发了email，通报了2021年3月1日起禁止菠萝入境的这个处理决定。按照台湾方面说法，是25日傍晚就召开会议处理这个问题。但是直到第二天上午，这个消息也没在台湾放出来。应该就是捂住了。所以大陆这面也没有看到台湾的反应，于是26日中午一过，国台办就发了这个消息。</li>
  <li data-pid="le3KT96b">之所以是国台办发消息而不是海关发消息，估计是因为海关那面没有什么台湾在大陆的媒体管道，发消息不会有影响。国台办有把握把这个消息传回台湾去吧。</li>
  <li data-pid="tzjS30yq">所以国台办发布会结束后（已经是周五下午了），台湾农委会马上召开发布会，主委陈吉仲开始做问题说明。其实这个操作已经是错失了悄无声息解决问题的最佳时机。从这个时间进展上看，陈吉仲接到海关总署的email后，根本就没有上报民进党高层和蔡英文苏贞昌等人。而是准备自己处理。等国台办捅出来以后，实际上已经处理不了了。这个官僚。不过对大陆来说，这个官僚的恰恰好。</li>
 </ol>
 <p data-pid="MHzkwjCL">有空的可以看看下面2015年的这个旧闻。当时台湾人是如何在菠萝里超量使用违规农药的。这事情后来貌似没有下文，没听说台湾谁被追查获罪。</p><a data-draft-node="block" data-draft-type="link-card" href="https://link.zhihu.com/?target=http%3A//www.mnw.cn/xiamen/news/910578.html" data-image="https://pic3.zhimg.com/v2-f3203a2bb0b79f3ad49a7d50f8e4129e_180x120.jpg" data-image-width="1055" data-image-height="140" class=" wrap external" target="_blank" rel="nofollow noreferrer">厦门检出台湾菠萝农残超标 或因台果农偷用禁药</a>
 <p data-pid="MsnRbXdy">其实，大陆自己的菠萝就很好吃，新果马上就要上市了：</p><a data-draft-node="block" data-draft-type="mcn-link-card" data-mcn-id="1349498136738598914"></a>
 <h2>接下来说一下台湾方面是会怎么处理。</h2>
 <ol>
  <li data-pid="bScTUTAX">菠萝事件对台湾社会的震撼是非常大的。因为多年来国台办第一次不那啥了。然后菠萝影响的是大量的农户。甚至是要断绝了这些农户全年的收成。台湾现在的判断极度悲观，一下子就往“彻底封禁”这个方向去了。完全没注意到国台办说的“暂时”禁止的这个用词。</li>
  <li data-pid="sRhS2ORk">这种错误判断，所以台湾压根就没想过要技术性解决这个问题。比如迅速整顿水果流通环节，本地除虫、提高品控。而是先抱怨中国大陆。具体的抱怨包括这几项：</li>
  <ol>
   <li data-pid="pwXTpnIk">我们有虫我们承认，但是有虫的菠萝你们可以用熏蒸杀虫啊，熏蒸杀虫以后就合格了啊；</li>
   <li data-pid="gWDE8WXV">我们卖给你们那么多批次，有虫的毕竟是少数嘛，那你能不能通报哪些批次哪些产地有问题，然后只要禁了它们就好了，不要把台湾的全都禁掉了嘛。</li>
   <li data-pid="X9j5Q-1t">你们这个完全不符合WTO规范嘛，我们要去WTO告你……</li>
  </ol>
  <li data-pid="U0v4nG_z">然后接下来很快就转为政治上的指责。</li>
  <li data-pid="20K7-7IJ">第一种典型的指责是大陆禁止菠萝是政治操作，没有善意；</li>
  <li data-pid="x8fAKf58">接下来就是展现执政党“爱护果农、同仇敌忾”的气质，农委会表示要拿出10亿新台币来补贴果农减少损失，然后就是蔡英文去表演吃菠萝，并且号召台湾人一起帮着吃菠萝。赖清德甚至还计算出来，说“只要每个台湾人每天多吃一两个菠萝”危机就会过去了。</li>
  <li data-pid="3kE3zLOG">然后各种媒体采访就开始展现政治人物是多么爱台湾爱农民，多么积极去帮着吃菠萝。展现台湾人民团结友爱共同啃菠萝的热情。编造出各种煽情懒人图……</li>
  <li data-pid="lHJqZIy4">然后国民党派系的媒体又开始煽动说就是因为台湾对大陆政治操作，导致大陆政治报复禁入菠萝……还是那句话，海关会那么无聊吗？</li>
  <li data-pid="3oYIp_5z">然后台湾上上下下忧心忡忡担心大陆接下来会对其它水果下手，比如释迦、莲雾……名单都替我们拉出来了，就等着大陆这面一一照单禁止，一个一个弄死算了。</li>
  <li data-pid="QoLSYQcH">台湾官方提出一个解决方案：由于大陆禁令，大概会导致5万吨菠萝无法外销。因此要在台湾岛自己解决掉2万吨处理（无非是鼓励大家吃吃吃，以及进行深加工水果罐头果汁之类），然后向大陆以外的国家外销3万吨。但是1个月内组织2万吨处理能力，我不相信。这个世界上除了大陆没有哪个地区能快速部署这个工业能力。后面那个外销3万吨，也不可能，没有任何地方可以在一个月内疏通渠道，并且完成3万吨吃货。</li>
  <li data-pid="k86AwWx7">昨天看到了网上的截图，台湾有说要把菠萝出口到东南亚的……额，俗称的把大同的煤运到平顶山，这次大家有机会看到这种蠢事了。为什么说这是把大同的煤运到平顶山，我不细说了，请看别的回答吧。</li>
 </ol>

 <h2>最后是我的判断：</h2>
 <p data-pid="1VkJXt_K">这两天我发了点关于台湾关于菠萝的东西，顺便也利用这个时间对台湾水果在大陆销售做了一点研究，</p>
 <ol>
  <li data-pid="Ak1PnRYx">基本上大陆这次禁止菠萝，是一个技术性的操作。我不敢说和政治一点关系都没有，但是还是那句话，没那么无聊。政治考量非常小。毕竟这点菠萝没多少钱，省下来的钱都不够一波火箭弹的，更不用说东风了。</li>
  <li data-pid="4acONjfn">按照海关的操作，大概还是希望台湾能解决品控的问题，不要用菠萝向大陆投毒就好，那么禁令几个月后，“暂时禁止”就可以取消，慢慢恢复一下。当然，这个过程中果农还是会受到损失，但是不会死光，还能缓过气来。</li>
  <li data-pid="QZamfKYI">但是如果台湾要把这件事上升到政治事件，那这个问题就没有技术解决的可能了，如果台湾炒作大陆政治打压菠萝，那我们为了维护海关尊严，只好把这个事情坚持下去。“暂时禁止”就得慎重重开，这一慎重没准就得好多年。等台湾菠萝田都铲光，也许重开有望。</li>
  <li data-pid="f2cxxzBB">国台办开会后，其实大陆官方就没有什么新声音了。只有一些半官方的媒体转载发布了这个消息，有不咸不淡的几句评论。但是民间的声音可就热闹了，我看了，基本上百分之百是支持国台办和海关的，而且要求把穷台政策进行到底，但其实国台办并没有准备“穷台”的脚本。</li>
  <li data-pid="czaNrmDH">但是对民间商人的影响也很大。 <a class="member_mention" href="https://www.zhihu.com/people/077beabaec12ddc4f99fbdc0e7967586" data-hash="077beabaec12ddc4f99fbdc0e7967586" data-hovercard="p$b$077beabaec12ddc4f99fbdc0e7967586">@古青</a> 观察发现市场上台湾释迦一夜之间从21元降价到16元，那就应该是水果商急于出清库存，避免接下来的风险。</li>
  <li data-pid="q_DCwDiy">今天（2月28日）爆出某大陆水果商通知台湾果农，一笔4000公斤的莲雾暂停向大陆发货。这个把台湾吓坏了，好多果农去问，政府官员也上蹿下跳，最终证实这只是这个水果商个人行为，也只是一个孤例，不代表大陆官方的意思。但是相关的讨论也开始多起来了。这个我觉得也是水果商担心风险，暂停发货准备观望一下的意思。但是也确实很吓人。</li>
  <li data-pid="0Kg-T6gh">据台湾那面说因为莲雾保鲜期很短，所以如果大陆停掉莲雾，造成的损失会更大。</li>
  <li data-pid="G7-x8FrJ">虽然大陆官方没有进一步禁止什么水果和其它产品进入大陆市场的新消息，但是菠萝事件确实挑起了大陆人民群众的热情，我已经收到很多私信表示说如果在超市看到台湾水果，就绝对不会买。这种表态实际上是可以反过来影响渠道商的。那么如果在（对岸的）政治操作下，这种情绪发酵，虽然海关不禁，但是客观上民间是可以完全禁掉台湾的很多种类产品的。那就热闹了。</li>
  <li data-pid="YrcBCuDZ">相信不少人和我一样，这几天正在搜索台湾向大陆出口的产品都有哪些，在消费领域上，应该快有人推出全新的“台湾出品”的清单了，估计再有几天我就能看到这份名单。这份名单也一定是可以上热搜的名单。而且，这个名单的影响才会很大。</li>
  <li data-pid="ZCOoZBuI">比菠萝大得多。</li>
 </ol>
