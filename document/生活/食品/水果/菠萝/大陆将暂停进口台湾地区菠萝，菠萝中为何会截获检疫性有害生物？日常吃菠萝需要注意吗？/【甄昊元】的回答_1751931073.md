# 大陆将暂停进口台湾地区菠萝，菠萝中为何会截获检疫性有害生物？日常吃菠萝需要注意吗？

- 更新时间：2021年02月27日23时08分58秒
- 回答url：https://www.zhihu.com/question/446510247/answer/1751931073
<body>
 <p data-pid="gfoMHQQf">暂停进口的禁令出台，台湾反应很快、也很大，指责大陆政治操作的声音也出来了，我觉得有必要把这件事情的始末认真帮大家捋一下：</p><a href="https://www.zhihu.com/zvideo/1349058166789943296" data-draft-node="block" data-draft-type="link-card"></a>
 <ol>
  <li data-pid="qWQtnZ1G">台湾菠萝大肆进入大陆，然后取得辉煌的市场成绩，其实一个重要的契机是2012年中国在菲律宾进口的香蕉和菠萝中检测到有害虫，于是停止了菲律宾的上述水果的进口，从此给台湾菠萝进口提供了契机。<sup data-draft-node="inline" data-draft-type="reference" data-numero="1" data-text="中国禁止菲律宾水果进口，台湾菠萝趁机补缺" data-url="https://taiwan.huanqiu.com/article/9CaKrnJvphW">[1]</sup></li>
  <li data-pid="ndcqPi4f">2014年以后，台湾菠萝对中国大陆的销售增速极快，从2014年的9000吨到2019年接近4万吨，高峰年增速甚至达到40%。</li>
  <li data-pid="f64kE0TC">快速增长的菠萝市场，当然给台湾菠萝农户带来了好日子。可以说转的盆满钵满。近两年每年台湾菠萝销往大陆销售额达到15亿元。</li>
  <li data-pid="tEHQNdZO">但是随着销售数量快速增加，台湾菠萝品质其实出现了快速下降的趋势。台湾学者焦钧在2019年曾经提出，台湾菠萝质量控制越来越糟，连日本专家都忧虑，台湾菠萝好光景剩不到几年。<sup data-draft-node="inline" data-draft-type="reference" data-numero="2" data-text="台湾学者担忧台湾菠萝品控下降" data-url="https://taiwan.huanqiu.com/article/9CaKrnKibVV">[2]</sup></li>
  <li data-pid="Bg-_sx1-">2015年，大陆海关就检出台湾菠萝农残超标<sup data-draft-node="inline" data-draft-type="reference" data-numero="3" data-text="台湾菠萝黑历史：农残超标7倍" data-url="http://news.sohu.com/20150519/n413368428.shtml">[3]</sup>。从此提高了对台湾菠萝的抽检比例。</li>
  <li data-pid="ujMDWo2j">此次海关总署提出暂停台湾菠萝入境的理由，是2020年内，发现台湾水果多个批次出现有害生物，根据台湾农委会方面新闻发布会的消息，这些有害生物是三种危害极大的蚧壳虫。</li>
  <li data-pid="bftEoGVf">目前掌握消息是，2020年内，台湾最少有13个批次，大概超过400吨菠萝是有蚧壳虫的。</li>
  <li data-pid="QDyKO4Lh">根据台湾方面的反应来看，大陆海关在2020年应该已经向台湾方面提出相关问题，台湾可能也进行了响应的整改。据农委会的一位女士说，台湾自2020年10月下旬起的菠萝就开始没有蚧壳虫类有害生物。这个说法就字面来说也许也没什么毛病，但有一个问题，就是菠萝的旺季是4月到8月期间，11月以后的秋果和冬果产量本来就低，同时由于季节的原因，秋冬季节蚧壳虫病害本来也就小得多。所以这个说法是回避了问题关键。</li>
  <li data-pid="NCioPGbH">有害生物入侵对环境、农业、人民健康影响很大。中国国土幅员辽阔，有独特的生态氛围，但是也容易受到外来生物入侵的影响。统计表明已经有超过400中生物入侵，每年给中国造成损失超过1300亿元。<sup data-draft-node="inline" data-draft-type="reference" data-numero="4" data-text="有害生物入侵给中国造成巨大损失" data-url="https://bbs.pinggu.org/jg/jingji_shijiejingji_602988_1.html">[4]</sup></li>
  <li data-pid="QFwWatkW">鉴于台湾菠萝多批次出现有害生物，暂停台湾菠萝入境，是一种必要处理方法。这种禁止入境可以让农产品运营方严肃对待相关问题，做好品质整改。</li>
  <li data-pid="vZYACap-">台湾菠萝的主要产地集中在中南部地区，包括台中、台南、高雄等地，这些地区是民进党的铁票仓，所以菠萝禁入，对这些地区农业打击和民进党票仓打击其实挺大的。</li>
  <li data-pid="3B2KA6yC">前面我们说了，菠萝采摘的旺季是4月到8月。所以如果3月开始禁止入境，实际上对全年菠萝的销售影响极大。4月果实就要采摘，一个月左右的时间，台湾根本没有足够的时间和资源去安排这些菠萝的销售。还有一种解决办法是深加工，但是一个月时间完成菠萝的深加工，这一样不是台湾能做到的。</li>
  <li data-pid="XrG0rXJM">台湾菠萝外销极度依赖大陆，有多依赖呢？外销菠萝销往大陆占比达到97%，销往日本2%，销往香港1%。所以大陆如果断掉入境，相当于直接给台湾菠萝判了死刑。</li>
  <li data-pid="mau_QE5l">除了中国大陆，你根本找不到一个市场，可以快速消化掉超过3万吨菠萝。对中国大陆来说，3万吨菠萝，相当于每个人吃不到一小口。</li>
  <li data-pid="YFH0-Nh0">所以出这么大事儿，台湾方面根本没法交到。第一时间就召开了新闻发布会，然后好像到现在已经全面动了起来，除了农委会以外，蔡蛙英文、苏蛙贞昌、各种大大小小的政客、各路媒体网红等等，都开始行动起来表态。</li>
  <li data-pid="ZOAryXpB">新闻发布会上，那个女的官员说，我们出口到大陆的菠萝不是每个批次都有有害生物，只有少数的一些批次有这样的问题。没有必要因为这个少数批次禁止掉所有的台湾菠萝。还说如果发现蚧壳虫，用熏蒸方法杀虫就能解决，就可以合格了。反正我是没听说过海关还要负责帮助进行熏蒸的。所以你也不知道菠萝里混了蚧壳虫，是故意的还是故意的？</li>
  <li data-pid="1ukpsojR">农委会主委陈吉仲在发言中已经语无伦次，谈到大陆的时候提出大陆去年疫情、水灾、集装箱不足、海运费用上调，预计2021年大陆粮食仍有危机，这个时候禁止台湾菠萝很不明智。</li>
  <li data-pid="T8e92bsh">2019年开始，全球菠萝市场需求出现了下行。主要包括美国菠萝需求下降、荷兰市场低迷、菲律宾菠萝价格下降等。</li>
  <li data-pid="bkYV2HrX">目前台湾已经乱成一锅粥，估计又开始要推爱心菠萝之类的项目，号召全民吃菠萝，以稳定市价、保障农户的收入。但是这种行动无济于事，基本上都还是一些政治人物作秀，安抚民心之举。</li>
  <li data-pid="pdY7xhyR">台湾一些人叫嚣要去WTO告中国。这个也不是不行，不过WTO的总干事貌似还都没有正式就任<sup data-draft-node="inline" data-draft-type="reference" data-numero="5" data-text="空悬许久的WTO总干事刚刚就任" data-url="https://m.sohu.com/coo/hsdt/450959518_167569">[5]</sup>，WTO去年被特朗普搞到瘫痪，就算人员到位，积压的案件很多，这样一笔15亿元人民币的案子，无论怎么也排不到前面去。就WTO的工作效率，这事要最后审结，怎么也得两年以后了。两年以后，别说菠萝，苏贞昌的骨灰都不一定在哪里了。</li>
  <li data-pid="ePJ34jPD">还有人指责说中国大陆去年发现有害生物，为什么不是去年禁止入境，而选择了当下。首先，去年肯定是有照会之类的发文。但是去年下半年没有在菠萝上和台湾纠缠，当然是因为去年的时候，特朗普蓬佩奥都在台上，和台湾沆瀣一气准备咬我们。这不是拜登上台一个月了，我们终于腾出手来处理这个海关遗留问题了。</li>
  <li data-pid="xQHyQTsf">暂停菠萝入境，这个操作可大可小。台湾方面显然准备把这一事件升级为政治事件，指责中国大陆不遵守WTO的公平贸易规矩，问题是，2019年以来，台湾强行下架的大陆产品和企业也很多，那个时候没听说台湾要援引WTO原则来处理这些问题。大陆这次禁止菠萝入境，留有非常充分的余地，表面上的理由仍然是有害生物。有害生物的处理遵循一定规程，是可解的。但是如果台湾要用政治手法来处理，最终是不可解的。那就只好学一句《赘婿》宁毅的话“春天来了，让台湾的菠萝破产吧”。</li>
  <li data-pid="PBOU249G">很多人猜测大陆这次行动具有政治上惩戒的意味，这个我不能确定。不过如果把这次行为看作是一种惩戒，那从台湾角度讲，这算是安知非福的一次事件。毕竟，如果大陆用穷台的方法来玩你，就意味着扔火箭弹的可能性降低了。当然，我喜欢火箭弹洗地的方案。我这个人特别爱洗地，我愿意用火箭弹给蔡英文卧室洗地……</li>
  <li data-pid="AMc4DrNl">这一事件，大陆方面是本着技术处理的态度，所以走的是海关总署-国台办的路线。其实如果按照这个路线应对 ，事情有转机。但是台湾人显然不会，他们一定认为，“我对你凶，你们自然就会软”……好吧……</li>
  <li data-pid="GGhTvTKS">这个事件的另一个有趣之处在于，海关总署的决定是2月25日下午发到台湾农委会的。农委会据说连夜研究这份文件了。但是到了26日上午，台湾风平浪静，好像根本没发生这件事一样。大概这些台湾人认为，只要不予理会，大陆也就是发个文而已。所以在一上午风平浪静以后，国台办可能是没收到反馈意见，真忍不住了，于是开了这么个新闻发布会，于是破岛一下午鸡飞狗跳……第二天是周末，然后就是三月一号了，接下来大陆两会12天。差不多两周时间，你想找人都找不到。这个时机确实玄妙。</li>
  <li data-pid="_iaHqGlQ">很多人在猜测菠萝是不是只是一个开始，后续是不是有别的产品，农委会貌似已经开始排查所有产品对大陆的依赖了……但是其他生产制造部门貌似还没有进行类似的工作。</li>
 </ol>
