# 为什么老外喜欢吃鸡胸？
- 点赞数：7234
- 更新时间：2019年09月25日10时54分09秒
- 回答url：https://www.zhihu.com/question/39441338/answer/834966120
<body>
 <ol>
  <li data-pid="nlzxNqQu">做法简单。我一个美国哥们Ken，每周上超市买一大包冷冻鸡胸，一大包冷冻西蓝花，午饭和晚饭内容三年不变：平底锅煎鸡胸肉和西蓝花，做法就是小火煎10分钟，闹表响后到厨房翻面，再煎10分钟，之后蘸酱吃。如果换了鸡其他部位的肉，那都要剃骨头的，用时可就大不一样了，因为至少要先化冻，然后上刀，没点手艺还可能受伤。但是牛肉也有整块不用切的啊，这就联系到了第二个原因</li>
  <li data-pid="APFthKi6">便宜。鸡肉是主流肉类里单价最便宜的了，只是牛肉的几分之一</li>
  <li data-pid="SGTvU28h">吃起来安全。美国人吃东西也挺笨的，每年好多人吃花生噎死，也不咋会吐鱼刺，特别怕鱼刺、鸡骨头卡嗓子。美国的饭店里都不敢做有刺的鱼，万一卡了嗓子，饭店容易摊上官司。鸡胸肉也占了这个便宜，里面一点骨头没有，不会出事故</li>
 </ol>
