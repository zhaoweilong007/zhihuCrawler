# 如果下个月世界大战，你最想做什么？
- 点赞数：89583
- 更新时间：2016年11月04日16时58分46秒
- 回答url：https://www.zhihu.com/question/40350229/answer/87428865
<body>
 <p data-pid="HU9Id9qi"><b>注意,以下物品要秘密搞来.不要一下买一堆,少量多次.也不要在一个地方采购.</b></p>
 <p data-pid="QceJMAe3">开战前:</p>
 <ul>
  <li data-pid="snDSPUL7">储备肥皂,剃须刀片,糖,盐,火柴,蜡烛,针线,牙膏,净水器,<b>维生素</b>,<b>各类药品</b>,以及卫生巾等生活用品.不止可以自己用,还可以进行交换.<br></li>
  <li data-pid="gvsQaaqY">储备种子.建议小麦,好保存.土豆也不错,成熟快.<br></li>
  <li data-pid="YSrlxZZh">一些金属和食物可能被用于战争.战前做好回收,开战后可以卖掉.<br></li>
  <li data-pid="ep7FLiGj">收音机和电池,手摇式收音机也不错.如果有条件,可以买一部电台.每天收听国际新闻.互联网不安全,很容易得到你的个人信息.<br></li>
  <li data-pid="9F8OuKAP">把钱换成贵金属,妥善保存.但是注意,<b>有些时候不要指望金子能换成食物</b>.<br></li>
  <li data-pid="lTshsnpc">为孕龄女性储备避孕药.<br></li>
  <li data-pid="6o2n07Bm">准备优质的自行车以及备用零件,轮胎,维修手册.可以在自行车上喷一些暗色的漆,让它显得更破旧.<br></li>
  <li data-pid="P2VgLGfp">准备电离辐射测量仪.</li>
  <li data-pid="NeTsfiZD">搭建掩体.尽可能建在农村.<br></li>
  <li data-pid="we1EmiZr">把掩体刷成白色,换上铅玻璃,内部铺上铅版和钢板并接地.白色是为了反射核战的热辐射,铅版防电离辐射,钢板防EMP.<b>EMP可以毁掉几乎一切借助晶体管电子设备</b>,应该将<b>电脑,收音机等妥善保存</b>在接地的密封钢箱子里.实际上,你的汽车估计也不能用了.</li>
  <li data-pid="n31TG83O">准备各类科学技术,医药等书籍(入门级).可以搜电子版存到微缩胶片里,<b>妥善保存</b>.</li>
  <li data-pid="b-C8ozx2">准备碘化钾和半胱氨酸.服用碘化钾防放射性碘,核爆后应服用两个月.<b>核爆后立即服用</b>(四小时内才有效),头两天服用1000mg,之后一周500mg,剩余每日100mg.半胱氨酸也可以预防一定辐射,尽早服用.</li>
 </ul>
 <p data-pid="yZwc3Pua">战败的话:</p>
 <ul>
  <li data-pid="8kiWW4oJ">决定要不要出国,通常刚开始占领者对边境管的比较松.<br></li>
  <li data-pid="X-h_Av8t">如果你是党员,你可能会很危险.尽可能换个身份.<b>并不是说背叛,而是保存力量</b>.与其送死,不如活着.<br></li>
  <li data-pid="5EFXAZn_">穿破旧的衣服,值钱的东西都别带.<br></li>
  <li data-pid="7C7DfExK">藏好你的财物.<br></li>
  <li data-pid="ewa98sMS">让女性不要上街.如果在一楼也要注意.历史上这时候会发生大规模强奸.通常是几个士兵冲入房内,拿枪抵住男人.然后进行强奸.如果男主人反抗,就会被杀死.<br></li>
  <li data-pid="GUBSUzaB">如果遭到了抢劫或强奸,不要向占领当局举报.他们很可能会把你列入一个名单里,派警察秘密监视.</li>
  <li data-pid="be8ZHQur">学习占领者的语言,这很重要.</li>
  <li data-pid="yjmkPtRR">避免和占领者发生冲突,尽量回避.</li>
  <li data-pid="yTm9nrbf">不要做傻事,将来有更好的机会.</li>
  <li data-pid="wN0jp7yZ">不要在小孩面前谈论侵略者,他们在学校可能被盘问.</li>
  <li data-pid="dF8dtZMH">随身携带一个小包.里面装着保暖的衣物,高热量食物和常见药品.这是因为侵略者可能把你带到集中营.</li>
 </ul>
 <p data-pid="RQAVKLh6">关于食物,<b>最低需求</b>:</p>
 <ul>
  <li data-pid="U6PbYCNG">水,400千克/季度.(应至少每三个月更换)</li>
  <li data-pid="-OXeJt07">小麦(主要提供糖类,且蛋白质和微量元素丰富),每人每年165千克.</li>
  <li data-pid="XAHV1x5g">食用油(提供脂肪.建议菜籽油,脂肪酸比例合理,且价格低廉),每人每年23千克.</li>
  <li data-pid="wwFxTw9s">盐,每人每年450g.</li>
  <li data-pid="7Lcb6ucC">复合维生素,每人每年365片(具体见说明书)</li>
 </ul>
 <p data-pid="ZGzq2awC">预计每人每年1600元.</p>

  <li data-pid="qA6v-ncV">针线.</li>
  <li data-pid="2A5KenV6">如果你有慢性病,请准备好药.</li>
 </ul>
