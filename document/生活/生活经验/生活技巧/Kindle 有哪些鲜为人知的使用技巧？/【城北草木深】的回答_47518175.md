 <p data-pid="VcOwImay">补充：看评论里大家关注的重点很多微信公众号的推送,下面说下微信公众号的rss制作和推送。<br>
  ==============================================================<br>
  1.如果用狗耳朵<a href="https://link.zhihu.com/?target=http%3A//www.dogear.cn/" class=" wrap external" target="_blank" rel="nofollow noreferrer">狗耳朵 - 全文RSS和微信公众号推送服务</a>的收费服务，狗耳朵提供一些有名的公众号的订阅。缺点是不全，不一定包含你需要的。最近服务器不稳定。<br>
  2.有专门针对微信订阅号推送到kindle的收费服务，<a href="https://link.zhihu.com/?target=http%3A//www.weikindle.com/na/home.htm" class=" wrap external" target="_blank" rel="nofollow noreferrer">WeiKindle - 提供为Amazon Kindle推送微信公众号文章的服务</a>和<a href="https://link.zhihu.com/?target=http%3A//www.taorenqi.com/" class=" wrap external" target="_blank" rel="nofollow noreferrer">TaoRenQi - 为Amazon Kindle订阅推送微信公众订阅号文章服务</a> 但是貌似都不成熟，还都不能用。<br>
  3.有些你喜欢的微信订阅号，需要自己制作了。主要是在第三方的微信平台上找，看有没有收录你需要的订阅号。制作的方法就是feed43+yahoo pipes。这些平台有：<br>
  微广场：<a href="https://link.zhihu.com/?target=http%3A//www.iwgc.cn/" class=" wrap external" target="_blank" rel="nofollow noreferrer">微信公众号列表</a> <a href="https://link.zhihu.com/?target=http%3A//www.vccoo.com/" class=" wrap external" target="_blank" rel="nofollow noreferrer">微口网 - 热门微信文章和公众号大全</a> 本身提供rss摘要。<br>
  爱微帮 <a href="https://link.zhihu.com/?target=http%3A//www.aiweibang.com/" class=" wrap external" target="_blank" rel="nofollow noreferrer">微信运营，自媒体社交媒体运营好帮手</a> <a href="https://link.zhihu.com/?target=http%3A//www.lewuxian.com/" class=" wrap external" target="_blank" rel="nofollow noreferrer">乐无线-微信公众平台精品内容聚合服务</a> <a href="https://link.zhihu.com/?target=http%3A//chuansong.me/" class=" wrap external" target="_blank" rel="nofollow noreferrer">传送门 -- 微信公众账号和文章的导航及推荐</a> 这个问过开发者了，不允许rss订阅</p>
 <p data-pid="bd-LVB3w">补充2：微信公众账号的rss哪里找？我主要用微口网和微广场：<a href="https://link.zhihu.com/?target=http%3A//www.vccoo.com/" class=" wrap external" target="_blank" rel="nofollow noreferrer">微口网 - 热门微信文章和公众号大全</a> <a href="https://link.zhihu.com/?target=http%3A//www.iwgc.cn/" class=" wrap external" target="_blank" rel="nofollow noreferrer">微信公众号列表</a> <a href="https://link.zhihu.com/?target=https%3A//chi.miantiao.me/zhihu-daily-rss/" class=" wrap external" target="_blank" rel="nofollow noreferrer">知乎日报 RSS</a></p>

 <p data-pid="1U7rG8Wz">补充3：评论里好多人说看不懂，彻底的小白这么做：</p>
 <ol>
  <li data-pid="RlaRSbhC">要么选择kindle4rss的服务<a href="https://link.zhihu.com/?target=https%3A//kindle4rss.com/" class=" wrap external" target="_blank" rel="nofollow noreferrer">Kindle4RSS: RSS Feed Reader for Kindle</a>，要么选择狗耳朵的服务：<a href="https://link.zhihu.com/?target=http%3A//www.dogear.cn/" class=" wrap external" target="_blank" rel="nofollow noreferrer">狗耳朵 - Kindle推送服务</a>（两个都收费，一个月几块吧很便宜），注册一个kindle4rss或者狗耳朵的帐号</li>
  <li data-pid="UW4MCCbA">然后把kindle4rss或者狗耳朵给你的推送邮箱添加到亚马逊的邮箱信任列表里kindle4rss的设置页面：狗耳朵的设置页面如下亚马逊的设置页面，在<a href="https://link.zhihu.com/?target=http%3A//www.amazon.cn/" class=" wrap external" target="_blank" rel="nofollow noreferrer">亚马逊(中国)</a>注册一个帐号，并在你的Kindle上用这个帐号登录。在亚马逊网站打开我的帐户下拉菜单 -&gt; <a href="https://link.zhihu.com/?target=https%3A//www.amazon.cn/mn/dcw/myx.html/ref%3Dkinw_myk_redirect%23/home/settings/payment" class=" wrap external" target="_blank" rel="nofollow noreferrer">管理我的内容和设备</a>, 点击设置</li>
  <li data-pid="sRtlTeUW">找一个rss，添加到你的kindle4rss或者狗耳朵账户，手动推送一下，成功的话，就添加自己喜欢的rss地址吧</li>
 </ol>
