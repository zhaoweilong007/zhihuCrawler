 <h1><strong>1. 数学刷题</strong></h1>

 <p>为什么会有即使用了题海战术也还是没有效果<span><span>？</span></span></p>
 <p><strong>因为基础都没有打牢<span><span>，</span></span>刷再多的题也没有用</strong><span><span>。</span></span></p>
 <h2><strong>1.1 数学怎么提高<span><span>？</span></span></strong></h2>
 <p>核心观点<span><span>：</span></span><strong>基础夯实+刷题+整理反思</strong></p>
 <h2><strong>1.2 各个步骤如何落实<span><span>？</span></span></strong></h2>
 <p>首先咱们得明白为啥一道简单的题看一眼就知道答案<span><span>，</span></span>而一道相对复杂的题看了几眼也搞不出答案<span><span>？</span></span></p>
 <p>因为简单题只有一个或两个知识点<span><span>，</span></span>但是一道所谓的难题包含了很多个知识点以及不同的方法<span><span>。</span></span></p>
 <p><strong>由此看来<span><span>，</span></span>深刻理解知识点+建立知识点的连结非常重要<span><span>。</span></span></strong></p>
 <p><strong>1.2.1 过基础</strong></p>
 <p>①在这个阶段<span><span>，</span></span>需要着重过一下自己的基础知识<span><span>（</span></span>可以参考一轮复习资料<span><span>）</span></span><span><span>，</span></span><strong>深刻理解基础概念<span><span>，</span></span>一定得一步一个脚印<span><span>，</span></span>不能认为自己懂了一点就对这个概念全部理解</strong></p>
 <p>其中<span><span>，</span></span>需要<strong>着重注意的是自己掌握不牢的<span><span>、</span></span>容易出错的点以及高考考点</strong><span><span>。</span></span></p>
 <p>②光是理解还不够<span><span>，</span></span>还需<strong>学会应用</strong><span><span>，</span></span>建议找到这个知识点涉及的题目<span><span>，</span></span>看完对应知识点后<span><span>，</span></span>做一做关于该知识点的题目<span><span>。</span></span></p>
 <p><span><span>（</span></span>建议<strong>可以找一些高考题进行练习</strong><span><span>，</span></span>看看高考题对于不同知识点是如何考察的<span><span>，</span></span>也可以用<strong>小题狂练那种分知识点模块</strong>的参考书<span><span>）</span></span></p>
 <p>③如果感觉这个知识点还不够牢固<span><span>，</span></span>建议<strong>反复多次过概念+对应练习巩固</strong></p>
 <p><strong>1.2.2 刷题</strong></p>
 <p>个人感觉我的数学能在短期内迅速提升<span><span>，</span></span>离不了我给自己制定的<strong>刷题计划</strong><span><span>，</span></span>这个制定建议分为两种<span><span>：</span></span></p>
 <p>①刷基础题</p>
 <p><strong>基础<span><span>，</span></span>是重中之重</strong><span><span>，</span></span>纵观近几年的高考题<span><span>，</span></span>大家也可以发现高考题也是以基础为主<span><span>，</span></span>因此<strong>每天做点基础题非常重要</strong><span><span>，</span></span>可以根据自己地区的高考卷结构制定自己的基础题刷题计划<span><span>。</span></span></p>

 <p>②刷难题</p>
 <p>基础打牢了还不够<span><span>，</span></span>不足以每次拿到 120+的成绩<span><span>，</span></span>咱们还需<strong>要进一步提升自己的能力</strong><span><span>，</span></span>这个时候推荐可以刷一些难题<span><span>。</span></span></p>
 <p>所谓难题<span><span>，</span></span>一般是圆锥曲线和导数的最后一问<span><span>、</span></span>选择最后两个以及填空最后一个<span><span>，</span></span>但是这些题是<strong>有套路</strong>的<span><span>，</span></span>特别是求导的最后一个题有几个常见的方法<span><span>，</span></span>多积累<span><span>、</span></span>多理解<span><span>，</span></span>在考试里面如果一下看不出<span><span>，</span></span>还可以一个一个去试<span><span>，</span></span>找到最佳方法<span><span>。</span></span></p>
 <p>因此<span><span>，</span></span>刷一些这样的难题很有必要<span><span>。</span></span></p>

 <p>难题不在多<span><span>，</span></span>在精<span><span>。</span></span></p>
 <p>但是刷难题有以下<strong>注意事项<span><span>：</span></span></strong></p>
 <ul>
  <li>每一道难题<strong>定时完成</strong></li>
 </ul>
 <p>一般来说<span><span>，</span></span>难的大题做题时间大概在 20min 左右<span><span>，</span></span>难的选填 5～8min 左右<span><span>。</span></span><span><span>（</span></span>主要看个人的习惯与掌握情况<span><span>）</span></span></p>
 <p>难题为什么难<span><span>，</span></span>因为需要大家在<strong>有限时间</strong>里想清楚比较复杂的概念和运用恰当的方法<span><span>，</span></span>所以无意义地在上面花太长时间并不会有益于自己提升<span><span>。</span></span></p>
 <p>因此切记<span><span>，</span></span><strong>不要在高三后期一道题僵持个一两个小时<span><span>，</span></span>非常浪费时间<span><span>。</span></span></strong></p>
 <p>建议<strong>可以在不会的题旁边写下大致思路</strong><span><span>，</span></span>对答案方便对比自己的思路与标答的思路<span><span>，</span></span>也很有效率<span><span>！</span></span></p>
 <ul>
  <li>有一些比较清奇的思路<span><span>，</span></span>可以<strong>积累总结一下</strong></li>
 </ul>
 <p>比如<span><span>，</span></span>今天导数遇到个下辈子都想不出来的精巧思路<span><span>，</span></span>花个 10min 积累一下<span><span>，</span></span>考试遇到了睡觉都笑醒<span><span>，</span></span>哈哈哈哈<span><span>！</span></span></p>
 <p>但是注意哦<span><span>，</span></span><strong>积累一定要有条理</strong><span><span>，</span></span>不要想到什么写什么<span><span>，</span></span><strong>尽量分门别类<span><span>，</span></span>这样复习回看的时候也不会想看的找不到<span><span>，</span></span>看到的缺乏系统和条理<span><span>。</span></span></strong></p>
 <p><strong>1.2.3 整理反思</strong></p>
 <p><strong>①错题本</strong></p>
 <p><strong>数学非常需要整理错题<span><span>！</span></span><span><span>！</span></span><span><span>！</span></span></strong>千万不要为了偷懒<span><span>，</span></span>就省略这个步骤<span><span>，</span></span>整理的题可以作为自己查漏补缺的重要材料<span><span>。</span></span></p>
 <p>整理错题需要注意以下几点:</p>
 <ul>
  <li><strong>不要抄题<span><span>，</span></span>直接裁题</strong><span><span>！</span></span></li>
 </ul>
 <p>我记得以前班主任说过一句话<span><span>，</span></span>别舍不得裁你做过的题<span><span>，</span></span>大部分同学回看这些练习册的几率为 0<span><span>，</span></span>所以咱班同学都是错题直接裁<span><span>。</span></span></p>
 <p>而且啊<span><span>，</span></span>抄一道题可能抄错而且还浪费时间<span><span>，</span></span>所以直接裁吧<span><span>，</span></span><strong>省时又省力</strong>～</p>
 <ul>
  <li><strong>正面贴题<span><span>，</span></span>反面写答案和简单步骤</strong></li>
 </ul>
 <p>重做错题时<span><span>，</span></span>看着正面就 OK 啦<span><span>，</span></span>在草稿本上好好算算<span><span>，</span></span>其中注意一<strong>点<span><span>，</span></span>一道题如果做对了三次以上<span><span>，</span></span>这道题可之后不再</strong>重做<span><span>。</span></span></p>
 <ul>
  <li><strong>一般来说<span><span>，</span></span>高考前 20 天左右就不再开新题<span><span>，</span></span>应把重心放在自己的错题上</strong></li>
 </ul>
 <p>每天找个时间<span><span>，</span></span>静下心来好好做错题<span><span>，</span></span>每次大概 40min<span><span>，</span></span>一定要认真对待这个行为<span><span>。</span></span></p>
 <p><strong>因为不认真搞<span><span>，</span></span>可能这次错<span><span>，</span></span>下次还错<span><span>，</span></span>下下次还错<span><span>。</span></span></strong><span><span>（</span></span>泪目的亲身经历<span><span>）</span></span></p>
 <p><strong>②应急本</strong></p>
 <p>你是不是有这样的问题<span><span>：</span></span></p>
 <ul>
  <li>考前复习很迷茫<span><span>？</span></span></li>
  <li>考前担心知识点记不住<span><span>？</span></span></li>
 </ul>
 <p>那<strong>你现在需要的是一个应急本</strong><span><span>。</span></span></p>
 <p>嘿嘿<span><span>，</span></span>是不是很懵<span><span>，</span></span>什么是应急本<span><span>？</span></span></p>
 <p>其实很简单<span><span>，</span></span>应急本的内容主要来自于<strong>对之前考试的分析</strong><span><span>，</span></span>包括<strong>薄弱知识点总结</strong><span><span>、</span></span>自己的常见错误总结<span><span>（</span></span>包括审题问题<span><span>、</span></span>不仔细<span><span>、</span></span>对某个概念不熟等等<span><span>）</span></span><span><span>。</span></span></p>
 <p><strong>应急本上收录的内容建议大伙在考前再看一遍<span><span>，</span></span>以加深印象<span><span>，</span></span>特别是自己容易遗忘<span><span>、</span></span>记混的概念～</strong></p>
 <p>这个本子可以从高三一直到高考<span><span>，</span></span>很有用<span><span>，</span></span>当时我和同桌都用了这个方法<span><span>，</span></span>每次考试前我们在别的同学比较浮躁不安的时候<span><span>，</span></span>安安静静地复习<span><span>，</span></span>真的能稍微缓解一点考试焦虑<span><span>。</span></span></p>
 <p><strong>1.2.4 小小经验</strong></p>
 <p><strong>最后想分享一下我高考失误的血的教训<span><span>，</span></span>唉<span><span>，</span></span>本来水平在 130+最后只考了 120<span><span>，</span></span>希望我的一些失败教训能给大家一些启发<span><span>。</span></span></strong></p>
 <ul>
  <li><strong>高考时<span><span>，</span></span>一定不要赶时间<span><span>！</span></span>保证能拿的分<span><span>，</span></span>一分不少<span><span>！</span></span></strong></li>
 </ul>
 <p>我高考的时候就是急着做难题去了<span><span>，</span></span>唉<span><span>，</span></span>难题虽然比平时做的好<span><span>，</span></span>但是选择题前 10 个<span><span>，</span></span>错了 3 个<span><span>。</span></span></p>
 <ul>
  <li><strong>高考时<span><span>，</span></span>不要打破自己的常规<span><span>！</span></span></strong></li>
