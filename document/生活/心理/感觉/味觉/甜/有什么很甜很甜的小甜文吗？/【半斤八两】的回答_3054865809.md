# 有什么很甜很甜的小甜文吗？
- 点赞数：14863
- 更新时间：2023年08月11日11时28分20秒
- 回答url：https://www.zhihu.com/question/584983180/answer/3054865809
<body>
 <p>我喝嗨了背着经纪人开直播。</p>
 <p>弹幕里提到京圈太子爷，我上头：「沈辞吗？谈过。」</p>
 <p>「就是年纪太小些，我喜欢大点的。」</p>
 <p>下一秒，我喜提微博热搜。</p>
 <p>「姜晚乔可是什么都敢说，什么都敢蹭啊！」</p>
 <p>「敢蹭沈总，恭喜姜晚乔喜提律师函。」</p>
 <p>隔天，沈辞找到我眼尾泛红，咬牙切齿：「当初甩我就是因为这个？」</p>
 <p>这还是我那清冷霸总吗？</p>
 <p>1</p>
 <p>尽管凌晨四点，经纪人冉姐还是杀来了我家。</p>
 <p>对上冉姐铁青色的脸，我瞬间酒意全无。</p>
 <p>我打着哈哈：「口误，真的是口误！」</p>
 <p>冉姐摁灭了指尖烟，一副悔不当初的模样。</p>
 <p>「姜晚乔，你可真是把脑子泡进酒里了，什么都敢说啊。我怎么就带了你这么一个没带脑子的花瓶。」</p>
 <p>这话我就不乐意了。</p>
 <p>我一把挽住冉姐的手，撒娇嘟囔道：「人家才不是花瓶，明明上周你才夸我演技好。」</p>
 <p>冉姐在我脑门落下一个爆栗：「你知道沈辞是什么人吗？京圈太子爷，人狠性子淡。你看多少女明星蹭他，哪个有好果子吃的？」</p>
 <p>我知道冉姐在说李薇。</p>
 <p>李薇是我的死对头。我俩因为类型相同，李薇经常给我使绊子，抢我的通告。</p>
 <p>前段时间，沈辞接受采访被主持人问起手中的戒指，沈辞说了个初恋。</p>
 <p>第二天，李薇就暗戳戳地在微博发了个同款，配文「年少的欢喜」。</p>
 <p>本想蹭一波热度，没想到沈辞直接律师函警告，还停掉了其公司与李薇所有的拍摄合作。</p>
 <p>圈里人都是人精，李薇的日子自然不好过。</p>
 <p>客厅电视正播放着财经新闻。</p>
 <p>镜头切换，下一秒，沈辞那张淡漠的脸毫无防备地出现在我眼前。</p>
 <p>曾经，我在这张脸上见过最极致艳丽的情欲。</p>
 <p>起起伏伏间，求我爱他。</p>
 <p>我抱着冉姐的胳膊，有些心虚：「应该没关系吧，现在凌晨，刚才直播间里也没多少人，我又不算火——」</p>
 <p>话到一半，冉姐把手机递给了我。</p>
 <p>我的名字已经挂在了文娱榜第一，直播录屏在热搜上满天飞。</p>
 <p>这可能会成为我出道以来最火的一天。</p>
 <p>我随手点开一个，下面的评论一半骂我蹭，一半夸我勇，还有一小部分在为我烧高香。</p>
 <p>「姜晚乔最近是不是有新剧上啊？蹭沈总涨热度，胆子真大。」</p>
 <p>「姜晚乔是谁？敢蹭京圈太子爷，恭喜喜提律师函。」</p>
 <p>「姜晚乔可是什么都敢说，什么都敢蹭啊！」</p>
 <p>我哭丧着脸：「冉姐，救救我。」</p>
 <p>「救你，我怎么救你。」冉姐顿了顿，「除非你能把你说的话变成真的。」</p>
 <p>我愣了一下，迎上冉姐的目光：「如果我说，是真的呢。」</p>
 <p>「你脑子没事吧，你谈过谁我还不知道？」冉姐可能觉得我病得不轻：「你现在赶紧上微博发道歉声明，我这边联系给你降热搜。」</p>
 <p>「趁现在事儿还没闹到沈辞那边，咱们立正挨打。」</p>
 <p>我点点头：「那陆遥那边，你帮我也通个气。」</p>
 <p>陆遥是我的绯闻 cp。</p>
 <p>因为最近在配合新电影宣传，为了吃 cp 粉红利，我们双方团队默认了这层关系，没有辟谣。</p>
 <p>2</p>
 <p>我连夜发了道歉声明，冉姐觉得诚意不够，还让我手写了封道歉信。</p>
 <p>一顿折腾下来，天快亮了我才睡下。</p>
 <p>在梦里，我回到了初见沈辞的那年。</p>
 <p>我曾经被「流放」过一段时间，原因是在学校打架。</p>
 <p>尽管我打她是因为她造我黄谣，可我始终是公众人物。</p>
 <p>为了避风头，爸妈把我送到了外婆家，一个苏杭小镇。</p>
 <p>那时候沈辞还不是什么京圈少爷，只是我外婆隔壁家丝绸铺老板娘的儿子。</p>
 <p>我对沈辞完全是见色起意。</p>
 <p>尽管在娱乐圈多年，但沈辞依旧是我见过最好看的人。</p>
 <p>沈辞素来淡漠清冷，却被我缠着做过很多大胆的事儿。</p>
 <p>那双淡漠好看的眼里盛满情欲，粗重的呼吸，还有少年炙热的体温。</p>
 <p>我性子娇，脾气差，沈辞对我百依百顺。</p>
 <p>前提是我乖的时候。</p>
 <p>所以分手的时候，我们闹得很难堪。</p>
 <p>烟雨江南难得地下了场暴雨。</p>
 <p>沈辞跪在小院里，求我看他一眼。</p>
 <p>我打开门，将生日时他亲手烧制的陶瓷娃娃扔给了他。</p>
 <p>陶瓷砸在地上，摔了个粉碎。</p>
 <p>「姐姐，求你，别丢下我。」</p>
 <p>我比沈辞大一岁，但沈辞很少叫我「姐姐」。</p>
 <p>每次都是在床笫间被我磨得不行，才红着脸哼哼唧唧地叫我几声。</p>
 <p>我隔着门，回应他的是长久的沉默。</p>
 <p>沈辞声音沙哑，捡起地上的陶瓷碎片握在手里。</p>
 <p>血混着雨水流进院旁的沟渠，显得格外动魄。</p>
 <p>「姜晚乔，你别后悔。」</p>
 <p>3</p>
 <p>几天过去，我的直播热搜不降反升。</p>
 <p>反倒是发出去的道歉声明一点水花也没有。</p>
 <p>「这点破事能在热搜上挂好几天，明显是有人买了热搜，想借沈辞的手搞你。」</p>
 <p>保姆车上，冉姐点了根烟，却依旧压不住眉间的烦躁：「算了，我再想想办法。」</p>
 <p>冉姐送我去了片场，上周杀青的戏，导演突然通知我要补镜头。</p>
 <p>我到了才得知，我要补的是剧本上本没有的亲密戏。</p>
 <p>我不愿：「导演，我不想拍。」</p>
 <p>导演笑着，但语调里透露着威胁：「乔乔，你是演员。」</p>
 <p>一时间，我晃了神。</p>
 <p>这句话，我曾经也对沈辞说过。</p>
 <p>盛夏，沈辞家中，电视里正播放着我参演的新电影预告。</p>
 <p>沈辞抬头望向我，嗓音里夹带着委屈：「经常会拍亲密戏吗？」</p>
 <p>我咬了口沈辞手里的西瓜，含糊不清：「偶尔吧。」</p>
 <p>沈辞沉默半晌：「乔乔，我不喜欢。」</p>
 <p>我笑笑：「可我是演员，我没有拒绝的权利。」</p>
 <p>沈辞不说话，我以为这事就过去了。</p>
 <p>当晚，沈辞拉着我去了酒店。</p>
 <p>沈辞将我压在浴室镜子前，反反复复折腾了好久。</p>
 <p>沈辞咬着我的脖子，我哼唧出声。</p>
 <p>「别吸那儿，外婆会看见。」</p>
 <p>沈辞扶着我的腰，在我耳旁，声音低哑：「姐姐，以后我会让你有拒绝的权力。」</p>
 <p>男人陷入情欲中的承诺，最是不堪一击。</p>
 <p>我没当回事儿，笑着说「好。」</p>
 <p>我只当是情侣间的情趣。</p>
 <p>沈辞一个和母亲相依为命长大的半穷小子，哪来的能力去做到让我在圈内为所欲为。</p>
 <p>思绪回笼，导演拍了拍我的肩：「快去准备吧。」</p>
 <p>补的是酒店的亲密戏。</p>
 <p>我化好妆出来，远远地就看见了站在床边的李震</p>
 <p>李震是这部剧的男一，而我是一个爱而不得的恶毒女二。</p>
 <p>这场戏就是我见不得男主被女主吸引，于是对男主下药想要生米煮成熟饭，逼男主就范。</p>
 <p>「乔乔，我说过你跑不了的。」</p>
 <p>李震手背拂上我的脸，被我一把拍开。</p>
 <p>李震笑着：「怎么？不装了？」</p>
 <p>我早知道他对我有些龌龊心思。</p>
 <p>但以李震在圈中地位，我每次只能装傻充愣地躲过去。</p>
 <p>只是没想到，他居然串通导演直接改剧本。</p>
 <p>我盯着他：「李震，你可真是让我恶心。」</p>
 <p>「恶心又能怎么样呢？」李震笑笑：「一会儿，你还不是得主动扑向我。」</p>
 <p>李震俯身在我耳旁：「要不我们假戏真做，我的下部戏让你当女主。」</p>
 <p>我笑盈盈地看着李震：「谢谢李老师。」</p>
 <p>李震笑得如脸上开花，可他并不知道我给他准备了大礼。</p>
 <p>我看了看李震躺的位置，当导演说开拍的那刻，我猛地向李震扑去，脚刚好被床单绊住。</p>
 <p>整个人扑偏，膝盖恰巧顶在了李震的命根子上。</p>
 <p>李震的脸瞬间红得发黑，工作人员瞬间围了上来。</p>
 <p>我急忙跑到李震身边，语气焦急：「李老师对不起，都是我的错，真的对不起。」</p>
 <p>「但是我感觉，我应该没有压到什么东西。」</p>
 <p>周围的工作人员，听出了我的隐喻，忍不住偷笑。</p>
 <p>李震去了医院，听说住了好几天。</p>
 <p>住院耽误了工期，导致剧组档期紧张，得加班加点赶原本进度。</p>
 <p>导演烦躁地删掉了我的这场亲密戏。</p>
 <p>4</p>
 <p>我没有被沈辞律师函警告，但我正在谈的工作受到了影响。</p>
 <p>毕竟现在谁都舔沈辞，上赶着做点什么巴结他。</p>
 <p>冉姐急得上火，托关系联系沈辞那边，但沈辞不愿意见我。</p>
 <p>我其实能猜到，毕竟当初我那般折辱过他。</p>
 <p>晚上，我床上翻来覆去地睡不着。</p>
 <p>鬼使神差的我登陆了早已废弃不用的 QQ.</p>
 <p>在列表里，我找到了沈辞。</p>
 <p>依旧是当初我逼他换的那个简笔小熊头像。</p>
 <p>我在对话框里，反反复复地打了几行字，最终又被我删光。</p>
 <p>我不知道应该用怎样的开场词揭开我和沈辞的这段过往。</p>
 <p>敲门声打断我的思绪。</p>
 <p>门被敲得「咚咚」作响，震得我心慌。</p>
 <p>之前我也有遇见过半夜找来的私生，所以我打算直接报警。</p>
 <p>正当我按下最后一个「0」时——</p>
 <p>「姜晚乔。」</p>
 <p>是沈辞。</p>
 <p>我打开门，沈辞喝得醉醺醺站在门外。</p>
 <p>沈辞一看见我就朝我扑过来。</p>
 <p>我推扰着他：「沈辞，站好。」</p>
 <p>沈辞环着我的腰，嗓音委屈：「姜晚乔，好不容易让我抓住个机会，我都做到这份上了，你怎么还不来找我。」</p>
 <p>我蒙住：「什么意思？不是你拒绝冉姐说不想见我的吗？」</p>
 <p>「不想见你，我何必花那么多钱买你说我太小我的热搜。」</p>
 <p>「你看哪个男人喜欢别人说自己小的。」</p>
 <p>「我都已经那么不要脸了，你都还不来找我。」</p>
 <p>？？？</p>
 <p>所以这热搜是沈辞自己买的？</p>
 <p>沈辞脸上泛着醉酒后的红，声音越来越小：「所以，你当初甩我真的是因为这个？」</p>
 <p>沈辞整个人贴上来，下巴搁在我的颈窝，蹭得我痒痒的。</p>
 <p>「那时候我年纪小。」沈辞含上我的耳垂，耳边是满足的叹息：「现在我长大了，姐姐。」</p>
 <p>舌尖的湿意滑过我的耳垂，像被电击中般酥麻。</p>
 <p>我忍住身体的颤抖，沈辞却见我没有回应，委委屈屈：</p>
 <p>「姐姐，你放心，我很干净的。这些年贴上来那些女人都被我处理了。」</p>
 <p>「只蹭蹭也不行，我可不想让你误会。」</p>
 <p>这还是财经周刊上那清冷霸总吗？</p>
 <p>怎么喝醉酒的沈辞是这个德行？</p>
 <p>「沈辞，直播是我喝酒上头，给你带来的一切负面影响，我向你道歉。」</p>
 <p>我叹了口气：「但是我们已经分手了，是你说，让我别后悔。」</p>
 <p>「乔乔，是我后悔了。」沈辞低头埋在我的肩窝，轻轻蹭着，「别分手，别不要我。」</p>
 <p>我轻轻推开沈辞：「沈辞，我们不合适。」</p>
 <p>沈辞抬头看向我，眼尾泛红。</p>
 <p>「合适？陆遥那样的就合适？」</p>
 <p>我借坡下驴：「嗯，他比你大还好玩。」</p>
 <p>沈辞红着眼尾，一言不发。</p>
 <p>陆遥的电话，好死不死的这时候打了进来。</p>
 <p>我如同看到救星，赶忙接起电话，不小心点到了扩音。</p>
 <p>电话那头，陆遥轻喘着：「乔乔，快来——」</p>
 <p>？？？</p>
 <p>还没等我作出反应，沈辞抢过电话，一把挂断。</p>
 <p>「沈辞，你听我说——」</p>
 <p>沈辞捂住我的嘴，眼里带着祈求：「姐姐，求你别去。」</p>
 <p>沈辞咬着唇，似乎下了很大的决心：「我真的有好好长大。」</p>
 <p>「你可以玩我，别找别人。」</p>
 <p>下一秒，沈辞引着我的手，拉开了他的衣服拉链。</p>
 <p>「可以吗？姐姐——」</p>
