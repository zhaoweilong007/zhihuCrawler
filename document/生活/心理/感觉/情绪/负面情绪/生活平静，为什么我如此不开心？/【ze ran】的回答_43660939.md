# 生活平静，为什么我如此不开心？
- 点赞数：7635
- 更新时间：2017年03月01日15时30分48秒
- 回答url：https://www.zhihu.com/question/29237238/answer/43660939
<body>
 <p data-pid="xsChAn8o">Life is so boring，考个证吧。那么多种认证，有聊的，无聊的，考几个呗，没准什么时候就用到了。</p>
 <p data-pid="ygd0OoTf">Life is so boring，跑跑步吧。1km，2km，5km，10km，半马，全马，听说跑着跑着就 high 了，停都停不下来。</p>
 <p data-pid="bDB-72yT">Life is so boring，打打拳吧。学个武术，跆拳道什么的，出门可以防身，在家可以打男人。</p>
 <p data-pid="t3WP2u6s">Life is so boring，当学霸吧。记记笔记，讲讲题，刷刷奖学金，还能保研呢。</p>
 <p data-pid="jW_8ocsZ">Life is so boring，旅个游吧。高铁那么快，随便去个地方逛一逛，吃碗馄饨再回来，拍照上传朋友圈，多好玩。</p>
 <p data-pid="-9yGoQ2P">Life is so boring，赏赏花吧。春天到了，梅花杏花桃花都开了，一年只开一次，很美的，去看看吧。</p>
 <p data-pid="q9Dklm5J">Life is so boring，吃点别的吧。抱着为民请命，神农尝百草的心，多转转，吃点不同的，发现些好吃的，吐槽下难吃的。</p>
 <p data-pid="-NNwhVIv">Life is so boring，调戏下男朋友吧。别和笨蛋计较了，调戏调戏他，具体方法就不说了。</p>
 <p data-pid="HuKW14Iy">Life is so boring，do something.</p>
