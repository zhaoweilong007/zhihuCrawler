# 你看过最爽的爽文有多爽？

- 更新时间：2024年03月11日18时31分40秒

<body>
 <p>刚上大学时，我热衷于跟室友分享便宜好物。</p>
 <p>后来她不耐烦了：「不会真以为我跟你一样穷酸抠搜吧？」</p>
 <p>「平时低调，是因为不想给你们造成压力，你知道我爸是本市最大的房地产公司老总吗？」</p>
 <p>大四时我拎了个 prada 回宿舍。</p>
 <p>她面露鄙夷：「我知道你很羡慕我的家境，但女孩子要自尊自爱，不能走上歧路啊。」</p>
 <p>？该怎么说。</p>
 <p>就，她最近正追的那部大制作甜宠剧，被买版权的原书作者是我。</p>
 <p>1</p>
 <p>收到 N 大录取通知书那天，我妈带我去买了个智能手机，然后满脸忐忑地告诉我。</p>
 <p>「小竹，妈妈知道现在的大学生都得有手机和电脑。」</p>
 <p>「但是家里实在周转不开，你看能不能……把你表姐之前那台旧电脑借来用一年？」</p>
 <p>说完，她又赶紧补上一句：「等明年情况好点，就给你换一台新的。」</p>
 <p>我强压住内心酸涩，笑着说：「不用换了妈，表姐那台电脑配置还不错，够用到大学毕业了。」</p>
 <p>和别人期待着大学生活度过的暑假不一样。</p>
 <p>一整个夏天，我都在打听，N 大附近有哪些好做的兼职。</p>
 <p>开学那天，我拎着两个巨大的编织袋走进宿舍，发现隔壁床的女生已经到了。</p>
 <p>她自我介绍说她叫盛瑶，是 N 市本地人。</p>
 <p>我笑了一下：「我叫孟听竹。」</p>
 <p>盛瑶穿热情地挽着我的手，跟我介绍学校哪个小超市的东西最便宜，怎么能买到上一届的二手书。</p>
 <p>礼尚往来，我也会把我买过便宜好用的东西分享给她。</p>
 <p>盛瑶不太回复我的消息，我也没放在心上。</p>
 <p>军训结束后，我找了好几份兼职。</p>
 <p>上课的间隙里，我在整座城市东奔西走，甚至无暇顾及到底有没有融入这个宿舍的友情关系里。</p>
 <p>那天下午，我把收到的家教费转了一大半给我妈，准备去洗衣服时，才发现洗衣粉袋子空了。</p>
 <p>正好 pxx 上有打折的大桶洗衣液，我下了单，顺手把链接发给了盛瑶。</p>
 <p>她人在宿舍，我就顺口说了一句：「盛瑶，我买了一桶特别划算的洗衣液，链接发给你了。」</p>
 <p>没想到盛瑶忽然从位子上站起来，转头看着我。</p>
 <p>哪怕她极力克制，语气中还是流露出几分不耐烦：</p>
 <p>「孟听竹，你不会真以为我跟你一样穷酸抠搜吧？」</p>
 <p>「我忍很久了，你看你发的这些便宜东西，我什么时候买过？」</p>
 <p>「你知道我爸是本市最大的房地产公司老总吗？」</p>
 <p>说话间，另外两个室友也转过头来，默不作声地看着我。</p>
 <p>我有些莫名其妙：「那我第一次给你发的时候，你把话说清楚不就好了。」</p>
 <p>「我可不是那种喜欢炫富的人，平时过得一直很低调，就是怕给你造成心理压力，没想到你还没完了。」</p>
 <p>她扬了扬眉，「别的不说，就你这个几块钱一大桶的洗衣液洗过的衣服，我穿了估计都过敏。」</p>
 <p>对面床的夏岚也说：</p>
 <p>「小竹，瑶瑶平时背的包都是 LV 罗意威，连拖鞋都是爱马仕的，她肯定不需要省这个钱的。」</p>
 <p>盛瑶笑了一下：「别说了，我估计她根本分不清这些牌子，说多了别人也会尴尬的。」</p>
 <p>我低头看了一眼时间，眼看去做家教快来不及了，匆忙点点头：</p>
 <p>「知道了，以后不给你发就是。我得出门了。」</p>
 <p>2</p>
 <p>如果事情到止于此，倒也没什么。</p>
 <p>毕竟这个世界上有穷就有富，大家泾渭分明，井水不犯河水就是了。</p>
 <p>结果不久之后，某天晚上，盛瑶在宿舍里翻箱倒柜找了半天，忽然一转身：</p>
 <p>「我的鱼子酱精华不见了。」</p>
 <p>夏岚惊讶地说：「不会吧，是不是之前你带回家忘记拿回来了？」</p>
 <p>「不可能！昨天回来的时候我亲手放进包里，今天早上我还用过的。」</p>
 <p>我正对着电脑屏幕疯狂码字，想尽快把甲方要的广告文案赶出来，拿到那五百块的稿费。</p>
 <p>结果眼前光线微微一暗，盛瑶已经站在了我身边。</p>
 <p>她居高临下地看着我，语气不善：「你看到我那瓶鱼子酱精华了吗？」</p>
 <p>「没有。」</p>
 <p>我头也不抬地写着文案，结果盛瑶居然就停在我身边不动了，</p>
 <p>「孟听竹，说实话，大家都是室友，我也不想怀疑你。但白天我和夏岚熊雅她们去玩剧本杀，只有你在宿舍。」</p>
 <p>在键盘上飞速游走的手一顿，我抬眼看着她：「所以？」</p>
 <p>盛瑶一点也不客气：「所以你把你的衣柜和矮柜都打开，让我检查一下。」</p>
 <p>「这事你可做不了主，就算把你那房地产公司老总的亲爹叫来，也做不了主。」</p>
 <p>我忍住心头的怒气，尽量平静地说，</p>
 <p>「报警吧，你让警察拿搜查令过来，把整间宿舍、整个学校翻一遍都行。」</p>
 <p>「你！」</p>
 <p>她脸色一变，「不见棺材不落泪是吧？」</p>
 <p>说着，当着我的面拿出手机，打了个电话。</p>
 <p>我本来以为她在报警，没想到是打给她家里人：</p>
 <p>「妈，你上周刚给我买的鱼子酱精华在宿舍被人偷了。」</p>
 <p>可能是想让亲妈帮忙撑腰吧，说完这句话，她就按了免提，一脸盛气凌人地觑着我。</p>
 <p>结果电话那头，盛瑶妈妈的声音传来：</p>
 <p>「瑶瑶，妈妈正要跟你说呢，你的鱼子酱精华忘在茶几上没带去学校——」</p>
 <p>话音没落，盛瑶已经猛地挂了电话。</p>
 <p>见我冷冷地望着她，盛瑶咬了咬嘴唇，转身就走，却被我一把握住手腕。</p>
 <p>「你干什么？！」她忍不住叫，「放开我！好疼！」</p>
 <p>当然疼，我这双从小帮家里做惯了农活的手，用起力来，娇生惯养的大小姐又怎么受得了。</p>
 <p>我压着怒火说：「道歉吧。」</p>
 <p>「我道什么歉？东西丢了，宿舍里就你在，我不该怀疑你吗？」</p>
 <p>「但你东西没丢，你妈亲口说的。」</p>
 <p>气氛一瞬间紧绷，眼见不对，夏岚直接跑出去，把宿管阿姨叫了进来。</p>
 <p>门一开，盛瑶就开始红着眼圈掉眼泪：</p>
 <p>「家境再好我也要节约，有什么错？几千块钱的东西，问一句怎么了？又没说就是你拿的……」</p>
 <p>宿管阿姨连忙道：「有话好好说，孟听竹，你先把人家盛瑶放开！」</p>
 <p>我松了手，盛瑶一边甩着胳膊一边哭，话里话外都是指责我得理不饶人。</p>
 <p>阿姨打圆场：「行了行了，盛瑶误会了你，你也把人家手腕捏红了，都是同学，各退一步。」</p>
 <p>我想算了，没想到盛瑶却不依不饶。</p>
 <p>「她把我弄伤了，得给我道歉！」</p>
 <p>我懒得再搭理她，重新拉开椅子坐下，继续写我的文案。</p>
 <p>盛瑶闹了好一会儿，最后才在夏岚和熊雅的劝说下回到了自己的位子。</p>
 <p>3</p>
 <p>宿舍其他三个人越走越近，平时吃饭上课都一起，完全把我排除在外。</p>
 <p>不过好在，我也交到了朋友，隔壁传媒学院的宋棠。</p>
 <p>很巧，我新带的家教学生是她堂弟，一来二去的，我们就熟悉了。</p>
 <p>宋棠的家境不逊盛瑶，但性格完全天差地别，跟我十分合拍。</p>
 <p>学期中，学校要举办宿舍装修大赛，第一名可以拿到定制的特殊徽章。</p>
 <p>盛瑶带头讨论了半天，最后跑来找我：「宿舍装修是集体活动，你也得出一部分钱。」</p>
 <p>「我不参与。」</p>
 <p>我说，「你们自己搞，别带我。」</p>
 <p>「搞什么啊你，宿舍本来就是一个整体，我们都装修得漂漂亮亮，就你这个位置破破烂烂，怎么可能拿第一名？孟听竹，你怎么这么自私？」</p>
 <p>我嗤笑一声：「行，你不自私，那你出钱把我这儿装修了吧，想弄什么风格我都不介意。」</p>
 <p>盛瑶鄙夷地看着我：「你是乞丐吗？我家再有钱，也不可能这么花给你，你家就穷到这个地步？」</p>
 <p>「是的，就穷到这个地步，所以我不参与。」</p>
 <p>「你天天跑出去做家教，现在几百块钱拿不出来，谁信啊？」</p>
 <p>「你不是说我家里太穷，所以赚到钱我都打回去了，有问题吗？」</p>
 <p>说完，我拿上书和电脑，转身去图书馆了。</p>
 <p>本以为这事会就这么了结。</p>
 <p>没想到盛瑶居然得寸进尺，不知道从哪里弄来了我家里的联系方式。</p>
 <p>下午我和宋棠从图书馆出来，我妈忽然在微信上给我转了一千块钱。</p>
 <p>又发来一条消息：</p>
 <p>「小竹，学校有什么活动还是要参加，和同学好好相处，不要不合群。家里不缺钱，以后你也别给妈妈打钱了。」</p>
 <p>我脚步一下子钉在原地，死死盯着屏幕，半晌没吐出一个字。</p>
 <p>宋棠察觉到不对劲，转头看着我：「小竹？」</p>
 <p>我直接把电话给盛瑶打了过去，气得声音发抖：「你找我妈要宿舍装修的钱了？」</p>
 <p>「当然了。」她得意洋洋地说，「我就说了一下这个比赛很重要，你妈马上就说一定出钱。孟听竹，不是我说，你妈起码比你懂事多了。」</p>
 <p>我妈没有固定工作，开了个小裁缝铺，缝缝补补，很辛苦地供我上学。</p>
 <p>我不在家的时候，她隔三岔五就吃白菜煮挂面，手里为数不多的一点存款，都是平日里省吃俭用留出来的。</p>
 <p>现在被盛瑶这么一说，就轻而易举转了一千块，就是因为她一直对我心有愧意，总觉得给不了我好的生活。</p>
 <p>一股怒火涌上我心头，然后越烧越烈。</p>
 <p>宋棠走回我身边，小声问：「怎么了，小竹？」</p>
 <p>电话已经被盛瑶不耐烦地挂断，我深吸一口气，压着火道：</p>
 <p>「你还记得学校前几天通知的那个宿舍装修大赛吗？」</p>
 <p>宋棠点点头：「记得，但费心费力弄那玩意儿干什么？花钱还麻烦。」</p>
 <p>「但盛瑶觉得很有必要，所以被我拒绝后，她直接打电话找我妈要钱去了。」</p>
 <p>「有毛病吧这女的？」</p>
 <p>宋棠脱口而出。</p>
 <p>最后她说，要陪我回去找盛瑶。</p>
 <p>我不想让她惹上麻烦，定了定神，勉强扯出一抹笑：</p>
 <p>「没事，我和她的事情，自己解决就行了，牵扯太大也不好。」</p>
 <p>4</p>
 <p>我回宿舍的时候，盛瑶她们三个都在。</p>
 <p>听到开门声，夏岚抬头扫了我一眼，然后戳戳身边的盛瑶。</p>
 <p>她头也不抬地说：</p>
 <p>「你回来了啊？我们已经买了壁纸、床帘和星星灯，平摊下来每人二百，钱你直接转我微信。」</p>
 <p>「转你微信怎么够？」</p>
 <p>我冷笑一声，「我还是烧在你坟头吧。」</p>
 <p>盛瑶猛地抬起头，愤慨地看着我：「你有病吧？」</p>
 <p>「有病的是你！」我大步跨到她面前，盯着她，「学校说了，这活动全凭自愿，我也说了，我不参加。你凭什么打电话找我妈要钱？」</p>
 <p>「找你妈怎么啦？你性格这么孤僻，什么集体活动都不参加，你妈还跟我赔不是，让我多带着你玩呢。」</p>
 <p>盛瑶翻了个白眼，一脸不屑一顾。</p>
 <p>我尽量让自己平静地看着她：「你觉得你自己配吗？」</p>
 <p>「你说什么？！」</p>
 <p>「你耳背我也懒得再重复第二遍，总之这钱我一分都不会出，你们爱怎么弄怎么弄。」</p>
 <p>我说，「但你要再给我妈打电话，别怪我把你们夜不归宿，还在宿舍用违禁电器的事情举报上去。」</p>
 <p>大概是我说的话的确威胁到了她们。</p>
 <p>盛瑶绝口不谈让我出钱的事，拉着夏岚和熊雅把宿舍装饰得像是网红样板间，也衬托得我这个灰扑扑的角落更加显眼。</p>
 <p>最后没拿到她梦寐以求的第一名，盛瑶也就更看我不顺眼。</p>
 <p>她干脆带着那两个人，开始正大光明地孤立我。</p>
 <p>学院里很快就有关于我的谣言，说我自私自利，逃避集体活动，欺负宿舍其他同学。</p>
 <p>宋棠试图替我澄清，但盛瑶立刻发了一条朋友圈。</p>
 <p>「有时候，的确要宽容一些，不要过多地去责怪身边的同学，只要自己尽力了，拿不到第一也没关系。」</p>
 <p>配图是她装饰得十分精致的床位，和旁边露出的我位置的一角。</p>
 <p>因为早上在查资料写东西，桌子上堆了不少书和纸页，看上去乱糟糟的。</p>
 <p>于是关于我的传言里又加上了一条——</p>
 <p>穷酸抠门，还邋遢。</p>
 <p>「听说她的床单一学期才换一次。」</p>
 <p>「其实孟听竹家里不穷，就是因为嫉妒盛瑶，所以故意让她拿不到第一的。都什么年代了，还有人几百块钱拿不出来吗？」</p>
 <p>「人穷不可怕，心穷才可怕。」</p>
 <p>听到这些传言，要说一点都不难受，那是不可能的。</p>
 <p>位于繁华城市的高校，就像是一个微型社会，把人分割成泾渭分明的阶级。</p>
 <p>为了转移注意力，我把学习和兼职之外的时间，用来写小说。</p>
 <p>没想到在这件事上，我竟然有几分天赋，于是很快就写出了一点成绩。</p>
 <p>接连三个月的稿费到账，我终于能换掉表姐那台稍微用久一点就卡到不行的电脑，又留了五千整给我妈打回去。</p>
 <p>她不肯收：「小竹，大城市消费高，你拿着钱多和舍友出去玩玩。」</p>
 <p>我笑了笑：「妈，你别担心，我和室友相处得挺好的，她们也不经常出去玩的。」</p>
 <p>虽然相处得好是我编的，但最近，盛瑶她们的确也不怎么出去玩了。</p>
 <p>因为大二专业课增多，又额外增加了两到三人一组的课程设计，而且老师还是学院里出了名的铁面无私。</p>
 <p>这三个人天天出去玩，基础根本没打牢，于是忙得焦头烂额。</p>
 <p>结果时间截止前，她们真的交上去了一份设计，而且还做得很成熟、很完整。</p>
 <p>我隐约猜到了几分，但没打算理会。</p>
 <p>最后老师给了盛瑶她们很高的分数。</p>
 <p>盛瑶专门发了朋友圈：</p>
 <p>「课设拿了第一名~爸妈奖励的 Kelly 金刚色，他们的小棉袄也是最优秀的小棉袄。」</p>
 <p>其实她并不是第一，我和宋棠的分数比她们高一分。</p>
 <p>但这条朋友圈我原本就看得一知半解，实在不值得耗费心神，于是很快就忘了这件事。</p>
 <p>没想到不久后，学院里流言四起，都说盛瑶的课设作品是花高价买的。</p>
 <p>因为上次我威胁盛瑶时说过的话。</p>
 <p>她理所当然地，把这口黑锅扣在了我头上。</p>
 <p>5</p>
 <p>那天下午，我刚更新完两章小说，盛瑶就红着眼圈站在了我身边。</p>
 <p>「孟听竹，我到底哪里得罪你了？大一刚开学，我还跟你分享过买东西的经验。我本来是真心想跟你做朋友的，你不领情就算了，为什么要害我？」</p>
 <p>「你那是公主偶尔下凡体验平民生活，可没打算真的做平民。」</p>
 <p>我无语地说，</p>
 <p>「再说了，还需要我害你？就你平时表现出来的学习水平，你自己看看那份课设，像你能做出来的吗？」</p>
 <p>其实像这种学生水平有限，做不出东西的情况，老师们多少都心照不宣。</p>
 <p>但盛瑶这事，一是因为那位老师向来是出了名的铁面无私，最近学校也因为某些新闻正抓得严，二是因为……她实在是太张扬了。</p>
 <p>买一份完全不符合自己实际水平的设计，又对这种近乎作弊得来的高调成绩大肆宣扬。</p>
 <p>别人又不是傻子。</p>
 <p>流言越传越烈，最后连院里领导都开始过问，据说盛瑶她们可能要挨处分。</p>
 <p>那天晚上，盛瑶回家了。</p>
 <p>第二天她回了寝室，说她妈妈要请我吃饭。</p>
 <p>我还以为自己听错了：「请我？要请也是请夏岚熊雅她们吧？」</p>
 <p>也不知道盛瑶她妈干了什么，昨天还红着眼圈六神无主的公主，一下子变得气定神闲：</p>
 <p>「我妈说你能从一个小村考到这里来，很了不起，她很欣赏你。」</p>
 <p>出于对长辈的礼貌，我答应了盛瑶妈妈的邀请。</p>
 <p>她在离学校很远的一家餐厅订了包间，开车过来接我们。</p>
 <p>看到她的时候，我心里有些酸涩。</p>
 <p>我和盛瑶能做室友，我妈和她妈妈的年龄也不会差太远。</p>
 <p>可疲倦苍老的我妈，和连头发丝都打理得一丝不苟的盛瑶妈妈比起来，就像是两个世界的人。</p>
 <p>「其实瑶瑶之前回家就提到过你，说你家境清贫，但为人却很自强，我也很欣赏你这样的女孩子。」</p>
 <p>盛瑶妈妈笑了笑，从包里取出一只印着一串英文的蓝绿色盒子，推到我面前，</p>
 <p>「送你的见面礼，希望你会喜欢。」</p>
 <p>打开，里面是一条光华璀璨的手链。</p>
 <p>虽然我不认识这些品牌，但宋棠有一条，只是不怎么戴。</p>
 <p>价值大概在两万块左右。</p>
 <p>盛瑶妈妈喝了口茶水，慢条斯理地说：</p>
 <p>「孟听竹同学，你和瑶瑶是室友，大学四年都要一起住的，有点小打小闹很正常。有误会，说开就行，但有些谣言事关学业，还是不能随便乱传的。」</p>
 <p>我笑了一下，把盒子盖好，退回去：「您误会了，那事跟我没关系，建议再问问盛瑶真实情况。」</p>
 <p>她轻轻挑了下眉毛，又拎出一个黑金色的袋子：</p>
 <p>「这套护肤品，你可以寄回去给你妈妈用。听瑶瑶说，你妈妈一个人带你很辛苦的，应该没用过什么好东西吧。」</p>
 <p>「你是聪明孩子，孰轻孰重，你应该能判断。更何况只需要你出面跟学校说明一下，你是因为之前的私人矛盾才乱传谣言的。」</p>
 <p>「你跟瑶瑶一起住，很清楚她的作业都是自己做的，对不对？」</p>
 <p>我听懂了。</p>
 <p>她知道这事其实和我没关系，但也不妨碍她觉得只要一条手链加一套护肤品，就能收买我这个穷酸女孩，出面替她的宝贝女儿顶锅。</p>
 <p>如果真像她所说，我主动跑去承认谣言是我乱传的。</p>
 <p>盛瑶当然不会被记过了，但我会。</p>
 <p>我慢慢抬起眼，看着对面的盛瑶和她妈。</p>
 <p>两个人那傲慢的表情如出一辙，只是老的比小的更会掩饰而已。</p>
