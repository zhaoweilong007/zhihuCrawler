 <h2><b>四、吊顶</b></h2>
 <ul>
  <li data-pid="VOXCK6zn">吊顶的作用就是埋线，什么情况下有必要做吊顶：装中央空调/新风/风管机、客厅光源做筒灯</li>
  <li data-pid="jN210ewx">吊顶建议用轻钢龙骨，工艺需“底板托侧板”</li>
  <li data-pid="CNwLl1jp">处于长远考虑以及生活体验，不建议做复杂的吊顶造型，压低层高+不耐看</li>

 <p data-pid="_lIaibpc">③地柜主要考虑水盆、消毒柜、洗碗机、灶台、隔板柜、拉篮、抽屉。水盆力荐台下盆，有大单盆比双盆好用的说法，建议遵循习惯；消毒柜近水远火；洗碗机不管好不好用，装就得留插座。从使用感受上对比，抽屉比拉篮和隔板柜好用，建议地柜收纳采用隔板+抽屉的组合，经济耐用；</p>

 <h2>八、卫生间</h2>
 <ul>
  <li data-pid="Fudi9y0J">空间小/有小孩/有老人的，忌选锐角大台面。没必要考虑台面储物，零七八碎的洗漱用品尽量上墙。既能保证台面整洁，又无需担心小孩老人误拿错用，还可保证一定的安全空间。图中洗手台和马桶的位置设计也是反面教材。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://pica.zhimg.com/50/v2-31156327f7a7db862ac63c9d4eff0107_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="500" data-rawheight="344" data-original-token="v2-31156327f7a7db862ac63c9d4eff0107" class="origin_image zh-lightbox-thumb" width="500" data-original="https://picx.zhimg.com/v2-31156327f7a7db862ac63c9d4eff0107_r.jpg?source=1940ef5c">
  <figcaption>
   （网图）
  </figcaption>
 </figure>
 <ul>
  <li data-pid="jpDxkQIQ">空间允许的话最好做干湿分离。好处不赘述，卫生间地面保持干燥可最大程度的减少意外摔滑，水汽少可保证卫生间整体清洁、美观、防霉。</li>
  <li data-pid="4vOTsB18">地面坡度，卫生间与其它房间不同，地面应有1%~3%的泛水坡度，保证地面下水顺畅，如果安装防臭地漏或超薄地漏，可以再适当增加排水坡度，且地砖选择务必防滑、质量好。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-ac14bb0e7eac56c0c9dcfef17318494f_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="686" data-rawheight="387" data-original-token="v2-ac14bb0e7eac56c0c9dcfef17318494f" class="origin_image zh-lightbox-thumb" width="686" data-original="https://pic1.zhimg.com/v2-ac14bb0e7eac56c0c9dcfef17318494f_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="rHzWB_pU">插座被挡在马桶后面、安智能马桶却没留插座、插座距离太远等惨案屡见不鲜，各种干货贴都在磨叽插座这些事不赘述。<b>只有一条宗旨</b>，卫生间非固定且可错开时间使用的电器可共用1-2个插座，比如吹风机和卷发棒可共用1个插座；洗衣机、智能马桶等固定且不可共用插座的电器就需要对号入座了。插座防水性要高，<b>避免电器“超期服役”</b>，杜绝漏电隐患。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-5fbccb6cedf0fd75ba52fbe7d5030c5c_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="1080" data-rawheight="1620" data-original-token="v2-0277088d3c623d5ee87c278604857f5c" data-default-watermark-src="https://picx.zhimg.com/50/v2-d2c4eeea9559c78cb28982ef0d51cc40_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1080" data-original="https://pic1.zhimg.com/v2-5fbccb6cedf0fd75ba52fbe7d5030c5c_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="jLoKcGup">其实不仅是老人，厕所也是成年人的意外高发地，卫生间门可尽量考虑向<b>外开的平开门或推拉门</b>。如发生意外，不会因撞门、空间局限导致抢救不及时。不希望意外发生，长远考虑，从设计上就做好防范。</li>
  <li data-pid="TqmvWaij">台上盆好看但总清理徒增烦恼，密封胶要质量好，长起霉来太脏了，推荐一体式台盆</li>
  <li data-pid="rw4wKRAB">以及，想好到底要不要装浴缸</li>
 </ul>
 <h2>九、去甲醛&amp;空气质量&amp;防噪</h2>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-683e50a6f85392ffcda2a0f468fab688_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="600" data-rawheight="800" data-original-token="v2-ffaad902a5dc0ad2d152cd64cf45d5b0" data-default-watermark-src="https://picx.zhimg.com/50/v2-814e5a85fb85884ca134241209f64fca_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="600" data-original="https://pic1.zhimg.com/v2-683e50a6f85392ffcda2a0f468fab688_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="wo6fwxY3">你始终要相信，开窗通风是最好的去甲醛方法，入住新房至少通风3个月，入住之后也要保持通风</li>
  <li data-pid="FfasMcqt">像成都北京冬天的雾霾天气，开窗通风是不可能的。室内装上新风系统，舒坦似神仙，尤其是有孩子老人的家庭。</li>
  <li data-pid="bGlFgX_4">住宅临近马路、广场，还是不可能开窗的，车辆穿梭喇叭齐名大妈跳舞链家开会种种噪音“不绝于耳”，所以防噪门窗了解一下。</li>
 </ul>
 <h2>十、储物</h2>
 <p data-pid="HCFtDYDU">不要多，但至少要够用，尤其是有小孩和老人的家庭。</p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-8e788c4c965232e3e76a6bfac8bf68b2_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="720" data-rawheight="405" data-original-token="v2-2c4264944441c645440b446c3ba156d2" data-default-watermark-src="https://picx.zhimg.com/50/v2-7387d1c26397913fa77dbeb690705928_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="720" data-original="https://picx.zhimg.com/v2-8e788c4c965232e3e76a6bfac8bf68b2_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="UCDPrVge">有条件的，入户做<b>储物柜+鞋柜</b>一体，放鞋放包放随身物品，柜底留空间放常穿的鞋；</li>
  <li data-pid="GiTPol6l">定制衣柜，根据自己收纳习惯，哪里挂衣服，多高，哪里折叠衣服等考虑柜体内部结构。有条件做衣帽间当然更好更私密。</li>
  <li data-pid="NBSdzQ4U">推荐可收纳式茶几、入墙式顶天立地式收纳柜</li>
 </ul>
 <figure data-size="normal">
  <img src="https://pica.zhimg.com/50/v2-e6a178b5ba6b22f13ea9a8a58ba522fb_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="1159" data-rawheight="2000" data-original-token="v2-5b524069618a389e58f9115b77de249a" data-default-watermark-src="https://picx.zhimg.com/50/v2-36ab358968e55d6383b38059d75b6693_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1159" data-original="https://picx.zhimg.com/v2-e6a178b5ba6b22f13ea9a8a58ba522fb_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="bAaTVEeP">收纳柜体要用尽容积，建议立体堆放收纳</li>
  <li data-pid="UtOBw3aN">抽屉利用储物盒、分隔件都小物做区分，收纳更高效</li>
  <li data-pid="v96g9xjE">不要忽略墙壁空间，比如挂钩，衣柜柜门收纳袋</li>
  <li data-pid="sXE0o4ov">不要忽略隐形收纳，比如收纳床、收纳凳</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-b1a02c82e636579f19ef870403d35976_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="558" data-rawheight="835" data-original-token="v2-0b2c5e28f50afb53d939ab3e209192c9" data-default-watermark-src="https://picx.zhimg.com/50/v2-1d522ab3530509175794782477e7aa65_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="558" data-original="https://pic1.zhimg.com/v2-b1a02c82e636579f19ef870403d35976_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="TylgWsW5">餐边柜考虑一下，让你的餐桌不再是杂物收容所，实用且美观</li>
  <li data-pid="-OHwq-3Y">卫生间妥善利用马桶上方空间，门后空间</li>
 </ul>
 <h2>十一、地板/地砖</h2>
 <ul>
  <li data-pid="N-2pFLC4">性价比高的地板当属复合地板了</li>
  <li data-pid="u84rUWVb">家里养宠物，常开窗，忌选深色</li>
  <li data-pid="SBhueVkk">地板表面平整度≤2.0mm，接缝高低差≤0.5mm，地板间缝隙宽度≤0.5mm，翘曲度≤0.5%，踢脚板接缝严密、平整，顶角线沿口平直度≤3mm</li>
  <li data-pid="1kdK2WR9">地板铺设无松动、自然行走一圈无明显响声，靠墙伸缩应8-12mm</li>
  <li data-pid="-mpq9DPr">地砖镶贴无空鼓，表面平整度≤2.0mm，接缝高低差≤0.5mm，接缝直线度≤3.0mm，接缝宽度误差值≤2mm；客厅门槛石与厨房、卫生间完成面高差15MM-20MM</li>
 </ul>

 <h2>十二、美缝</h2>
 <p data-pid="WzY2MSsU">建议卫生间、厨房墙砖地砖都做美缝，不仅好看，防霉防水效果也很好，不易发黑，主要是好清理，动手能力差的或者工程量略大的请专业作美缝的或者监理，能做的可以找教程自己做。</p>

 <p data-pid="59bD53U2">1、 试一试把房间的颜色最简化，刷成单一的浅色系，会得到意想不到的空旷感，对，房子变大啦！</p>
