# 筷子存在的价值是什么？
- 点赞数：7444
- 更新时间：2023年05月24日12时36分52秒
- 回答url：https://www.zhihu.com/question/544802631/answer/3039062078
<body>
 <p data-pid="_PHADG7V">在美国留学的时候，有一次几个同学请一个一直一起玩 DOTA 的老美室友 C 吃中餐。</p>
 <p data-pid="MqOYkpup">上菜之后，我们想让店家拿副刀叉，C 君连忙摆手，说不想搞特殊化。说自己会用筷子，并且附上了一张自豪脸。</p>
 <p data-pid="HWihyTgj">结果上菜之后，他有些为难，因为菜肴和他概念里面的中餐显然有些出入。这里提一下，在美国同样的中餐馆，英文菜单和中文菜单点出来的东西是不一样的。英文菜单更迎合美国当地人的刻板印象，比如 { 酸甜、黑椒、橙皮、左宗棠 } 和 { 鸡、猪、牛、鱼、虾 } 的笛卡尔乘积。</p>

 </figure>
 <p data-pid="hqC5yGps">那种菜肴的食物块比较大，质地比较坚硬，对筷子使用的新手相当友好。而我们直接刷脸获得了中文菜单，C 君大开眼界，凑上去看热闹。菜单是使用到轻微卷边的塑封打印 A4 纸张，布满密密麻麻中文，有红烧肉、辣子鸡、爆炒猪腰、麻婆豆腐、烧鸭烧鹅等可选，光是种类就超过了英文版好多倍，他意识到中餐厅的中英菜单这件事并不是简单的平替翻译，而有云泥之别。</p>
 <p data-pid="8nZkdHrW">上菜之后，C 君端着碗筷怵了。他指着蒜苔炒肉片说：你在开玩笑么，这玩意怎么用筷子夹？！</p>

 <p data-pid="NVgQrpD5">我们几个面面相觑，然后各自轮流轻松地用筷子夹了几根蒜苔、几片肉片。C 君大跌眼镜，伸手尝试。我们看到他拿筷子的手，青筋暴起，颤颤巍巍，试着夹了好几次都掉落。</p>
 <p data-pid="0iatEgN0">终于，在某一次尝试中，C 君成功地夹到了菜，放到碗中。给了一个 "yes!" 的动作庆祝。我们在座的欢呼和鼓掌。</p>
 <p data-pid="SeNP_Bb6">然后一个女生就把一碟花生米放在了他面前。我们对他说，恭喜通过筷子使用水平的第一层，let's kick it up a notch.</p>

 <p data-pid="oLajHl8F">他在尝试好几次之后，花生米老是滑落，汗都出来了。然后无助地看向我们，说以花生这种几何构造，这在物理上根本不可能用筷子夹起来，我们一定是在玩弄他。他还说不相信我们能用筷子吃这玩意，肯定是手拿着吃的。</p>
 <p data-pid="2nm-oFeO">于是我们就又轮流各自夹了一粒。一个男生还一次一筷子夹了两粒，不为别的，只为炫技。C 君十分惊奇。</p>
 <p data-pid="Y5bUCPYn">我们好心地帮他纠正了握筷子的姿势，调整了筷子的位置，告诉他不用那么用力，要放松要自然。他再次尝试了几次，终于晃晃悠悠地夹起了一粒花生，吃到了，开心地大笑起来。</p>

 <p data-pid="lvALvLXQ">就在 C 君一边练习着筷子的握法、一边抓耳挠腮、跃跃欲试时……</p>
 <p data-pid="yVRcn4v_">……店家拿上了几个勺子搭在碟边。</p>
 <p data-pid="fOCrMBkz">C 君的表情是一种如释重负和大失所望的混合。他感叹，果然就算是中国人也是有极限的。</p>
 <p data-pid="R9cg4q1r">一桌子人开始坏笑。纷纷拿起筷子，轮流夹起了一块块完整的豆腐…</p>
