# 厨房有哪些合理又实用的布局？
- 点赞数：5737
- 更新时间：2014年12月10日21时59分13秒
- 回答url：https://www.zhihu.com/question/20867347/answer/34743490
<body>
 <p data-pid="aHhxaWrL">关注这题很久了。接下来我家厨房要装修，把一些思考的厨房要点心得写在这里和大家分享。</p>
 <br>
 <p data-pid="EMhWHIPY"><b>1.开放式厨房并不太适合常做饭的中国人。</b></p>
 <ul>
  <li data-pid="4QqZN1gD">厨房是家中的工作区、餐厅是休闲区。建议分开。</li>
  <li data-pid="_d0XqFG0">为什么做饭的人不想吃饭？因为闻了做饭的味道会增加饱腹感。在开放式厨房吃饭，很多人都不会吃的特香。</li>
  <li data-pid="UkRXPtPf">虽然现在油烟机马力大了，但是还要看你家的烟道状况。</li>
  <li data-pid="fd-caIB6">虽然有油烟机，但厨房的窗吹一点风，烟气就进房间了。</li>
  <li data-pid="JcsN-XG9">烤箱、空气炸锅、电炸锅、蒸笼的烟气能够确保都被吸走吗？</li>
  <li data-pid="lyyqrRqz">如果觉得互动感和敞开开放感是非常必要的，可以设一个上菜窗口、磨砂玻璃门等。</li>
 </ul>
 <p data-pid="mB0pQynP"><b><br><br>
   2.厨房内中岛也不一定适合你家。</b></p>
 <figure>
  <img src="https://pica.zhimg.com/50/d0b76cb908bd3ab94c94a674c9b5094c_720w.jpg?source=1940ef5c" data-rawwidth="500" data-rawheight="500" data-original-token="d0b76cb908bd3ab94c94a674c9b5094c" class="origin_image zh-lightbox-thumb" width="500" data-original="https://picx.zhimg.com/d0b76cb908bd3ab94c94a674c9b5094c_r.jpg?source=1940ef5c">
 </figure>
 <br>
 <ul>
  <li data-pid="RdbW13Yt">首先中岛式就很可能是开放式厨房。</li>
  <li data-pid="bX3446Qe">中岛的厨房动线会被打乱，地插也要重新设置。</li>
  <li data-pid="qqMSzy1Y">中岛式的油烟机很好看但是很贵，品牌不多。（吸力没关注过）</li>
  <li data-pid="UoVSLo-H">如果你家厨房很大，做中岛不如把厨房台面加宽。从60cm的台面宽度加到80cm。</li>
 </ul>
 <br>
 <br>
 <p data-pid="6QjqtnYU"><b>3.按照动线设计你的厨房。</b></p>
 <ul>
  <li data-pid="bMfLNGVz"><b>烹饪流程</b>：冰箱(储物空间)→台面(预处置空间)→水槽（清洗空间）→台面(切配空间)→灶台（烹饪空间）→台面(上菜空间，也可以和切配空间共用)</li>
  <li data-pid="aOgqQY7d"><b>清洁流程：</b>台面（放置空间）→水槽（清洗空间，也可放洗碗机）→台面(沥水空间)→碗柜锅柜(储存空间)</li>
  <li data-pid="7CmyJvIp">注意考虑左撇子和右撇子，线路完全相反</li>
 </ul>
 <p data-pid="bSSonFmw"><b>4.厨房照明顶灯只是辅助照明。正对操作台的柜下照明才是主要照明。</b></p>
 <ul>
  <li data-pid="v_H9g-AS">和电工师傅及早定位预留好电路</li>
  <li data-pid="IQstyRq4">一图说明原因：就是影子。自己画的渣图，就这个意思，哈哈哈。</li>

 </ul>
 <p data-pid="UrAn_ioZ"><b>5.窗户附近留台面及水槽</b></p>
 <ul>
  <li data-pid="MGWr8IUu">窗户前留工作台面，为的是方便照相。下面这张照片的用光就是利用窗口的自然光，属于好看好拍类型，特别适合新手。（如果你没有拍照的爱好，可忽略）</li>
  <figure>
   <img src="https://pic1.zhimg.com/50/bed1bb67a3f7af4e5c19c47e2d76e35b_720w.jpg?source=1940ef5c" data-rawwidth="461" data-rawheight="648" data-original-token="bed1bb67a3f7af4e5c19c47e2d76e35b" class="origin_image zh-lightbox-thumb" width="461" data-original="https://pic1.zhimg.com/bed1bb67a3f7af4e5c19c47e2d76e35b_r.jpg?source=1940ef5c">
  </figure>
  <br>
  <li data-pid="PpHaq-Ze">水槽放在窗户下，为的是保持风干和日照杀菌。</li>
 </ul>
 <p data-pid="cOLA-hTM"><b>6.我推荐三个厨房清洁电器。</b></p>
 <ul>
  <li data-pid="nfJnYU4O">垃圾处理机。大概1000元左右。买个稍微好点牌子的。<br></li>
  <ul>
   <li data-pid="6mHERNXY">可以把湿垃圾骨头、菜渣、果皮打碎冲入下水道。这样一方面省事，另一方面干垃圾可以好几天不倒都行。</li>
  </ul>
  <figure>
   <img src="https://pica.zhimg.com/50/d0dbc482a77229fcccd8bfa26557e314_720w.jpg?source=1940ef5c" data-rawwidth="600" data-rawheight="450" data-original-token="d0dbc482a77229fcccd8bfa26557e314" class="origin_image zh-lightbox-thumb" width="600" data-original="https://picx.zhimg.com/d0dbc482a77229fcccd8bfa26557e314_r.jpg?source=1940ef5c">
  </figure>
  <br>
  <li data-pid="r01jflzV">洗碗机+即加热式热水龙头。</li>
  <ul>
   <li data-pid="G64Pa59T">知乎洗碗机推荐也很多了。反正目测省了好多事儿和减少了家庭矛盾。</li>
   <li data-pid="4XmUvTjh">因为有了洗碗机和垃圾处理机。热水用量需求并不大，可以用即热式水龙头（不过山寨厂比较多，建议选购稍微好点的。）</li>
   <figure>
    <img src="https://picx.zhimg.com/50/ba14a3ea8bd2a405c65b3e19059eb2ee_720w.jpg?source=1940ef5c" data-rawwidth="215" data-rawheight="209" data-original-token="ba14a3ea8bd2a405c65b3e19059eb2ee" class="content_image" width="215">
   </figure>
   <br>
  </ul>
  <li data-pid="IZHSdDXU">手持式的充电吸尘器。</li>
 </ul>
 <p data-pid="afTTyRXf">才100块左右，随手吸一下脏东西渣渣、面粉很方便。</p>

 <p data-pid="jYTdMhW1"><b>6.水槽附近及下方，</b><b>请预留空间及插座。</b></p>
 <ul>
  <li data-pid="_V2HIppX">有些东西虽然不一定现在装，但是未来说不定你就想通了，随着你家庭成员变化就有脑子有新变化太正常。暂时不用的空间，空出来可以作为洗洁精、新抹布等物品的储存空间。</li>
  <ul>
   <li data-pid="iXuA87Ph">净水器，有小孩后可能需要。</li>
   <li data-pid="jxRBSWGj">洗碗机，为家务吵架后可能需要。<br></li>
   <li data-pid="GDmX4hPS">小厨宝/热水龙头，家里长辈同住的时可能需要。</li>
   <li data-pid="9R5wKp_G">垃圾处理器。</li>
   <li data-pid="298hIxCX">……谁知道未来还会有啥？</li>
  </ul>
  <li data-pid="TKNWhspo">如果在装修时不空出插座和空间，回头想装不能装就太麻烦了。<br></li>
 </ul>
 <p data-pid="zRo3IJf3"><b>7.厨房请安排10个以上的插座面板。</b></p>
 <ul>
  <li data-pid="rSbH2ppw">如果有必要可以设一个电箱。东西多的话，务必使用4平方线路，2.5平方线路不够的。</li>
  <li data-pid="wf-Wjs4D">大部分围绕操作台面附近，各种小家电，种类太多，你不知道你自己、伴侣、你家长未来会进什么爱好坑。</li>
  <ul>
   <ul>
    <li data-pid="F6pbJR-n">咖啡爱好2插（咖啡机、磨豆机)</li>
    <li data-pid="iVcBMO2A">烘焙爱好2插（烤箱，厨师机）</li>
    <li data-pid="r5kxL_v7">果汁爱好2插（搅拌机、榨汁机)</li>
    <li data-pid="sbrUc3TY">儿童偏好2插（辅食机、蒸笼消毒）</li>
    <li data-pid="JN7NPN7Z">老人偏好2插（搅拌机、煎药机)</li>
    <li data-pid="tM4tsB86">自家做/养生偏好（面条机、面包机、早餐机/烤面包机、电饼铛、豆浆机、电蒸笼、电炖锅etc）</li>
   </ul>
  </ul>
  <br>
  <li data-pid="dcm9Mdp2">2个在灶台边备用。</li>
  <li data-pid="PYJzfd_r">3个水槽边上面说了不再赘述。</li>
  <li data-pid="E1_rjXb7">冰箱1个、微波炉1个、电饭煲1个、电水壶1个常占。</li>
 </ul>
 <p data-pid="DurNNNqP">反正我的想法就是多配几个。不会亏的。</p>
 <p data-pid="MVAG5ijU">现在不买面板，以后就要拉接线板。</p>
 <br>
 <p data-pid="klSmXJLX"><b>8.垃圾桶的位置</b></p>
 <ul>
  <li data-pid="Jkse55Bx">位置要接近操作台和水槽</li>
  <li data-pid="kebPmAHm">可以放在橱柜内部，拉开门丢垃圾。内藏。不过很多人不习惯。</li>
  <li data-pid="Sc0QUbMH">也可以不做门，在台面下空置一个储物格，直接放个垃圾桶。这是我个人的偏好。</li>
 </ul>
 <p data-pid="on7QXdLi"><b>9.可以预留一个活动厨房台，需要时拉出来做桌子。</b></p>
 <ul>
  <li data-pid="-Gcua4Sq">用来吃烤肉或火锅（我的爱好！）</li>
  <li data-pid="LkWd-K6y">在餐厅吃味道和烟气太大了···</li>
 </ul>
