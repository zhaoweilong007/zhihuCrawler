# 家庭装修应该把重要的资金预算放到什么环节？
- 点赞数：9225
- 更新时间：2014年09月14日00时07分34秒
- 回答url：https://www.zhihu.com/question/24053376/answer/26657539
<body>
 <p data-pid="stoFFNf2">注重品质的家装只在某个环节加大资金投入是远远不够的。装修的水平<b>一半靠钱，一半靠人</b>，例如水电改造等基础环节是非常重要的，多花些钱是应该的，但是<b>大把钱花出去了，设计阶段没有花精力反复考虑和沟通，施工环节没有盯好，后期一点小问题足以击溃你用多花钱换高品质享受的期望。</b></p>
 <p data-pid="emoX9UR5"><b>========================我是干货========================</b></p>
 <p data-pid="F5bVTvJi">先罗列些干货。装修中，我多花了资金/多花了精力觉得很值的部分：</p>
 <ul>
  <li data-pid="dE22HyBw"><b>松下的新风系统，以及专业的安装队。</b></li>
  <li data-pid="DZOA_QsH"><b>林内的燃气热水器，以及tb上能买到最好的热水循环泵</b></li>
  <li data-pid="faVps-UD"><b>水电改造</b></li>
  <li data-pid="ADH3Bcn9"><b>雅鼎的浴室五金件</b></li>
  <li data-pid="nAIlPImn"><b>芬琳漆</b></li>
  <li data-pid="7QAuxMnR"><b>爱格板的家具、爱格板的橱柜、爱格板的地板</b></li>
  <li data-pid="jhlHP9wn"><b>客厅的投影、幕布</b></li>
  <li data-pid="ReCWJRMm"><b>家里所有的家电，不一定很贵但是都是反反复复研究过的，包括西门子的洗碗机，lg的洗衣机，三星的冰箱，三菱的空调等。</b></li>
  <li data-pid="3nQx-qvv"><b>水处理系统，包括德淘的前置过滤、怡口的软水、还有RO净水器。</b></li>
  <li data-pid="bz6k1eya"><b>淘宝上付出的大量精力，省了不少钱，包括灯具、家电、床垫、卫浴洁具、各种零配件。感谢万能的淘宝。</b></li>
 </ul>
 <p data-pid="0bB5w16m">=======================<b>我是干货</b>=======================</p>
 <p data-pid="5nu38A9S">以下是我是根据自己折腾近一年的经验，<b>针对热爱生活，想要提升家的品质的朋友</b>的建议（以下内容仅仅是个人能想到的经验，虽然除了软装各个环节都大致覆盖了，但是还不够完整，所以如果没有装修经验的请先百度学习几篇完整的装修日记）：</p>
 <ol>
  <li data-pid="q4K6TQuE">首先<b>找个好装修公司</b>，其次是找个好工长。没必要狂砸钱，在市场正常价格范围内找个负责任的公司，规模不一定要大（而且个人经验是找个有一点基础，正在追求扩大规模和知名度的中小型公司更省心），关键是口碑好。<b>优先级一定是公司&gt;工长</b>，只有负责任的公司+完善的管理制度才能对工长形成有效的约束力，网上那种名气大的工长单接活儿搞砸了的例子太多了，为啥？<b>没有完善的管理制度，对工长没有约束力。</b>公司抽四成以上的份子不是没有道理的。公司的增项，如果公司靠谱的话，我个人的经验是大部分增项都是很有道理的，比如多刷防水，下水道包管静音，价格觉得离谱可以砍价，甚至可以不找公司做，回头自己买点材料塞给工长一条烟就能搞定。找个好设计师，可以是装修公司自己的，也可以自己通过别的渠道找，名气可以不大，但是一定要有经验，如果热爱生活（爱折腾家居，喜欢折腾厨房）就更好了，因为这样的人会比一般人考虑的更全面。设计水平固然重要，<b>但是经验和细节细致才是决定你以后生活体验的决定性因素。</b></li>
  <li data-pid="mplqgVfg"><b>水电改造</b>找专业人来做。第一步找好公司后，谈谈水电。<b>插一句话，装修过程中一定不要犯懒，想通过多花钱来省心的想法要不得。钱可以多花，但是学习的过程不能省，一定要多问多学，你自己把装修的知识搞的明明白白的才能知道怎样能装修出好的效果来，这点没有人能帮助你做好，即使是监理公司也不行。</b></li>
 </ol>
 <ul>
  <ul>
   <li data-pid="BXtUxp3z">水电环节非常重要，自己多学学然后跟工长谈，如果发现工长对水电了解不多，立马找专业水电公司来做。水电不要偷懒，能多设计几个插座就多设计几个插座，方案一定要反复推敲，而且很重要的一个原则是：<b>多给以后留余地</b>，能多做些基础工作（如高清线、网口、插座）就要多做些，多花个几百块钱多装几个插座网口也许你一直用不到，但是一旦几年后有了需求，你会觉得当时的决定太值了。</li>
   <li data-pid="5ahIG6K0">多考虑房间的布局，你的房子布局和功能不可能一直一成不变，随着以后生活的需要，大到卧室改书房，小到家具挪位置都是很有可能的，所以我的建议是<b>不影响整体美观的情况下，每面连续的墙上至少留一个插座</b>（比如一面墙上有个门，这就不算连续的一面墙，要两边各留个插座为佳）</li>
   <li data-pid="vKhFonyq">水电材料问题，说几个经验：</li>
   <ul>
    <li data-pid="0RN39QTA">底盒一定要用好的，跟水电谈好自己加钱换，甚至可以自己买好了不用它的，这是很容易忽略的一个细节，具体可以看下<a href="https://link.zhihu.com/?target=http%3A//www.chiphell.com/thread-1025335-1-1.html" class=" wrap external" target="_blank" rel="nofollow noreferrer">电气布线容易忽视的一个小配件</a> 的讨论。</li>
    <li data-pid="PRGbDEri">电线、水管没必要刻意升级，因为水电公司也好装修公司也好，包括开发商原带的材料，用的一定都是达标级别以上的产品。但是厨房、卫生间如果大功率电器多的话，考虑下把对应的一路电线加粗。</li>
    <li data-pid="3nsxgqaH">网线换掉开发商送的线，全部换成六类网线，原因请参见我上面说的原则。</li>
   </ul>
  </ul>
 </ul>
 <p data-pid="ZbU3Z9Ir">3. 厨房要下力气设计。不仅仅是布局、结构、橱柜的位置和空间利用，下水的设计、水处理设计也是要提前做好准备的，<b>最理想的情况是橱柜方案、水处理（净水器、软水机）在水电改造前确定，并且三方（橱柜、水电设计、净水器公司）碰面谈好具体的方案；如果不能，你自己一定要研究好安装的原理、顺序，精确测好设备的尺寸、橱柜内的安装空间，水电工一般不会装，即使会也不可能像你自己一样认认真真仔仔细细的考虑到日后的使用便利。</b>我建议可能的话，让工长或者瓦工也参与进来。</p>
 <ul>
  <ul>
   <li data-pid="5NVMVRI5">追求品质的话，建议必备的几样设备：<b>洗碗机、净水器、软水机（当地水质超级好可以忽略）、热水循环系统（配合燃气热水器）、嵌入式烤箱。</b>优先级依次递减。可能的话让水电工测好水压，如果水压不够支撑水处理设备（有压损）还要加增压泵。</li>
   <li data-pid="yKz8fIGx">进口五金件、增配五金件虽然贵，但是很好用。淘宝可以买到便宜的正品，不过非常折腾。</li>
   <li data-pid="QnCQKVjQ"><b>净水器</b>也是个很大的话题，我把自己的经验写在了<a href="http://www.zhihu.com/question/24130896/answer/26773385" class="internal">家庭用中央净水器和软水机用什么好？怎么安装？</a> 这个回答里，可以参考下</li>
  </ul>
 </ul>
 <p data-pid="HkoseJbS">4. 到了施工阶段了，首先的建议就是<b>关键环节、关键时点（水电改造、防水、瓦工铺砖、各个验收环节、各种设备和用具的安装）一定要亲力亲为，自己盯好。</b>其次呢现在装修一般是轻工辅料，装修公司提供的材料方面我的建议如下：</p>
 <ul>
  <ul>
   <li data-pid="TCvzme9B">考虑环保，用成品腻子粉，不一定要进口的，国产美巢的就很好了，跟普通腻子的区别简单点说就是一个加胶（甲醛），一个加水。这点是很容易被忽略的一定，其实环保方面个人感觉比选好漆还重要。</li>
   <li data-pid="7yMzsFPj">吊顶木板用当地建材市场环保些的，例如澳松板、大亚板，这个钱不要省。检查下装修公司吊顶龙骨的质量。龙骨和木板的质量决定了吊顶的稳固性。</li>
   <li data-pid="lKjFVrOj">厨房的烟道最好能让瓦工帮忙再密封一遍，以免日后串味。</li>
  </ul>
 </ul>
 <p data-pid="oE8WJcC0">5. 主材环节。可能对于装修没太多概念的人一说装修品质，最先想到的是地板、地砖、油漆这些主材。我的建议如下：</p>
 <ul>
  <ul>
   <li data-pid="3psMH8uk">进口的强化地板不一定比实木复合的差，环保性和耐X程度反而更好。</li>
   <li data-pid="mFq-tT-u">不懂得分辨砖品质的话，两个比较省心的要点，一是大品牌，二是要广东佛山产的砖。</li>
   <li data-pid="s-DRDWFu">油漆用纯进口的（芬琳、都芳）低端就可以了，品质比国产的好很多，但是前提是工人有施工经验&amp;愿意做，这个很重要，提前要确定好，否则还是立邦多乐士吧。</li>
   <li data-pid="99qld6Kc">壁纸效果确实比油漆好，但是费用高&amp;寿命比油漆稍微差点。</li>
   <li data-pid="rpTncneT">不要用免漆木门。</li>
  </ul>
 </ul>
 <p data-pid="TzgMV9br">6. 卫浴。个人建议是，<b>财力允许的前提下，能用多好的就用多好的~</b> 我个人是把主材上省的钱用在了这里。针对生活品质的建议如下：</p>
 <ul>
  <ul>
   <li data-pid="IiOe1JOH">买个带空气增压的大顶喷吧，至少20cm以上。用了之后你就不会想再用手持花洒了。</li>
   <li data-pid="ynpO-Cbx">马桶要买釉面好的，toto的智洁很不错。</li>
   <li data-pid="25_rYgig">没必要花一大笔钱买整体智能马桶。卫洗丽是个好东西，韩国产的比日本的好。冲的菊部那叫一个酸爽啊~</li>
   <li data-pid="vbXi32qL">五金件要舍得花钱，国产九牧的跟科勒的，一两年后差别就出来了。</li>
   <li data-pid="_70x5vze">浴霸建议用松下的暖浴快。好看、小巧、功能多，总之高大上。</li>
   <li data-pid="jPskcr92">地漏买好点的，个人不建议潜水艇，粘上毛发后就失去密闭性了。那种硅胶鸭嘴的或者深水封的都行。</li>
   <li data-pid="lIrwLZWE">我觉得小卫生间浴帘比淋浴房好用。</li>
  </ul>
 </ul>
 <p data-pid="em7XiTgy">7. 家具：</p>
 <ul>
  <ul>
   <li data-pid="71jT9w_H">自己非常喜欢、而且有财力的话上实木，否则板材家具绝对是质优价廉的选择。</li>
   <li data-pid="6XwQjyU1">低端实木假货多，环保性也不如合格的板式家具。</li>
   <li data-pid="3fnvKqrY">进口的爱格板我用的非常满意，钱花的很值。</li>
   <li data-pid="gtBxH_sD">订制家具比成品家具无论效果、还是空间利用都更贴心。</li>
   <li data-pid="ku-l60zE">芝华士沙发并不贵，而且摆在客厅很小气的，还占空间（后部要留空），个人建议喜欢的话可以买个单座的。</li>
   <li data-pid="go3zDlpO">慕斯这个假洋鬼子品牌建议别碰，不是说东西不好，主要是不值那个价。</li>
  </ul>
 </ul>
 <p data-pid="YNOSfUwW">8. 家电：</p>
 <ul>
  <ul>
   <li data-pid="RJxm64Wz">建议不用太奢侈的地方：电视、烟机灶具。电视么，我觉得不论是从更新换代的频率，还是对高清有较高的要求来考虑，都没必要买太好的，前者可以等等4k电视，我是后者，我自己装了套投影。烟机的理由下面会说</li>
   <li data-pid="ae1Yh1Gu">建议舍得花钱的地方：空调、冰箱、洗衣机、热水器。<b>加上洗碗机（<a href="http://www.zhihu.com/question/19798040" class="internal">为什么中国人（总的来说）不用烘干机、洗碗机？</a> ），这些是最能体现生活品质的家电。</b></li>
   <li data-pid="QHF4hokE">家电<b>尺寸很重要</b>，提前量好，最好开工前就确定好位置。特别是洗衣机、冰箱、空调。</li>
   <li data-pid="VEooH8zq">侧吸烟机比其他的效果都好。缺点是噪音略大，不过一般低风量也够用了。烟机灶具因为技术含量不大，所以目前市场上产品的<b>同质化严重</b>。根据你的喜好和需求，选对了油烟机的种类（能侧吸尽量侧吸，从原理上来说效果是最好的），面板的种类（玻璃、陶瓷面板好打理），挑个当地口碑服务不错的品牌就可以了，不一定要买四五千起的，尤其是西门子的….鄙视下这个牌子，除了洗碗机，其他厨房家电性价比低的一塌糊涂，做工也就那样。</li>
   <ul>
    <li data-pid="w8PfZmVy">update：评论里有些朋友说侧吸效果并不好。我以自己用过几个烟机的经验来看侧吸确实好用，另外在<a href="http://www.zhihu.com/question/19793706" class="internal">开放式厨房有哪些优缺点？</a> 中前两位用过开放厨房的答主也用经验证明了这点。此外侧吸的深度也解决了我等碰头党的问题啊…</li>
    <li data-pid="jhTNzVyR">烟机好用不好用跟烟道关系很大，发现不好用的话不妨检查下烟道，特别是开发商送的止逆阀有没有换掉。</li>
   </ul>
  </ul>
 </ul>
 <p data-pid="97okdYfG">9. 其他小建议</p>
 <ul>
  <ul>
   <li data-pid="m_1NGdlk">换把<b>指纹锁</b>吧，虽然锁比一般防盗门还贵，但是效果真的非常好，而且用着也爽。不要用进口的，例如三星的，做工不如国产的，而且也没有天地勾，不安全。</li>
   <li data-pid="EhTi2-sO"><b>弱电箱</b>考虑下可以换个大点的。</li>
   <li data-pid="MVPEdB5I"><b>新风系统</b>很好用，性价比比空气净化器高的多了。</li>
   <li data-pid="WVPVX_gT"><b>中央空调</b>会比普通空调好看、舒服，但是跟新风系统一样，三分材料七分安装，一定找专业的施工队按照施工规范来做。</li>
   <li data-pid="bE0BGaEx"><b>地暖</b>是好东西。集中供暖改造地暖可以参考下我的回复：<a href="https://link.zhihu.com/?target=http%3A//www.chiphell.com/forum.php%3Fmod%3Dviewthread%26tid%3D973712%26page%3D2%23pid23003360" class=" wrap external" target="_blank" rel="nofollow noreferrer">抛砖引玉，扒扒能提升生活品质的那些玩意【原创】</a></li>
  </ul>
 </ul>
 <p data-pid="3XNPw4QY">软装我就不说了，这个根据个人的需要和家装风格来吧，装修到软装，基本上家的品质和功能也就固定了，软装的主要作用就是提升视觉的效果了。</p>
 <p data-pid="2WRZD9B6">家装真的是一个庞大的工程，其实每个环节都能写一大篇。能力和精力有限目前就写了这些，如果大家感兴趣我会继续补充~</p>
