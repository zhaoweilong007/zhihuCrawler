 <p data-pid="oIXjiry4">重要的循环的日子特别适合用自带日历做提醒，比如驾照分到期日、周年纪念日、会员卡到期日等等，记得在日历里选好相应时间，然后设置好每年循环或者每月循环，X天前提醒等，即可及时收到通知哦。</p>

 <p data-pid="uJvPXiRs">在iPhone的图片，长按即可直接抠除背景，把需要的主体拿出来，非常方便。同时如果需要照片中的信息，比如地址、电话号码等，也可以长按选择，直接复制，电话号码可以直接点拨打。</p>

 <h2>充电要改变观念</h2>
 <ul>
  <li data-pid="5jnp9zNq">首先明确,即使你不用,电池也会自己老化的,损耗是不可逆的过程.</li>
  <li data-pid="VENYoy2-">电池有循环次数大家都知道,iPhone一般为500次</li>
  <li data-pid="Q0vM1TO3">完全充电到完全放电为一次,不是说你有电情况下插上去拔下来算一次</li>
  <li data-pid="acFpj1b2">新手机不需要放完电再充,现在用的是锂电池没有记忆性质</li>
  <li data-pid="_hkSJEPi">不要傻傻的拿了新iPhone等它放完电才给充电，这对电池不好其实</li>
  <li data-pid="SliIqHNk">充电记得随用随充，千万不要等低于30%了才充,不好</li>
  <li data-pid="HqUME0zD">如果可以,每个月完整的进行一次充电放电可以保证电池活性</li>
  <li data-pid="Gg_FDe7O">最好不要边充电边玩游戏，不好(容易过热)</li>
  <li data-pid="TFrAts8V">如果在不过热的情况下,是可以边充边玩的</li>
  <li data-pid="bVMLq4gX">低电量模式如果喜欢可以一直开着，不会有什么事情</li>
  <li data-pid="FG7hL4Lx">保修期内如果电池健康低于80%可以免费换电池</li>
  <li data-pid="Sxa_FmUz">冬天温度过低充不进去电是正常的,手机自我保护而已</li>
  <li data-pid="0ri1pslv">长时间不用的话最好电量保持在50%左右,电压稳定最长对电压损耗最少,这个也是为什么很多刚出厂新机拿到手你会发现电量在50%或者40%左右的原因</li>
 </ul>

 <h2>手机丢了别忘记先试试用“查找”</h2>
 <ul>
  <li data-pid="LUHkIbDl">很多人会有手机丢了或者一下子忘记在哪儿的情况，如果实在想不起来，抓紧时间，用别人的手机登入网页版iCloud地址，输入自己的Apple ID，可以查到自己iPhone所在地的定位，如果关机或者断网也可以留下最后一次记录的位置。</li>
  <li data-pid="mVvmoaD8">我就有个朋友聚会结束后找包一直找不到（手机又放包里），后来通过这个办法发现，原来手机落在自己车里了，包也找到了，虚惊一场。</li>
  <li data-pid="DFIYi_v6">如果真丢了,建议先打开丢失模式(丢失模式下可以给捡到手机的人留言 ),锁定住手机,然后打电话给Apple400客服,必要时候也可以先报警备案.千万不要点短信链接,切记切记.</li>
 </ul>

 <h2>挂Apple牌子的店那么多,别认错了</h2>
 <ul>
  <li data-pid="UfrUtBCu">Apple的线下店一般分两种，自营的叫直营店很高大上的装修，其他的像有些商场的专柜叫授权经销商。不是所有挂着Apple牌子的都是正规的Apple店铺，有些街边的小店可能是没有资质的，卖给你是假货或者翻新,卖给你后就不管你了。</li>
  <li data-pid="IWMaRRFP">Apple直营店，服务最好，一般都开在城市的最繁华地段，有着巨大的精致的落地玻璃（每块玻璃都定制的，巨贵），宁波这家还有块巨大的索尼定制非常细腻的显示屏用来上课,直营店除了负责销售也负责售后，有问题先预约天才吧，然后拿设备和身份证去就可以。</li>
  <li data-pid="ZdaI2xke">如果你的城市没有直营店，那应该会有指定维修网点，有些会开在写字楼里，Apple官网可以查询具体地址和联系方式，或电话400询问。</li>
  <li data-pid="K6davc1I">Apple的直营店的灯光设计师真的牛，你进去的时候注意下，在里面打开手机看屏幕的时候，会发觉异常清晰好看</li>

 <h2>iPhone刷公交地铁很方便其实</h2>
 <ul>
  <li data-pid="dFlqbtS5">方法一:在钱包APP中绑定银行卡,靠近公交地铁闸口时,双击电源键,自动弹出指纹或Face ID解锁,完成,中间无需特意打开任何APP.优点是有时银联会有优惠活动,也可以绑定信用卡无需充值.</li>
