# 如何评价何同学最新视频：我做了苹果放弃的产品——AirDesk?
- 点赞数：11771
- 更新时间：2021年10月18日00时38分33秒

<body>
 <ol>
  <li data-pid="xoAmqTHj">这是一个自媒体视频，是 Entertainment，不是 Education；</li>
  <li data-pid="hb4rTFDp">这张桌子是一个「创意道具」，不是「商品」，它甚至不是产品原型，最终目的不是量产；</li>
  <li data-pid="nadlr1jW">这是一个从商单需求出发，设计出来的「道具」，它最终要引到，落脚的，是甲方的升降桌产品。</li>
 </ol>
 <p data-pid="YTvKKakF">搞清楚这三个点，你再想想：</p>
 <ol>
  <li data-pid="KJdWekUj">如果你是观众，UP 主咔咔烧钱，花费时间精力给你整活，你看得爽吗？</li>
  <li data-pid="WPBqg7Y9">如果你是甲方，这几个小时内千万级的流量，冲得爽吗？</li>
  <li data-pid="LvySCiX3">如果你是平台，特别是睿站这种裤衩亏掉都上不了贴片的平台，有这样的 UP 接单，给你返 xx 个点的商务手续费，你舒服吗？</li>
  <li data-pid="rfWvsuEv">如果你是何同学，新公司刚成立，出了这么第一个片，至少未来 xx 个月的公司员工的工资有着落了，你今晚睡得香吗？</li>
 </ol>
 <p data-pid="TutFn_Ec">都舒服了，还轮得到谁来评价？</p>
