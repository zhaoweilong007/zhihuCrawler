# 写小说空有设定和人物形象，却编不出故事剧情怎么办？
- 点赞数：16995
- 更新时间：2019年10月11日16时19分43秒
- 回答url：https://www.zhihu.com/question/329077765/answer/796515610
<body>
 <ol>
  <li data-pid="PWrVbUUf">如果是在三年前，那么我或许会告诉你这样一些技巧。这些思路乍一看，对你来说都能起到醍醐灌顶的作用。</li>
 </ol>
 <ul>
  <li data-pid="gsB9bqzs">角色存在的意义。</li>
  <li data-pid="hzrgisDQ">人物重点事件规划。</li>
  <li data-pid="ydDaAWBM">源源不断地自我需求。</li>
  <li data-pid="uvpcNXI8">外部矛盾的冲击。</li>
  <li data-pid="nNwxZ0xG">一条明确的主线。</li>
  <li data-pid="h7eS7AMx">充满亮点的转折。</li>
  <li data-pid="zvlNpM4U">从人性弱点制造困难。</li>
  <li data-pid="XZ-q4hiO">天降馅饼式的高潮伏笔。</li>
  <li data-pid="pYNiD_WP">时机成熟时新人物出现。</li>
  <li data-pid="KWHTxgsh">配角的人生哲学。</li>
  <li data-pid="Zk_upQuy">女主艺术与生活的兼容。</li>
  <li data-pid="xQV9F4KC">充满期待感的情节点。</li>
  <li data-pid="RzGkIOot">爱情与事业的部分重合。</li>
  <li data-pid="EWLpdchu">主角背景所带来的责任。</li>
  <li data-pid="SqgN7vJZ">生活中一波三折的日常。</li>
 </ul>
 <ol>
  <li data-pid="yN2Sxcl-">除了上面我说的这些，还可以写一大篇出来，我相信仅凭上面随手写的几笔，对于大部分人来说，起码开窍能做到了。某些同学看了或许会心急火燎地立马动笔，开始规划自己的书，恨不得当场写出一篇鸿篇巨著。</li>
  <li data-pid="vRRm5EoL">但是在这之前，我可能会打击你一下，实话实说，新人小白看了我上面的东西肯定有用，也能用的上，但是，他依旧写不出内容。因为，它们依旧是一个小说的框架，并没有真正逃离设定的范畴。所以给你讲了你也依旧写不出来。</li>
  <li data-pid="y3iz7kzl">下面给大家谈谈真正要编出一个好故事所需要的东西。上面提到的只是技巧，而接下来讲的才是你需要培养的地方。</li>
 </ol>
 <ul>
  <li data-pid="_rIee_qv">拥有强烈的表达欲望。</li>
  <li data-pid="tMlVvyKP">极具立体化的简单剧情想象能力。</li>
  <li data-pid="h8aKD11I">大脑对假定人物心理反应的捕捉。</li>
  <li data-pid="qRqHOHRx">超强的文字伏笔功底。</li>
  <li data-pid="yju7IurB">深层次地剧情钻研。</li>
  <li data-pid="z4RxkPgb">大脑和身体同层次的高强度训练。</li>
  <li data-pid="9_X8o05h">拥有火书思维和高层次剖析动机。</li>
  <li data-pid="IzisnD6o">精准的好书仿写训练。</li>
  <li data-pid="Ov-oaWTy">日常剧情的高强度重复写作。</li>
  <li data-pid="zCyqu4mG">缜密且成熟地逻辑思维。</li>
  <li data-pid="LDX7wqZg">大脑图书馆式的知识储备。</li>
  <li data-pid="N2H5KsrU">每个周期剧情强制性进步的写作要求。</li>
  <li data-pid="EyZbUFvN">从培养语感开始。</li>
 </ul>
 <ol>
  <li data-pid="31yrli7j">以上这些东西才是你需要真正去学习的东西。才是一个写作者能够长期写作的根源，这决定了你能不能真正吃写作这碗饭。</li>
  <li data-pid="-txqh4kP">作家可以培养，把第一种风格强制性的用在一个人身上，培养出一个底层网络小说作者简直绰绰有余。而优秀的作家没法培养，就是因为第二种学习风格，老师根本无法参与，只能自行攻略。</li>
  <li data-pid="Wxe32Gex">如果你对写作有短期需求，那么我建议你生搬硬套第一种公式，写出来的时候也的确像那么回事儿。如果你对写作是一种长期追求，并且要靠它吃饱饭，甚至吃撑，并且吃得体面，那么第二种训练模式，一定要优先使用。</li>
  <li data-pid="5YAH0_9H">该说的就说这么多，其他的我也不再多阐述，就看你能用上几分。顺便留一下我的微信公众号，看了的，最好关注，多少给我涨点儿粉，不然白写了。</li>
 </ol>
