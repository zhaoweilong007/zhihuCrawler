# 如何以「你脸红什么」为开头写个故事？
- 点赞数：8729
- 更新时间：2024年03月11日19时57分32秒
- 回答url：https://www.zhihu.com/question/422195582/answer/1513612510
<body>
 <p>「你脸红什么？」将房卡插好，我走进酒店房间，回头看着还停留在门口的男孩，「进来。」</p>
 <p>他只有十八岁，称之为男人，实在是稍微有些勉强——身形是高大的，但总归藏不住眼神中的青涩和慌张。他的皮肤白皙、柔嫩，看得出来是没经过什么保养的，偏偏更能透出此刻的潮红来。</p>
 <p>修剪整齐的指甲在 T 恤下摆留下一道道皱痕，他紧紧地揪住自己的衣角，指节发白，指肚却泛红。</p>
 <p>看得出来他是紧张极了，像是第一天上岗的工蜂，经不住蜜糖的诱惑，却还怕我，仿佛我是一朵蓄势待发的食人花。</p>
 <p>我换上酒店的拖鞋，散开头发，坐在床边看着他，又重复了一遍，「进来。」</p>
 <p>他往前迈了一步，鞋底踏在松软的地毯上，很快又退回去，仿佛我的房间是沼泽，有去无回。</p>
 <p>我因此笑出了声，看着他问：「雏？」</p>
 <p>他的脸更因这个字涨得通红，惊慌地看着我，半天才反驳道：「不，不是！」</p>
 <p>我没理会，给自己开了一罐啤酒，泡沫争先恐后地从拉环的小孔里涌出，有一些淌到了我的手背上，被我嘬净。</p>
 <p>脚步声踢踢踏踏地响起，男孩快步走了进来，攥住我的手腕。我抬头看他，他的脸还是那样红，呼吸急促不稳，眼中神色复杂。</p>
 <p>「怎么了？」</p>
 <p>话音未落，我便仰面倒在酒店蓬松的大床上，这一推又狠又重，男孩随即压了上来。他有些汗湿的手伸到我的背后，急不可耐地摸索，找到金属小扣后便专心致志地对付起来，然而却总是不得章法，于是摸索变成了多少有些粗暴地扯拽，我因此皱起了眉，叫了声停。</p>
 <p>或许是这一声停实在不带有什么欲拒还迎的意味，男孩还真的停了下来。</p>
 <p>我坐起来，从床头柜的抽屉里找出小盒，丢给他。</p>
 <p>「戴上。」见他迟迟不动，我笑了一下，「怕什么，不用你付钱。」</p>
 <p>他还是不动。</p>
 <p>「不想戴？你不怕，我还怕艾滋呢。」我的手悬在空中，就这样僵持了一会儿，直至我有些不耐烦了，「你该不会是不知道怎么用吧？」</p>
 <p>他从我手中接过盒子，没有打开，只是捏在手里，半天才问我：「这算怎么回事？」</p>
 <p>「什么怎么回事？」</p>
 <p>「我们总不能就这么不明不白地......那样了吧，总要有个说法吧？」</p>
 <p>「你管我要说法？」他这句台词是我始料未及的，我也因此发了笑，从提包里找出钱夹，抽出一叠粉色，「给，你的说法。」</p>
 <p>半天没人伸手来接，我挑眉看着他：「不够？那可就要看你表现了。」</p>
 <p>他依旧盯着我看，看了一会儿，忽然一下子站了起来，对我说：「我不是来搞这个的！」</p>
 <p>他的脸红得不像样子，看来是羞愤交加。</p>
 <p>「那你想干吗？谈恋爱？不好意思，我结婚了。」我抬起手，给他看无名指上的戒指，「而且没有离婚的想法。」</p>
 <p>可能是差一点就做了「男小三」，让这男孩觉得蒙受了奇耻大辱，听了我的话，他后退几步，逃也似的离开了房间。</p>
 <p>现在还有这么纯情的男孩吗？真稀奇。</p>
 <p>我把这件事讲给张子衿听，他笑得不行。</p>
 <p>张子衿是我丈夫，也是我的初恋，今天恰好是我们认识十周年的日子。</p>
 <p>我们在酒店楼下的餐厅预订了位置，他应该已经到了，果然，我下楼的时候，他正在前台等我。</p>
 <p>「那个男孩一直在看你。」碰杯之后，他和我说。</p>
 <p>我回头看了远处的男孩一眼，跟他讲了刚才的遭遇，惹得他笑出声来。</p>
 <p>「所以，你伤害了一个纯情少男？」他摇着头发笑，几乎拿不稳刀叉，「真有你的。」</p>
 <p>「他跟了我一路，要不是他跟着我进了酒吧，我还以为他未成年呢。」我说，「他走过来要微信，叫我学姐，还说他是什么校园新闻部的部员，听过校领导讲我的事情，想采访我。」</p>
 <p>「那你是怎么说的？」</p>
 <p>「我说可以啊，我们换个地方聊吧。」</p>
 <p>「然后你就把人领到酒店来了？」他又笑了，用手中的叉子指了指我，「我都不知道怎么夸你，周柳，你可真行。」</p>
 <p>这一顿饭我们吃得很慢，彼此都有许多话想讲，男孩不知道为什么不肯离开，只点了一份意面，已经黏成了一团糨糊。</p>
 <p>直到我们准备离开，走到酒店门口的时候，他果不其然跟过来拦住了我。</p>
 <p>他说：「学姐，没有别的意思，我真的很希望能采访到您。」</p>
 <p>我没什么反应，张子衿反倒又被他逗笑了，他拍了拍我，说会在车里等我，便先行离开。</p>
 <p>男孩告诉我，他叫孟舟，是 C 大的大一新生，我给他留了一个电话，告诉他明天下午两点，我有一个小时的空余时间。</p>
 <p>回到车上，我和张子衿都没怎么说话，车载广播随机放到我们俩都会唱的歌，我们就会一起哼唱一段，跳到下一首不会的，便又沉默下来。</p>
 <p>车窗外的霓虹星星点点，将夜幕刺穿一个又一个小洞，然而这光又冷又模糊，在车流中闪烁着，照不亮行人匆匆的面孔，只能照亮污浊的积水、惶乱的飞蛾，和一颗颗晕头转向的灰尘。</p>
 <p>广播里温柔的女声缓缓发问，「您有多久没有抬头仰望过星空了呢？」</p>
 <p>没有人给她答案，只有音乐的前奏静静地流淌出来。</p>
 <p>「这是我会唱的第一首流行歌曲，你知道吗？」张子衿调大了音量，对我说，「那时候看《流星花园》，言承旭真的很帅。」</p>
 <p>「你有没有幻想过自己是道明寺？」</p>
 <p>他哼笑一声，漫不经心地说：「幻想？我幻想自己是杉菜，道明寺都为我倾倒。」</p>
 <p>于是我也笑了。我们一同发笑，又一同安静下来，这首歌我们都会唱，但谁都没有开口，只是静静地听。</p>
 <p>陪你去看流星雨，落在这地球上，让你的泪落在我肩膀。</p>
 <p>要你相信我的爱，只肯为你勇敢，你会看见，幸福的所在。</p>
 <p>「啊——」</p>
 <p>我是被自己的尖叫吓醒的，其实醒来的瞬间，我已经不记得在梦里见着了什么，但犹记得那种恐惧，直到我已经醒了，也依旧控制不住地颤抖和尖叫。</p>
 <p>伴随着开关啪的一声响，惨白的光照亮了整个房间，我将头埋在膝盖上，意识到这恐怖的声音是我的叫喊混杂着哭泣。</p>
 <p>我感到张子衿抱紧了我，他的胳膊牢牢地将我圈住，手掌在我背上来回地安抚。</p>
 <p>「周柳，冷静点，是我。」他的手扶住我的后脑，按在他的肩头，一遍又一遍地说，「现在是二〇二〇年，你在家里。」</p>
 <p>这两句话如同咒语，使我躁动又惊恐的灵魂暂时温顺下来。</p>
 <p>「子衿，你不能......你不能离开我。」冷静下来之后，我看着他。</p>
 <p>他歪着头，摸了摸我的脸，拭去那些汗珠，「你在哪我就在哪，我们永远不会分开，我永远不会离开你。」</p>
 <p>我依旧紧紧地攥住他的手和袖子——曾有人说我发狂的时候像一只野兽，随时都在捕猎，预备攻击。</p>
 <p>除了张子衿，没有人告诉过我，其实脆弱也是我的本性，而除了他，我无法信任任何人，我无法保证，别人不会趁着我舔舐伤口的片刻，一口咬断我的脖子。</p>
 <p>夜晚极长，又极难挨，我只记得睡前我央求张子衿不要离开我，等到我醒来时，他正坐在我旁边，靠着枕头看书。</p>
 <p>见我醒了，他将书翻回折了角的某页，问我：「这个词是什么意思？」</p>
 <p>「浪漫。」我说。</p>
 <p>「英文真的很难学，读书太难了。」他将书倒扣在床头柜上，揉揉眼睛，「每次我看这些东西，都觉得你真了不起。」</p>
 <p>「没办法，读书改变命运。」</p>
 <p>洗漱完毕，我坐在梳妆台前化妆，张子衿没事做，从镜子里看我，看着看着，便笑了出来。</p>
 <p>「怎么了？」我问。</p>
 <p>「没什么，就是觉得你涂那么红的嘴唇，有必要吗？」说完，他走到我身后，低下身来观察我，「你的确是太惨白了一点。」</p>
 <p>「有必要，否则我看着太像好人了。」说着，我又盖了满满一层红色。</p>
 <p>「红嘴唇，红指甲，金戒指，其实我一直觉得这些东西很艳俗。」他顿了顿，抱起手臂上下打量我，「为什么在你身上这么合适？」</p>
 <p>「因为我就是俗人嘛。」</p>
 <p>我从烟盒里摸出一根烟来，叼在嘴里，张子衿凑过来点燃。</p>
 <p>薄薄的烟雾中，我眯起眼睛看他，他也同样看着我。</p>
 <p>「你弟弟怎么样了？」过了半天，他问。</p>
 <p>我掸落烟灰，不说话，又渡了一口，取乐般地吐出两个烟圈。</p>
 <p>「还那样，不能动，不能讲话，但是也死不了。」将烟掐灭在烟灰缸里，我看见滤嘴处鲜艳的口红印，伸出手去捻了一把，下意识又拧开口红补了一遍。</p>
 <p>我今天正好打算回家去看看我弟弟，他瘫痪八年了，脖子以下都不能动，也不能说话，吃喝拉撒都要靠管子，整日拿药物和营养剂吊着，精神虽是清醒的，也跟废人没什么两样。</p>
 <p>他叫周林，比我小一岁，今年是二十五岁，按我们家原本的安排，应该是找个女孩谈婚论嫁的年龄，不过他如今这副样子，有什么打算也早就化作泡影了。</p>
 <p>苹果皮被我削得很薄，几乎是半透明的状态，但还连着没有断开，其实很像此刻他的生命。</p>
 <p>削皮完毕，我用拇指抵住刀背，刀刃剜下一块果肉送进嘴里，甘甜饱满的汁液溢满了整个口腔。</p>
 <p>「周林，我最近有些忙，没来看你。」苹果被我搁在一旁，我放下刀，为床上的人掖好被角，「妈走的时候，最放心不下的就是你，一直嘱托我，一定要照顾好你。」</p>
 <p>「爸妈在世的时候最疼你了，可惜你出了事，没能好好孝敬他们二老。」我摸了摸他的头发，怕他听不清，便低下身子凑在他耳边，「你不用担心，钱，我跟你姐夫有的是，治多久都不是问题。」</p>
 <p>周林的喉咙里沙沙作响，不知是想说什么。桌上被搁置的苹果很快开始氧化变黄，我将它切成小块，榨成汁，交给了护工。</p>
 <p>「我平时工作有点忙，你多费心。」我关上卧室的房门，对护工说，「需要什么就买，钱不够的话，随时联系我。」</p>
 <p>护工满口答应，却在我离开后将果汁倒进了洗碗池，正遇上我折回来取东西，把她吓了一跳。</p>
 <p>「周女士，不是我偷懒，现在他喝不了果汁的，他不能吞咽的。」她有些惶恐地看着我，解释道。</p>
 <p>我只是点点头，「是吗？那你多费心。」</p>
 <p>我出来时是下午一点，还有一个小时，我来到事先约定的咖啡馆，孟舟已经坐在那里等我。</p>
 <p>见我来了，他站起来，走到我这一侧，拉开椅子，又叫来服务生让我点些喝的。</p>
 <p>看得出来他今天细心打扮过，似乎是想往成熟的路子上靠，奈何他长了张娃娃脸，加上本来就年轻，还是很显小。</p>
 <p>「看来你还是没打消这个艳遇的心思啊。」点好咖啡，我冲着他笑了笑。</p>
 <p>「我，我没别的意思，我只想给你留个好印象。」他的脸又红了，「上次的事对不起，我不知道你已经结婚了。」</p>
 <p>我摆摆手，示意他不用再说了。</p>
 <p>「能不能问一下，你丈夫是做什么工作的？」他忽然问。</p>
 <p>我咬着小勺看着他，「采访已经开始了吗？」</p>
 <p>「不是不是！」他连连摆手，可能是察觉自己声量提高，反应过来以后很快又压下去，「其实我一看见你，就觉得你简直太漂亮了，所以那时候你跟我说你结婚了，我就在想，什么样的男的才能站在你旁边。」</p>
 <p>可能是他的表情太真挚了，听起来跟那些恭维并不一样，于是我笑了，「然后呢？」</p>
 <p>「然后我见到了你丈夫，真的，我没话说了！我就想问问，他是不是明星啊？」</p>
 <p>「不是。」我低头往杯子里丢了两块方糖，细细搅匀，「他好看吧？我也觉得。我见他第一面也在想，哪个女人才能站在他旁边。」</p>
 <p>抬起头，我看着他，「他前几年刚从监狱回来。」</p>
 <p>「狱，狱警吗？」</p>
 <p>我垫着下巴，微笑地回答他，「他服刑满三年，五年前刚放出来。」</p>
 <p>看得出来孟舟很惊慌，但更多的是好奇，若不是碍于礼貌，他恨不得立刻就刨根问底，弄个清楚明白。或许是嘴巴实在发痒，他仰头灌了一大口柚子茶。</p>
 <p>我觉得有些好笑，「怕了？是不是忽然很庆幸，那天没有跟我上床？」</p>
 <p>他摇摇头，又点点头。</p>
 <p>「你不好奇吗，他是因为什么进去的？」</p>
 <p>他还是先摇摇头，又点点头。</p>
 <p>「防卫过当，致人重伤瘫痪，犯事的时候还没满十八周岁，一共判了四年，在里面表现好，减了一年。」</p>
 <p>他太年轻了，脸上藏不住一点事，听到「防卫过当」四个字，明显松了一口气。</p>
 <p>「你知道受害人是谁吗？」我压低声音，盯着他看，「是我弟弟。」</p>
 <p>他的表情果然又纠结起来，过了半天，他十分纠结地开了口，「那你，那你为什么还要嫁给他？」</p>
 <p>「他是个富二代啊，又长那么好看。」我笑了笑，「我小时候很穷苦的，豪门阔太这个身份，对我的诱惑太大了。」</p>
 <p>「你骗人！」他的声音又高了起来，「不可能，你骗我！」</p>
 <p>「为什么不可能？孟舟，我是个彻头彻尾的俗人。」</p>
 <p>他看着有些生气了，不说话，只是端坐着，像前一天在酒店那样盯着我看。</p>
 <p>我摇摇头，看了一眼表，「采访还不开始吗？你只有一个小时哦。」</p>
 <p>他的表情依旧生硬，摊开笔记本，对着我抛出了一个个毫无新意的问题。只有最后一个问题令我有了片刻的失神——其实问题本身倒不是非常有趣，但在那一刻，我的话术模板指令忽然停止运行，迫使我不得不认真思考起我内心最真实的答案。</p>
 <p>「请问，你想成为一个什么样的人？」</p>
 <p>思考片刻，我回答他，「一个浪漫的人。」</p>
 <p>看得出他情绪还有些别扭，以至于我的答案并未引起他的注意。他埋着头，唰唰地在本子上写着什么，只把这当成程式化的回答之一。察觉到了这一点，我将涂了红指甲的手攥成空拳，叩响了他面前的桌面。</p>
 <p>他因此抬头看着我，眼睛同我对视。</p>
 <p>「一个浪漫的人。」我重复道。</p>
 <p>采访结束后，我问他是否还有别的问题，他说他对我很好奇，但他觉得，我不会回答他的问题。</p>
 <p>我笑了笑，对他说：「这样吧，孟舟，这篇采访稿你先不要急着发，下次有机会，我们去学校走走，我很久没有回 C 大了。」</p>
 <p>「我现在正要回 C 大。」他很快说。</p>
 <p>「今天不行，我还有事。」我谢绝了他，「不过待会儿我丈夫会来接我，我们送你回学校吧。」</p>
 <p>「不用了。」可能是拒绝得太过生硬，他又补上一句，「我觉得太尴尬了。」</p>
 <p>我耸耸肩，又被他逗笑，「你不是对我没有非分之想了吗？」</p>
 <p>他站起来，看了我半天，只说了一个字：「有。」</p>
 <p>他可真是坦诚。</p>
 <p>张子衿来接我，十分自然地和孟舟打招呼，反倒是孟舟的表情，仿佛他才是我法律意义上的丈夫。</p>
 <p>「确定不需要搭个便车吗？」张子衿问孟舟，得到拒绝的答案后，他点点头，「那我们先走了。」</p>
 <p>「你们一点都不恩爱！」</p>
 <p>我其实没想到孟舟会这样说，因为这句话挺绿茶的。</p>
 <p>孟舟似乎长在了张子衿的笑点上——他这句话把张子衿逗得哈哈大笑，搭在车门上的手一抖一抖的，笑够了，他回过头来看着孟舟。</p>
 <p>「孟舟，我是这个世界上最爱周柳的人。」</p>
 <p>张子衿说这句话的时候，孟舟的表情很有趣，像是也有话要说。</p>
 <p>不过我没给他这个机会——我上了车，张子衿也随之坐到了驾驶座上，车子很快发动，把表情复杂的孟舟留在原地。</p>
 <p>「他很喜欢你。」张子衿对我说。</p>
 <p>「子衿。」我轻声叫出他的名字，打断他之后的话，「我也是这个世界上最爱你的人。」</p>
 <p>我侧过头看着他，他的嘴角带着微笑，目视前方，手掌覆上我的手背，轻轻拍抚。</p>
 <p>稍晚一点，我们还有一个采访，是关于这一季主打的新品——张子衿是这几年来最受瞩目的调香师，许多人都说，他是这个行业中百年一遇的天才。</p>
 <p>嗅觉是调香师的艺术生命，所以张子衿周围的人是不被允许抽烟的，除了我。</p>
 <p>我学的是珠宝设计，作品中最出名的一个系列是我大学时的毕业设计，我将幼儿的乳牙打磨光滑，如珍珠一般，再为它们戴上由红宝石装饰的皇冠，或让它们手持金色权杖和宝剑。</p>
 <p>这个系列的作品，被我命名为 Fighter，也就是战士，我毫不避讳地告诉所有人，之所以用红宝石做装饰，是因为红宝石是我最喜欢的珠宝，红色即是血色。</p>
 <p>当时大家对此褒贬不一，有人认为我很变态，或是在哗众取宠，也有人认为这是暴力美学，这些作品很有创意。</p>
 <p>总之，最初，我的风评并不算很好，后来能够得以扭转，是因为我在事业上升期嫁给了刚刚刑满释放的初恋，还上了微博热搜，爆款美文铺天盖地，把网友们感动得不行。</p>
 <p>当时好多人都在刷，「又相信爱情了」什么的，不过我觉得他们明明是被张子衿的脸感动了。</p>
 <p>每一次我仔细地看他，都会一遍遍地感叹，他实在是太好看了，这种好看不是五官多么英俊，也不是姿态多么优雅，而是没有争议的美，是一种不论你的肤色、受教育程度，不论你的文化背景、取向，都能够欣赏的客观的、标准的美。</p>
 <p>我有个朋友，曾经一度觉得我是世界上最好看的人，见过张子衿之后，其他人再夸我好看，她总会在旁边接上一句，那是你没见过她老公。</p>
 <p>此刻，面前的记者也频频望着他的脸孔出神，甚至问出了一个明显不在采访稿上的问题。</p>
 <p>「应该有不少人夸过您好看吧？」</p>
