# 常见的社会潜规则有哪些？
- 点赞数：6257
- 更新时间：2023年01月29日11时05分26秒
- 回答url：https://www.zhihu.com/question/19605307/answer/1734196155
<body>
 <p data-pid="w1QvWRtg">1、真正成大事的人，都懂得装傻和藏拙。<br>
  2、风光的背后，要么沧桑，要么肮脏。<br>
  3、所谓失言，就是一不小心说了实话。<br>
  4、不肯学挨打，就不要去学打人。<br>
  5、人没钱不如鬼，汤没盐不如水。<br>
  6、越厉害的人，往往越懂得低调。<br>
  7、你成功了有人妒嫉你，你失败了有人笑话你。<br>
  8、一个人越在意的地方，就是他越自卑的地方。<br>
  9、一个烂好人的忍让，总会被人当成软弱来践踏。<br>
  10、条条大路通罗马，而有些人就生在罗马。<br>
  11、金钱可以解决99%的问题，剩下的权力可以解决。<br>
  12、事情没有做成之前别声张，万一失败了就是打脸。<br>
  13、永远对人保留30%神秘感，人家不会轻易冒犯你。<br>
  14、和人交心要慢一点，再慢一点；和人绝交要快一点，再快一点。<br>
  15、永远不要自以为是，哪怕真的对了。世界无时无刻不在变化，时刻保持清醒，保持怀疑，谨慎行事。<br>
  16、在同事面前议论同事，基本就相当于在他本人面前议论他，一定会传到他耳边。<br>
  17、主管领导决定你的一切，他说你不行，压你几年，基本上就废了。<br>
  18、 很多人嘴上说着希望你过得好，但是他不希望你真的过得比他好。<br>
  19、 不用跟别人比来比去了，你就是个普通人，总有人比你好，也总有人比你差。<br>
  20、一定要和人精又有背景的人搞好关系，十年以后你会发现这有多少好处。<br>
  21、生活接触中的交谈，人们大多记住的是与你对话的感觉，而非内容。</p>
