 <p>「有没有心软的神……」</p>
 <p>我话音还没落下，突然，有人伸出温暖的手握住我。</p>
 <p>我刚想大喊「有流氓」，却听到对方无奈且带着笑意地说：</p>
 <p>「怎么办，系统好像把我绑定成你的专属女二了。」</p>
 <p>这个声音我再熟悉不过。</p>
 <p>「老婆！」</p>
 <p>我尖叫着扑进了她的怀里。</p>

 <p>番外之《震惊全校的毕业照是如何产生的》</p>
 <p>x 大最近流传着一个恐怖传说：</p>
 <p>「晚上不要靠近博士生宿舍楼，因为有女鬼出没！」</p>

 <p>「老婆，听说咱们这栋宿舍有女鬼，你晚上从图书馆回来一定要小心！」</p>
 <p>说完这句话，我抱住女二，哭丧着脸：</p>
 <p>「我真没用，要不是因为我现在是个盲人，我一定不会让你一个人走夜路的！」</p>
 <p>对此，女二一脸平静地问：</p>
 <p>「你最近是不是又大晚上不睡觉，偷偷在楼里摸地形了？」</p>
 <p>我点点头：</p>
 <p>「是呀，晚上人少，正是我摸地形的好时机，毕竟我也想尽快熟悉盲人的生活嘛。」</p>

 <p>「下次你半夜摸地形的时候别穿那套白色睡衣了。」</p>
 <p>我乖巧说：</p>
 <p>「好。」</p>
 <p>尽管不知道我老婆的用意何在，但她一定有自己的道理。</p>
 <p>……</p>
 <p>和女二携手来到这个世界后，我从来没有这么热爱过学术。</p>
 <p>我本以为自己永远也无法完成「取得博士学位」的任务。</p>
 <p>但我老婆何许人也，一个人弄了两份研究成果，连我的博士论文也一条龙服务全都搞定了！</p>
 <p>于是我混吃等死，一路躺到了毕业。</p>
 <p>毕业典礼上，女二作为优秀毕业生代表发言，而我作为身残志坚的盲人青年模范发言。</p>

 <p>「除了导师之外，我还想感谢一个人。</p>

 <p>听到女二的这番话，我在台下激动地拍师妹：</p>
 <p>「她在说我诶，她在说我诶！她好爱我！」</p>
 <p>师妹欲哭无泪地捂住我的嘴：</p>
 <p>「师姐你小点儿声，你毕业倒是潇洒离开了，我还得再待一年呢！」</p>
 <p>然而台上的女二话锋一转，道：</p>
 <p>「这个人就是我的母亲，如果没有她，就没有现在的我。」</p>
 <p>我：「……」</p>
 <p>见状，毒舌师妹暴露本性，凑近我的耳边，给我插了一刀：</p>
 <p>「师姐，我好像听到有什么东西碎了，哇，原来是你的心。」</p>
 <p>我强词夺理：</p>
 <p>「丈母娘比我多认识我老婆二十几年，交情深点也是应该的。」</p>
 <p>等到校领导说完昏昏欲睡的致辞，毕业生终于能够上台合影。</p>

 <p>「我师姐就托付给你了，你一定要带着她抢到毕业合照的 C 位。」</p>

 <p>上台站好合照队形后，女二低声叫了我的名字。</p>
 <p>我闻声侧过头，便感受到有东西轻轻碰了碰我的嘴。</p>
 <p>于是在摄影师「三二一茄子」的喊声中，毕业合照定格住了我和女二有史以来最猖狂的吻，而这张毕业合照要在 x 大挂到学校倒闭，供千万名 x 大学子瞻仰。</p>
 <p>从此以后，我在 x 大留下的传说，除了「我们仍未知道那天在博士生宿舍所见的女鬼的名字」，就是「那个身残志坚但是和全校最厉害的博士生学姐在毕业典礼上啵了嘴的盲人女生」。</p>
