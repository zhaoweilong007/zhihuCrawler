# 如何以“不过是一个穿越女，怎么斗得过世家贵族培养了十几年的大家闺秀”写一篇文章？

 <p>「不过是一介穿越女，怎么斗得过世家贵族培养了十几年的大家闺秀。」</p>
 <p>我看着面前女子骤然苍白的脸色，朱唇微弯，「说到底，你不过是我袭爵称王路上的垫脚石罢了。」</p>
 <p>夫君带回一个女子，是我爹娘的亲生骨肉。</p>
 <p>她夺我所爱，栽赃陷害，让我众叛亲离，人人厌弃，还说原书女主永远赢不了系统女主。</p>
 <p>我淡然一笑，看来我的真正身份，是时候公布了。</p>
 <p>1</p>
 <p>我是镇守南境的宁王独女，八省明珠，自幼受尽宠爱，百姓拥戴。</p>
 <p>周岁礼，是长公主千里迢迢，自京城赶来，亲临祝愿。</p>
 <p>及笄时，得圣上赐宴，三天流水席，万民庆贺。</p>
 <p>将来所嫁良人，更是千挑万选，慎重至极。</p>
 <p>可我心里只钟情于许云渊。</p>
 <p>他是忠义侯之子，文武双全，才华横溢，只早年不幸，家中惨遭灭门横祸，因此自幼寄养在宁王府，与我青梅竹马，情深意笃。</p>
 <p>所以及笄之后，我便便暗暗央求父亲，请圣上赐婚。</p>
 <p>父亲拗不过我，多番上书陈情，终是在我十六岁生辰时，得了允准。</p>
 <p>但许云渊不想委屈了我，留下一纸书信，便骑马北上，上京赴考。</p>
 <p>信中，他字字藏锋，力透纸背，郑重立誓必将金榜题名，状元及第，以魁首之位作聘，进士之身迎娶，予我红妆十里，凤冠霞帔。</p>
 <p>字里行间中的恳切与情意，我自是深深明白。</p>
 <p>之后的半年，我满心欢喜，满怀期待，没有一日不盼望着他快些回来。</p>
 <p>可到头来，他却食了言。</p>
 <p>那日，他帽插宫花，身着红袍，高头大马，衣锦还乡，身边还带着一个风姿拂柳，纤弱窈窈的美貌女子。</p>
 <p>宁王府前，人群之中，我远远望着他行来，面上的喜色还未绽开，便对上了他冷漠疏离的双眸。</p>
 <p>他说我是假明珠，那女子才是真千金，她才是父王的亲生女儿。</p>
 <p>她不仅心口的胎记与我一样，还有相同的玉佩信物。</p>
 <p>甚至当年将我寻回时，因太过激动而疏漏的细微裂痕，都分毫不差。</p>
 <p>母亲颤抖着指尖取过我腰间佩饰，与她的并排而列，细细对比。</p>
 <p>只一眼，便紧紧抱住了她，泪如雨下，口中声声念着「我苦命的女儿」。</p>
 <p>2</p>
 <p>近半年来，真假千金的话本层出不穷，坊间流传甚广，其中不是影射帝王流落民间的公主，便是暗喻靖王早年风流的爱女，亦或是丞相新寻回的掌珠……</p>
 <p>偏偏这些落难小姐，虽并未在世族官家长大，却个个出落得容貌惊人，才情绝世，回归位后，真真是一时风头无两，荣光无限。</p>
 <p>而原来那些名门闺秀，却性情大变，丑态尽出，不是嫉妒发疯，便是恶毒害人，个个名声损毁，下场凄惨。</p>
 <p>但听得再多，真正发生在自己身上时，还是如此的猝不及防。</p>
 <p>而且冥冥中，我总觉得这是事情中，隐隐透着蹊跷。</p>
 <p>于是为了少起争端，我便主动奉还了舒禾郡主的朝服与封号，自请去做外使掌柜，以报答父母的养育之恩。</p>
 <p>但父王和母亲却极力挽留，说无论发生什么，我都是南境的舒禾郡主，除了他们多了一个女儿，我多了一个妹妹，其它一切如旧。</p>
 <p>可偏心就像风寒咳喘，难以掩藏。</p>
 <p>他们疼她爱她，也愧对于她，于是翻涌成千万倍的荣光纵宠，奇珍珠宝，源源不断地捧到她的面前，生怕有半分亏待。</p>
 <p>于是我和许云渊一同设计的庭院，变成了她的居所。</p>
 <p>我自幼一起长大的贴身仆婢，全遣到了她的身边伺候。</p>
 <p>甚至我的名字林月河，也因着她的一句意境很美，直接换给了她。</p>
 <p>而现在，我叫林月影，意为林月河的替影。</p>
 <p>一字之差，便将十六年的娇宠疼爱，讽刺得辛辣。</p>
 <p>可变了的，何止名姓，更是其他人的态度。</p>
 <p>自她来的第一天，所有人便都围着她团团转，眼中再也看不见我。</p>
 <p>以前，只我精通兽语，能与动物交流。</p>
 <p>那是我在天赋之上，苦苦研磨十几年，才终于修得的绝技，多年来一直用此为百姓预卜天象，提示收成。</p>
 <p>她却观摩几眼，短短片刻，就熟练掌握。</p>
 <p>甚至常常我的翻译到了嘴边，最后哑然无声，她却怡然自得，侃侃而谈。</p>
 <p>各县农官听完，神色且惊且喜，全都闪烁着狂热的光亮。</p>
 <p>而看向我的目光里，却是难以掩饰的嫌弃。</p>
 <p>曾经，南境世家贵族，王孙子弟都夸我美艳精灵，冰雪聪明，总能提出意料不到的奇思妙绝。</p>
 <p>如今，一个想法刚刚冒头，她就已先于我开口，便于灌溉的水利风车，提高清洁的顺滑胰皂，还有光影折射的太阳储能……</p>
 <p>各种新奇好物，层出不穷。</p>
 <p>于是，人人赞她蕙质兰心，聪颖过人。</p>
 <p>转而叹我，今非昔比，技不如人。</p>
 <p>自幼，我学神农试验千针，尝遍百草，终成高超医术，小有医仙名号。</p>
 <p>而她一来，不用开方抓药，只需直饮圣水，便可妙手回春，止痛消灾。</p>
 <p>渐渐，周围乡镇，医仙之名，便成了她独有的赞誉。</p>
 <p>即便她一直推辞，说不敢受领。</p>
 <p>即便她总是问我：「大家如此过奖，姐姐不会不高兴吧？」</p>
 <p>即便她转脸又是嘲讽轻笑：「哦，不该叫你姐姐的，毕竟野种不知出生年月，难分大小。」</p>
 <p>即便我明知这一切，她都是有人暗中相助。</p>
 <p>倾听兽语时，她说：「系统，把她变成我的嘴替。」</p>
 <p>苦思办法时，她说：「系统，对她使用读心术。」</p>
 <p>看诊治病时，她说：「系统，把她的方子抄过来，做成饮料，再加点罂粟进去，不怕他们不上瘾。」</p>
 <p>「……」</p>
 <p>林林总总，时时刻刻，她总是得意笃定地与人交谈。</p>
 <p>可这些，只有我能听见。</p>
 <p>于是，不过短短半月，人人都认定她温婉，聪慧，娇弱玲珑，仿佛世间所以美好都集于一身，获得了全部人的喜欢。</p>
 <p>而我的形象，则由原来的温婉，聪慧，娇俏玲珑，变成了嫉妒，恶毒，面目全非。</p>
 <p>所以父王寿诞那日，她不过略施小计，我便成了暗害她的歹毒之人。</p>
 <p>明明她吃的是我桌前的芙蓉糕，若是有毒，我该首当其冲。</p>
 <p>却无人听我解释，只围在她身侧安慰，说我早知她最爱芙蓉糕，故意一口不吃，就等着她主动上钩，实在用心险恶。</p>
 <p>众目睽睽，众口铄金，几乎立刻就要将我的罪名定了下来。</p>
 <p>一时间，我如溺海独舟，孤立无援。</p>
 <p>而林月河花团锦簇，纤弱嫣然，绣帕掩唇轻声道：</p>
 <p>「虽然昨日姐姐才不小心给我尝了我会过敏的菜肴，今日又意外在芙蓉糕中藏毒，可我相信，姐姐只是怕我分了父母宠爱，又见云哥哥对我太好，一时起了邪念，并不是真心要害死我。」</p>
 <p>话音刚落，无人不赞她善良大度，更彰显我的嫉妒恶毒。</p>
 <p>我恍若未闻，只将一双眼眸落在许云渊身上，他知道的，我不是那样的人。</p>
 <p>他望了我一眼，扬声制止了众人的窃窃私语。</p>
 <p>然后，缓步行至我的身前。</p>
 <p>日光下，墨色的鎏金云光锦袍耀眼夺目，他目光如炬，缓缓开口。</p>
 <p>「你我自幼一同长大，我素来把你当做亲妹妹，可你如今的所作所为，真是太让我失望了。」</p>
 <p>我心头一跳：「妹妹？」</p>
 <p>「没错。」他抬眸望来，语色笃定，「以前是我年少不懂，错将鱼目作珍珠，如今遇见月河，才明白何为真爱可贵。」</p>
 <p>好一个真爱可贵。</p>
 <p>我的心慢慢沉落，一片寒凉，回忆起近日种种，便都明白了。</p>
 <p>也终于想通了。</p>
 <p>原来他最近的态度冷淡，不是因为我做错了什么，或是哪里做的不够好，而是因为他移情别恋。</p>
 <p>水因有性山难转，情爱之事，最是勉强不得。</p>
 <p>我不是纠缠不清的性子，若他早同我坦言，我会成全，实在不必闹得如此难看。</p>
 <p>海誓山盟终不留，你若无情我便休，定下了心思，我反而释然了。</p>
 <p>3</p>
 <p>而林月河见我未如预料中悲痛欲绝，甚至连半分失态也无，略略有些讶异，但眼眸轻转，便秀眉微蹙，落下几滴泪来：</p>
 <p>「姐姐，我知你心中吃味，深感委屈，十分同情于你，才未将你几次三番暗害我的事情告知父母。」</p>
 <p>「可现今身份已定，云哥哥与我两情相悦，生死互许，你还是尽早悬崖勒马，回头是岸，莫要执迷不悟，苦苦纠缠。」</p>
 <p>「非也。」我轻轻弯唇，淡定开口：「男人多的是，你想要，我让给你。」</p>
 <p>「让？」她冷笑一声，脸上几乎维持不住那副伪善的楚楚可怜情态，一步一步走到我的面前，咬牙切齿，「这个世界里，我才是天选之女，注定的赢家，你区区封建王朝的原文女主，见识浅薄，木讷无趣，拿什么跟我比？」</p>
 <p>「我不必跟你比，你哪配。」我玩味的看着她，仿佛看着世界上最大的笑话，</p>
 <p>「你的父母宠爱你，可不及曾经对我的十分之一，」</p>
 <p>「你的心上人说喜欢你，可受过七刀，为我拼过性命，为我鞍前马后，舍生忘死，」</p>
 <p>「你现在得到的对待，都是我玩儿剩下的。」</p>
 <p>「如今区区身份转变，便翻脸无情，天堂地狱，来日风水轮转，谁是皎皎天边月，谁是凡土脚下泥，自有定论。」</p>
 <p>「你！」她本想激怒我，让我出丑，却不料最后，是自己失态，瞪圆了眼睛，目中恶毒的恨意几乎化成刀，「好一个伶牙俐齿！我今日便要让你看看，究竟谁是月，谁是泥！」</p>
 <p>说罢，她猝然转向宁王夫妇和许云渊，「父王母亲，云哥哥，你们都亲眼瞧见了，姐姐如此妒恨于我，不惜多番陷害暗杀，如今在众人面前，都如此的容不下我，我实在是心生惶惶，不敢再待下去了，请父亲母亲允我回家去吧。」</p>
 <p>她的话还未说完，母亲已急急心疼地抱住了她，泪如雨下：「傻孩子，这里就是你的家，你要去哪？你还能去哪？」</p>
 <p>说罢，又看向父王，哀泣道：「王爷，过去多年，我们已经对不起月儿了，如今还要纵容别人欺负到她头上吗？」</p>
 <p>父王目光如炬，视线在我面上如刀刮过，又看了一眼许云渊。</p>
 <p>许云渊微微颔首，再转向我时，便冷了神色，语气冰寒：</p>
 <p>「南境宁王府，素以仁善闻名，你虽为无名弃婴，却也是父王母亲养育多年，倾注了无数心血。」</p>
 <p>「念在过往情分，二老允你继续留在宁王府，你却不知感激，惹起诸多争端不说，还将歹毒心肠用到了月河身上，甚至出言离间我们之间的感情，当真是负恩昧良，暗室欺心，再容你不得！」</p>
 <p>说罢，他猛然转身，大步流星走上高台。</p>
 <p>秋风猎猎，将他的暗色衣摆扬起，锋利如刀。</p>
 <p>至高之处，他眸光凛冽，神色肃冷，自怀中掏出明黄退婚圣旨，逐字宣读。</p>
 <p>语罢，顿了顿，又将漠然的目光看向我。</p>
 <p>「今日南境所有世家贵族，皆在座上，俱为证人，自此刻起，被驱逐出府，宁王府和你断绝关系，我与你的婚约也彻底解除，无论以后荣耀落魄，都再无半分瓜葛。」</p>
 <p>万籁俱静，片刻，又响起窃窃私语，众人的视线全部落在了我的身上，机锋暗藏，心思各异，但掩饰不住的，仍是一张又一张的看笑话嘴脸。</p>
 <p>我心痛如绞，死死捏紧手心，不肯露出半分脆弱难堪，只将目光投向寿宴的主桌主位，冷静开口。</p>
 <p>「父王母亲，也是如此想法吗？」</p>
 <p>二老默不作声，半晌，宁王洪声开口：</p>
 <p>「我们养育你多年，循循善诱，悉心教导，却依旧难改你恶毒本性，宁王府怕是再容不下你这尊大佛，你自寻去路吧。」</p>
 <p>我轻轻点头，我确实是尊大佛，你宁王府，也确实是容不下。</p>
 <p>我可以走。</p>
 <p>但绝不是担着污名走。</p>
 <p>本打算在寿宴之后，再同父母坦承我一直隐藏的真实身份，但现在看来，要提前揭晓了。</p>
