# 有什么让人听了背脊发凉的恐怖故事？
- 点赞数：64903
- 更新时间：2024年04月03日16时30分43秒
- 回答url：https://www.zhihu.com/question/317945181/answer/913998136
<body>
 <p>我有个发小，死得特别惨，但没想到更恐怖的事情还在后头……</p>
 <p><br></p>
 <p>那时候我们一共五个小孩，在火车道那边玩。</p>
 <p><br></p>
 <p>那时候他也不知道怎么的，腿就卡在了轨道上。</p>
 <p><br></p>
 <p>一开始我们还没在意，后来他发现，自己的腿怎么也拔不出来，就有点慌了神，嚎啕大哭。</p>
 <p>我当时年龄最大，比较有主见，于是喊了两个小孩抓紧跑去，找大人帮忙。</p>
 <p><br></p>
 <p>然后我跟另一个力气比较大的一起，就使劲拔这个发小的腿。</p>
 <p><br></p>
 <p>大概过了十几分钟，一声火车鸣笛声，把我们的魂都惊出来了，我当时脑子嗡的一下，心想完了，我记得当时我的裤腿直接就湿了。</p>
 <p><br></p>
 <p>我那个被卡住腿的发小也不哭了，整个人傻待在那里，愣住了。</p>
 <p><br></p>
 <p>我一巴掌扇在他脸上，他才反应过来。</p>
 <p><br></p>
 <p>「妈的，赶紧拔啊！艹！」</p>
 <p><br></p>
 <p>另一个小伙伴见状，也反应过来，把吃奶的力气都用上了，我倆为了拔他的腿，脸给憋得通红。</p>
 <p><br></p>
 <p>然而，火车的速度很快，没一会，就能看到了。而且火车头眼瞅着越来越大。</p>
 <p><br></p>
 <p>我当时没管其他的，只想把我朋友给救出来。</p>
 <p><br></p>
 <p>突然，我发现不对劲！</p>
 <p><br></p>
 <p>我那个发小不动了，我抬头看他，发现他也在看着我。</p>
 <p><br></p>
 <p>只见他特别认真地跟我说了一句。</p>
 <p><br></p>
 <p>「哥，算了吧。」</p>
 <p>然后猛的一使劲，把我俩给推了出去。</p>
 <p><br></p>
 <p>两秒钟后，火车轰隆一声，过去了。</p>
 <p><br></p>
 <p>我坐在地上，眼睁睁的看着那个发小，被火车撞碎了，血像雾一样，喷洒在空中。人瞬间就没有了。</p>
 <p><br></p>
 <p>我一辈子也忘不掉那个画面。</p>
 <p><br></p>
 <p>后来我大病了一场，父母就把我带离了那个地方，搬家去了别处。</p>
 <p>怕我有心理阴影。</p>
 <p><br></p>
 <p><br></p>
 <p>再后来，我长大了，一次出差，碰巧又回到小时候那个地方。</p>
 <p><br></p>
 <p>那里已经没人住了，房子都空了，而且看样子马上就要拆迁。</p>
 <p><br></p>
 <p>虽然我晚上还要赶大巴车回去，但我心里有些怀念小时候。</p>
 <p><br></p>
 <p>而且这个地方说不定什么时候就拆迁了，再不好好看看这个地方，这个地方就没了。</p>
 <p></p>

 <p></p>
 <p><br></p>
 <p>于是就瞎胡转了转，也不知道是怎么着，这个小时候我经常玩的地方，竟然让我走迷路了。</p>
 <p>我左走右走，来到了一幢特别破旧的房子前。</p>
 <p><br></p>
 <p>这个时候，背后突然传来一声狗叫，我小时候被狗咬过，从小就有心理阴影。</p>
 <p><br></p>
 <p>这狗一叫，我腿就开始发软，想都没想，赶紧躲进屋子，把门给关起来了。</p>
 <p><br></p>
 <p>这一关不要紧，关上以后才发现，这屋里没有灯，里面乌七八黑的。</p>
 <p><br></p>
 <p>狗也不叫了，我也开始冷静下来。</p>
 <p><br></p>
 <p>这屋里，阴森森，还有些冷，感觉挺瘆人的，我心里默数了十个数，心想，等那个狗走远了，我再出去。</p>
 <p><br></p>
 <p>1，2，3，4，5，6...... 10！</p>
 <p><br></p>
 <p>好了，我伸手去拉门。</p>
 <p><br></p>
 <p>妈的！ 我头发吓得都立起来了！</p>
 <p><br>
   这门，锁死了...</p>
