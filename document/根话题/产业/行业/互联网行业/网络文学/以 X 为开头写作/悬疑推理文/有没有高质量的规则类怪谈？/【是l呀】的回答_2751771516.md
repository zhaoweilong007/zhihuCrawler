# 有没有高质量的规则类怪谈？
- 点赞数：7534
- 更新时间：2024年03月19日16时06分18秒
- 回答url：https://www.zhihu.com/question/505507304/answer/2751771516
<body>
 <blockquote>
  半夜十二点，手机收到一条短信。
  <br>
   「请不要给你的室友开门。」
  <br>
   我顿时毛骨悚然，三个室友都在寝室里。
  <br>
   这时，门口传来了敲门声。
 </blockquote>
 <p>1</p>
 <p>「咚咚咚——」那声音急促又短暂，忽大忽小。</p>
 <p>「来了别敲了。」室友 A 离门口最近。</p>
 <p>我准备提醒她的时候，她已经打开了门。</p>
 <p>「谁呀。」</p>
 <p>开门的瞬间，一阵冷风袭来，可是现在才九月份。</p>
 <p>「没人敲什么门！是不是有病！」室友 A 骂骂咧咧地关上了门。</p>
 <p>我想出声提醒，却看见手机上那条消息又消失了。</p>
 <p>似乎刚才只是我的错觉。</p>
 <p>或许是最近恐怖小说看多了，产生了幻觉。</p>
 <p>我安慰自己。</p>
 <p>我和室友都是夜猫子，不到凌晨两点不睡觉。</p>
 <p>她们打游戏，我看小说。</p>
 <p>但是最近看的一篇小说让我难以入睡。</p>
 <p>寝室里的规则怪诞，而我们这因为疫情封校，每天只能待在寝室。</p>
 <p>我有时觉得自己的寝室和里面所说的十分相似。</p>
 <p>比如刚才的「不要给室友开门」。</p>
 <p>小说里的规则是「不要相信你的室友」。</p>
 <p>如果违规了会怎样呢？</p>
 <p>我想起小说里形容的：七窍流血，从手脚开始腐化，最后被大家赶出寝室，被走廊上的「脏东西」吃掉。</p>
 <p>一晚上，我睡得迷迷糊糊，总感觉周围有什么东西在我的床边转悠，快要喘不过气。</p>
 <p>我是被冻醒的。</p>
 <p>九月的天气还未转凉，我醒来时室友们都还没醒。</p>
 <p>今天没有早八，况且还是上网课。</p>
 <p>我准备上床继续睡回笼觉，可是怎么都睡不着。</p>
 <p>我索性下床写作业。</p>
 <p>刺目的白炽灯照到泛黄的纸页上，上面突然出现一行字：</p>
 <p>「不听话的孩子都是坏孩子。」</p>
 <p>字迹充满戾气，仿佛是用刀一笔一笔刻上去一般。</p>
 <p>鸡皮疙瘩蔓延了我一整个手臂，我抖着手去摸上面的字。</p>
 <p>它却突然消失了。</p>
 <p>就和昨天晚上的短信一样。</p>
 <p>九点半，室友陆陆续续起来了。</p>
 <p>我一直观察 A 室友，但她好像并没有什么反应。</p>
 <p>「你今天有没有不舒服。」我只好明着问她。</p>
 <p>但她却皱着眉，一副觉得我有病的样子。</p>
 <p>我和室友 A 关系不算太好，上个学期还曾经因为一些事情吵过架。</p>
 <p>平时也不怎么说话，突然的关心确实有点唐突。</p>
 <p>但我实在没办法了。</p>
 <p>如果第一次的短信还能说是自己的幻觉的话，那么字迹呢？</p>
 <p>一切都是那么奇怪。</p>
 <p>然而更加奇怪的事发生了。</p>
 <p>用得好好的饮水机突然坏了，烧不了热水。</p>
 <p>已经晚上十一点多了，还有几分钟就到十二点。</p>
 <p>室友 A 突然说要去走廊上打热水。</p>
 <p>「别去！」</p>
 <p>我急忙叫住她。</p>
 <p>一股诡异的感觉涌上我的心头。</p>
 <p>太不正常了，一切都太不正常了。</p>
 <p>为什么偏偏是十二点，为什么偏偏是室友 A。</p>
 <p>这次她扭头看了我一眼，似乎对于我今天对她的过分关注有些不耐烦。</p>
 <p>「我姨妈来了，肚子痛，现在就要喝热水！」</p>
 <p>我好像看见她背后有什么东西，但一眨眼，又不见了。</p>
 <p>她拉开门出去了。</p>
 <p>似乎一切都有预料一般，我看着手机上的手机，一分一秒地过去了。</p>
 <p>时间来到十二点整。</p>
 <p>「咚咚咚——」门口传来了敲门声。</p>
 <p>像昨天晚上似的，短暂急促，忽大忽小。</p>
 <p>只是还混杂着室友 A 的声音。</p>
 <p>「开下门，我忘带钥匙了。」</p>
 <p>——「不要给室友开门。」</p>
 <p>——「不听话的孩子都是坏孩子。」</p>
 <p>室友 B 在洗澡，室友 C 在床上。</p>
 <p>我盯着门口的方向。</p>
 <p>心脏狂跳不止。</p>
 <p>「外面冷死了，快开门！」</p>
 <p>咚咚咚——</p>
 <p>我僵硬地移着我的脚走向门口。</p>
 <p>手搭在门把上却迟迟不敢开。</p>
 <p>「好冷啊，开开门吧。」</p>
 <p>九月的天，怎么会冷呢？</p>
 <p>我的手慢慢从把手上拿下来。</p>
 <p>敲门声持续不断，我站在门口处，贴着门似乎要把门看穿。</p>
 <p>直到敲门声停止，我看了眼手机。</p>
 <p>十二点零一分。</p>
 <p>我浑身的血液已经僵住了，撑着室友 A 的书桌在她的位置上坐了下来。</p>
 <p>过了好一会才缓过来。</p>
 <p>这时，门把手突然动了。</p>
 <p>我条件反射地站起来，只见室友 A 提着水壶回来了。</p>
 <p>「今天真倒霉，走廊上的热水器也坏了，害得我还要去楼上接热水。」</p>
 <p>汗水浸透了我的衣衫。</p>
 <p>我咽了咽口水：「你刚刚敲门了吗？」</p>
 <p>室友 A 放下水壶：「敲什么门，门又没关。」</p>
 <p>我按住自己发抖的手，转头问向床上的室友 C：「你刚刚听见敲门声了吗？」</p>
 <p>室友 C：「没有啊。」</p>
 <p>我顿时跌坐在室友 A 的椅子上。</p>
 <p>「你怎么了？」室友 A 皱着眉。</p>
 <p>「我、我不知道。」手一直抖个不停，我抓住室友 A 桌上的水壶。</p>
 <p>尽量让自己平复语气：「我昨天收到一条短信，上面说让我别给室友开门。」</p>
 <p>「短信是不是很快就消失了。」</p>
 <p>室友 C 突然从床上坐起来。</p>
 <p>「对！」我像是抓住了救命稻草，猛地点头。</p>
 <p>「我昨天也收到了。」</p>
 <p>「上面写的是：寝室里没有惨叫声。」</p>
 <p>「谁这么无聊发这些短信啊。」</p>
 <p>室友 A：「你就被这吓到了？」</p>
 <p>室友 A 将我从椅子上提起来，自己坐上去。</p>
 <p>「这学校一个社团搞的。」</p>
 <p>说完，她拿出自己的手机，滑到一个「恐怖社团」的群里。</p>
 <p>群主：「让我们看看有多少人会被吓到。」</p>
 <p>管理员：「今夜是个无眠。」</p>
 <p>……</p>
 <p>「每年这个时候都有，不用怕。」室友 A 给我倒了杯热水。</p>
 <p>我心里的石头终于落下。</p>
 <p>接过水猛灌一口。</p>
 <p>「真无语，这些人没事多吃溜溜梅，整这些乱七八糟的东西。」</p>
 <p>「对了，你收到的消息是什么。」</p>
 <p>恐惧一过，好奇心就滋滋地往上冒。</p>
 <p>室友 A 勾起嘴角：「让室友喝下你递的热水。」</p>
 <p>2</p>
 <p>时间仿佛静止了，我盯着她不说话。</p>
 <p>「啊——」</p>
 <p>浴室内传来一声惨叫。</p>
 <p>我顾不上刚刚的心惊，急忙跑到浴室门口问道：「怎么了？」</p>
 <p>室友 B 用浴袍勉勉强强遮住身体，身上的沐浴露都还没擦干净。</p>
 <p>洗澡水和泪水混在一起，她一手扶着浴袍，一手指着厕所：「好、好多头发。」</p>
 <p>我顺着她手指的地方看过去，却什么都没看见。</p>
 <p>「哪有什么头发，该不是你自己掉的吧。」室友 A 闻声走过来。</p>
 <p>床上的室友 C 伸出头：「别这么大惊小怪，吓死人了。」</p>
 <p>「盈盈，你相信我，我真的看到好多头发。」室友 B 哭着拽着我的手，不肯放开。</p>
 <p>我只好陪她一起洗完。</p>
 <p>她像是真的被吓到了，晚上不敢一个人睡觉，非要跑来和我睡。</p>
 <p>我和她挤在逼仄狭窄的空间里，突然想起短信的事。</p>
 <p>「你知道学校的『恐怖社团』吗？」</p>
 <p>室友 B 想起什么似的，身体突然一抖。</p>
 <p>「我昨天收到一个短信，上面让我不要去洗澡。</p>
 <p>「我、我以为是谁的恶作剧。」</p>
 <p>室友 B 的声音已经带上了哭腔，整个身体抖成了筛子。</p>
 <p>「是那个『恐怖社团』搞的，别担心了。」</p>
 <p>我安抚着室友 B 的情绪，内心却忐忑不安。</p>
 <p>我总觉得哪里奇怪，可又说不上来。</p>
 <p>如果这是一个社团搞的，敲门声还能勉强说得过，那厕所里的头发呢。</p>
 <p>若真是室友 B 自己的头发，那她以前怎么没被吓到。</p>
 <p>更何况，我们都看不见头发，就像之前，她们都没听见敲门声。</p>
 <p>而且，大家的短信都不一样。</p>
 <p>我突然一惊，我喝下了室友 A 的水。</p>
 <p>不能违背规则，但为什么室友 A 的规则和我们的又有些不一样。</p>
 <p>或许是因为太害怕，晚上睡觉时一直处于高度紧张中。</p>
 <p>「滴答滴答——」像是水龙头没关紧的声音。</p>
 <p>我迷迷糊糊睁开眼。</p>
 <p>走廊上的灯光从门上的玻璃窗透进来。</p>
 <p>床帘处出现了一个人影！</p>
 <p>我闭上眼睛，用被子将自己蒙住。</p>
 <p>但外面的水滴声还在继续。</p>
 <p>「滴答滴答——」</p>
 <p>却又和水龙头漏水的声音不一样，更加缓慢，更加浓稠。</p>
 <p>接着，物品拖拽的声音响起。</p>
 <p>或许不是物品。</p>
 <p>我能清楚的感受到是一个人被拖拽着。</p>
 <p>因为那个被拖拽的人是我！</p>
 <p>我动不了，只感觉浑身的血液往一处流去。</p>
 <p>「滴答滴答——」</p>
 <p>在这个安静的寝室显得刺耳极了。</p>
 <p>我好像死了，被人拖着走了好久好久。</p>
 <p>外面好冷啊。</p>
 <p>我好想回家——</p>
 <p>3</p>
 <p>那不是我的声音！</p>
 <p>我好想回家——</p>
 <p>不是我说的。</p>
 <p>那不是我！</p>
 <p>我惊醒过来，满身的冷汗。</p>
 <p>室友 B 已经起床了。</p>
 <p>她听见我的动静掀开我的帘子：「怎么了？」</p>
 <p>「没，做了个噩梦。」</p>
 <p>但那个梦太真实了，到现在我都心有余悸。</p>
 <p>室友 B 见我清醒，又去室友 A 的床下叫她上课。</p>
 <p>「家琪，快起床了，腾讯会议签到。」</p>
 <p>「你昨天没睡吗？黑眼圈这么重。」室友 B 一边拉开帘子，一边问。</p>
 <p>室友 A：「不知道，我感觉睡了一觉好累啊，比跑了八百米还累。」</p>
 <p>「你少来了，昨晚肯定又熬夜了，快点下床洗漱，还有五分钟上课了。」</p>
 <p>室友 A 磨磨蹭蹭地下了床，路过我身边的时候突然停住，欲言又止。</p>
 <p>等到她洗完脸后，又站在我身后。</p>
 <p>「你昨天有收到短信吗？」</p>
 <p>我摇了摇头问道：「你收到了什么。」</p>
 <p>室友 A 没回答我，魂不守舍地回到了座位。</p>
 <p>因为不能出寝室，吃饭都是专人送到的。</p>
 <p>所以当敲门声又响起的时候，我下意识地抖了一下。</p>
 <p>直到室友 C 去开了门，把饭拿进来，我才松了口气。</p>
 <p>今天的午饭是冬瓜排骨和辣子鸡丁。</p>
 <p>我打开一部下饭剧，正准备吃。</p>
 <p>「呕——」</p>
 <p>室友 A 干呕得虚脱，满头的汗。</p>
 <p>「咋了？」离她最近的室友 C 问道。</p>
 <p>「生、生的——呕——」室友 A 一边说一边吐。</p>
 <p>「没有啊。」室友 C 塞了一块排骨进嘴里，然后又夹了一块室友 A 碗里的排骨。</p>
 <p>「挺香的啊，都熟了。」</p>
 <p>室友 A 看着室友 C 又夹了一块，直接跑去厕所吐了起来。</p>
 <p>我看着碗里的肉，瞬间明白。</p>
 <p>短信。</p>
 <p>今早室友 A 问我有没有收到短信，说明她自己收到了。</p>
 <p>所以，她到底收到了什么。</p>
 <p>4</p>
 <p>「你们别再玩这个无聊的恶作剧了！再发短信我报警了！」</p>
 <p>厕所里传来室友 A 愤怒的声音。</p>
 <p>恐怖社团、恐怖社团。</p>
 <p>我突然想起来，我这里有学校所有社团的名单。</p>
 <p>社团的建立是要通知学生会的。</p>
 <p>我从抽屉地下翻出去年的文件。</p>
 <p>从第一页扫到最后一页。</p>
 <p>恐怖社团：解散时间——三年前！</p>
 <p>一个消失了三年的社团，一条条莫名其妙的短信，还有那出现在纸页上的笔记。</p>
 <p>当推翻所有不可能后，那真相只有唯一的可能。</p>
 <p>我拿出一张白纸，在上面写道：</p>
 <p>「你是谁？」</p>
 <p>没人回答，白纸依旧干干净净。</p>
 <p>但我知道，一定有人在这。</p>
 <p>室友 A 从厕所中走出来，面色如肝。</p>
 <p>我拦住她，直奔主题：「你昨天是不是收到短信了。」</p>
 <p>另外的室友 B 和室友 C 转过身来。</p>
 <p>室友 B 举起手：「我又收到一条。」</p>
 <p>「上面说，两点后请不要拉开你的床帘。」</p>
 <p>室友 C：「我也有一条。」</p>
 <p>「寝室里没有血迹。」</p>
 <p>大家说完，都下意识地看着室友 A。</p>
 <p>我严肃地问道：「所以你收到的是什么。」</p>
 <p>「我什么都没收到！」</p>
 <p>室友 A 一把推开我：「这么无聊的游戏你们玩上瘾了吗！</p>
 <p>「别来烦我了！」</p>
 <p>室友 A 一定收到了短信，但看样子，她并不想告诉我们。</p>
 <p>「家琪。」我叫住她：「你是什么时候加入那个恐怖社团的群。」</p>
 <p>室友 A 愣住了，整个人呆在原地，僵硬地扭着脖子，眼睛死死地盯着我。</p>
 <p>「家琪！」我有点害怕，又叫了一声她的名字。</p>
 <p>「我不知道，我不知道，我不知道。」她将脖子扭正，低着头，一直重复着这句话。</p>
 <p>她一边说，一边往床上走。</p>
 <p>我一直看着她，直到她将帘子拉上的瞬间，突然朝我露出一个阴森的笑。</p>
 <p>5</p>
 <p>真是疯了！</p>
 <p>我拿着笔一笔一笔在白字上写着。</p>
 <p>「你到底是谁！</p>
 <p>「你要干什么。</p>
 <p>「算我求你，你快出现吧。」</p>
 <p>终于，白纸上浮现出一行字迹。</p>
 <p>「凌晨两点后，请不要靠近尖锐的物品。」</p>
