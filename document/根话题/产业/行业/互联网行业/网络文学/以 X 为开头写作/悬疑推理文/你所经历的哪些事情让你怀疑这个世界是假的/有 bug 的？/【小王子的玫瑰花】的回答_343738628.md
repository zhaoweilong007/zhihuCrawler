# 你所经历的哪些事情让你怀疑这个世界是假的/有 bug 的？
- 点赞数：15720
- 更新时间：2018年03月17日18时02分01秒
- 回答url：https://www.zhihu.com/question/61917117/answer/343738628
<body>
 <p data-pid="_kL77I_7">我爷爷生前最疼我，有一天我刚从老家回新乡上班，中午我妈打电话说我爷爷去世了，爷爷爱干净，去外边洗澡不小心摔池塘里了。开车回家，一路上都是八十码，那一天车意外的少，回家守了七天孝，然后去祖坟埋，我哭的很伤心，埋完后回家里去的时候，发现刮起了大风，爷爷的坟头上刮起一个小旋风，跟着我们跟了两条街，我姑姑一边哭一边说，回去吧爸，回去吧！下午就准备回新乡上班，毕竟在家了七天，结果车刚开出村口，轮子就卡在路边上了，老家的路你们都知道，挨着垄沟的都是空的，结果给我爸打电话把车给拖出来了，换了个备用轮胎，忙完就是俩仨小时，结果回新乡的路上，发现前方两三个小时前出了大车祸，好几个车里的人都没了，还有一辆大车也翻了，追尾追了好长……我开车一般都是60.70码……细思密恐，爷爷总是最爱我的，想尽一切办法保护我……愿天堂安好</p>
