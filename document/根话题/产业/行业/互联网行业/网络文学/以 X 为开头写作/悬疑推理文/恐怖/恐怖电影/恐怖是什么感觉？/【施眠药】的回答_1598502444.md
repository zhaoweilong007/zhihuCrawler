# 恐怖是什么感觉？
- 点赞数：9360
- 更新时间：2024年03月11日15时51分40秒
- 回答url：https://www.zhihu.com/question/23055391/answer/1598502444
<body>
 <p>前女友在 qq 上给我发了段监控录像，录像里她老公的脖子 180 度向后扭，之后她发出一句话：「救我！」。</p>
 <p>「许泽，你相信因果报应吗？」</p>
 <p>深夜，前女友阿珉突然发来这么一段话。</p>
 <p>「我好怕。」还没等我回复，阿珉又发来信息。</p>
 <p>「你怎么了？」我有些困惑，阿珉是我的初恋。她给我的感觉是个挺严肃的人，几乎不会开玩笑。</p>
 <p>「你相信这世上有鬼吗？」她重复。</p>
 <p>「相信。」我想了想：「举头三尺有神明。」</p>
 <p>「不在头顶。」对面一字一顿：「就在我的身边。」</p>
 <p>「你指的是什么？」我彻底懵了。</p>
 <p>对面却沉默了。</p>
 <p>「你是遇到什么事了吗？」我打字。</p>
 <p>毕竟也是曾经喜欢过的人，她如果真的遇到什么难事，比如经济方面的，我愿意想办法帮帮她。</p>
 <p>大概有十分钟的时间，对面一直是沉默状态。</p>
 <p>「你是在玩真心话大冒险吗？」我问道。</p>
 <p>自从阿珉结婚后，我俩基本没有怎么联系过，现在她突然找我，话又说得莫名其妙，如果不是她神经了，那么很有可能是在玩什么整人游戏。</p>
 <p>而现在这个认真又严肃的我，俨然已是她和朋友们的笑料了吧。</p>
 <p>就在我以为这是一个恶作剧，阿珉不会再回复我时。</p>
 <p>电脑右下角的 QQ 又闪了起来，我点开聊天框。</p>
 <p>「阿珉请求与你视频。」</p>
 <p>我看着视频请求陷入沉默。</p>
 <p>我有些轻微的社恐症，今天晚上她突然发来这些莫名其妙的话，如果一会打开视频发现真的是个恶作剧的话。</p>
 <p>我想我接下来的一个月都会做噩梦吧。</p>
 <p>可万一是真的有事呢，我内心纠结着。</p>
 <p>聊天框嗡嗡地一直震动着，犹如一个定时炸弹，我思前想后，还是没有接。</p>
 <p>「有什么事直接打字说吧。」我说道。</p>
 <p>「救我！」</p>
 <p>许久，对面传来一张图片文件，文件署名：陆葵。</p>
 <p>如果阿珉是在拍悬疑电影。那么毫无疑问，她成功勾起了我的好奇心。</p>
 <p>文件按格式来看是动态图，我移开停在关闭键上的鼠标，选择接收文件，想看看到底是怎么回事。</p>
 <p>图片打开，虽然不知道为什么被加了一层灰色滤镜导致看上去有些模糊，但仍旧看得出是一个青年男人。</p>
 <p>「这是什么？」我问阿珉。</p>
 <p>画面似乎有些跳帧，轻微闪动着，男人背对着画面。</p>
 <p>我盯着看了五六秒也没看出什么名堂，于是准备关掉图片。</p>
 <p>就在鼠标移到关闭按钮的一瞬间，男人动了。</p>
 <p>已经有些倦意的我差点被吓到桌子底下去。</p>
 <p>他慢慢转过头，突然好像是发现了什么似的抬头看向我。</p>
 <p>男人面无表情，再加上灰色滤镜，有一种说不出的诡异。然而还没等我看仔细，画面突然跳动，又回到了男人背对着我的样子。</p>
 <p>QQ 上阿珉还没有回复我。</p>
 <p>阿珉不是一个喜欢装神弄鬼的女人，这一点我再了解不过。</p>
 <p>我看着图片上的男人，尽管令人不舒服，但莫名地越看越熟悉。</p>
 <p>我又看向文件名：陆葵，看来是这个男人的名字。</p>
 <p>我和阿珉的大学朋友里没有叫这个名字的，突然我的脑子里灵光一闪。</p>
 <p>陆葵，不就是阿珉老公的名字吗！</p>
 <p>虽然搞清楚了男人的身份，但是我却更加迷惑了：阿珉把自己老公发给我干什么，难道是想介绍我俩认识不成。</p>
 <p>我想了想作为前男友的我和现老公见面的场景，不寒而栗。</p>
 <p>我看了一眼阿珉的 QQ 头像，已经变成了灰色下线的状态。</p>
 <p>事情越来越奇怪，如果和我最开始猜测的那样，阿珉是在和朋友玩整蛊游戏，那么完全没有必要把自己老公的照片都给我发过来。</p>
 <p>莫非她们是想看看我这个前男友的反应？</p>
 <p>我几乎是立刻排除了这个猜测，太离谱了，正常人是不会拿朋友的家庭和睦开玩笑的。</p>
 <p>她的最后一句话是：「救我。」</p>
 <p>我想了想，总觉得不是什么大事，因为如果真的到了快没命的程度，正常人应该都知道警察比前男友管用。</p>
 <p>阿珉可能真的遇到了什么事，而这件事或许和我有关？我心里这么想着。</p>
 <p>我看着图片，男人最后那个抬头看向镜头的动作，说明摄像机在他上方较高的位置上，再加上那个似曾相识的灰色滤镜。</p>
 <p>我的脑子里突然闪电般掠过一个念头：</p>
 <p>这根本就是监控摄像的画面啊！</p>
 <p>如果真的是监控的画面，那么一切就好解释了：</p>
 <p>阿珉在家中安了监控来监视自己的老公，但是很明显，陆葵最后的动作说明他发现了监控的存在。</p>
 <p>所以阿珉因为被自己老公发现了家里的摄像头而向我求救？</p>
 <p>我想了想，总觉得另有蹊跷，我看向图片，画面中的男人不断重复转头的动作，我内心不舒服的感觉愈加强烈。</p>
 <p>当然不是出于前任对现任的那种奇怪情愫，而是一种生理上的不适应</p>
 <p>我点起烟，盯着动图看起来，试图找出让我觉得不舒服的地方。</p>
 <p>整个动图以十四秒为一个循环，分别是男人背对着监控、男人转头看向监控。</p>
 <p>就这样一直重复着，就在我以为不会有什么新发现准备掐灭烟去睡觉的下一个瞬间，我终于意识到了整个动图最诡异的地方：</p>
 <p>也许是时间太晚了，我的注意力并不集中，竟然在一开始没有发现图片中这么明显的异样，我颤抖着看向屏幕：</p>
 <p>画面中男人无论是背对还是转头看向摄像头时，永远都只有头转了过来！</p>
 <p>而自始至终，他的身子都是背对着监控。</p>
 <p>把脑袋像橡胶一样 180 度扭向身后，这根本不可能是人！</p>
 <p>我突然有一种想吐的感觉，努力压抑着狂跳的心脏。</p>
 <p>人的头不可能做到 180 度转向。</p>
 <p>除非画面中的不是人。</p>
 <p>手在颤抖，我开始后悔自己没有把卧室的灯打开，有那么几分钟我甚至不敢回头。</p>
 <p>结果导致我只能僵硬地坐在电脑旁，看着画面中的那个叫做陆葵的男人一次又一次地把脖子向后扭出非人的角度。</p>
 <p>好在人的适应力是很强大的，重复看了几次之后渐渐没了之前恐怖的感觉，理性又占据了主导，我开始重新思考事情的合理性。</p>
 <p>首先，除非对方是猫头鹰转世，否则人的脑袋是绝对不可能做到像图上那样的扭动。</p>
 <p>很有可能是合成照片，但我的前女友合成了一张自己老公的鬼图就为了在二半夜吓我一跳？</p>
 <p>我觉得自己和阿珉的脑子总要坏掉一个。</p>
 <p>我看了看图片，目光又落在阿珉发过来的「救命」上。终于忍不住，咬咬牙，发送了视频邀请。</p>
 <p>总不能当什么也没发生过。</p>
 <p>视频邀约静静地闪动着，直到画面显示无人应答。</p>
 <p>挂掉视频，比起心中的一丝失落，我更多的是松了一口气，社恐症让我做到这一步已经是极限了。</p>
 <p>就在我准备关掉 QQ 时，阿珉的头像又亮了起来。</p>
 <p>我的心几乎是在同时猛地沉了下去。</p>
 <p>很快，聊条框上方显示：对方正在输入中。</p>
 <p>「你看过图片了？」对面问道。</p>
 <p>我刚想打字说我看过了，猛然间又觉得哪里不对劲，对面的语气明显透着一股陌生。</p>
 <p>我删掉了打好的话：「没有，不会是什么鬼图吧。」</p>
 <p>「是。」对面的回复速度很快，就像是在等我一样：「你最好不要点开。」</p>
 <p>「你又不是不知道我最怕这种了，我才不点开。」</p>
 <p>「那就好。」对面干脆地回复：「没事了，删了它吧。」</p>
 <p>我想了想：「行。」</p>
 <p>「你录制屏幕给我看看。」对面突然说道：「录制一下你删掉图片的过程。」</p>
 <p>「凭什么？你到底是谁？」我不知怎么的来了勇气，质问道。</p>
 <p>对面没有再说话，阿珉的头像又暗了下去。</p>
 <p>就在我准备追问时，一只手突然搭在我的肩膀上。</p>
 <p>我被肩膀突如其来的触感吓得差点原地去世。</p>
 <p>「你在干嘛？」手的主人发出声音。</p>
 <p>我向后看去，只看到白斐那张惨白的脸。</p>
 <p>一个月前，我和这位网聊三年却从未谋面的网友面基，谁曾想那之后发生了一件诡异的事情。</p>
 <p>简而言之，白斐救了我的命，为了报恩，我收留了据说是无家可归的他。</p>
 <p>白斐看了看我，似乎对吓到我并不在意：「一晚上了，你在干什么？」</p>
 <p>他的声音总有种不可抗拒的魔力，我虽然被吓了一跳很生气，但还是如实回答了他。</p>
 <p>「前女友？」白斐上下打量着我。</p>
 <p>「什么意思？」我被他看的浑身不自在。</p>
 <p>「没什么……」白斐道：「能让我看看吗？」</p>
 <p>我起身把电脑让给他。</p>
 <p>「唔。」白斐死盯着屏幕，这一声算是回应了我。</p>
 <p>白斐看得很快，几乎是一目十行。就在我怀疑他到底有没有认真时，他直接打开了那张动图。</p>
 <p>接着，一声轻微的：「咦。」从他嘴里发出来。</p>
 <p>「有头绪？」我忙问道。</p>
 <p>白斐没有说话，只是盯着那张图。</p>
 <p>大概有五分钟的沉默，就在我以为时间静止了时，白斐突然扭头看向我。</p>
 <p>「是合成图吗？」我问。</p>
 <p>白斐摇头。</p>
 <p>「你给她发送了视频但是没人接？」白斐把鼠标放在聊天框里。</p>
 <p>「是，我想确定一下到底是什么事。」我如实答道。</p>
 <p>「图片不是合成的。」白斐说道。</p>
 <p>「真的有人可以把头扭到这个角度？」我有些诧异。</p>
 <p>「可能不是『人』。」白斐说道。</p>
 <p>「你的意思这个男人不是人？」我笑道：「别扯了。」</p>
 <p>「你不记得之前的事了吗？」白斐道：「你觉得那时遇到的是什么呢？」</p>
 <p>我顿时无言：「我只记得你救了我。」</p>
 <p>「那就够了。」白斐笑了一下。</p>
 <p>我刚想追问他关于那件事的真相，白斐却打断了我：「你和这个女的关系好吗？」</p>
 <p>「现在倒也谈不上。」我说：「只是多年前的恋人关系罢了。」</p>
 <p>「哦。」白斐淡淡地回应：「她可能已经死了。」</p>
 <p>「你说什么？」我的脑袋像是被人用木棍敲了，嗡的一声。</p>
 <p>「只是说存在这种可能。」白斐依旧一脸无所谓的样子。</p>
 <p>对于他的冷漠我并不感到奇怪，这家伙从我们认识起就是这样，仿佛世间的一切都与他无关似的。</p>
 <p>「为什么这么说？」我问道：「与陆葵有关吗？」</p>
 <p>「还不清楚。」白斐摇头。</p>
 <p>我看向 QQ 聊天框，阿珉的头像始终是灰色的，就像不会再醒来一样。</p>
 <p>「不会这么快吧。」我抱着最后一丝希望：「一个小时前我们还在聊天。」</p>
 <p>「你知道她家在哪吗？」白斐沉默了一会说。</p>
 <p>我是知道阿珉住哪里的，但是她结婚后我们两年没有怎么聊过，所以不太确定她有没有搬家。</p>
 <p>「我们现在过去。」白斐迅速说道：「或许来得及。」</p>
 <p>到 Z 市时天已经快亮了，兜转了几圈后我还是凭着记忆找到了阿珉所在的光明园小区。</p>
 <p>车越往目的地开，我内心的不安就越多，终于在到达光明园小区的那一刻达到了顶峰。</p>
 <p>有什么事情已经发生了，就在小区的里面。</p>
 <p>出现在我们的面前的，是不断跑动呼喊的保安。</p>
 <p>我俩将车停在稍远的路口，然后装做刚下夜班回家的样子走了过去。</p>
 <p>来往都是神色紧张的保安，一些人不停地用对讲机呼喊着什么，甚至没有人注意到我俩。</p>
 <p>「这是怎么了？」白斐拦住一个保安问道。</p>
 <p>保安忙得甚至看都没看我们：「出事了出事了。」</p>
 <p>说完便急忙向远处跑去。</p>
 <p>我和白斐对视了一眼，急忙跟在他后面。</p>
 <p>小区很大，我和白斐跟在保安后面，估计是把我俩当成了看热闹的人，所以保安并没有管我们。</p>
 <p>虽然还只是清晨，但远远就看到 C 栋楼的前面围了一圈人。</p>
 <p>我的心猛地抽了一下，记忆中 C 栋是阿珉住的地方。</p>
 <p>保安为了保护现场把人群分割开来，不少心里素质差的人已经吐了，只有少数人还在看着楼梯口。</p>
 <p>我和白斐来到人群前面，眼前的一切让我的灵魂瞬间被抽离身体……</p>
 <p>阿珉死了。</p>
 <p>据旁边的邻居说，阿珉是被清晨巡逻的保安发现的，当时就已经没有了生命体征。</p>
 <p>「闹鬼了。」不断有人这么说道。</p>
 <p>很明显不是自杀，任谁看到现场都会做出这样的判断。</p>
 <p>因为这样的死亡方式根本不可能是自杀，令人匪夷所思的是她的死法。</p>
 <p>1986 年，著名魔术师大卫·科波菲尔在众目睽睽之下成功穿过长城。</p>
 <p>如今看起来，阿珉更像是表演失败的魔术师。</p>
 <p>她的头和下肢完好无损，不过它们之间隔了一道厚厚的墙。</p>
 <p>我的身体不受控制地瘫软，白斐像早有预感似的扶住了我。</p>
 <p>「她死了。」白斐轻声道，像是对一切做了总结。</p>
 <p>我什么也听不进去，脑子里像过电影一般不断重复着有关阿珉的画面。</p>
 <p>「救我。」她说。</p>
 <p>我努力张开嘴，却发现自己根本发不出任何声音。</p>
 <p>白斐拍拍我的后背。</p>
 <p>「你知道她家在几楼吗？」白斐问道。</p>
 <p>我和阿珉大学毕业后就分手了，老实说我还真没来过这里，之所以知道她的住址也是在以前的聊天里听她提起才知道的。</p>
 <p>「六楼。」</p>
 <p>白斐自言自语地抬头看向楼上，我也顺着他的目光看去。</p>
 <p>已是深秋，清晨的空气有些冷，整栋楼只有六楼的窗户开着。</p>
 <p>「你说他会不会正在看着这里。」白斐突然说道。</p>
 <p>「你说阿珉的老公？」我立刻反应过来。</p>
 <p>「你见过老婆死了还不出现的人吗？」白斐问道。</p>
 <p>「或许他还在睡觉呢？」我反问道，</p>
 <p>「这么冷的天气还开着窗。」白斐道：「他已经醒了。」</p>
 <p>「他在窗口？」我抬头看向六楼，隐约中似乎真的看到人的衣角，不禁不寒而栗。</p>
 <p>「现在怎么办？」我问白斐。</p>
