# 有没有什么细思恐极的恐怖故事？

- 更新时间：2024年03月19日16时53分07秒

<body>
 <blockquote>
  室友穿男装很帅，我把两人合照发网上，有人回复：【小心这个男的，他眼露三白，好色贪淫，是个变态杀人犯！】
  <br>
   【你胡说什么，她是女的！】
  <br>
   我不以为然，晚上，室友却悄悄摸上了我的床。
 </blockquote>
 <p>1</p>
 <p>我像往常一样，把自己跟室友周然做饭的短视频发到网上，很快，粉丝们就纷纷点赞。</p>
 <p>【周然帅出新高度！】</p>
 <p>【不是，你怎么跟张凌赫住一起啊？死丫头，你搬出去，给我住两天！】</p>
 <p>【我宣布我的取向是帅，不管男女。】</p>
 <p>一会儿工夫，留言就增加到了几百条，我随意扫了几眼，正准备关掉，其中一条评论却飞快地被顶到了最上面。</p>
 <p>我仔细一看，顿时哭笑不得。</p>
 <p>【同学，小心这个男的，他眼露三白，好色贪淫，是个变态杀人犯！】</p>
 <p>对方的 ID 名叫老神棍，头像是一张太极阴阳鱼的图片。</p>
 <p>我点进他的页面一看，只见主页上挂着几张灵符，下面还带了小黄车，但是符文都很贵，几万十几万一张，一单成交的都没有。</p>
 <p>这年头，骗子的胃口怎么越来越大了。</p>
 <p>不等我回复，其他粉丝早就纷纷反驳。</p>
 <p>【笑死，果然是老神棍，装神弄鬼骗人。】</p>
 <p>【对啊，我们周然是女生啊，还变态杀人犯，离谱死了。】</p>
 <p>熟悉我的网友都知道，周然是女生，也是我合租的室友。</p>
 <p>我们住的这个小区都是两室一厅的户型，由物业公司直租，而且只允许同性别的人合租。</p>
 <p>当初进门时看见周然，我还以为我走错房间了。周然个子很高，留着利落的短发，眉骨硬朗，看着简直不比男明星差。</p>
 <p>我拉着行李箱要走，周然直接掏出身份证给我看。</p>
 <p>「你好，我叫周然，性别——女，不要弄错了哦。」</p>
 <p>我这才明白，是自己搞了个乌龙。</p>
 <p>周然性格很好，为人大方，而且很爱做饭，经常买好吃的跟我分享。我也会把两人相处的视频发到网上，几天时间，就积攒了不少粉丝。</p>
 <p>2</p>
 <p>怕掉粉，我不敢直接骂人，而是礼貌地回复：【不好意思啊，我室友不是杀人犯，她是个女生。】</p>
 <p>老神棍：【不可能！</p>
 <p>【男人藏精于骨，你看他眉骨分明，这是乾命，肯定是个男人。</p>
 <p>【小姑娘，你相信我，我看了几十年面相了，不会搞错的。这人头尖额窄，眼露三白，性格非常暴躁，漠视人命，是穷凶极恶之辈。</p>
 <p>【他生性好淫，现在又扮成女人接近你，八成是有什么特殊的虐待癖好，你快跑吧！】</p>
 <p>老神棍发了一大串留言，我看得很生气。</p>
 <p>【你胡说八道什么，周然脾气很好的！</p>
 <p>【我们楼上的邻居老是半夜三更发出很大的噪音，周然都没跟她吵过架。】</p>
 <p>我楼上住着个当主播的漂亮女生，半夜三更经常在房里拖动椅子，穿着高跟鞋走来走去，弄出很大的动静。</p>
 <p>我上去说了几次，那女生却连门都不肯开。</p>
 <p>还是周然劝我，说她虽然吵，但每天晚上也就折腾半小时左右，大家都是邻居，忍一忍算了。</p>
 <p>搬进小区以来，我就没见过周然跟谁红过脸，这么好的脾气，怎么可能是杀人犯啊。</p>
 <p>我不想跟网友吵，直接把这条评论折叠了。</p>
 <p>没想到对面却不依不饶，发了一条私信给我。</p>
 <p>【你头发枯黄，六亲无缘，日角低陷且有横纹，你爸是不是在你五岁那年意外去世了？】</p>
 <p>屏幕上短短的几行字，我心里却顿时「咯噔」一下，情不自禁地捏紧了手机。</p>
 <p>我读幼儿园的时候，我亲爸就车祸去世了。后来我妈带着我改嫁到外地，跟现在的继父在一起，也让我喊爸爸。</p>
 <p>继父对我还不错，连同学们都不知道我们没有血缘关系。</p>
 <p>这个算命的，又是怎么知道的？</p>
 <p>【你疾厄宫有两条横纹，去年肯定发生过意外，还有一条是今年的，能不能避过去就看你自己了。】</p>
 <p>发完最后一条消息，老神棍就没反应了，手机屏幕转黑，但我的心里却久久无法平静。</p>
 <p>我去年从楼梯上摔下来，小腿骨折，在医院躺了半个月。</p>
 <p>这个老神棍，真的有点东西啊！</p>
 <p>3</p>
 <p>周然她——该不会真的是男的吧？</p>
 <p>我呆呆地坐在床上，仔细回忆着两人相处的画面，就在这时，手机忽然又振动了起来。</p>
 <p>我打开微信界面一看，发现是隔壁的邻居小美，给我发了好几条信息。</p>
 <p>【陈希，你快看群啊。</p>
 <p>【好几个女生被尾随了，真可怕，我刚准备出去吃夜宵呢，吓得我不敢出门。】</p>
 <p>我忙切到业主群，翻看聊天记录。</p>
 <p>邻居们讨论得热火朝天，大部分在骂物业，也有少数几个男邻居，隐晦地提醒大家衣服要穿保守一点，被其他女生追着骂。</p>
 <p>我们这个小区，夹在一个废弃的旧厂房和一个老旧的小区中间，前面大门正在修路，两头被封死了，出去的话只能从北门走。</p>
 <p>北门是条狭窄的小路，车子进不来，要穿过很长一条幽深的巷道，才能到外面大马路上。</p>
 <p>那几个女生就是在那条巷道里被尾随的。</p>
 <p>莹莹：【大家可千万别出去，我手现在还在抖，五六个文身的肌肉男，喝得烂醉，真的好吓人。】</p>
 <p>老许（3-1001）：【报警了吗？】</p>
 <p>莹莹：【报了，警察说等会儿会派人去看的，但我估摸着也不会有什么结果，他们就跟着我说下流话，没来得及动手我就跑了。】</p>
 <p>路人甲：【是不是看你穿得太漂亮了想搭讪啊？误解别人意思不好吧？】</p>
 <p>莹莹：【滚！这人怎么没备注房号，麻烦物业把他踢出去！】</p>
 <p>路人甲：【踢你妈，你自己也没备注啊，你算什么东西？】</p>
 <p>两人吵了起来，我好奇地点开莹莹的头像，想看看这是哪栋的业主，一点进去，才发现我们两人之间有好友。</p>
 <p>看着她朋友圈背景图那张招摇的低胸照片，我这才反应过来，她就是我们楼上那个女主播。</p>
 <p>4</p>
 <p>我回复小美。</p>
 <p>【这个莹莹我认识，就住我们楼上，做直播那个，长得很漂亮。】</p>
 <p>小美回了我一连串问号：【你在说什么，那个女生不是早就搬走了吗？】</p>
 <p>我心头重重一跳。</p>
 <p>【不可能啊！她昨天晚上特别吵，周然还上楼找她了，我听见他们说话的声音了。】</p>
 <p>小美：【你肯定弄错了，她搬走的时候，我还帮她把行李拿到楼下，她跟我说要回老家以后不来了。】</p>
 <p>我顿时头皮发麻。</p>
 <p>这到底是怎么回事，如果莹莹已经搬走了，她为什么要在群里说自己遇见色狼，而且每天晚上在我们楼上发出那么大动静的，又会是谁呢？</p>
 <p>我不敢往深了想，手脚发凉，心脏「怦怦」直跳。</p>
 <p>就在这时，屋外忽然传来防盗门被打开的声音。</p>
 <p>「陈希，我带夜宵回来了，你睡了吗？」</p>
 <p>我吓一跳，慌乱地在屋子里转了一圈，想往床下躲，一弯腰，才发现我这床是矮床，没有床底。</p>
 <p>衣柜里又塞满了衣服，这卧室里，根本没有可以藏人的地方。</p>
 <p>「陈希——你这么早睡着了？」</p>
 <p>周然的声音已经到了房间门口，我不知道该怎么办，只能脱掉鞋子往床上一躺，抓过被子蒙住头。</p>
 <p>「嘎吱——」</p>
 <p>门把手旋转，卧室的房门被打开。</p>
 <p>低沉的脚步声响起，我屏住呼吸，心跳到了嗓子眼。</p>
 <p>「今天睡这么早啊。」</p>
 <p>一只手探过来，把被子从我头顶往下掀。</p>
 <p>我闭着眼睛，一动不敢动。</p>
 <p>5</p>
 <p>松软的床垫往下陷，周然在我旁边坐了下来。</p>
 <p>我虽然闭着眼睛，也能感觉到她的视线在我身上扫来扫去，带着一股不怀好意的阴森感。</p>
 <p>「真睡着了？」</p>
 <p>周然伸手摸我的脸，两根手指顺着脸颊，一路往下。</p>
 <p>冰冷粗糙的触感，激起一阵战栗，我几乎要控制不住身体的颤抖了。</p>
 <p>就在这时，楼上忽然发出「咚！」的一声巨响，像是椅子倒地的声音，周然的手一顿。</p>
 <p>过一会儿，好像有人拖着椅子走来走去，响声更大了。</p>
 <p>「真他妈烦！」</p>
 <p>周然不耐烦地站起身，朝头顶喊。</p>
 <p>「你有完没完，能不能安静点？」</p>
 <p>她喊得太响，我不能再装睡了，我犹豫片刻，深深吐出一口气，睁开眼睛，做出一副迷迷糊糊的样子。</p>
 <p>「周然，你回来了，怎么了？」</p>
 <p>「没事，楼上又开始了，我上去跟她说一下，你继续睡吧。」</p>
 <p>周然安慰似的朝我笑了一下，转身带上门出去了。</p>
 <p>等她一走，我立刻扑过去反锁房门，身体贴在门上，大口大口喘气。</p>
 <p>放在口袋里的手机又振动起来，小美发了一张图片过来。</p>
 <p>【这是物业给我的，我就说吧，楼上根本没有人住！】</p>
 <p>图上是一张小区各户的动态表，表格里显示，6 栋 1301，状态为「空置」。</p>
 <p>正常的空房都是「待租」，那些房东没有出租需求的空房，才会特别标注为「空置」两个字，</p>
 <p>我脑子有点僵。</p>
 <p>【可是楼上又发出噪音了，周然刚上去啊。】</p>
 <p>楼上那个莹莹，脾气差，又是个话痨，每次周然上去，都能听见她在那边大喊大叫，周然压低了嗓音小声劝她。</p>
 <p>两个人要说好一会儿话，周然才会下来，楼上是绝不可能没人住的。</p>
 <p>但小美发的这个图片，也不像是假的。</p>
 <p>我大脑机械地转着，联想到刚才那个老神棍说的话，心里忽然有了一个大胆的猜测。</p>
 <p>难道说，周然在楼上，囚禁了一个女人？</p>
 <p>周然不会真的是变态吧！</p>
 <p>6</p>
 <p>我被自己的念头吓到，忙掏出手机，想找小美把情况说清楚，然后去她房里躲着，再报警。</p>
 <p>可一翻到我们两人的聊天界面，才发现小美又发了一连串信息。</p>
 <p>【你怎么老不信我，我骗你有什么好处？</p>
 <p>【楼上根本不可能有人啊！</p>
 <p>【周然上去也没用！八成一个人在走廊里傻站着呢，你等着，我去给你拍个照片！】</p>
 <p>我握紧手机，倒吸一口冷气。</p>
 <p>完蛋了，小美做事情怎么这么莽撞啊！</p>
 <p>这要是去楼上，正好撞到周然——我不敢再想下去，立刻冲到客厅，打开防盗门。</p>
 <p>我跑到走廊上的时候，看见电梯门正缓缓合上，小美穿着个粉色的摇粒绒睡衣，开心地朝我挥手。</p>
 <p>我扑过去按下电梯键。</p>
 <p>「小美，你听我说，你——」</p>
 <p>「哎呀你来得正好，走，我带你上去看看。」</p>
 <p>小美不由分说，一把拉住我的手，把我往电梯里拽。</p>
 <p>我大惊。</p>
 <p>「你放手，我不去！</p>
 <p>「你听我说，周然是个杀人犯，他在楼上囚禁了一个女人，我们得快点报警！」</p>
 <p>「哈哈哈，陈希，你在胡说什么啊？</p>
 <p>「你忘记了，周然那个朋友就是警察啊，不是来你家好几次了，警察能跟杀人犯当朋友？」</p>
 <p>我一愣，这才想起来，周然确实有个当警察的朋友，叫李俊杰，长得人高马大的。</p>
 <p>有一次周然喝醉，还是李俊杰把周然送回家的，他还穿着警服，吓了我一跳。</p>
 <p>如果周然是在逃的杀人犯，李俊杰不可能不知道吧？</p>
 <p>这么一想，我又感觉是自己疑神疑鬼了，周然回家之后，确实也没对我做什么啊，都是那个老神棍在乱说话。</p>
 <p>7</p>
 <p>「奇怪，如果他不是变态，那 1301 里面的女人，到底是怎么回事啊？」</p>
 <p>我百思不得其解，小美按下 13 楼的电梯键。</p>
 <p>「哪里来的女人，去看看不就知道了。</p>
 <p>「你就放心吧，现在都什么年代了，你看走廊里都是监控，哪里来的杀人犯啊？</p>
 <p>「就算他真的是变态，我可是跆拳道黑带，看我的，嘿——哈——我们两个还打不过他一个？」</p>
 <p>小美对着空气劈砍了几下，她胆子向来很大，人又直爽，我和她第一次相遇，就是我和几个跳广场舞的大妈起了冲突，她冲出来替我打抱不平。</p>
 <p>她一个人舌战一群大妈，两手叉腰，唾沫横飞，丝毫不弱下风。可以说，小美是我在这个小区交到的第一个朋友。</p>
 <p>她天不怕地不怕的情绪感染了我，我心里也轻松不少。</p>
 <p>电梯很快就到了 13 楼，电梯门左手边尽头那户就是 1301，门口空荡荡的，不像其他人家，摆着鞋架铺着地毯。</p>
 <p>小美指着那扇黑色的大门。</p>
 <p>「看见没有，我就说这里没人，周然应该在——」</p>
 <p>我们边说边朝走廊深处走，很快，小美脸色僵硬，说不下去了。</p>
 <p>因为那扇黑色的防盗门，居然开着一条缝。</p>
 <p>「奇怪，这门怎么开着？」</p>
 <p>小美握住门把手朝外拉，门开得更大了，她探着身体朝里看了一眼，眼珠子立刻瞪得老大。</p>
