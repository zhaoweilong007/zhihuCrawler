# 你所经历的哪些事情让你怀疑这个世界是假的/有 bug 的？
- 点赞数：18494
- 更新时间：2018年02月28日15时45分26秒
- 回答url：https://www.zhihu.com/question/61917117/answer/330578412
<body>
 <p data-pid="5Qg6o13E">读大学时有一个雨夜，躺在寝室的床上睡得很沉。到凌晨的某一刻，睡梦间耳畔忽然响起一个中年女声：“她忽然被一声炸雷惊醒。”话音刚落，窗外“轰隆隆”一声巨响，像是整个天空被什么劈裂的声音。我瞬间被惊醒，睁开了眼睛。回想起刚才的一切，惊诧。 在这个电闪雷鸣的雨夜里，开始思考一种可能：我是不是活在某个人的剧本里，我每天的生活都只是剧情需要。到现在已经有七八年了，每次想起来都要细细思忖自己是不是睡糊涂了，先醒了然后自己编出那句话的。但是怎么回忆都是那样，一个女人先宣读了剧情，雷声炸，我才醒的。真是匪夷所思。</p>
