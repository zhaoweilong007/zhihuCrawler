# 你所经历的哪些事情让你怀疑这个世界是假的/有 bug 的？
- 点赞数：12081
- 更新时间：2018年04月06日17时14分09秒
- 回答url：https://www.zhihu.com/question/61917117/answer/360018245
<body>
 <p data-pid="2PZBUl6R">小时候，一次妈妈生病了，好几天没吃饭，念叨着想吃鱼，那时候家里挺穷的，根本就没钱买鱼。我家有一个池塘，那是夏天，池塘里的水很深。我不知道怎么想的，强烈的想捉条鱼给妈妈吃。我从池塘边上下去了，慢慢的往池塘中间走，我就一个人，手里啥都没拿。走了没几步，突然有条鱼撞我腿上了，然后这条鱼一跃而起，自己跳到岸上去了，我赶紧冲回岸上捉住了鱼。就这样，妈妈吃上鱼了。当时真的觉得好神奇，连老天爷都在帮我。不然那么大的池塘，赤手空拳怎么能抓到鱼。后来，还被爸爸惩罚了，说我这么下水，会被淹死。但我还是特别骄傲，妈妈吃上我捉的鱼了。</p>
