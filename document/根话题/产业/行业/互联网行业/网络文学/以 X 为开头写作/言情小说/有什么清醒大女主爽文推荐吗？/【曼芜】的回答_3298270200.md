# 有什么清醒大女主爽文推荐吗？
- 点赞数：6217
- 更新时间：2024年03月11日16时54分23秒
- 回答url：https://www.zhihu.com/question/627267758/answer/3298270200
<body>
 <p>我和姐姐是双生凰鸟。</p>
 <p>江国皇帝将我们姐妹两人一起迎入宫，双双册封为皇后，享受齐人之福。</p>
 <p>皇帝为民祈福那日，天降异象。</p>
 <p>两只凰鸟一只坠落，一只腾飞。</p>
 <p>国师断言，坠落的那只凰鸟会为江国带来厄运。</p>
 <p>我被下令斩杀，尸体于熊熊烈火中燃烧。</p>
 <p>再睁开眼，我从林间醒来。</p>
 <p>姐姐稳坐凤位，她掩面而笑：「江国有本宫一位皇后就够了。」</p>
 <p>可她不知道，我一手策划自己假死，只为了逃脱皇宫那个囚笼。</p>
 <p>凰鸟本该自由翱翔于九天，而非困于深宫。</p>
 <p>我和姐姐一样都是旺夫命，但我选择旺自己。</p>
 <p>1</p>
 <p>我和孪生姐姐虞惜芝一同被册封为皇后那日。</p>
 <p>国师向皇帝进言：「双生凰鸟要并肩齐飞，两位皇后理应一同侍寝，方可保陛下洪福齐天。」</p>
 <p>江曜听信了国师之言。</p>
 <p>是夜，他召我和姐姐一同去给他侍寝。</p>
 <p>我们坐着双人轿往皇帝寝宫而去。</p>
 <p>路上，虞惜芝秀眉轻挑望着我：「就国师事多，就算有两位皇后又如何？长幼有序，今夜理应我来侍寝。」</p>
 <p>话落，她问我：「你说呢？栖月。」</p>
 <p>「是啊，今夜姐姐一人侍寝便可。」我叹气，撩开车帘。</p>
 <p>轿子在宫道里穿行，窗外只能看见高高的城墙，和头顶那方灰蒙蒙的苍穹。</p>
 <p>皇宫宛如一个巨大的鸟笼，将我和姐姐囚禁于深宫。</p>
 <p>姐姐想要努力争夺笼主的宠爱，可我却只盼望着能够飞出困住我的这方囚笼。</p>
 <p>虞惜芝很满意我的回答，她勾唇浅笑：「栖月，姐姐和你商量个事。」</p>
 <p>我点头：「姐姐直说便是。」</p>
 <p>她握着我的手道：「待会到了陛下寝宫，我先进去侍寝，你在外面候着。</p>
 <p>「待我侍寝完毕，出来换你时，我们只是走个过场，进去的人还是我，如何？」</p>
 <p>虞惜芝怕我不答应，又向我撒娇：「求你了，我的好妹妹，你就让姐姐这一次吧。」</p>
 <p>2</p>
 <p>我试探着问：「阿姐，你当真甘心被困在深宫里？」</p>
 <p>虞惜芝像是听见了天大的笑话：「妹妹，你说的是什么话？什么叫被困于深宫？</p>
 <p>「我们入宫当皇后，分明就是享受荣华富贵，天下多少女子求之不得？」</p>
 <p>说完，她又嘲讽道：「你啊，真是得了便宜还卖乖，又当又立。</p>
 <p>「也不知你怎么想的。这世间，哪里还有比皇宫更好的地方？」</p>
 <p>虞惜芝说话历来尖酸刻薄，尤其在我面前。</p>
 <p>我脸沉下来。</p>
 <p>她看见后，意识到自己还有事求我，当即转变了态度：</p>
 <p>「哎呀，瞧姐姐这张嘴，竟说些大实话，妹妹别放在心上。」</p>
 <p>她说完又转回刚开的话题：「妹妹，待会你的侍寝机会，可以让给姐姐吗？」</p>
 <p>我不想侍寝，虞惜芝让我将侍寝的机会让给她，我求之不得。</p>
 <p>可我担心这么做，会引来大祸。</p>
 <p>陛下若知道两次进去的都是姐姐，不计较倒也无妨。</p>
 <p>可若他计较，往重了说，是欺君之罪。</p>
 <p>我不想侍寝，但也不想死。</p>
 <p>我反问：「阿姐，你可想过，若陛下发现又当如何？」</p>
 <p>虞惜芝不以为然：「怕什么？我们长相一致，声音更是可以以假乱真，只要我不说，你不说，陛下他不会知道的。</p>
 <p>「就算他知道，我也能有法子哄得他不去计较，妹妹就放心吧。」</p>
 <p>我面色平静道：「阿姐既这么想要，我让给你便是。」</p>
 <p>虞惜芝笑颜如花，嘴角上扬：「我就知道，栖月最好了。」</p>
 <p>3</p>
 <p>我站在寝宫外等候。</p>
 <p>寒风从窗台灌进来，吹碎了香案上的袅袅香烟。</p>
 <p>虞惜芝在寝宫里侍寝，里面隐隐传来让人面红耳赤的声音。</p>
 <p>约莫过了一炷香的工夫，里面消停了。</p>
 <p>江曜对虞惜芝吩咐道：「退下吧，将栖月叫进来。」</p>
 <p>「是，陛下。」虞惜芝穿好衣裳，出来唤我：「栖月，陛下叫你进去。」</p>
 <p>「嗯，知道了。」我应道。</p>
 <p>我们交换了一个眼神，她转身走进寝宫。</p>
 <p>我在外面站了会，听见姐姐学着我的语调和江曜说话。</p>
 <p>她学得很像，连我自己都有些愕然。</p>
 <p>别说江曜分不清楚，恐怕就是我阿娘听着也不一定能分得清楚。</p>
 <p>江曜果然没有起疑。</p>
 <p>翌日，陛下例行赏赐。</p>
 <p>虞惜芝来向我炫耀：「看来陛下很喜欢我，给我的赏赐比妹妹你的好。」</p>
 <p>随后又道：「对了，栖月，我昨夜一人分饰两角，演得还不错吧？陛下他没有起疑呢。」</p>
 <p>我敷衍应道：「姐姐真厉害。」</p>
 <p>虞惜芝面色红润，抚摸着小腹，娇羞道：「希望我能早日怀上龙嗣。</p>
 <p>「对了妹妹，往后但凡陛下宣你侍寝，就由我代替你去，直到我怀下龙嗣。</p>
 <p>「我为我们虞家争光，你也会为我高兴，是吗？」</p>
 <p>我波澜不惊，颔首：「是呢，只要阿姐开心，我就开心。」</p>
 <p>虞惜芝笑得花枝乱颤。</p>
 <p>4</p>
 <p>这样的日子持续了三个月。</p>
 <p>姐姐怀孕了。</p>
 <p>高兴之余，她来试探我：「栖月，我怀孕前三个月不可再侍寝了，你不会趁机和我抢陛下的宠爱吧？」</p>
 <p>「不会。」我摇头，顺口说，「从今日起，我称病，如何？」</p>
 <p>「太好了，妹妹真是体贴入微，不过......」她话锋一转，「我怕你装病被陛下发现，不如......你真病吧？」</p>
 <p>我微讶：「怎么真病？」</p>
 <p>「这还不简单。」虞惜芝从怀里摸出一包药粉。</p>
 <p>原来她早有准备，挖着坑等我跳。</p>
 <p>她拆开药粉倒进我面前的茶里。</p>
 <p>晃匀之后，端至我面前，浅笑盈盈地望着我：「你把这杯水喝下去，就会真病。</p>
 <p>「到时候就算是陛下想要宠幸你，恐怕也会乘兴而来，败兴而归。</p>
 <p>「我的好妹妹，快喝了吧。</p>
 <p>「这茶，只是让你身上起红疹子罢了，待我可以侍寝的时候，你这病自然就好了。」虞惜芝连哄带骗。</p>
 <p>我心里门儿清，她压根没想过让我得宠，这次让我真病，下次还有别的法子阻止我受宠。</p>
 <p>可我也确实不想得宠，更不想给江曜侍寝。</p>
 <p>「我喝便是。」我端着茶水，一饮而尽。</p>
 <p>5</p>
 <p>喝下茶水后，我身上开始起红疹子。</p>
 <p>虞惜芝猜得很准。</p>
 <p>是夜，江曜来了我的寝宫。</p>
 <p>他想要宠幸我，可看见我一身红疹子，便朝我摆了摆手：「罢了，改日吧。」</p>
 <p>「是。」我颔首，庆幸自己逃过一劫。</p>
 <p>可他竟还不走，在窗台旁的棋台上坐了下来。</p>
 <p>他朝我招手：「栖月，过来陪朕下一局棋。」</p>
 <p>「是。」我走过去，坐在江曜对面，陪他下棋。</p>
 <p>姐姐棋技一般，他和姐姐想来也下过棋。</p>
 <p>为了不露出马脚，我刻意收敛锋芒。</p>
 <p>一局结束，我输了，他胜了。</p>
 <p>谁知，他眸色一深，盯着我道：「栖月，你在让朕？这不是你真正的本事。」</p>
 <p>我含笑道：「陛下说笑了，是陛下棋技精湛，臣妾自愧不如。」</p>
 <p>下一瞬，他将棋子扫翻在地，眸底染满怒意：「虞栖月，你和惜芝还要骗朕到何时？」</p>
 <p>我心下一惊，他竟然察觉出来了。</p>
 <p>「陛下......」我正想解释。</p>
 <p>外面传来婢女的通传声：「芝皇后驾到。」</p>
 <p>虞惜芝被婢女搀扶着走进来，她看见棋子扫翻在地，以及江曜一脸怒容，已然明白发生了什么。</p>
 <p>她连忙跪下，禀道：「陛下息怒，是臣妾的错，妹妹说不想侍寝，求臣妾替她侍寝，臣妾一时糊涂，才会答应她。」</p>
 <p>我不敢置信地望着虞惜芝。</p>
 <p>姐妹一场，她就这样干脆利落地将我卖了？</p>
 <p>江曜眉头深锁，冷声问：「她为何不想侍寝？你们还有什么事瞒着朕？」</p>
 <p>「回陛下。」虞惜芝抢先答道，「栖月不想侍寝是因为她在宫外......有心上人。</p>
 <p>「她要为她的心上人守贞，所以今日特意服下起红疹的药，就是为了避宠。</p>
 <p>「殿下，妹妹她只是一时糊涂，我这个当姐姐的没有劝住她，求陛下网开一面，给妹妹一次改过自新的机会。」</p>
 <p>6</p>
 <p>虞惜芝无中生有，将矛头全都指向我。</p>
 <p>她很清楚，一个帝王最介意的事便是自己的妃子心中有别的男人。</p>
 <p>还为了别的男人，不让自己碰。</p>
 <p>这犯了江曜的大忌。</p>
 <p>他冷眸凝向我：「你就这么不想让朕碰？为了避宠，竟狠心给自己下药？」</p>
 <p>若此时我还不为自己辩解，那长着这张嘴有何用？</p>
 <p>我否认：「陛下，臣妾没有，药是姐姐哄骗我喝下的。」</p>
 <p>江曜冷睨向虞惜芝：「惜芝，可有此事？」</p>
 <p>「陛下，臣妾冤枉，真是栖月自己不想侍寝。」虞惜芝辩解，火上浇油道，「陛下，若您不信，不妨试试。</p>
 <p>「她想不想让您碰，您试试便知了。</p>
 <p>「她只是身上起了红疹，可若真要侍寝，也不影响，这红疹又不会传染。」</p>
 <p>我冷眸凝向虞惜芝，真恶毒。</p>
 <p>江曜采纳了她的建议，他一把将我拽入怀里，就要来吻我。</p>
 <p>我不敢推开，被迫承受着。</p>
 <p>江曜感应到我的僵硬，低声命道：「回应朕。」</p>
 <p>我真不知道怎么回应，更何况，我并不喜欢他。</p>
 <p>江曜没有得到我的回应，彻底怒了。</p>
 <p>他将我摁在膝下，冷声命道：「跪下——」</p>
 <p>7</p>
 <p>我被迫跪下，江曜又挑起我的下巴：「告诉朕，你的心上人是谁？」</p>
 <p>我哪有什么心上人，都是姐姐瞎说罢了。</p>
 <p>我如实答：「回陛下，臣妾没有别的心上人。」</p>
 <p>江曜对我的话半信半疑：「你最好没骗朕，若不然朕会杀了你的心上人。」</p>
 <p>我态度诚恳：「臣妾不敢欺骗陛下。」</p>
 <p>身后传来虞惜芝的声音：「陛下，臣妾知道栖月的心上人是谁，是在春华街......」</p>
 <p>我眸光一寒，在预料到虞惜芝要说什么的时候，立刻打断她：「姐姐休要胡言乱语。」</p>
 <p>我知道姐姐要说的那个人，是在春华街开纸鸢铺的纸鸢匠人，沈清霁。</p>
 <p>沈家世代做纸鸢为生，到了沈清霁这一辈，他传承了这门手工艺活。</p>
 <p>沈清霁年纪轻轻，可做纸鸢的手艺已经超过了他爹爹。</p>
 <p>我从小喜欢放纸鸢，经常拿着零用钱去找沈清霁定制纸鸢。</p>
 <p>我是他小时候第一个找他定制纸鸢的客人。</p>
 <p>我的闺房里摆满了他制作的纸鸢，各式各样，应有尽有。</p>
 <p>虞惜芝曾打趣道：「妹妹，你这么爱纸鸢，干脆嫁给沈清霁算了。」</p>
 <p>我欣赏沈清霁的手艺，与他是朋友，并非她所说的心上人。</p>
 <p>若因为我给他带来杀身之祸，那我的罪过就大了。</p>
 <p>我不给虞惜芝说话的机会，立刻俯身对江曜说：「陛下，臣妾的心上人是您，前些日子不侍寝，是因为姐姐说要等她先怀上龙嗣，臣妾是妹妹，理应让着姐姐。」</p>
 <p>「你的心上人，是朕？」江曜眸光深深望着我。</p>
 <p>我口是心非颔首：「是，陛下是九五之尊，年轻又英俊，臣妾没有理由不喜欢陛下。」</p>
 <p>我的话取悦了江曜，他愉悦地笑出声来。</p>
 <p>随后话锋一转：「惜芝，你真是令朕很失望。」</p>
 <p>「陛下......」虞惜芝还要解释。</p>
 <p>江曜扬声命道：「来人，扶芝皇后回宫，从今日起，芝皇后禁足半月。」</p>
 <p>「什么......」虞惜芝愣在原地，还没来得及求情，就被皇帝的近身侍卫扶下去。</p>
 <p>8</p>
 <p>「起来，陪朕再下一局棋。」江曜指了指棋盘，语气较先前温和了许多，「这回拿出你的真材实料来，若是赢了朕，朕重重有赏。」</p>
 <p>「是。」我起身，继续和江曜下棋。</p>
 <p>这一次，我没有再让他。</p>
 <p>一局结束后，他输了。</p>
 <p>「栖月，你真是令朕刮目相看。」他看向我的目光多出几分赞赏之意，临走前对我说道，「你好好养伤，待身上红疹子好了，也该给朕侍寝了。」</p>
 <p>我压下心头的翻涌，颔首应道：「是，恭送陛下。」</p>
 <p>江曜走后，派人送了许多赏赐过来。</p>
 <p>我走到窗前，看着深深的宫墙，在心底轻叹。</p>
 <p>何时，我才能离开这座宫墙？</p>
 <p>我低头修剪着花瓶里的花枝，再抬头时，被苍穹里的一只凰鸟吸引住了目光。</p>
 <p>我连忙走出寝宫，来到后花园，仰头望着越来越近的凰鸟。</p>
 <p>片刻后，凰鸟坠落，掉进花园里。</p>
 <p>我俯身捡起来，原来这是一只凰鸟样式的纸鸢。</p>
 <p>能将纸鸢做得如此栩栩如生，我猜普天之下只有一人。</p>
 <p>那人便是沈清霁。</p>
 <p>我端详着手里的纸鸢，在纸鸢的背上看见绑着一卷纸条。</p>
 <p>我拿起纸条拆开来看，是沈清霁的字迹：【凰鸟栖明月，纸鸢寄相思。】</p>
 <p>栖月，是我的名字。</p>
 <p>沈清霁他这是什么意思？</p>
 <p>借纸鸢表达对我的思念之情？</p>
 <p>他不会是喜欢我吧？</p>
 <p>我惊出一身冷汗，幸好纸鸢被我捡到，若是被旁人捡到，送到虞惜芝或是江曜手里。</p>
 <p>将会为我和沈清霁惹来大祸。</p>
 <p>我连忙拿着纸鸢回屋，将纸条丢进火盆里烧掉。</p>
 <p>烧掉纸条后，我正想将凰鸟也烧掉。</p>
 <p>栩栩如生的凰鸟在烈火中化为灰烬，定会有一种凤凰浴火重生的美。</p>
 <p>我淡漠的眸子，忽然燃烧起光芒。</p>
 <p>那是对自由的向往。</p>
 <p>我改变了主意，这凰鸟纸鸢，先不烧了。</p>
 <p>我从箱子里拿出纸鸢绳，将凰鸟纸鸢重新绑起来。</p>
 <p>我走到院中，将凰鸟纸鸢朝姐姐的琼芝宫放去。</p>
 <p>纸鸢飘到琼芝宫，随后被我用剪刀剪断纸鸢线，坠落进琼芝宫。</p>
 <p>姐姐今夜，怕是睡不着了。</p>
 <p>9</p>
 <p>江曜每日派御医来给我请平安脉。</p>
 <p>他下令让御医必须在七日内，将我身上的红疹子治好。</p>
 <p>御医给我开了药膏。</p>
 <p>我身上的红疹子一日比一日淡。</p>
 <p>我知道，等红疹子全部褪去那日，便是我侍寝之时。</p>
 <p>第六日，我借着回娘家的由头，出宫去了春华街逛街。</p>
 <p>我来到纸鸢店，沈清霁正在后院制作纸鸢。</p>
 <p>他面前摆着两只纸鸢架，看形状，正是凰鸟的形状。</p>
 <p>他抬头看见我，英俊的面容闪过一抹喜色。</p>
 <p>他将手中的纸鸢放在一旁，起身道：「栖月，可是收到了我为你做的凰鸟？」</p>
 <p>我面色凝重道：「沈清霁，你这样做很冒险。昨晚那枚纸鸢若是被有心之人捡到，后果不堪设想。」</p>
 <p>「对不起。」沈清霁道歉，从怀里摸出一张皇宫手绘地图，摊开来给我看，「我知道你住在明月宫，我放纸鸢时，便是朝着明月宫而去，不会被旁人捡到。」</p>
 <p>「你真是大胆，快把皇宫地图收起来。」我胆战心惊，以前我怎么不知道沈清霁这么疯狂？</p>
 <p>在我的印象中，他历来循规蹈矩。</p>
 <p>怎么如今竟干些掉脑袋的事？</p>
 <p>沈清霁听话地将皇宫地图折起来，塞进怀里。</p>
 <p>「进来坐。」他请我移步茶室，用小茶炉烧水煮茶。</p>
 <p>院子外的树枝上有鸟儿在鸣叫，水开了，烟雾袅袅，茶香四溢。</p>
 <p>入目处，充满了人间烟火气。</p>
 <p>不像在皇宫里，处处透着森寒与落寞。</p>
 <p>收起思绪，我开门见山：「沈清霁，我姐姐前几日是不是来找你定制了两只凰鸟纸鸢？」</p>
