# 知乎上有没有双男主的文?

- 更新时间：2024年03月11日16时55分38秒

<body>
 <p>沈家小少爷把我掰弯后，我才得知这是他和兄弟无聊玩的「直男大冒险」游戏。</p>
 <p>【玩脱了。】</p>
 <p>【他真喜欢男人，恶心。】</p>
 <p>我识相滚蛋。</p>
 <p>撞见他死对头。</p>
 <p>男生高扬眉峰，懒懒地笑问我：</p>
 <p>「要不，你换个男人谈谈？」</p>
 <p>我谈了。</p>
 <p>不可一世的沈小少爷却发疯了。</p>
 <p>1</p>
 <p>沈简承千方百计把我掰弯。</p>
 <p>答应他告白后没多久，我收到匿名短信发来的群聊记录。</p>
 <p>【玩脱了。</p>
 <p>【他真喜欢男人，恶心。】</p>
 <p>我一路上划，弄清事情原委。</p>
 <p>原来沈简承会追我，只是出于和兄弟无聊玩的「直男大冒险」游戏。</p>
 <p>我是他选中的试验对象。</p>
 <p>一个他嘴里走路骚里骚气，最有男同潜质的班长。</p>
 <p>完美兼具拆盲盒的刺激感。</p>
 <p>所以他追我，再表白。</p>
 <p>2</p>
 <p>我找到沈简承的时候，他正微俯在台球面，高高的眉骨隆起，指骨紧贴球杆。</p>
 <p>胸口的十字架下坠晃动。</p>
 <p>好几个围观的女生拿手机偷拍，一脸激动和朋友分享。</p>
 <p>我走过去，把沈简承掀翻在地，对准他眼眶一拳。</p>
 <p>「沈简承，你就是个混蛋！」</p>
 <p>我完全被他耍了一通啊。</p>
 <p>血液升温直冲脑门。</p>
 <p>还想再抬起拳头，沈简承已经将我反压在地，顺势坐我腰腹上，长腿卡在我腰两侧。</p>
 <p>我动弹不得。</p>
 <p>「发什么疯？」</p>
 <p>他歪头睨我，语气不咸不淡。</p>
 <p>我怒视他，从牙关蹦出几个字：</p>
 <p>「直男大冒险。」</p>
 <p>「哦。」</p>
 <p>沈简承了然地点点头。</p>
 <p>「本来还想多玩几天，看看你们这类人谈恋爱是什么光景，不过你既然知道了也没什么意思了。」</p>
 <p>他从我身上离开，体温还残留在我身上，带出一种眷恋。</p>
 <p>可下一秒，沈简承长腿一抬，用力往我腰部踹上一脚。</p>
 <p>他练过搏击，这一脚，我感觉骨头都要散了。</p>
 <p>腰部肯定出现瘀青了。</p>
 <p>果然。</p>
 <p>沈简承看着我白皙的腰部出现他凌虐的痕迹，眼眸变深。</p>
 <p>却说：</p>
 <p>「两清。</p>
 <p>「滚吧。」</p>
 <p>3</p>
 <p>我不带一点犹豫就离开。</p>
 <p>台球室内却翻了天。</p>
 <p>沈简承很烦躁，折了那把价值六位数的专属球杆。</p>
 <p>兄弟陈泽靠过来。</p>
 <p>「哥，心情不爽？</p>
 <p>「到天上人间玩玩怎么样，听说新来的公主挺嫩。」</p>
 <p>沈简承咬着烟头，烟雾熏得那双狭长的眸更具压迫感。</p>
 <p>其实他真没多大兴致，但有件事的确需要他亲自确认。</p>
 <p>沈简承靠在沙发上，新来的公主一路揉着他筋脉勃发的小臂，然后到硬实的胸膛。</p>
 <p>红艳的唇快贴上来时——</p>
 <p>沈简承脑海突然出现一张脸。</p>
 <p>他妈的。</p>
 <p>又是他。</p>
 <p>最近他和女人亲热，总会时不时闪过李言澈的脸。</p>
 <p>男同这玩意也传染？</p>
 <p>他染上了？</p>
 <p>沈简承像触电般推开身上的人，离开时一路踹翻桌上的瓶瓶罐罐。</p>
 <p>他低咒：</p>
 <p>「草，他个李言澈。」</p>
 <p>酒水倒了一地。</p>
 <p>一片狼藉。</p>
 <p>屋内的人面面相觑，不知道小少爷又发的什么疯。</p>
 <p>4</p>
 <p>我在便利店买了一颗热腾腾的鸡蛋，坐在路边，龇牙咧嘴地揉瘀青。</p>
 <p>沈简承这狗崽子，下手是真狠。</p>
 <p>揉到后面，我把鸡蛋壳剥了，大口嚼在嘴里，眼睛有些冒汗。</p>
 <p>沈简承一定不知道，他追求我的那六个月对我来说意味着什么。</p>
 <p>从小因为性格文静，男同学总联合起来排挤我，他们说我长得娘。</p>
 <p>正因如此，我更不敢让人知道我的性取向。</p>
 <p>直到上大学遇见沈简承。</p>
 <p>他是沈家的小少爷，从小离经叛道，龟毛又挑剔，偏偏对我唯命是从。</p>
 <p>我大半夜牙痛，他能从被窝爬起顶着满头乱发陪我去医院。</p>
 <p>约会我想吃糖炒栗子，他会剥好一颗颗排在桌子上，告诉我按顺序吃就不会烫嘴。</p>
 <p>他以为掰弯我是他的功劳，但其实我一直就喜欢男人。</p>
 <p>何况是他温柔又热烈的攻势。</p>
 <p>沈简承和我表白。</p>
 <p>词很简单，但也是真的说进我心里去。</p>
 <p>「能不能给我一个站在你身边，陪你一起经受世俗眼光的机会？」</p>
 <p>我答应了。</p>
 <p>一时上头，还鼓起勇气主动凑上去亲他。</p>
 <p>当时沈简承往后退了好大一步。</p>
 <p>我还傻傻地以为他是惊到了。</p>
 <p>李言澈。</p>
 <p>你真就一傻蛋。</p>
 <p>5</p>
 <p>「咳咳。」</p>
 <p>一激动，蛋黄噎住喉咙，我难受得抡胸口。</p>
 <p>我现在模样肯定很像农村里正下蛋的鸡。</p>
 <p>脖子抻得老长。</p>
 <p>旁边递过来一瓶矿泉水。</p>
 <p>我想都没想拧盖，喝了一大半，舒服多了。</p>
 <p>「谢——」</p>
 <p>话说一半，看见一张打过几次照面的脸。</p>
 <p>沈简承的死对头，齐誉。</p>
 <p>我顿时一脸戒备，屁股一抬，挪出石凳几里地。</p>
 <p>齐誉，校游泳队的。</p>
 <p>利落的寸头，近一米九的身高，还有八块腹肌公狗腰，学校不少人私底下传他是男同天菜。</p>
 <p>但我本能对他有点排斥，因为这人流里流气，还调戏过我。</p>
 <p>6</p>
 <p>当时沈简承校运会被抽中男子三千米，跑完体力透支，喊我去接他。</p>
 <p>我冲好葡萄糖水过去。</p>
 <p>路上遇见一群体育生在训练，还都光着膀子。</p>
 <p>我红着脸加快脚步经过。</p>
 <p>手突然被齐誉攥住，他敛下眼皮看我，一颗小小的黑痣藏在薄薄的单眼皮线里，扯开嘴角。</p>
 <p>「去给沈简承送葡萄糖水？」</p>
 <p>沈简承说过他家和齐誉家是商场上多年的竞争对手。</p>
 <p>两人在学校一向不和。</p>
 <p>我自然对齐誉没什么好感。</p>
 <p>「关你什么事！」</p>
 <p>要走，没想到齐誉捞过我，公主抱那种，轻轻松松做了五十多个上下蹲。</p>
 <p>我大爷的，当时跟个朝天乌龟一样在他怀里挣扎。</p>
 <p>可齐誉发烫的肌肉紧紧箍在我胳膊上。</p>
 <p>眼看玩够了，他才把我放下，饶有兴致地扫了眼我手中的葡萄糖。</p>
 <p>「抱歉啊，体力太旺盛了，队内又要负重练习，没找到称手的东西。」</p>
 <p>所以我被他当成训练器材了？</p>
 <p>「有病！」</p>
 <p>我结结巴巴骂他。</p>
 <p>离开时他兄弟一直在后头起哄：</p>
 <p>「誉哥，看上人家了？」</p>
 <p>齐誉轻笑。</p>
 <p>「哪能啊，人是沈简承的小媳妇，没看见巴巴跑去送补给呢。」</p>
 <p>我现在回想当时的情景，谢谢，汗流浃背了。</p>
 <p>可齐誉厚脸皮地在我身边坐下。</p>
 <p>「我全看见了，你男朋友打你呢，沈简承那家伙多少有点不知道怜香惜玉了。」</p>
 <p>我把矿泉水瓶捏得咔咔响——</p>
 <p>为了增加点威慑力。</p>
 <p>「你就这么闲吗，专门跟出来看我笑话？」</p>
 <p>「谁说我看笑话？」</p>
 <p>「那你来干吗？」</p>
 <p>「来撬墙脚的。」</p>
 <p>我准备好的一顿输出全卡在喉咙里，瞪大眼睛看他。</p>
 <p>齐誉高扬眉峰，懒懒地笑问我：</p>
 <p>「要不，咱换个男人谈谈？」</p>
 <p>7</p>
 <p>我吞了口唾沫。</p>
 <p>「谁……」</p>
 <p>齐誉注视我，嘴角一直带着抹邪气的笑，一切尽在不言中。</p>
 <p>我迟缓地反应过来，这街溜子又在调戏我！</p>
 <p>淦！</p>
 <p>我从凳子上跳起来，指着齐誉一顿输出：</p>
 <p>「你费尽心思撩拨我不就是为了让沈简承不爽吗？</p>
 <p>「没用，沈简承铁直，根本对我没有半点喜欢，他和我在一起全是他和兄弟一场玩闹！」</p>
 <p>我破罐子破摔，也不管自己的音量吸引到路人。</p>
 <p>「是，我是喜欢男人，但就要被你们两个人这样耍吗？」</p>
 <p>我把对沈简承的怨气全发泄在齐誉身上。</p>
 <p>齐誉似是没想到我会爆发，神色收敛正经，迟疑出声：</p>
 <p>「你想……」</p>
 <p>「有意思吗？」</p>
 <p>我不等他说完就反问，眼眶不争气地微热。</p>
 <p>齐誉半张的嘴又合上。</p>
 <p>半晌，他无奈一笑。</p>
 <p>「林言澈，你还真是……」</p>
 <p>真是什么，齐誉没说。</p>
 <p>害我一路回宿舍都在纠结，脚步踏上台阶，声控灯亮起。</p>
 <p>倚靠在墙角的人在灯光下被照亮，吓我一跳。</p>
 <p>沈简承。</p>
 <p>他来干什么？</p>
 <p>看到他就腰痛。</p>
 <p>我目不斜视从他身边走过，闻到淡淡的酒精味。</p>
 <p>还没走出几步，人就被按在墙上。</p>
 <p>沈简承双手撑在我耳旁两侧，眼尾勾着红。</p>
 <p>「李言澈，你说，我是不是被你传染了？要不然，我怎么总是……总是想着你？」</p>
 <p>我阻止他靠得我更近，偏过头。</p>
 <p>「我不懂你在说什么。」</p>
 <p>「呵。」</p>
 <p>沈简承手摸到我腰间的瘀青，轻轻揉起来。</p>
 <p>「还在生我气？</p>
 <p>「我跟你道歉，不应该打你。你也知道从小到大除了我爸妈，还没人敢动我。」</p>
 <p>他把头深深埋在我颈侧。</p>
 <p>「李言澈，我们和好。」</p>
 <p>8</p>
 <p>我用力踹在沈简承膝盖，报复他那一脚。</p>
 <p>他吃痛松开我。</p>
 <p>我冷声说：</p>
 <p>「沈简承，从我看见那些聊天记录起，我们就没可能了。</p>
 <p>「我不是在和你小打小闹，我们分手。」</p>
 <p>声控灯超过续航时间，灭了，楼道陷入黑暗。</p>
 <p>视觉消失，听觉就越敏感。</p>
 <p>沈简承默声。</p>
 <p>但我能从他不稳的呼吸频率里，知道他处在爆发的边缘。</p>
 <p>「如果你没什么事，我先走了。」</p>
 <p>踏上台阶，后衣领被人攥住。</p>
 <p>沈简承几乎是运着气，大手掌着我后脑勺，将我脸摁贴在墙面。</p>
 <p>「别跟我横。」</p>
 <p>我挣扎。</p>
 <p>「你有病啊沈简承！放开我。」</p>
 <p>他凑上来，扳过我脸狠吻下来。</p>
 <p>我咬破他嘴角。</p>
 <p>得了自由，扶着墙角干呕。</p>
 <p>「李言澈，我让你恶心？」</p>
 <p>黑暗中，沈简承眼睛闪着火星。</p>
