# 《三体》中有哪些细思恐极的细节？
- 点赞数：5151
- 更新时间：2016年01月12日23时03分09秒
- 回答url：https://www.zhihu.com/question/22855640/answer/81262887
<body>
 <p data-pid="SVbRKiqo">我来说两个细节吧，谈不上极其恐怖。《三体》我最喜欢的两个男人，常伟思和章北海。</p>
 <p data-pid="7U0oZmAb"><b>一、</b>常伟思对即将冬眠的章北海说：<b>“此生能相聚已经很幸运了,代我向未来的同志问好。”</b></p>
 <ul>
  <li data-pid="d92IMf0G">刚看到这句话的时候，非常感动。可是看到后来才发觉，常伟思一直都不信任章北海，怀疑他的真实动机。并且留给了未来一封信，隐晦地提醒太空军同志们要提防章北海是否是逃亡主义者。<br></li>
  <li data-pid="SUQhNc72">那么问题是：<b>常伟思本人是秉持着什么主义的呢？</b><br></li>
  <li data-pid="PppnT4Tu">首先可以确定，<b>常伟思绝不是一个坚定的胜利主义者。</b>这里有两方面可以佐证：</li>
 </ul>
 <ol>
  <li data-pid="NnbUgO7v">如果他是个坚定的胜利主义者，他完全可以说服自己接受「章北海是胜利主义者」这个事实。<br></li>
  <li data-pid="nE7PBJib">如果他是个坚定的胜利主义者，当他对章北海有所怀疑的时候，他完全可以禁止章北海加入「支援未来计划」。</li>
 </ol>
 <ul>
  <li data-pid="1ACi5lSf">其次可以确定，<b>常伟思绝不是一个逃亡主义者。</b>为什么呐？</li>
 </ul>
 <ol>
  <li data-pid="HiUtrFGQ">如果他是个坚定的逃亡主义者，并且认为章北海是逃亡主义者。绝不会留给未来的同志一封要提防章北海的信。因为章北海是被太空军绝对信任的政委。没有必要画蛇添足给章北海制造麻烦。<br></li>
  <li data-pid="JitiTB-I">如果她是个坚定的逃亡主义者，但是认为章北海是个坚定的胜利主义者，会对未来人类逃亡做出干扰，所以要对章北海进行反干扰的话，他完全可以禁止章北海参与「支援未来计划」。</li>
 </ol>
 <p data-pid="Kd7Pi_jM">那么<b>常伟思本人究竟是什么主义呢？</b></p>
 <ul>
  <li data-pid="PdFXJ5C9">我给出的结论是，他在人类未来的出路上，没有什么主义。他只是一个军人，一个纯粹的军人，勤勤恳恳的尽自己的义务，兢兢业业的缔造太空军，完成自己作为太空军司令的责任。他对人类的未来没有一个明确的构思，但是他深知无论是战斗还是逃亡，都需要一支太空舰队，他把选择权留给了未来的同志，留给了章北海。他让章北海向未来的同志问好，他对未来是有着殷切希望的。这样的人物设定让我极其感动。</li>
  <li data-pid="Q4eOEQuQ">但是他身为一个纯粹的军人，又不能无视军中可能出现一个「叛徒」，但是他又无法肯定这个叛徒对于人类究竟是希望，还是毒草。于是到最后，他选择了折衷。留下一封信，但是说的很隐晦。他尽了自己的义务，一切都留给未来的同志。</li>
 </ul>
 <p data-pid="obvhtOkC"><b>二、</b>章北海与东方延绪的对话：</p>
 <blockquote data-pid="XDOjQSzi">
  “东方。”章北海轻轻地唤了一声。
  <br>
  “嗯？”<b>美丽的舰长</b>转过身来，她的双眸中映着银河系的星光。
  <br><b>“如果有一天我不得不杀了你，请原谅。”</b>章北海轻声说。
  <br>
  东方延绪对这话付之一笑，“你看我像钢印族吗？”
  <br>
  章北海看看她，在从五个天文单位外照来的阳光中，她像是一根飘浮在星海背景上的轻盈的羽毛。
  <br><b>“我们属于大地和海洋，你们属于星空。”</b>
  <br>
  “这样不好吗？”
  <br><b>“不，这样很好。”</b>
 </blockquote>
 <p data-pid="VMoYkbZJ">“<b>如果有一天我不得不杀了你，请原谅。</b>”这句话极其值得玩味。</p>
 <ol>
  <li data-pid="_i1Ko5OZ">章北海这种性格的人，没有任何理由，任何必要对任何一个人说出这种话，这种话只能引起别人对他的反感，或者怀疑。只有一种解释：<b>他对东方延绪产生了男女间的情愫</b>。据我判断，是男女之情，而非父女之情。如果你们仔细看一下他们初见那一段文字描写，我相信你们会得出和我一致的结论。</li>
  <li data-pid="IXa9WTbo">他是其实是在提醒自己不要陷入这种情愫中，影响自己的判断。</li>
 </ol>
 <ul>
  <li data-pid="8DSIWj77">在水滴暴露了三体的真实面目后，章北海的先知得到了太空军五舰的认同，崇拜。然后章北海早在决定逃亡的时候，就知道了自己所在的舰船没有任何办法坐长途太空旅行。他没有做出任何添加燃料的尝试，因为这太冒险了，所以，他只能寄希望于追兵。</li>
  <li data-pid="FEVSeu9z">但是既然算到了这一步，<b>为什么不直接在控制权交出之前，就将其他四舰歼灭，完成燃料收集呢？这样自己也能存活下来！</b></li>
 </ul>
 <ol>
  <li data-pid="86d_66kr">章北海做出的选择已经够多了，<b>他将人类的希望带到了太空，他的使命已经完成了</b>。</li>
  <li data-pid="bOVdtqpC">人类以后的命运，不是他能够决定的，而是<b>要依靠新人类自己去探索，他不能越疱代俎</b>。</li>
  <li data-pid="dbidJmli">究竟是<b>哪一艘舰能够最先出手，那么哪一艘舰就最有资格，也最有希望在这个未知的未来中，代表人类继续前行</b>，而其他的都是可以牺牲的，包括自己所在的「自然选择」号。</li>
  <li data-pid="9Yyp1WMA">章北海选择离开东方延绪独处，就是为了<b>让各舰船处于公平的环境里竞争，让各自做出选择</b>。</li>
  <li data-pid="0viCYNdz">章北海终究还是没忍住，帮了东方延绪走出第一步，，然后东方延绪没有这个狠心，<b>她的惊讶迟疑让她丧失了代表人类继续前行的机会</b>。</li>
  <li data-pid="QH_frSQ4">无论「自然选择」号是否能成为「跑得最快的精子」，<b>章北海一定会死</b>。要么被其他四艘舰炸死，要么就是「自然选择」继续前行，章北海选择自尽。</li>
  <li data-pid="c0VYHvZC">章北海自尽的方式极有可能是「<b>水手的方式</b>」，从太空舰的舢板上跌入汪洋般的真空中。</li>
  <li data-pid="HmW_gZTB">经历这次爆炸后，幸存者完成了对「人类」的告别，进化成「太空人」，这是<b>用他人生命完成的对自身的洗礼，是进化的代价。</b></li>
  <li data-pid="I34rEnOB">这种自相残杀不涉及道德，上帝死了，旧道德崩溃了。地球死了，新道德也失去了其自适的土壤。<b>太空人需要重建自己的道德体系。而这第一课就是与过去的道德告别。</b></li>
  <li data-pid="c3pBJWnD">这种残杀<b>只涉及生存，作为人类整体的延续</b>。</li>
  <li data-pid="ygTNu-jE"><b>新人类不需要先知，章北海的存在对于人类未来是一个妨碍</b>。</li>
  <li data-pid="z1L4cVXp"><b>文明的第一需求是生存。</b></li>
  <li data-pid="zIiL-dyg"><b>给岁月以文明，而不是给文明以岁月</b></li>
 </ol>
