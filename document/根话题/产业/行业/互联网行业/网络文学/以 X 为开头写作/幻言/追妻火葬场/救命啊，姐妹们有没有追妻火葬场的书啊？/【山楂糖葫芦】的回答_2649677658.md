# 救命啊，姐妹们有没有追妻火葬场的书啊？
- 点赞数：13416
- 更新时间：2024年05月13日20时42分27秒
- 回答url：https://www.zhihu.com/question/525548504/answer/2649677658
<body>
 <p>「你到底喜欢我什么？我改还不行么？」</p>
 <p>这是我暗恋的男神对我说的第一句话。</p>
 <p>再见面，他真的把眼角那颗，我喜欢的痣点掉了。</p>
 <p>后来，我找了个同样眼角有痣的小奶狗。</p>
 <p>他却疯了，红着眼求我：「喜欢我就不能再喜欢别人了，你知不知道？」</p>
 <p>01</p>
 <p>褚辰哪都好，就是命不好。</p>
 <p>大一刚报到就让我给盯上了。</p>
 <p>腰窄腿长肩宽，目测一米八，是人群中亮眼的高度。</p>
 <p>微风拂过，白 T 恤紧贴身前。</p>
 <p>隐隐约约能看到肌肉的沟壑。</p>
 <p>戴着帽子口罩，帅哥气质还是扑面而来。</p>
 <p>只见他走进校门，站住，拿出手机左转转，右转转。</p>
 <p>三分钟后，傻乎乎地直奔我们迎新处走来。</p>
 <p>我忍不住笑出声。</p>
 <p>男孩子路痴也这么可爱的吗？</p>
 <p>人还没到，迎新处几个老油条，已经蠢蠢欲动。</p>
 <p>室友撞一下我的腰，使眼色：「哎哎，脆弱纤瘦系！你的菜！」</p>
 <p>其他人跟着起哄，闹得我有点不好意思。</p>
 <p>「姐姐好，我叫褚辰，请问是在这报到吗？」</p>
 <p>「姐姐」两个字他叫得很轻，扭捏的语调直接戳我心巴上。</p>
 <p>「嗯，这边登记。」</p>
 <p>我抬头撞进他勾人的眼睛里。</p>
 <p>他朝我笑笑，眉眼弯弯，从我手上拿过笔，弯腰签字。</p>
 <p>我的眼睛不受控制地从他领口，锁骨……一路看到没有赘肉的小腹。</p>
 <p>直到他骨节分明的大手在我面前晃晃。</p>
 <p>褚辰一脸无辜道：「姐姐？看什么呢？」</p>
 <p>「没……没什么。」</p>
 <p>被抓包，我低头假装收拾文件。</p>
 <p>不知道是不是错觉，我好像听见一声嗤笑。</p>
 <p>很轻。</p>
 <p>室友贱兮兮凑到我耳边，用是个人就能听到的声音说：「你看他眼角还有一颗痣！」</p>
 <p>「你不是说对这样的男生没有抵抗力么？快上啊！」</p>
 <p>我赶紧捂住她的大嘴巴，递给褚辰一张学校的地图。</p>
 <p>褚辰接过地图，手指不经意地轻点眼角的痣。</p>
 <p>救命，他果然听到了！</p>
 <p>02</p>
 <p>当褚辰拉着行李箱，第 3 次出现在新生报到处时。</p>
 <p>报到处的姐妹们都笑疯了。</p>
 <p>褚辰不好意思地挠挠头，看着我问：「姐姐，你能带我去宿舍吗？」</p>
 <p>我本来是想矜持一些的，结果室友直接把椅子抽走，给我往前一推。</p>
 <p>我踉跄着撞进褚辰怀里。</p>
 <p>嗯，有点矜持，但不多。</p>
 <p>褚辰反应很快，伸手垫住我的头。</p>
 <p>不知道是怕我磕疼了，还是在维护自己的清白。</p>
 <p>褚辰的宿舍在新校区，我没去过，不熟悉路。</p>
 <p>更要命的是，我也路痴。</p>
 <p>在我第二次带他经过同一座孔子雕像的时候，褚辰愣了愣。</p>
 <p>「姐姐，你看这个雕像眼不眼熟？」</p>
 <p>「哈哈哈！是不是跟我们刚刚路过的那个很像？」</p>
 <p>我企图蒙混过关，随口胡诌道：「咱们学校有好几座一模一样的雕像，时间久了，你就知道了。」</p>
 <p>我偷偷背过身，点开百度地图。</p>
 <p>「唉，还以为姐姐是想多跟我待一会呢？」</p>
 <p>「嗯？你说什么？」</p>
 <p>褚辰摇摇头。</p>
 <p>「没什么，我说我对学校不太熟悉，我收拾完，你能带我逛一下校园吗？」</p>
 <p>可怜巴巴的恳求，配上眼角那颗泪痣。</p>
 <p>唐僧来了也说不出拒绝的话吧？</p>
 <p>我鬼迷心窍地回了个：「好。」</p>
 <p>「那加个微信吧，收拾好了我找你。」</p>
 <p>褚辰大方亮出二维码。</p>
 <p>我扫上，申请信息：陈知蕴。</p>
 <p>「陈知蕴？」</p>
 <p>「嗯？」</p>
 <p>褚辰笑笑，那双漂亮的眼睛盯着我说：「真美啊！」</p>
 <p>阳光透过树叶的缝隙，落在他身上。</p>
 <p>第一次听到这么真心的赞美，我的心被狠狠戳中。</p>
 <p>然后我听到褚辰用好听的声音大喘气道：「你的名字。」</p>
 <p>03</p>
 <p>淦！</p>
 <p>我合理怀疑褚辰跟山东有一腿。</p>
 <p>不然倒装怎么用得这么 6？</p>
 <p>在这里真诚感谢百度、高德还有谷歌地图。</p>
 <p>让我终于在太阳落山前，把褚辰送到了寝室。</p>
 <p>刚往回走，褚辰就下来叫住我。</p>
 <p>「姐姐，你还是先带我熟悉校园环境吧？」</p>
 <p>「嗯？你不收拾东西了？」</p>
 <p>他挠挠头，小声嘟囔：「收拾，但——姐姐更重要……」</p>
 <p>说完，他不好意思地往前走。</p>
 <p>啧，现在的弟弟真是了不得。</p>
 <p>带着褚辰逛了一圈校园，我们边走边聊天。</p>
 <p>才知道他老家跟我是一个地方的。</p>
 <p>而且我们是同一个初中高中，只不过他一直低我一届。</p>
 <p>大学又是跟我一个系的直系师弟。</p>
 <p>「这也太巧了吧？」</p>
 <p>我不由惊呼。</p>
 <p>褚辰低头轻笑，声音里是如释重负的轻巧：「是啊，好巧。」</p>
 <p>我踮脚拍拍他的肩膀。</p>
 <p>「那你不用买教材了，我的借你，上面还有笔记和重点哦~」</p>
 <p>「恭敬不如从命，那我请姐姐吃饭吧！」</p>
 <p>我一抬头才发现，不知不觉已经逛到学校的「情侣餐厅」。</p>
 <p>顾名思义，专门坑小情侣的。</p>
 <p>装潢好看，菜价更好看。</p>
 <p>我拉拉褚辰的衣角，在他耳边小声提醒：「这个店可黑啦，我们不要做冤大头……」</p>
 <p>褚辰喉结滚动，一手拉开门，一手环抱着我的肩膀，把我搂进餐厅。</p>
 <p>我傻眼了。</p>
 <p>小声嘟囔：「你干吗呀，我不是都说这家店很贵了吗？」</p>
 <p>他弯下腰，学着我小声说：「又不是天天都吃，再说了，我觉得姐姐比这家店贵多了……」</p>
 <p>04</p>
 <p>温热的呼吸落在我的脸上，好热。</p>
 <p>我拉开距离，深呼吸，思考去医院开点救心丸的可行性。</p>
 <p>点菜时，刚好遇到商学院的江卿尘，和他女朋友吃完饭结账。</p>
 <p>江卿尘看见我，愣了一瞬，随即点头笑笑，算是打招呼。</p>
 <p>可我整个人却局促得呼吸都有些不顺畅。</p>
 <p>我「暗恋」江卿尘很多年，从初中开始。</p>
 <p>你有没有在年少时喜欢过一个人？</p>
 <p>然后为了他，努力变优秀。</p>
 <p>江卿尘从小就长得好看，常年年级大榜第一。</p>
 <p>我为了更靠近他一点点，做过很多傻事。</p>
 <p>每天去操场跑圈，拼命减肥变瘦，以为瘦下来就会好看。</p>
 <p>熬夜学习，早起背错题，成绩每爬升一名，我都觉得离他更近一些。</p>
 <p>还偷偷打听他喜欢的学校、专业，把他的梦想也当成自己的目标……</p>
 <p>我想那么优秀的少年，暗恋他的女生一定很多。</p>
 <p>但只有我最轰动，也最无耻。</p>
 <p>我用 QQ 小号写日志，里面密密麻麻都是一个名字：江卿尘。</p>
 <p>日志被扒，所有人都觉得我癞蛤蟆想吃天鹅肉。</p>
 <p>我还没有跟江卿尘说过一句话。</p>
 <p>可我连告白的机会都没有了。</p>
 <p>我的暗恋日志闹得很大，江卿尘还来主动找我。</p>
 <p>他对我说的第一句话是：「你喜欢我什么？我改还不行么？」</p>
 <p>后来他真的把我喜欢的那颗，落在眼角的痣点掉了。</p>
 <p>江卿尘上大学后，女友不断。</p>
 <p>这次这个背影都透露着精致的漂亮。</p>
 <p>真般配啊，可我的心里怎么这么酸呢？</p>
 <p>「姐姐？」</p>
 <p>突然褚辰那张好看的脸出现在我面前。</p>
 <p>他双手捧着我的脸，有些赌气：「跟我在一起的时候，不许想别的男人。」</p>
 <p>05</p>
 <p>我轻敲褚辰的脑袋。</p>
 <p>「小屁孩，别拿你对付小女生那套……唔」</p>
 <p>话没说完，嘴被剥好的虾堵住。</p>
 <p>褚辰自然地吮吸指尖上的汤汁，笑得阳光无邪：「甜的！」</p>
 <p>心漏跳一拍。</p>
 <p>对付小女生那套确实很好用……</p>
 <p>教训的话再也说不出口，我红着脸低头吃饭。</p>
 <p>「姐姐。」</p>
 <p>「嗯？」</p>
 <p>「没有别的小女生。」</p>
 <p>这……是在跟我解释自己没谈过恋爱吗？</p>
 <p>应该是我想多了吧，算了算了，闷头干饭要紧。</p>
 <p>奇怪，今天这个食堂的菜，还挺好吃的。</p>
 <p>都是我喜欢的菜。</p>
 <p>吃完饭，褚辰提议去刚刚路过的小树林散散步。</p>
 <p>检测他的认路能力。</p>
 <p>能力还没检测出来，蚊子倒是给我叮了个大包。</p>
 <p>见我挠了半天，褚辰扯过我的胳膊。</p>
 <p>认真地在上面画了个十字：「这样就不痒了。」</p>
 <p>「哼，被蚊子叮的又不是你。」</p>
 <p>褚辰突然把手臂伸进灌木丛，我伸手过去拉。</p>
 <p>「你干吗呀？」</p>
 <p>他把胳膊伸到我面前的时候，上面赫然鼓着三个大包。</p>
 <p>「姐姐，痒。」</p>
 <p>我白了他一眼，用力按了三个十字上去。</p>
 <p>褚辰看着惨不忍睹的胳膊，笑得像个二傻子。</p>
 <p>小树林越往里走，卿卿我我的小情侣越多。</p>
 <p>不时能听到奇怪的声音。</p>
 <p>气氛变得微妙起来。</p>
 <p>最后一段石板路设计得很不合理，一块一块走显得做作，两块一起跨又卡裆……</p>
 <p>「你小心点啊，这里容易绊倒。」</p>
 <p>我刚提醒，就听见褚辰在身后焦急了喊了声：「姐姐！」</p>
 <p>转身回头的瞬间被他扑倒。</p>
 <p>他的大手护在我的脑后，我们亲在了一起。</p>
 <p>！</p>
 <p>那可是我的初吻啊！</p>
 <p>06</p>
 <p>更尴尬的是，我们挡住了后面同学的路。</p>
 <p>慌忙起身才发现是江卿尘和他女朋友。</p>
 <p>江卿尘没有着急离开，皮笑肉不笑地打量我一番，最后目光落在我的唇上。</p>
 <p>他嗤笑一声，搂住身边人，拥吻下去。</p>
 <p>不忍再看，我拉着褚辰离开。</p>
 <p>我喜欢江卿尘这么多年，已经变成了我心里的一颗毒刺。</p>
 <p>没有资格的伤心最残忍。</p>
 <p>我讨厌爱情。</p>
 <p>它不像去自习室占座位，只要我起得够早，去排队，就一定有我的位置。</p>
 <p>明明是我先来的，我先喜欢的，可也是我最先出局的。</p>
 <p>年少的喜欢融入骨血，以至于我现在还学不会告别。</p>
 <p>褚辰没说话，一路送我到宿舍楼下。</p>
 <p>在我要上楼的时候，他一把拽住了我。</p>
 <p>褚辰摸了摸自己的嘴唇，委委屈屈道：「姐姐，我的初吻……」</p>
 <p>宿舍门口人多眼杂，我赶紧捂住他的嘴，小声说：「我也是，我们这算扯平了！」</p>
 <p>说完，我一路小跑回寝室。</p>
 <p>手机振动，褚辰发来消息：「姐姐，不要扯平好不好？」</p>
 <p>这就是小奶狗吗？比我想象中的黏人。</p>
 <p>纠结怎么回复，室友一脸贱兮兮地凑过来：「啧啧啧，拿下了？」</p>
 <p>「拿下了！我刚才还看见他们俩在宿舍楼底下搂搂抱抱！」</p>
 <p>停停停，我赶紧阻止室友们的臆想。</p>
 <p>再聊下去，估计孩子都有了。</p>
 <p>还好下铺进屋的时候，跟我说楼下有人找。</p>
 <p>下楼发现居然是江卿尘。</p>
 <p>他递给我一个袋子，嘴角有血，语气不是很好：「你们班的作业。」</p>
 <p>大课我们班是跟江卿尘他们班一起上的。</p>
 <p>但是也不用特意过来送作业吧……</p>
 <p>「你的脸？」</p>
 <p>江卿尘侧过身子，不想让我看见。</p>
 <p>「没事。」</p>
 <p>我接过作业：「辛苦你这么晚还跑一趟，我就先回去了……」</p>
 <p>「等一下！」</p>
 <p>江卿尘塞给我两张票，草莓音乐节的。</p>
 <p>「买多了，送你两张，你……可以带你男朋友一起去。」</p>
