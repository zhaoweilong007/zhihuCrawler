# 有没有好看的追妻火葬场文？

<body>
 <p>我标记了一个笨蛋 omega。</p>
 <p>为了圆他的北大梦，我放弃兼职赚钱，吃糠咽菜地给他补课。</p>
 <p>某天却在学校的保研名单上看到了他。</p>
 <p>那天，他和情人笑着说：「她真够蠢的，一次标记能让她为我付出这么多。」</p>
 <p>我才知道，他是夜店有名的浪荡子，后颈的腺体不知道被多少人咬过。</p>
 <p>于是，我不要他了：「你太脏了。」</p>
 <p>后来听闻，浪荡子硬生生承受剥皮之疼洗干净了腺体，只为从良。</p>
 <p>1</p>
 <p>「姐姐，你标记我吧。」</p>
 <p>巷子里的少年精致白皙得发光，眉眼间却露出一抹破碎的红意。</p>
 <p>我安抚着他的后颈：「忍一下，我们先回家，回去就给你。」</p>
 <p>我舍不得在这个昏暗的小巷里标记他，这里配不上他。</p>
 <p>第一次遇见安颂是在图书馆。</p>
 <p>他正处于发情期，一只手死死捂住后颈的抑制贴，倚靠在墙上细密地颤抖，见我走来，湿漉漉的眸光一亮。</p>
 <p>「别走，求你给我一个标记。」</p>
 <p>我顿了顿，二话没说，打了 120 送他到医院。</p>
 <p>在我看来，他那副模样与耍流氓无异。</p>
 <p>我是一个对待感情很专一的人，做不到随意给陌生人标记，临时标记也不行。</p>
 <p>那天过后，他就缠上我了。</p>
 <p>奶茶店里，他笑容很甜地来找我赔罪。</p>
 <p>「姐姐，谢谢你那次送我去医院。」</p>
 <p>我没说话，他好像又想起什么，抿起唇，脸颊染上绯红：「上次……我不是在故意耍流氓。」</p>
 <p>我淡淡点头表示理解。</p>
 <p>他却突然冲进我怀里，仰起头小心翼翼地看我道：「姐姐，我喜欢你。」</p>
 <p>我慌乱下推开了他。</p>
 <p>直到我的易感期提前到来。</p>
 <p>2</p>
 <p>我在 KTV 兼职，安颂一如往常来找我。</p>
 <p>我躲进员工休息室里，他却步步紧逼。</p>
 <p>眼神执拗地看着我，如同献祭的羔羊般，他摘下后颈的抑制贴，在我面前低下头。</p>
 <p>之后，我便对他担负起了责任。</p>
 <p>我是一个孤儿，除了安颂外没有得到过其他人的爱意，我不懂该怎么爱人。</p>
 <p>只能在我能力范围内最大程度地满足他的需求。</p>
 <p>他很会撒娇，总是抱着我软软道：</p>
 <p>「姐姐，我喜欢这个球鞋。」</p>
 <p>「姐姐，我也想要最新款的手机。」</p>
 <p>「姐姐，给我买台游戏机好不好？」</p>
 <p>我没怎么犹豫，便给他买了。</p>
 <p>在我看来，他值得。</p>
 <p>3</p>
 <p>我拼了命地赚钱，每天晚上几乎都是凌晨才能回家。</p>
 <p>二百多的蛋糕，三千的球鞋，五千元的手机……</p>
 <p>他想要的东西很贵，我要一刻不停地努力。</p>
 <p>临近考研，他突然说想和我一起去北大。</p>
 <p>我既高兴又担忧。</p>
 <p>时间太短了，他没有任何准备，问我的题目更是教了一遍又一遍还是不会。</p>
 <p>研究生考试需要学习的东西太多，我为了给他复习，辞去了白天的兼职。</p>
 <p>上晚班赚的那些钱既要交房租，还要保证安颂的生活品质。</p>
 <p>我能省则省，吃糠咽菜的时候，安颂总是满脸心疼地抱住我。</p>
 <p>「姐姐，你以后别给我买好吃的了，那些都很贵。」</p>
 <p>我只感到失败，让一个 omega 跟在我身边受苦。</p>
 <p>「没有很贵，你太瘦了，要多吃点好的，我一个 alpha 吃什么都行。」</p>
 <p>傍晚出门时，我给他勾了些题目，让他自己做完去睡觉。</p>
 <p>没想到他今天发情期又到了，想到这我有些生气。</p>
 <p>「你这个状态不在家待着，跑出来做什么？</p>
 <p>「还有身上的柠檬味哪来的？接触 alpha 了？」</p>
 <p>我皱着眉闻着他身上的味道，平日温柔的茉莉奶香味中混杂了刺鼻的柠檬味，让人闻着不太舒服。</p>
 <p>他讨好地亲我：「刚才点了杯柠檬水不小心洒在身上了，才没有接触 alpha，我那么乖姐姐还不相信我吗？」</p>
 <p>说着他撒娇似的捏住我的手指摇晃：「姐姐以后多陪陪我嘛，我不想自己一个人在家。」</p>
 <p>我松了口气，揉了揉他的头发：「姐姐周末抽空陪你玩。」</p>
 <p>他惯会懂得怎么让我心软。</p>
 <p>我拿他没法，见他难受得厉害，加快脚步将他带回家。</p>
 <p>4</p>
 <p>我请了一天假，陪安颂到游乐园玩。</p>
 <p>他牵着我的手，笑得天真可爱：「姐姐，陪我坐过山车嘛。」</p>
 <p>我恐高，犹豫片刻，不想扫他的兴，还是答应了。</p>
 <p>一圈下来，他兴致正高，又拉着我去坐海盗船。</p>
 <p>我站在原地，胃里翻腾：「我在这等你，你自己去玩好不好？」</p>
 <p>安颂眨了眨眼，嘟着嘴道：「不好。」</p>
 <p>他背过身生气：「姐姐是不是不爱我了？」</p>
 <p>我一阵头疼，哄着他：「爱你，永远都爱你。」</p>
 <p>他这才转过身，眼里亮晶晶的：「那姐姐去坐跳楼机，我就相信。」</p>
 <p>我发觉他此时的恶趣味，心里有些难过。</p>
 <p>见我许久不说话，他好像知道错了，抿着嘴巴给我道歉。</p>
 <p>「姐姐对不起，我太自私了。</p>
 <p>「那我们不玩跳楼机了，我们去坐摩天轮吧。」</p>
 <p>说着，他牵着我的手朝另一个方向走，但他的眼睛还在跳楼机上留恋，一副很想玩的样子。</p>
 <p>我叹了口气，想着还是要满足自己的 omega 的。</p>
 <p>「走吧，我陪你去玩。」</p>
 <p>他瞪大眼睛，高兴地点头：「好！」</p>
 <p>一趟下来，我吐得昏天暗地，他依旧兴致勃勃。</p>
 <p>休息时面前突然来了三个 alpha。</p>
 <p>5</p>
 <p>身高挺拔，面容俊美，叼着烟，一副浑不吝的样子。</p>
 <p>他们的目标不是我，而是安颂。</p>
 <p>其中一个将手搭在安颂的肩上：「怪不得最近颂颂不陪我们玩了，原来是找到新伴了。」</p>
 <p>这都是安颂的朋友？还是三个 alpha？</p>
 <p>我疑惑向安颂看去，却看到他正低着头，脸颊上浮出一抹明显的红晕。</p>
 <p>我觉察到不对，抬眸看去，那个男人的手正按在安颂后颈的腺体上！</p>
 <p>这个动作对于 omega 和性骚扰无异。</p>
 <p>一把火烧到心头，我钳住那人的手甩开，挥舞着拳头砸上那人的脸，一脚踹翻他。</p>
 <p>这时安颂突然冲上来拽住我：「姐姐别打了，我们是朋友，刚才是开玩笑的。」</p>
 <p>「什么朋友会做那种动作？」</p>
 <p>我脸色难看地质问。</p>
 <p>6</p>
 <p>不待我说完，他眨着眼，啪嗒啪嗒地落泪。</p>
 <p>那人站起来，挑衅般释放信息素，打量我后：「你是他新相好？怎么一副穷酸样，他怎么看上你的？」</p>
 <p>我身上的衣服鞋子加起来没有超过五十元，而面前的男人满身名牌。</p>
 <p>我掐住指尖，不动声色地问：「你和安颂是什么关系？」</p>
 <p>他一脸坏笑地凑近：「不如你猜猜我俩是什么关系。」</p>
 <p>「够了，有什么事冲我来，离姐姐远点！」安颂突然跑到我面前，倔强地瞪着面前的 alpha。</p>
 <p>我的心一下子就软了起来，揽过 omega 后肆意地释放属于强 A 的信息素。</p>
 <p>烈酒味道的信息素瞬间压得几人起不来身。</p>
 <p>强 alpha 天性中的暴虐感在这一刻显露无遗，透过骨子里的威压让几个人身体不断颤抖。</p>
 <p>「这是给你们一个教训，以后少来招惹他。」</p>
 <p>说完，我收回信息素，淡淡地看他们一眼后转身带着安颂离开。</p>
 <p>「说吧，你们是什么关系？」</p>
 <p>我没那么好糊弄，找了处拐角，问他。</p>
 <p>他委屈极了，哭哭啼啼地：「我和他不熟的，他之前追我，逼着我和他在一起，我不同意，他还带人欺负我，我腰上还有他打的伤疤。」</p>
 <p>他掀起衣角，露出里面雪莹莹的细腰，侧面有一处长长的疤痕，如同蜈蚣一般。</p>
 <p>我叹了口气，他太清楚怎么让我心软了。</p>
 <p>扯落他的衣角，我紧紧地将他扣入怀中：「安颂，永远都别背叛我。」</p>
 <p>即便如此，心里的不安感仍在加剧，好像怀里的 omega 注定要离开我。</p>
 <p>我缓了缓心悸，放开安颂，转身打算找那几个 alpha 算算这道疤痕的账，却被安颂牵制住。</p>
 <p>他细白的手搭在我的手腕上：「姐姐我累了，我们回家吧。」</p>
 <p>我定定地看了他片刻，他的眸光没有丝毫退却。</p>
 <p>我拉过他的手：「好，我们回家。」</p>
 <p>7</p>
 <p>我重新找了份酒吧的工作，薪水比较高。</p>
 <p>趁休息时间动身去学校送保研的资料。</p>
 <p>将资料递给导员。</p>
 <p>导员接过看了两眼，笑着道：「今早北大发来了保研录取名单，上面第一个就是你。」</p>
 <p>他打开那份名单，我随意扫了下，却被一个名字吸引住视线。</p>
 <p>「安颂？」</p>
 <p>「对，也有他，挺聪明漂亮的 omega。」</p>
 <p>我心里重重一跳，将手机里安颂的照片调了出来：「是他吗？」</p>
 <p>「是他。」</p>
 <p>走出办公室，我靠着墙壁，思绪混乱，不明白安颂为什么要骗我。</p>
 <p>回到家，屋子里空无一人。</p>
 <p>我翻着桌上摆放凌乱的考研真题。</p>
 <p>笔触凌乱地写着错误的答案，可他既然能获得北大的保研资格，怎么会连这些题目都不会做。</p>
 <p>故意的吗？</p>
 <p>我压下疑惑，时间来不及了，得赶去酒吧上班，等回来后再问清楚好了。</p>
 <p>他有些小性子，大概是想借此让我多陪陪他吧，我皱着眉想着。</p>
 <p>昏暗的酒吧里鱼龙混杂，我穿梭在各个卡座去送酒。</p>
 <p>无意识间看到了眼前这幕。</p>
 <p>安颂和上次游乐场的三个男人暧昧地缠在一起，身躯如蛇般缠着一个男人接吻。</p>
 <p>我脑海空白，愣在原地，只听到他含糊的声音。</p>
 <p>「哥哥别和她计较，我和她只是玩玩而已。</p>
 <p>「她特别蠢，一次标记就能让她给我付出这么多，下次我让她给我交研究生的学费她估计也不会拒绝。」</p>
 <p>男人笑着说：「她都那么穷了，你也忍心骗。」</p>
 <p>他语气嫌弃：「确实太穷了，再玩几天就算了，没意思。」</p>
 <p>我稳住身形，颤抖地叫他：「安颂。」</p>
 <p>他猛地抬头，看到我时，脸上闪过震惊、心虚、不知所措。</p>
 <p>他起身，朝我迈出一步：「姐姐……」</p>
 <p>8</p>
 <p>他拉住我的手，嘴唇动了动：「姐姐，不是你想的这样……」</p>
 <p>「那你告诉我，我该想什么？」</p>
 <p>他嘴唇嗫嚅，没了声音。</p>
 <p>「小许，过来给 25 号桌的客人送箱啤酒！」领班大声喊道。</p>
 <p>我垂下眼睫，一点点扯开他紧握的手：「安颂，我们结束了。」</p>
 <p>转身去库房搬啤酒。</p>
 <p>「颂颂过来，别告诉我你真的喜欢上了这个穷鬼。」</p>
 <p>身后 alpha 隐含着威胁的声音响起。</p>
 <p>我回眸，却只看到 omega 投入男人怀抱的背影。</p>
 <p>「哎哥，那个 omega 看着真清纯漂亮，他有主没？」男人的穿着打扮明显是酒吧的新客。</p>
 <p>我搬着啤酒来到 25 号卡座，沉默不语地顺着他手指的方向看去，是安颂。</p>
 <p>另一个人拍了刚才问话那人的脑袋，笑得不怀好意。</p>
 <p>「什么啊，他在我们这出名得很，浪着呢，只要给钱，他来者不拒。」</p>
 <p>「要不要试试？那方面贼带劲。」</p>
 <p>「行啊。」</p>
 <p>突然那人起来推了我一把：「酒搬完了就赶紧滚，别搁这儿碍事。」</p>
 <p>我僵硬地起身，心脏已经痛得麻木。</p>
 <p>原来一切都有迹可循。</p>
 <p>怪不得他对标记那么驾轻就熟，怪不得他身上沾染的信息素总是洗不掉，怪不得他第一次见面可以这么随便……</p>
 <p>在他眼里贞洁大概什么都不是，乖巧听话也不过是骗我为他付出的把戏。</p>
 <p>大概是心里早有对此的预料，所以痛苦之外还有一丝解脱。</p>
 <p>那天我问他身上的柠檬味道是怎么来的，他说是柠檬水洒了，可我在游乐场的时候便闻到那个 alpha 的信息素是柠檬味的。</p>
 <p>其实早就有所猜测了不是吗？</p>
 <p>他和我的相处中早就撒过无数个谎言，只不过因为我爱他，所以孤注一掷地赌上我所有的信任。</p>
 <p>可惜，我还是赌输了。</p>
