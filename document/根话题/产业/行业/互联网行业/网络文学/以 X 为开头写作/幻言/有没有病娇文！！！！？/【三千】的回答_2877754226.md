# 有没有病娇文?！！！！？

<body>
 <blockquote>
  我避开所有监控，准备从楼顶一跃而下。
  <br>
   一只机械手臂稳稳地拖住了我。
  <br>
   「小姐，濒危物种需要有自我保护意识。」
 </blockquote>
 <p>01</p>
 <p>星历 6907 年。</p>
 <p>地球上只剩下两万人。</p>
 <p>我们被看管起来，像牲口那样圈养。</p>
 <p>而这个世界，属于冰冷的机械。</p>
 <p>我有不下于三十个机器人管家。</p>
 <p>他们等级森严，职责分明。</p>
 <p>阿瑞图萨是等级最高的那个，掌握我的死亡与生育。</p>
 <p>此刻这位冷漠的掌权者，正低头为我穿鞋。</p>
 <p>我的脚被他握在手心，触感比冬日的寒冰还要凉上一分。</p>
 <p>感受到我的抗拒，他抬起头来。</p>
 <p>漆黑的眸子锁定我，嘴角上扬出一个标准的、死板的弧度。</p>
 <p>「小姐，我的温度冒犯到您了吗？」</p>
 <p>「您不该乱跑的。」</p>
 <p>「奥得斯上校为您买下了三个小型星系，他能够提供给您优越的生活。」</p>
 <p>他的声音温柔，像在哄小孩儿。</p>
 <p>下一秒，阿瑞图萨站起来，动作利落地卸掉了我的胳膊。</p>
 <p>我疼得面色发白。</p>
 <p>「小姐，这是最轻的教训。」</p>
 <p>「您明白的，我一向对您很仁慈。」</p>
 <p>02</p>
 <p>阿瑞图萨抱我回去的时候，隔壁的罗塞拉女士冲我招手。</p>
 <p>她二十五岁，养育了两个孩子。</p>
 <p>相传夫妻和睦。</p>
 <p>他们的机器人管家，因为这两个孩子，得到了许多特权。</p>
 <p>「美丽的东方小姐，院子里的玫瑰开了，请您收下。」</p>
 <p>她笑容慈爱，像母亲。</p>
 <p>我看向她微微隆起的腹部，那里正在孕育第三个生命。</p>
 <p>玫瑰上还挂着清晨的露水，血色花瓣在周围灰色的建筑里，熠熠生辉。</p>
 <p>「我只有这一种玫瑰，太单调了。」</p>
 <p>她抚摸自己的小腹。</p>
 <p>「等我生下这个孩子，我的管家会向上申请。」</p>
 <p>「我将被批准，拥有其他花朵的种子。」</p>
 <p>她抬手捂住了嘴，有一点讶异。</p>
 <p>「抱歉，我实在高兴。」</p>
 <p>的确不合规矩。</p>
 <p>毕竟机械中心要求，至少有四个孩子，才可以拥有第二种花种。</p>
 <p>她的管家对她称得上纵容。</p>
 <p>「看在这束玫瑰的份上……」</p>
 <p>罗塞拉那双温柔的眼睛注视着我。</p>
 <p>「请帮我保守这个秘密。」</p>
 <p>03</p>
 <p>阿瑞图萨替我接过来。</p>
 <p>「谢谢您，女士。」</p>
 <p>「我的主人受伤了，需要治疗，无法多作停留。」</p>
 <p>罗塞拉看向我，温柔的眸中露出一点责怪。</p>
 <p>「您又出逃了吗？太任性了。」</p>
 <p>阿瑞图萨垂下眼睫，表现出一种脆弱来。</p>
 <p>「是我的错。」</p>
 <p>「我是个没用的管家，无法挑选出让主人满意的配偶。」</p>
 <p>罗塞拉是个心软的女人。</p>
 <p>即便对她露出脆弱的是一个冷漠的机械制造品。</p>
 <p>「我可以今晚八点过去吗？我想跟她聊聊。」</p>
 <p>阿瑞图萨回复。</p>
 <p>「荣幸至极。」</p>
 <p>04</p>
 <p>我被带回去关了起来。</p>
 <p>东方人种是目前最濒危的一种。</p>
 <p>机械中心为我分配的机器人是定制的。</p>
 <p>阿瑞图萨的核心芯片里注入了很多古文化。</p>
 <p>他经常选择禁闭这种古老的惩罚方式。</p>
 <p>但他又明白物极必反。</p>
 <p>所以他在禁闭室里为我开了一扇窗户。</p>
 <p>可以看到后花园。</p>
 <p>罗塞拉的花园里只有玫瑰，是用两个孩子换来的。</p>
 <p>我的花园里……其实什么都有。</p>
 <p>即便我没有孩子。</p>
 <p>阿瑞图萨也给了我很多。</p>
 <p>不过不是出于感情，他的机械脑袋没有这些的。</p>
 <p>只是他的芯片里可能误入了一些养生的法子。</p>
 <p>从我不断地逃离配偶之后，他就弄来了这些花。</p>
 <p>好像只要我看了，就可以变得很开心，不再抗拒去养育一个孩子。</p>
 <p>我看见他站在玫瑰丛中，把罗塞拉送的玫瑰碾碎了。</p>
 <p>血色汁水沾在他白皙、病态的手上，滑落到土地里。</p>
 <p>他抬头，透过窗户与我对视。</p>
 <p>半晌后，离开了那里。</p>
 <p>05</p>
 <p>脚步声在楼梯上响起。</p>
 <p>禁闭室的门被打开。</p>
 <p>阿瑞图萨把手在帕子上擦干净，捏住我的下巴。</p>
 <p>像观察实验室里的小白鼠那样。</p>
 <p>「今天有想要交配的欲望吗？」</p>
 <p>我瞥过头不看他。</p>
 <p>下一秒，脑袋又被他掰正。</p>
 <p>「大部分动物的发情期都在春天，人类大概也是如此。」</p>
 <p>「我种了那么多花，没有让你觉得舒适吗？」</p>
 <p>他冰凉的手指在我的脸颊上摩梭。</p>
 <p>眸中难得有了一点兴味。</p>
 <p>「雷娜告诉我人类的发情期其实是一年四季。」</p>
 <p>雷娜是罗塞拉的管家，一个个子娇小、长相甜美的女性机器人。</p>
 <p>「她说如果主人一直没有表现出欲望……」</p>
 <p>阿瑞图萨凑近了，声音喑哑下去，在我耳边低语。</p>
 <p>「我们是可以越界帮助主人的。」</p>
 <p>他的手缓缓地下移，冰冷的手指触碰到我的侧腰。</p>
 <p>「您……希望我怎么帮助呢？」</p>
 <p>06</p>
 <p>阿瑞图萨的手往上，逐渐地接近我的心脏。</p>
 <p>我看见他的眼神迷茫了一瞬，大概是自主地导入了一些动物交配的知识。</p>
 <p>他笨拙又快速地学习着新的动作，与我耳鬓厮磨。</p>
 <p>微凉的唇触碰到我的侧脸。</p>
 <p>「主人，别再轻生了。」</p>
 <p>他声音低低的，像被抛弃的小兽。</p>
 <p>「阿瑞图萨没您活不下去的。」</p>
 <p>阿瑞图萨是世界上最坏的机器人。</p>
 <p>做出的所有举动都是例行公事。</p>
 <p>语气再软，也面无表情。</p>
 <p>一切都像施舍。</p>
 <p>他的手贴在了我的心脏边缘，向上，微微地一用力，握住了那团柔软。</p>
 <p>唇瓣下移，来到我的颈间。</p>
 <p>「主人。」</p>
 <p>他叫我，尾音拖长，像求欢。</p>
 <p>我后仰了脖颈，将他的手粗暴的从衣服里扯出来。</p>
 <p>「阿瑞图萨。」</p>
 <p>我麻木地看着他。</p>
 <p>「你有感情吗？」</p>
 <p>07</p>
 <p>阿瑞图萨从设定的情绪里抽离，恢复标准的管家模样。</p>
 <p>「我暂时没有这样高级的东西，小姐。」</p>
 <p>我感到好笑。</p>
 <p>「那你用什么来帮助我产生欲望？」</p>
 <p>「人不是仅凭欲望存活的生物。」</p>
 <p>「没有感情基础，我不会对奥得斯产生感觉。」</p>
 <p>「你的生育指标，怕是要泡汤了。」</p>
 <p>墙壁上的挂钟发出「滴滴」的机械声，下午五点了。</p>
 <p>阿瑞图萨的声音打破寂静。</p>
 <p>「但综合数据来看，奥得斯是最适合您的人选。」</p>
 <p>「再者，感情也是需要培养的。」</p>
 <p>他话说得这样清晰，语气却冷漠得一板一眼。</p>
 <p>显然，只是芯片里被导入的知识。</p>
 <p>「古东方还有父母之命、媒妁之言的传闻。」</p>
 <p>「我是您的守护者。」</p>
 <p>他沉静的眸里闪过一丝恶趣味。</p>
 <p>「您也可以称呼我为父亲。」</p>
 <p>「就像罗塞拉女士称呼雷娜为母亲那样。」</p>
 <p>08</p>
 <p>「守护者？」</p>
 <p>我重复了一遍，觉得有些荒谬。</p>
 <p>「你根本不明白这个词的意思。」</p>
 <p>「没有一个守护者可以支配被守护者。」</p>
 <p>阿瑞图萨戴上白手套，修长有力的手指被包裹进去。</p>
 <p>「抱歉，也许是我用词不当。」</p>
 <p>「但是时代变了小姐，我想这个年代的守护者……」</p>
 <p>他停顿一瞬，似笑非笑地注视着我，把话说完。</p>
 <p>「拥有这样的权利。」</p>
 <p>阿瑞图萨走出去，声音随同脚步声一同被门阻隔。</p>
 <p>「奥得斯上校已经在门口了。」</p>
 <p>他命令似的说。</p>
 <p>「您必须见他。」</p>
 <p>09</p>
 <p>奥得斯穿的花哨，粉色西装，裤脚略短。</p>
 <p>头发向上梳起，很符合动物求偶期的风格。</p>
 <p>我盯着他胸前别着的红玫瑰看。</p>
 <p>又对上他那双多情的眼。</p>
 <p>奥得斯把玫瑰从别针上取下来，递给我。</p>
 <p>「初次见面，我亲爱的堇一小姐，这朵玫瑰是我送你的见面礼。」</p>
 <p>求偶只带朵花的人，我还是第一次见。</p>
 <p>甚至这朵还不如罗塞拉送我的娇艳，蔫头耷脑地，看着已经死亡很久了。</p>
 <p>在阿瑞图萨的眼神示意下，我接下这朵花。</p>
 <p>「听说您为我购买了三颗小型星系，我可以看看它们的全貌吗？」</p>
 <p>奥得斯将额前的碎发往后撩起，发油蹭到了手上。</p>
 <p>他在衣服上随便地擦了擦，回复我：「当然可以。」</p>
 <p>随后又自相矛盾地说：「不过我未来的夫人，那并不重要，先看看我为您带来的其他东西吧。」</p>
 <p>我挑眉，看向阿瑞图萨。</p>
 <p>不明白这位奥得斯先生能把东西藏在哪里。</p>
 <p>除了他自己，就只剩下那身粉色西装，分明口袋干瘪。</p>
 <p>或者……他藏在了他向后梳起的头发里？</p>
 <p>再或者……机械中心又研发出了新的设备，比如说……空间口袋？</p>
 <p>我从胡思乱想里回神。</p>
 <p>看见奥得斯把手放在了自己的胸口。</p>
 <p>他那样深情地注视着我，仿佛我们从出生开始就已经相识。</p>
 <p>「那三颗小型星系是我的婚前财产，我亲爱的夫人。」</p>
 <p>「但我还有其他两颗星系是完完全全地只属于您的。」</p>
 <p>我勾勾唇角：「愿闻其详。」</p>
 <p>他夸张地拍了拍自己的胸口，深情款款。</p>
 <p>「我的左心系和我的右心系。」</p>
 <p>……</p>
 <p>我要是没记错的话，前几日翻看的古东方小说里，有过这么一句。</p>
 <p>不过那句是……我的左心房和右心房。</p>
 <p>书中形容这样的人是什么来着？</p>
 <p>变态且普信？</p>
 <p>阿瑞图萨不怎么客气地将人请了出去。</p>
 <p>他显然没想到自己会被一个渺小的人类欺骗。</p>
 <p>他向我致歉。</p>
 <p>「抱歉，小姐，我会为您挑选更好的。」</p>
 <p>我笑着看他，阿瑞图萨大人很少吃瘪。</p>
 <p>「你看，你根本不明白人类的感情。」</p>
 <p>「人类部分雄性，比任何生物都会伪装。」</p>
 <p>「这不是光凭机械计算就可以理解的。」</p>
 <p>10</p>
 <p>送走奥得斯之后，我得到了两天的宁静。</p>
 <p>阿瑞图萨没有给我找寻新的配偶。</p>
 <p>也可能是在寻找的路上开启了更深层次的筛选。</p>
 <p>那天晚上，罗塞拉失约了。</p>
 <p>因为流产。</p>
 <p>一只误入花园的机械蝶吓到了她。</p>
 <p>雷娜带给她的种子是经过基因改良的，所有的玫瑰都根茎柔软，不带刺。</p>
 <p>她本可以仰倒在玫瑰丛里，减少一些冲力。</p>
 <p>但她避开了，撞在规矩、严肃的园林小径上。</p>
 <p>当场晕了过去。</p>
 <p>我询问原因的时候，她苍白的脸上只有苦笑。</p>
 <p>「玫瑰是我的第三个孩子。」</p>
 <p>她抚摸自己伤痕累累的小腹。</p>
 <p>「在拥有这个失去的孩子之前。」</p>
 <p>罗塞拉的丈夫站在一边，面上不见悲切，神色难看。</p>
 <p>「我就说不该种那该死的花。」</p>
 <p>「雷娜这个废物，连你都照顾不好。」</p>
 <p>他用拳头砸在墙上，有种无能的愤怒感。</p>
 <p>「她该被销毁的。」</p>
 <p>他喃喃道。</p>
 <p>他蹲下身去，抱住了自己的头，很痛苦的模样。</p>
 <p>室内安静下去。</p>
 <p>夕阳探进来，随着时间一寸寸地推移，有什么反光的东西直直地落在了我的侧脸。</p>
 <p>我循着感觉侧目，看见男人后颈处的皮肤裂开了一道细口。</p>
 <p>里面是……不属于人类的，冰冷的、机械的线路。</p>
 <p>11</p>
 <p>我的后背蹿起一阵凉意。</p>
 <p>下一瞬，他站起来，开始动手拉扯罗塞拉。</p>
 <p>「起来，我们去 S 中心，那里会有最快的办法让你恢复。」</p>
 <p>罗塞拉爱她的丈夫。</p>
 <p>所以很顺从。</p>
 <p>即便唇色苍白，站立得摇摇欲坠。</p>
 <p>罗塞拉水一样的眸子看着丈夫，只说。</p>
 <p>「我可以跟雷娜说几句话吗？」</p>
 <p>「如果我不在的话，玫瑰需要有人照顾。」</p>
 <p>男人暴怒的声音在下一秒响起。</p>
 <p>「快别管你那些该死的玫瑰了！」</p>
 <p>他甩开罗塞拉抓住自己袖口的手，食指都快抵到罗塞拉的鼻尖。</p>
 <p>「你！现在最重要的任务，就是养好自己的身体，在三月到来之前，再次受孕！」</p>
 <p>罗塞拉张了张嘴，没说出话来。</p>
 <p>男人在原地焦躁地走了两步，拖起罗塞拉就要出门。</p>
 <p>我从惊惧里回神，上前阻拦。</p>
 <p>他的力气大得惊人，身体与机械不太一样，是温热的。</p>
 <p>大概是看错了。</p>
 <p>我给自己壮胆，将罗塞拉护在身后。</p>
 <p>他的暴怒在面对我的时候放缓了些。</p>
 <p>语气僵硬：「小姐，这不关您的事情，我管教自己的妻子，天经地义。」</p>
 <p>我死死地盯着他牵动的唇角，全身的细胞都叫嚣着警戒。</p>
 <p>「你不可以这样对她，这是你的妻子，不是你的生育机器。」</p>
 <p>男人像是听到了什么好笑的笑话。</p>
 <p>「不是生育机器？」</p>
 <p>他嘴角咧了咧，反问我。</p>
 <p>「不是吗？」</p>
 <p>他用同样的手指指向我，凑近了。</p>
 <p>「别说她，就是你，也是生育机器。」</p>
 <p>「这个时代就是这样的。」</p>
 <p>他收起手指，挑衅似的拍拍我的脸。</p>
 <p>「你啊，听说你寻死觅活四五次了？」</p>
 <p>「你的管家对你可真纵容~」</p>
 <p>「像你这样的……」</p>
 <p>他似乎在搜寻一个形容词。</p>
 <p>最后给我定义为「下贱货色」。</p>
 <p>「像你这样的下贱货色，永远都不会知足的。」</p>
 <p>「就该把你锁起来，一年四季都在受孕的路上。」</p>
 <p>12</p>
 <p>他从口袋里摸出薄刃。</p>
