# 救命啊，姐妹们有没有追妻火葬场的书啊？
- 点赞数：5521
- 更新时间：2024年03月11日18时35分34秒
- 回答url：https://www.zhihu.com/question/525548504/answer/2726436355
<body>
 <p>我出事之前，曾经给顾江打过一通电话，哽咽着说：「我做噩梦了。」</p>
 <p>顾江冷淡的声音带着几分无奈与不耐：「悦悦，我们走到头了。」</p>
 <p>我听见那边有女人的声音，撒娇地说她又孕吐了。</p>
 <p>他温柔地哄着她，挂了电话。</p>
 <p>我出事之后，他却花钱请人，四处招魂。</p>
 <p>我的鬼魂就站在他面前，听他颤声问别人：「如果我现在死了，能见到我夫人一面吗？」</p>
 <p>他红着眼圈：「我想再见她一面，不惜代价。」</p>
 <p>我怔了怔，转身就走，赶着投胎。</p>
 <p>天上地下，今世来生，我们再也不要见面了。</p>
 <p>1</p>
 <p>我以为人死以后，万事皆休，什么都解脱了。</p>
 <p>可我错了。</p>
 <p>我死以后，成了鬼魂。</p>
 <p>而且在第七天，我还被拉回人间，看到了最不想看到的人。</p>
 <p>他红着眼，站在我们的家里，形销骨立，眼底发青，仿佛一阵风就能给吹飞了，浑然不像从前的意气风发。</p>
 <p>他抓着做法事的道长，颤抖着问：「如果我现在出事，是不是就能追上我夫人，一起投胎？」</p>
 <p>我悚然一惊。</p>
 <p>道长也悚然一惊：「顾总，不至于。」</p>
 <p>顾江的声音，脆弱得仿佛易断的弦：「我只想再见她一面，不惜任何代价。道长您想想办法。」</p>
 <p>听着很深情，道长都感动了。</p>
 <p>可我靠在墙角，却很想笑。</p>
 <p>他大概忘了，我死那天晚上，还给他打过电话。</p>
 <p>当时我哭得抽抽噎噎，我想跟他说我做噩梦了，我梦见五岁时，顾江爸妈把我领回家，告诉我从此我爸妈不再回来了。</p>
 <p>五岁的我听不懂，但也知道从此在世上孤孤单单，所以悲恸大哭。</p>
 <p>是六岁的顾江搂着我，反复说他要保护我，要照顾我，要陪伴我。</p>
 <p>他郑重地发过誓的。</p>
 <p>我哭着问顾江，他在哪里，他为什么发誓又食言。</p>
 <p>顾江没说话，但有个女人的声音娇娇柔柔地传过来：「顾江，我又想吐了。」</p>
 <p>我听见顾江温柔的声音，带着一丝宠溺：「等着，我给你拿话梅。」然后挂了电话。</p>
 <p>毫不留恋。</p>
 <p>我才想起来，我已经被他抛弃了。</p>
 <p>他和我结婚没几年，就跟我恩断义绝，为了他的秘书卓烟。</p>
 <p>他因为卓烟跟我吵了半年，冷战分居半年，然后又跑回家找我，拿着卓烟的孕检报告。</p>
 <p>他说卓烟肚子里是他的孩子，他不能让孩子没有父亲。</p>
 <p>我彼时刚回家，手里还拿着重度抑郁症的诊断书。</p>
 <p>看着孕检单，我把诊断书死死攥住，朝他笑笑：「哦，可我能。我不离婚，孩子就是私生子，多刺激。」</p>
 <p>顾江皱眉：「吴悦你得有基本的人性，再不喜欢卓烟，她肚子里也是个无辜的孩子。」</p>
 <p>我真的笑了。我笑得眼泪都快出来了，真的很想抬手给顾江一个嘴巴子，可我不能。</p>
 <p>我一抬手，袖子撸起来，我胳膊上那些深深浅浅的疤痕就会露出来。</p>
 <p>那是我在一个个孤独等天亮的深夜，在这段婚姻里，痛苦挣扎的痕迹。</p>
 <p>是重度抑郁症无法自控的症状。</p>
 <p>可落在顾江眼里，可能就是我用来求怜悯的手段了。</p>
 <p>我不想这么卑微。</p>
 <p>顾江说：「强扭的瓜不甜，这是常识。吴悦你清醒点。」</p>
 <p>我胸口闷得发酸。</p>
 <p>当初求婚时，他不是这么说的。他半跪在我面前，可认真了，说他这辈子都会替我遮风挡雨。</p>
 <p>可我依然笑着，打开冰箱，拿出一颗蜜瓜：「我在农家乐强扭的，挺甜。」</p>
 <p>我切开那颗蜜瓜，递给顾江：「不信你尝尝？」</p>
 <p>顾江冷冰冰地瞪我一眼，目光扫过我抬起的手腕，顿了一顿。</p>
 <p>我迅速垂下手臂，让袖子垂落，遮住那冒头的小伤疤。</p>
 <p>顾江过来想抓我的手腕，我直接袭上他胸口：「怎么卓烟不好睡，还是觉得我好？」</p>
 <p>顾江便忘了这事儿，一把推开我，像推开一件垃圾：「吴悦，你以前没这么惹人厌的。」</p>
 <p>我垂了垂眼。</p>
 <p>呵，他说我惹人厌。</p>
 <p>那我就再加把劲。</p>
 <p>我边吃蜜瓜边告诉他，我永远不会离婚，我就喜欢看私生子见不得人的戏码。</p>
 <p>顾江静静地看着我，那眼神让我觉得在他眼里，我又变成了一件垃圾。</p>
 <p>他说吴悦你真的让我恶心。</p>
 <p>说完转身就走，毫不留恋。</p>
 <p>我看着他的背影，眼泪不知不觉掉下来。</p>
 <p>转瞬又被我狠狠擦掉。</p>
 <p>我怕他回头看见，虽然我知道他是不会回头的。</p>
 <p>但是万一呢。</p>
 <p>我的回忆和爱情，都被他糟蹋干净了，现在只有一点可怜的尊严了。</p>
 <p>那天过后，顾江和卓烟公然同居，因为顾家二老不同意，顾江连父母家都不回了，爱了个感天动地。</p>
 <p>而我开始独自生活。</p>
 <p>顾家二老想来陪我，被我拒绝了。</p>
 <p>我打小就是他们照顾长大，不能这么大了还让他们跟我操心。</p>
 <p>我拉紧窗帘，终日躺在床上，有一顿没一顿地吃药，没一顿有一顿地吃饭。</p>
 <p>我不想这样的，可抑郁症真的很可怕，它会让人变成活死人。</p>
 <p>我想如果我当初不嫁给顾江，不把他当成全世界，不跟他和卓烟拉扯长达一年，我是不是就不会得这个病。</p>
 <p>这谁也不知道。</p>
 <p>可有一天，我刷手机看见朋友圈里卓烟的孕妇照，三个月的孕妇，身材丰腴，神情安逸。</p>
 <p>而我站在镜子前，肋骨一根根清晰可见，脸色蜡黄，无精打采。</p>
 <p>那天我看着镜子，突然就哭了。</p>
 <p>哭完我梦见了五岁时的场景，爸妈再也回不来了，顾江抱着我发誓会保护我照顾我。</p>
 <p>他说得那么真，以至于我醒来后还以为他会继续保护我。</p>
 <p>我迷迷瞪瞪给他打电话，抽抽噎噎告诉他我做噩梦了，我哭了，我很害怕。</p>
 <p>我当时真的是处于迷瞪状态。</p>
 <p>顾江没说话，卓烟说话了。她孕吐了。</p>
 <p>顾江说去给她拿话梅，走开后，就跟我说了一句话：「吴悦，除了离婚的事，其他事你一概不要骚扰我。」</p>
 <p>说完就挂了。</p>
 <p>我看着手机，看了很久，看到身子都麻了。</p>
 <p>可我没哭。</p>
 <p>好像是大脑怕我难过，屏蔽了我的感情，让我连悲伤都体会不到了。</p>
 <p>也可能是抑郁症又加重了。</p>
 <p>我恍恍惚惚记起我该吃药了，又恍恍惚惚下床，吃药。</p>
 <p>吃完没两分钟，突然一阵窒息，我就倒在了地上。</p>
 <p>我这才意识到，我可能误吃了顾江留在家里的消炎药。</p>
 <p>能导致我严重过敏那种。</p>
 <p>窒息让我的胸口火辣辣地疼痛，我鼻腔因为吸不上气也开始疼得神经一跳一跳。</p>
 <p>可我的大脑此时却十分清醒，一点都不再恍惚，前尘旧事如走马灯一样在我眼前飘过。</p>
 <p>我慈爱的父母，友善的顾家爸妈，以及陪了我二十多年的顾江。</p>
 <p>我脑海里的顾江，颀长英俊，爱我如珠似宝。</p>
 <p>我看着他笑了，可眼角一滴泪，却淌过脸颊，安静地落在地毯上。</p>
 <p>那是我留在人间最后的东西。</p>
 <p>2</p>
 <p>我以为人死了就万事皆休了。</p>
 <p>我以为我死了就可以摆脱伤心了。</p>
 <p>可我没想到，在我死后的第七天，我又被一股力量拉回了阳间，回到我死去的那间房。</p>
 <p>房间里摆着法器，有个穿着卫衣的年轻人，拿着罗盘在做法。</p>
 <p>我回来时，他看了我一眼，放下罗盘：「法事做完了。」</p>
 <p>顾江环视着房间，神色急切而凄苦：「人呢？悦悦在哪里？道长，我怎么看不见？」</p>
 <p>年轻人又看了我一眼，发现我靠在墙边，只是看着顾江却默不作声，便什么都没说。</p>
 <p>顾江又在房间里转了一圈，红了眼睛：「你是不是在骗我？」</p>
 <p>他瘦得很厉害，才七天就瘦了整整一圈，眼底大片的青，让我怀疑这七天他到底睡没睡觉。</p>
 <p>我从来没有见他这么着急过，不，是没这么为我着急过。</p>
 <p>当初我流产，打电话给他，让他快来救救我和孩子。</p>
 <p>可他连电话都没接。</p>
 <p>他说他在开会，可我给他公司打过电话，他根本没在公司。</p>
 <p>他跟卓烟出去考察了。</p>
 <p>我躺在床上，看着他姗姗来迟的身影，一语不发。</p>
 <p>他当时握着我的手红了眼：「悦悦真坚强，自己来医院，下次老公不让你自己受罪了。」</p>
 <p>可是后来他食言了。</p>
 <p>他让我在孤独的深夜受罪一次又一次，直到以死亡为解脱。</p>
 <p>我低下头，一滴泪掉落在空气中，又迅速消失得无影无踪。</p>
 <p>年轻道长又看了我一眼，问顾江：「尊夫人有没有什么未尽的愿望？兴许你帮她解决了，她就出来见你了。」</p>
 <p>顾江怔了怔，眼圈更红，嗓音嘶哑得像是扎了鱼刺一样：「她的愿望……我永远做不到了。」</p>
 <p>是的，他当然做不到。</p>
 <p>世界这么大，这么繁华，可我唯一的愿望就是顾江能爱我、陪我。</p>
 <p>可这一点，我生前他没做到，我死后他再也做不到了。</p>
 <p>我听见他空洞地问道长：「她死前，胳膊上全是深深的刀疤，下了阴间，鬼魂还会不会疼啊？」</p>
 <p>我低下头，看着我即便变成鬼，依然深可见骨的疤痕。</p>
 <p>不会，我不会疼。</p>
 <p>因为疼也不会有人心疼。</p>
 <p>道长也看了我胳膊一眼，声音冷了下来：「她为什么会有那么多刀疤，是你弄的吗？」</p>
 <p>顾江摇了摇头，眼神没有一点生气，空洞得可怕：「不是，我不知道，我根本不知道她有刀疤。我要是知道，我……」</p>
 <p>他没有说下去，倒是我看着他，又被逗笑了。</p>
 <p>他要是知道能怎么样呢？又像看一件垃圾一样看我？还是觉得我又换了种方式博得他注意？</p>
 <p>觉得我丧心病狂地想破坏他和卓烟这对有情人？</p>
 <p>我闭上了眼睛。</p>
 <p>我宁死，也不会让他知道。</p>
 <p>我宁死，也不让他有机会羞辱我。</p>
 <p>顾江嗓子哑得厉害，咳了一声，却还是嘶嘶哑哑地：「我从她房间里找到了这个……」</p>
 <p>他摊开手心，一张皱巴巴的诊断书摊在他掌中。</p>
 <p>他的声音有些颤抖，问道长：「她有重度抑郁症，她为什么不告诉我？」</p>
 <p>他低头看着那张诊断书，整个人都开始发抖：「她为什么不告诉我，她要是告诉我，我真的会留下来照顾她的，为什么……」</p>
 <p>道长有些好奇：「哦？那你那天是在做什么呢？」</p>
 <p>顾江愣住了。</p>
 <p>他的神情有些绝望，仿佛想到了什么，抖得更加厉害，连声音都在抖：「我在……逼她……离婚。我还……告诉她……我让我秘书怀孕了。」</p>
 <p>他捉住道长的胳膊，像是捉住了救命的稻草：「她会不会因为这个不回来？她是不是不想见我？道长她是不是不想见我？」</p>
 <p>道长微笑着，一根根手指地掰开他的手，淡然道：「你觉得呢？」</p>
 <p>他低头收拾法器，边收拾边说：「她想见你时候就会出现。要是不想，我也没办法。」</p>
 <p>顾江的眼神又开始空洞起来。</p>
 <p>仿佛有人把其中的生机抽走了，抽得一点都不剩。</p>
 <p>他面无表情，看着道长的身影，轻飘飘地问：「道长，如果我现在出事，我是不是就一定能见到我夫人？」</p>
 <p>我听了悚然一惊。</p>
 <p>道长也悚然一惊，直起身子看了看他，又看了看我害怕的神情，认真道：「顾总，不至于。」</p>
 <p>顾江不说话，只是看着手里那张诊断书，不知在想什么。</p>
 <p>道长自顾自收拾好，跟顾江打声招呼，便走出了房间。</p>
 <p>经过我的时候，他念了句咒语，一道金光打在我身上。</p>
 <p>他看看我斑驳的胳膊，声音温柔起来：「去留随意，不许伤人。」</p>
 <p>我不知道他是谁，我也不想知道。</p>
 <p>我更不想伤人。</p>
 <p>我有过伤害卓烟的机会，可我没有。</p>
 <p>我知道变心的是顾江，今天没有卓烟，明天也会有卓云、卓雾、卓雨。</p>
 <p>所以在卓烟莫名其妙滚下楼梯后，我还替她叫了救护车。我是和她抢男人，但我没抢到丧良心。</p>
 <p>可后来，顾江说我嫉妒卓烟，伤害卓烟。</p>
 <p>他第一次和卓烟开房在一起，就是那天。</p>
 <p>现在，他拿着我的诊断书发呆，一副无法接受的样子。</p>
 <p>像极了我在卓烟朋友圈看到顾江披着浴袍时表情。</p>
 <p>3</p>
 <p>道长走后，我没有即刻就走。</p>
 <p>毕竟再回去，我可能没有任何机会返回人世了。</p>
 <p>我待在我死去的房间，奇怪地看着顾江。</p>
 <p>他不回去陪卓烟吗？不是那么宝贝卓烟肚子里的孩子吗？</p>
 <p>他一直坐在我卧室的地上，抱着个紫檀匣子做什么。</p>
 <p>我凑近看了看，怔住了。</p>
 <p>匣子上，是我的照片，我十八岁时的照片。</p>
 <p>这应该是我的骨灰盒。</p>
 <p>可是这是谁给弄的骨灰盒，是恶作剧还是什么，上面贴的不是我的单人照，而是我和顾江的合照。</p>
 <p>那天是我答应顾江在一起的第一天，顾江举着手机跟我合照，两张青春的脸上洋溢的全是对未来的期待和坚定的决心。</p>
 <p>谁能想到，有一天我们会分开，还分得那么难看。</p>
