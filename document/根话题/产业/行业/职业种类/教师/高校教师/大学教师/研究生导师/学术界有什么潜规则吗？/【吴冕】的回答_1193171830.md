# 学术界有什么潜规则吗？
- 点赞数：7679
- 更新时间：2021年08月08日13时24分39秒
- 回答url：https://www.zhihu.com/question/271530655/answer/1193171830
<body>
 <ol>
  <li data-pid="aX1gxW30">紧跟你研究领域中的风口。再烂的学术背景和方法论站在风口上也能飞。</li>
  <li data-pid="U6gKdImo">千万不要只跟着导师做。导师顶多能占你20%的精力。至于这20%能不能让导师产生80%的价值，这就要看你导师的能力了。如果你给他/她20%了还不够，那么给更多更是浪费。记住，你不是要成为他/她，而是要去未来某一个他/她到不了的地方。</li>
  <li data-pid="EGfEK9px">除了不要围着导师转，还要记得广撒网。</li>
  <li data-pid="RIxWqxBc">Fake it till you make it. 实力不重要，有很多人看起来成功其实只是因为装得住。</li>
  <li data-pid="U46CG-zt">学会给自己做背书/endorsement。学术人面前说资源；其他人面前说自己的学术成就。</li>
  <li data-pid="qdU3qdYs">永远要有退出机制，无论做到什么阶段了。一定要有一个立马不做学术也立马能挣钱发展的很好的退路。</li>
  <li data-pid="QEWEuGlc">永远不要在低谷选择退出，因为这只是看上去的退出，那个低谷会永远伴随你，像一个诅咒一样。</li>
  <li data-pid="Ywp8Mty7">关于读博，大多数人都会用四年以上的时间来拿到学位，因为博士没有deadline，不管学制怎么要求，但凡读博读发达的，基本重点都不在写完博士论文上。是上课，发表，甚至做公司做的很大的都有。反而我身边太多太多读了四年读完了结果除了一张毕业证什么都不剩的。有的连导师和同事同学的关系甚至都没维持住。这样的话，继续把学术发展下去的难度可以说是相当大。</li>
  <li data-pid="ILInvuGg">很多科研工作者心里都有些问题。很多人做学术是为了逃避现实问题，就像很多人打游戏一样。这样的学者或者游戏玩家是没有灵魂的。对自己做的事情一定要有自己独到的vision/理解/假设/猜测/见解。不一定要有data/evidence支持，但一定要meaningful/有建设性/毁灭性idea。</li>
  <li data-pid="5hEwCn72">自古以来知识分子圈子都很难混，所以切记不要妄加评论任何人，以及任何人的research/presentation。作为听众或一般的读者，必须要给评价的时候可以：只说自己听懂的部分有多棒+听不懂的地方自己非常希望presenter能帮助自己弄清楚。说的写的人都可以叫presenter，说和写出来的东西都可以叫presentation。当然，做reviewer或其他类似情况除外。</li>
 </ol>
