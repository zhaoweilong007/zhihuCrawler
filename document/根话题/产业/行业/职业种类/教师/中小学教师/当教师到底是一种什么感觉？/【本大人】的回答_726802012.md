# 当教师到底是一种什么感觉？
- 点赞数：24730
- 更新时间：2022年08月01日23时51分19秒
- 回答url：https://www.zhihu.com/question/324705910/answer/726802012
<body>
 <p data-pid="DgWZJD-X">老师范生一个，随便说点吧。</p>
 <p data-pid="Vsxp3GXI">当教师的好处：</p>
 <ol>
  <li data-pid="QYIuIMmz">工作稳定。</li>
  <li data-pid="YEM-JCg1">收入稳定。</li>
  <li data-pid="GI8E0ZeM">假期较多。</li>
  <li data-pid="zJX3HG_I">人际关系相对简单。</li>
  <li data-pid="rro92PR8">桃李满天下，容易获得成就感。</li>
  <li data-pid="hnHjlD-B">因为长期和孩子们相处，在心态上会相对年轻些。</li>
 </ol>
 <p data-pid="rHQAIasN">可能的不好的地方：</p>
 <ol>
  <li data-pid="2I1zYoXQ">收入不高。</li>
  <li data-pid="nYEBJW2f">社会地位不高。（经济较发达的城市，老师的收入和地位会好一些。）</li>
  <li data-pid="ehiR5yRL">如果是男生，家境又一般，难娶。（但还是能娶到的。）</li>
  <li data-pid="YCXc4Wxb">工作任务重，事务繁杂。备课、上课、改作业、教研、家访、开会、培训、迎接各种检查等，让你忙得停不下来。</li>
  <li data-pid="mZ3d_xd6">一些熊孩子和狼家长会相当棘手，相关问题很难处理妥当。</li>
  <li data-pid="iTPtjGu1">咽喉炎和颈椎病等职业病。</li>
 </ol>
 <p data-pid="1NJqmsh3">怎么成为一个好老师？</p>
 <ol>
  <li data-pid="6u26VsZc">任何情况下，都禁止做这两件事情：体罚和变相体罚学生；和学生谈恋爱。</li>
  <li data-pid="aPh6nN45">希望你在入职后的第一个学期，每一节课都有详案。详案的基本要求是：把你准备在课堂上说的每一句话、安排的每一项教学活动，都写在教案里。如果实在做不到，至少坚持写一个月详案。</li>
  <li data-pid="x7bLxPCD">每节课后必须要有反思。文字可多可少，少至三言两语亦可。</li>
  <li data-pid="dT3-kdE2">在征得对方同意的情况下，多听老教师的课，评课时只说对方好的地方，不说不好的地方。因为你的目的是学习他人的优点。说缺点这种事情，让教研组长、教务处长、教育局的评课专家来说。</li>
  <li data-pid="IKwiKY1Z">在适当的时候邀请老教师来指导你的课，认真倾听他们的批评意见。</li>
  <li data-pid="tylbOrZn">要尊重领导，路上碰到，至少主动打个招呼，不要视而不见。不要与同事有大的矛盾冲突。</li>
  <li data-pid="A8QPNPuT">爱你教的那些孩子们。虽然不是每一个，但一定要是绝大多数。师者父母心，记住了，你的爱的出发点不是要他们获得好成绩，而是希望他们能健康快乐成长，希望他们未来有出息。</li>
  <li data-pid="9xVj1aGG">在条件允许的情况下， 一定要完整地带一届毕业生。争取能从高一一直带到高三毕业。这样你才有资格说自己是“有经验的”老师。这三年，是你职业生涯的起点和基础，一定要不怕苦、不怕累，该准备的材料，该收集和编撰的资料，尽可能地都处理好并留下存档。容留日后不断增删，不断反思，不断进步，直至形成自己稳定的教学风格。</li>
  <li data-pid="jQ7yBQb3">一旦形成自己稳定而高效的教学风格，教学工作就会慢慢变得轻松惬意起来。因为有之前经验和积累起来的资料为基础，各项教学工作对付起来都会比较得心应手，偶有突发事件，也能凭经验妥善处理。学生的成绩也会稳步上升。</li>
  <li data-pid="27SluDYF">最后关于班级管理，我只强调两点：其一，言不出则已，言出必践。惩戒和表扬学生之前，一定要想清楚，自己能不能做到，自己做不到的事就不要说出口。比如，你说罚某生打扫教室三天，你如何确保你能让他做到这一点？如果你没有办法让他老老实实接受这三天的惩罚，那就不要这样罚。如能做到言出必行，不出一月，威望可立。威望既立，诸事可行。其二，批评教育学生之前，一定要提前了解他的家庭情况和成长背景。必要时，最好先和家长聊聊，能家访更好。这样做，还能避免趟雷，以免惹上狼家长，不得全身而退。</li>
  <li data-pid="DrLBkqpi">学校也是社会的一部分，照样有恶人恶事。有些领导、有些同事、有些家长甚至有些学生，会有恶言恶语，有的甚至真的想要对你做恶事，所以要有安全意识，注意保护好自己的人身财产安全。</li>
 </ol>
 <p data-pid="_PB7-zJW">一旦你成了一个好老师，深受学生爱戴，学生成绩又好，家长喜欢，领导喜欢，同事敬重，各种优秀和奖励拿到不想拿，人生还是很惬意的哈。</p>
 <p data-pid="1RkvsDc7">当然，要到这一步，肯定要付出很多努力，受很多委屈的，人生哪有那么容易呢？</p>
 <p data-pid="Ke6AEowK">最后一句，老师这个职业有一个非常大的特点，那就是：如果你真的愿意克服一些困难，成为一个好老师，真的没有什么事物能够阻止你的。</p>
 <p data-pid="ym4KFLuS">从师十二载，弟子三千人。</p>
 <p data-pid="hsZG_Gld">愿君前程似锦！</p>
