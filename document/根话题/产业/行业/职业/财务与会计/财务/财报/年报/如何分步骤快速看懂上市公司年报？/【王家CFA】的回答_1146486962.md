# 如何分步骤快速看懂上市公司年报？
- 点赞数：11489
- 更新时间：2024年03月11日15时41分23秒
- 回答url：https://www.zhihu.com/question/381804940/answer/1146486962
<body>
 <p><strong>我是以投资人视角来看的。</strong></p>
 <p>看年报，最主要的目的，<strong>是了解一个公司是怎么赚钱的；</strong></p>
 <p>而最主要的作用，<strong>是评估公司股票未来价值是否提升。</strong></p>
 <p><strong>本文 12000 多字，可能需要半小时，请大家耐心阅读。</strong></p>
 <p><strong>或者先收藏着，用到时回来备查</strong>。</p>
 <p>你可能觉得这并不是一个快速阅读的方法，相信我这是一个阅读框架，当掌握了这种模板化的阅读方法后，剩下只是时间上的磨炼问题，熟能生巧。</p>
 <p>如果是第一次覆盖一家公司，可以完整地按照以下步骤完成；如果很熟悉了，可以把你已经形成的要点做一次重新梳理。</p>
 <p><strong>先打击一下大家的预期：</strong></p>
 <p>分析一家上市公司，本来是很复杂的，因为年报本身都有 100 多页，想想工作量就是很大的。</p>
 <p>我想这也是很多人希望「快速看懂上市公司年报」的动因。</p>
 <p>「是不是有某种秘籍，让我们 10 分钟？1 小时？2 小时？看懂呢？」</p>
 <p>10 分钟的高手我没见过，但是我这个前券商分析师可以做到 1 个小时完成最终的 5 页纸年报点评——</p>
 <p>对年报点评这种级别的报告，如果你是第 3 名以后给到买方的，基本只会给你的邮件拖进垃圾箱。</p>
 <p>除非是行业大佬，那他们第二天给报告都有人等着看。</p>
 <p><strong>今天分享的方法，根本不是什么秘籍，只是一个训练手册。</strong></p>
 <p>甚至对于专业选手来说，只是一个训练大纲，把每一个 Step 按照你覆盖的行业和公司进行详细展开，都可以按照至少 1:3 的篇幅进行拓展。</p>
 <p>每个行业都有自己的「窍门」，工作关系可能我只是熟悉券商、SAAS、线下零售几类公司的分析。</p>
 <p>更详细的手法都是吃饭的家伙，大家都不会轻易泄露——到现在我都不知道有色、芯片、医疗应该怎么看。</p>
 <p><strong>然后给大家写点鼓励：</strong></p>
 <p>100 多页的公司年报，很多部分对投资者来说是没有用的；</p>
 <p>如果我们熟悉一个行业，熟悉这家公司，又可以剔除很多不重要的文字讨论与数据；</p>
 <p>熟能生巧的过程最终就是让你看到一个年报，你可以准确地在目录里的第几个条目下的第几点寻找你需要的内容；</p>
 <p>当你积累好自己的分析模型，自己的打分体系，积累好自己的写作模板，输出也都不是难事。</p>
 <p>如果你不是老板，那么第一步先做到：</p>
 <ul>
  <li>有分析套路；</li>
  <li>逻辑自恰；</li>
  <li>有自己独到的观点；</li>
  <li>无关对错。</li>
 </ul>
 <p><strong>1 年时间足矣</strong>。</p>
 <p>如果你已经成为了中级分析人员，提高的目标就是——在熟练的基础上，尽量准确、尽量合理。</p>
 <p>快速看动上市公司年报，不是一项无法完成的任务，这和减肥、健身是一样的，存在明确的路径。</p>
 <p><strong>看你愿不愿意，沉下心提升自己的核心能力了。</strong></p>
 <h2><strong>Step0 我们看年报是为了什么</strong>？</h2>
 <p>解释一下。</p>
 <p>作为投资者，如果你希望通过研究一家上市公司的年报，寻找投资机会。</p>
 <p>意味着你认可价格会围绕价值波动，需要通过上市公司年报去计算公司价值。</p>
 <p>分析公司历史，与猜测公司未来。</p>
 <p><strong>业务端，和财务端。</strong></p>
 <p>作为投资人，公司年报就是他们的体检报告，财务报表是体检的重要数据，业务讨论（业务概况、经营讨论、重要事项）的表述就是诊断的总结与评估。</p>
 <p><strong>分析历史（财务、业务）→评估财务数据及业务讨论→去伪存真→得出评价结论</strong></p>
 <p>阅读年报后，有以下几种用途：</p>
 <ul>
  <li>完整全面的投资分析；</li>
  <li>完成 DCF 模型、比较法估值模型里的相关参数、假设；</li>
  <li>写作行业监控/竞品分析报告；</li>
  <li>写作财经新闻；</li>
 </ul>
 <p><strong>对于我来说，所有关于年报的分析都会做进 DCF 模型中，不至于让我的分析变成简单的观点和数据堆砌，示意一</strong>下：</p>

 <p>对于写作报告的人来说，你也可以制作自己的模板，哪些指标或分析放在哪些段落。</p>
 <p>本次介绍就不进行更深入的如何制作模型、如何写作报告、如何做新闻的部分了。</p>
 <h2><strong>Step1 了解行</strong>业</h2>
 <p>下面我们要开始纯干的部分了，先给大家上一个图，我自己习惯的一个分析框架：</p>
 <figure>
  <img src="https://picx.zhimg.com/50/v2-3f03fac7b5c3c8507fba0f55f5e87894_720w.jpg?source=1940ef5c" alt="" data-rawwidth="1990" data-rawheight="2264" data-original-token="v2-3f03fac7b5c3c8507fba0f55f5e87894" class="origin_image zh-lightbox-thumb" width="1990" data-original="https://pic1.zhimg.com/v2-3f03fac7b5c3c8507fba0f55f5e87894_r.jpg?source=1940ef5c">
 </figure>
 <p>上面的图，最多只到 4 级菜单。</p>
 <p>不同行业、不同公司都需要添加不一样的第 5 级或者第 6 级菜单，仍然有很大的拓展空间，这里就写到我觉得合适的级别位置。</p>
 <p>大家可以根据自己的目的，适当补充。</p>
 <p><strong>如果年报里有相关内容，主要工作就是阅读和判断真实性；</strong></p>
 <p><strong>如果年报里没有相关内容，留个记号，自己通过找人、上网、亲自调研寻找。</strong></p>
 <p>在开始阅读一家公司的年报之前，一定要做一些准备工作，就是对上市公司所在行业有一个了解。</p>
 <p>有的人在对行业很陌生的时候，直接开始阅读年报，会有几个问题：</p>
 <ul>
  <li>分析不对路子，比如你拿着财务分析教材去看手游公司和证券公司的存货周转率，显然有问题；</li>
  <li>找不到合适的分析工具，比如你要是不知道房地产行业的运行规则，直接拿前三年销售做线性回归，错的连亲妈都不认识；</li>
  <li>甚至都不知道该看哪些内容，比如你要是不知道一个行业是不是出口导向型，都不知道去看国际宏观经济和汇率走势；</li>
  <li>还有很多坑.........</li>
 </ul>
 <p><strong>所以在阅读年报以前，务必先了解行业。</strong></p>
 <p>有些上市公司的年报非常温馨，他会把行业的盈利模式介绍一遍，这种情况在美股更常见，A 股上市公司就不愿意这么费劲——科普这件事情并不要求强制披露。</p>
 <p>有以下几个要点我认为必须提前阅读：</p>
 <ul>
  <li><strong>市场特征：</strong>自由竞争/垄断竞争/寡头垄断/完全垄断？这决定了行业内部玩家的定价策略、成本控制能力、市场分割特征等因素，比如分析两桶油，寡头市场里你就自然而然知道定价相关驱动力与影响因素；</li>
  <li><strong>是否为周期性行业：</strong>这非常重要，我们一般认为有色、钢铁、建筑建材、房地产、金融（等等等等）都是周期性行业；白酒、农业、餐饮、食品饮料（等等等等）属于非周期性行业。<br>
    区分是否为周期性行业，对你评价一家公司影响巨大。<br>
    比如一个连锁餐饮一路业绩下滑，它可能真的不行了，但是一个做稀土的，一路下滑可能只是探底过程，有一天会回来；</li>
  <li><strong>行业阶段：</strong>新兴行业/增长行业/成熟行业/衰退行业？这有助于你去分析公司的各类指标，特别是增长能力。<br>
    都是 20% 增长，对于一个芯片公司可能就是不及格，但是对一个钢铁企业，那简直是爆炸了；</li>
  <li><strong>对资产依赖程度：</strong>重资产/轻资产？这个不用多解释，重资产行业你去看人效作用没那么大；轻资产行业你盯着折旧也没太多意义；</li>
  <li><strong>行业驱动力：</strong>政策驱动/产品驱动/销售驱动/运营驱动/资本驱动？这有助于你迅速定位到底哪些财务指标是你应该关注的。<br>
    比如游戏行业你就得看几个拳头产品的数据，比如有色行业你就要关注除了市场价格外它们的资源储备，又比如信托公司你就得去看它们的投资回报和坏账情况；</li>
  <li><strong>其他一些要素</strong>：地域特征/客户分层/产品波动性，etc。<br></li>
 </ul>
 <p><strong>只有预先储备了行业知识，你阅读上市公司的年报才能更高效。</strong></p>
 <h2><strong>Step2 弄明白一个上市公司的基本商业模</strong>式</h2>
 <p><strong>商业模式决定了我们在后续阅读业务分析+财报的时候，关注那些段落和财务指标。</strong></p>
 <p>即使同一个行业，不同公司的商业模式也是不同的，一般来说上市公司的年报都会稍微提一句自己的商业模式。</p>
 <p>同样，这多见于美股公司。</p>
 <p>一般来说，最粗放的分类方法包括线型公司、平台型公司。</p>
 <p>如果不能区分公司的类型，我们很多分析都是错误的。</p>
 <p>举个例子：都是给别人授权品牌，很多品牌仅仅是给授权使用并不参与任何生产过程，这是典型的平台型企业；有些品牌深入参与设计、品控、销售，这就很类似线性公司。</p>
 <p>（如果你有自己的一套划分商业模式的手法，只要能够更准确地帮助识别商业模式，都是可以的。）</p>
 <p><strong>线型公司——公司为客户直接提供价值，直接从中获益，以此向下可以进行更细致的划分。</strong></p>
 <p>我们首先可以划分为 To C 和 To B 型公司。</p>
 <p><strong>1）To C——终端产品与服务直接卖给个人</strong></p>
 <p>我们需要在年报中找到以下个关键要素：</p>
 <p><strong>什么产品？</strong></p>
 <p>我们可以继续细分，举例：</p>
 <ul>
  <li>实物/虚拟（边际成本，比如分析纯硬件和硬件+增值公司的不同）；</li>
  <li>高单价/低单价（消费决策机制，比如分析白电和小家电的不同）；</li>
  <li>奢侈品/必需品（消费弹性，比如分析疫情对高端白酒和低端白酒的差别）；</li>
  <li>高毛利/低毛利（产品盈利方式，比如分析潮牌和普通服装的不同）；</li>
  <li>高频/低频（消费节奏，比如分析商旅酒店和度假酒店的不同）</li>
  <li>其他我们认为有助于了解一个产品到底有哪些特性的要素；</li>
 </ul>
 <p><strong>客群什么特征？</strong></p>
 <p>我们可以继续细分，举例：</p>
 <ul>
  <li>是否有地区性（产品扩展及增长空间，比如分析地区餐饮和全国性餐饮）；</li>
  <li>年龄；</li>
  <li>支付能力；</li>
  <li>有助于更精确描绘目标客户的一系列画像；</li>
 </ul>
 <p><strong>解决了消费者怎样的需求？</strong></p>
 <p>我们可以继续细分，举例：</p>
 <ul>
  <li>基本需求还是高级需求；</li>
  <li>个人消费需求/家庭消费需求/送礼需求/社交需求；</li>
  <li>其他我们认为的重要需求，<strong>有助于我们判断未来</strong>；</li>
 </ul>
 <p><strong>如何实现销售的？</strong></p>
 <p>我们可以继续细分，举例：</p>
 <ul>
  <li>主要渠道；（例如：线上/线下、合作/自营等）</li>
  <li>营销方式；（例如：裂变、打折、代言等）</li>
  <li>品牌力；（比如白酒、食品饮料行业要特别注意这点）</li>
  <li>其他我们认为有助于分析<strong>「产品如何找到人」</strong>的要素。<br></li>
 </ul>
 <p><strong>2）To B</strong></p>
 <p>我们需要在年报中找到以下个关键要素：</p>
 <p><strong>什么产品？</strong></p>
 <p>我们可以继续细分，举例：</p>
 <ul>
  <li>实物/虚拟（边际成本，比如分析纯设备和设备+咨询公司的不同）；</li>
  <li>大额/小额；（采购决策机制，比如分析大型软件和 SAAS 软件，评估销售增长特征）；</li>
  <li>长期/一次性；（收入模型）</li>
  <li>过程产品/终端产品（借以分析各项财务指标影响因素）；</li>
  <li>高毛利/低毛利（产品盈利方式，比如分析通用设备和特种设备的不同）；</li>
  <li>高频/低频（消费节奏，比如分析数字广告和传统广告的不同）</li>
  <li>其他我们认为有助于了解一个产品到底有哪些特性的要素（<strong>公司通过什么方式、在多长时间、以多大代价、卖给怎样决策流程的、什么客户、满足他们什么样的需求？</strong>）；</li>
 </ul>
 <p><strong>产业链什么位置？</strong></p>
 <p>我们可以继续细分，举例：</p>
 <ul>
  <li>上中下游；</li>
  <li>市场地位（垄断、头部）；</li>
  <li>多行业通用/垂直行业专用；</li>
  <li>其他有助于我们分析公司<strong>议价能力、定价能力、价格销量的要素</strong>；</li>
 </ul>
 <p><strong>客群特征？</strong></p>
 <p>我们可以继续细分，举例：</p>
 <ul>
  <li>大 B/小 B；</li>
  <li>G；</li>
  <li>在客户采购中的地位（主营业务、通用业务、边缘但必须业务）；</li>
  <li>客户决策模式（谁出钱、谁获益、谁拍板等）；</li>
  <li>其他有助于我们<strong>理解销售对象采购细节的要素</strong>。<br></li>
 </ul>
 <p><strong>平台型公司</strong></p>
 <p>公司为买卖双方或者多方提供各类交易的场所，从撮合中获益，以此向下可以进行更细致的划分。</p>
 <p>我们需要在年报中找到以下个关键要素：</p>
 <p><strong>怎样的平台价值？</strong></p>
 <ul>
  <li><strong>信息/交易/履约/混合；</strong></li>
  <li>技术驱动/资源驱动/规模驱动（上市公司一般会进行夸张，这个问题一般要我们自己寻找）；</li>
  <li>平台对谁收费？怎么收费？对哪类资源收费？</li>
  <li><strong>其他有助于识别平台型企业核心价值的要素；</strong></li>
 </ul>
 <p><strong>平台参与方都是谁？</strong></p>
 <ul>
  <li>B/C/G;</li>
  <li>分别有什么特点点；</li>
  <li>平台满足用户怎样的需求；</li>
  <li>其他有助于<strong>分析用户粘性、付费意愿、续费意愿的要素；</strong></li>
 </ul>
 <p>对 To C 和 To B 平台有不同的分析方法，大家可以自己去补充。</p>
 <ul>
  <li>C 端交易价格透明、决策时间短、决策流程短、标品居多，etc</li>
  <li>B 端交易价格不透明、决策时间长、决策流程长、非标居多，etc</li>
 </ul>
 <p><strong>混业经营公司</strong></p>
 <p>上市公司普遍存在混业经营现象，除了上面对不同板块一一进行分析以外，还要研究不同板块间的协同作用（<strong>Synergy，包括业务上、财务上协同</strong>）。</p>
 <ul>
  <li>是否能够实现<strong>交叉销售</strong>；（做零售的自己有品牌，做社交的自己有直播或游戏）</li>
  <li>是否能够<strong>平滑业绩</strong>（比如某些板块好的时候另一些板块一般，某些板块一般另一些板块好——白酒、食品饮料）；</li>
  <li>是否有<strong>财务协同</strong>（比如赚钱的板块和现金流好的板块，可以在财务上互补）。<br></li>
 </ul>
 <h2><strong>Step3 业务总</strong>览</h2>
 <figure>
  <img src="https://pic1.zhimg.com/50/v2-3cc59c41668002432c019619d6951c2e_720w.jpg?source=1940ef5c" alt="" data-rawwidth="2560" data-rawheight="781" data-original-token="v2-3cc59c41668002432c019619d6951c2e" class="origin_image zh-lightbox-thumb" width="2560" data-original="https://picx.zhimg.com/v2-3cc59c41668002432c019619d6951c2e_r.jpg?source=1940ef5c">
 </figure>
 <p>年报最开始的部分一定是公司管理层对集团业务、战略、市场和重点数据的解读。</p>
