# 导师遇到一个很 push 的学生是一种怎样的体验？
- 点赞数：6264
- 更新时间：2019年10月10日17时05分26秒
- 回答url：https://www.zhihu.com/question/299061588/answer/841754038
<body>
 <p data-pid="PH8CYJNi">某top N 研究生在读</p>
 <ol>
  <li data-pid="jASVZf67">我导师沉迷打游戏，每次我去找他讨论问题的时候，他都慌张的把游戏机收起来。</li>
  <li data-pid="HBFqcoT8">从来不计划自己要求学生做什么，甚至也不计划自己平时要做什么，上班就是打个游戏。我和我学弟经常跟老师说“L老师你能不能帮我看一下XXXX”，最后发现三个星期之后，ddl的前两天，他还纹丝不动（内心MMP）</li>
  <li data-pid="vY6mSceL">即便他的基金快结题了，对方不催他，他就不会整结题材料。如果哪天看他在努力写什么，一定是他被甲方爸爸骂了，并且我们不想给他擦屁股。</li>
  <li data-pid="U4KUK4GW">曾经有学弟对他嘶吼：“你整天搞些没价值的课题，然后打游戏混日子，你考虑过学生未来的学术生涯吗？”他说我们很自私，不考虑他的感受。</li>
  <li data-pid="ylzphHzp">他其实在学生时代是个学霸，但是从目前来看，为了自己未来的学术生涯，我们不得不push他。</li>
  <li data-pid="7ySd7XNC">L老师最近突然特别努力，不知道他是不是看到了我这个回答</li>
 </ol>
