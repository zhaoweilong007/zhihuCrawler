# 你的导师对你说过什么让你至今难以忘怀的话？
- 点赞数：5939
- 更新时间：2023年10月23日15时29分27秒
- 回答url：https://www.zhihu.com/question/359740428/answer/2966061733
<body>
 <p data-pid="5CpLOp1m">再说一遍:内容保真。</p>
 <p data-pid="mavnXnoE"><b>1大一篇：</b></p>
 <ul>
  <li data-pid="Pw8tc5xD">【01】</li>
  <li data-pid="-1EJw4JF">我问：老师，来这个鸟不拉屎的地方读书，看不到出路啊？</li>
  <li data-pid="iUeatEXk">师答：能退学吗？</li>
  <li data-pid="QWaqFQyd">我问：老师，我没想退学，我就是迷茫，不知道怎么办？</li>
  <li data-pid="yfy0zuBr">师答：把“我是这个学校的学生”的思维方式变成“这是我的大学”。</li>
  <li data-pid="u-qPXm0P">我问：老师，这不是车轱辘话吗？</li>
  <li data-pid="g7DPhpKD">师答：“我是这个学校的学生”的思维是以学校为准，“这是我的大学”的思维是以我为准。以学校为准，则喜怒为其所役。以我为准，则学校为我所用。学会利用学校提供的一切发展自己。</li>
  <li data-pid="ZHMl4LI8">【02】</li>
  <li data-pid="TF0ZXbSU">我问：老师，学院安排的课程太多了，其他老师讲的啰嗦无聊，咋整啊？</li>
  <li data-pid="gPwV8kOq">师答：逃。</li>
  <li data-pid="C_UhSngX">我问：老师，万一逃课被点名，期末挂科多不好啊。</li>
  <li data-pid="VQJrxJSd">师答：不是让你逃出去玩，是让你逃到图书馆，自己找书学。上课听讲效率低。</li>
  <li data-pid="w9xRMHml">【03】</li>
  <li data-pid="Rqgo5jYa">我问：老师，我想早点看咱们专业的书，行不行？</li>
  <li data-pid="C1awMeIg">师答：不行。</li>
  <li data-pid="mCdBYVLs">我问：为什么？</li>
  <li data-pid="2kKUXa4c">师答：仔细阅读培养方案，找出公共课-专业基础课-专业方向课三个层次，按这个顺序读书。</li>
 </ul>
 <p data-pid="60l5shZK"><b>2大二篇：</b></p>
 <ul>
  <li data-pid="wTxHm8k0">【04】</li>
  <li data-pid="_nRdtLcU">我问：老师，我想找个实习挣钱？</li>
  <li data-pid="0zIE_ahT">师答：不建议没有专业知识指导下的盲目实习。</li>
  <li data-pid="JQLnzz5A">【05】</li>
  <li data-pid="v9usd-WO">我问：老师，好几个女生看上我了，我该选哪个？</li>
  <li data-pid="PylqUr4n">师答：从男人的角度，都选一遍，从老师的角度，选性格互补的那个。</li>
  <li data-pid="U-EKJqa3">我问：老师，你怎么选漂亮的？</li>
  <li data-pid="z9wEDryP">师答：不好看的被我淘汰了。</li>
  <li data-pid="oz371QYc">【06】</li>
  <li data-pid="mGZXADHi">我问：老师，我竞选上社团的会长了，不知道怎么干？</li>
  <li data-pid="eh7Ex6yG">师答：别干黄了就行。社团组织可能会是你唯一一次按自己想法可劲霍霍的实验平台。</li>
  <li data-pid="91jhF0jd">【07】</li>
  <li data-pid="K8VXVSlB">我问：老师，你上课有PPT吗？</li>
  <li data-pid="jplvkkAQ">师答：讲课是人与人的交流，不是垂帘听政。</li>
  <li data-pid="2SVg5VsA">【08】</li>
  <li data-pid="RSxElIjW">我问：老师，我作业交了五遍了，怎么还不合格？</li>
  <li data-pid="_HyXoIKv">师答：为了让你长记性，训练你一次就达到标准的做事风格。</li>
  <li data-pid="vUVU7ooQ">我问：老师，我本来想按您的标准做的，但是被其他事情耽误了。</li>
  <li data-pid="alna9-Qk">师答：要给自己的成功找理由，不要给自己的失败找借口。事情只有做与不做，做的结果只有好坏，没有但是。</li>
  <li data-pid="KATYcBHN">【09】</li>
  <li data-pid="bqluJElQ">我问：老师，教师节快乐。</li>
  <li data-pid="caeABaPV">师答：好像我很郁闷似的。</li>
  <li data-pid="XaYOfCic">我问：老师，给您买了一个果篮，放在楼下大爷那了。</li>
  <li data-pid="r1yqfK3D">师答：践行我的精神就是最好的礼物。果篮就给看门大爷吧，送人东西不能要回去。</li>
 </ul>
 <p data-pid="K2tpRStg"><b>3大三篇：</b></p>
 <ul>
  <li data-pid="QAAULQy9">【10】</li>
  <li data-pid="SoS9giSO">我问：老师，我组了个团打创业比赛，想请您当指导老师？</li>
  <li data-pid="24t7g522">师答：没空，不要打着我的旗号招摇撞骗。不过，可以帮你把关，免得丢人。</li>
  <li data-pid="oEbBWfTs">【11】</li>
  <li data-pid="xHBPqhyZ">我问：老师，上次那个比赛，我们得了二等奖，想请您吃饭庆祝一下？</li>
  <li data-pid="F-qje22_">师答：二等奖有什么可庆祝的。</li>
  <li data-pid="7etyhc4x">【12】</li>
  <li data-pid="b_U8IAV6">我问：老师，昨天又偷着蹭您的课了，您还是那一套？</li>
  <li data-pid="f5NUJ8tD">师答：我昨天看到你鬼鬼祟祟溜进教室了，你也还那样。</li>
  <li data-pid="vvG_Ql9S">【13】</li>
  <li data-pid="QwipPR2C">我问：老师，周围同学说我不合群，苦恼？</li>
  <li data-pid="_GGKHFZu">师答：不合此群，便合彼群。</li>
  <li data-pid="P_VD50SV">【14】</li>
  <li data-pid="X9NosVI1">我问：老师，为什么我跟别人讲东西达不到您的效果？</li>
  <li data-pid="d7Ll2j6Q">师答：你轻易达到了，我这么多年岂不是白混了。</li>
 </ul>
 <p data-pid="oQ5Pgtjm"><b>4大四篇：</b></p>
 <ul>
  <li data-pid="6uVjOz1w">【15】</li>
  <li data-pid="OODEPXtJ">我问<b>：</b>老师，我有两个offer，要签哪个？</li>
  <li data-pid="Bhuha_ax">师答：签一个，备一个。</li>
  <li data-pid="ZWjE4s3-">【16】</li>
  <li data-pid="4EK6iBw-">我问：老师，你每天在办公室呆那么晚都在忙啥？</li>
  <li data-pid="2iJLmB6X">师答：忙着躲避中年妇女的唠叨。</li>
  <li data-pid="IIJ5MmVL">【17】</li>
  <li data-pid="ci34651d">我问：老师，毕业论文我一个字没写呢？</li>
  <li data-pid="q_Am1hkf">师答：再不写我也救不了你了。</li>
  <li data-pid="HKGImDvF">我问：老师，论文我写完了，其他老师说要您签字才能过？</li>
  <li data-pid="prV8r5Bm">师答：等我出差回来再签。</li>
  <li data-pid="WFNF760C">我问：老师，你为啥拖了我这么久才签字？</li>
  <li data-pid="7Yi13Ikb">师答：磨磨你的脾气，这是最后一课。到社会要学会收敛自己的锋芒。不要觉得委曲，男人的心是由委曲撑大的。</li>
 </ul>
 <p data-pid="ibFPPiZI">【番外】：后来听学弟说，老师每年都会在课堂上以“我有一个学生”的句式讲述我大学的事迹。可我从来没有跟别人说过这个糟老头对我的影响。</p>
