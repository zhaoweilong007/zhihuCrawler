# 适合新手的化妆品有哪些​​推荐？
- 点赞数：11589
- 更新时间：2015年05月11日01时51分21秒

<body>
 <p data-pid="6WOQyxt9">买买买是新手最头疼的问题，又花时间又花钱。</p>
 <p data-pid="njRVl5JE">买来不适合或者不会用就更惨了，然后还不敢再买，想化妆却迟迟下不去手。</p>
 <p data-pid="Tfl63vnx">你可能碰到过如下问题：</p>
 <ul>
  <li data-pid="FIPmSYfg">想买一整套化妆品，该有哪些必不可少的？<br></li>
  <li data-pid="Dy4rxSAa">粉底液粉饼散粉，唇釉唇膏唇彩，眉粉眉笔染眉膏，眼线液眼线笔眼线膏。。。我到底该选什么啊？<br></li>
  <li data-pid="uSJTeWXC">怎么才能挑到适合自己的颜色的粉底液、眼影、唇膏和腮红呢？</li>
 </ul>
 <p data-pid="T-aOB-EV">有很多彩妆教程都会告诉你，必须去一个一个试过来，或者买之前要做好多功课，这对很多有耐心又追求完美妆感的女生来说是非常必要的。</p>
 <br>
 <p data-pid="extT2yeF">但并不是每个女生都有这么多闲功夫和好耐心的。大部分女生因为怕麻烦，或者因为地理位置的原因没有办法去试妆，一旦买错了，再也不敢再下手买别的。</p>
 <br>
 <p data-pid="URUPQbGG">所以为了拯救大部分懒妹子的这个问题，为你们量身订做了一个新手购物指南，只要照着步骤做就可以。</p>
 <ul>
  <li data-pid="mi7DAWXC">不用动脑子<br></li>
  <li data-pid="mQglpO-J">不用去店里试<br></li>
  <li data-pid="0T8Ioe1-">花钱炒鸡少</li>
  <li data-pid="IvMlT1Z7">效果炒鸡好</li>
 </ul>
 <p data-pid="XBFiYIze">请按照一步一步来，不要囫囵吞枣。</p>
 <br>
 <p data-pid="2T7LPE31">*已经买了化妆品的同学可以直接查看第三步检查表，检查自己买的产品是否有问题并根据第一步和第四步补买。</p>

 <p data-pid="zHcP5Fsk">▋第三步：查</p>
 <p data-pid="MOxNjtQU">检查颜色：</p>
 <ul>
  <li data-pid="Yn19IJ3x">比较4种色号的口红，你更适合哪种？这就是你以后可以长期坚持买的安全颜色类型了。</li>
  <li data-pid="7agkhzYu">比较2种眼影，是粉色的还是大地色你涂上去看着眼神更干净？这就是你以后的安全眼影参考色了。</li>
  <li data-pid="8zGQQ3_u">比较2种腮红，是橘色还是粉色让你看上去气色更好？这就是你以后的安全腮红参考色了。</li>
 </ul>
 <p data-pid="0dE6LFLw">检查问题：</p>
 <p data-pid="kctWfwgg">如果回答是“是”请看第四步对应数字的内容。</p>
 <ul>
  <li data-pid="3pb66k4L">粉底用后脸变黑了或者感觉掉进面缸子里（请根据色号图提示换色号）</li>
  <li data-pid="GJyOacWM">粉底涂不均匀而脏脏的？（-&gt;1）</li>
  <li data-pid="p5ArUjX2">眼影试过的两种颜色都脏脏的？（-&gt;2）</li>
  <li data-pid="AvU0v2gn">唇膏涂布均匀，尤其是唇线处感觉不整齐？（-&gt;3）</li>
  <li data-pid="1kyxfyx0">按照教程涂完底妆以后有没有感觉脸像一张大白饼特别平和大？（-&gt;4）</li>
  <li data-pid="CBlcCSCQ">粉底涂够量了但仍没有盖住脸上的红血丝，感觉红扑扑？（-&gt;5）</li>
  <li data-pid="L2X-vlMj">粉底色号和皮肤一样，但感觉脸色蜡黄？（-&gt;6）</li>
  <li data-pid="r7H_Brbs">粉底色号和皮肤一样，但感觉脸色苍白？（-&gt;7）</li>
  <li data-pid="9HUYElaY">是否仍然有明显的痘坑、黑眼圈、雀斑、嘴角暗沉？（-&gt;8）<br></li>
  <li data-pid="CMfszg8s">是否有明显毛孔？(-&gt;9)</li>
  <li data-pid="piOYBdE4">4小时后再检查你的妆容，是否有起皮或者干纹？（-&gt;10）</li>
  <li data-pid="-pCt6HwB">4小时后再检查你的妆容，是否有脱妆或者暗沉？（-&gt;11）</li>
  <li data-pid="ijx16sJL">4小时后再检查你的妆容，眼妆是否有晕妆？（-&gt;12）</li>
 </ul>

 <p data-pid="C7zBnBIP">对照检查问题表再购买：</p>
 <ol>
  <li data-pid="neIPrbaA">Everyday粉底刷（性价比最高的新手刷）<br></li>
  <li data-pid="H_gN8oma">艾诺琪小号眼影刷+眼影晕染刷+斜角眼线刷（根据自己经济情况选择价位即可）<br></li>
  <li data-pid="9AFlCGb4">黄狼毛唇刷（没有特别推荐的，买的个挺便宜的也好用）<br></li>
  <li data-pid="LuYDLZKG">双色修容粉（KATE双色V字脸修容修颜高光粉饼）<br></li>
  <li data-pid="W3lWWPs3">绿色饰底乳/隔离霜（悦诗风吟）<br></li>
  <li data-pid="_BTfQ37w">紫色饰底乳/隔离霜（悦诗风吟）<br></li>
  <li data-pid="Tpl3Ccd6">粉丝饰底乳/隔离霜（悦诗风吟）<br></li>
  <li data-pid="nwxH-qvc">遮瑕膏（ipsa三色）<br></li>
  <li data-pid="hntcu-ZX">遮盖毛孔的妆前乳（too cool for school rules of pore, 贝玲妃猪油膏/反孔精英，Smashbox等）<br></li>
  <li data-pid="7RqPa0ZO">保湿妆前乳（Laura Mericer Hydrate）或荷荷巴油（<a href="http://zhuanlan.zhihu.com/nvshenjinhualun/19952072" class="internal">无瑕裸底妆全攻略</a>）<br></li>
  <li data-pid="GTJCvP51">散粉（Laura Mericer，嫌贵就买KATE自然型的散粉，KATE的散粉因为没有被炒作过，效果好又没有假货）</li>
  <li data-pid="KHPPuT0j">e.l.f的眼部打底霜（质量更好的用Urban Decay）</li>

 <p data-pid="_t9ouIx3">▋第五步：换</p>
 <p data-pid="4KuBM6m-">把不合适的化妆品都和别人换掉吧（眼影、腮红、口红），或者二手卖掉好了。</p>
 <br>
 <p data-pid="mvhTmhr-">▋Tips</p>
 <ul>
  <li data-pid="Lzug9a27">为什么推荐这些产品？（眼影、口红都是挑的便宜颜色正质量也还行的，主要目的是试色，试完色以后可以根据自己的喜好去买贵一些的口红。 ）</li>
  <li data-pid="o7QHR481">粉底如果有专柜仍然需要去专柜挑选最合适，由于地理原因没有专柜的再用这种网购方法挑选。<a href="http://zhuanlan.zhihu.com/nvshenjinhualun/19947387" class="internal">3步挑选完美粉底液 </a><br></li>
  <li data-pid="G61zU5U3">为什么推荐水粉霜？因为水粉霜非常自然，遮盖力最小，即使买错了色号影响并不会像别的粉底液那么大。适合干皮用，但据油皮反应脱妆和闷痘也很少发生。</li>
  <li data-pid="hhebps-P">关于粉底液色号：20#色是选用最多的，如果你的肤色一般人的话就买20#基本没有问题。（寺主人已经给肤色深浅不同的妹子都试过这个号，大家涂上以后都感觉不错。）</li>
  <li data-pid="V16dWzBu">为什么推荐中样？1.便宜，出错了也损失不大 2.新手很多都不频繁化妆用不掉。</li>
  <li data-pid="LTjik2kj">如果是学生党，强烈推荐眼影、腮红唇膏宿舍一起买一套然后都试试，瓜分一下。</li>
  <li data-pid="97TXq2Tr">试色一定要和朋友一起来尝试，他人眼中的建议很重要。<br></li>
  <li data-pid="M4tIZEJZ">
   <p data-pid="-I3mwjPw">已经买了化妆品的同学可以直接查看第三步检查表，检查自己买的产品是否有问题并根据第一步和第四步补买。<br></p></li>
 </ul>
 <p data-pid="Ps5-SYtX">指南根据个人经验总结，难免存在个人偏好、遗漏以及在”操作简单“、”价格合适“和“完美效果”中间进行纠结的权衡。</p>
