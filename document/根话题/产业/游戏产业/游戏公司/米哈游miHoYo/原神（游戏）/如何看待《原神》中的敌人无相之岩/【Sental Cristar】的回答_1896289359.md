# 如何看待《原神》中的敌人无相之岩?
- 点赞数：5768
- 更新时间：2021年05月20日15时17分33秒
- 回答url：https://www.zhihu.com/question/460131449/answer/1896289359
<body>
 <p data-pid="re1KxUtM">错误的五香岩打法：口袋锚点传送，30s搞定。</p>
 <p data-pid="Xb4mVJgF">正确的五香岩打法：</p>
 <ol>
  <li data-pid="j798aRja">传送到岩本门口</li>
  <li data-pid="Mu6nPK8W">抓3只蝴蝶，拾取1个石珀，2个鸟蛋，2个蘑菇</li>
  <li data-pid="gxgZwgB4">跳下正面的断崖抓一只蝴蝶</li>
  <li data-pid="IcxNJDW0">左拐，打死雷莹术士</li>
  <li data-pid="y1-NzZve">挖矿</li>
  <li data-pid="U6xPpBO-">到前边的岛上挖矿</li>
  <li data-pid="mx2UEkfS">游泳过海，挖矿，打死龙蜥</li>
  <li data-pid="BN9IFiy-">游泳回来，打死F4（4个铁甲小宝）</li>
  <li data-pid="ts6vIknG">传送回岩本门口，左转出门</li>
  <li data-pid="6h0a56A_">门口抓2只蝴蝶</li>
  <li data-pid="zr_tOAZx">爬上坡，如果有蝴蝶就抓2只</li>
  <li data-pid="rBq2Q5tI">在歪脖子树那里拾取2个鸟蛋</li>
  <li data-pid="zx8IIj5W">拾取2个石珀，打死3个法师（水火冰各一个）</li>
  <li data-pid="9vSZJzoS">跳下山，打死火法师1个，牛头人2只，弓箭手1个</li>
  <li data-pid="efr4AmrT">拾取石珀一个</li>
  <li data-pid="IzdeLAtZ">看心情打死周围部落居民若干</li>
  <li data-pid="b1-xofoW">花费30秒击败无相之岩</li>
  <li data-pid="55tZpe-7">传送到旁边的传送点</li>
  <li data-pid="ozh5w0g6">爬到石头上，拾取2个琉璃袋，1个鸟蛋</li>
  <li data-pid="Xi403wvu">跳下石头，挖矿</li>
  <li data-pid="0RVM2M64">左转，拾取石珀一个</li>
  <li data-pid="zHfnGDnQ">收工</li>
 </ol>
