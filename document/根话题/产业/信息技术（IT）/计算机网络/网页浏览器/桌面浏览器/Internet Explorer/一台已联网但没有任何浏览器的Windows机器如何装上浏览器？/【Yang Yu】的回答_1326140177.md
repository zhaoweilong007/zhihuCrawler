 <p data-pid="7zU35nk8">打开 PowerShell</p>
 <div class="highlight">
  <pre><code class="language-powershell"><span class="n">PS</span><span class="p">&gt;</span> <span class="nv">$url</span> <span class="p">=</span> <span class="s2">"https://go.microsoft.com/fwlink/?linkid=2108834&amp;Channel=Stable&amp;language=zh-cn"</span>
<span class="n">PS</span><span class="p">&gt;</span> <span class="nv">$filePath</span> <span class="p">=</span> <span class="s2">"$env:userprofile\Downloads\MicrosoftEdgeSetup.exe"</span>
<span class="n">PS</span><span class="p">&gt;</span> <span class="nb">Invoke-WebRequest</span> <span class="n">-Uri</span> <span class="nv">$url</span> <span class="n">-OutFile</span> <span class="nv">$filePath</span>
<span class="n">PS</span><span class="p">&gt;</span> <span class="p">&amp;</span> <span class="s2">"$env:userprofile\Downloads\MicrosoftEdgeSetup.exe"</span></code></pre>

 <p data-pid="OLrNiQQ8">补充，那要是没有 PowerShell 或者 PowerShell 版本小于 3.0 怎么办</p>
 <p data-pid="cf7YsN0_">新建一个 .vbs</p>
 <div class="highlight">
  <pre><code class="language-aspx-vb">Set winHttp = CreateObject("WinHttp.WinHttpRequest.5.1")
url = "https://go.microsoft.com/fwlink/?linkid=2108834&amp;Channel=Stable&amp;language=zh-cn"
winHttp.open "GET", url, False
winHttp.send ""
Set adodbStream = CreateObject("ADODB.Stream")
with adodbStream
    .type = 1
    .open
    .write winHttp.responseBody
    .savetofile "MicrosoftEdgeSetup.exe", 2
end with
msgbox "Download completed"</code></pre>
