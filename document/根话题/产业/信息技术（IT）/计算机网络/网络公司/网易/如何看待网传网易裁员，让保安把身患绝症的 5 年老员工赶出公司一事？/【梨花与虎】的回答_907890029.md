# 如何看待网传网易裁员，让保安把身患绝症的 5 年老员工赶出公司一事？
- 点赞数：53435
- 更新时间：2019年12月30日12时26分18秒

<body>
 <p data-pid="Y7VkWbBU">《HR不花一分钱解雇员工实操全网最全手册》</p>
 <h2>初阶版</h2>
 <ul>
  <li data-pid="EEOORFfQ">甭管员工表现怎样，绩效考评一律给D（配合措施：劳动合同中约定，绩效考评连续三个月打D，则解除劳动合同，没有补偿金。）</li>
 </ul>
 <blockquote data-pid="Ou2nCjSx">
  3月底主管找我谈绩效，说他跟代理主管聊过了，这次准备给我评D绩效，因为我现在不适合在这里继续工作了。当时我很懵，因为我和代理主管每天吃饭的时候都在讨论项目的开发方向，很多重要的功能也都是我俩加班讨论确定的，重要功能他也都交给我做，组内的业绩排名也基本是他第一我第二。
 </blockquote>
 <ul>
  <li data-pid="3fOIy36I">诱逼员工主动申请离职，威胁“拿补偿金不利于以后找工作”。比如“你的下家做背景调查时我们会说你坏话”，又如“在离职证明上写不利于员工的离职原因”。</li>
 </ul>
 <blockquote data-pid="Tb98GNiJ">
  我不愿发申请，主管和HR就轮番找我谈话逼我，并且变相威胁说拿N+1的话会对我非常不利，句句都是以“怕影响我找下一份工作”的角度劝我不要拿N+1。
 </blockquote>
 <ul>
  <li data-pid="omFTziZd">如果员工识破了这个阴谋，拒绝签离职申请，就撤掉员工工位，收回电脑，解除门禁权限，从打卡机里删掉员工，让他打不了卡、上不了工、甚至进不了公司；（某福报公司用过这一招）</li>
  <li data-pid="-IEGbic2">骗员工说，特批你从明天开始不用来公司打卡了，方便后续以“旷工”理由辞退；</li>
  <li data-pid="vuvhS8KR">诱逼员工请年假，不请就给员工算旷工（此处存疑：这他妈是什么鬼操作？法务表示，无法律依据啊）</li>
  <li data-pid="BXvcaoVQ">如果战况进入拉锯战，则发动扯皮攻击。往员工身上泼脏水，让他背锅，让他委屈，让他心累，让他绝望，陷入跳进黄河洗不清的境地，则有望一举击溃其心理防线，精疲力竭之下签字。</li>
 </ul>
 <blockquote data-pid="x-Lrw9F1">
  被我反驳后主管又强行挑刺说我有一个bug，我反驳说这个bug是接口bug，不是我的bug，而且影响很小。结果主管说所有的问题都是策划的问题，问我同意吗。真心觉得自己患病后还那么卖力地工作没有一点意义。
 </blockquote>
 <ul>
  <li data-pid="PW3ZdleM">安排不明身份人员在员工工位附近往来监视，给对方制造心理压力。人在重压之下就会有漏洞，趁机收集他违反公司规定的证据。</li>
 </ul>
 <h2>进阶版</h2>
 <ul>
  <li data-pid="WB0ciqle">打电话、寄信件给员工家人，让家人知道员工摊上事儿了，丢了工作了，给员工制造更大心理压力。（<b>良心警告：</b>心不狠不要用这招。中国人的信仰是家庭，擅用家人威胁别人，有可能引发匹夫之怒，血溅五步，让自己家宅不宁、日夜难安。）</li>
 </ul>
 <blockquote data-pid="iwc5Nwc9">
  “如果你觉得现在收不方便，那我们寄回你老家，让你家人帮你收也可以的。”我想如果他们也有父母的话，应该也会知道父母年龄大了受不了刺激。
  <br>
  于是HR给我母亲打电话，说我生了这个病就没法继续在这工作了。HR说让我母亲劝我，否则他们也不愿意看到保安来清退我。第二天一早，又有工作人员来敲门，拿着电话说有关部门人员要找我母亲，问我母亲我的情绪怎么样，跟我母亲说我现在还没离职，让看好我不要做出什么事来。我母亲怕出事，于是劝我说要不主动签字算了。
 </blockquote>
 <ul>
  <li data-pid="0MUR952F">骚扰员工，让他难以安宁，心力交瘁。这招对付生病、怀孕或在哺乳期的员工特别有效。</li>
 </ul>
 <blockquote data-pid="RZDFl3tJ">
  结果HR说“收个稿件你觉得会影响你养病？”
  <br>
  主管回复说必须接听HR电话才能继续休病假。
 </blockquote>
 <ul>
  <li data-pid="jWQF1Uy8">修改员工考勤记录，无中生有制造迟到、早退、旷工。</li>
 </ul>
 <blockquote data-pid="oUFnlV_B">
  结果7月底，我上系统看了一眼，发现自己竟平白多出了4次早退记录。5月10号是我请病假前在公司的最后一天，可以看到，当时还没有早退记录。而5月10号开始我一直在休病假。所以这4次早退记录到底是从医院早退，还是从家里早退？
 </blockquote>
 <h2>高阶版</h2>
 <ul>
  <li data-pid="tHWE7rx4">收集员工的QQ签名、朋友圈动态，如果出现“此去泉台招旧部，旌旗十万斩阎罗”之类诗词，就向有关部门举报员工有反动和自杀倾向。（某电脑生产商在应对顾客质量维权时用过类似手段，让顾客坐牢十个月，成功化解危机；某爱国厂你怎么肥四，不知道的朋友怕不是要赖我把你们教坏了鸭！）</li>
  <li data-pid="ygkfSi3F">如果纠纷被捅到董事局等高层处，事态有恶化可能，则建议把员工交给法务和专业人员，建议员工接受由公司免费提供的心理咨询，或建议员工去指定医疗机构检查，方便收集对公司有利的证据。</li>
 </ul>
 <h2>后续版</h2>
 <ul>
  <li data-pid="GFCe-01y">即使公司在仲裁中失败，也通过不断上诉拖延执行。（此处为引用当事人引用的HR表述关于「一裁终局」的表述删除，关于劳动纠纷仲裁和诉讼的详细条文见评论区置顶的知友表述）</li>
 </ul>
 <blockquote data-pid="9D6h0AXi">
  于是接下来HR找我谈话时，换了更隐晦的威胁方式。跟我说<b>仲裁就算我赢了，公司也会通过不断上诉拖我两年时间</b>，我耗不起的。
 </blockquote>
 <ul>
  <li data-pid="e3pbMKCr">如果事态有曝光到媒体、彻底失控的可能性，则同意法定补偿金额，但分12个月发放，因为12个月正好是员工劳动维权的有效期。</li>
 </ul>
 <blockquote data-pid="Zx5hX5W2">
  不过他们承诺如果同意签字，可以给到高于N+1的金额。关键是要拿这笔钱是有附加条件的，要分12个月拿。这意味着不仅要放弃后续一切的法律途径维权，而且变得没有保障。
 </blockquote>
 <ul>
  <li data-pid="nVU5aA00">补偿金和工资拖延发放，让员工往来奔波，求告无门，疲于维权。</li>
 </ul>
 <blockquote data-pid="qJ1WAhUF">
  在公司后来给出的单方解除通知书上，明确写着N+1和应发的工资会在9月9日前发放。然而9月9日我没有收到一分钱。最后N+1还是我去申请劳动仲裁后才拿到的。（其他同事的大额赔偿金也有被公司一直拖着不给，去申请了仲裁公司才给的。）
 </blockquote>
 <ul>
  <li data-pid="TfAtqnNx">不配合员工办理失业金。不但让他失业，还让他失业金也领不到。</li>
 </ul>
 <blockquote data-pid="ljVR0HYF">
  离职证明公司也拖了一个半月不给我，即便我找劳动监察投诉，又跟HR说等着离职证明要领失业金，HR还是跟我说他们要考虑给不给我。
  <br>
  之后好不容易拿到离职证明去办理失业登记，又被工作人员告知公司开的证明不合规，于是我又联系HR说明了马上就要过了失业金办理期限，让帮忙盖公司公章。
  <br>
  结果HR回复我说她请示过了，因为我和公司对于解除劳动关系的理由存在争议，所以不能给我盖公章。言外之意就是我要放弃劳动仲裁，他们才肯给我盖章。
 </blockquote>
 <ul>
  <li data-pid="ow5jDMYZ">不提供工资条，或仅提供一个数额，失去证据效力。</li>
 </ul>
