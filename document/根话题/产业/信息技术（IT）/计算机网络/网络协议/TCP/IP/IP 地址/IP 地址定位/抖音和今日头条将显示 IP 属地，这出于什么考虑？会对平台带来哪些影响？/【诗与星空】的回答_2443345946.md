# 抖音和今日头条将显示 IP 属地，这出于什么考虑？会对平台带来哪些影响？
- 点赞数：5636
- 更新时间：2022年04月17日09时20分41秒
- 回答url：https://www.zhihu.com/question/528090210/answer/2443345946
<body>
 <p data-pid="bP4ein8b">对这件事不做详细评价。</p>
 <p data-pid="EFJwc9TL">仅对“<b>今天显示省份，明天就有可能精确到市区甚至街道</b>”这句格式化明显的群发句子做一下评判。</p>
 <p data-pid="NoPWIVzJ">1、但凡是个真正的大陆老网民，就知道早期BBS是显示IP和地理位置的，只不过最后一位用星号代替。</p>
 <p data-pid="lOOnbKuq">现在显示IP属地不过是恢复而已。</p>
 <p data-pid="xUZ1q23R">2、但凡是个真正的大陆网民，都知道现在所有APP都是实名注册的。</p>
 <p data-pid="03sK_ZEz">对于监管部门的管理人员来说，早就精确到身份证手机号了，精确到市区、街道对于监管部门的管理者来说没有任何难度。监管部门拿到这些资料后，就没有必要在平台上精确显示。</p>
 <p data-pid="31qeejp7">已经被监管部门拿到的资料，你担心的是什么？</p>
 <p data-pid="5qS_AHq1">3、让我感到震惊的是，竟然还有认证信息为法律专业的答主也使用这句格式化群发句子。</p>
 <p data-pid="3w-2i6TC">中国的法律界人士，是如何做到对中国互联网企业的格式化条款一无所知的？</p>
</body>
