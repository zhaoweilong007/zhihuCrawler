# 如何巧妙地使用旅游类 App？
- 点赞数：5230
- 更新时间：2019年06月26日14时14分29秒
- 回答url：https://www.zhihu.com/question/30469466/answer/48203804
<body>
 <p data-pid="ZzQiJVET">作为一个不折腾会死星人，每年旅行是必须的；作为一个非常事儿多的细节控，选择自助游是妥妥的~为了让旅行的时候省心点儿，每次出游时我会利用互联网资源做好很多准备工作。以前常用一些旅行网站、论坛，现在基本都有推出APP，使用起来就更方便了。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="o6i5mlsR">我的旅行按行前准备、旅行途中、旅行结束会分别用到很多不同的APP，截图中留下的这些是我现在常用的，介绍里出现但是本截图中没有的并不是其本身不好，通常是因为我觉得用着不（长）顺（得）手（丑）就删了。</p>

 <p data-pid="WMSeTV4P"><b>一、行前准备</b></p>
 <p data-pid="IUC271oo">在确定旅行目的地之后，我会罗列出目的地的主要景点以及个人感兴趣的特色景点，根据旅行天数、出行季节的不同，参考其他人的游记、攻略后合理确定行程。机票、酒店会在行程确定后预定或与行程安排同时进行。</p>
 <ul>
  <li data-pid="QqpLCGpR">游记攻略——穷游、蚂蜂窝、蝉游攻略<br></li>
 </ul>
 <p data-pid="EDi1eNOS">穷游的攻略类最好，蚂蜂窝的阅读体验比较好。</p>
 <ul>
  <li data-pid="InisiEBF">行程安排——穷游助手<br></li>
 </ul>
 <p data-pid="LPiQ6FEZ">穷游的行程助手可以导出PDF和EXCEL，也可以在APP中直接修改、查看，非常方便。</p>
 <ul>
  <li data-pid="OAPn5iFN">机票比价、预定——去哪儿、天巡<br></li>
 </ul>
 <p data-pid="7kktEz3i">这一项去哪儿的优势还是非常明显的，同时间同航线基本都会比携程艺龙更便宜。境外机票会同时参考天巡。</p>
 <ul>
  <li data-pid="JckoLai6">特价信息——E旅行网<br></li>
 </ul>
 <p data-pid="IrOYz2ot">特价信息全面且更新非常及时，特别适合假期灵活的同学。</p>
 <ul>
  <li data-pid="x7A7Ah_e">酒店比较、搜索——tripadvisor（到到网）<br></li>
 </ul>
 <p data-pid="_--rn2Kc">到到网的评价还是比较客观的，参考价值很大。可以看到世界各地入住者的评语，比单看预定网站客观许多，还可以横向纵向各种对比，是选择障碍症患者的好盆友~</p>
 <ul>
  <li data-pid="nkMgnKpD">酒店比价、预定——去哪儿、booking、airbnb<br></li>
 </ul>
 <p data-pid="oMrZvoyK">和机票一样，酒店比价这一块去哪儿是很有优势的，境内酒店主要用这个，境外酒店会同时参考booking。</p>
 <p data-pid="tycBr_SA">住腻了标准化的酒店，新宠是民宿bnb，在境外尝试了几次体验很好，在国内也开始体验了，同样也还不错。</p>
 <ul>
  <li data-pid="ratEu7ah">目的地一日游预定——klook</li>
 </ul>
 <p data-pid="qculI0WT">无论是买迪士尼门票，还是海岛一日游都可以从Klook app来定，还有很多优惠的套票～或者在当地不知道玩什么的时候，来看看其他人的选择和评价，省心而且性价比特别高，不用去某宝众多第三方卖家里仔细分辨，很安心。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="z6e82Xo8"><b>二、旅行途中</b></p>
 <p data-pid="ltteudlb">旅行中使用的APP，很多都是生活服务类的，不仅仅是旅行类。每个人的习惯不同，就不多评价了。比较下来，我个人习惯用的主要是下面这些：</p>
 <ul>
  <li data-pid="ox6Y906X">航班动态——航旅纵横<br></li>
  <li data-pid="OHAbTkbq">地图路线——google离线地图、citymapper<br></li>
  <li data-pid="yAURj7Vk">天气预报——墨迹天气<br></li>
  <li data-pid="cVo9m_Yr">翻译词典——有道翻译官、旅行翻译官<br></li>
  <li data-pid="Pl2MozB8">滤镜修图——snapseed、instagram<br></li>
  <li data-pid="_vigXHw5">旅行社交——蚂蜂窝嗡嗡</li>
  <li data-pid="KmgH8-r1">货币转换——imoney、Currency</li>
  <li data-pid="s2cRzMEn">目的地玩法——携程当地、穷游折扣<br></li>
 </ul>
 <p data-pid="Fnk3ZAd5">除此之外，时差、货币转换等等也可以自行选择用着顺手的APP（我暂时还没发现相对完美的，求推荐！）</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="b5YgoDpl"><b>三、旅行结束</b></p>
 <p data-pid="MFemJScx">旅行结束后趁着热情可以写写游记、攻略，一方面是对自己的旅行进行总结和纪念，一方面也是分享经验造福大家~</p>
 <ul>
  <li data-pid="JfXoB22Q">如果你的游记以分享攻略为主，推荐使用穷游、蚂蜂窝。<br></li>
  <li data-pid="88eaGCLv">如果你的游记以分享美图为主，推荐使用去哪儿旅图。<br></li>
 </ul>
 <p data-pid="o-Q1uZmu">穷游、蚂蜂窝、去哪儿、携程等都推出了系列应用对旅行中的各类需求进行了全覆盖，但各自旗下依然有其主打的特色应用，<u>比如穷游的行程制定、蚂蜂窝嗡嗡的社交功能、去哪儿的搜索及信息整合功能等等都是同类应用中的佼佼者，</u>非常推荐。</p>
