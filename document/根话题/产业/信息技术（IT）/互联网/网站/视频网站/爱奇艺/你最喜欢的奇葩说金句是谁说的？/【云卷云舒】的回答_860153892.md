# 你最喜欢的奇葩说金句是谁说的？
- 点赞数：6312
- 更新时间：2019年12月05日15时17分54秒
- 回答url：https://www.zhihu.com/question/35831057/answer/860153892
<body>
 <p data-pid="BlmDX3EY">我追了一部综艺节目5年，期期不落，这里面有一个宝藏男孩，特别感谢他，让我见识到了一个全新的思想境界。让我学会了思考，让我学会了反省，让我学会了接纳，让我知道了这个世界上原来有那么多不同的人，那么多值得我们学习的地方，让我知道了一件事情的很多不同的看法，他让我知道了不要轻易对一件事情和一个人下定论，为之感慨。他说：“浩瀚宇宙从归零开始”。</p>
 <p data-pid="OThVOu41">东边来的一匹马语录：</p>
 <ol>
  <li data-pid="uW-azZ34">我们这个世界上大约只有5%的人，有愿望，积累知识，了解过去，创造文明。而剩下95%的人呢？他们活着，仅仅就只是在生活。</li>
  <li data-pid="mva2cbad">人要有归零的精神，空的瓶子才能装水，每隔几年就全面的归零，这样活 着才更有意义。</li>
  <li data-pid="2EjgCPAS">心里有很多苦的人，只要一丝甜就能填满。</li>
  <li data-pid="gmoJe3Oo">被误会是表达者的宿命，我们没什么可抱怨的。</li>
  <li data-pid="E0cPlYhq">中国有一个成语叫刻舟求剑，我特别喜欢这个成语，因为我们即便对这个成语再熟悉，我们经常会容易成为那个人，我们特别容易忘记自己是在一条河上面，我们特别容易记得自己是在一条船上面。</li>
  <li data-pid="2VCCsDzc">今年是我父亲去世十周年。他是突然去世的，大家都知道，亲人的突然离去会给人带来特别大的打击。大概三年后，我做了一个梦。我梦见我父亲跟我说，直到今天我才真正的走了，很高兴和你做一世父子，有缘再聚。我不信，我不信那是我父亲跟我说的。因为在我的印象里，他十年前就走了，在一呼一吸之间离开了我们。所以我知道那是我潜意识里，自己放掉了自己。</li>
  <li data-pid="l1r_9nQo">我们自以为做出了公正的判断，其实往往都带有自己的主观色彩。不要做他人生活的审判者。</li>
  <li data-pid="0HuXErwI">文化是沉淀的结果，所以追求文化是本末倒置。</li>
  <li data-pid="wxdZrtIp">生而为人，这是必须面对的事啊，无法阻止，所以也不必害怕。我们共同度过的岁月，一起制造的回忆，这些都是陪伴我走下去的勇气。感谢命运让我们来到彼此的生命里。</li>
  <li data-pid="yMkVByUs">曹雪芹在整部《红楼梦》里最喜欢的是姑娘，最讨厌的是婆子，所以姑娘要写死 ，因为只要她还活着 ，她就会变成婆子。</li>
  <li data-pid="rgeR-6wv">罗:“脏话是人的情绪到了尽头。” 马:“不是，人的情绪到了尽头是沉默。”</li>
  <li data-pid="J0Lr5A5z">青山不老，为雪白头，我们不知道青山何时所始，何时所终，但不耽误青山在我们心中的美。</li>
  <li data-pid="c256Tam0">这个世界不会好，但这个世界也不会更坏了。因为这个世界，从成为世界的那一天开始，就和现在一模一样。”</li>
  <li data-pid="ZY6gaE7n">所谓的完美人生，就是中国人说的命好。生在一个无饥馑之忧之家，小时候即便遇见了危险，也大难不死，必有后福。读书的时候，她虽然不是最聪慧的，但在中游之上。及长，她爱她所爱，居然也能追到她爱的那个人，但却未必跟他走进婚姻。即使，她有朝一日结婚，但一日情尽，两个人可各奔东西，但依旧是好朋友。她工作当中不是没有困难，但往往能够得到自己想要的结果。在我看来，这就是完美人生，就是命好。</li>
  <li data-pid="7MbbtDXR">什么是教育成果？当你完成教育过程，把所学的全忘了的时候，剩下的就是教育成果。</li>
  <li data-pid="czy6cvBj">追求幸福圆满应该坦诚，追求自虐兼虐人的就坦白，此乃鄙人心中坦白与坦诚的区别。我们要的是恋爱终得圆满，能坦诚相待求得幸福，何苦坦白相对劫后余生呢。</li>
  <li data-pid="TVhdizIe">活这个字，是三点水旁的，它是流动的。什么叫活着，就是你还会变化。</li>
  <li data-pid="oftJ9F9r">所有人都是自己的所有经历叠加到一块儿，成为了今天的自己。</li>
  <li data-pid="V9sjWQbH">只有做到最大程度的不一样，才能做到最大程度被需要。</li>
  <li data-pid="nBt_FQbZ">任何一件事从不同的角度切入，都会有不同的观点和认知，甚至是截然相反的论断。</li>
  <li data-pid="8JqcLk67">正因为婚姻如此美好，才不应该被催，尤其不应该被催着去做什么</li>
  <li data-pid="hQ_KNa0_">人生最好的态度就是把自己放没有</li>
  <li data-pid="14AWruLd">不是时间捆绑你的人生，是你的执念，恐惧，妄念会捆绑你的人生</li>
  <li data-pid="n6IYB2g1">每一个人都步步是绝境，那个绝境被触碰到的时候，就是人最深刻思考，或者多去想一想的那个时候。</li>
  <li data-pid="u2Ye823B">梦想在这里不是用来实现的，而是用来被调整的。</li>
  <li data-pid="0TRPrfvw">世事洞明，却不以世故待人。</li>
 </ol>
