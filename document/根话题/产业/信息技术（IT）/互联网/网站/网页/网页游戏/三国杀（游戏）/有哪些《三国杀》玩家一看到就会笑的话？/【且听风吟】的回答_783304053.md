 <h2>【标准包】</h2>
 <ul>
  <li data-pid="tL5EmOCe"><b><i>刘备</i></b></li>
 </ul>
 <p data-pid="5cvjD_7f">『仁德』：出牌阶段，你可以将任意张手牌交给其他疑似队友角色并和真正队友对喷</p>
 <p data-pid="rgCUuYg7">『激将』：主公技，每当你需要使用或打出一张杀时，你可以大叫一声快帮我出杀</p>
 <ul>
  <li data-pid="4WwaGXsk"><b><i>张辽</i></b></li>
 </ul>
 <p data-pid="08tbFNx4">『突袭』：摸牌阶段，你可以大叫一声拿来吧！然后从两名其他角色手牌中各获得一张垃圾牌</p>
 <ul>
  <li data-pid="3c-1wL8p"><b><i>许褚</i></b></li>
 </ul>
 <p data-pid="7vi4-sf-">『裸衣』：摸牌阶段，你可以大叫一声脱！然后获得牌堆顶的一张闪。出牌阶段，你可以对手牌中有闪的角色使用一张杀，或对手牌中杀比你多的角色使用一张决斗</p>
 <ul>
  <li data-pid="YXEjFW0t"><b><i>夏侯惇</i></b></li>
 </ul>
 <p data-pid="UfWpXr8q">『刚烈』：每当你受到伤害后，你可以展示牌堆顶的红桃牌并弃置之</p>
 <ul>
  <li data-pid="c66oqxfm"><b><i>司马懿</i></b></li>
 </ul>
 <p data-pid="0OdjLo5n">『反馈』：每当你受到伤害后，将一张乐不思蜀置入你的判定区</p>
 <p data-pid="EnbOOGLS">『鬼才』：每当你的手牌中没有红桃牌/梅花牌时，将一张乐不思蜀/兵粮寸断置入你的判定区</p>
 <ul>
  <li data-pid="XXczsRD0"><b><i>曹操</i></b></li>
 </ul>
 <p data-pid="cS4jxpp_">『奸雄』：每当你受到伤害后，将一张乐不思蜀置入你的判定区</p>
 <p data-pid="ENcVFCKS">『护驾』：当你需要使用或打出一张闪时，你可以请求反贼魏国角色出一张闪</p>
 <ul>
  <li data-pid="4HiW-plX"><b><i>郭嘉</i></b></li>
 </ul>
 <p data-pid="fdsdXBuB">『天妒』：当你判定区内乐不思蜀的非红桃判定牌生效后，你可以获得此牌</p>
 <p data-pid="vI9GQkLp">『遗计』：每当你受到一点伤害后，你可以获得两张垃圾牌，然后将一张乐不思蜀置入你的判定区</p>
 <ul>
  <li data-pid="_aCSZJLV"><b><i>华佗</i></b></li>
 </ul>
 <p data-pid="A_bLqk-A">『急救』：锁定技，你只能摸到黑色牌</p>
 <p data-pid="pLHv6pt7">『青囊』：锁定技，每当你拥有手牌时，你成为兵粮寸断，过河拆桥和顺手牵羊的目标</p>
 <ul>
  <li data-pid="yh1F2x_V"><b><i>孙尚香</i></b></li>
 </ul>
 <p data-pid="7DzhoK-Y">『枭姬』：锁定技，每当你摸到装备牌，你可以失了智地使用</p>
 <p data-pid="slNdi3yh">『结姻』：你可以看着全场满血男性角色骂娘</p>
 <ul>
  <li data-pid="WvWYfWt0"><b><i>甄姬</i></b></li>
 </ul>
 <p data-pid="rQCYe6s9">『洛神』：准备阶段开始时，你可以展示牌堆顶的一张红色牌并弃置之</p>
 <p data-pid="ZIuKu5M-">『倾国』：当你拥有黑色牌时，将一张贯石斧置入敌方装备区</p>
 <ul>
  <li data-pid="WTRSaLFc"><b><i>貂蝉</i></b></li>
 </ul>
 <p data-pid="eDGaLQiZ">『离间』：锁定技，当你出场时，场上至多只能有一名男性角色</p>
 <p data-pid="PfYysSWx">『闭月』：下家的摸牌阶段结束后，你可以大叫一声卧槽忘摸牌了</p>
 <ul>
  <li data-pid="o4EjRSCU"><b><i>吕布</i></b></li>
 </ul>
 <p data-pid="012NcrcJ">『无双』：每当你使用一张杀时，目标角色依次使用两张闪。每当你使用一张决斗时，目标角色依次使用两张杀并对你造成一点伤害</p>
 <ul>
  <li data-pid="TchctGmG"><b><i>华雄</i></b></li>
 </ul>
 <p data-pid="Nto9xtl1">『耀武』：每当你没有闪时，其他角色的红色杀只能指定你为目标</p>
 <ul>
  <li data-pid="5D-WcWhR"><b><i>陆逊 </i></b></li>
 </ul>
 <p data-pid="Sz6gNSSh">『谦逊』：每当你的判定阶段结束后，你可以大叫一声卧槽谁给我放的乐。每当你因顺手牵羊而失去手牌后，你可以悻悻没有被抽走好牌</p>
 <p data-pid="Df3_K8m_">『连营』：每当你失去最后的手牌时，你可以获得牌堆顶的一张闪</p>
 <ul>
  <li data-pid="_uSJRL8u"><b><i>周瑜</i></b></li>
 </ul>
 <p data-pid="_C3KfTRs">『英姿』：摸牌阶段开始时，你忘记此技能并从牌堆顶摸两张牌</p>
 <p data-pid="RNJC3cRF">『反间』：出牌阶段限一次，你可以送给一名角色一张牌</p>
 <ul>
  <li data-pid="K63hswBF"><b><i>诸葛亮</i></b></li>
 </ul>
 <p data-pid="E0hfWru8">『观星』：每当你的判定区有乐不思蜀/兵粮寸断/闪电时，你可以观看牌堆顶的X张非红桃牌/非梅花牌/黑桃2-9的牌(X为全场势力数并至多为5)，并改变其中任意数量的牌的顺序然后将剩余的牌放入牌堆底</p>
 <p data-pid="oQDN5Lp0">『空城』：锁定技，每当你失去最后的手牌时，将一张万箭齐发/南蛮入侵置入敌方手牌</p>
 <ul>
  <li data-pid="Xg9-Uz_J"><b><i>赵云</i></b></li>
 </ul>
 <p data-pid="L08wOL0o">『龙胆』：锁定技，你必然将杀当闪，闪当杀使用并可以在队友问你为什么总是脱裤子放屁时自信反驳我就是想用一下技能</p>
 <ul>
  <li data-pid="wRw6a6j5"><b><i>张飞</i></b></li>
 </ul>
 <p data-pid="pMiaHnvf">『咆哮』：摸牌阶段，你可以砸桌子并大叫卧槽这游戏是不是没有杀</p>
 <ul>
  <li data-pid="yCFlfWpn"><b><i>关羽</i></b></li>
 </ul>
 <p data-pid="g9xYt7ji">『武圣』：摸牌阶段，你可以一直默念AKAKAKAKAK</p>
 <ul>
  <li data-pid="JyblUWko"><b><i>大乔</i></b></li>
 </ul>
 <p data-pid="DlN7YJDc">『国色』：出牌阶段，你可以将一张方片牌置入一名角色的判定区并在其判定阶段开始时弃置牌堆顶的一张红桃牌</p>
 <p data-pid="TJLtDKTc">『流离』：锁定技，每当你的攻击范围内没有目标时，你成为场上杀的目标</p>
 <ul>
  <li data-pid="ry1-hO1_"><b><i>马超</i></b></li>
 </ul>
 <p data-pid="MZDYgEWw">『马术』：每当你的顺手牵羊和兵粮寸断打出后，你可以大叫一声卧槽忘了有马术</p>
 <p data-pid="ASPrVnu4">『铁骑』：每当你使用杀指定一名角色为目标时，若其手牌中有闪，你展示牌堆顶的一张黑色牌并弃置之。若其手牌中无闪，你展示牌堆顶的一张红色牌并弃置之</p>
 <ul>
  <li data-pid="NwkWM3dN"><b><i>孙权</i></b></li>
 </ul>
 <p data-pid="BX8N-pNK">『制衡』：出牌阶段限一次，你可以弃置任意张垃圾牌，然后摸等量更垃圾的牌</p>
 <p data-pid="4vL20KKJ">『救援』：其他吴势力没有桃时，你陷入濒死状态</p>
 <ul>
  <li data-pid="mNlLLCDO"><b><i>黄盖</i></b></li>
 </ul>
 <p data-pid="kr5M5m31">『苦肉』：你的回合开始时，你可以立即死亡并被队友问候家人</p>
 <ul>
  <li data-pid="vpGFDaS6"><b><i>黄月英</i></b></li>
 </ul>
 <p data-pid="MTqQtOA2">『集智』：摸牌阶段，你展示牌堆顶的两张非锦囊牌并获得之</p>
 <p data-pid="8J5ZEjPc">『奇才』：当你使用顺手牵羊时，你可以感叹可惜距离不够</p>
 <ul>
  <li data-pid="LN0g6p7M"><b><i>吕蒙</i></b></li>
 </ul>
 <p data-pid="V87l_ZUe">『克己』：锁定技，当你出场时，成为所有攻击性卡牌及技能的目标</p>
 <ul>
  <li data-pid="-KYi9Tr3"><b><i>甘宁</i></b></li>
 </ul>
 <p data-pid="oKIOMIBS">『奇袭』：每当你使用一张过河拆桥，你可以无情嘲笑你的牌太多啦</p>
 <ul>
  <li data-pid="IZ5oDjw2"><b><i>袁术</i></b></li>
 </ul>
 <p data-pid="d3KUw7Db">『妄尊』：你可以声称自己是忠臣</p>
