# 在校生非正常下载 2578 篇文献，社科大 IP 遭数据库商封禁，该学生将面临哪些处罚？
- 点赞数：7977
- 更新时间：2022年07月11日08时02分13秒
- 回答url：https://www.zhihu.com/question/542506737/answer/2568120520
<body>
 <p data-pid="hIyBzBQh">同样都是学生，俄罗斯（更正：哈萨克斯坦）有个学生，就是专门把这些论文都下载下来，做了一个免费下载论文的系统，供大家下载。全球范围内，这些靠论文为生的平台，寄生在整个科研学术之上，赚取了大量金钱，而实际的贡献却非常有限。</p>

 <p data-pid="zDsVzWab">论文是科研工作者和学者撰写的，他们自己却并不能因此而获得收益。全球几大巨头，构建了用论文谋利的网络，可以轻易获取收入，而这些收入最终的源头来自于社会和国家支持科研学术的资金。</p>
 <p data-pid="k2bNaI29">从公共利益的角度，应该尽可能降低知识流通的成本，减少阻碍，但是既得利益者们已经构建起了高墙。</p>
 <p data-pid="FEK6wEmH">而一个学生因为下载论文被处罚，仅仅是这个落后体系下的一个缩影。</p>
