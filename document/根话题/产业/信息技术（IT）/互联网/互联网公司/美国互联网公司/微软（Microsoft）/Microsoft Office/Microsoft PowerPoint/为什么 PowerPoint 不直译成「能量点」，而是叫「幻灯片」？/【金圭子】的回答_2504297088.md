# 为什么 PowerPoint 不直译成「能量点」，而是叫「幻灯片」？
- 点赞数：11327
- 更新时间：2022年05月27日18时44分39秒
- 回答url：https://www.zhihu.com/question/492717143/answer/2504297088
<body>
 <p data-pid="qWYMkQ7W">“有点儿软办公室”(MicroSoft Office)的三大主流软件是：<br>
  “单词”(Word)、“卓越”(Excel)和“力量点”(PowerPoint)。</p>
 <p data-pid="KgjRe8Sf">此外还有“通道”(Access)、“出来看”(Outlook)、“一个记事本”(OneNote)、“印刷人员”(Publisher)、“通信人员”(Communicator)和“信息路径”(InfoPath)等不常用软件。</p>
