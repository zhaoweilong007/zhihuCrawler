# 据说 iPhone 仅凭A13 和 iOS，而其他参数都比不过国产手机却达到销量第一，是真的吗？

- 更新时间：2020年08月01日14时11分37秒
- 回答url：https://www.zhihu.com/question/384048016/answer/1119607874
<body>
 <p data-pid="kavCsKwu">用个手机而已，没必要这么大戾气，我的观点一直都是手机没有最好一说，最适合自己的才是最好的。</p>

 <p data-pid="beCVwXDO">下次提这种问题请务必先把您心目中“除了处理器和系统，其他参数都完全超过iPhone”的那一款国产手机说出来再让我们对比好不好呢？</p>
 <p data-pid="9a5O7F2e">不会您所谓的其他参数指的是旗舰A的a参数，加旗舰B的b参数，再加旗舰C的c参数吧？iPhone不是叶phone，没法一个打十个。</p>
 <p data-pid="DPr_reMT">既然题主没明说是具体哪款国产旗舰，那我就单说一下，除了处理器和系统，iPhone在友商围堵下依然处于领先/顶尖的方面：</p>
 <ol>
  <li data-pid="271fAq2g">屏幕：displaymate连续多年将年度最佳屏幕颁给iPhone，这块屏幕，就是世界第一；</li>
  <li data-pid="hTxMlyeD">色彩管理：这应该算在系统中，其他安卓旗舰譬如三星也有不输于iPhone的屏幕硬件，但色彩管理差的不是一点半点，这锅得Android来背；</li>
  <li data-pid="PRhjhlL0">摄影摄像：一流摄影（拍照）和顶级摄像（视频），至于iPhone11系列摄像头白平衡的统一和拍摄界面的流畅度，不是我针对谁，我是说在座的各位都…为什么都2020年了某些旗舰机取景界面还是能卡出翔？就不用说HDR插帧视频了，在座的各位都…</li>
  <li data-pid="1MwEO5sK">内存和闪存：大家都知道iPhone处理器强，其实iPhone的内存和闪存一直用的也是顶级硬件，当友商还在为emmc和ufs打得不可开交时，iPhone早就用上了nvme。下面我放了passmark的跑分，可以感受下差距有多大；（这图很老了，我就给删掉了，链接还在，感兴趣的朋友可以去看看）</li>
  <li data-pid="qNOu6bP2">隐私保护：FBI和苹果的小故事大家应该都有所耳闻，隐私方面应该没什么异议吧；（有朋友提到了棱镜门，我解释一下，我这里提到的安全性针对普通消费者，棱镜门中政府行为无法预测，更无法逃避，不光苹果，谷歌Facebook等公司同样会把用户信息给美国政府，但对于普通消费者来说，与其担心政府得到自己的信息，更应该担心的是无良公司得到你的信息；另外还有人提到了好莱坞艳照泄露事件，我只想说当你一个密码走天下的时候，就应该预料到有一天你的密码会因为某家公司被撞库而泄露…苹果提供的双重验证可以很好地避免被人轻松登录自己AppleID事件的发生，iPhone用户一定记得打开）</li>
  <li data-pid="_FPaQ9lc">动画：苹果过度动画的流畅，让安卓经历数个大版本更新依旧停留在拙劣模仿的程度，也可能有人不喜欢动画觉得拖泥带水，iOS很贴心的给了关闭选项；</li>
  <li data-pid="lvcfM9Cw">设计：不只是外观设计，包括UI和操作设计，苹果的产品有很明显的传承感和一致性；</li>
  <li data-pid="V-QbOtHw">carplay：懂得都懂；</li>
  <li data-pid="Iz-0i1X7">App Store：Google Play勉强可一战，国内的应用市场就算了…</li>
  <li data-pid="D5O63NIc">生态：苹果的产品线很多，从电脑到耳机，只要你拥有AppleId，你会惊奇发现各个硬件间竟然可以协同得如此自然流畅；</li>
  <li data-pid="FOZMaf2M">震动马达：哒哒哒，iPhone震动的优势，不止在于手感，更在于系统层面的支持，不然你们双击试一下（狗头）；</li>
  <li data-pid="ODRNqzLp">扬声器：360°立体声，真的不一样；</li>
  <li data-pid="m7QX4JI0">麦克风：立体声麦克风；</li>
  <li data-pid="4rlqdm2y">Face ID：安全性不是指纹识别可比，易用性平时没问题，在疫情期间是真的难受…</li>
  <li data-pid="w8-ckbHI">Apple Store：不仅提供产品试用，还有多种课程可以免费学习，另外不同于某些厂商产品出问题后的扯皮，苹果会像车企一样召回有问题的产品，并免费更换硬件；</li>
  <li data-pid="6tB6TEMT">残障友好：题主可以看下<a href="https://link.zhihu.com/?target=https%3A//b23.tv/BV17s411N7v3" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">https://</span><span class="visible">b23.tv/BV17s411N7v3</span><span class="invisible"></span></a>，苹果对残障人士的关爱值得每个厂商学习；</li>
  <li data-pid="m28CrV5e">环境友好：苹果在几代产品之前就换用了无胶水包装，线材也用的可降解材料，金属用回收金属，这条其实与上一条一样，人文、环境，即使你不做，也不会有什么人挑理，一旦做不好还会遭受批评，但苹果还是选择去做，这是企业文化还是责任心使然，我不知道，但我希望能有越来越多的厂家可以参与到其中；</li>
  <li data-pid="1XHSYy-P">……</li>
 </ol>
 <p data-pid="JydH2qNw">以上是iPhone依然处于一流/第一的方面，有的方面被其他手机超过，有的方面其他手机依然难望其项背，综合来看，每年的iPhone旗舰依然是最值得购买的手机之一。</p>
