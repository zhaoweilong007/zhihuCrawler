# 如何看待搜狐科技CEO张朝阳提出5G电磁波可能对人体有害？
- 点赞数：7636
- 更新时间：2019年05月23日15时44分56秒
- 回答url：https://www.zhihu.com/question/324827114/answer/688789282
<body>
 <ol>
  <li data-pid="v-ska9LN">张朝阳清华大学物理系毕业，美国麻省理工大学博士、博士后。我不知道这个问题下面讥讽张朝阳的到底什么来路？有几斤几两能有资格嘲笑MIT物理学博士的技术言论？没有一点敬畏之心？甚至连礼貌都没？</li>
  <li data-pid="lU_fLEGM">5G确实规划了毫米波波段。</li>
  <li data-pid="fEXuJaN7">有不少研究表明毫米波确实会对生物产生危害。≥３ｍＷ／ｃｍ２的毫米波导致成年子鼠学习、记忆功能明显降低，成年子鼠海马区Ｍ－Ｒ的最大受体结合数（Ｂｍａｘ）明显升高，成年子鼠及足孕胚胎脑中多巴胺（ＤＡ）含量明显降低；≥５ｍＷ／ｃｍ２辐射导致成年子鼠下丘脑中ＡＶＰ含量明显降低，以上指标与功率密度呈明显相关。结论５３ＧＨｚ毫米波辐射导致胚胎损伤的阈值在１～３ｍＷ／ｃｍ２，其导致成年子鼠学习、记忆功能降低。</li>
  <li data-pid="W5-GkKFo">角膜对毫米波辐射极敏感，３５ＧＨｚ毫米波辐射１ｈ，功率密度为５ｍＷ／ｃｍ２时，可引起角膜上皮轻度损伤，功率密度为１５ｍＷ／ｃｍ２，除造成角膜上皮脱落或坏死外，甚至引起角膜内皮的肿胀、脱落。</li>
  <li data-pid="aaTxRBfH">5G波段频率太高，所以穿透和绕射能力就差，室内没有覆盖的话，起不了作用。</li>
  <li data-pid="udOF46hY">5G确实很大程度上超出了实际的需求。毕竟我们4G都没有充分的利用起来。5G网络投资巨大，需要数万亿的投资，是否真的对国民生产生活有明显帮助，是否值得投资确实是一个需要斟酌的问题。</li>
  <li data-pid="3gErk-4U">最近有个新闻让我很郁闷，就是说有研究表明，网络技术的普及并没有真正有效提高劳动生产率。中国的全要素生产率最近几年没有增长甚至下降的，虽然我们的网络发展最近几年很快。</li>
  <li data-pid="Rq-rjEIB">张朝阳选择在这个节骨眼上发表这个言论，确实有够……佛系。在我看来他没有什么企图，反而是一片赤子之心。</li>
  <li data-pid="vvXHlJs7">最近几年，我们确实投资了很多看上去很高上大、先进、时髦，却没有多少经济社会效益的东西，比如光伏、风电、VR等，但是很多基础产业，比如DRAM、农业、畜牧业这些不高上大，但是却很落后，要非常需要投资的行业呢？想象下如果我们投入万亿到这些产业，提高了生产率，得到了更多更廉价的粮食、更优质的蛋白质来源、更多的新鲜水果，是不是比手机网速更快一点有价值呢？毕竟我们光草原面积就比整个印度国土都大，我们没有很好的利用起来，也许我们利用的好，牛肉就能降到20块钱一斤也说不定。或者我们用这笔钱投资集成电路、存储器等基础产业呢？塑造几个三星、台积电这样的企业呢？</li>
  <li data-pid="UT7X3JH4">当然，我觉得整个事件中，床破班子对5G的热情令我感到很意外。为了打压华为5G，不惜利用国家力量卑鄙手段来放倒华为，让我这个对5G不怎么感冒的旁观者，突然也觉得5G是未来的命脉了。关于这个5G，要么床破班子是对的，要么是错的。但是我们自己要仔细想想，5G到底真的那么重要，还是只是我们被感觉这个东西很重要。</li>
 </ol>
