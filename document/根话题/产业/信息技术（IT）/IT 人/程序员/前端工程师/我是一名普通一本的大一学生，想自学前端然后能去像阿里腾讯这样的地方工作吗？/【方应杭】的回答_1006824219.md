# 我是一名普通一本的大一学生，想自学前端然后能去像阿里腾讯这样的地方工作吗？
- 点赞数：12239
- 更新时间：2022年07月21日03时20分11秒
- 回答url：https://www.zhihu.com/question/369283077/answer/1006824219
<body>
 <p data-pid="g0cbvNMq">给你个路线，看完并照做能去阿里腾讯（丑话说在前面，这条路需要花费大量时间，但有效）：</p>
 <ol>
  <li data-pid="TE8Crtjn">《编码——隐匿在计算机软硬件背后的语言》，如果身边没人问，你看到「第12章 二进制加法器」就可以了，后面不勉强。这本书让你理解什么是计算机。</li>
  <li data-pid="p_wIqukn">现代 JavaScript 教程 <a href="https://link.zhihu.com/?target=http%3A//zh.javascript.info" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">http://</span><span class="visible">zh.javascript.info</span><span class="invisible"></span></a> ，比《JavaScript高级编程》更适合入门</li>
  <li data-pid="lW4EEJSO">然后你需要学习另一门语言 <a href="https://link.zhihu.com/?target=https%3A//www.bilibili.com/video/av9297989/" class=" wrap external" target="_blank" rel="nofollow noreferrer">4小时讲完《C语言》</a> ，为什么要学第二门呢？为了避免你变成只会 JS 的井底之蛙，开阔开阔眼界，知道什么是链接、编译、可执行文件也是好的。（只想吃青春饭35岁转行的同学可以跳过这一步）</li>
  <li data-pid="ZubwEqko">接下来你可以看《算法（第四版）》，看到子字符串查找就可以了。（只想吃青春饭35岁转行的同学可以跳过这一步）</li>
  <li data-pid="KGJzi8O9">练手，去 leetcode 刷几十道题就当复习了，用 c 用 js 都能刷 leetcode，这一步可以测试自己学得如何。（只想吃青春饭35岁转行的同学可以跳过这一步）</li>
 </ol>
 <p data-pid="5mJ-3dKj">好了，现在你基本把科班生该学的基础学了一部分了，可以进入实践了。</p>
 <ol>
  <li data-pid="42n8FvUM">看《CSS揭秘》，提升自己对 CSS 的掌握。不用看完全书，挑着看十几个自己感兴趣的例子即可，这本书的优点就是不用全看完。</li>
  <li data-pid="vGPVjSiK">开始做项目，比如自己做个画板、做个计算器、做个扫雷，反正就是把你见过的东西都自己实现一次，能不能做成功不要紧，关键是过程。做完三四个这样的项目之后，开始学习框架</li>
  <li data-pid="n2cuTpJD">Vue 和 React 随便学一个，把官方文档一字不漏地看完，再说一次，一 字 不 漏！</li>
  <li data-pid="TctzPD-G">看完之后把你之前的项目用 Vue 或 React 重写</li>
  <li data-pid="5BcYGQtj">开始做更大的项目，比如仿一个 Todo MVC，仿一个购物车应用，总之把你见过的东西都拿过来用 Vue 或 React 做做看。</li>
  <li data-pid="aObQXYP6">提升，这一部分可能性很多，你已经会写点小应用了，但是你只学了编程的皮毛，为了脱颖而出，你可以深入研究一个方向，比如编译原理、工程化、数据结构和算法等，不用贪多，选一个方向就够你研究几个月的。这要看很多书和文章，花的时间可能比前面所有步骤的总和还要多，但回报相当高。（只想吃青春饭35岁转行的同学可以跳过这一步）</li>
 </ol>
 <p data-pid="7TBA8xF6">你在做的过程中会学会科学上网、Stackoverflow、GitHub、Git 等。</p>
 <p data-pid="PfOGYMR5">你会无数次地查看 API 文档，并逐渐记住常用的 API。</p>
 <p data-pid="7fCq_8vf">你不得不了解 babel webpack typescript http，遇到了不要怂，把文档看一下就直接干。</p>
 <p data-pid="mD544wQC">这个过程你会卡壳很多次，请自己想办法搞定。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="DM7c43Qx">幸好你才大一，花两年时间按上面的方法做，大三下学期就可以找阿里腾讯的实习。</p>
 <p data-pid="82Y1NPpr">如果你是按上面方法做的，把简历给我（简历里把你会的东西、做的项目都描述清楚），我帮你内推。除了实习能内推，校招我也能内推。</p>
 <p data-pid="xPlp0wkm">实习结束后，要么留用，要么去竞争对手那里（因为有 Special Offer，工资更高）。</p>
