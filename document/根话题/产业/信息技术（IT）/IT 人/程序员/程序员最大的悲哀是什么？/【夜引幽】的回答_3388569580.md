# 程序员最大的悲哀是什么？
- 点赞数：7747
- 更新时间：2024年04月03日16时29分21秒
- 回答url：https://www.zhihu.com/question/50327690/answer/3388569580
<body>
 <blockquote>
  公司卸磨杀驴，项目开发完就把我裁了。
  <br>
   领导觉得，换个便宜的新人来做维护更划算。
  <br>
   但他不知道，因为需求频繁改动，代码里补丁盖补丁，导致屎块堆叠，可不是随便来个人就能扛得住的。
  <br>
   果不其然，等我一走，屎山喷发了。
 </blockquote>
 <p>1</p>
 <p>人事说要辞退我的时候，我大感意外。</p>
 <p>就在上周，我负责的重要项目刚刚收尾，产品顺利上线。</p>
 <p>老板亲口说要给项目组的每个人发奖金，甚至拍着我的肩膀夸我是股肱之臣。</p>
 <p>这才几天，我这个肱股之臣居然就要被裁了？</p>
 <p>人事面无表情。</p>
 <p>「该给你的补偿公司会一分不少给你的，今天你就把离职手续办了吧。」</p>
 <p>这话说的，好像他们很大方一样。</p>
 <p>我在睿云待了刚好一年。</p>
 <p>赔偿最多就是俩月工资而已。</p>
 <p>但钱是一方面，最让我接受不了的，是他们这种过河拆桥的行为。</p>
 <p>招我来的时候，正是项目启动初期。</p>
 <p>市场上各种人工智能应用异常火爆，睿云也想在自己的产品中加入 AI 来提升竞争力。</p>
 <p>而我正好有 AI 模型的开发和部署经验，入职后，就立刻跟项目组几位同事全身心投入到新业务的研发中。</p>
 <p>小公司岗位职责划分不明确，一个人得干四五个人的活，我是毫无怨言。</p>
 <p>这一年艰辛自不必多说。</p>
 <p>可万万没想到。</p>
 <p>产品上线第一个就裁掉我。</p>
 <p>讽刺的是，昨晚我还在隐形加班，熬夜到两点多整理项目中不足的地方，以便于后续优化改进。</p>
 <p>回到工位上，经理徐峰正在跟其他人说话。</p>
 <p>看见我，他很自然地说：「小陈，你不要有什么太多的想法，这是公司正常的人员优化，你能理解吧？」</p>
 <p>我理解他奶奶个腿。</p>
 <p>项目组其他几个人似乎已经知道了我被裁的事，表情都有几分尴尬。</p>
 <p>我没错过徐峰眼里一闪而过的得意。</p>
 <p>我擦。</p>
 <p>这个老登，估计是早就等着这一天了。</p>
 <p>2</p>
 <p>徐峰虽然是技术部经理，但有着很多领导的通病：不懂技术，还贼特么爱瞎指挥。</p>
 <p>睿云主要业务是口腔数字化服务，我来之前，系统框架问题就很多。</p>
 <p>因为频繁更改需求，每次项目还催得紧，开发和测试时间都远远不够，再加上徐峰这个搅屎棍喜欢乱发指令，导致架构越来越复杂混乱。</p>
 <p>时间一久各种毛病就出来了。</p>
 <p>组件臃肿、代码重复、逻辑纠缠……</p>
 <p>而每一次新的需求变动，就意味着要在原有的 bug 上增加更多的 bug，就好像撒了一个谎却需要用更多的谎去圆，说是堆屎一点也不夸张。</p>
 <p>我愿称之为祖传屎山。</p>
 <p>睿云的业务就是在这座屎山上支撑着。</p>
 <p>如果直接接入 AI，难以想象日后维护起来会有多困难。</p>
 <p>所以开始新项目之前，我不止一次提出更改框架。</p>
 <p>但徐峰不同意。</p>
 <p>他认为这样会大大拖慢项目进度，而且完全没有必要。</p>
 <p>「现在那些 AI 模型不是都现成的吗？直接拿过来不就能用？重写框架，亏你想得出来。公司给你开那么高的工资，不是让你出这种馊主意的！」</p>
 <p>我争不过，只能硬着头皮上。</p>
 <p>果然，等到新项目开始以后，旧框架的设计不符合现有需求，各种预料之中和预料之外的问题接踵而至。</p>
 <p>本着为项目为公司负责任的态度，我跟徐峰频频出现分歧，好几次在会议上爆发了激烈的争执。</p>
 <p>他气极的时候大吼：「别给我找这么多借口！什么框架冲突，什么不兼容，出问题了就多反思反思自己！」</p>
 <p>现在回想，这大半年，他早就忍够了我这个不听话的下属了吧。</p>
 <p>可他还不明白，产品虽然上线了，但是隐患太多。</p>
 <p>只有我这个不听话的员工，才清楚新项目里到底有多少坑，也只有我才知道后续运营中怎么避免踩坑。</p>
 <p>3</p>
 <p>徐峰吩咐我：「把你需要交接的内容整理一下，今天下班前交给我就行。」</p>
 <p>说完他就回自己办公室了。</p>
 <p>我的工作日志，开发流程，运维部署文档都早早提交了，而我又没开始新的任务，还真没有什么需要特别整理交接的。</p>
 <p></p>

 <p></p>
 <p>这也是为什么公司敢突然辞退我。</p>
 <p>组里几个小伙伴纷纷给我发私信，无非就是安慰我的一些话。</p>
 <p>负责测试的芸芸是 00 后，平时我俩最聊得来。</p>
 <p>她特别生气。</p>
 <p>「我前两天就看到徐老贼跟人事嘀嘀咕咕的，不知道在说什么，原来是针对你！你是项目最大的功臣，公司怎么能这样！他们是不是有病？」</p>
 <p>我回她：「是啊，确实有病。」</p>
 <p>不就是觉得反正项目已经完成了，没必要留着我，大不了回头再找个便宜的人来维护就行了。</p>
 <p>我写了份总结文档发给了徐峰。</p>
 <p>「徐经理，您看看还有什么问题吗？」</p>
 <p>徐峰秒回：「行，就这样吧。」</p>
 <p>老登，看都不看。</p>
 <p>找徐峰签字的时候，他嘴上说着舍不得并肩作战的伙伴，但那笑比 AK 还难压。</p>
 <p>「小陈啊，临别之际给你一句忠告，别成天觉得自己技术牛就了不起。对企业来说，最不缺的就是能干活的人。你那自以为是的性格再不改改，到哪儿都干不长久。」</p>
 <p>我回敬他：「您说的对，我肯定不是公司大动脉，最多就是个括约肌。说起括约肌，建议您没事儿多提肛，否则搞不好哪天漏屎。」</p>
 <p>徐峰认为我是不爽，才故意说这种话来发泄。</p>
 <p>他嗤笑一声，做出一副大人不记小人过的恶心模样。</p>
 <p>我也不再废话，扭头就走。</p>
 <p>4</p>
 <p>第二天八点半被闹钟吵醒。</p>
 <p>我把被子一掀才突然想到，自己已经不用去上班了。</p>
 <p>手机提醒我收到一封新邮件。</p>
 <p>打开一看，困意瞬间没了。</p>
 <p>「Jie，您好，我是锋鸣科技的 CEO 裴争鸣。我和我的团队在 Github 上看到，过去几年您一直在做 AI 应用到阿尔茨海默症检测的项目，这跟我们团队想做的事不谋而合。我们真诚地邀请您加入我们团队中，期待您的回复。」</p>
 <p>阿尔茨海默症，其实就是老年痴呆。</p>
 <p>我因为个人兴趣，在 Github 上传过自己写的几个项目。</p>
 <p>这是怎么个事？</p>
 <p>失业第一天，我还没开始投简历呢，offer 居然直接从天而降了？</p>
 <p>我打开搜索引擎查了下锋鸣科技和 CEO。</p>
 <p>公司没搜到多少信息，可这个 CEO 来头不小。</p>
 <p>裴争鸣，杜克大学博士毕业，一直从事于计算机视觉相关领域的研究。</p>
 <p>他的个人主页里，都是他在上学期间独立开发的各类开源项目。</p>
 <p>以我专业毒辣的眼光鉴定，妥妥的技术大牛。</p>
 <p>真没想到，这样的牛人居然会注意到我在 github 上的项目，一时竟让我有些受宠若惊。</p>
 <p>不过我对他们公司还不了解，肯定是不能直接答应的。</p>
 <p>我在邮件里表达了自己愿意深入沟通的意思，然后附上了手机号。</p>
 <p>邮件刚发出去，芸芸的微信马上来了。</p>
 <p>「杰哥，组里今天来了个新人！听徐老贼意思，就让他以后维护咱们那个项目呢。你敢信？一个刚毕业的本科生！」</p>
 <p>「谁不知道应届生最便宜啊！啧啧啧，把你踢了换个便宜的来替代，真是一群傻逼！」</p>
 <p>我乐了。</p>
 <p>徐峰他没事吧。</p>
 <p>好歹是公司的重要业务，再怎么样，也不能找个刚毕业的新兵蛋子来维护啊。</p>
 <p>他到底是蠢，还是对我写的代码太有信心？</p>
 <p>无所谓，哥已经不关心了。</p>
 <p>因为 CEO 的电话来了。</p>
 <p>5</p>
 <p>我正要客套几句，那头直截了当地问：「你也在上海？」</p>
 <p>「对。」</p>
 <p>「明天周六，方便约个午饭吗？地点你定。」</p>
 <p>通话时间不到一分钟，我们就非常干脆地约好了明天在我家附近的川菜馆子见面。</p>
 <p>全程没有一句废话。</p>
 <p>这就是大佬的效率吗？</p>
 <p>芸芸还在给我口头直播。</p>
 <p>「新人在看你留下的文档呢，我瞅他那样就知道费劲。」</p>
 <p>「老板在群里说 AI 检测上线以后，用户活跃度大幅提升，狠狠夸了徐老贼。你是没看到，他一笑满脸褶子，菊花怒放了属于是。」</p>
 <p>我想象了一下徐峰大笑的样子，觉得芸芸这个比喻实在太贴切了，忍不住把刚喝进去的水喷了出来。</p>
 <p>第二天我在饭店见到了裴争鸣。</p>
 <p>不知道是不是程序员之间的惺惺相惜，我俩一见如故。</p>
 <p>锋鸣科技是裴争鸣一年前才创立的一家公司，致力于 AI 在医疗领域的应用研究，目前已经获得了两轮投资。</p>
 <p>「……阿尔茨海默症能诊断出的时候，大脑神经元已经死了太多了，基本上很难再逆转。但是如果疾病发现得早，是可以通过药物治疗和控制的，提前介入和干预，就能帮助患者改善认知功能、延缓临床病程 10—15 年。所以，我们现在要去做的这件事就是尽量帮助人们在发病初期就检测出来。」</p>
 <p>当下 AI 在各行各业的火热程度不肖多说，裴争鸣想做的事并不稀奇，毕竟我自己也因为兴趣在研究这方面的东西。</p>
 <p>但最吸引我的，是他说起这些话时眼里闪烁的光。</p>
 <p>饭馆里人声鼎沸，人们讨论着工资、房贷、娶老婆、生孩子，眼前这个高谈阔论要为人类医疗事业做出颠覆性改变的男人显得那么格格不入。</p>
 <p>但我却感觉自己跟他产生了共鸣。</p>
 <p>6</p>
 <p>快结束的时候我提醒他：「裴总，你全程没有问过我任何技术相关的问题。」</p>
 <p>裴争鸣笑道：「你在 github 上的那几个项目，50k star（点赞）已经很能说明问题了。而且我看过你的代码，我们团队非常需要你这样的优秀人才。」</p>
 <p>啧，你听听。</p>
 <p>CEO 就是不一样。</p>
 <p>「您太过奖了。我还有最后一个小问题，你们招人难道都是你在网上发掘？」</p>
 <p>裴争鸣哈哈大笑。</p>
 <p>「当然不是，但是我们有缘分，今天坐在一起吃饭是缘分，如果未来可以一起共事那就是圆满了。」</p>
 <p>啧啧，你听听，你听听。</p>
 <p>徐峰只会认为，公司花钱雇我们是我们的荣幸，企业给员工提供就职机会是恩赐，必须要对公司感恩戴德。</p>
 <p>人跟人的差距比人跟狗的差距都大。</p>
 <p>入职睿云的当天，不到十个人的团队特地为我开了个小型欢迎会。</p>
 <p>跟新同事们互加微信，恰好这时候有个陌生人也来加我，我一个没注意通过了好友申请。</p>
 <p>「徐经理让我来问问你，为什么图像库版本忽然不兼容了？」</p>
 <p>我立刻反应过来，这是徐峰招的那个新兵蛋子。</p>
 <p>没有礼貌不说，问话还没头没尾。</p>
 <p>而且现在睿云又不给我开工资，版本兼不兼容关我屁事，不能我被裁了还得负责售后服务吧？</p>
 <p>我二话不说就把他给删除。</p>
 <p>没两分钟，徐峰的电话来了。</p>
 <p>我想了想，按下了接通。</p>
 <p>「小陈你怎么回事？刚从公司离开就要划清界限了？就找你问个小事你有必要这样吗？我跟你说，做人留一线，日后好相见。」</p>
 <p>这训斥的口吻，还把我当他手下呢。</p>
 <p>「徐经理，我是爱莫能助。你也知道的，代码那么复杂，出了问题是要挨个排查的，不是我隔空就能诊断出来的啊。」</p>
