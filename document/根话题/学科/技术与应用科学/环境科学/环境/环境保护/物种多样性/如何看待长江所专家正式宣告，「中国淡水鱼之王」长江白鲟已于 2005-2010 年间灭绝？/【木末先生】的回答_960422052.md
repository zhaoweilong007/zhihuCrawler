# 如何看待长江所专家正式宣告，「中国淡水鱼之王」长江白鲟已于 2005-2010 年间灭绝？
- 点赞数：9098
- 更新时间：2020年01月03日00时33分23秒
- 回答url：https://www.zhihu.com/question/364374536/answer/960422052
<body>
 <p data-pid="dMRh8RAT">忧伤……白鲟成了“20”年代里，第一个成为恶化生态牺牲品的物种。</p>
 <p data-pid="5iZJq54j">它不如中华鲟有名——我听中华鲟听得更多，但它有一个不知是荣耀还是悲伤的称谓——“水中大熊猫”。</p>
 <p data-pid="Eq_uGl3E">不应该套用“受害者有罪论”，但白鲟确实有其“繁衍生息”上的“短板”（唉）</p>
 <ul>
  <li data-pid="PEuh26lA">性成熟周期长（7~8年）</li>
  <li data-pid="Km3NlP8E">味美</li>
  <li data-pid="tskIH84a">体型大，商业价值高（唉）</li>
 </ul>
 <p data-pid="ytPdkleX">后两者引起的备受青睐（捕捞）在俗语“千斤腊子万斤象”里体现得淋漓尽致。腊子指的是中华鲟，象指的是白鲟（因为其吻部长状如象鼻，又俗称为象鱼），这句话意思是：只要捕获一尾，不论中华鲟还是白鲟，就比打十几网小鱼的收入还多。</p>
 <p data-pid="waWj0DYH">50年前，白鲟还有很多，而在半个世纪里，它消失殆尽。</p>
 <ul>
  <li data-pid="kzB_C5ZO">20世纪70年代，每年白鲟的捕捞量有5吨左右，那时候渔民说：“(鲟)鱼肉用板车拖到街上去卖，像卖豆腐一样，要多少切多少，也不用称。”</li>
  <li data-pid="YFDBMCRG">1999年初步统计，白鲟资源量已不足400尾</li>
  <li data-pid="X_34cR3K">2002年一只误捕的白鲟，被拍下了，或许成了活体白鲟的最后影像资料<sup data-draft-node="inline" data-draft-type="reference" data-numero="1" data-text="2020第一个悲伤的消息！专家称长江白鲟已灭绝" data-url="https://baijiahao.baidu.com/s?id=1654627981251088859&amp;wfr=spider&amp;for=pc">[1]</sup></li>
 </ul>

 </figure>
 <ul>
  <li data-pid="inW8Rxnd">科研人员自2003年以后，再没有发现过白鲟</li>
  <li data-pid="rMTH4i5C">2020年开年，科学家宣布白鲟灭绝……</li>
 </ul>
 <p data-pid="v3rgqc_Q">在这个过程中，我们也做过努力，列过珍贵稀有动物、国家一级保护动物IUCN极度濒危物种。</p>
 <p data-pid="Cy5-UGEa">但白鲟无疑是“金贵”的，即使保护，也没有保护下来</p>
 <ul>
  <li data-pid="9B-k5hNi">人工饲养从未成功</li>
  <li data-pid="T2vhsna8">未能保留下卵、精子、受精卵，或可培养的活细胞，或普通冷冻组织</li>
 </ul>
 <p data-pid="8RIJ666p">而它的灭绝综合了种种因素，<b>这些因素最终都指向一个方向：人为</b></p>
 <p data-pid="s1uhFdfR">1.捕捞</p>
 <p data-pid="f-qCC6bo">2.环境破坏，这包括<sup data-draft-node="inline" data-draft-type="reference" data-numero="2" data-text="危起伟：白鲟可能已经灭绝 | 但请不要放弃，就像对白鱀豚一样 " data-url="http://www.sohu.com/a/274065584_100001695">[2]</sup></p>
 <ul>
  <li data-pid="WQKSwURX">葛洲·坝分割：把长江分成了上下两段，上游的白鲟去不了下游，下游的白鲟也到不了上游，它们原本就数量较少的种群被分割得更小。</li>
  <li data-pid="Y8ooL6hq">生存空间缩小：白鲟是大型食肉动物，活动范围缩小也意味着觅食范围的减小，即使上游食物短缺，白鲟也无法再到下游觅食。</li>
  <li data-pid="QkAY1p8U">船活动的影响：船来船往的码头、不分昼夜施工的采砂场，都会影响白鲟的生存。</li>
  <li data-pid="FZKckPsX">繁殖地的破坏：白鲟游经过一条条危机四伏的河流，在轮船下左躲右闪，忍饥挨饿回到家乡产卵。本以为安全了，没想到祖祖辈辈沿用的“产床”早已不见踪迹，自己的家园变成了人类的大工地。</li>
 </ul>
 <p data-pid="FNUpbmNU">长江所的论文，无疑为“白鲟”开了死亡证明。这种离开，不是找不到，而是再也见不到。更可悲的是，我们连白鲟的一个活体标本也没有留下来。</p>
 <p data-pid="cBFOL_de">“邮票证明我来过……”</p>
