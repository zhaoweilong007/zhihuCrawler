# 长城汽车举报比亚迪污染物排放不达标，比亚迪回应「反对不正当竞争」，对比亚迪有哪些影响？如何看待此风波？

- 更新时间：2023年05月25日15时14分25秒
- 回答url：https://www.zhihu.com/question/602925644/answer/3043727931
<body>
 <p data-pid="adyB_ItJ">看到举报文件了，感叹长城好勇。</p>
 <p data-pid="9VSbI8bU">简单说一下结论：</p>
 <ol>
  <li data-pid="DTEggoi-">情况是真的，这俩车我都拆过，很震惊。</li>
  <li data-pid="c_J51yb1">怀疑是真的，要达标必须要用高压，甚至部分合资企业还上了电控电动主动吸附系统，确保不能超标。</li>
  <li data-pid="_gnVxj83">监管部门很难收场，因为多家自主品牌技术中心都进行了市场车辆的逆向排放测试，CD模式测试确实过不了。长城举报之前，做了充分的准备，而且这个问题是行业内已经是一个普遍的质疑了。</li>
  <li data-pid="7T8Vzubd">中国的法规严，监管弱，对自主品牌相对宽松的灰色地带，被长城这次举报给掀桌子了。</li>
 </ol>
 <p data-pid="U-DoYS42">有啥直接后果：</p>
 <ol>
  <li data-pid="wDNgcN18">把监管部门给怼墙角了。查出来有问题，车企的召回方案大概率是巨资，再追溯当时是怎么过的监管测试，简单来说就是掀桌子了。查不出来有问题，那更有问题。</li>
  <li data-pid="2i3dHkyu">油耗、排放、有害物质检查，监管部门的灰色地带一直是个黑洞，尤其是对自主品牌。但是过去车企一般来说就是看破不说破，一般很少有举报的。本次举报扩大化后，又是一场腥风血雨。</li>
 </ol>
 <p data-pid="SKJGzBlm">为什么？</p>
 <ol>
  <li data-pid="k-UhI059">市场竞争条件下，比亚迪的突然崛起，宣传过度完美后遭到的反噬。比亚迪目前在三电系统产业链带来的产品大概整体phev驱动系统6000-10000的成本优势别人学不来，但是其他的部分还是过去的比亚迪。很多自主品牌的传统技术人员对比亚迪并不认可。缺陷接受标准、供应链级别、材料选择、防锈工艺、有害物质使用控制等等，这个还是国产自主品牌的一般水平，但是被消费市场传播推到国货之光、国内第一销量，然后所有人都要自主品牌学比亚迪，其实好多自主企业是闷一口血的。尤其是一些花了很多钱，硬件全体系对标合资企业甚至按照欧盟标准已经可以大量出口海外的，非常不服气这个结果。所以长城抓住了油箱这个东西，打了一记闷棍。</li>
  <li data-pid="yFATUi1B">比亚迪目前其实也在适应这个国内自主品牌销量第一的地位，其实也在研究今年年中开始也要换上高压油箱，而且很多关键零件开始寻找新的供应商，以确保量大了之后质量能支撑住。长城这记闷棍的时间点也掐的很好。</li>
 </ol>
 <p data-pid="9POIeoQv">最后，行业密切关注下一步，随着自主品牌的提升回归市场主流，在法规方面的灰色地带应当尽快消除并承担相关的企业责任。</p>
