 <h2>1. 注册Apple ID</h2>
 <p data-pid="GBN6hhrW">Apple ID是所有iOS设备的门票，拥有Apple ID就意味着可以在iPad上安装正版app，使用iCloud服务，以及防止你的iPad丢失或被盗（超级重要）！</p>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-ca0f4e3e8e9c99142ed6743dfd8cb13a_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="1130" data-rawheight="381" data-original-token="v2-fe4a7057087e7c2056354731756bad58" data-default-watermark-src="https://picx.zhimg.com/50/v2-64a314575886699f0f5485ccda4b2913_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1130" data-original="https://picx.zhimg.com/v2-ca0f4e3e8e9c99142ed6743dfd8cb13a_r.jpg?source=1940ef5c">
 </figure>
 <p data-pid="MZ13nuQf">一般注册Apple ID前往下列网址即可，如果你懒得去，也可以在初始化iPad的时候进行注册：</p><a href="https://link.zhihu.com/?target=https%3A//appleid.apple.com/account%23%21%26page%3Dcreate" data-draft-node="block" data-draft-type="link-card" data-image="https://pic4.zhimg.com/v2-f5562edaf787849bebffee610e80f713_180x120.jpg" data-image-width="1200" data-image-height="630" class=" wrap external" target="_blank" rel="nofollow noreferrer">Apple ID</a>
 <p data-pid="c4M1ripw">目前注册都需要使用手机号，且强制要求<b>开启双重认证</b>，因此！设置双重认证的手机号一定是<b>常用手机号</b>，如果预计后续会换号，记得换号之前改绑定Apple ID，否则后面改绑会需要找人工客服，非常麻烦。</p>

 <h3><b>4.2 辅助功能</b></h3>
 <p data-pid="C88seFN3">这一部分很多新手可能会懵！听说很厉害但是又玩不明白，不过虽然功能非常复杂，但是有不少神器，这里详细介绍一下吧！</p>

 <h3><b>5.4 手势操作</b></h3>
 <p data-pid="11T1kzLl">由于iPad的手势非常丰富，所以单独拿出来说（设置上这一条居然没有说明，也太不专业了），iPad上的手势可以根据使用的手指数量来划分：</p>
 <ul>
  <li data-pid="MO9BaCEH"><b>单指：</b>基本操作有<b>点击</b>、<b>长按</b>、<b>按住拖动</b></li>
 </ul>
 <ol>
  <li data-pid="gz_vyspn">从屏幕底部往上滑：解锁屏幕、返回主屏幕</li>
  <li data-pid="6tTw6Y6R">从屏幕顶部往下滑：查看通知</li>
  <li data-pid="Vl0hWuUN">主屏幕页面，屏幕中间下拉：唤出搜索框</li>
  <li data-pid="DJDtwkOO">从屏幕底部往上轻滑，到屏幕的一半停住：唤出多任务应用切换</li>
  <li data-pid="Q9tKmZrQ">在应用内从屏幕底部往上滑动两厘米停住：唤出程序坞（方便分屏幕或切换应用）</li>
  <li data-pid="fe9MU5Nl">屏幕底部左右滑动：切换应用</li>
  <li data-pid="w9aGpR9G">打字时长按空格：移动光标到指定位置输入</li>
  <li data-pid="WrZxnjYS">打字时单次点击移动光标、两次点击选中词语，三次点击选中段落</li>
  <li data-pid="xn46Bh94">其他都是基本操作，不多说</li>
 </ol>
 <ul>
  <li data-pid="n6yABe-M"><b>双指：</b>多用于<b>缩放</b>功能</li>
 </ul>
 <ol>
  <li data-pid="KepRIWzy">双指靠近缩小，远离放大：查看图片、文件、网页等可缩放内容适用</li>
  <li data-pid="0ZuxIxQs">双指同时按住，往下拖动：信息、备忘录支持多选</li>
  <li data-pid="pCrTP3GO">Safari浏览器：双指靠近查看当前所有标签</li>
  <li data-pid="AEtAQCGA">照片图库：双指缩放、远离，在屏幕上展示更多或更少照片</li>
  <li data-pid="bZBrpoVn">键盘：双指在键盘上滑动，快速移动光标</li>
  <li data-pid="iWlYv8OY">键盘：双指捏合缩小成手机键盘，远离时还原</li>
  <li data-pid="7lm0Xssd">拆分键盘：有主屏幕按钮（圆形Home按钮）的设备：双指远离键盘拆分到屏幕两侧</li>
 </ol>
 <ul>
  <li data-pid="eJaSG6nj"><b>三指：</b>主要针对<b>输入界面</b></li>
 </ul>
 <ol>
  <li data-pid="K9fXn5li">打字的窗口三指轻点：弹出快速编辑栏，iPad上可以快速撤销或重做输入！</li>
  <li data-pid="YpWhXS9m">打字时从右往左快速滑动：撤销，从左往右：重做（上面视频有讲哦）</li>
  <li data-pid="rENQ5HAG">三指靠近：复制，三指远离：粘贴</li>
 </ol>
 <ul>
  <li data-pid="zHNATJ6K"><b>四指或五指：</b>主要用于<b>切换界面</b></li>
 </ul>
 <ol>
  <li data-pid="_F4xOIf1">任意位置往上滑：唤出多任务应用切换</li>
  <li data-pid="aZPM7qu_">任意位置左右滑动：切换应用</li>
  <li data-pid="EPra8RS9">五指专属操作：捏合停住打开多任务，不停返回主屏幕</li>
 </ol>
 <p data-pid="2spuCv2P"><b>稀奇古怪：</b>一只手指<b>长按选中</b>后按住不放，用另一直手指<b>点击选择</b>其他内容。常用于一次移动多个文件或桌面图标。</p>
 <p data-pid="qHhKWzSj">iPad的手势操作看似复杂，其实都比较人性化，多试试就会用了。如果担心老人记不住这些功能，只要记得不同数量的手指主要是干什么的就行。</p>

 <h3><b>5.5 多任务与画中画开启方式</b></h3>
 <p data-pid="nHrbqqwA">这应该也是非常多iPad新手不太会操作的内容，其实学会了之后就很简单了，具体步骤如下：</p>
 <ol>
  <li data-pid="6d7Z4hv2">打开第一个应用，然后从屏幕底部上滑三厘米左右，打开程序坞</li>
  <li data-pid="PLYvdjH6">按住你要打开的App，拖到屏幕中间（比如做笔记时忽然想看相册里拍的课堂笔记）</li>
  <li data-pid="ISHeee5U">这样就得到了满屏的主App和小窗的次App，重复操作可以获得多个小窗的次App</li>
  <li data-pid="36qmFwf8">但是这还没有结束，因为次App是可以切换的，小屏<b>下方灰条</b>可以切换App的！左右滑动或者往上轻扫就可以切换啦！另外<b>屏幕顶部灰条</b>按住拖动窗口，或者往最右一滑关掉小窗（不小心关掉，屏幕右侧往左一滑就回来了）！</li>
 </ol>

 <p data-pid="OZX6ppkV">尽量避免重度使用时使用快充，特别是10.5寸的iPad，这貌似是10.5寸的一个通病，重度使用时iPad的主板会非常烫，而快充会导致电池和充电IC位置很烫，一方面这非常不利于电池寿命（<b>锂电池过热容易造成永久容量损失</b>），其次会导致10.5寸iPad出现通病：Home键上方亮斑！</p><a href="https://www.zhihu.com/question/312877010?sort=created" data-draft-node="block" data-draft-type="link-card" class="internal">Ipad Pro 10.5寸是否大概率会出现屏幕白斑问题？</a>
 <p data-pid="lpBKOXAH">不过这个亮斑问题也是设计缺陷，即使小心使用，时间长了也有概率中招，放宽心就好，<b>屏幕有两年的保修，出问题了在保修期直接去换</b>！个人之前用过的三台iPad Pro 10.5就有两台出现了该问题，其中一台由于经常连着快充吃鸡，电池也迅速降到了70%的健康度。</p>
 <p data-pid="EaesmtW4">关于查看电池健康的，一般是将iPad 连接到电脑，使用itools或爱思助手查看，不过实测当健康水平较低时非常不准（充满前显示健康度70%，充满后80%），另外要提一下电池循环，比如说78次循环意味着你的电池理论上经历了78次从100%用到0%的过程（直观地来说，比如两次100%用到50%叠加起来就是一次循环），不过其实际计算方法更复杂。</p>
 <p data-pid="d4_NfC2X">另外对于锂电池，一般建议每两到三个月进行一次完整的充电循环，需要这样做：</p>
 <blockquote data-pid="Amcclkjq">
  将电池充满，并保持一段时间，然后保持正常使用（不要长时间持续使用），直到电池耗完关机，稍微等一段时间，重新将电池充满（建议使用慢速充电器，比如苹果祖传充电头的五福一安）。
 </blockquote>
 <p data-pid="quZehvjb">这样有助于避免锂电池出现虚电的问题，但是对于延长锂电池使用并无帮助，如需延长锂电池使用，建议日常使用尽量让电池电量保持再30%-80%之间，更多相关内容可以看看少数派的这篇文章：</p><a href="https://link.zhihu.com/?target=https%3A//sspai.com/post/55393" data-draft-node="block" data-draft-type="link-card" data-image="https://pic3.zhimg.com/v2-017363c7714c00c15066c5049ecf3f86_180x120.jpg" data-image-width="1440" data-image-height="480" class=" wrap external" target="_blank" rel="nofollow noreferrer">科学保养锂电池，这些方法让你的手机更耐用 - 少数派</a>
