 <h2>充电要改变观念</h2>
 <ul>
  <li data-pid="5IgZ-wBQ">首先明确,即使你不用,电池也会自己老化的,损耗是不可逆的过程.</li>
  <li data-pid="eqqPqXGJ">电池有循环次数大家都知道,一般的充电循环次数为1000左右</li>
  <li data-pid="LKUm5Q6O">完全充电到完全放电为一次,不是说你有电情况下插上去拔下来算一次</li>
  <li data-pid="lMpfpFS4">新iPad不需要放完电再充,现在用的是锂电池没有记忆性质</li>
  <li data-pid="IKrSNoz3">不要傻傻的拿了新iPad等它放完电才给充电，这对电池不好其实</li>
  <li data-pid="SYpebykG">充电记得随用随充，最好不要等低于30%了才充,不好</li>
  <li data-pid="W2DonmbU">如果可以,每个月完整的进行一次充电放电可以保证电池活性</li>
  <li data-pid="7Dy4Ka2z">最好不要边充电边玩游戏，不好(容易过热)</li>
  <li data-pid="kw_qrJ9I">如果在不过热的情况下,是可以边充边玩的</li>
  <li data-pid="a8KSqxw8">保修期内如果电池健康低于80%可以免费换电池</li>
  <li data-pid="BeUaZgUf">冬天温度过低充不进去电是正常的,iPad自我保护而已</li>
  <li data-pid="6gayxIhw">长时间不用的话最好电量保持在50%左右,电压稳定最长对电压损耗最少,这个也是为什么很多刚出厂新机拿到手你会发现电量在50%或者40%左右的原因</li>
  <li data-pid="0i0ee3ed">觉得有帮助别忘记 ,你们懂的</li>
 </ul>

 <h2>iPad是可以插U盘的,连接鼠标的</h2>
 <ul>
  <li data-pid="AT1OH5Ci">iPad升级iOS13后，是可以插U盘或者通过转接线接普通U盘的,具体方法可以到我主页看视频,我专门录了一个.</li>
  <li data-pid="ebDRO1RY">不仅可以接U盘,升级到iOS13.4后,连接鼠标后,也可以更像电脑的鼠标了.不管是苹果的鼠标还是普通的蓝牙鼠标都可以.</li>
  <li data-pid="gr7g8DRV">鼠标设置路径:iPad设置--辅助功能--触控--辅助触控打开--设备--蓝牙设备,搜索到鼠标信号,连接即可.</li>
  <li data-pid="Etm1dFkK">基本可以实现常规鼠标的大部分操作,拖拉表格线什么也没问题.</li>
  <li data-pid="8XLXio3g">当然，也可以连接带触控板的键盘，iPad pro的妙控键盘虽然重一些，但是打字手感，触控板体验超级好（本段文字即为妙控键盘打下的），iPad2019，iPad Air3 罗技也出了带触控板的键盘。从体验上来说，在iPad上触控板的体验大部分场景下会比鼠标更好。</li>

 <h2>了解下你不知道的Apple售后</h2>
 <ul>
  <li data-pid="QPDYBnrQ">Apple的售后总的来说是非常好的</li>
  <li data-pid="ncunE51x">iPad官网直营店或者天猫旗舰店购买均可以14天免费退换，即使你激活了（天猫写着7天是因为天猫系统改不了改不了…估计全天猫也就Apple这么任性吧，14天内都可以。在线销售中，天猫旗舰店和官网一样都是Apple自己在运营的，其他都是第三方）</li>
  <li data-pid="9WeK9mQp">你的数据线在保修内如果坏了，是可以去免费换的，记得去直营店，或者正规售后，街边有些冒充的是不会给你换的。</li>
  <li data-pid="hfXjtuOx">有关于iPad的问题，你打400客服一般都可以解决。如果比较忙，选择英文频道进去，会比较容易找到客服，当然，要说英语。</li>
 </ul>
 <h2>挂Apple牌子的店那么多,别认错了</h2>
 <ul>
  <li data-pid="IEH7FwoS">Apple的线下店一般分两种，自营的叫直营店很高大上的装修，其他的像有些商场的专柜叫授权经销商。不是所有挂着Apple牌子的都是正规的Apple店铺，有些街边的小店可能是没有资质的，卖给你是假货或者翻新,卖给你后就不管你了。</li>
  <li data-pid="8hSEj6Ob">Apple直营店，服务最好，一般都开在城市的最繁华地段，有着巨大的精致的落地玻璃（每块玻璃都定制的，巨贵），宁波这家还有块巨大的定制非常细腻的显示屏用来上课,好奇问了问,说是索尼定制的(索尼,这么大=奢侈奢侈),直营店除了负责销售也负责售后，有问题先预约天才吧，然后拿设备和身份证去就可以。</li>
  <li data-pid="2yGcYLd-">如果你的城市没有直营店，那应该会有指定维修网点，有些会开在写字楼里，Apple官网可以查询具体地址和联系方式，或电话400询问。</li>
 </ul>

 <h2>Apple ID注意事项</h2>
 <ul>
  <li data-pid="xVYcmhmF">注册Apple ID最好不要用QQ号等纯数字邮箱，会很容易被垃圾短信算法算到污染</li>
  <li data-pid="M6vkAp6q">一定要开启“双重认证”（每次电话Apple400客服都会提示你开这个），否则你的iPad有可能被黑客绑架（我朋友碰到过），具体方法放在评论区了，还不会的去看就行了。</li>
  <li data-pid="NxtQppF3">Apple ID是你在iPad上的最重要的钥匙，不要轻易给别人。</li>
  <li data-pid="9t-4p_1p">也别登别人的Apple ID ,特别是在iCloud里登入,如果要下软件什么,在Apple store中登入就行了,千万不要从iCloud里登入(我一个朋友又遇到了)</li>
 </ul>
 <h2>Apple ID的好功能你却浪费了</h2>
 <ul>
  <li data-pid="J1tAlIks">Apple ID是你iPad的账号，用这个账号你可以实现很多交流的功能。</li>
  <li data-pid="814CnwA9">除了常规的发iMessage和FaceTime交流</li>
  <li data-pid="zpcmEi_l">比如和你的对象，在系统相册里建立一个共享的相簿，两个人可以把想分享的照片直接放进去，某一个人放进去后另外一个人在他iPad或者iPhone上也能看到，一起经营属于你们的回忆吧</li>
  <li data-pid="xfxSEs4K">比如在系统备忘录，你们可以共同编辑一个备忘录，一起在一个备忘录上记日记也好，作事项备忘录也好，或者一起做旅行list，都可以。在你iPad或者iPhone的备忘录上修改，对方iPhone和iPad上的备忘录就可以看到变化。这个功能其实在很多APP里都可以，共享写作。</li>
  <li data-pid="GTXfbyPY">比如设置提醒的时候，把相关的人的Apple ID一起加进去，一起提醒他们明天聚会别迟到等等。</li>
 </ul>

  <li data-pid="cgDsml5t">记得关闭“APP内评分及评论”，不然会比较烦，每次提示你给APP评个分</li>
  <li data-pid="tbigTTjM">开启“卸载未使用的APP”，可以让你不充裕的内存又恢复过来。</li>
  <li data-pid="Qe-sej6D">下载的软件想要批量移动其实很简单，先长按一个APP，让APP都扭起来，不放手，另外一只手去点你想一起移动的APP，想移哪个点哪个，点了就会被吸过来，超级简单</li>
 </ul>
 <h2>iPad是可以长截图的</h2>
 <ul>
  <li data-pid="Qf-zHQuJ">iPad在iOS11后，网页等截面，截图后是可以选择“整页”的长截图的，也可以转为PDF导出来。</li>
  <li data-pid="Z63iprIS">当然你也可以通过下载第三方APP实现，比如免费的“滚动截屏”APP，收费的“速拼”（也不贵功能更多）</li>
 </ul>
 <h2>iCloud是个好东西</h2>
 <ul>
  <li data-pid="V2PGhx8C">如果可以，尽量保持iCloud备份，即使iPad丢了，数据都可以被“复活”</li>
  <li data-pid="u9ZIODTy">照片备份在iCloud，可减少对内存的占用，需要的时候才从云端抓下来</li>
 </ul>
 <h2>订阅记得及时关闭</h2>
 <ul>
  <li data-pid="NaBtMu3_">在APP里订阅过什么服务后，记得及时关闭订阅，以免被自动续费扣款</li>
 </ul>
 <h2>控制中心好东西</h2>
 <ul>
  <li data-pid="RzggqUkb">右上角下滑可以调出控制中心快捷键</li>
  <li data-pid="GG6l6MaE">在设置-控制中心，可以把常用的添加到这里</li>
  <li data-pid="BKbkwRHE">试试去长按每一个图标感受下功能</li>
  <li data-pid="f7fRb3Ec">记得WiFi也长按下会有新发现，打开后再长按下WiFi又会有个新发现。</li>
 </ul>
 <h2>隔空投送</h2>
 <ul>
  <li data-pid="7UARRR6g">iPad和iPhone之间或与其他苹果设备之间用隔空投送传图片，文件等不会压缩且很快，没流量也可以</li>
  <li data-pid="X159_Gr2">不用的时候记得关闭或者设置为仅限联系人，不然有可能被陌生人骚扰，我朋友在地铁上被人通过这个发很恶心的图。</li>
 </ul>
 <h2>后台APP刷新可关闭几个</h2>
 <ul>
  <li data-pid="gKhOIycI">设置-通用中，打开后台APP刷新，关闭不常用的，对减少电量损耗会有点帮助</li>
 </ul>
 <h2>文本替换记得用</h2>
 <ul>
  <li data-pid="xCR7hX4F">设置-通用-键盘--文本替换</li>
  <li data-pid="K8ZIJ9SA">可以把你常用的句子甚至段落，用快捷键打出来，比如用DZ,代表你浙江省宁波市鄞州区XX街道XX路的收件地址，用的时候键盘打DZ就可以调出来，超级方便。</li>
 </ul>
 <h2>关于照片</h2>
 <ul>
  <li data-pid="hl4snQgU">删除的照片，其实还在你手机最近删除相册里，有些在这里会待40天才会彻底删除，所以万一删错了，这里还有后悔药</li>
  <li data-pid="6qysDAPP">相册里，点选择，按住滑动照片，可以快速选择多张。</li>
  <li data-pid="nak63GId">双指捏合，可以放大缩小相册</li>
 </ul>
 <h2>ipad的基本手势和操作（iOS新系统）</h2>
 <ul>
  <li data-pid="CDqC0ylH">任意界面，点按屏幕顶端中间位置，可快速回到前面</li>
  <li data-pid="Hml6Idh9">从屏幕下边框上滑调出多任务，上滑APP关闭后台运行（一般不需要经常关闭后台）</li>

  <li data-pid="A1y49ofg">从右上角下滑可调出控制中心</li>
  <li data-pid="TFiWrnPo">从左上角下滑可调出通知栏</li>
  <li data-pid="5ifg0IhE">截图：同时按住锁屏键和音量+键（全面屏）</li>
  <li data-pid="9Xe_O-Wx">关机：长按锁屏键和和音量+键（全面屏）</li>
