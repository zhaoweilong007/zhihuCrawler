# 一个新的笔记本电脑到手后应该做哪些事情？
- 点赞数：17074
- 更新时间：2019年07月12日23时46分08秒
- 回答url：https://www.zhihu.com/question/37031635/answer/741291558
<body>
 <p data-pid="xz0hFm8l">吃水果要剥皮，用 Windows 也一样，如果是 Win 10 的话，刚装完系统就需要屏蔽一些功能，才能更顺畅快速：</p>
 <ol>
  <li data-pid="IDvuAcT7">隐藏任务栏上的搜索框和小娜，需要搜索时win+s就会出现</li>
  <li data-pid="V0I-9v2Q">禁用开始目录的app自动推荐</li>
  <li data-pid="Jyrxq5UD">删除人脉图标</li>
  <li data-pid="Hx0fYzR2">删除任务栏上默认固定的邮件和商店（不是卸载）</li>
  <li data-pid="V_Mp2N29">清理无用磁力贴，或者缩小，以图将开始菜单宽度缩减为原来的 2/3。</li>
  <li data-pid="YCiYZJWw">禁止快速启动：经常会重启不干净。</li>
  <li data-pid="SzkdTk--">卸载5-8个左右系统捆绑应用。</li>
  <li data-pid="rOBqBVi5">尽量让系统更新帮你自动装驱动，原厂驱动包经常各种捆绑。</li>
  <li data-pid="1RVcM31r">装完驱动后打开任务管理器，选择启动，然后把所有随驱动安装一起安装的自动启动项目全部禁止（直接用任务管理器做不要安装任何电脑医生软件来搞）。</li>
  <li data-pid="JpTYTc5n">资源管理器里，允许查看文件扩展名并取消文件使用历史。</li>
  <li data-pid="cZesaZSu">修改注册表，删除2-3个系统默认的置顶文件夹，比如 “3D对象”</li>
  <li data-pid="J6wVsJh7">用windows 更新安装完所有补丁。</li>
  <li data-pid="I_If7pVk">浏览一遍所有系统服务，禁用1-3个没鸟用的服务，比如某些自动更新服务。</li>
  <li data-pid="LWAwNTwG">此时应该做一次系统盘备份或者ghost。</li>
  <li data-pid="aQMOZdEA">打开添加删除windows功能，允许沙盒功能，网上下软件时尽量先在沙盒中安装，没病毒没捆绑时再正式安装。</li>
  <li data-pid="8kaDfMQj">每安装完一个软件，比如微信和qq，记得查看设置里是否有开机自动启动，有的话就禁止掉。</li>
  <li data-pid="50pNIm-q">避免安装一些黑名单内的软件，比如百度360全家桶，搜狗输入法及好几个国内杀毒软件，各种国内压缩软件。</li>
  <li data-pid="HXPdAnSY">创建账号不要默认创建在线账户，先创建本地离线用户，再绑定微软账号。</li>
  <li data-pid="sb810wjb">安装网盘软件时要记得关闭后台p2p加速服务。</li>
  <li data-pid="xd9rZuZr">新装软件时不要盲目自动安装，先用手动安装查看会装些什么，有没有捆绑？有没有shell集成？每一个选项都要仔细阅读，不要被按钮的位置和颜色迷惑，看清楚按钮上面的字，反复确认三遍再点击。</li>
 </ol>
 <p data-pid="ui4AkZev">国内用 Windows 就像骑着摩托飙高速，看着很爽，稍不留神就万劫不复。所以需要时刻留心，小心驶得万年船。</p>
