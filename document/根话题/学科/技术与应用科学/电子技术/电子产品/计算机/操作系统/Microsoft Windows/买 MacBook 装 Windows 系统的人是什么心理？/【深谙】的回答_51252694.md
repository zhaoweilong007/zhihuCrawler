# 买 MacBook 装 Windows 系统的人是什么心理？
- 点赞数：15139
- 更新时间：2015年06月13日21时31分58秒
- 回答url：https://www.zhihu.com/question/31198093/answer/51252694
<body>
 <p data-pid="HRw95o9n">我曾经在火车站候车室休息的时候，看到我旁边的小孩在玩Macbook Air，系统装的却是Win7。</p>
 <p data-pid="eh3Jqpeo">我问这个小孩儿，你苹果电脑居然装Win7，太low了吧？</p>
 <p data-pid="Zwh6kMjn">小孩说：“我爷爷一直在Macbook上使用Windows系统，我爷爷今年97岁了，长寿吧。”</p>
 <p data-pid="6KATGHtK">“你的意思是说，在Mac上使用Windows系统使人长寿？”</p>
 <p data-pid="PCDZwAdB">小孩说：“不是的，我爷爷长寿是因为他从来不多管闲事。”</p>
