# 有没有觉得 Windows 越做越烂，尤其是 Win11？
- 点赞数：5507
- 更新时间：2021年10月17日12时05分25秒
- 回答url：https://www.zhihu.com/question/476531075/answer/2174819572
<body>
 <p data-pid="lYf2W04N">WIN11能够安装安卓程序，我还挺害怕这点的。</p>
 <p data-pid="p0PncFYO">目前在桌面端，我们通过一个浏览器（不管是谷歌edge还是360），可以看知乎，搜百度，看B站，看直播等等，就这一个软件我们就可以自如的在网上冲浪，其他就是工具软件和游戏安装下。</p>
 <p data-pid="Lfez2Mzc">而手机则要装一堆APP，现在手机浏览器的作用越来越小越来越小简直就成了个新闻软件，我手机就不没装百度APP，可是如果有什么要查一下用浏览器百度，很多点进去直接指引到百度APP下载界面，你不装就看不了；还有比如临时要查个公司，于是浏览器搜索天眼查，点进去根本没有查询界面，就是一个下载界面，你不下载手机上就是查不了。可是我就是偶尔用用我根本不想装个APP。我觉得这简直可怕，手机浏览器基本就废掉了。</p>
 <p data-pid="d9GU_5JK">所以如果WIN11后边全面放开可以安装安卓APP，那么很多公司会不会也来这么一手，浏览器根本不提供功能入口，你点进去就是要你下载安装软件，本来一个浏览器能搞定的事，我得装个B站APP，装个百度APP，装个斗鱼APP，装个知乎APP……我很不希望会这样。</p>
</body>
