# Windows 没有 mac OS 流畅吗，为什么？
- 点赞数：5529
- 更新时间：2020年09月05日03时53分12秒
- 回答url：https://www.zhihu.com/question/419188932/answer/1453822912
<body>
 <p data-pid="Tif07hQb">因为你穷，买不起真正的高端windows本。</p>
 <p data-pid="RVZKsv29">只买得起mac本的人当然会有这种莫名其妙的结论。</p>
 <p data-pid="eVy1SI-E">像我这种有钱人，win本选高端产品线诸如xps和hp specture的满配，macbook选最贵价的16寸满配，台式3990x，内存256g，硬盘nvme 2t*2，从来不考虑这个牌子好那个牌子流畅之类的破事，也从来不上知乎问各种没营养的问题。</p>
 <p data-pid="DYx9IxBb">我寻思有些苹果粉作祟的优越感由来不就是价格么？要用魔法打败魔法。老子最便宜的零件都比你的mac贵，建议往后靠靠。</p>
