# 为什么 macOS 比 Windows 稳定？
- 点赞数：6761
- 更新时间：2023年10月24日12时02分26秒
- 回答url：https://www.zhihu.com/question/578996067/answer/2979988991
<body>
 <p data-pid="z-jmUEFF">因为windows出问题了会蓝屏，老老实实承认：我出问题了，问题是0x00XXXX，你查查原因。</p>
 <p data-pid="pPUvTKLl">macOS出问题了会偷摸的重启，并且把你之前开着的程序都重新打开，假装没出问题。</p>
 <hr>
 <p data-pid="l82yzzxX">好多人评论说：mac多么多么稳定，用了多少年都没有崩溃过。你到底用过mac没有。</p>
 <p data-pid="BMwOsQJ2">我想说，这个东西因人而异，跟电脑负载有关。不能用个例代表别人。</p>
 <p data-pid="1rwP7k4F">贴个我电脑的cores证明一下，我不在laptop上编译调试，写代码都是在开发机上。这个只是日常使用，所有的core都是系统崩溃。</p>
