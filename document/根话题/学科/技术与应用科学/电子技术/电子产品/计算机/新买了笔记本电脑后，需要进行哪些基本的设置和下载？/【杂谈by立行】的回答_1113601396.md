 <h2><b>一 系统优化</b></h2>
 <h3><b>1.1 软件卸载</b></h3>
 <p data-pid="w5Ma4vnp">新买的电脑或者重装的系统可能会有一些预装软件，如果对你而言软件无用的话，建议右键直接卸载。</p>
 <p data-pid="V2RFWNvW">列几个常见的预装软件</p>
 <ul>
  <li data-pid="VyZqE5CM">Microsoft Store 里面的某些游戏<br></li>
  <li data-pid="jJWQFjdx">McAfee迈克菲全方位实时保护<br></li>
  <li data-pid="1yk8g6yW">纸牌游戏<br></li>
  <li data-pid="4tI-M22K">XBOX(如果不玩游戏的话)<br>
    如果不知道一个软件有什么用，请自行复制查询，或者干脆不管它。<br>
    关于常见品牌的预装软件及其作用，可以参考<a href="https://link.zhihu.com/?target=https%3A//www.windows10.pro/which-notebook-preinstalled-software-can-be-uninstalled/" class=" wrap external" target="_blank" rel="nofollow noreferrer">这篇文章</a><br></li>
 </ul>
 <h3><b>1.2 自启动管理</b></h3>
 <h3><b>1.2.1 管理自启动</b></h3>
 <p data-pid="80_16qS4">同时按住 Ctrl+Shift+Esc （都在你键盘左侧）调出任务管理器，左键单击『启动』</p>
 <p data-pid="1RVkR7XD">在你需要禁止的项目上右键调出菜单即可禁用该自启动项目。</p>

 <h3><b>1.4.2 OneDrive 设定位置</b></h3>
 <p data-pid="sKu8OgNq">云盘应该是当下使用电脑的一个标配。文件同步/备份都离不开一份安全可靠方便的云服务。</p>
 <p data-pid="nbkZWWnq">OneDrive 是微软亲儿子，和 Win10 的结合比较紧密。普通用户有 5G 的空间可用，Office 365 用户有 1T 空间可用。建议重要文件及时放到 OneDrive 或者是其他云盘里备份/同步。</p>
 <p data-pid="XEvaVKzX">这里主要是提醒诸位第一次设置 OneDrive <b>「请谨慎选择默认位置」</b>，之后无法修改。C 盘是 128G 的朋友就最好别放在 C 盘了，或者及时开启 『按需同步』的功能减少本地空间占用。</p>

 <h3><b>1.6.1 创建还原点</b></h3>

 <p data-pid="pURr_qYv">建议装完系统做好基本配置并且安装好常用软件时手动创建一个还原点，方便出了问题时还原；同时也建议折腾电脑前创建一个还原点，给自己留一个退路，毕竟重装系统成本其实挺高的。</p>

 <h3><b>2.0 多说几句</b></h3>
 <p data-pid="1SpOOfJ3"><b>「工具最终还是为人服务的」</b>。不要本末倒置，沉迷于工具本身。</p>
 <p data-pid="Pf1KfpTg"><b>「利用工具优化自身的工作流才是我们的最终目的」</b>。一款软件的某个功能再厉害，但是如果它不能优化你的工作流，这款软件于你的价值就是 0 。</p>

 <ol>
  <li data-pid="fzSxx5RU">电脑软件请尽量<b>「去官网」</b>或者 GitHub下载，小心各种下载站。</li>
  <li data-pid="YiIsqKjk">请谨慎选择国产软件安装。</li>
  <li data-pid="NX-n_haJ">国产软件请<b>「不要点击」</b>『快速安装』、『极速安装』这类选项。</li>

  <li data-pid="cpYSw7g-">电脑软件利用软件自身的更新功能就可以了，<b>「无需使用」</b>软件管家这类产品进行更新管理。</li>
  <li data-pid="eLBGE2ZF">小众软件做好安装包备份，第三部分也会讲这个问题。</li>

 <h2><b>三 日常使用</b></h2>
 <h3><b>3.1 软件新装</b></h3>
 <h3><b>3.1.0 下载</b></h3>
 <p data-pid="lFJna2Yq">文章开头提过，这里再强调一遍。</p>
 <ul>
  <li data-pid="VA_Ocf8p">尽量<b>「去官网」</b>或者 GitHub 下载，遇上流氓的概率会小很多。</li>
  <li data-pid="poTcJrCj">建议如果空间足够的话，下载的安装包不要删除，直接移入“重装系统备份”文件夹。重装的时候能减少很多麻烦。</li>
  <li data-pid="CezwPoBc">小众软件、小工具记得做好<b>「安装包备份」</b>。</li>
  <li data-pid="6yjGgpb0">大型软件最好<b>「本地」</b>留也留有备份，不然几个 G 的文件再从网上拉下来可能会很耗时。</li>
 </ul>
 <h3><b>3.1.1 安装</b></h3>
 <ul>
  <li data-pid="5ysW55NE">所有国产软件，请<b>「不要选择」</b>『快速安装』、『一键安装』这类选项，请一步一步检查。</li>
  <li data-pid="Tz4oFmME">安装位置最好可以统一管理，具体可以参考奶酪的这篇文章：<a href="https://link.zhihu.com/?target=https%3A//www.runningcheese.com/windows" class=" wrap external" target="_blank" rel="nofollow noreferrer">大道至简，如何高效使用 Windows？</a>。当然，懒得管理的话选择默认位置也没关系。</li>
  <li data-pid="IXEsBSLS">安装过后记得<b>「去自启动管理检查」</b>一下，不要相信软件自身的自启动选项。</li>
  <li data-pid="jJuQFGh1">可以利用开始菜单作为软件入口，为软件进行分类管理。</li>
 </ul>
 <h3><b>3.2 文件备份</b></h3>
 <p data-pid="02E6tPq-">电脑丢了，无非就是损失一点金钱，日后也能赚回来。但数据要是丢了，<b>「你花再多的钱也找不回来了」</b>。</p>
 <p data-pid="9LiwnnFb">再怎么重视数据安全也不为过，这方面展开讲也可以单开一个栏目，限于主题就简单提几嘴网络备份的选择。</p>
 <p data-pid="CN6_8VMH">普通 Windows 用户其实用好 OneDrive 做备份就足够了，5G 的空间不够的话可以拼车 Microsoft 365 家庭版，一年 50 多元，1T 空间可用。 当然，也可以直接购买 Microsoft365 个人版，省去很多麻烦。</p><a data-draft-node="block" data-draft-type="mcn-link-card" data-mcn-id="1230571857822552064"></a>
 <p data-pid="aHMSTXTz">这里的 Microsoft 365 不光是 OneDrive 的 1T 空间，还包括 Word、Excel、PowerPoint 三件套的多平台使用权。</p>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-78dc45c435f9b464db2be7223a574858_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="789" data-rawheight="1054" data-original-token="v2-666e65096d0fc959b87adb11c3d04daa" data-default-watermark-src="https://picx.zhimg.com/50/v2-17e9c9fe31de951925759e1ee34ab631_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="789" data-original="https://pic1.zhimg.com/v2-78dc45c435f9b464db2be7223a574858_r.jpg?source=1940ef5c">
  <figcaption>
   Office 产品对比
  </figcaption>
 </figure>
 <p data-pid="S9P5tgkC"><b>「非常不建议你使用淘宝几块钱 5T 的 OneDrive 账号进行文件备份」</b>，这种账号是虚假注册的，随时可能被微软<b>「封号」</b>。而且账号里的文件可以被系统管理员<b>「随时查看和删除」</b>，毫无隐私性可言。</p>
 <p data-pid="PYZ61kNr">至于百度网盘，如果你是会员的话，Windows 倒也支持最多 5 个本地文件备份，这也不失为一种选择。</p>
 <p data-pid="gzD5g3wZ">而坚果云，上面也提到过，如果有多设备同步的需求，它在国内会是首选。如果没有这方面的需求，也可以下载尝试一下，免费用户不限制空间、速度，只限制容量。（坚果云看到请给我续费打折，谢谢）</p>

 <h3><b>3.3 系统安全</b></h3>
 <p data-pid="M3cZCTaL">虽然大家都劝说微软自身的『Windows 安全中心』足够维持系统安全，没必要安装第三方安全软件。</p>
 <p data-pid="x6TeKT-e">但如果你对电脑了解不是很多，或者面对流氓弹窗束手无策，我还是建议你安装第三方安全软件。『Windows 安全中心』<b>「防得住病毒，防不住流氓」</b>。</p>

 <h3><b>3.4 发现并解决工作流上的痛点</b></h3>
 <p data-pid="xEe5pAVC">尽管上面推荐了 70+ 的软件，但是其实本文价值最高的部分藏在下面的几段话里面。</p>
 <p data-pid="uL75oss_">太阳底下无新鲜事，其实<b>「我们在操作电脑时遇到的问题里面 80% 前人都遇到过」</b>，并且可能给出了软件或者解决方案。</p>
 <p data-pid="dpI3XGlM">你需要做的其实就是：</p>

  <li data-pid="MIf_vXrE">不断发现工作流中的痛点、可优化点</li>
  <li data-pid="FnQ5thhY">以合适的方式进行搜索、提问</li>
  <li data-pid="D5Eez7mO">找到/创造解决方案并记录（印象笔记剪藏或者打印成 PDF）</li>
 </ul>
