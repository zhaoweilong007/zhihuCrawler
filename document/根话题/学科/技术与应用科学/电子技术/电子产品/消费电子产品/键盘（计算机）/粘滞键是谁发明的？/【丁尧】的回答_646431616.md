# 粘滞键是谁发明的？
- 点赞数：7671
- 更新时间：2021年12月12日17时22分26秒
- 回答url：https://www.zhihu.com/question/29748629/answer/646431616
<body>
 <p data-pid="zNuPX_mQ">粘滞键是我博士老板 (Gregg Vanderheiden) 发明的，因此我大致了解发明粘滞键的历史，以及各个操作系统决定预安装粘滞键的背景和时间线，或许可以解答题主的疑问。</p>
 <h2>粘滞键有什么用</h2>
 <p data-pid="kkFOXVbE">粘滞键是80年代初发明的，当时叫 1-Finger，顾名思义初衷是帮助只有「一根手指」的用户。比如键入「@」，需要同时按住 Shift 和 2，这时候粘滞键就可以起作用了。当然，这个功能帮助到的人群远不止「一根手指」。有个天生双臂缺失的大爷给我演示过他是怎么用电脑的，粘滞键发明前他是怎么解决 Ctrl Shift 这些功能键的：嘴里咬一根木棍儿，一头缠几圈橡皮筋——这是他操作电脑唯一的工具；然后他让家人帮他把5个一美分硬币摞在一起，用透明胶缠住，一坨硬币的一面再缠上一个小铁环，让木棍儿的橡皮头刚好可以穿过。需要按住 Shift 的时候，用木棍儿把一坨硬币吊到 Shift 上，重量刚好可以把键按下，再去按别的键，用完再把硬币吊走。</p>
 <h2>现在有人用粘滞键吗</h2>
 <p data-pid="LxWIFAL7">前面说到粘滞键一开始叫 1-Finger，容易理解但不够准确，因为发现很多其他用户群体也在用，比如：</p>
 <ul>
  <li data-pid="Z5zhTKq8">口杆 (mouth stick) 用户</li>
  <li data-pid="MsG4Rd9-">头杆 (head stick) 用户</li>
  <li data-pid="9DEhZGTm">关节炎用户</li>
  <li data-pid="0KA6V2Jy">腱鞘炎用户</li>
  <li data-pid="6S6NV3a1">手抖的用户</li>
  <li data-pid="mmGPuaxC">手小的用户</li>
  <li data-pid="vObS5jXM">手部力量弱的用户</li>
  <li data-pid="NAwLYnlM">动作比较慢没办法持续按下一个键的用户</li>
  <li data-pid="hu7nV1tS">……</li>
 </ul>
 <p data-pid="OC5fbvBS">加上现在老龄化的趋势，需要经常用到粘滞键的用户可能不比常用功能键的 gamer 少呢。</p>
 <h2>为什么不去掉「连按五下」快捷启动</h2>
 <p data-pid="9p1u8SBz">我认为这个问题既有历史原因，也有目标用户基数的考量。从历史看，粘滞键最早是给DOS开发的（命令行界面），后来推广到最早期的 Apple 和 Windows 图形界面系统，那时候可能还没有激烈的对战游戏需要连按 Shift 或 Ctrl 造成按键冲突。后来即使有一些 gamer 觉得连按启动很烦，这也是可以关掉一劳永逸的。而且 gamer 大多是 power user，权衡之下，让 power user 关掉这个功能，比去掉快捷启动、让残疾人花一番功夫才能开启这个功能，可能更容易让广大用户接受吧。</p>
 <h2>怎样彻底避免触发粘滞键</h2>
 <p data-pid="R9FFb_d_">有不少评论说，粘滞键关了还会再触发。这里追加说明，如何彻底避免触发粘滞键。</p>
 <p data-pid="uriFiTMJ">Windows 10 里，粘滞键第一次触发会弹出这个窗口：</p>

  <figcaption>
   Windows 粘滞键确认启动的窗口截屏。截屏显示：你想启用粘滞键吗？通过一次按一个键，粘滞键允许使用 Shift、Ctrl、Alt 或 Windows 徽标键。启用粘滞键的键盘快捷方式是按 5 次 Shift 键。以及「是」「否」按钮。
  </figcaption>

 <p data-pid="YncZUxY5">选「否」只是选择<b>这次</b>不开启粘滞键，下次连击5次还是会弹出这个窗口。点上面窗口里的链接，反选「允许使用快捷启动」，就能彻底避免连击触发了。</p>

 </figure>
 <p data-pid="VobvNR6v">有评论提到能控制系统设置的软件，比如360管家恢复默认设置，或者一些公用电脑会自动恢复默认设置（比如 Deep Freeze ），就需要在360和 Deep Freeze 里设置了。</p>
