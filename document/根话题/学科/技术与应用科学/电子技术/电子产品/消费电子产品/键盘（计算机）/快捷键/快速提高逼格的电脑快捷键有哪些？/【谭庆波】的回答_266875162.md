# 快速提高逼格的电脑快捷键有哪些？
- 点赞数：39803

<body>
 <h2>1 电脑锁屏</h2>
 <ul>
  <li data-pid="svO6I_Gh">有些时候，需要暂时离开座位去处理其他事，可是电脑还有数据再跑。</li>
  <li data-pid="f9ae_y1U">关掉的话，数据就白跑了，不关的话，又不想让别人看到我电脑的资料。</li>
  <li data-pid="WUQ7IVLS">那么就按住<b>windows键后，再按L键。</b></li>
  <li data-pid="dR_uDCiG">这样电脑就直接锁屏了，不用担心电脑的资料外泄。</li>
 </ul>

  <img src="https://picx.zhimg.com/50/v2-ca805d0c1c3abec4383f7ac13a2799ce_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="469" data-rawheight="329" data-original-token="v2-ca805d0c1c3abec4383f7ac13a2799ce" class="origin_image zh-lightbox-thumb" width="469" data-original="https://pica.zhimg.com/v2-ca805d0c1c3abec4383f7ac13a2799ce_r.jpg?source=1940ef5c">
 </figure>

 <h2>2 步骤记录器</h2>
 <ul>
  <li data-pid="L8b3Thon">实验室来学弟了，老师让你带带，你把一些软件演示了好几次，他还是没学会，而你又没更多的时间再去手动演示。</li>
  <li data-pid="5sWU7tCh">这个时候，<b>步骤计步器</b>就体现它的作用了。</li>
  <li data-pid="8FgMm4vL"><b>按下windows键+R，输入psr.exe回车，然后点击“开始记录”就可以了。</b></li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-97296f608c9fdee20d15d0de5d82a9f1_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="614" data-rawheight="166" data-original-token="v2-97296f608c9fdee20d15d0de5d82a9f1" class="origin_image zh-lightbox-thumb" width="614" data-original="https://picx.zhimg.com/v2-97296f608c9fdee20d15d0de5d82a9f1_r.jpg?source=1940ef5c">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="0tTdTyYy"><b>它会记录下你做的所有操作，然后生成一个图文并茂的详细文档,单击停止记录后，会将详细记录文档保存到一个压缩文件中。</b></li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-1a10f8694e02c8981eab9404379f4a50_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="271" data-rawheight="163" data-original-token="v2-1a10f8694e02c8981eab9404379f4a50" class="content_image" width="271">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <h2>3 快速显示桌面</h2>
 <ul>
  <li data-pid="3kbCNxOG">在实验室里，正高兴的开黑玩游戏的时候，老师突然闯进来了。</li>
  <li data-pid="hPYY2qpl">这时让他看到自己在玩游戏肯定不好，<b>游戏界面又没法一下子最小化，</b>怎么办？</li>
  <li data-pid="YOUPhuKA">别紧张，直接<b>按下Windows键和D键</b>，桌面出现了。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-4cdd194a57c4d0478d40242abf7b73ab_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="429" data-rawheight="226" data-original-token="v2-4cdd194a57c4d0478d40242abf7b73ab" class="origin_image zh-lightbox-thumb" width="429" data-original="https://pica.zhimg.com/v2-4cdd194a57c4d0478d40242abf7b73ab_r.jpg?source=1940ef5c">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>

 <h2>4 屏幕放大镜</h2>
 <ul>
  <li data-pid="1uVzUBzC">有时出门忘记带眼镜了，电脑上的字太小，看不清怎么办？</li>
  <li data-pid="NhUAwvJn">别着急，教你使用窗口放大镜，立刻解决这一问题。</li>
  <li data-pid="p_UxheQG"><b>按住windows键，然后使劲连续按加号键</b></li>
  <li data-pid="jg6NTcF-">屏幕就会变得很大，直到你满意为止。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-507ebe4054f1e0f0ac26af2f75e0a2fc_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="640" data-rawheight="453" data-original-token="v2-507ebe4054f1e0f0ac26af2f75e0a2fc" class="origin_image zh-lightbox-thumb" width="640" data-original="https://pica.zhimg.com/v2-507ebe4054f1e0f0ac26af2f75e0a2fc_r.jpg?source=1940ef5c">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="qRPPCuNJ">然后<b>按住windows键，连续按减号键</b>，屏幕又可以回到原来的样子。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <h2>5 新建文件夹</h2>
 <ul>
  <li data-pid="etOazmrE">通常新建文件夹都需要3个步骤，“<b>右键-&gt;新建-&gt;文件夹</b>”,费时费力，效率太低了，那么怎么办呢？</li>
  <li data-pid="dAma3v5k"><b>按住“Ctrl+Shift+N”键，立刻新建文件夹。</b></li>
 </ul>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-28d2205ff1e3c5d6990993c2378de5f3_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="206" data-rawheight="165" data-original-token="v2-28d2205ff1e3c5d6990993c2378de5f3" class="content_image" width="206">

 <h2>6 屏幕虚拟键盘</h2>
 <ul>
  <li data-pid="v-xItnQy">有时突然急需要一台电脑操作文件，但发现它们的键盘居然是坏的。<b>稳住，别慌~</b></li>
  <li data-pid="anAn6-3A"><b>这时候，用鼠标打开运行栏，输入osk，电脑自动会出现炫酷虚拟键盘！</b></li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-0028d86ff56e2c06dbfabaf9789a27c5_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="640" data-rawheight="250" data-original-token="v2-0028d86ff56e2c06dbfabaf9789a27c5" class="origin_image zh-lightbox-thumb" width="640" data-original="https://picx.zhimg.com/v2-0028d86ff56e2c06dbfabaf9789a27c5_r.jpg?source=1940ef5c">
 </figure>

 <h2>7 关闭浏览器当前页面和当前文档</h2>
 <ul>
  <li data-pid="Q5vk4oPy">浏览器打开了很多页面，用鼠标挨个去关太费劲，没关系，按<b>Ctrl+W</b>键，快速帮你搞定。</li>
  <li data-pid="vX2G5vmD">打开的文档太多，找不到你想要的文档，拖慢电脑速度，买关系，按<b>Ctrl+W</b>键，帮你关掉它。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-7b103697d88a6a302b145ee228db2dfe_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="624" data-rawheight="327" data-original-token="v2-7b103697d88a6a302b145ee228db2dfe" class="origin_image zh-lightbox-thumb" width="624" data-original="https://picx.zhimg.com/v2-7b103697d88a6a302b145ee228db2dfe_r.jpg?source=1940ef5c">

 <h2>8 一次性快速设置</h2>
 <ul>
  <li data-pid="yRMWS6uq">如果有很多设置需要重新操作，比如调整显示器亮度、音量大小，打开无线网，还要看本本电池电量。</li>
  <li data-pid="CFhr_rq9">只需要按下<b>Windows键+X</b>，一次性满足所有要求。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-4bec98fe12045a687d44a932223b641e_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="640" data-rawheight="315" data-original-token="v2-4bec98fe12045a687d44a932223b641e" class="origin_image zh-lightbox-thumb" width="640" data-original="https://pica.zhimg.com/v2-4bec98fe12045a687d44a932223b641e_r.jpg?source=1940ef5c">
 </figure>

 <p class="ztext-empty-paragraph"><br></p>
 <h2>9 清除浏览器缓存刷新</h2>
 <ul>
  <li data-pid="UyJEn13M">很多程序员在写网页的时候，明明源码已经改过来了，可是浏览器上显示的页面还是不多，这个时候你就得考虑是不是缓存的问题了。</li>
  <li data-pid="BV0kDe1z"><b>按住“Ctrl+F5”键，清缓存刷新，帮你解决因为缓存而带来的问题。</b></li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <h2>10 文档打印</h2>
 <ul>
  <li data-pid="DvUTmrdm">去打印店打印东西，打印店的小哥太忙没时间招呼你。</li>
  <li data-pid="WxP1nf2V">别急着，没有他，自己也能打印。</li>
  <li data-pid="dHfq-kwk"><b>按住“Ctrl+P”键，帮你打印文档。</b></li>
 </ul>
