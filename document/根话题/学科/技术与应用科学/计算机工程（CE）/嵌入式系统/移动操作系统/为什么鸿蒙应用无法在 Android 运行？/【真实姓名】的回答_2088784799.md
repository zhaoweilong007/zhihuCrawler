# 为什么鸿蒙应用无法在 Android 运行？
- 点赞数：5414
- 更新时间：2021年08月28日11时40分59秒
- 回答url：https://www.zhihu.com/question/482631812/answer/2088784799
<body>
 <h2>因为没有找到诀窍，我分享你一些经验：</h2>
 <ol>
  <li data-pid="UpS7UPVt">找钓点：春秋冬季找开阔水边，夏季找水草边或流动水边平缓处。</li>
  <li data-pid="u2iKaDaE">打窝：先试钓几下，确定水面下竿点，洒下窝料。</li>
  <li data-pid="oEEImS1u">有耐心：窝料打下去以后要等鱼群，一般要半小时至两小时，这段时间千万不能走开，这可是黄金时间。</li>
  <li data-pid="Ge8BfhVh">合适的鱼钩鱼线：野钓一般用1.0号主线，2-3号钩。</li>
  <li data-pid="nz8ekV2S">要投准：垂钓时，鱼钩要对准窝点，不要投偏。</li>
 </ol>
