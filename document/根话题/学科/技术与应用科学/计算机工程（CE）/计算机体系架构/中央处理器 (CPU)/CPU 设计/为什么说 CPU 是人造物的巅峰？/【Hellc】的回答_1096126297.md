# 为什么说 CPU 是人造物的巅峰？
- 点赞数：12113
- 更新时间：2020年03月27日12时55分34秒
- 回答url：https://www.zhihu.com/question/378861322/answer/1096126297
<body>
 <p data-pid="cx3WXP-e">以新冠病毒为代表的病毒，其整体<i>尺寸</i>一般在30-80nm。</p>
 <p data-pid="2axJfZ6C">而Intel长期被人嘲讽”已经落后“的工艺制程为14nm。</p>
 <p data-pid="d4uKZPsm">而以14nm作为最小设计尺寸（很不严谨的说法，但可以这么粗略理解）的晶体管，在一枚小小的CPU中有几十亿到几百亿个，与全人类数量在同一个量级上。</p>
 <p data-pid="V_ZyS97v">同时，需要小白注意的是，这些晶体管不是通过物理或化学反应批量生成的重复物（如一滴水中有几十亿个水分子），而是<b>每一个都有特定设计以及明确功能的实体</b>，他们是一个个被画出来并精确地制造在硅晶圆上的。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="IMdwt0od">当然，最应让人感到震撼的是，在无数科学家与工程师的努力下，在波澜壮阔的人类产业链合作之下，这个人造物的巅峰，仅以<b>最低几角钱</b>最高不过几千元的价格，就能来到你的手上。</p>
 <p class="ztext-empty-paragraph"><br></p>
