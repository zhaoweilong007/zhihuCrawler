# 如何看待小米最新发布的冰封背夹散热器不敌萝卜?
- 点赞数：5889
- 更新时间：2020年06月25日21时47分11秒
- 回答url：https://www.zhihu.com/question/372904943/answer/1024459682
<body>
 <p data-pid="LSPH73pR">“萝卜散热”是图吧渊源已久的一个梗，这次算是老梗新吃了。</p>
 <p data-pid="vyH1e04P">准确的来说，应该是<b>白萝卜</b>的散热能力比部分散热器强，因为白萝卜的<b>含水量非常高</b>，所以<b>热容</b>也很高，加之切面会<b>渗水</b>，水作为<b>填充剂大幅提升热交换效率</b>，白萝卜曾经在15分钟测试中<b>战胜了大量下压式风冷</b>，因而白萝卜散热一战成名，被誉为<b>图吧基佬专属散热器</b>。</p>
 <p data-pid="obFdk-3C">所以，这种针对手机的高温差热交换散热器打不赢萝卜，实属正常。</p>
