 </ul>
 <blockquote data-pid="4_CMTmZW">

 <ul>

 </blockquote>
 <ul>

 </ul>
 <blockquote data-pid="bfL77t3X">

 <ul>
  <li data-pid="IOzvGshe"><b>我轻蔑而又不屑一顾地看了他们一眼。</b></li>
 </ul>

 </blockquote>
 <ul>
  <li data-pid="_oEKJElV"><b>语气中是那么的简单而又轻松，确实，事情的处理结果，最多不就是她们赔礼道歉吗？还能是什么呢？</b></li>

 </ul>
 <blockquote data-pid="SeHJphg5">

  <b>又是焦急的等待。</b>

 <h2>⭕<b>吃瓜</b></h2>
 <ul>
  <li data-pid="YEf0di3a"><b>随后，年龄较小女生开始拿起手机开始给朋友发微信语音消息，大概是说了一下自己地铁上的遭遇，「<u>给你们吃个瓜……</u>」</b></li>
 </ul>
 <blockquote data-pid="Qf69AVB_">
  <b>她十分得意得说了一下「<u>我们是对其他人负责嘛</u>」「<u>他其实离我们很远，不会是偷拍我们</u>」之类的话语。</b>
 </blockquote>
 <ul>
  <li data-pid="5Tijsxib"><b>具体的内容由于是窃窃私语，不得而知。</b></li>
 </ul>
 <blockquote data-pid="Iu2EYVS4">
  <b>听到这里，我本想友善地提醒一句「在事情没有结果之前，不能直接宣传开来，这样是不负责任且不严谨的。」</b>
 </blockquote>
 <ul>
  <li data-pid="Q_1SiB5y"><b>确实，在这个网络社会，大家吃瓜的心情，分享瓜的热情，似乎根本阻挡不住。还是不要阻挡一件能让别人开心的事情。</b></li>
 </ul>
 <blockquote data-pid="z27zAUFg">
  <b>几分钟过后，我拿起手机来，看着手机还剩一点点电了，在朋友微信小群里说了一句「我快到了。」</b>
 </blockquote>
 <ul>
  <li data-pid="Uw4ri_BV"><b>随后，两位女生开始议论起来。</b></li>
 </ul>
 <blockquote data-pid="McCUXi5m">
  <b>其中年龄大的女生更是把头伸了过来，十分吃瓜的问我在干什么，一副十分担心我在销毁证据的样子。</b>
 </blockquote>
 <ul>
  <li data-pid="Gis1HjQH"><b>随后我把手机拿给他看，给他看了一下我的微信，给她解释我在给朋友群发消息，她十分不屑的说道，「<u>我不是要看你微信消息，我看你微信消息干嘛</u>！」</b></li>
 </ul>
 <blockquote data-pid="yELbcfms">

 </ul>
 <blockquote data-pid="LdTyNyYx">

 </ul>
 <blockquote data-pid="1umY9WOt">

 </blockquote>
 <ul>
  <li data-pid="Ci_iYHgM"><b>还是有些许不舒服的，警察到了现场，向她们询问了情况，我就在旁边，不管是合理的不合理的，我都听着，出于礼貌，我没有打断。</b></li>
 </ul>
 <blockquote data-pid="8MBr36Jc">
  <b>在两个女生说完之后，我十分礼貌的说我需要补充一下现场情况，却没有一个发声的机会。</b>
 </blockquote>
 <ul>
  <li data-pid="K-t_HVqY"><b>就这样，我被前后警察包围着，我们一起上了扶梯，又一起下扶梯。</b></li>
 </ul>
 <blockquote data-pid="tRTK0CoZ">
  <b>来到地铁上后，警察指示其一位同事看着我坐在列车上，然后自己则是过去再次询问了另两位女生当时的场景。</b>
 </blockquote>
 <ul>
  <li data-pid="I0B5WYiL"><b>过了一会儿，警察A走了过来，坐在了我对面的座椅上，然后仔细观察着什么。</b></li>
 </ul>
 <blockquote data-pid="CjaHjkea">
  <b>地铁大概驶出了两个站台，警察好像明白了什么，拿出手机，对着我的鞋子的位置录了视频。</b>
 </blockquote>
 <ul>
  <li data-pid="9vjIacVP"><b>终于，漫长的列车终于达到了火车南站，伴随着四位警察，一行人全部去往警务室。</b></li>
 </ul>
 <blockquote data-pid="wE_vnQNR">
  <b>在到达火车南站上楼梯的时候，警察A走到我旁边对我说「我想应该是误会了，你也不要有太多情绪。」</b>
 </blockquote>
 <ul>
  <li data-pid="FH9ebi9h"><b>我大概理解了警察A的意思，十分无奈摇了摇头，并对离我比较近的警察B说道「男生现在真的是弱势群体呀，你可以想一下，要是我在没有任何证据的情况下说一个女生偷拍我，结果会是怎样的呢？列车安全员潜意识里女生应该是弱势群体，在得到投诉后，第一时间将我控制住，但是，事实到底谁才是弱势群体呢？」</b></li>

 <blockquote data-pid="E6864SeY">
  <b>在出站到达警务室的时候，年级小的女生对另一个女生说道，「<u>这是我们第一次去警察局唉</u>，」随后露出来些许兴奋且开心的笑声。</b>
 </blockquote>
 <ul>
  <li data-pid="bONQ20Uf"><b>警察A拿出了手套及执法记录仪，在执法记录仪的拍摄下，在办公室内全员的目光下（包括两位女生），帮我脱掉了右脚的鞋子，仔细的检查了鞋带，鞋舌，鞋垫下位置，随后又脱掉了左脚的鞋子，重复了一遍操作。</b></li>
 </ul>
 <blockquote data-pid="S2j3gUzd">
  <b>随后警察站了起来，对我和旁边的同事说道，「我已经检查过了，鞋子没有任何问题。」</b>
 </blockquote>
 <ul>
  <li data-pid="fJzJ06vC"><b>事情呢也应该是我猜想的那样，列车快到站时两边的广告牌照在鞋子上，折射出了绿光，让两个女生误以为是摄像头，然后列车到站，绿光消失了，两位女生误以为是男生关掉了摄像头。说着并把自己在地铁拍摄的同位置还原视频让大家看一看。</b></li>

 <blockquote data-pid="BrpuEXCs">
  <b>说着就走到两位女生面前，并把两位女生带了出去。</b>

 <h2>⭕调解</h2>
 <ul>
  <li data-pid="ajM_4BHl"><b>留在屋子里的警察开始转变了态度，对我说道「其实这种被怀疑是正常的，他们报警了我们也只能这样去做一个处理，我刚刚在地铁上就知道大概是怎么一回事了，你先看一下，缓和一下情绪，看看这个事情怎么处理呢，你也可以说一说你的想法和诉求。」</b></li>
 </ul>
 <blockquote data-pid="JLXl47g4">
  <b>随后我说道「首先我需要说明的是，请你们警察要理解我一下，这个事情我很气愤，但是我也很理性，全程都在配合包括列车乘务员，包括你们的工作。」</b>
 </blockquote>
 <ul>
  <li data-pid="Q9Zg-1OH"><b>「你确实是很配合我们的工作，感谢你们的理解！」</b></li>
 </ul>
 <blockquote data-pid="iYtW78La">
  <b>可能是被情绪左右了，警察较为诚恳的话我却觉得是带了一点官腔。</b>
 </blockquote>
 <ul>
  <li data-pid="C8dV6Ad-"><b>随后我补充道，「我诚恳的希望你们要明确一点，这个事情，你们处理的时候不要优先的去考虑和谐社会，把事情弄小来做一个调解，这样的话，这种事情以后还是会发生（自己也觉得，我这个事情本身不就是广州地铁事件的一个延续吗？所有的事情都想着压着捂着和谐社会，最后却是没有了警示作用，不和谐的事情一天一天的上演），而且对于我来说这是不公平的。」</b></li>
 </ul>
 <blockquote data-pid="e2-BedK-">
  <b>随后警察说道「确实我很同情你的遭遇，这件事情发生了我肯定是会对两人进行一个教育的。」「其实我也看的出来，你全程都比较配合我们，感谢你的配合，帅哥你学历应该也很高吧？」</b>
 </blockquote>
 <ul>
  <li data-pid="fXL4I0l0"><b>这让我十分诧异，我腼腆地说道「没有没有，我只是一个很普通的普通人，可能说话做事比较客气。」</b></li>
 </ul>
 <blockquote data-pid="bg9cREho">
  <b>警察A说道，「其实我们把你们叫过来，作为我们其实相当同情你的遭遇，」说到这里，我难过的看向了警察B，警察B在与我目光对视的时候，似乎给了我一个肯定的眼神。</b>
 </blockquote>
 <ul>
  <li data-pid="wPF8wPzi"><b>「任跟谁在地铁上，在大庭广众之下，在没有证据之下，被这样要求脱掉鞋子，接受检查，都是难以理解的，而且你是年轻小伙子，应该还是看起来比较有正气的。」</b></li>
 </ul>
 <blockquote data-pid="c1YYC7O2">
  <b>「我……」</b>
 </blockquote>
 <ul>
  <li data-pid="9cyfYnu-"><b>没有等我开口，警察A说道「我也明白你要说的意思，就好比我们经常说的那样，我们遵纪守法了一辈子，老老实实的充满善意的对待身边所有的人，到头来事情发生在了你自己的身上，却没有受到一点善意的对待，心里肯定是很难过的。」</b></li>
 </ul>
 <blockquote data-pid="HF4sh4VE">
  <b>「现在呢，我们尽力的去做一个调解，如果你满意这里的调解呢，我们就调解解决，你要是不满意呢，你可以按照法律程序去起诉他们，主张一个对你名誉，精神损失的赔偿。」</b>
 </blockquote>
 <ul>
  <li data-pid="t9S7DQet"><b>警察A继续问道，「那现在你这边有没有一个明确的诉求呢，比如他对你造成了精神损失，名誉损失，能不能你来要求她们对你一个经济赔偿。」</b></li>
 </ul>
 <blockquote data-pid="DAyrAyyK">
  <b>我说道，「经济是我不考虑的，我需要的是一个有态度的道歉，她们必须要认识到自己的错误，我不要钱，我觉得要钱的话是对我第二次侮辱。」</b>
 </blockquote>

 <h2>⭕道歉</h2>
 <ul>
  <li data-pid="NsQ3erTf"><b>随后警察A出去对两个女生说了些什么，然后把两人叫了进来。</b></li>
 </ul>
 <blockquote data-pid="KQhHnAA4">
  <b>年龄大一点的女生用飞快的语速说道「<u>帅哥，对不起</u>，」随后转身就要走出办公室。</b>
 </blockquote>
 <ul>
  <li data-pid="AkXmqbl0"><b>我当时十分诧异了，这一种道歉语气，还有这种深怕多带一秒的场景，真的让我一瞬间仿佛回到了上学时代，那时候调皮之后，老师让我们给他道歉的时候，我们就会拖着一个长长的语气说道，「老~师~~，对不起（老师两个字拖着腔调，对不起加快语速）。」</b></li>
 </ul>
 <blockquote data-pid="y8f4i25J">
  <b>也许也带一点情绪原因，让我马上联想到了她们之前说的话「<u>你这么自信干嘛，放心，我们错了我们会给你认错的</u>。」于是，「我不接受道歉，」我脱口而出。</b>
 </blockquote>
 <ul>
  <li data-pid="5xZlulQB"><b>后来我想了一下我这句话似乎是有表述上的错误，我正确的说法应该是，「你根本就没有道歉。」</b></li>
 </ul>
 <blockquote data-pid="8H6hvooJ">
  <b>警察A问道，「不接受道歉是什么意思？」</b>
 </blockquote>
 <ul>
  <li data-pid="3cWIKdE7"><b>我答到「就是他们没有道歉的态度呀，也没有认识到自己的错误。」</b></li>
 </ul>
 <blockquote data-pid="e9r7bAyq">
  <b>年龄小的女生看到这样，也就没有了要道歉的意思。</b>
 </blockquote>
 <ul>
  <li data-pid="nZZ9kvNs"><b>是呀，道歉似乎是为难他们了，现在办公室里面有着四个地铁警察，当着这么多人说句对不起，对于两个女生来说是多么难得一件事情呀。</b></li>
 </ul>
 <blockquote data-pid="DKU4pGYm">
  <b>而我自己就不一样，被别人当着全车乘客的面要求脱鞋，然后被铁路安全员「搀扶着走下了地铁」，再安安静静地站在角落，最后在一群警察的前后「保护」下走进了警察室，全程都没什么为难，因为你成熟，因为你配合，因为你守法。</b>
 </blockquote>

 <h2>⭕争论</h2>
 <ul>
  <li data-pid="jp9tfNiT"><b>随后，警察A开始把我们都叫到办公桌前，对我们说道，「现在我们要把事情的起因经过写一下，你们稍等一下，我写完了还需要你们签字确认。」</b></li>
 </ul>
 <blockquote data-pid="P8NKAWtU">
  <b>随后，站在后面的警察B开始尝试着劝导两位女生认识到自己的错误，还是要道歉处理这件事情。</b>
 </blockquote>
 <ul>
  <li data-pid="U5MthsoJ"><b>但是年龄小的女生始终目光呆滞，十分不屑目无表情的看着周围。</b></li>
 </ul>
 <blockquote data-pid="EJ9SooK0">
  <b>随后女生开始强调，「<u>这个我们又不是为了我们自己，我们是看到旁边其他人的隐私受到了影响。</u>」</b>
 </blockquote>
 <ul>
  <li data-pid="tkmpuyQQ"><b>听到这里，我感觉到了年龄小的女生没有丝毫的歉意，也并没有感觉到自己错误。</b></li>
 </ul>
 <blockquote data-pid="aLEhN09P">
  <b>于是我也开始不礼貌地插嘴说道「我可以给你还原一下当时的场景，右边一排，全都是男生，你们在最左边的一排的最远处，我站在列车的立着的扶手的地方。当时大家的衣着，除了年龄大一点的女生外，其他的女生都是穿的裤子。我面前站了一个阿姨，站在要下地铁的门口，年龄比较大，穿的裤子，手拉着一个小朋友，穿的校服，」随后我又强调了一下，这个场景是在她们说我偷拍后我留意并记下来的。</b>
 </blockquote>
 <ul>
  <li data-pid="Q90wHcsJ"><b>随后年龄小的女生开始暴躁起来，「<u>你身后好像还有一个女生……</u>」</b></li>
 </ul>
 <blockquote data-pid="NxD0fiD3">
  <b>随后我说道，「列车是有监控的，你可以随时去看监控，不要好像好像……」年龄大的女生似乎看到了事情如果这样发展下去她们会比较被动，于是使了一个眼色叫小女生不要再说了。</b>
 </blockquote>
 <ul>
  <li data-pid="uXhAsyS3"><b>我说道，「我需要明确的是，你的出发点并不是在为别人发生，在我站在墙前的时候，你一直在对你的朋友发消息，十分迫不及待的发表自己的见闻，我就开始不相信你是在帮别人发声。」</b></li>

 </blockquote>
 <ul>

  <b>随后，我跟着警察B走了出去，在上地铁附体时，我笑着对警察B说，「现在男生真是弱势群体呀。」</b>
