# 三分之二同意加装电梯后，是否无力回天？
- 点赞数：5883
- 更新时间：2023年05月03日21时47分02秒
- 回答url：https://www.zhihu.com/question/422169261/answer/3011088163
<body>
 <ol>
  <li data-pid="JDasCiDG">公示期间，书面，实名反对声明送到居委会。</li>
  <li data-pid="OIVI-P05">同时写挂号信到街道办，住建局，信访办，市长热线，省长热线，电话要录音，挂号信要拍照录音。</li>
  <li data-pid="OnbxCqGd">对方，包括居委会要剧中协调，即协商，不要出面，不要参加。</li>
  <li data-pid="ApI2icp-">自己要上报单位组织领导，得到领导批准后才能正式参加协商，但是领导很忙很忙。</li>
  <li data-pid="bwVi4fl2">不要以采光隐私等理由拒绝，不给任何理由，就是拒绝。</li>
  <li data-pid="7CX8hyCB">下面几条是，如果你按步骤完成了前面几步，依然强行安装，请走下面流程。</li>
  <li data-pid="NegWTYMm">在现场表达坚决反对但不要和任何人冲突。</li>
  <li data-pid="pMF__vE5">向住建局投诉没有安全审核就施工（没有哪个设计院的人敢在安全审核上签字，尤其是看房子）。</li>
  <li data-pid="sONA_nXX">楼上如果有行政企事业单位的，直接找它们上级部门的jw，书面反应侵害群众利益，不让群众安居乐业，保留所有记录，包括jw的收据。</li>
  <li data-pid="--iIs9Qc">拿着书面材料加上jw的收据等，复印若干份，到对方单位，发给大家，不要起冲突，不要祥林嫂一样，只说jw接受了。记得要实名，自己和对方全部实名。</li>
  <li data-pid="VHKcWPYc">以上不要添油加醋，要实事求是。</li>
  <li data-pid="ZLDTVTkV">去一次后，对方没反应，一个月（一定时间后）后再去一次，一次接一次，但是不要闹，切记。</li>
  <li data-pid="OcVOy_Hn">因为邻里关系不和谐，它就已经没有升迁的机会了。</li>
  <li data-pid="cdb5I3ek">直到它来找你道歉。</li>
  <li data-pid="10EeBMJk">它不来道歉，它的竞争对手会nong它，你放心，它有一堆竞争对手。</li>
  <li data-pid="FEKRluuE">当它损失超过收益后，就自然放弃了。</li>
  <li data-pid="XgW2iUGN">还有一点，不要协商，只要协商没有通过，你保留所有证据，谁审批同意，谁违法。不要听信网上说2/3同意就强制安装，这是假的，只要你保留所有记录和证据。</li>
  <li data-pid="19RP47wF">如果强行施工，投诉没有安全许可。民法典上，谁审批谁担责。谁建设，谁终身负责。</li>
  <li data-pid="s7XyF4X0">根据民法典，加装后卖出房子，如果没有在合同中加入电梯的责权移交新业主，那么老业主终身负责。</li>
  <li data-pid="KLuo519M">其它。</li>
 </ol>
