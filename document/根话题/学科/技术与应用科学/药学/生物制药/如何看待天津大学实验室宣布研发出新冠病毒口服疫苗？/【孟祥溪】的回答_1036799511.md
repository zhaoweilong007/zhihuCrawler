# 如何看待天津大学实验室宣布研发出新冠病毒口服疫苗？
- 点赞数：5259
- 更新时间：2020年02月25日12时33分58秒
- 回答url：https://www.zhihu.com/question/374459725/answer/1036799511
<body>
 <ol>
  <li data-pid="OC1x-w9y">目前的状态是制剂OK，体外已经验证，有人口服。尚未临床试验，未见动物水平的报道。</li>
  <li data-pid="cnILlcMZ">做一个新药然后自己口服不违背政策法规，在某些（比如我这个）小圈子里是常态的事情。这体现了研发者的一种态度，不是好事也不是坏事。但必须指出，在药物研究领域这一N=1的行为没有科学意义，也不应该拿来论证科学问题。只是个人态度的宣示，没有吹点也没有喷点。</li>
  <li data-pid="YAIsBZWa">口服疫苗不是新东西，一种广为流行的脊髓灰质炎减毒疫苗就是。但是口服疫苗的临床总体做的不好，减毒疫苗还有一定风险。由于消化系统的特点，口服疫苗对质控要求要低一些，研发周期较短，较为适合在疾病流行期间快速推出。但是药剂的挑战比较大。</li>
  <li data-pid="4vuvb5tX">药物研发淘汰率高，大概率这东西无法坚持到上市。有疫苗了不代表疫苗安全有效，疫苗安全有效不代表能及时投入市场，投入市场了也未必能被接受。如果要预测的话，显然预测它不成功是更稳妥的。</li>
  <li data-pid="htCj70Kh">但是还是要支持。只要按照规定的研发流程走下去，多一种药就多一个选项。这次不成，下次流行病大爆发可能就有用了；现在看不靠谱/机会渺茫，也许经过尝试人们的认识就会加深，把不可能变可能。现在国家愿意投钱，科学家把钱拿来做具体的科研工作，本无可厚非。</li>
  <li data-pid="R7FA2-u_">标题党是媒体的错（传媒生态的错），不应苛责科学家。据我所知，某家做核酸疫苗的上市公司，其宣传的标题党程度不亚于此，然后股票应声而涨。本报道内容大体上把能披露的细节披露了一些，还是基本可以看的。</li>
  <li data-pid="S71zaMOx">虽然不看好，但希望能有奇迹发生，让我们早日吃到口服抗原疫苗。即使没有奇迹发生，也希望大家都能从疫苗研发中获得更深刻的认识，建立起可行的研发流程，保护未来人类的公共卫生安全。</li>
 </ol>
