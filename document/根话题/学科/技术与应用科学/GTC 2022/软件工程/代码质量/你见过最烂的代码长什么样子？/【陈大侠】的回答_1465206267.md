# 你见过最烂的代码长什么样子？
- 点赞数：5342
- 更新时间：2020年09月10日19时51分24秒
- 回答url：https://www.zhihu.com/question/265453795/answer/1465206267
<body>
 <p data-pid="1VHv60iL">以前在csdn上看到过一个帖子，被发帖子哥们的经历雷到了。</p>
 <p data-pid="P5NXeIls">这哥们跳槽到一家公司当c#程序员，在这家公司写代码有个规矩，代码必须要用中文写，而且要阅读通顺。</p>
 <p data-pid="1JyuWDQu">代码要求用中文写倒也没什么大不了，世上公司千千万，总有些奇怪嗜好的技术领导，毕竟有人还发明了用纯中文编程的易语言。然而更奇葩的是这家公司不但要求用中文编程，而且还必须代码阅读符合中文语义，读起来要通顺。</p>
 <p data-pid="Sl-mxgWa">先来看看传统的中文编程，比如定义一个中文类</p>
 <div class="highlight">
  <pre><code class="language-csharp"><span class="k">class</span> <span class="err">数据库连接对象</span>
<span class="p">{</span>
    <span class="k">public</span> <span class="k">void</span> <span class="err">连接</span><span class="p">(</span><span class="kt">string</span> <span class="err">服务器名</span> <span class="p">,</span> <span class="kt">string</span> <span class="err">用户名</span><span class="p">,</span> <span class="kt">string</span> <span class="err">密码</span><span class="p">,</span> <span class="kt">string</span> <span class="err">数据库名</span><span class="p">)</span>
<span class="p">}</span>
</code></pre>
 </div>
 <p data-pid="YAmM1YDm">然后这样调用方法</p>
 <div class="highlight">
  <pre><code class="language-csharp"><span class="err">数据库连接对象</span><span class="p">.</span><span class="err">连接</span><span class="p">(</span><span class="err">服务器名</span> <span class="p">,</span> <span class="err">用户名</span><span class="p">,</span> <span class="err">密码</span><span class="p">,</span> <span class="err">数据库名</span><span class="p">)</span>
</code></pre>
 </div>
 <p data-pid="YOpEJYN4">这种只属于中文编程，但不算阅读通顺。这家公司要求的是代码符合汉语阅读习惯</p>
 <div class="highlight">
  <pre><code class="language-text">数据库连接对象.连接到(服务器名).用户为(用户名).密码为(密码).数据库为(数据库名称)</code></pre>
 </div>
 <p data-pid="wemQV7kr">而且连c#本身的关键字也要用中文替换掉，封装成中文的函数调用</p>
 <p data-pid="vds567Vw"><u>比如</u></p>
 <div class="highlight">
  <pre><code class="language-csharp"><span class="err">遍历器</span><span class="p">&lt;</span><span class="err">登录信息</span><span class="p">&gt;.</span><span class="err">计数遍历</span><span class="p">(</span><span class="n">_详细信息集合</span><span class="p">).</span><span class="err">从</span><span class="p">(</span><span class="m">0</span><span class="p">).</span><span class="err">到</span><span class="p">(</span><span class="n">_详细信息集合</span><span class="p">.</span><span class="err">长度</span><span class="p">()).</span><span class="err">每隔</span><span class="p">(</span><span class="m">1</span><span class="p">).</span><span class="err">执行</span><span class="p">((</span><span class="err">当前行索引</span><span class="p">,</span> <span class="err">登陆信息</span><span class="p">)</span> <span class="p">=&gt;</span>
            <span class="p">{</span>
                <span class="err">判断</span><span class="p">.</span><span class="err">如果是</span><span class="p">(</span><span class="n">XXX</span><span class="p">).</span><span class="err">则</span><span class="p">(()</span> <span class="p">=&gt;</span>
                <span class="p">{</span>
                    <span class="err">判断</span><span class="p">.</span><span class="err">如果是</span><span class="p">(</span><span class="n">YYY</span><span class="p">).</span><span class="err">则</span><span class="p">(()</span> <span class="p">=&gt;</span>
                    <span class="p">{</span>
                        <span class="err">表格</span><span class="p">.</span><span class="err">行</span><span class="p">(</span><span class="err">当前行索引</span><span class="p">).</span><span class="err">背景色设为</span><span class="p">(</span><span class="n">Color</span><span class="p">.</span><span class="n">Gray</span><span class="p">);</span>
                    <span class="p">});</span>
                <span class="p">}).</span><span class="err">否则</span><span class="p">(()</span> <span class="p">=&gt;</span>
                <span class="p">{</span>
                    <span class="err">判断</span><span class="p">.</span><span class="err">如果是</span><span class="p">(</span><span class="n">ZZZ</span><span class="p">).</span><span class="err">则</span><span class="p">(()</span> <span class="p">=&gt;</span>
                    <span class="p">{</span>
                        <span class="err">判断</span><span class="p">.</span><span class="err">如果是</span><span class="p">(</span><span class="n">KKK</span><span class="p">).</span><span class="err">则</span><span class="p">(()</span> <span class="p">=&gt;</span>
                        <span class="p">{</span>
                            <span class="err">表格</span><span class="p">.</span><span class="err">行</span><span class="p">(</span><span class="err">当前行索引</span><span class="p">).</span><span class="err">背景色设为</span><span class="p">(</span><span class="n">Color</span><span class="p">.</span><span class="n">Gray</span><span class="p">);</span>
                        <span class="p">});</span>
                    <span class="p">});</span>
                <span class="p">});</span>
            <span class="p">});</span>
</code></pre>
 </div>
 <p class="ztext-empty-paragraph"><br></p>
 <div class="highlight">
  <pre><code class="language-csharp">  <span class="err">异常</span><span class="p">.</span><span class="err">对以下语句进行异常检测</span><span class="p">(()</span> <span class="p">=&gt;</span>

 <p data-pid="fe_kboYp">这家公司的领导认为这样写代码可读性更高</p>
 <p data-pid="9O-lZN1c">我觉得论代码比烂，这种应该是巅峰了，相比其他的烂代码，在不同维度，比如外国人基本看不懂，其他烂代码在怎么说也是无国界的，这种代码已经烂到隔离国籍了</p>
