 <p data-pid="3Nwph2AH"><b>类型：</b>可视化图表设计</p>
 <p data-pid="Pwq3brqM"><b> GitHub Star ：</b>5985</p>

 <ol>
  <li data-pid="RS2Fjz3l">简洁的 API 设计，使用如丝滑般流畅，支持链式调用</li>
  <li data-pid="JNoaAhrP">囊括了 30+ 种常见图表，应有尽有</li>
  <li data-pid="1Q0qDvdN">支持主流 Notebook 环境，Jupyter Notebook 和 JupyterLab</li>
  <li data-pid="pHccJnIn">可轻松集成至 Flask，Django 等主流 Web 框架</li>
  <li data-pid="1r2adViU">高度灵活的配置项，可轻松搭配出精美的图表</li>

 <h2><b>Superset </b></h2>
 <p data-pid="oEqPjY6d"><b>类型：</b>开源的 企业级 轻量BI工具</p>
 <p data-pid="zTGofd4S"><b> GitHub star ：</b>24937</p>
 <p data-pid="bhbzU_Z-"><b>功能：</b></p>
 <ol>
  <li data-pid="1-xDj7fp">创建和分享可视化面板</li>
  <li data-pid="I1vW0nd9">有丰富的可视化方法来分析数据，且具有灵活的扩展能力</li>
  <li data-pid="cdpwgEtk">具有可扩展的、高粒度的安全模型，可以用复杂规则来控制访问权限。目前支持主要的认证提供商：DB、OpenID、LDAP、OAuth、和Flask AppBuiler的REMOTE_USER</li>
  <li data-pid="vad7qgy4">使用简单的语法，就可以控制数据在UI中的展现方式</li>
  <li data-pid="coLdsGHI">与Druid深度结合，可快速的分析大数据</li>
  <li data-pid="111t1lgC">配置缓存来快速加载仪表盘</li>
  <li data-pid="KS3rY8HG">Superset最初是在<a href="https://link.zhihu.com/?target=http%3A//Druid.io" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">http://</span><span class="visible">Druid.io</span><span class="invisible"></span></a>的基础上设计的，但是通过使用SQLAlchemy（一种与大多数常见数据库兼容的Python ORM），迅速扩展了范围以支持其他数据库。</li>
 </ol>

 <h2><b>plotly</b></h2>
 <p data-pid="iQ7RUWjU"><b>类型：</b>非常著名且强大的交互式开源数据可视化框架</p>
 <p data-pid="nDM1C62V"><b>GitHub star ：</b>5235</p>
 <p data-pid="ck3iKwHz"><b>功能：</b></p>
 <ol>
  <li data-pid="GkJX9icU">交互式开源可视化框架，支持超过40种独特图表类型，涵盖统计、财务、地理、学术、三维等。</li>
  <li data-pid="AfmaAKuJ">建立在Javascript plotly库基础上，能使python用户创建基于web的可交互的可视化作品，其能在jupyter notebook上展示，而且可以导出为HTML。</li>
  <li data-pid="3ZWKnku8">plotly还可以在非web编辑器上（如pycharm、spyder）绘制图表</li>
  <li data-pid="ghoMGw9E">能导出出版级别的图片</li>
 </ol>

 <h2>Bokeh</h2>
 <p data-pid="52bs8Saq"><b>类型：服务于浏览器的炫酷的交互式可视化库</b></p>
 <p data-pid="O4e2kV4-"><b>GitHub star ：</b>11061</p>
 <p data-pid="iqT_bE_t"><b>功能：</b></p>
 <ul>
  <li data-pid="yQ4fJs8m">专门针对Web浏览器的交互式、可视化Python绘图库</li>
  <li data-pid="vgzPbTwz">提供优雅简洁的多功能可视化展示，能快速创建图表、仪表板和可视化应用</li>
  <li data-pid="ELQkI50w">可以做出像D3.js简洁漂亮的交互可视化效果，但是使用难度低于D3.js。</li>
  <li data-pid="xmy_nHi1">独立的HTML文档或服务端程序</li>
  <li data-pid="L6opPeFw">可以处理大量、动态或数据流</li>
  <li data-pid="NbDXInGq">支持Python (或Scala, R, Julia…)</li>
  <li data-pid="JHJvvOwj">不需要使用Javascript</li>
 </ul>
 <p data-pid="k2epWPXo"><b>使用方法：</b></p>
 <div class="highlight">
  <pre><code class="language-text">from bokeh.plotting import figure, output_file, show
# 创建图表
p = figure(plot_width=300, plot_height=300, tools="pan,reset,save")
# 图表中添加圆
p.circle([1, 2.5, 3, 2], [2, 3, 1, 1.5], radius=0.3, alpha=0.5)
# 定义输出形式
output_file("foo.html")
# 展示图表
show(p)</code></pre>
