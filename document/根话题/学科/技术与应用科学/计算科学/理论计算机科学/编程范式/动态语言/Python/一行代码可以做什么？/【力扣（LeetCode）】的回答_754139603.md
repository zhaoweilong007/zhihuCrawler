 <h2><b>能解决一道算法题</b></h2>
 <p data-pid="T96RfvQU">比如 <u><a href="https://link.zhihu.com/?target=https%3A//leetcode-cn.com/problems/power-of-two/%3Futm_campaign%3Dlcsocial%26utm_medium%3Dquestion%26utm_source%3Dzhihu%26utm_content%3D285586045%26utm_term%3D754139603" class=" wrap external" target="_blank" rel="nofollow noreferrer">力扣 231. 2 的幂</a></u>，input 是一个整形，要求判断这个数字是否是 2 的幂，并返回布尔值。</p>
 <div class="highlight">
  <pre><code class="language-python"><span class="k">def</span> <span class="nf">isPowerOfTwo</span><span class="p">(</span><span class="bp">self</span><span class="p">,</span> <span class="n">n</span><span class="p">:</span> <span class="nb">int</span><span class="p">)</span> <span class="o">-&gt;</span> <span class="nb">bool</span><span class="p">:</span>
    <span class="k">return</span> <span class="n">n</span> <span class="o">&gt;</span> <span class="mi">0</span> <span class="ow">and</span> <span class="n">n</span> <span class="o">&amp;</span> <span class="n">n</span> <span class="o">-</span> <span class="mi">1</span> <span class="o">==</span> <span class="mi">0</span></code></pre>
 </div>
 <p class="ztext-empty-paragraph"><br></p>
 <div class="highlight">
  <pre><code class="language-python"><span class="k">def</span> <span class="nf">isPowerOfTwo</span><span class="p">(</span><span class="bp">self</span><span class="p">,</span> <span class="n">n</span><span class="p">):</span>
	<span class="k">return</span> <span class="n">n</span> <span class="o">&gt;</span> <span class="mi">0</span> <span class="ow">and</span> <span class="mi">2</span><span class="o">**</span><span class="nb">int</span><span class="p">(</span><span class="n">math</span><span class="o">.</span><span class="n">log2</span><span class="p">(</span><span class="n">n</span><span class="p">))</span> <span class="o">==</span> <span class="n">n</span></code></pre>
 </div>
 <p data-pid="HkCOpJ7B">n 小于等于 0 时一定为否，n 大于 0 的前提下，第二种解法就是简单的通过数学运算比较，第一种解法则是基于二进制理论：如果一个数是 2 的幂，它的二进数最高位是 1，其它都为 0；对它减 1 后最高位为 0，其余都是 1，做与运算会得到 0。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="QzNWJOt3">再比如 <u><a href="https://link.zhihu.com/?target=https%3A//leetcode-cn.com/problems/reverse-words-in-a-string/%3Futm_campaign%3Dlcsocial%26utm_medium%3Dquestion%26utm_source%3Dzhihu%26utm_content%3D285586045%26utm_term%3D754139603" class=" wrap external" target="_blank" rel="nofollow noreferrer">力扣 151. 反转字符串里的单词</a></u>，也可以用一行代码解答。</p>
 <div class="highlight">
  <pre><code class="language-python"><span class="k">def</span> <span class="nf">reverseWords</span><span class="p">(</span><span class="bp">self</span><span class="p">,</span> <span class="n">s</span><span class="p">:</span> <span class="nb">str</span><span class="p">)</span> <span class="o">-&gt;</span> <span class="nb">str</span><span class="p">:</span>
    <span class="k">return</span> <span class="s2">" "</span><span class="o">.</span><span class="n">join</span><span class="p">(</span><span class="n">s</span><span class="o">.</span><span class="n">split</span><span class="p">()[::</span><span class="o">-</span><span class="mi">1</span><span class="p">])</span></code></pre>
 </div>
 <p data-pid="3MCGG4Zl">首先用 split 方法将字符串转化成 list 并去除头尾的空格，[::-1] 可以用 <code>list[&lt;start&gt;:&lt;stop&gt;:&lt;step&gt;]</code> 理解，即从头到尾以每步 -1 的方式获取元素，也就是反转了 list，最后合并。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="AOXkz25c">当然，上面举的这些例子不过是现代程序员们的小把戏，早在上个世纪 60 年代，<code>one-liner program</code> 的概念在就已经在计算机领域的上古巨佬之间流传。在维基百科上，你可以搜索到相关词条。</p>
