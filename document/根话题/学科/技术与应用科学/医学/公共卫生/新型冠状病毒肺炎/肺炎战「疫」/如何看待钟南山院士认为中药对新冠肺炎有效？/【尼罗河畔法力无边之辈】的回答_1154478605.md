# 如何看待钟南山院士认为中药对新冠肺炎有效？

- 更新时间：2020年04月15日14时17分48秒

<body>
 <p data-pid="HcYxt8XU">1、钟南山是一个纯粹的接受现代医学，有过国外医学经历的中国顶级专家，他的经历里没有中医学，以钟的能力和资历，甚至可以质疑临床试验的标准，所以别扯他不懂西医、不懂双盲、不懂循证医学。</p>
 <ul>
  <li data-pid="pjq9QzgK">1960年7月至1971年8月，担任北京医学院（现北京大学医学部）放射医学教研组助教。</li>
  <li data-pid="i71X-0T5">1971年9月至1974年4月，担任广州医学院第一附属医院内科住院医师。</li>
  <li data-pid="r1oOwm-e">1974年4月至1978年8月，担任广州医学院第一附属医院内科主治医师。</li>
  <li data-pid="aUV-RaaR">1978年9月至1982年12月，担任广州医学院第一附属医院内科副主任医师、讲师。</li>
  <li data-pid="oNpiFXLK">1979年至1981年，赴英国爱丁堡大学医学院及伦敦大学呼吸系进修。</li>
  <li data-pid="g3z-r7vA">1983年1月至1986年12月，担任广州医学院副教授（于1985年后被指定为中央领导保健医生）。</li>
  <li data-pid="wCDahwMk">1986年12月至1995年，担任广州医学院呼吸内科教授、硕士生导师。</li>
  <li data-pid="n2NAs-aj">1987年12月至1992年10月，担任广州医学院第一附属医院院长。</li>
  <li data-pid="HtFpLJht">1992年5月至2002年12月，钟南山担任广州医学院院长、党委书记。</li>
  <li data-pid="7SJV8YDE">1992年2月至1997年12月 广州医学院附一院广州呼吸疾病研究所 副研究员</li>
  <li data-pid="hPr1mjS8">1995年8月，担任北京大学医学部呼吸内科教授、博士生导师。</li>
  <li data-pid="2iaUPT4s">1995年10月至1996年10月， 公派到加拿大蒙特利尔市麦吉尔大学 访问学者。</li>
  <li data-pid="C0JyoIa6">1996年5月，当选为中国工程院院士。</li>
  <li data-pid="oh9Tx7BY">1997年12月至1999年12月广州医学院附一院广州呼吸疾病研究所 主任医师。</li>
  <li data-pid="Trey0TDK">1999年12月至2012年10月广州医学院附一院广州呼吸疾病研究所 教授/副所长。</li>
  <li data-pid="tlsNOqOx">1998年6月，任中国工程院医药卫生工程学部副主任。</li>
  <li data-pid="Ef6O6pVT">2003年，“非典”爆发，以钟南山为代表的医护工作者经长期努力，抗击了非典。</li>
  <li data-pid="4lOkaGQE">2004年，被评为“感动中国2003年度”十大人物之一。</li>
  <li data-pid="wToQZ7c9">2005年4月13日，当选中华医学会第23届会长。</li>
  <li data-pid="KxHfU-vL">2007年，获英国爱丁堡大学荣誉博士学位。</li>
  <li data-pid="ekS5gKd_">2007年10月，担任呼吸疾病国家重点实验室主任。</li>
  <li data-pid="RCVq6Lfz">2009年7月，担任贵州医科大学名誉院长。</li>
  <li data-pid="c-luj8ab">2009年9月10日，被评为“100位新中国成立以来感动中国人物”。</li>
  <li data-pid="0dOWLFNq">2010年，获澳门科技大学荣誉博士学位。</li>
  <li data-pid="WOp3EItY">2012年10至2013年7月，任广州医学院附一院广州呼吸疾病研究所教授/常务副所长。</li>
  <li data-pid="UUbayLJb">2013年7月至今，任广州呼吸疾病研究所所长。</li>
  <li data-pid="eSAvnavB">2014年，获颁香港中文大学荣誉理学博士学位。</li>
  <li data-pid="3h0SIoNm">2020年1月20日，国家卫健委高级别专家组召开发布会，钟南山任专家组组长。</li>
  <li data-pid="JzXAUgnq">2020年1月21日，新型冠状病毒联防联控工作机制科研攻关组第一次会议，钟南山任组长。</li>
 </ul>
 <p data-pid="VKI_iVIW">2、他是本次疫情中，第一线的科研人员，拿到的是最第一手的资料、数据和结论，比那些连《内科学》都没吃透，甚至连正经医学都没学过的键盘侠们可信得多。</p>
 <p data-pid="B5OC-ZBu">3、最终结论有可能是不严谨的，但是事后诸葛亮没意思，伟大如牛顿爱因斯坦的理论也有错误理论。</p>
 <p data-pid="rOHLFXup">4、有些人非常可笑，一方面说着要尊重科学，却空口白牙质疑人家院士拿着科学数据科学分析做的结论。你有本事发论文反驳啊？看看医学界能不能认可你的反驳？还是你觉得一个顶级院士的论文不值一驳只需要在知乎敲敲键盘就足以把问题说清楚？</p>
