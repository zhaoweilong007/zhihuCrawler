# 戒烟后如何将肺部清洁干净？

 <h2>事实一：肺部是无法清理的。</h2>
 <p data-pid="uNxTD2RW">如果你吸烟，想要清理肺部，戒烟是最重要的一步。但对于那些从未吸烟的人来说，肺部的黑化（非碳化）是自然现象，无法逆转，也无法清理。</p>
 <p data-pid="muOGLpMN">戒烟者会在戒烟后的几年内每隔一段时间咳出一些褐色的痰。这是来自香烟烟雾的残留焦油，由支气管和较小气道（称为细支气管）的粘液腺排出。剩余的焦油在肺部表面可见，就像你生命中的纹身一样，终身无法去除。</p>

 <p data-pid="zmCL96Lp">有心的烟民可以看看自己的胸片CT，找个不吸烟的朋友对照以下，对比相当明显，甚至完全不用解剖或者任何医学常识，肉眼可辨。</p>

 <p data-pid="flJDA4kS">肺功能的损伤是不可逆的，当然人体都是可自我修复的。焕然一新的美梦就不要在想了，从生理机能角度来说，戒烟之后可以恢复到未吸烟的感知状态，但是从医学角度来看，长期吸烟者不论戒烟多久，肺部功能都不可恢复至非吸烟者的水准。</p>
 <p data-pid="XCaXsqWX"><b>简单的说就是：你可以成为一个更好的你，但是你不可能成为那个本应该的你！</b></p>

 <h2>你的肺部不会恢复到100％，那么戒烟对你的意义何在？</h2>
 <p data-pid="3ynb_jtB">一旦你戒烟，你的身体就开始自我修复。</p>
 <p data-pid="Oa73nv-q">戒烟的典型好处是：</p>
 <p data-pid="p_jyDUOL"><b>6小时内</b></p>
 <ul>
  <li data-pid="CoZC2ERM">你的心率减慢，你的血压变得更稳定。</li>
 </ul>
 <p data-pid="LJcfXrat"><b>一天之内</b></p>
 <ul>
  <li data-pid="Ah3vKrIg">几乎所有的尼古丁都不在您的血液中。</li>
  <li data-pid="1CtJM15h">血液中的一氧化碳水平下降，氧气更容易到达心脏和肌肉。</li>
  <li data-pid="R6Tcqz_b">你的指尖变得温暖，你的手更稳定。</li>
 </ul>
 <p data-pid="H_n0Amuk"><b>一周之内</b></p>
 <ul>
  <li data-pid="xhCa8a8h">你的味觉和嗅觉可能会改善。</li>
  <li data-pid="wTGUuhIv">你有更高血液水平的保护性抗氧化剂，如维生素C.</li>
 </ul>
 <p data-pid="Xtj0E5JR"><b>3个月内</b></p>
 <ul>
  <li data-pid="QGMmJmZ1">你咳嗽和喘息的时间更少。</li>
  <li data-pid="dUSbU87i">你的肺部的自然清洁系统正在恢复，更好地去除肺部的粘液，焦油和灰尘（运动有助于清除肺部）。</li>
  <li data-pid="H52qZHmI">你的免疫系统正在开始恢复，所以你的身体更能抵抗感染。</li>
  <li data-pid="EIaLoo9a">你的血液不那么厚实和粘稠，你手脚的血液流动得到了改善。</li>
 </ul>
 <p data-pid="CGNhNYaV"><b>6个月内</b></p>
 <ul>
  <li data-pid="uk1M-tR1">你不太可能咳嗽。</li>
  <li data-pid="oqk2fCnk">与吸烟时相比，你可能感觉压力更小。</li>
 </ul>
 <p data-pid="jIGe0CF4"><b>1年后</b></p>
 <ul>
  <li data-pid="OP2smE8I">你的肺部现在更健康了，你比你吸烟更容易呼吸。</li>
 </ul>
 <p data-pid="JZB03ie0"><b>在2至5年内</b></p>
 <ul>
  <li data-pid="fyoeuhrH">心脏病发作和中风的风险大幅下降，这种风险将随着时间的推移逐渐减少。</li>
  <li data-pid="U9pVbPMJ">对于女性，在五年内，宫颈癌的风险与从未吸烟的人相同。</li>
 </ul>
 <p data-pid="L3dXBukK"><b>10年后</b></p>
 <ul>
  <li data-pid="RrkfD836">您患肺癌的风险低于持续吸烟者（如果您戒烟时疾病尚未出现）。</li>
 </ul>
 <p data-pid="JNE2zPq_"><b>15年后</b></p>
 <p data-pid="Y6oiEF3F">您心脏病发作和中风的风险接近于从未吸烟的人。</p>

 <p data-pid="JodmP332"><b>肺部不可清理，但是可以通过一些手段来提高自我修复的过程，当然也仅仅是一些简单的食疗，功效因人而异，仅供参考。</b></p>
 <p data-pid="6Aysft-A">采取高抗氧化饮食，充足的水果（特别是浆果），大量的蔬菜，谷物（尤其是燕麦），以及大量的坚果。</p>
 <p data-pid="9qvvjYM_">多喝水，少喝茶/咖啡。</p>
 <p data-pid="0zAkTAO0">有氧运动</p>
 <p data-pid="ImmXUoVZ">PS：临睡前减少乳制品的摄入，尤其是刚刚戒烟的阶段。</p>
