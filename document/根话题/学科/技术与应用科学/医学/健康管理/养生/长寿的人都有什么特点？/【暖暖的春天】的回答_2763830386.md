# 长寿的人都有什么特点？
- 点赞数：5036
- 更新时间：2022年11月18日11时30分24秒
- 回答url：https://www.zhihu.com/question/61419687/answer/2763830386
<body>
 <p data-pid="AwC5Hfyo">我外婆，102岁，和英女王死因一样，无疾而终。总结一下她的长寿秘诀：</p>
 <ol>
  <li data-pid="zI4tPGi-">第一条也是最重要的一条——有母胎自带的优秀基因！她外婆和妈妈都是90多岁去世的。她家族里的其他女人也基本上没有早于80去世的。</li>
  <li data-pid="hlnsokId">身体底子好，没有基础病。小时候家境殷实，父母宠爱，吃穿用度都很宽裕，所以身体好，精神状态也很好。</li>
  <li data-pid="DH3sMDy5">有文化，旧时读过私塾和女子中学，满腹诗书，出口成章，熟读史书，引经据典。喜欢看报纸，听评书，记忆力特别好，思路清晰，文学造诣高于所有子孙辈。用现在的话是妥妥的学霸。</li>
  <li data-pid="Tf3KTpZh">心态好，从来不操心，不生气，不担忧，不上火。儿孙自有儿孙福，对各家的困难出钱能支持就出钱支持，没钱就口头鼓励大家，告诉子孙办法总比困难多。不用着急。</li>
  <li data-pid="EofRsfVj">作息有节，从来不熬夜。</li>
  <li data-pid="mxD__26C">饮食有度，不吃生冷油腻辛辣。每天早上都要吃一个油煎荷包蛋。喝一碗粥，吃点小菜，这个小菜不是咸菜，是现拌或现炒的菜。很少吃剩饭剩菜。三顿饭按时按点吃，每顿大半碗米饭，不因为好吃而多吃一口</li>
  <li data-pid="ywYWZjnv">喜欢自己盘腿打坐，自己拿简易的按摩器给自己做足底按摩，这个按摩器可能是不用的小梳子，或者陶瓷的小勺子。</li>
  <li data-pid="XAeOEr51">自己的事情自己做。我妈妈是护士，比较懂护理，所以她一直和我妈妈一起生活，但是她从来没有想要任何人伺候她。她90多岁还在用电饭锅给我们煮饭，自己的贴身衣服也都是自己手洗，虽然劝过很多次不让她洗衣服，怕把地板弄湿了她摔倒会有危险。但是她还是坚持偷偷自己洗衣服。每天早上起来都会自己叠被，整理自己房间的家务。</li>
 </ol>
 <p data-pid="ZrkwGSK8">外婆人很好，有些不了解她的人可能会觉得她给人一种冷漠疏离的感觉，但是我们知道她是有大智慧的，她能记住每个孙辈重孙辈的生日。她情绪稳定，从来不大喜大悲，她对这个世界温柔，所以这个世界也回报她温柔。</p>
