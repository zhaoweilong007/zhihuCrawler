 <h2><b>Part 1：带到医院</b></h2>
 <h2>产妇用</h2>
 <ul>
  <li data-pid="-JCdcrd4"><b>个人洗漱用品——必备！（不洗脸的人除外）</b></li>
  <ul>
   <li data-pid="BgI7zxip">住院的几天时间，按照自己的日常所需，带好自己的洗漱用品。牙膏牙刷洗面奶擦脸油之类的。</li>
   <li data-pid="eLyIhUkb">强烈推荐一次性洗脸棉，棉柔巾，干净卫生，洗脸擦桌子怎么都好用。目前用过N多款，最推荐ITO这个，厚实，带颗粒感，洗的干净，耐用，别的有些牌子洗一次就OVER了，这一款，一张大概洗三次都可以。孩子出生以后，用得到的场景更多，必囤！</li>
  </ul>
 </ul><a data-draft-node="block" data-draft-type="mcn-link-card" data-mcn-id="1302220716558118912"></a>

 <blockquote data-pid="m9F30VMK">

 </blockquote>
 <ul>
  <li data-pid="aCSG0FvD"><b>吸奶器——背奶上班人群+奶牛体质必备，贫奶妈妈或者产后不打算背奶上班的酌情可以省略——提前备好</b></li>
  <ul>
   <li data-pid="JsJeXIEP">奶牛妈妈建议在预算允许的情况下，买电动双边的。在产后非常虚弱的情况下，又要频繁起夜喂奶或者吸奶，这个时候能省点力气真的是太重要了。强烈建议吸奶器也带到医院去，（我开始以为医院用不到，后来专门让家人回去取了），因为如果是奶牛体质，出奶比较快，有可能前几天就有涨奶乳房痛的情况，这个时候及时排空乳房很重要，大大缓解疼痛，也能避免堵奶。 相信我，哺乳是个大关，​​月子里两个乳房就像两个炸弹，任何让它俩舒服点的东西都值得。</li>
   <li data-pid="EZGucTk4">另外这里提到奶牛体质：如何判断奶牛体质呢？据说，这个和遗传有点关系，自己妈妈奶比较多的话，自己奶多的可能性也比较大。但是也不绝对。</li>
  </ul>

   <ul>

   <li data-pid="KjdMFlqL"><b>储物空间</b></li>
   <ul>
    <li data-pid="2IfcXety">有了小孩之后，家里物品的数量是几何式增长，所以在生孩子之前，请做好储物空间的规划，小孩子的东西都比较柔软比较小，细碎，小孩子的衣服不建议挂在衣柜里，因为都很小，一般也是棉质地的，叠放最节省空间。所以最实用的就是抽屉，我家族空间是比较充足的，但是在孩子出生之前还是添置了几组亚克力的抽屉柜。相比甲醛的污，同等体积的亚克力抽屉，因为比较薄，所以储物空间会更大，而且也不会有甲醛的污染。，很轻盈，推荐爱丽丝品牌，质量很好。无数人私信问我这个柜子的链接了～～～</li>

 <h2>Part5：其他重要的准备</h2>
 <ul>
  <ul>
   <li data-pid="ohxmlofi"><b>联系好通乳师</b></li>
   <ul>
    <li data-pid="zdC697Jv">坐完月子才明白一个道理，分娩只是生孩子的第1关，哺乳也有一个大关要过，尤其在月子里，堵奶之类的问题很让人困扰。建议提前联系好通乳师，了解一下有哪些是可以随时上门的。堵奶，产妇不方便出门，但是又会很疼，需要尽快的有通乳师来帮忙。如果家里请了月嫂，请确认月嫂是否具备通乳技能。非常重要。如果堵奶的问题没有解决，引发乳腺炎，孩子和大人都会很遭罪。</li>
   </ul>
   <li data-pid="ZLC1mBOr"><b>考察好家旁边的社区医院及环境</b></li>
   <ul>
    <li data-pid="Nd4l8pPh">孩子出生满月后很快就要开始打预防针，这个时候产妇的身体比较虚弱，孩子也非常小，经不起折腾。，强烈建议提前考察好家旁边的设计院在哪里，去了解一下相应的程序，尽量确保到了医院后能够快速的打完针</li>
   </ul>
   <li data-pid="kgHnPHT4"><b>安排好月子期间的居住问题</b></li>
   <ul>
    <li data-pid="FEHUm3hR">有了小孩家里不是仅仅添了一口人，很有可能也有老人要来家里帮忙照料，提前和家人沟通好。照料者是谁，考虑好如何安排居住的问题。各家对孩子如何睡都有自己的观点，可以选择在小床上，然后小床放在宝爸宝妈的大床边。也有家庭。，孩子是和老人在一个房间的，不论选择什么方式，都应该提前做好安排。</li>
   </ul>
