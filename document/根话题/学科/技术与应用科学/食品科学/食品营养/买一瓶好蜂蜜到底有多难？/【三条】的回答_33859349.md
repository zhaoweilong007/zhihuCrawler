# 买一瓶好蜂蜜到底有多难？
- 点赞数：7235
- 更新时间：2015年12月11日11时22分55秒
- 回答url：https://www.zhihu.com/question/26727733/answer/33859349
<body>
 <p data-pid="8XyWKWNI">这个问题我非常喜欢，作为一个卖蜂蜜的，我非常希望所有的蜂蜜消费者都能成为蜂蜜专家，要不然消费蜂蜜的顾客都被便宜的垃圾蜜抢去了。</p>
 <p data-pid="vTTJNeKr">虽然题主没给出具体的好蜂蜜标准，但是从其描述来看应该也是业内普遍认可的好蜂蜜标准：优质成熟蜜。<b>先说结论：买一瓶好蜂蜜很难，非常难。</b></p>
 <p data-pid="Udn4LdJj">前几天还和一个做蜂蜜的同行聊到中国的商品蜜中到底有多少高质量成熟蜜的问题，我之前只了解到的大概比例是低于5%，而这位同行说几年前5%还有可能，现在是远不足2%了。</p>
 <p data-pid="QTmkpLwD">假蜂蜜的数量是不可能得到靠谱数据统计的，卖水蜜的商家也不会承认自己卖的是水蜜，所以这些数据都是业内人士根据实际情况估算出来的，仅供参考。先给大家普及一下商品蜜的分类再说为什么优质蜂蜜这么难买。</p>
 <p data-pid="D8K64BqY"><b>【假蜂蜜】</b></p>
 <ul>
  <li data-pid="hf-l85OA">简单的用蔗糖、葡萄糖、果葡糖浆、淀粉、各种添加剂假造的时代已经过去，在先进仪器的加入下，80%以上的假蜜不但外观已经可以以假乱真，在气味和口感上也有不错的相似度，普通消费者想要鉴定出来难度很高。</li>
  <li data-pid="Mv3H-B3D">通过给蜜蜂喂食蔗糖造出来的假蜂蜜，业内人士好判断，普通消费者基本判断不了。</li>
  <li data-pid="Cb8ciEHS">还有一种在最上面一层掺入真蜂蜜的假蜂蜜，如果鉴定不仔细，连蜂农都有可能骗过，造假已经到了丧心病狂的地步。</li>
  <li data-pid="7e0q1Ous">假蜂蜜有多少市场份额呢？这是没法统计的，业内人士大多认为在四分之一左右。</li>
  <li data-pid="amShuhGP">注：那些鉴定假蜂蜜的土方法（诸如拉丝法、浸纸法）早已被登峰造极的造假技术秒成渣渣了，切勿迷信。</li>
 </ul>

 </figure>
 <p data-pid="nXSWces0">这是我上个月看到的一瓶假蜂蜜，有些土蜂蜜外形真的是这样，只看外形真认不出是假的。</p>
 <p data-pid="XdM58Oqe"><b>【水蜜】</b></p>
 <ul>
  <li data-pid="0KNBe1V9">为了大幅度提供蜂蜜产量，在未经过蜜蜂充分酿造就提前取出的蜂蜜被称为水蜜。</li>
  <li data-pid="NZGv4kM2">水蜜营养成分严重缺失且极不稳定，喝这种蜜和喝糖水差别已经不大。</li>
  <li data-pid="tPsNhFbP">相关部门并没有明令禁止水蜜，水蜜的销售是合法的，所以水蜜占有了绝大部分蜂蜜市场。</li>
  <li data-pid="SeXnd9Kx">因为水蜜的含水量比较高，无法通过国家标准，所以商家会想法降低水蜜的含水量。最常见的方法就是高温加热蜂蜜蒸发水分，超过40℃的温度就会破坏蜂蜜的营养成分，可以想象本来营养成分本来就极为缺失的水蜜遭到这样一通破坏会变成什么样……</li>
  <li data-pid="0nWlniNv">蜂蜜的波美度和含水量是可以相互换算的参数，波美度越高则含水量越低，别以为波美度越高蜂蜜就越好，毕竟你不知道这波美度是通过什么办法达到的。所以欧美国家基本不采用波美度作为蜂蜜好坏的标准，遗憾的是我国一直以波美度给蜂蜜划分等级……</li>
 </ul>

 <br>
 <p data-pid="Grd4UXZ0">图中那些水汪汪的格子里装的都是水蜜，右下角有些格子开始结白色的盖子了，完全结好盖子的就算成熟蜜了。</p>
 <br>
 <p data-pid="7g2-1sf5"><b>【成熟蜜】</b></p>
 <ul>
  <li data-pid="sIVAr5PM">首先，我国没有明文规定成熟蜜的标准，“成熟蜜”属于行话，是在业内被认可的一个标准。</li>
  <li data-pid="61J0l3vs">蜂蜜是蜜蜂酿造给自己吃的，当蜜蜂认为蜂蜜已经达到自己的要求时，会对蜂蜜进行“封盖”，所以业内普遍认为完全封盖的蜂蜜为成熟蜜。</li>
  <li data-pid="rUa-FQXf">在不同的经纬度、不同的环境、不同的气候下蜜蜂对蜂蜜进行封盖的含水量标准是不同的。如果在一个温度较低比较潮湿的环境，超过20%的含水量蜂蜜也可以长期保存不变质，那么蜜蜂很可能在含水量超过20%时就封盖。如果在一个温度较高比较干燥的环境，含水量过多可能导致蜂蜜发酵变质，那么蜜蜂就会在含水量足够低的时候才会进行封盖。只要完全封盖了的蜂蜜那就是真资格的蜂蜜，封盖时的含水量不同只能说明采蜜的环境不同。</li>
  <li data-pid="mXCN7YV2">为什么我国要对蜂蜜的含水量做要求呢？因为商品蜜是要卖到全国各地的，在蜂蜜的产地有22%的含水量不会发酵变质不代表运输到其它地方也不会发酵变质，所以我国规定含水量低于20%的蜂蜜为一等蜜。这个规定较大程度上避免了商品蜜发酵变质。</li>
 </ul>

 <br>
 <p data-pid="KVl2uzZi">这就是完全成熟的封盖蜜，用刀削掉盖子就可以开始摇蜜了。</p>
 <br>
 <p data-pid="Iq2q_ZUG"><b>【优质成熟蜜】</b></p>
 <ul>
  <li data-pid="EgUDcVsX">大多数的成熟蜜都是好蜜，但不代表所有的成熟蜜都是好蜜。</li>
  <li data-pid="8yBQ8MZj">如果蜜源地附近有个排污工厂，产出的蜂蜜可能含有有害物质。</li>
  <li data-pid="M7qihIYQ">如果蜜源是农作物，那么就可能有施肥、喷洒农药等情况，进而导致在蜂蜜中形成残留。所以油菜花蜂蜜之所以廉价，不只是因为油菜花多好采蜜，还因为油菜花是农作物。</li>
  <li data-pid="ES9ejBph">海拔高、日照充足、森林覆盖率高、无任何污染源、有稀有蜜源花朵，这算是顶级蜜源地了，这种地方产出的成熟蜜足以傲视群雄。</li>
 </ul>
 <p data-pid="TZoCnBe8"><b>【意蜂蜜 vs 中蜂蜜（土蜂蜜）】</b></p>
 <ul>
  <li data-pid="o6yGHRJk">意蜂很温顺，可以开车拖着蜂箱全国跑，这样一年可以赶几拨花期采蜜，可取多次蜜，产量比较高。</li>
  <li data-pid="lXXgtR0g">中蜂很难完全驯服，不能轻易搬动其蜂箱，所以现在只有一些偏远地方或者深山里还在养中蜂，通常一年只取一次蜜，产量相当低。</li>
  <li data-pid="gM9iDj6w">但是蜜蜂只是搬运工，有好的蜜源才能产出好的蜂蜜，所以我并不认同中蜂蜜一定好于意蜂蜜的说法，虽然我是卖中蜂蜜的。</li>
 </ul>

 <br>
 <p data-pid="FLtXd8-O"><b>做了上面的知识普及，相信大家对蜂蜜的种类和价值有了一定的了解，但是成熟蜜毫无生产难度，为什么就这么难买到呢？</b></p>
 <ul>
  <li data-pid="hpf2pXGu">原因就是消费者对蜂蜜知识了解得太少。</li>
  <li data-pid="Uzwa8qGD">中老年人作为蜂蜜的消费主力，他们最为看重的就是价格，喜欢购买有价格优势的蜂蜜。优质成熟蜜的成本是水蜜的数倍，在价格竞争当中肯定毫无战斗力，厂家要生存只好投消费者所好推出廉价蜂蜜。</li>
  <li data-pid="doE-7hKn">绝大多数的商品蜜是意蜂产的，养意蜂的蜂农大多数是专职蜂农，每年会拖着几十上百个蜂箱去赶花期。每一波花期产出来的蜂蜜往往上吨重，是不大可能带着蜂蜜去赶下一波花期的，需要在产出来后就尽快卖掉，所以必须投收蜜厂家所好，厂家要低价蜂农就产水蜜增加产量。两者心里都倍儿清楚，但都不去捅破。</li>
  <li data-pid="S9ReAeUv">所以啊，好蜜不是难找，是难卖。商人是趋利的，肯定是什么好卖就卖什么啊，所以好蜜的市场份额被越挤越低。</li>
  <li data-pid="2BpDKJeM">（这句是广告，不喜就跳过）我推出蜂蜜时，一大堆人质问我：别家蜂蜜一二十块钱一斤，你凭什么要上百，太暴利了吧？油菜花水蜜几块钱一斤就能收购到，老式树干蜂箱的成熟蜜我要几十块钱一斤才能收购到；一个意蜂蜂农一个花期能产几吨水蜜一辆卡车就拉进厂了，山里的中蜂老蜂农一年就产几十斤最多上百斤蜂蜜，还要肩挑背扛运出山来。这两者间的成本差距是十几倍啊，我也知道价格和销量成反比，但成本在那摆着我也没办法。</li>

  <br>
  <li data-pid="zvB14ZWx">商超的数据不好搞，但是淘宝的数据大家是可以轻易查到的。在淘宝搜索“蜂蜜”，然后按照销量从高到低排序，看看这些销量最好的蜜大多数是什么价位的。要知道淘宝购物的大多是获取知识相对容易的年轻人，商超购物主力是知识相对固化的中老年人，细细琢磨一下大家就知道上文中的2%是怎么来的了。我虽然知道成熟蜜的大概成本，但是我不能告诉你说什么价格以上的就是成熟蜜什么价格以下的就是假蜜水蜜，毕竟不同品种的蜜成本不同，不同商家的价格策略也不同，大家心里有数看个大概就成。</li>
  <li data-pid="5wsyK96D">价格高的就一定是好蜂蜜吗？NO！连全球最牛的新西兰麦卢卡蜂蜜都被英国查出高达80%以上的造假，我们伟大的名族又会是怎么样呢？（新闻链接：<a href="https://link.zhihu.com/?target=http%3A//www.chinanews.com/gj/2013/08-27/5210681.shtml" class=" wrap external" target="_blank" rel="nofollow noreferrer">全球售量远高于产量 新西兰蜂蜜有“李鬼”充数</a>）</li>
 </ul>
 <br>
 <p data-pid="woJ-TQqi"><b>给大家讲述了好蜂蜜难买的原因，再分享一些普通人不知道的行业知识，避免在购买蜂蜜时犯下低级错误。</b></p>
 <p data-pid="XJhTYqpO">【有些蜂蜜我不知道是哪来的】</p>
 <ul>
  <li data-pid="kT4xyJRF">雪莲蜂蜜根本不存在：雪莲通常生在在高山雪峰之中，气候恶劣，这样的环境下蜜蜂根本无法生存。</li>
  <li data-pid="NyX8ioKQ">葡萄蜂蜜根本不存在：葡萄只有花粉，没有蜂蜜。</li>
  <li data-pid="fqX6vuXc">玫瑰蜂蜜根本不存在：玫瑰的雌雄蕊退化，根本没有蜜，只有极少的花粉，所以根本没有蜂农愿意去采玫瑰花。</li>
  <li data-pid="pzvV6TTA">金银花蜂蜜根本不存在：金银花的花冠又长又细，蜜蜂嘴短根本采集不到。</li>
 </ul>
 <p data-pid="DIQ0xwHJ">【购买这些蜂蜜一定要谨慎】</p>
 <ul>
  <li data-pid="m1DBuLBY">益母草蜂蜜产量很低：由于益母草的资源非常有限，只有风调雨顺的年景才能产出非常有限的一点蜂蜜，难以在市场大规模供应。</li>
  <li data-pid="KxbXexMF">桂花蜂蜜非常的稀少：桂花的花期是10-12月份，意大利蜂已冬眠，只有产蜜量极低的中华蜂可以在好天气下能产出一点蜜，不可能在市场大规模供应。</li>
  <li data-pid="og4u1hxa">苹果花蜂蜜寥寥无几：苹果开花时，蜂蜜的量是非常少的，基本只够蜜蜂自身的消耗，所以基本无法形成商品苹果蜂蜜。</li>
  <li data-pid="qnwm3Afz">野菊花蜂蜜少有商品：大多数野菊花的花期在9下旬，大多数蜜蜂已准备冬眠，冬眠5个月蜜蜂本身就需要消耗大量蜂蜜，所以即使采到野菊花蜜，蜂农也通常会留给蜜蜂。</li>
  <li data-pid="sUkZjE_x">黄连蜂蜜非常的稀少：黄连生于山林阴处，花期2-3月，这期间适合大规模培育黄连的省份由于气温较低蜜蜂基本还需要人工喂养。</li>
 </ul>
 <p data-pid="yVkZRE5R"><b>最后总结一下：</b></p>
 <p data-pid="qCy7up_H"><b>消费者贪图便宜倒逼蜂农生产水蜜……</b></p>
 <p data-pid="tUV_u5PF"><b>近年大量消费者购买进口蜂蜜，导致进口蜂蜜产量跟不上而造假……<br>
   全球的奸商是一家啊，很多人可能吃了一辈子的蜂蜜，但从来不知道正宗好蜂蜜是啥味道……</b></p>

 <ul>
  <li data-pid="7Cieg2M6">不同品种的蜂蜜有不同的外观，有几乎透明的、白色的、淡黄色的、深黄色的、棕色的、黑色的等等等等，蜂蜜本身的颜色就很丰富，造假者也都按照常规颜色造假，所以要通过颜色判断真假几乎不可能。当然如果有造假者傻乎乎的搞出什么绿色蜂蜜、蓝色蜂蜜我也只好给他跪了。</li>
  <li data-pid="5OeCj3fv">在合适的温度（通常是12-15℃）将蜂蜜静置一段时间，蜂蜜基本都会结晶，结晶通常是从容器底部开始逐渐往上直至完全结晶，结晶需要的时间较长，快则数周慢则数月。当温度很低，但不是适合的结晶温度时，蜂蜜会变得浓稠，静置久了依然会慢慢开始结晶。所以通过是否结晶来判断蜂蜜的真假也很困难。（有个买家是内蒙的，他说现在环境温度是零下30多度，我猜想蜂蜜快递过去最多几小时就会硬邦邦吧，真想去亲眼看看。）</li>
  <li data-pid="FICMpTVp">很多卖假蜂蜜的，直接卖结晶蜜，有部分假蜂蜜的结晶体非常的硬。通常当年的新蜂蜜结晶硬度和凝固的猪油差不多，远超这个硬度的结晶蜜嫌疑巨大。</li>
  <li data-pid="S8Sc96SB">结晶后很硬的有没有真蜂蜜呢？有，但那极可能是陈年老蜜。蜂蜜虽然可以保质很长时间，但放太久了营养成分会稀释，也不是很值得购买。</li>
  <li data-pid="i1cKzFJI">有一种叫岩蜜的东西，就是硬得像石头一样的蜂蜜，我了解甚少无法评论，唯独能肯定一点：这种蜂蜜不可能大量在市场供应。</li>
 </ul>
 <br>
 <p data-pid="bGRFNMNN">【气味】</p>
 <ul>
  <li data-pid="IG0EACAW">很多假蜂蜜用香精来制造香味，这种香味一点也不像花的香味，问起来有点闷，不愿意多闻。闻过真蜂蜜香味的人很容易鉴定出来。</li>
  <li data-pid="mZRV28-x">掺了白糖或者给蜜蜂喂了大量白糖的蜜蜂，也很容易通过闻味鉴定出来，我个人觉得白糖的气味蛮好鉴别的，不熟悉的人可以挖一勺白糖兑一点点水闻闻。</li>
  <li data-pid="-_naU5g-">只采自一种花的蜂蜜基本上都会带有这种花的香味，比如你在槐花蜜中就能闻到槐花的香味。</li>
  <li data-pid="oLqtVu4h">采自百花的土蜂蜜香味混合了多种花香，所以香味会比较浓郁，不属于那种清新的香味。</li>
  <li data-pid="ve6K4Kre">如果闻到蜂蜜有酒糟的味道，那么就是真蜂蜜发酵了，发酵后营养流失细菌超标，不能再吃了。</li>
 </ul>
 <p data-pid="OE632DC1">【口感】</p>
 <ul>
  <li data-pid="PmuywFbj">直接取一勺液态蜂蜜干吃，真蜂蜜糊嘴，入口后直下喉咙，不满嘴跑味，有略微的辣喉感。但凡有一条不满足的，都有大大的嫌疑。</li>
  <li data-pid="EjgQmSJd">结晶的蜂蜜和蜂巢蜜嚼起来会有颗粒感，因蜂蜜品种不同、加工方式不同，颗粒的大小会有不同，嚼起来略微有点齁嗓子。</li>
  <li data-pid="Ft7k58eI">品尝是最能检验真假蜂蜜的办法，但前提是你得知道真蜂蜜是什么味儿。</li>
 </ul>
 <br>
 <p data-pid="SV2U5nEy"><b>注：以上提到的方法只适合鉴定假蜜，无法很好的鉴定水蜜，哪怕资深的业内人士也是靠长年累月的口感来判断，普通消费者没有专业仪器的帮助实难识别。</b></p>
