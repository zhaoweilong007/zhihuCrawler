# 如何看待2020年3月3日美联储降息？会产生哪些影响？
- 点赞数：5918
- 更新时间：2020年03月12日19时47分44秒

<body>
 <p data-pid="4hAv0A4C">北京时间3月3日晚，美联储将基准利率下调<b>50个基点</b>，还将超额准备金率（IOER）<b>降低50基点</b>至1.1％。</p>
 <p data-pid="RzlVrdKx">美联储对本次降息的定义为emergency rate cut，即紧急降息，这是<b>2008年金融危机以来第一次使用该政策</b>。</p>
 <blockquote data-pid="ksiu3ZuO">
  联储一般只会在议息会议上公布目标基准利率，本次议息会议将于北京时间3月19日凌晨2点召开，而为了应对新冠病毒的经济冲击和稳住美股市场，3月4日联储就选择了紧急降息，并且降息50bp，较之前幅度扩大一倍。自2019年以来的三次降息，每次幅度均为25bp。

 <h2><b>降息有什么影响？</b></h2>
 <p data-pid="AXdLSLBJ">对美股、黄金、A股、外汇、中国利率、中国地产、全球经济一一阐述如下：</p>
 <ul>
  <li data-pid="jGyyKqyG">对于美股而言，<b>短期有支撑作用，但是长期或意味着美股十年牛市的终结</b>。降息后，美股巨震，道琼斯指数跌超1%在1分钟内拉至+1%，随后转跌，截止我的回答写完，目前跌约1%。短期降息有支撑，但长期美股或难回历史高点。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-8197c034d82e7fc455f145840163bca3_720w.jpg?source=1940ef5c" data-rawwidth="810" data-rawheight="362" data-size="normal" data-caption="" data-original-token="v2-79d5b85057f8b95182c7171a898a10fd" data-default-watermark-src="https://pic1.zhimg.com/50/v2-64a25acdc7a0832debaeee4102443af9_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="810" data-original="https://pica.zhimg.com/v2-8197c034d82e7fc455f145840163bca3_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="1Sc9uJCs"><b>对黄金而言自然是利好。</b>黄金有抗通胀和避险的作用，降息意味着货币宽松，通胀压力加大，且美联储降息也隐含着对未来经济前景的担忧，存在一定风险，因此黄金价格跳涨3%。根本上，<b>黄金价格=通胀-利率</b>，现在潜在通胀升高，利率下降，黄金价格自然上涨。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-74a1363fd666533d0351f0166a3ae4e1_720w.jpg?source=1940ef5c" data-rawwidth="866" data-rawheight="415" data-size="normal" data-caption="" data-original-token="v2-250a1f9f703df2b256373307e9b6f459" data-default-watermark-src="https://picx.zhimg.com/50/v2-74675ab168cbaef15e440606387824a6_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="866" data-original="https://picx.zhimg.com/v2-74a1363fd666533d0351f0166a3ae4e1_r.jpg?source=1940ef5c">

 <ul>
  <li data-pid="ixwmoDlV"><b>美联储降息会引起美元贬值，人民币升值。</b>美联储降息意味着美元吸引力下降，美元汇率自然会下跌，而人民币汇率自然升高。</li>
 </ul>

 <ul>
  <li data-pid="10rBzoSQ"><b>对中国A股市场而言，短期有冲击，长期则凸显A股的投资价值，A股市场将会更加聚焦市场内在因素而非外围环境。</b>就在昨天，大摩将中国A股的投资评级上调至“<b>增持</b>”，其认为仅有A股和新加坡股市能够发挥避险的作用。如果美股发生波动，“水往低处流”，A股目前估值水平合理，再加上有政策护航，A股对外资的吸引力加强。长期而言，<b>外资持续增持A股是趋势，A股逐步向好也是趋势。</b></li>
 </ul>

 <ul>
  <li data-pid="nZmV31Q9"><b>全球降息潮重启。</b>在美联储降息前，澳洲央行已然超预期降息0.25bp,马来西亚央行也跟随降息，美联储降息后，部分绑定美元的地区如香港预计会跟随降息，全球降息潮开启。最新消息，阿联酋宣布降息50bp。</li>
  <li data-pid="uIvAeu25">中国央行会跟随降息吗？<b>今年预计人民银行仍有30-50bp的降息空间，但是3月或不会跟随降息，但会用定向降准来释放流动性以支持实体经济发展。</b>当然，如果全球疫情规模迅速扩大，不排除人行3月就会降息。</li>
  <li data-pid="aHBb0HrZ">对大家关心的中国房地产有何影响？目前房地产市场仍然还是政策主导没有变，短期为了抑制经济下行压力，<b>房地产政策会以“因城施策”方式部分城市高压政策有所放松</b>，但房价大幅上涨的时代确实过去了，<b>长期不看好趋势性房地产投资机会</b>，但在长三角、珠三角等地或有结构性机会。</li>
  <li data-pid="vqw8exvs">全球经济复苏前景如何？原本2020年全球经济有望步入弱复苏的轨道，但受疫情影响，昨天经合组织将2020年经济增长预期从2.9%下调至2.4%，全球经济下行压力陡增。<b>单就降息而言，能够延缓经济下行速度，但是真正经济好坏还是取决于疫情控制情况。</b></li>
 </ul>

 <p data-pid="CX-NA9VB">因为除了中国有真正的财政政策外，其他国家政府动员能力过差，只有货币政策可用，当前无论疫情防控还是稳经济，中国的制度优势都会显露无疑。</p>
