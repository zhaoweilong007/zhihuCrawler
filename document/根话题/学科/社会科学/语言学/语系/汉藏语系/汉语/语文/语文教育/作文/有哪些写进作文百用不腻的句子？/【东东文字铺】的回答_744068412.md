# 有哪些写进作文百用不腻的句子？
- 点赞数：22485
- 更新时间：2021年07月04日16时48分56秒
- 回答url：https://www.zhihu.com/question/288806462/answer/744068412
<body>
 <p data-pid="gy27Dgmj">极力推荐《南方周末》新年贺词，以下是我总结的近十年的贺词精华</p>
 <ul>
  <li data-pid="jVOICnl-">什么才是生活的本来面貌？是消除一切压力与焦虑，摆脱所有传统习俗与社会舆论，随心所欲，自由自在，还是体认到自由与责任永远相伴相生，人必须在责任与压力中淬炼生命，前一种是乌托邦，后一种才是真实世界。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="W5FnTsEc">我们期待有更多的普通人把个人的生涯与历史的嬗变，时代的进步融汇到一起。这是个人的幸运，更是公众的福祉。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="vqoTO-3q">成长的方向是独立人格，进步的标志是自由宽容。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="TrNaQgAJ">100多年前，先贤们发现固步自封已不足以救亡图存，因此埋藏了心中痛苦，远渡重洋，以寻求振兴之道。因此西风东渐，建兵工厂以御外侮，建学校以期未来，建报馆以开民智，因此德先生，赛先生带来了复兴之光。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="b2rR2Ywp">我们在一起就像一滴水融入另一滴水，就像一束光簇拥着另一束光，因为我们知道唯有点亮自己，才有个体的美好前程，唯有簇拥在一起，才能照亮时代的未来。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="9aNvXOVz">我们对自己的期许从来不能脱离时代的进程，“苟日新，日日新”，唯有融入文明进步的潮流，期许才不会退化为桎梏。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <figure data-size="normal">

 </figure>
 <p data-pid="GARAA3LR">还有一些平日积累的百用不腻的句子(很多是在人民日报，光明日报等报刊的时评文章上摘抄的)</p>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="wNwvmYeG">“千千万万普通人最伟大”，无论是奋战在实验室中刻苦攻关的科研人员，无论是托举起“超级工程”的普通工人，还是保家卫国，固守边疆的边防战士……改革的时代潮流奔腾不息，每个人都可以成为“弄潮儿”。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="t7rO3wjp">以理想绘就底色，青春才真正饱满丰盈。新时代呼唤更多充满理想主义的年轻人，在“干”字当头的社会氛围中，源源不断地注入青春动能。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="qv8V0XI3">揆诸华夏文明史，从“汤武革命，顺乎天而应乎人”，到“周虽旧邦，其命维新”，从魏文侯重用李悝变法，到商鞅变法开启秦国统一之路，变革之风为历朝历代所传承。从张骞凿通西域，“西北国始通于汉”，到盛唐“南海商船来大食，西京祆寺建波斯”，从"使者相望于道，商旅不绝于途”的丝路盛景，到"舶交海中，不知其数"的贸易繁荣，开放潮流见证我国"天下大同”，"协和万邦"的宽广胸怀。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="C8Sf4rX_">以改革开放的姿态继续走向未来，变革与开放的深厚历史渊源、深厚文化根基，是一笔宝贵精神和经验财富。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="4FeEC7JH">王继才在孤岛坚守30余年，用自己的坚韧和奉献换来千家万户的平安；钟扬在青藏高原跋涉16个春秋，用自己的艰辛和智慧留下创造美好未来的4000万颗种子；李保国在太行山扶贫35个年头，用自己的汗水和知识造就了千百群众的福祉……</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="Prb2tgvG">在亿万人民奔跑的脚步中，960多万平方公里土地呈现曙光升腾，万物生长的复兴气象，可以说，以奔跑、追梦比喻我国的发展再合适不过了。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="Jyfi2vQH">挑战前所未有，机遇稍纵即逝，等待和迟疑只会错失机遇窗口，奔跑和奋斗才能引领时代潮流。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="aDP0_esk">回首改革开放40年的历程，我们研发运载火箭“上九天揽月”，我们制造深海探测装备“下五洋捉鳖”，一切伟大成就都不是一蹴而就。然而，正如古人所言，“奋然为之，亦未必难。”正是靠着亿万人民的“奋然为之”，我们闯关夺隘，攻坚克难，书写了改革开放的辉煌篇章。实践证明，砥砺奋斗精神，激荡奋斗情怀，就能不惧一切艰难险阻。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="f0P5nr5z">从神州陆沉，九原板荡而来，向中华崛起，民族复兴而去。大千宇宙，浩瀚长空，全纳入赤子心胸，惊世两弹，冲霄一星，尽凝铸中华豪情。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="UdRUcAl6">繁霜尽是心头血，洒向千峰秋叶丹。”我国知识分子历来有浓厚的家国情怀，有强烈的社会责任感，重道义，勇担当，他们把自身的前途命运同国家和民族的前途命运紧紧联系在一起，留下了可歌可泣的事迹，建立了彪炳史册的功勋。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="QGDQoXmW">“一座岛，两个人，三生不离。”王继才夫妇默默坚守开山岛32年，这座“馒头”大的岛，缺食少淡水，石多泥土少，飞鸟不做窝，台风时常扰，王继才夫妇却以“守岛就是守家，国安才能家安”的信念在这里深深扎下根。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="Vv-vy09V">“一寸赤心惟报国”，超越一己之私，贡献一己之力，这是我们这一代人书写人生，成就不凡的坦途。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="IzwJddIi">宋人张载有言，“为天地立心，为生民立命，为往圣继绝学，为万世开太平。”讲道义，重德行，勇担当，尊崇“先天下之忧而忧，后天下之乐而乐”，倡导“修身齐家治国平天下”，这是中华优秀传统文化的价值底色，绵延至今仍熠熠生辉。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="oiDxCdrR">从钱学森，华罗庚，李四光等老一辈“海归”毅然回到祖国怀抱，到西安交通大学“西迁人”科技报国的爱与坚守，再到赓续传统的“两弹一星”精神，“载人航天”精神，无数知识分子心中有“大我”，以铁肩担起道义责任，以智慧推动历史前行，今天黄大年，李保国，南仁东，钟扬等优秀人才的故事为什么能如此感染心灵，触发共鸣，就是因为他们“一寸赤心惟报国”，让生命为时代而澎湃，为人民而奋进。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="QwJ8sZMN">这个美好时代，需要无数中华儿女无怨无悔地守以坚韧，付以辛劳，献以智慧。舍小我谋大我者，必以天下为己任，忧患着大我的忧患，快乐着大我的快乐，憧憬着小我的憧憬，幸福大我的幸福。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="IFtOmeqe">“慨然抚长剑，济世岂邀名。”多少英雄儿女为国家的富强而埋名献身，当今时代，奋斗是最生动的报国，奉献是最可贵的报国，新时代人仍需无数“盖以身许国，但求福利民”的中华赤子，赓续自强不息的民族精神，爱国奉献的红色基因，书写俯仰无愧的的报国华章。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="FhQxsbCb">日出东方，其道大光；前途似海，来日方长。云山苍苍，江水泱泱；今我青年，追逐梦想。勇作先声，引领风潮；责在你我，成必家国！</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="tFyD32hs">青年向上，时代向前，逆风的方向永远最适合飞翔。100年前，一群踏着时代节拍的青年人，用一本《新青年》开启了新文化的篇章；70年前，一群青年时就敢于到中流击水，浪遏飞舟的时代翘楚，让星星之火成燎原之势，带领人民站起来；40多年前，一群青年乘着浩荡春风，成为创造中国奇迹的生力军，让改革开放成为祖国最好的名片，无论何时，奋斗者都是那个时代的风流人物！</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="hLv0fHgm">青年的脚步，写下青春之歌，让人们对青春之中国充满希望。北大女生宋玺远赴亚丁湾护航，怀揣万里赴戎机的壮心；外卖小哥雷海为在中国诗词大会夺魁，平凡生活难掩诗心澎湃；有90后博导走向工作岗位，芳林新叶吐翠绿；归国创业的留学生日益增多，中国的时与势撑起他们的雄心；平均年龄30多岁的中国航天科研团队，对探索更遥远的灿烂星空充满信心。青春理想，青春活力，青春奋斗，正是时代精神和生命力所在。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="AYdAqiAL">枪林弹雨，烽火硝烟里走出来的英雄是英雄，如今为国家建设和社会平安默默奉献，忠诚担当的人同样是英雄。他们坚守信仰的忠诚、以身许国的壮志、民族复兴的宏愿，不仅挺起了民族的脊梁，也树立了价值的标杆。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="Pyam2PsL">“小家”与“大国”同声相应、同气相求、同命相依。正因为感念个人前途与国家命运的同频共振，所以我们主动融家庭情感与国家情感于一体。“家国情怀”如同川流不息的江河，流淌着中华民族的精神血液，滋润着每个人的精神家园。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="p1Fs5MES">群之所为事无不成，众之所举业无不胜。忆往昔，几多风雨，我们依靠人民创造了伟业，树立了丰碑；看今朝，无限风光，我们依靠人民凝聚磅礴力量，不惧风雨，新时代属于每一个奋斗者，每个人都是新时代的创造者和见证者。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="pyZAD7hQ">山是屏障，水是载体，承载着生命也背负着希望，只有把水体保护好、修复好、利用好，才能让生命的律动听到流淌的声音。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="QWfYtlKU">蓝天白云、繁星闪烁，是民之所愿；清水绿岸、鱼翔浅底 是民之所盼；鸟语花香、田园风光，是民之所望。环境就是民生，青山就是美丽，蓝天也是幸福。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="us-QT36R">让老百姓呼吸上新鲜的空气，喝上干净的水，吃上放心的食物，生活在宜居的环境中，是生态文明的必然要求，也是建设美丽中国的根本保障。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="sto2W8ce">回顾中华文明的生态观念，无论是“天育物有时，地生财有限”的客观认识，还是“顺天时，量地利，则用力少而成功多”的自然规律，抑或是“竭泽而渔，岂不获得？而明年无鱼；焚薮而田，岂不获得？而明年无兽”的发展理念，都表明中华民族向来尊重自然、热爱自然，强调天地人统一，要把自然生态同人类文明联系起来，按照大自然规律活动，取之有时、用之有度。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="yONLuEFQ">从“良好生态环境是最公平的公众产品，是最普惠的民生福祉”到“绿水青山就是金山银山”；从安吉的“绿色变奏曲”到长白山的“冰雪黑白调”，无不为世界生态文明建设贡献“东方智慧”。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="XZxK85Jp">老子说：“人法地，地法天，天法道，道法自然。”荀子提倡：“草木荣华滋硕之时，则斧斤不入山林，不夭其生，不绝其长也。”人与自然和谐相处自古以来的智慧。</li>
 </ul>
