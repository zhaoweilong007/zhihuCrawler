# 0~50哪个数字最容易被忽视？
- 点赞数：6096
- 更新时间：2023年07月22日21时56分50秒
- 回答url：https://www.zhihu.com/question/593868136/answer/3038537853
<body>
 <p data-pid="sx9eKVm-">0-23是小时数排除，1-31是日期数排除，0-40是常见的气温排除，35-42是体温范围排除，这样剩下43-50这几个数。其中44两个数字相同难以被忽视，45和48有很多因数，49是完全平方数排除，43和46都可以作为身份证号的前两位，故最有可能被忽视的数字是47。</p>
