# 父母一定要我把户口转去香港怎么办?
- 点赞数：15217
- 更新时间：2021年11月03日17时36分50秒
- 回答url：https://www.zhihu.com/question/462421370/answer/2195778770
<body>
 <p data-pid="DocyNmrx">先说结论：中国香港身份是中华人民共和国<b>普通人</b>逆天改命的一个大Bug。</p>
 <p data-pid="IJ6UaXAo">这个问题换成另一个问法是香港身份在可以拿的情况下，该不该拿？或者能不能想办法拿一个香港身份，好处是什么，能否列一个全面一点的对比，而不是用一两个身边人的经历大喊大叫的。</p>
 <p data-pid="I8MOdNUN">其实题主在提这个问题的时候都已经高二了，看情况也是需要先注销户口走单程证这个途径，高考的优待政策有些赶不及，因为单程证要排一到三年的队。那我这个回答就以更为广泛的角度来答。</p>
 <ol>
  <li data-pid="VIRFInl3">拿了香港身份，<u>内地的身份</u>可以<b>换一种形式更好地保留</b>。因为只要是<u>中国籍香港永久居民</u>，可以很方便地办理一张内地城市的<u><b>港澳台居民居住证</b></u>，配发18位<u>810000开头的公民身份号码。</u>以前没有居住证，拿着回乡证还真是有各种麻烦，现在有了居住证，获发了身份证号，完全可以享受社会主义制度的优越性，一举获得<b><i>（超）</i></b>市民待遇<sup data-text="国务院办公厅关于印发 《港澳台居民居住证申领发放办法》的通知" data-url="http://www.gov.cn/zhengce/content/2018-08/19/content_5314865.htm" data-draft-node="inline" data-draft-type="reference" data-numero="1">[1]</sup>。政策里面表面上是说<u>给予同等待遇</u>，但是每当城市<b>设置了限制措施的时候，港澳人士都是豁免的。</b>比如内地一线城市和一些新一线城市对于住宅采取限购措施，又是比社保年限，又是比各种复杂的积分，而港澳人士不受此限制。但是要注意，万赞答案那个小学入学的情况<sup data-text="隔壁老丁的回答。市教委优待港澳台籍人士子女的规定这种是随时可以调整的。" data-url="https://www.zhihu.com/question/484107763/answer/2117167572" data-draft-node="inline" data-draft-type="reference" data-numero="2">[2]</sup>，<b>反而不要觉得会一直如此</b>，随着当地港澳人士的孩子增多，估计会回到规定所说的“同等待遇”。很多<b>好处都是要闷声享受的</b>，一嚷嚷，媒体一关注，这种本来就只是某些领导为了统战搞的照顾措施，是很容易取消的。</li>
  <li data-pid="cU7wvBFy">下一代如需<b>回内地</b>上<b>双一流</b>，则<b>简单太多</b>。很多人常说<b>香港身份清北随便选，只能说想多了</b>，<b><i>清北复交</i></b>这四所，即便是港澳身份也仍面临激烈的竞争。倒是其他的985大学，或者双一流大学，容易了太多。所以如果下一代想读一个不差的985大学，那么用香港身份是很轻松惬意的，但若希望进清北复交这四所的话，DSE考出来其实都可以在香港读医科了，港澳台联考的话，文理科目前大概每边是2000人，你需要考到前3左右才可能上北大（清华大学不在联考中录取），前10才可能复交，真的不简单，但前1500估摸着都能上211了，尤其是暨南大学，基本上港澳生肯读就能上。</li>
  <li data-pid="CSGryHqK">想<b>读内地硕士</b>的话，香港身份会容易很多。内地考研现在卷成什么样子了，可是如果你是香港身份，申请清北的硕士，那就容易多了，只需要递交申请，不用参加研究生入学统考，经过面试即可入读。<sup data-text="北京大学 2021 年面向香港、澳门、台湾地区招收硕士研究生、博士研究生招生简章（校本部）" data-url="https://admission.pku.edu.cn/docs/20200921114017601058.pdf?CSRFT=5OHB-ES1X-VN40-O3IZ-NDTY-H24H-XDTM-LPIP" data-draft-node="inline" data-draft-type="reference" data-numero="3">[3]</sup>会刷人，但是相比之下，简单太多了。我身边的例子，广东某个四线城市的高考生，来香港很靠后的大学读了本科，之后在香港读了两个硕士，接着去申请北大的硕士，被淘汰，紧接着就去暨南大学读硕士了，一年之后转学去了北大读硕士。就是这么神奇。【留意：报考内地<b>本科</b>的香港考生，可以是持有单程证<u>仅有香港居民身份证</u>但还没有拿到永居的情况；而报考内地<b>硕士</b>，必须是永居。所以题主至少要等7年以后对于读硕才有优势。】</li>
  <li data-pid="G_cJ1V04">在香港定居想买住宅的话，香港永居身份可以节省约27%的税。香港住宅是不限购的， 任何人只要肯付30%的税即可购买，而且套数不限。永居首次置业只需要缴纳约3%的税。</li>
  <li data-pid="X94yMYGh"><b>特区护照</b>依据基本法第一百五十五条，中央人民政府协助或授权特区政府与各国或各地区缔结免签证协议<sup data-text="如何在香港基本法测试中取得满分（含基本法串讲和独家真题）" data-url="https://zhuanlan.zhihu.com/p/412453099" data-draft-node="inline" data-draft-type="reference" data-numero="4">[4]</sup>，所以可以<b>享受一百多个国家的免签或落地签</b>。当然，目前疫情之下，用处倒没那么明显。</li>
  <li data-pid="h07CuvDw">对于某些<b>工匠专业</b>的人来说，Run来香港可以<b>享受更好的人力估值</b>。我所在的某个港漂群，里面有香港各大地产商的工程师，考上公务员的工程师等土木、结构等国内压根看不起的“工匠”，结果他们的工资说出来吓死个人，工作5年的，普遍都是5万以上了，考入政府的，7万多一个月，而且根本就不怎么加班啊，不卷啊。香港新界这边的房子比深圳的还便宜，还是实用面积，这你怎么搞，对吧。按道理我不应该多说的，毕竟内地卷王纷纷run来香港，万一把市场run烂了怎么搞哦。对了，今天（2021年10月6日）特首宣布了<b>施政报告</b>，一方面要推行一大堆基建计划，另一方面，<b>优才计划</b>的配额再提高了一倍，每年由2000人提高到了4000人。<sup data-text="行政長官 2021 年施政報告" data-url="https://www.policyaddress.gov.hk/2021/chi/pdf/PA2021.pdf" data-draft-node="inline" data-draft-type="reference" data-numero="5">[5]</sup>另外，我身边好多IT专业的朋友，都称香港简直是码农的天堂，轻松惬意，收入丰厚。算了，真的不能多说。</li>
  <li data-pid="zkOj6yHI">在内地遇到需要帮助的事情，可以找当地的侨办予以协助。</li>
  <li data-pid="hwnDUpoO">大湾区可能会额外再给港澳人士一些利好。比如目前推出的前海扩容方案，提到了一些利好。<sup data-text="扩容8倍的大前海，对香港人有几多利好？" data-url="https://opinion.southcn.com/node_49184e8afe/dfa2b54143.shtml" data-draft-node="inline" data-draft-type="reference" data-numero="6">[6]</sup></li>
 </ol>
 <p data-pid="7IaBh4ha">好，暂时先说这些优点，接下来补些<b>缺点</b>：</p>
 <ol>
  <li data-pid="_afiamuF">港澳身份目前无法参军，如有军队理想的人士，香港身份不适合。</li>
  <li data-pid="h4jZUmGT">仕途方面，香港人士目前最高最高做到全国政协副主席。有远大理想为人民服务的人士，香港身份不适合。</li>
  <li data-pid="ZJ9-4vmK">只是想考个普通内地公务员的话，目前只有广东这边非常少量的岗位试点港籍公务员，所以可能香港身份不适合。</li>
 </ol>
 <p data-pid="ngHzIS6k">撇除情绪化的东西，<b>香港的很多优势是写进基本法的</b>，目前来看，还有二十多年好日子，所谓五十年不变嘛。比如基本法第一百零九条，关于香港的国际金融中心地位，提到：特区政府需要提供适当的经济和法律环境，保持香港的国际金融中心地位。其实你想啊，一个人均GDP这么高的发达经济体，竟然还不用给中央政府缴纳一分钱国税，国防开支（驻港部队）也不花钱，全世界除了澳门，谁能这么爽？真的想了解香港身份的优势，建议大家读一读香港基本法，看看国家是怎么设立这个特别行政区的，为什么会对香港这么好，好得让马爸爸直言“如果阿里能享受一国两制，不知道会做成什么样子”。<sup data-text="马云：如果阿里能享受一国两制，不知道会做成什么样子" data-url="https://www.bilibili.com/video/BV1Ma4y1p7FH/" data-draft-node="inline" data-draft-type="reference" data-numero="7">[7]</sup>那么就以我写的基本法考满分的文章作为结尾吧，里面有基本法160条内容的串讲。祝大家学习愉快~</p><a href="https://zhuanlan.zhihu.com/p/412453099" data-draft-node="block" data-draft-type="link-card" data-image="https://pic2.zhimg.com/v2-9a55cef13ddefb941b706ec0763194c5_ipico.jpg" class="internal">如何在香港基本法测试中取得满分（含基本法串讲和独家真题）45 赞同 · 7 评论文章</a>
