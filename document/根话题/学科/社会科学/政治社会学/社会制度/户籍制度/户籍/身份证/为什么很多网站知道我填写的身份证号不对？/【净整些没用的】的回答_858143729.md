# 为什么很多网站知道我填写的身份证号不对？
- 点赞数：7015
- 更新时间：2019年12月11日09时00分42秒
- 回答url：https://www.zhihu.com/question/270299828/answer/858143729
<body>
 <p data-pid="UWQoTjQr">身份证号的组成结构：</p>
 <p data-pid="oX3Pg8v8">区域码(6位) 出生日期码(8位) 顺序码(2位) 性别码(1位) 校验码(1位)</p>
 <p data-pid="8k-vZgC9">共18位。</p>
 <ul>
  <li data-pid="GJiD_AjU"><b>区域码</b> 指的是公民常住户口所在县（市、镇、区）的行政划区代码，如110102是北京市-西城区。但港澳台地区居民的身份号码只精确到省级。</li>
  <li data-pid="tM9IoUl-"><b>出生日期码</b> 表示公民出生的公历年（4位）、月（2位）、日（2位）。</li>
  <li data-pid="ESRjHqWU"><b>顺序码</b> 表示在同一区域码所标识的区域范围内，对同年、同月、同日出生的人编定的顺序号。</li>
  <li data-pid="SDlZj9zm"><b>性别码</b> 奇数表示男性，偶数表示女性。</li>
  <li data-pid="uGwf6zk6">最后一位是<b>校验码。</b></li>
 </ul>
 <p data-pid="w_6VMOdy">前面的没啥好说的，最主要的是最后这个<b>校验码</b>。</p>
 <blockquote data-pid="1DiBEO7s">
  最后一位是校验码，这里采用的是ISO 7064:1983,MOD 11-2校验码系统。校验码为一位数，但如果最后采用校验码系统计算的校验码是“10”，碍于身份证号码为18位的规定，则以“X”代替校验码“10”。
 </blockquote>
 <p data-pid="Jc2OuHOn">可能这个解释你不能看懂，那么我下面手撸一遍计算过程。</p>
 <p data-pid="_qg51_l9">校验码计算方式（小学生能看懂版本）：</p>
 <p data-pid="a74RJvbF">假设一个身份证前17位：33078119850907809</p>
 <ul>
  <li data-pid="_SxulWzp"><b>取身份证第1位</b></li>
 </ul>
 <ol>
  <li data-pid="BaVIorSs">计算17减0[身份证位数减1] 17-（1-1）=17</li>
  <li data-pid="cKSGXuBw">计算2的17次方 2^17=131072</li>
  <li data-pid="H_Mm0QtZ">计算131072除以11，取余数 131072➗11=11915……7 下面以 131072%11=7 表示</li>
  <li data-pid="IItQ0gMV">用7乘以3[身份证号第1位数字] 7*3=21</li>
  <li data-pid="lHtaWRJ8">记录下21这个数字</li>
 </ol>
 <p data-pid="uJ07xN1_"><b>取身份证第2位</b></p>
 <ol>
  <li data-pid="-seEaIsD">计算17减1[身份证位数减1] 17-（2-1）=16</li>
  <li data-pid="iWcPfIoc">计算2的16次方 2^16=65536</li>
  <li data-pid="AeXWA5A0">计算65536除以11，取余数 65536%11=9</li>
  <li data-pid="wzpzIHMy">用9乘以3[身份证号第2位数字] 9*3=27</li>
  <li data-pid="Gb1js-2e">记录下27这个数字</li>
 </ol>
 <p data-pid="JHDMt8t6"><b>取身份证第3位</b></p>
 <p data-pid="n6Qh8ePF">………………</p>
 <p data-pid="x1B1boIF"><b>取身份证第17位</b></p>
 <ol>
  <li data-pid="bXzV0M2n">计算17减16[身份证位数减1] 17-（17-1）=1</li>
  <li data-pid="kJelSV9v">计算2的1次方 2^1=2</li>
  <li data-pid="xHRHvt7R">计算2除以11，取余数 2%11=2</li>
  <li data-pid="sJyl66rl">用2乘以9[身份证号第17位数字] 2*9=18</li>
  <li data-pid="hzhUHwyx">记录下18这个数字</li>
 </ol>
 <ul>
  <li data-pid="_t5afnym">我们把这17个数字相加最后结果是423（不信可以自己手动算）</li>
  <li data-pid="lwM4mqoa">计算最后结果423除以11 取余数 423%11=5</li>
  <li data-pid="1YX3SoGR">计算12减5 12-5=7</li>
  <li data-pid="ulTfykPX">计算7除以11取余数 7%11=7</li>
 </ul>
 <p data-pid="UIXrzUnP">最后的计算结果只可能是0-10 这11个数字（感谢@<a href="https://www.zhihu.com/people/qiao-song-19" class="internal">乔松</a> 的指正），因为身份证号只有18位 所以如果最后这个校验码是10那么就记为X</p>
 <p data-pid="mwJCOkqv">那么我们得到最后一位的<b>校验码是7</b></p>
 <p data-pid="7tBPGJQs">如果你有时间，可以手动计算一遍自己的身份证校验码。</p>
 <p data-pid="ivL-Q6Il"><b>为了避免重复计算已经帮你把前面17步重复计算的前三步结果计算好了。</b></p>
 <p data-pid="i9XP3z1M"><b>7，9，10，5，8，4，2，1，6，3，7，9，10，5，8，4，2 校验码权重：1</b></p>
 <p data-pid="CI6JvaYn"><b>也就是你每位身份证号乘以对应位数的数字就行。</b></p>
 <p data-pid="aHUoW07N">后面的计算 拿计算器的话 大概一分钟之内能算完。</p>
 <p data-pid="bupcjDHZ"><b>如果算对了的话 记得回来点赞</b></p>
 <p data-pid="j_QekfIq"><b>下面给出校验码计算的代码 --python3</b></p>
 <div class="highlight">
  <pre><code class="language-text"> def get_check_digit(idnumber):
        """通过身份证号获取校验码"""
        check_sum = 0
        for i in range(0, 17):
            check_sum += ((1 &lt;&lt; (17 - i)) % 11) * int(idnumber[i])
        check_digit = (12 - (check_sum % 11)) % 11
        return check_digit if check_digit &lt; 10 else 'X'</code></pre>
