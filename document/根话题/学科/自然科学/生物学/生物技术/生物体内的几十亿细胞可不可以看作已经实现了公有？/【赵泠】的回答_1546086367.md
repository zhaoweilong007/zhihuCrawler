# 生物体内的几十亿细胞可不可以看作已经实现了公有？
- 点赞数：5562
- 更新时间：2020年11月13日12时07分30秒
- 回答url：https://www.zhihu.com/question/427519857/answer/1546086367
<body>
 <p data-pid="mzF_dw59">人体约有三万亿到三十七万二千亿个细胞，“几十亿细胞”大概是老鼠的规模。题目对生物的认识是天真的。多细胞生物体内的细胞并没有所谓的“社会化”，没什么生产力水平，更不要谈什么共产主义或公有制了——无论你是误解了课本或科普文章对免疫系统的描述，还是《工作细胞》之类看太多了，这都是个美丽的误会。</p>
 <p data-pid="fr8nTxBG">实际上人体内是相当混乱的状态，到处都是错误、无效命令和误启动，经常自我破坏乃至自我毁灭，和人类社会的运转完全是两码事。人体的细胞之间既不讲“平等”也不讲“生存权”，营养物质的分配既不是按需分配也不是按劳分配、经常让具有重要生理功能的细胞饿死·闷死或被代谢废物毒死，对遭病原体感染的细胞的处置方法基本上是“当街击毙”——将这情景反映到人类社会的话，是由暴力与死亡威胁支配的北斗神拳状态。</p>
 <p data-pid="rX5NeZMf">人脑的剪枝机制、梦存在的原因之类，很可能就是和细胞之间的混战状态有关<sup data-text="你就是你的大脑 - 赵泠的文章 - 知乎 https://zhuanlan.zhihu.com/p/290760759" data-url="" data-draft-node="inline" data-draft-type="reference" data-numero="1">[1]</sup>。</p>
 <p data-pid="pUKgjIB4">你今年不是经常听说新冠病毒导致的肺炎么，病毒性肺炎就是人类免疫系统猛烈攻击人体细胞、连正常细胞也一起干掉所导致的。别说共产主义社会了，反乌托邦有多少这样反的呢。</p>
 <p data-pid="r_cK22fP">癌症可以理解为细胞层次的自然选择：复制错误、辐射、病毒感染、药物干扰等造成不同细胞的性状差异，癌细胞重启继承自古老单细胞祖先的多个基因、在对抗免疫细胞和普通细胞时获得优势。你从人类社会里找一个癌性不死化的个体出来看看啊。</p>
 <p data-pid="v_TrzYal">癌细胞的特征：</p>
 <ul>
  <li data-pid="5BcBcF0A">抑癌基因沉默或丢失</li>
  <li data-pid="FKcuQ6x_">关闭细胞凋亡，尤其是细胞从细胞外基质分离时的程序性死亡<sup data-text="失巢凋亡" data-url="" data-draft-node="inline" data-draft-type="reference" data-numero="2">[2]</sup></li>
  <li data-pid="LFi7pjFX">用端粒酶修复受损的端粒</li>
  <li data-pid="P2NPPn-q">去除表面受体来避开免疫细胞的攻击</li>
  <li data-pid="jxBP-Jk5">改变粘弹性特征来促进运动、侵袭、定植</li>
  <li data-pid="m5dEznaa">分泌腐蚀性酶在组织上开辟通道，以进入血液和淋巴循环</li>
  <li data-pid="wMJjmXx6">改变接触到的细胞外基质与组织的理化性质以便定植</li>
  <li data-pid="lzghOnsH">在缺氧条件下关闭氧化磷酸化代谢、启动糖酵解</li>
  <li data-pid="z_vuNmF9">比癌变前耐酸</li>
  <li data-pid="hgdareuv">独立产生有丝分裂信号和生长因子</li>
  <li data-pid="LzDc-KgZ">增加遗传不稳定性以加速演化</li>
  <li data-pid="GC9ODQK5">快速适应的细胞骨架使有丝分裂能在非整倍体下进行<sup data-text="癌症晚期患者体内的许多癌细胞的细胞核严重不正常，对普通细胞来说在那个状态下立即就会自我毁灭，但人家活得好着呢，还能一次分裂成3~5个细胞。" data-url="" data-draft-node="inline" data-draft-type="reference" data-numero="3">[3]</sup></li>
 </ul>
 <p data-pid="UyuTmuXM">这类特征的共同性质是：</p>
 <ul>
  <li data-pid="luaYlUx_">放弃让复杂多细胞动物成立的“多组织合作”，</li>
  <li data-pid="yTSSdRBT"><b>没有放弃组织内合作</b>，癌细胞对微环境的影响对其它癌细胞是有利的。</li>
 </ul>
 <p data-pid="jng53ep1">你可以在海绵等结构简单的多细胞动物的正常细胞里找到这类特征，基本上这些功能是多细胞生物共同祖先持有的复杂工具箱和武器库的一部分。所以多种细胞癌变的结果都差不多，多个物种的癌细胞的表现看上去都相似。</p>
 <p data-pid="sUelhOH7">对人癌细胞来说，对免疫疗法的抵抗力很大程度上依靠182个“核心癌症固有免疫逃逸基因”的随机缺失<sup data-text="https://doi.org/10.1038/s41586-020-2746-2" data-url="" data-draft-node="inline" data-draft-type="reference" data-numero="4">[4]</sup>，当你将细胞视为个体的时候，这是怎么回事呢：</p>
 <ul>
  <li data-pid="duFj-Bs0">普通人被植入了各种自爆开关，警察按下遥控器你就会当场爆裂而死。</li>
  <li data-pid="B5xJbjcC">伸手将开关拔掉，普通人就会变成警察无力抵挡的怪兽，把社会打瘫痪。</li>
 </ul>
 <p data-pid="ZkHC2KSD">这似乎是二十世纪末美国B级片的情景。《心理测量者》系列的当街击毙、《自新世界》的“愧死”之类弱智机制相比之下都要更好吧。</p>
