 <p data-pid="8APPmzAW">没错，就是白菜箱子里长出来的！</p>
 <p data-pid="vfuEclt8">今年因为疫情嘛，全家几天去大超市采购一次，正月十六那天中午，我们全家从超市回来，我爸妈说把目前不吃的菜放到另一栋的老房里（俩房在一个小区，刚搬家完），现在住的新房冰箱搁不下了，于是我在车里等，我爸妈上楼。</p>
 <p data-pid="UFwvBJBh">百无聊赖之际，我妈突然打电话给我，说发现一猫！我蹭蹭蹭就按电梯上楼，然后就看到我爸妈正在围观猫猫！</p>

 <p data-pid="6Iidtxpw">北方冬天，楼道里是天然冰箱，我妈把白菜放纸箱子里，上面用一个破棉服盖上。因为跟对门人家也比较熟，我们两家都把一些蔬菜水果啊放门口。当时看到对门人家的大葱和柿子都被掀翻了，估计就是她干的</p>
 <p data-pid="W0yzJpW6">我妈当时想拿棵白菜，结果一掀开棉服，发现一个毛茸茸的东西！我妈当时吓一跳，说以为是狐狸，再一看是只猫猫，猫猫很乖，也不叫，我爸妈赶紧叫我上来围观～</p>
 <p data-pid="lnDcgxwH">说实话我之前一直是喜爱狗甚于猫，可能是因为小时候养过狗的缘故，总觉得猫不跟人亲（大误）所以从来没有想过养猫，只是在网上云吸猫，但是看到这只猫猫，发现长得还不错，这也是第一次亲密接触猫，立马变身好奇宝宝，上手摸了几下，猫猫非常乖！没有任何拒绝反抗！</p>
 <p data-pid="YDjIcVt0">当时楼道冷，我们一家三口就先把它抱进屋里。因为刚从这边房子搬家到新房，所以屋里没啥东西，我们就把猫猫放下来让她自由活动，然后看看我们小区的微信群有没有找猫的，翻了两三天的，然而并没有，我们决定先回新房吃午饭，再商量怎么办。（后面我有详细讲，为什么推测她是被遗弃，为什么找不到原主人）</p>
 <p data-pid="1K_mYheL">当时我有点动心思想养了，因为第一次接触猫，感觉不错呦，以后就可以上手撸猫了！但是我爸妈不同意，一是觉得现在都没有精力养，都得忙工作，二是当时疫情还非常严重，正月十六，2月9号，那一阵也正传猫狗可能感染病毒传人啊什么的，我们猜这个猫猫可能是被遗弃了，我爸妈心里也比较介意，毕竟当时疫情形势还非常紧张，人命关天，不怕一万就怕万一啊。我就讲，这是没有科学依据的啊，说猫猫的可爱啊，讨论了一会，感觉我妈口风有点松动，应该是看上这只猫猫了⁽˙⁸˙⁾</p>
 <p data-pid="9XRIaPSS">然后！吃完饭后，我就开始抱着手机查猫猫怎么养，想再说服说服爸妈，结果我妈提出要去老房子把猫猫带过来，让她吃点东西喝点水，我爸当时很不情愿，应该是发现我妈在逐步沉沦ᶘ ͡°ᴥ͡°ᶅ 但是没有办法拒绝，于是俩人就把猫猫带回新房了！</p>
 <p data-pid="CaqgoPiJ">带回来后，我顺势就说，我们养吧！我妈说:我们先养着，等有人找再给回去！大家听出来了吧这就是想养啊！我妈五十岁的养猫少女心焕发了哈哈哈！开心！与此同时，我爸特别生气地对我俩说了一句:“现在人都是闲的吧！天天人还养不好还养猫！”请注意十分钟后他就将被打脸 ヽ(๑Θ｡Θ๑)ﾉ</p>
 <p data-pid="4XlqDdEU">于是我赶紧问基友圈养猫的家长们怎么养猫，置办东西，同时猫猫开始不见外地踏巡新领地:</p>

 <p data-pid="3vQ80-4b">石榴生产是在2月20号，猫怀孕按63天算，石榴配种差不多在12月20号左右，那时候绝对谁都不知道会有疫情，还一片祥和呢，1月下旬关于疫情的消息才真正紧张起来</p>
 <p data-pid="xst7NXm1">我捡到石榴，是在正月十六，那时候疫情真的很严峻 ，差不多是在人们最紧张的时段。那个时候，也恰好有“猫狗能传染病毒”的各种消息传出，我爸养石榴后才注意到，跟我说小区有流浪猫，以前我们一家是真的不太注意，这可能就跟“孕妇效应”差不多……</p>
 <p data-pid="2phnX97i">所以， 因为之前配了种，疫情期间就肯定不会丢猫，这个逻辑不成立。当然，绝大部分家长，都不会丢猫，但不可否认，肯定有人选择相信了那些消息，丢猫了，包括我爸妈一开始犹豫不想养，也是担心病毒，因为真的是人命关天啊，当时也没有个科学准确的官方消息，谁都没法保证，所以，石榴由于疫情被遗弃的可能性，是有的 ( 0 x 0 )</p>
 <p data-pid="QcRVydIF">……………………………………………………………………</p>
 <p data-pid="v-EPwLZp">然后，我来怼一怼杠精们五花八门的没有逻辑的指责:</p>
 <p data-pid="qkIO4XUJ">一、杠精们说的，石榴不可能被遗弃的种种原因都太正确了，石榴是品种猫，可爱漂亮，又被配了种，但是，这也是你们逻辑中最大的漏洞，</p>
 <p data-pid="MPHLvQOF">那就是 : 如果是不小心跑丢的猫猫，这么珍视猫猫，为什么原主人不找？你们提到的种种办法，都应该是丢猫的人找猫的办法，而不应该是我必须全部用上找原主人的办法！ ( 0 x 0 )</p>
 <p data-pid="4435Rl77">如果原主人用了其中任何一种办法，我很大概率都是能知道的，如果他各种方法都用了，我肯定知道（后面在各个方法上我会说原因）但是！我一点找猫讯息都没碰到；</p>
 <p data-pid="1pSaaopz">不要问我为啥不想方设法找到原主人？在我发布过信息后，确认没有人找猫，我不可能一直发信息，那是扰民，而且如果原主人像被抓犯人一样终于被我揪出来了，他表面假意接回，后面又扔了，那我是把石榴给害了，我不是圣母，也没有猫病……</p>
 <p data-pid="GXZnQSO9">二、假设原主人找了，还原当时的特殊时期，注意那时候是疫情时期，一个丢了猫猫想找回猫猫的人，都会用什么办法呢？</p>
 <p data-pid="mCoi8rs7">杠精们提到的我应该找猫主人的各种办法，我们反向依次来看，假设原主人想找猫:</p>
 <p data-pid="-6ppQB8k">1.宠物医院贴告示，先pass，因为当时宠物医院都关门了，我指的是全部，大家应该还有印象，那时候街面所有商铺，饭店商店衣服店等等，全都关门了，我后面也有说，就在我打电话那一天，正被通知关门……而且，即便是平时，宠物医院也不是志愿救助队啊，真的不可能把寻猫告示贴前台或者贴门玻璃的……起码我去宠物医院看到的都是人家医院自己的广告，没有任何寻猫信息啥的；</p>
 <p data-pid="LjgPoART">2.小区内贴告示，非常好，这也是我很质疑的一点，正月十六那时候疫情真的还很严峻，我回老房子取东西都有点心虚，总觉得碰见人都不好似的，但我也冒着风险出去看了看，没有任何告示。</p>
 <p data-pid="SWYELzGO">我妈家小区是返迁房小区，一共就37栋楼，还没有高层，最高是11层的，我家新房还是一楼，所以如果贴了告示，我是一定能看到的，肯定没贴；我爸那时候做为村支委，四个庄的村干部曾经一起挨家挨户做过入户信息登记调查，要是有，咋滴他也能看见一张啊，但是半张都没有，小区里没有任何告示。</p>
 <p data-pid="LJ1iOGLr">不要问我为啥不贴告示找原主人，为啥我爸不贴告示 ……贴告示其实本身是不符合物业规定的啊，当然如果是找猫的人肯定着急，贴了就贴了，但我发布出去的微信消息渠道，门卫消息渠道已经有了，我再贴告示不合适，一是我爸这村支委显得带头破坏工作； 二是，当时关于猫狗消息的情势并不明朗，当时还有新闻，有些小区把流浪猫狗捕杀了，我怕万一哪天消息变了，政策变了，如果我们这边也让把流浪猫狗捕杀，石榴会被当做流浪猫处理，到时会很棘手，所以我不可能大张旗鼓地贴告示，我当时心里是提心吊胆的 = =</p>
 <p data-pid="p8zoD4wV">3.微信问。这其实是在当时疫情期间，最可行的一个办法，当时疫情，小区出入都得登记，人们几乎全都待在家，没有出来溜达的，物业还有值班人员一直在巡视，不让聚集，所以，假设主人去外地了，或者不在小区，他主要靠网络寻找，没毛病，因为我也是主要靠微信来找主人的。</p>
 <p data-pid="s7__ZoXA">我们小区一共俩业主微信群，我都加了，发问，但是都没有人回应，我翻了之前两天群消息，没人找猫，又盯了捡到石榴后两天的群消息，都是发广告的，什么卖口罩卖草莓，但是没有人找猫，最主要，也没有人回应我。</p>
 <p data-pid="f6HxdSBA">如果主人想找猫，最先想到的应该就是微信群里问，进我们小区这俩群真不是啥难事。我下面截的图是今天的其中一个微信群，群里会有找钥匙的，找水杯的，还有叫孩子回家吃饭的 ……之前的消息我没有截图，我为了省内存都是当天清理消息 ( 0 x 0 )</p>

 <p data-pid="DWzLluZJ">但是如果原主人找了，或者他想找回猫，咋地也会回复我，当时怕错过找猫信息，我这个群消息都没屏蔽，一天没用的消息也挺多，但是我想如果原主人真的急着找猫，他肯定跟我一样不会错过任何一条消息。</p>
 <p data-pid="zvhtuNol">我还让家里亲戚，我姑妈，表哥表姐帮我留意他们所在的各种群，因为原来都在一个庄，现在都在一个小区，我们家族的整体消息覆盖面，可以说挺广的，起码在小区里，面对小区里这四个庄的人，以及周围村庄，如果有人有找猫的消息，我们很大概率能知道，但是没人找猫。</p>
 <p data-pid="YFkAX_N4">我所在的城市是五六线，举个栗子 ，去年夏天下大雨，我车牌子被冲走了，我发朋友圈找，第二天就通过朋友传播找到了，是一个快手博主发了朋友圈，别人看到其中有我的车牌子……当然我能找到也有偶然性，但是我举这个栗子，就是想说，对于一个市区尚且如此，对于我们一个返迁小区，如果想找猫猫这种明显活物，我是肯定能获知消息的，但是没有，那很大概率就是他没找。</p>
 <p data-pid="Wqpl1H-4">4.按照我前面说的，杠精还会问，你们小区这么小，你这么神通广大，为啥不知道猫是谁的？</p>
 <p data-pid="LB5MNdH8">我不是神探啊，我就算是公安局查户口也不可能查到每一只猫啊……只能说，只有在我和找猫者双向发布消息的情况下，才能互相碰上，单向的话，就没有后续了 ( 0 x 0 )</p>
 <p data-pid="Po2GmpPB">虽然小区里大部分人都认识，都是原来周围四个庄的，但是！有一部分人是外地租户，因为小区租金便宜啊，这部分人我是不认识的 = =</p>
 <p data-pid="EMClHMDj">而且，小区入住率不高，我们这一单元，住满是12户，但实际才住了三户，猫又不像狗总出来溜，石榴可能压根没怎么见过外人，不可能见过这猫的人都记得，所以靠人跟人口口相传，就能打听到这是谁家猫的几率，很低，尤其当时疫情期间，不像平时聚堆唠嗑，根本小区里没人聚堆，所以没法打听 ( 0 x 0 )</p>
 <p data-pid="Z439kQbH">5.物业、门卫发布信息，太对了！这个渠道我之前一直没说 ，因为我爸经常在门卫值班，我真没想起来单说这一条。</p>
 <p data-pid="yrKGIb32">情况就是 : 我爸是党员 ，还是村支委，四个庄的村干部轮流在门卫的防疫帐篷那值班，那时候疫情严重期间，是两天一轮班，我爸跟其他几个庄的干部都认识，跟门卫更熟，门卫是我们庄的，所以如果原主人去门卫问过（这是最正常的找猫方法之一），或者上门口那个值班帐篷那问过，我爸肯定是第一个知道的，但是没有人问过任何找猫消息。</p>
 <p data-pid="pAvIxf8t">我爸值班，从一开始两天一轮，到现在疫情没那么严重，是八天一轮班，石榴生完孩子都仨月了，一直就没有人找猫。如果原主人真想找，现在也不封村封小区了，回来问问总可以吧？没有，那就是真没找了( 0 x 0 )</p>
 <p data-pid="-bPkh8dw">其实如果原来的主人想找，还有一个方法，小区物业有大喇叭广播，或者豁出去自己带个大喇叭喊也是可以的，以前小区卖豆腐，磨剪子啥的都是用大喇叭喊，虽然挺没面子，但如果是我想找，我挨家挨户敲门也会找的，但是原主人并没有找</p>
 <p data-pid="cAaJTx1_">(･᷆ω･᷇)不要说他可能敲门了但是我没在家， 从正月初三从我对象老家回来，我就住我妈家，天天家里有人，我们出去买菜时候我奶也在家</p>
 <p data-pid="CvIMwRbJ">……………………………………………………………………</p>
 <p data-pid="30w-_PoQ">综上所述:</p>
 <p data-pid="rQn6QsjF">原主人有可能是因为疫情的缘故，也有可能是租户要回老家了，等等等等，也许有很多种可能，但我不想猜测，没有意义了，结果就是他没有找猫，现在小猫都仨月了，所以杠精们真的别yy了，也别空口杠了，</p>
 <p data-pid="NFw7FHA_">最终结果就是:石榴和石榴崽崽们是我家一份子啦～ 即便是后面真的找来，就是亲妈，我也是绝不可能还回去的！就这么硬气了！</p>
 <p data-pid="LMPOmFWC">有杠精说我得精心伺候 ，但即便精心伺候一年，原主人来找我也得给？emm</p>
 <p data-pid="QIvFd68W">还有的说原主人给钱我就得给？还帮我算了我最多要一万五emmm</p>
 <p data-pid="EEIIe0eh">还有的说我不找到原主人是为了卖小猫赚钱emmm</p>
 <p data-pid="ZwncudU0">只能说你们没养过猫，或者养猫目的不同，虽然我们家是第一次养猫，但是养猫不仅付出物质，还是有感情的，虽然是养宠物 ，但不知不觉，她们就成了家人，可能你们理解不了 ( 0 x 0 )</p>
 <p data-pid="xyoTjl5d">通过后面的观察，石榴不会主动走出家门，我家是一楼，有时我妈开门倒垃圾时，我会在石榴身后暗中观察 ，石榴会在门口张望，但是不会走出去，就是看着人的背影；每次有人回家时，她都会颠颠地到门口迎接；</p>
 <p data-pid="S_ML6zsf">上周带她去医院绝育，通过这几次出门经历，发现她挺怂，出门就害怕，必须得抱着，然后小爪子紧巴着人的胳膊～</p>
 <p data-pid="V7vGC7WL">由此更加相信，她当时在怀孕期，生产的前十天，不是发情期，不太可能是自己跑丢的。不管怎么样，感谢缘分，让我们一家遇到了大石榴！！</p>
 <p data-pid="6YN1Ew89">2.石榴刚到家时就上床上沙发～</p>
 <p data-pid="Q_JPAGyt">是的！当时她刚到家，我跟有经验的朋友打听完之后，先用湿巾简单地擦了擦她的身子和爪子，然后别人告诉我要体检，驱虫，但是，当时由于疫情，我们市区基本所有的宠物医院都关门了……我特意发朋友圈问了问，有人回复我说有个医院还没关门，我一打电话，人家说，刚接到统一通知，正在关门，所以我印象很深刻 ……</p>
 <p data-pid="MF2zbdmW">然后就没啥办法了，只能检查检查她的毛，发现挺干净，没虫子，当时淘宝物流还不通，只能京东买各种用品，想着医院开门了给她体检去，没想到十天后她就生了……</p>
 <p data-pid="N-aUqHKA">我也一直提心吊胆，生怕有什么病，幸好石榴棒棒哒，崽崽们也很健康，没有我害怕的猫瘟猫鼻支猫传腹啥的，只是两个月后出现了一次耳螨，幸好很快治愈了～</p>
 <p data-pid="GoAXhgHb">3.石榴发情和绝育</p>
 <p data-pid="aJdAZ0yj">她在崽崽出生俩月后开始发情，第一次发情三天，间隔一周后第二次发情，这次发情了一周（有的大夫说这是一次发情期）发情症状结束后就带去医院噜，先体检，又做了抗体检测，一切正常，顺利绝育！昨天刚拆完线，再戴三天伊丽莎白圈，就解放噜！喵生大事了却一桩～</p>

 <p data-pid="_w-GUVYx">绝育:</p>
