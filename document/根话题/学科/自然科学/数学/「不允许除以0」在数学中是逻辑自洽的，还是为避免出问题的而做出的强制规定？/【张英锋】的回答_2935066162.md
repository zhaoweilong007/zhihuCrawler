# 「不允许除以0」在数学中是逻辑自洽的，还是为避免出问题的而做出的强制规定？
- 点赞数：5965
- 更新时间：2023年10月30日11时39分59秒

<body>
 <p>杀鸡不要用牛刀，能用中学数学解释的问题，就不需要用微积分或抽象代数来解释。</p>
 <p>首先，「不能除 0」与其说是强制规定，倒不如说是人们的自然选择。</p>
 <p>其次，你可以除 0，只不过要付出一个小代价，这个代价就是数学大厦的轰然倒塌。</p>
 <p>很多教科书上说「<strong>除以 0 没有意义</strong>」，这种简单粗暴的回答既不准确，又无法帮我们真正理解数学。</p>
 <p>让我们换一种提问方式：</p>
 <p><strong>如果允许「除以 0」，你将会进入一个怎样的数学世界</strong>?</p>
 <figure>
  <img alt="​《除零之门》MidJourney 生成" data-rawheight="720" data-rawwidth="720" src="https://pic1.zhimg.com/50/v2-dded88d6554c607a695aed111b2a2213_720w.jpg?source=1940ef5c" data-original-token="v2-dded88d6554c607a695aed111b2a2213" class="origin_image zh-lightbox-thumb" width="720" data-original="https://pica.zhimg.com/v2-dded88d6554c607a695aed111b2a2213_r.jpg?source=1940ef5c">

 <p>让我们从这个不起眼的问题出发，层层递进的提问，一起探索一个有趣的数学世界。</p>
 <p>在这个世界里，你不仅能更好的理解这一规则背后的思想，还能帮助你避免在证明时犯低级的错误。</p>
 <p>先说结论：</p>
 <p>加入「不能除 0」这条限制，并不是因为除以 0 没有意义，而是因为这会让计算的结<strong>果出现不</strong>确定性，让计算的过<strong>程出</strong>现悖论，专业一点说，这破坏了<strong>数学的</strong>一致性。</p>
 <h1><strong>为什么计算结果会出现不确定性？</strong></h1>
 <p>先解释什么是计算的确定性</p>
 <p>比如，1+1=2，1/2=0.5</p>
 <p>这些计算的结果是确定，你不会随便给出其他结果，这就是计算的确定性。</p>
 <p>有人说，一元二次方程有两个解，三角函数有无穷多个解，这是确定的吗？</p>
 <p>这也是确定的，因为根据这些函数的方程</p>
 <p><img src="https://www.zhihu.com/equation?tex=y%3Dax%5E2%2Bbx%2Bc" alt="y=ax^2+bx+c" eeimg="1"></p>
 <p><img src="https://www.zhihu.com/equation?tex=y%3Dsin%28x%29" alt="y=sin(x)" eeimg="1"></p>
 <p>输入一个 x 的值，比如 0，我们就明确知道 y 的计算输出结果，而不是随便一个数值，所以这也是确定的。</p>
 <p>但是，1/0=?，这个计算的结果是多少？你能确定有多少解吗？</p>
 <p>有人说，很简单啊，结果是<strong>无穷大</strong> ∞ ，因为如果写成函数的形式</p>
 <p><img src="https://www.zhihu.com/equation?tex=y%3D%5Cfrac%7B1%7D%7Bx%7D" alt="y=\frac{1}{x}" eeimg="1"></p>
 <p>当分母 x 越趋近于 0，y 就会越趋近于无穷大 ∞ 。</p>
 <p>例如下图，当 x 从大到小无限靠近 0 时，y 就会快速增大为无穷∞</p>
 <figure>
  <img alt="" data-rawheight="720" data-rawwidth="778" src="https://pic1.zhimg.com/50/v2-1c25d6ea34deb8daa17938c36ae304dd_720w.jpg?source=1940ef5c" data-original-token="v2-1c25d6ea34deb8daa17938c36ae304dd" class="origin_image zh-lightbox-thumb" width="778" data-original="https://pica.zhimg.com/v2-1c25d6ea34deb8daa17938c36ae304dd_r.jpg?source=1940ef5c">
 </figure>
 <p>​如果 y 的结果是无穷大，那计算的结果还是确定的，<strong>因为无穷大也是确定的解</strong>，很多函数在特定的点上也会出现无穷大。这不是随便给出结果，所以并不是问题所在。</p>
 <p>真正的问题在于，这个计算结果是不完整的。</p>
 <p>这个计算只考虑从正数一侧靠近 0，没有考虑到还有负数一侧。</p>
 <p>如果 x 从负数一侧靠近 0，y 的计算结果就会靠近负无穷-∞。</p>
 <p>例如下图给出了双侧的结果。</p>
 <figure>
  <img alt="" data-rawheight="720" data-rawwidth="782" src="https://pic1.zhimg.com/50/v2-250126db7c62f936a8a1e736c8e09a08_720w.jpg?source=1940ef5c" data-original-token="v2-250126db7c62f936a8a1e736c8e09a08" class="origin_image zh-lightbox-thumb" width="782" data-original="https://picx.zhimg.com/v2-250126db7c62f936a8a1e736c8e09a08_r.jpg?source=1940ef5c">
 </figure>
 <p>不同的趋近方向计算结果是完全不同。你就说，挣 1 个亿和赔 1 个亿，能一样吗?</p>
 <p>那在 0 这个点上，y 的取值究竟是正无穷+∞，还是负无穷-∞？</p>
 <p>你能确定吗？是不是不能？</p>
 <p>也就说，对于函数</p>
 <p><img src="https://www.zhihu.com/equation?tex=y%3D%5Cfrac%7B1%7D%7Bx%7D" alt="y=\frac{1}{x}" eeimg="1"></p>
 <p>当它站在 x=0 这个点上成为 <img src="https://www.zhihu.com/equation?tex=%5Cfrac%7B1%7D%7B0%7D" alt="\frac{1}{0}" eeimg="1"> 时，突然进入了一种不确定的状态，y 既不是正无穷，也不是负无穷，或者可以都是。</p>
 <p>这就是<strong>不确定的</strong>。</p>
 <p>有人说，两个结果好像也没问题啊？不就是正负两个无穷大吗？</p>
 <p>问题在于，除了正负无穷大， <img src="https://www.zhihu.com/equation?tex=%5Cfrac%7B1%7D%7B0%7D" alt="\frac{1}{0}" eeimg="1"> 还可以是从负无穷到正无穷中间的<strong>任何一个值</strong>。</p>
 <p>也就是随便给个数，都可以算是 <img src="https://www.zhihu.com/equation?tex=%5Cfrac%7B1%7D%7B0%7D" alt="\frac{1}{0}" eeimg="1"> 的正确结果，不再有对错之分，这是一种更诡异的不确定性。</p>
 <p>怎么会这样？很奇怪是吧！具体原因后面再解释。</p>
 <p>有人可能会问：</p>
 <p><strong>这有什么问题吗，结果不确定又怎么了，我就想让除法保留「除以 0」不可以吗</strong>？</p>
 <p>可以，当然可以了！</p>
