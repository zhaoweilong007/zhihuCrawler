# 行列式的本质是什么？
- 点赞数：7370
- 更新时间：2024年01月05日10时49分58秒
- 回答url：https://www.zhihu.com/question/36966326/answer/162550802
<body>
 <p data-pid="zKz7eBmw">先说答案：行列式是线性变换的伸缩因子。</p>
 <p data-pid="0ZtGbpHg">理解行列式一定要从线性变换出发去理解，直接去理解它的代数形式是没有意义的。</p>
 <p data-pid="TjurrlvP">这篇文章的结构是：</p>
 <ul>
  <li data-pid="oNlDMT0c">线性变换的几何直观</li>
  <li data-pid="75S7zEIH">实现线性变换的矩阵</li>
  <li data-pid="XYmyc_uU">行列式</li>
 </ul>
 <p data-pid="ObHa6cWH"><b>1 线性变换的几何直观</b></p>
 <p data-pid="DuKJqYiS">线性变换的几何直观有三个要点：</p>
 <ul>
  <li data-pid="WAm9j9gH">变换前是直线的，变换后依然是直线</li>
  <li data-pid="zoQXbRWn">直线比例保持不变</li>
  <li data-pid="DpcuCmyu">变换前是原点的，变换后依然是原点</li>
 </ul>
