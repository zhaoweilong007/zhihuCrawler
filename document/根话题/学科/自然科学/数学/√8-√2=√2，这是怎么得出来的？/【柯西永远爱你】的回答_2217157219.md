# √8-√2=√2，这是怎么得出来的？
- 点赞数：6039
- 更新时间：2021年11月21日09时24分21秒
- 回答url：https://www.zhihu.com/question/461209225/answer/2217157219
<body>
 <p data-pid="EKZBs11U">这是一个<b>有难度</b>的问题，要想证明它，我们需要一些<b>数学分析</b>的知识</p>
 <p data-pid="B6wClUnY">本题要证明 <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Csqrt+8++-+%5Csqrt+2++%3D+%5Csqrt+2+%5C%5D" alt="\[\sqrt 8  - \sqrt 2  = \sqrt 2 \]" eeimg="1"></p>
 <p data-pid="Vm5rJIp9">移项，等价于证明 <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Csqrt+8++-+2%5Csqrt+2+%3D0%5C%5D+" alt="\[\sqrt 8  - 2\sqrt 2 =0\] " eeimg="1"></p>
 <p data-pid="v-ZFuupN">构造函数 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29+%3D+%5Csqrt+%7B4x%7D++-+2%5Csqrt+x+%5C%5D" alt="\[f\left( x \right) = \sqrt {4x}  - 2\sqrt x \]" eeimg="1"></p>
 <p data-pid="EPIwqkbV">则要证明 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+2+%5Cright%29+%3D0%5C%5D" alt="\[f\left( 2 \right) =0\]" eeimg="1"></p>
 <p data-pid="gY_ViMm9"><b>因此我们只要证</b> <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Cforall+x+%5Cin+%5Cleft%5B+%7B0%2C3%7D+%5Cright%5D%2Cf%5Cleft%28+x+%5Cright%29+%5Cequiv+0%5C%5D" alt="\[\forall x \in \left[ {0,3} \right],f\left( x \right) \equiv 0\]" eeimg="1"></p>
 <p data-pid="G2Ob9qC4">因为 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29%5C%5D" alt="\[f\left( x \right)\]" eeimg="1"> 连续（利用<b>初等函数在定义域内是连续函数</b>）</p>
 <p data-pid="eNAuA_pT">所以 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29%5C%5D" alt="\[f\left( x \right)\]" eeimg="1"> 在闭区间 <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Cleft%5B+%7B0%2C3%7D+%5Cright%5D%5C%5D" alt="\[\left[ {0,3} \right]\]" eeimg="1"> 上能取到最大值和最小值</p>
 <p data-pid="pTnnIvyy">不妨设为 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+%7B%7Bx_1%7D%7D+%5Cright%29+%3D+M%2Cf%5Cleft%28+%7B%7Bx_2%7D%7D+%5Cright%29+%3D+m%5C%5D" alt="\[f\left( {{x_1}} \right) = M,f\left( {{x_2}} \right) = m\]" eeimg="1"></p>
 <p data-pid="X4SpfnLF">其中 <img src="https://www.zhihu.com/equation?tex=M%2Cm" alt="M,m" eeimg="1"> 分别表示 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29%5C%5D" alt="\[f\left( x \right)\]" eeimg="1"> 的最大值和最小值， <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_1%7D%2C%7Bx_2%7D+%5Cin+%5Cleft%5B+%7B0%2C3%7D+%5Cright%5D%5C%5D" alt="\[{x_1},{x_2} \in \left[ {0,3} \right]\]" eeimg="1"></p>
 <p data-pid="VNvQw-9B">根据幂函数、复合函数求导法则以及导数的四则运算，可得</p>
 <p data-pid="7P9vFAYp"><img src="https://www.zhihu.com/equation?tex=%5C%5Bf%27%5Cleft%28+x+%5Cright%29+%3D+%5Cfrac%7B4%7D%7B%7B2%5Csqrt+%7B4x%7D+%7D%7D+-+2%5Cfrac%7B1%7D%7B%7B2%5Csqrt+x+%7D%7D+%3D+%5Cfrac%7B2%7D%7B%7B%5Csqrt+%7B4x%7D+%7D%7D+-+%5Cfrac%7B1%7D%7B%7B%5Csqrt+x+%7D%7D+%3D+%5Cfrac%7B%7B2%5Csqrt+x++-+%5Csqrt+%7B4x%7D+%7D%7D%7B%7B%5Csqrt+x+%5Csqrt+%7B4x%7D+%7D%7D+%3D++-+%5Cfrac%7B%7Bf%5Cleft%28+x+%5Cright%29%7D%7D%7B%7B%5Csqrt+x+%5Csqrt+%7B4x%7D+%7D%7D%5C%5D" alt="\[f'\left( x \right) = \frac{4}{{2\sqrt {4x} }} - 2\frac{1}{{2\sqrt x }} = \frac{2}{{\sqrt {4x} }} - \frac{1}{{\sqrt x }} = \frac{{2\sqrt x  - \sqrt {4x} }}{{\sqrt x \sqrt {4x} }} =  - \frac{{f\left( x \right)}}{{\sqrt x \sqrt {4x} }}\]" eeimg="1"></p>
 <p data-pid="6e_sv4Iv"><b>以下需要分类讨论</b></p>
 <h3><b>情形一：</b> <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_1%7D%2C%7Bx_2%7D+%5Cin+%5Cleft%28+%7B0%2C3%7D+%5Cright%29%5C%5D" alt="\[{x_1},{x_2} \in \left( {0,3} \right)\]" eeimg="1"></h3>
 <p data-pid="TcF2Xziw">此时 <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_1%7D%2C%7Bx_2%7D%5C%5D" alt="\[{x_1},{x_2}\]" eeimg="1"> 均为函数 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29%5C%5D" alt="\[f\left( x \right)\]" eeimg="1"> 的极值点</p>
 <p data-pid="q4mpsrTs">于是根据<b>费马引理</b>可知 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%27%5Cleft%28+%7B%7Bx_1%7D%7D+%5Cright%29+%3D+f%27%5Cleft%28+%7B%7Bx_2%7D%7D+%5Cright%29+%3D+0%5C%5D" alt="\[f'\left( {{x_1}} \right) = f'\left( {{x_2}} \right) = 0\]" eeimg="1"></p>
 <p data-pid="UVBnbMHP">代入至 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%27%5Cleft%28+x+%5Cright%29+%3D++-+%5Cfrac%7B%7Bf%5Cleft%28+x+%5Cright%29%7D%7D%7B%7B%5Csqrt+x+%5Csqrt+%7B4x%7D+%7D%7D%5C%5D" alt="\[f'\left( x \right) =  - \frac{{f\left( x \right)}}{{\sqrt x \sqrt {4x} }}\]" eeimg="1"> 可知 <img src="https://www.zhihu.com/equation?tex=M%3Dm%3D0" alt="M=m=0" eeimg="1"></p>
 <p data-pid="iDTsYQjt">也就是说 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29%5C%5D" alt="\[f\left( x \right)\]" eeimg="1"> 的最大值和最小值均为 <img src="https://www.zhihu.com/equation?tex=0" alt="0" eeimg="1"></p>
 <p data-pid="r6-vC6iL">故 <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Cforall+x+%5Cin+%5Cleft%5B+%7B0%2C3%7D+%5Cright%5D%2Cf%5Cleft%28+x+%5Cright%29+%5Cequiv+0%5C%5D" alt="\[\forall x \in \left[ {0,3} \right],f\left( x \right) \equiv 0\]" eeimg="1"></p>
 <h3><b>情形二：</b> <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_1%7D%2C%7Bx_2%7D%5C%5D" alt="\[{x_1},{x_2}\]" eeimg="1"> 之中至少有一个 <img src="https://www.zhihu.com/equation?tex=%5C%5B+%5Cnotin+%5Cleft%28+%7B0%2C3%7D+%5Cright%29%5C%5D" alt="\[ \notin \left( {0,3} \right)\]" eeimg="1"></h3>
 <p data-pid="83eQQId6">也就是说 <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_1%7D+%5Cin+%5Cleft%5C%7B+%7B0%2C3%7D+%5Cright%5C%7D%5C%5D" alt="\[{x_1} \in \left\{ {0,3} \right\}\]" eeimg="1"> 或者 <img src="https://www.zhihu.com/equation?tex=%7Bx_2%7D+%5Cin+%5Cleft%5C%7B+0%2C3%5Cright%5C%7D" alt="{x_2} \in \left\{ 0,3\right\}" eeimg="1"></p>
 <p data-pid="zQ7wn1iX">假如 <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_1%7D+%3D+%7Bx_2%7D%5C%5D" alt="\[{x_1} = {x_2}\]" eeimg="1"> ，则显然有 <img src="https://www.zhihu.com/equation?tex=M%3Dm" alt="M=m" eeimg="1"> ，故 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29%5C%5D" alt="\[f\left( x \right)\]" eeimg="1"> 为常值函数</p>
 <p data-pid="znny9O0o">注意 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+0+%5Cright%29+%3D0%5C%5D" alt="\[f\left( 0 \right) =0\]" eeimg="1"> ，所以 <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Cforall+x+%5Cin+%5Cleft%5B+%7B0%2C3%7D+%5Cright%5D%2Cf%5Cleft%28+x+%5Cright%29+%5Cequiv+0%5C%5D" alt="\[\forall x \in \left[ {0,3} \right],f\left( x \right) \equiv 0\]" eeimg="1"></p>
 <p data-pid="Px2N7WUb"><b>以下讨论 </b><img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_1%7D+%5Cne+%7Bx_2%7D%5C%5D" alt="\[{x_1} \ne {x_2}\]" eeimg="1"><b> 的情况，此时有四种情形</b></p>
 <ol>
  <li data-pid="gVlq2Zrp"><img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_1%7D+%3D+3%2C%7Bx_2%7D+%5Cin+%5Cleft%5B+%7B0%2C3%7D+%5Cright%29%5C%5D" alt="\[{x_1} = 3,{x_2} \in \left[ {0,3} \right)\]" eeimg="1"></li>
  <li data-pid="6NBWFFBN"><img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_1%7D+%3D+0%2C%7Bx_2%7D+%5Cin+%5Cleft%28+%7B0%2C3%7D+%5Cright%5D%5C%5D" alt="\[{x_1} = 0,{x_2} \in \left( {0,3} \right]\]" eeimg="1"></li>
  <li data-pid="v6ZERbUA"><img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_2%7D+%3D+0%2C%7Bx_1%7D+%5Cin+%5Cleft%28+%7B0%2C3%7D+%5Cright%5D%5C%5D" alt="\[{x_2} = 0,{x_1} \in \left( {0,3} \right]\]" eeimg="1"></li>
  <li data-pid="NjSf-dqt"><img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_2%7D+%3D+3%2C%7Bx_1%7D+%5Cin+%5Cleft%5B+%7B0%2C3%7D+%5Cright%29%5C%5D" alt="\[{x_2} = 3,{x_1} \in \left[ {0,3} \right)\]" eeimg="1"></li>
 </ol>
 <p data-pid="r6bF6a6K">显然这四种情形只需要证明一种，剩下的同理即可</p>
 <p data-pid="T9nYMkNe"><b>这里我们证明 1. 的情形</b></p>
 <p data-pid="5SilxIc9">则有 <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_1%7D+%3D+3%5C%5D" alt="\[{x_1} = 3\]" eeimg="1"> ，那么 <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_2%7D+%3D+0%5C%5D" alt="\[{x_2} = 0\]" eeimg="1"> 或者 <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_2%7D+%5Cin+%5Cleft%28+%7B0%2C3%7D+%5Cright%29%5C%5D" alt="\[{x_2} \in \left( {0,3} \right)\]" eeimg="1"></p>
 <ul>
  <li data-pid="z1WKznKM">如果 <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_2%7D+%3D+0%5C%5D" alt="\[{x_2} = 0\]" eeimg="1"> ，则有 <img src="https://www.zhihu.com/equation?tex=%5C%5Bm+%3D+f%5Cleft%28+%7B%7Bx_2%7D%7D+%5Cright%29+%3D+f%5Cleft%28+0+%5Cright%29+%3D+0%5C%5D" alt="\[m = f\left( {{x_2}} \right) = f\left( 0 \right) = 0\]" eeimg="1"></li>
 </ul>
 <p data-pid="UEhx9lud">此时必有 <img src="https://www.zhihu.com/equation?tex=%5C%5BM+%3D+0%5C%5D" alt="\[M = 0\]" eeimg="1"> ，<b>否则</b> <img src="https://www.zhihu.com/equation?tex=%5C%5BM+%3D+f%5Cleft%28+%7B%7Bx_1%7D%7D+%5Cright%29+%3D+f%5Cleft%28+3+%5Cright%29+%3E+0%5C%5D" alt="\[M = f\left( {{x_1}} \right) = f\left( 3 \right) > 0\]" eeimg="1"></p>
 <p data-pid="VL2va8Q7">因为 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29%5C%5D" alt="\[f\left( x \right)\]" eeimg="1"> 在 <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Cleft%5B+%7B0%2C3%7D+%5Cright%5D%5C%5D" alt="\[\left[ {0,3} \right]\]" eeimg="1"> 连续，在 <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Cleft%28+%7B0%2C3%7D+%5Cright%29%5C%5D" alt="\[\left( {0,3} \right)\]" eeimg="1"> 可导</p>
 <p data-pid="rKB9X3Zn">故由<b>拉格朗日中值定理</b>得 <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Cexists+%5Cxi++%5Cin+%5Cleft%28+%7B0%2C3%7D+%5Cright%29%5C%5D" alt="\[\exists \xi  \in \left( {0,3} \right)\]" eeimg="1"> 使得 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%27%5Cleft%28+%5Cxi++%5Cright%29+%3D+%5Cfrac%7B%7Bf%5Cleft%28+3+%5Cright%29+-+f%5Cleft%28+0+%5Cright%29%7D%7D%7B%7B3+-+0%7D%7D+%3D+%5Cfrac%7BM%7D%7B3%7D+%3E+0%5C%5D" alt="\[f'\left( \xi  \right) = \frac{{f\left( 3 \right) - f\left( 0 \right)}}{{3 - 0}} = \frac{M}{3} > 0\]" eeimg="1"></p>
 <p data-pid="KlLphtnV">另一方面， <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%27%5Cleft%28+%5Cxi++%5Cright%29+%3D++-+%5Cfrac%7B%7Bf%5Cleft%28+%5Cxi+%5Cright%29%7D%7D%7B%7B%5Csqrt+%5Cxi+%5Csqrt+%7B4%5Cxi%7D+%7D%7D%5C%5D" alt="\[f'\left( \xi  \right) =  - \frac{{f\left( \xi \right)}}{{\sqrt \xi \sqrt {4\xi} }}\]" eeimg="1"> ，因此 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+%5Cxi++%5Cright%29+%3C+0%5C%5D" alt="\[f\left( \xi  \right) < 0\]" eeimg="1"></p>
 <p data-pid="Qms4HpEm">这与 <img src="https://www.zhihu.com/equation?tex=m%3D0" alt="m=0" eeimg="1"> 矛盾（注意 <img src="https://www.zhihu.com/equation?tex=m" alt="m" eeimg="1"> 是 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29%5C%5D" alt="\[f\left( x \right)\]" eeimg="1"> 的最小值）</p>
 <p data-pid="xdUPYw8R">因此 <img src="https://www.zhihu.com/equation?tex=%5C%5BM+%3D+m+%3D+0%5C%5D" alt="\[M = m = 0\]" eeimg="1"> ，即 <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Cforall+x+%5Cin+%5Cleft%5B+%7B0%2C3%7D+%5Cright%5D%2Cf%5Cleft%28+x+%5Cright%29+%5Cequiv+0%5C%5D" alt="\[\forall x \in \left[ {0,3} \right],f\left( x \right) \equiv 0\]" eeimg="1"></p>
 <ul>
  <li data-pid="sH7Xdgtz">如果 <img src="https://www.zhihu.com/equation?tex=%5C%5B%7Bx_2%7D+%5Cin+%5Cleft%28+%7B0%2C3%7D+%5Cright%29%5C%5D" alt="\[{x_2} \in \left( {0,3} \right)\]" eeimg="1"> ，则 <img src="https://www.zhihu.com/equation?tex=%7Bx_2%7D" alt="{x_2}" eeimg="1"> 是 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29%5C%5D" alt="\[f\left( x \right)\]" eeimg="1"> 的极小值点</li>
 </ul>
 <p data-pid="s_032QFD">根据<b>费马引理</b>可知 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%27%5Cleft%28+%7B%7Bx_2%7D%7D+%5Cright%29+%3D+0%5C%5D" alt="\[f'\left( {{x_2}} \right) = 0\]" eeimg="1"></p>
 <p data-pid="b7mHSnhX">结合 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%27%5Cleft%28+x+%5Cright%29+%3D++-+%5Cfrac%7B%7Bf%5Cleft%28+x+%5Cright%29%7D%7D%7B%7B%5Csqrt+x+%5Csqrt+%7B4x%7D+%7D%7D%5C%5D" alt="\[f'\left( x \right) =  - \frac{{f\left( x \right)}}{{\sqrt x \sqrt {4x} }}\]" eeimg="1"> 可知 <img src="https://www.zhihu.com/equation?tex=%5C%5Bm+%3D+f%5Cleft%28+%7B%7Bx_2%7D%7D+%5Cright%29+%3D+0%5C%5D" alt="\[m = f\left( {{x_2}} \right) = 0\]" eeimg="1"></p>
 <p data-pid="IOUPyLsj">此时同前面的讨论（应用<b>拉格朗日中值定理</b>）可得 <img src="https://www.zhihu.com/equation?tex=M%3D0" alt="M=0" eeimg="1"></p>
 <p data-pid="SH7vWK7W">因此 <img src="https://www.zhihu.com/equation?tex=%5C%5BM+%3D+m+%3D+0%5C%5D" alt="\[M = m = 0\]" eeimg="1"> ，即 <img src="https://www.zhihu.com/equation?tex=%5C%5B%5Cforall+x+%5Cin+%5Cleft%5B+%7B0%2C3%7D+%5Cright%5D%2Cf%5Cleft%28+x+%5Cright%29+%5Cequiv+0%5C%5D" alt="\[\forall x \in \left[ {0,3} \right],f\left( x \right) \equiv 0\]" eeimg="1"></p>
 <p data-pid="l8AxHwXD"><b>此外，另外三种情形也可以同理证明</b></p>
 <p data-pid="AMAg0FKN">综上，总有 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+x+%5Cright%29+%5Cequiv+0%5C%5D" alt="\[f\left( x \right) \equiv 0\]" eeimg="1"> 成立，故 <img src="https://www.zhihu.com/equation?tex=%5C%5Bf%5Cleft%28+2+%5Cright%29+%3D+%5Csqrt+8++-+2%5Csqrt+2++%3D+0%5C%5D" alt="\[f\left( 2 \right) = \sqrt 8  - 2\sqrt 2  = 0\]" eeimg="1"></p>
 <p data-pid="zaTmZ6RU"><b>证毕！</b></p>
