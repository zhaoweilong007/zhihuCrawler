# 海洋里到底有什么可怕的东西？
- 点赞数：5302
- 更新时间：2018年04月27日10时27分39秒
- 回答url：https://www.zhihu.com/question/271550823/answer/364317604
<body>
 <ul>
  <li data-pid="A5q1-pLy">令潜水员心慌意乱的超声波；</li>
  <li data-pid="oy0btzuG">4-8摄氏度的暗流，会诱发潜水员大面积肌肉抽筋；</li>
  <li data-pid="F5KVfL14">海底火山岩附近的高硫质海水（能够把潜水服腐蚀出洞来）；</li>
  <li data-pid="MBlQt-7g">海底火山岩附近上浮的有毒气体可以立刻烧伤人的呼吸系统；</li>
  <li data-pid="_iIa_U6A">无声无迹尾随你潜水的人鱼或其它海怪；</li>
  <li data-pid="eKn8z7kW">各类海兽的侵害和病菌感染；</li>
  <li data-pid="v_xZjl8d">质量低劣的呼吸混合气体（充罐的时候就是不能省钱！）；</li>
  <li data-pid="KbbJl_iB">锋利无比的礁岩；</li>
  <li data-pid="a7d5l88x">割你呼吸管子的同伴；</li>
 </ul>
