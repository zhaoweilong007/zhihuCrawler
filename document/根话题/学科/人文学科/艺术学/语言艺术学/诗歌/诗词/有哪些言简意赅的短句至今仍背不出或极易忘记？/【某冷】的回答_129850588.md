# 有哪些言简意赅的短句至今仍背不出或极易忘记？
- 点赞数：10574
- 更新时间：2016年11月05日06时40分25秒
- 回答url：https://www.zhihu.com/question/52286424/answer/129850588
<body>
 <p data-pid="euBvffcO">sin α + sin β = 2sin [(α + β) / 2] cos [(α - β) / 2]</p>
 <p data-pid="SMzjHl8w">sin α - sin β = 2cos [(α + β) / 2] sin [(α - β) / 2]</p>
 <p data-pid="cmzgyOKd">cos α + cos β = 2cos [(α + β) / 2] cos [(α - β) / 2]</p>
 <p data-pid="jaLMnS9t">cos α - cos β = -2sin [(α + β) / 2] sin [(α - β) / 2]</p>
