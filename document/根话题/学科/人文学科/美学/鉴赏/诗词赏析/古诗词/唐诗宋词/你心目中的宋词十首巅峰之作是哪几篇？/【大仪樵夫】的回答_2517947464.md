# 你心目中的宋词十首巅峰之作是哪几篇？
- 点赞数：7965
- 更新时间：2022年11月05日01时18分09秒
- 回答url：https://www.zhihu.com/question/495059487/answer/2517947464
<body>
 <p data-pid="NoxzmHH6">一首敬过往，一首敬现实；</p>
 <p data-pid="zXfMH2ck">一首敬风骨，一首敬袍泽；</p>
 <p data-pid="hZSynyti">一诗敬前任，一首敬糠妻；</p>
 <p data-pid="ze_BYq9O">一首敬相逢，一首敬别离；</p>
 <p data-pid="MrS4cLHD">一首敬家国天下，一首敬田园成诗。</p>
 <p data-pid="c8gAbzzD">豪放派没有伤春悲秋，没有无病呻吟，只有家国天下。个人领悟，贻笑大方。</p>
 <hr>
 <h2><b>一首敬过往：</b></h2>
 <ul>
  <li data-pid="m0Awc7ZJ">渔家傲·秋思 范仲淹</li>
  <li data-pid="YwnrMRbr">塞下秋来风景异，衡阳雁去无留意。四面边声连角起。千嶂里，长烟落日孤城闭。</li>
  <li data-pid="H1Ag5QhD">浊酒一杯家万里，燕然未勒归无计。羌管悠悠霜满地。人不寐，将军白发征夫泪。</li>
 </ul>
 <hr>
 <h2><b>一首敬现实：</b></h2>
 <ul>
  <li data-pid="cPqHstac"><b>江城子·密州出猎 苏轼</b></li>
  <li data-pid="ls7gf7Vw">老夫聊发少年狂，左牵黄，右擎苍，锦帽貂裘，千骑卷平冈。为报倾城随太守，亲射虎，看孙郎。</li>
  <li data-pid="n5hu9Dqb">酒酣胸胆尚开张。鬓微霜，又何妨！持节云中，何日遣冯唐？会挽雕弓如满月，西北望，射天狼。</li>
 </ul>
 <hr>
 <h2><b>一首敬风骨：</b></h2>
 <ul>
  <li data-pid="KIpN-UFi"><span class="nolink">蝶恋花·春景 </span> <span class="nolink">苏轼</span></li>
  <li data-pid="KGNIWMvR">花褪残红青杏小，燕子飞时，绿水人家绕。</li>
  <li data-pid="ZV3NPgDG">枝上柳绵吹又少。天涯何处无芳草。</li>
  <li data-pid="vzaVY54j">墙里秋千墙外道。墙外行人，墙里佳人笑。</li>
  <li data-pid="IfCnPKMF">笑渐不闻声渐悄。多情却被无情恼。</li>
  <li data-pid="LP82PqY1">（豪放派描写忠肝义胆/节操风骨的好词很多，但是很明显苏大胡子在这首词里已经超脱了。官场就是个围城，外面的人想进来，里面的人想出去。皇帝不急太监急，人微言轻的苦楚，化作一句“多情反被无情恼”）</li>
 </ul>
 <hr>
 <h2><b>一首敬袍泽：</b></h2>
 <ul>
  <li data-pid="cWQOcQcK">破阵子·为陈同甫赋壮词以寄之 辛弃疾</li>
  <li data-pid="BtfkX19V">醉里挑灯看剑，梦回吹角连营。八百里分麾下炙，五十弦翻塞外声，沙场秋点兵。</li>
  <li data-pid="3Ci77PoT">马作的卢飞快，弓如霹雳弦惊。了却君王天下事，赢得生前身后名。可怜白发生！</li>
 </ul>
 <hr>
 <h2><b>一首敬前任： </b></h2>
 <ul>
  <li data-pid="jNmTkRbT">江城子·乙卯正月二十日夜记梦 苏轼</li>
  <li data-pid="Y6auUK63">十年生死两茫茫，不思量，自难忘。千里孤坟，无处话凄凉。纵使相逢应不识，尘满面，鬓如霜。</li>
  <li data-pid="pt_dTQa5">夜来幽梦忽还乡，小轩窗，正梳妆。相顾无言，惟有泪千行。料得年年肠断处，明月夜，短松冈。</li>
 </ul>
 <p data-pid="2H5klfNd">更新一下：在从前，像陆游与唐婉、徐志摩与那谁分手后还扯不清的毕竟太少，继续纠缠可是要浸猪笼的。所以，分手跟天人两隔没有什么区别，大多是死生不复相见。用这首致敬前任没毛病吧？</p>
 <hr>
 <h2><b>一首敬糠妻：</b></h2>
 <ul>
  <li data-pid="niFf1_WH"><span class="nolink">少年游·润州作代人寄远</span> 苏轼</li>
  <li data-pid="ht_hp1sA">去年相送，余杭门外，飞雪似杨花。今年春尽，杨花似雪，犹不见还家。<br>
   对酒卷帘邀明月，风露透窗纱。恰似姮娥怜双燕，分明照、画梁斜。</li>
 </ul>
 <hr>
 <h2><b>一首敬相逢：</b></h2>
 <ul>
  <li data-pid="EEpPt2Ik">诉衷情·花前月下暂相逢 张先</li>
  <li data-pid="4UHao_WL">花前月下暂相逢。苦恨阻从容。何况酒醒梦断，花谢月朦胧。</li>
  <li data-pid="cCTrABrf">花不尽，月无穷。两心同。此时愿作，杨柳千丝，绊惹春风。</li>
 </ul>
 <hr>
 <h2><b>一首敬离别：</b></h2>
 <ul>
  <li data-pid="OXLUgrH-">雨霖铃 柳永</li>
  <li data-pid="J79HZ8Kt">寒蝉凄切，对长亭晚，骤雨初歇。都门帐饮无绪，方留恋处，兰舟催发。</li>
  <li data-pid="9jItfzD-">执手相看泪眼，竟无语凝噎。念去去，千里烟波，暮霭沉沉楚天阔。</li>
  <li data-pid="3UknXzZm">多情自古伤离别，更那堪，冷落清秋节。</li>
  <li data-pid="ki9Jug0d">今宵酒醒何处?杨柳岸，晓风残月。</li>
  <li data-pid="L-3OLXET">此去经年，应是良辰好景虚设。便纵有千种风情，待与何人说!</li>
 </ul>
 <hr>
 <h2><b>一首敬家国天下：</b></h2>
 <ul>
  <li data-pid="3B6MMfHx">满江红 岳飞</li>
  <li data-pid="WiLSZrKz">怒发冲冠，凭栏处潇潇雨歇。抬望眼，仰天长啸，壮怀激烈。</li>
  <li data-pid="VM7XKSYd">三十功名尘与土，八千里路云和月。</li>
  <li data-pid="eYPMXFJ2">莫等闲白了少年头，空悲切。</li>
  <li data-pid="RxtrqTHf">靖康耻，犹未雪；</li>
  <li data-pid="6sElGn5m">臣子恨，何时灭！</li>
  <li data-pid="oRcS_kJ_">驾长车踏破贺兰山缺。</li>
  <li data-pid="bb_WTXv6">壮志饥餐胡虏肉，笑谈渴饮匈奴血。</li>
  <li data-pid="5-bUCioA">待从头收拾旧山河，朝天阙。</li>
 </ul>
 <hr>
 <h2><b>一首敬田园成诗：</b></h2>
 <ul>
  <li data-pid="8TFqjQ1m">归字谣 袁去华（江西老表）</li>
  <li data-pid="feETnPNg">归</li>
  <li data-pid="4l5JCgT-">目断吾庐小翠微。</li>
  <li data-pid="XIsO5ob0">斜阳外，白鸟傍山飞。</li>
  <li data-pid="QVjo1u_b">（归字谣是个人最喜欢的宋词之一，大师级留白，笔墨精炼，意境深远）</li>
 </ul>
