# 如何判断一个人是否喜欢你？
- 点赞数：6117
- 更新时间：2019年04月09日15时42分42秒
- 回答url：https://www.zhihu.com/question/27966721/answer/240118474
<body>
 <p data-pid="2_fVM1hg">作为“好人卡”收集大神，总结了一套<b>绝对简单快速判断一个人（特别是女生）不喜欢你</b>的方法：</p>
 <ol>
  <li data-pid="2ENn6F0i">不主动给你发信息 = 不喜欢你</li>
  <li data-pid="UDiw5W1B">聊天过程中，对你的问题遮掩、敷衍、回避 = 不喜欢你</li>
  <li data-pid="d74G3_53">对你发的信息长时间（1小时以上）不回复，并且再次聊天时也没有任何解释 = 不喜欢你</li>
  <li data-pid="LfYkZZDQ">你评论她朋友圈，她不回复 = 不喜欢你</li>
  <li data-pid="LyTLtYNN">邀请约见，表示拒绝（近乎任何理由的拒绝）并且不提下次具体约见时间（类似于“以后有空再约见吧”） = 不喜欢你</li>
  <li data-pid="UgtWaBlM">如果约见</li>
  <ol>
   <li data-pid="brwjyld5">两个人一起走路，她走得很快，或者你想驻足拍一张照片，她还是继续往前走 = 不喜欢你</li>
   <li data-pid="-T8eKCL5">约见后，两人分开后，到家不告知（比如“我到家啦，你到了吗？”）= 不喜欢你</li>
   <li data-pid="QsJMyRDu">约见后，不主动给你发信息 = 不喜欢你</li>
  </ol>
  <li data-pid="daGIMS9n">如果你们一起出去玩，你给她拍完照片，她觉得拍得挺不错，p完但是没发朋友圈和微博等社交软件 = 不喜欢你</li>
  <li data-pid="xySszUIR">如果刚开始认识的时候，聊天她透露出来的意思是“家里人催的着急” “自己挺着急”…但是聊了一段时间她又和你说“现在的自己也不着急”。这说明她对你不感兴趣。</li>
 </ol>
 <p data-pid="mK9SwsD8">以上几点，凡是符合任意一点，就可以判定这个女生不不不不不不不喜欢你！</p>
 <p data-pid="QgCMMEvd">相反，如果一个女生喜欢你，那肯定就像你喜欢一个人的状态一样</p>
 <ol>
  <li data-pid="GEB36o88">经常主动发信息</li>
  <li data-pid="Jv9uZLAZ">平常的琐事，“现在小区里全是桂花的香味”，“今天在图书馆遇到了个神经病”，“下雨了，没带伞，湿了一身”，包括一些细节都想分享给你</li>
  <li data-pid="g0mun20o">尽可能及时的回复你信息，而且经常顺带说出一大串。</li>
  <li data-pid="Rm19I3BH">会找时间约见你，或对于你的约见近乎全力以赴，不找借口。她会洗头，甚至是专门去剪头发；化妆，好好打扮一番.....</li>
  <li data-pid="NRPTVt8d">约见的时候，</li>
  <ol>
   <li data-pid="PrLFhqt5">会喊你走慢点“哎呀，你走路好快呀...”</li>
   <li data-pid="AB09zQn_">很善解人意，说你挑的地方吃饭太贵（就是想为你省钱），会主动提出“我们去吃XX吧？”</li>
   <li data-pid="ioVbFqGo">饭后不会主动要求回家，常常是“哎呀，吃的好饱啊，我们先走一走吧”</li>
   <li data-pid="Cj6MupFF">分开后，会主动给你发信息，告知到家等等。</li>
  </ol>
 </ol>
 <p class="ztext-empty-paragraph"><br></p>
