# 为什么一部分人明知表白成功率不高，失败了都当不成朋友，还要去表白？
- 点赞数：5329
- 更新时间：2015年07月23日01时12分13秒
- 回答url：https://www.zhihu.com/question/23178532/answer/56070470
<body>
 <p data-pid="21sa7ILF">其实表白是可以多赢的：</p>
 <ul>
  <li data-pid="0lMzg-L7">表白让对方得到虚荣心的满足——赢！<br></li>
  <li data-pid="AzOLc3LJ">表白节约了彼此的时间——赢！<br></li>
  <li data-pid="O7I8nW0T">表白锻炼了口头表达能力——赢！<br></li>
  <li data-pid="M8Hhhg3C">表白时，心跳加快，有益健康——赢！<br></li>
  <li data-pid="eYEF4boV">表白失败，心如死灰，心情跌入谷底，人生的九九八十一劫，又过了一道坎——赢！<br></li>
  <li data-pid="qJ0xxRNe">少不经事的你好像又多懂了一些什么——赢！<br></li>
  <li data-pid="f3a_GiQi">当别人都在当键盘侠时你已经行动了——赢！<br></li>
  <li data-pid="WQumh4Tv">表白后对方对待你的方式，让你领悟，原来你爱得值得，或不值得——赢！<br></li>
  <li data-pid="cZeaolIm">失败当然能让你又认清一些事实，比如原来是你自作多情，或者原来你也没那么帅嘛——赢！<br></li>
  <li data-pid="IKp006Nb">表白万一成功了呢——真赢！</li>
 </ul>
