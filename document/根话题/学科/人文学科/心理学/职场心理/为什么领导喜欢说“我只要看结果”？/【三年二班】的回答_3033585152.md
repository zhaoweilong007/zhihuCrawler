# 为什么领导喜欢说“我只要看结果”？
- 点赞数：12664
- 更新时间：2023年05月18日14时25分47秒
- 回答url：https://www.zhihu.com/question/593292093/answer/3033585152
<body>
 <p data-pid="Y2OIYoZt">正常是这样的：</p>
 <ol>
  <li data-pid="Wsl-pC-T">大领导需要结果</li>
  <li data-pid="9dWWG6im">中层领导制定策略，获得大领导同意</li>
  <li data-pid="WWP8IIlP">基层领导制定计划，获得中层领导同意</li>
  <li data-pid="1kUJuOk1">工程师负责具体方案，获得基层领导同意</li>
  <li data-pid="RWvhbA7i">找资源，按照方案开展项目</li>
 </ol>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="HW9NZ7Sv">但是在一些奇葩单位，就变成了这样</p>
 <ol>
  <li data-pid="mWe317Jz">大领导需要结果</li>
  <li data-pid="amd6jL2z">中层领导负责传话</li>
  <li data-pid="2AiSiSUr">基层领导负责传话</li>
  <li data-pid="JSXkMMsg">工程师一脸懵逼，求助无方，承担所有责任。</li>
 </ol>
</body>
