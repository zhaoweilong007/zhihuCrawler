# 腾讯究竟强大在哪？是否被高估？
- 点赞数：5080
- 更新时间：2015年07月31日20时08分29秒
- 回答url：https://www.zhihu.com/question/32063844/answer/57242367
<body>
 <p data-pid="yRaaKEfL">随便说几个：</p>
 <ul>
  <li data-pid="dqTzE5QQ">每到特殊的节日，有定制的礼物送给每一个员工，像月饼这样的礼物，可以选择公司帮你寄给父母；</li>
  <li data-pid="T1oLxLs4">各路班车基本覆盖全市，准点到达，根本不知道地铁和公交;</li>
  <li data-pid="L38bJrgK">那年北京暴雨，淹死人，不只是提前下班，而且班车上都给每一个人准备好食物和饮料；</li>
  <li data-pid="_f-Z1jPk">公司茶水间里有各种饮料，随便喝；</li>
  <li data-pid="SGKl9WyC">卫生间一尘不染，会有公司内部新闻刊物；</li>
  <li data-pid="kRr2M_Vb">会议室都支持视频会议，腾讯多地远程办公非常方便；</li>
  <li data-pid="Xvo11FT4">RTX用作内部沟通，很方便，组织架构清晰，每个人都有专用电话，很多人更愿意电话沟通；</li>
  <li data-pid="Z_SuncIe">加班可以调休，公司规定上班不能连续超过7天，遇到之前假期调休时，第8天上班的话公司会放假；</li>
  <li data-pid="0-N4Qv0p">出差住宿标准还是不低的；</li>
  <li data-pid="eiyqrs67">过年回公司第一天，很多公司发开门红红包，作为早早收假来上班员工的奖励。但腾讯这一天本身就是放假，所有人可以向所有管理层索要红包，这一天只能用来拿红包的。虽然可以说是小恩小惠，但这个传统，营造的节日气氛，值得很多公司借鉴。</li>
  <li data-pid="AEKL34xh">年前会封版，系统，软件都不再更新了，为了让大家放松，不用在节日里遇到紧张情况。</li>
 </ul>
 <p data-pid="1iCZxYjh">听起来可能都是平常的事情，但可以说很少有公司能做的到</p>
