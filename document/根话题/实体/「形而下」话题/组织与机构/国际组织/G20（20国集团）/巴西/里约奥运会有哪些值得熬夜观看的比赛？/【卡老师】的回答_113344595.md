# 里约奥运会有哪些值得熬夜观看的比赛？
- 点赞数：7614
- 更新时间：2016年08月10日11时36分49秒

<body>
 <p data-pid="_h2MsYGI">没想到一口气写了这么多，做了一点微小的观赛指南，希望可以帮助到大家。</p>
 <p data-pid="cQhMmYXg"><b>本文已经授权《外滩画报》微信公众号the-bund 进行编辑发布</b></p>
 <p data-pid="V6zVjpWn">首先，最值得看的当然是本届奥运会的<b>开幕式</b>。（<b>8月6日7时</b>）</p>
 <p data-pid="n39qHR9L">本届奥运会并没有像世界杯一样为了迎合欧洲观众而把开幕式安排在东八区的凌晨，只要早睡早起，中国的观众观看开幕式的难度还是不大的。据说这届奥运会史无前例地设置了<b>两个圣火台</b>，而且点燃圣火的方式也非常有意思。根据往届奥运会的惯例，<b>6时30分</b>开始可能会有接近一小时的文艺表演，大概8点开始会进行<b>运动员入场仪式</b>，击剑奥运冠军<b>雷声</b>将担任中国代表团的旗手。点燃圣火大概会发生在北京时间的<b>11时</b>左右。按照今年央视的安排，<b>CCTV1、5、7</b>还有<b>新闻频道</b>都会转播开幕式，而且很可能会沿用伦敦奥运的模式，每个频道安排不同的解说（C5张斌+沙桐，C13白岩松压阵），观众可以自行选择。（CCTV5凌晨5时起开始预热直播，CCTV5app将提供直升机航拍直播）</p>
 <p data-pid="AhD9pzGE">接下来讲讲正赛。（具体比赛时间与转播安排请以官方及CCTV为准，以下赛事均可通过CCTV5APP收看）</p>
 <p data-pid="oW08lzCa">开幕前 关键词：女足，<b>内马尔</b></p>
 <ul>
  <li data-pid="7R-r6CzD">8月4日<b>凌晨3点</b>，中国女足将与东道主巴西女足进行中国代表团本届奥运会的第一场比赛，虽然胜算不大，不过作为女足重返奥运舞台的第一场比赛，有兴趣的球迷还是可以看一看的。</li>
  <li data-pid="OnNZMO_0">8月5日<b>凌晨3点</b>，<b>内马尔</b>率领的巴西国奥队将踏上冲击金牌的道路，他们的对手是南非国奥队。巴西足球能否一扫美洲杯小组赛出局的奇耻大辱，实现奥运男足金牌零的突破，一切都要从这场揭幕战开始。5</li>
 </ul>
 <p data-pid="P8KcjlpY">第一比赛日（国内8月6日晚上-7日上午） 关键词：首金，中美之战，<b>孙杨</b></p>
 <ul>
  <li data-pid="xLNMWIbZ">大约<b>21时30分</b>起，女子10米气步枪将决出本届奥运会第一枚金牌。奥运冠军杜丽、易思玲都将有机会成为历史上第一位拿下两届奥运会首金的运动员（杜丽是雅典首金得主，易思玲则在伦敦夺下首金）（<b>CCTV5</b>现场直播，沙桐解说）</li>
  <li data-pid="QNC6PuhH">北京时间<b>22时35分</b>，三大球里唯一的夺金热门<b>中国女排</b>将会对阵小组赛首个对手荷兰队，不出意外胜算是很大的。值得一提的是在这场比赛之前的<b>20时30分</b>，日本女排和韩国女排会先上演一场东亚德比，排球迷有兴趣的可以关注。</li>
  <li data-pid="oe2x6DWe">看完两位女射手和女排姑娘的比赛之后没多久，<b>中国射箭队</b>可能就要创造历史了。北京时间大约<b>3时</b>，中国男子射箭队将冲击男子团体比赛的奖牌。在仁川亚运会，男团的小伙子们击败了八连金的东道主韩国，拿下队史第一枚亚运团体金牌。在里约奥运的测试赛上，男团也顺利获得冠军，这一次里约之旅，值得我们期待。（<b>CCTV1</b>现场直播）</li>
  <li data-pid="t3FnUDot">紧接着进行的是男子10米气手枪的比赛，中国选手庞伟、蒲琪峰都有希望摘金，冠军诞生可能会在北京时间7日<b>3时前后</b>，如果还没睡着还是值得看一看的。看完这场赶紧补觉，因为天一亮就有好戏要开始了。</li>
  <li data-pid="--AGaoiI">7日<b>6时</b>，亚洲冠军<b>中国男篮</b>就要与众星云集的<b>美国</b>梦之队一起揭开男篮小组赛的序幕，不需要太多介绍，这是本届奥运会绝对不能错过的比赛之一。（<b>CCTV1</b>现场直播）</li>
  <li data-pid="4iwAdCR5">7日<b>6时</b>还有一场<b>女子48公斤级</b>的举重赛，中国队临时撤换掉了选手，对举重和篮球不是很感兴趣的朋友大可以继续补觉。因为大概在北京时间<b>9时</b>，<b>孙杨</b>就要出场冲击<b>男子400米自由泳</b>的金牌了。孙杨我就不做太多介绍了，这场比赛想必也是7日关注度最高的游泳比赛。而在孙杨比赛之前，<b>叶诗文</b>也会挑战女子400米混合泳，小叶子能否一扫颓势证明自己也是大家关注的焦点。（<b>CCTV5</b>直播）</li>
 </ul>
 <p data-pid="qJnHcr6F">第二比赛日（国内8月7日晚上-8日上午）关键词：吴敏霞</p>
 <ul>
  <li data-pid="ziw7Uuek">7日晚上大约<b>22时30分</b>，郭文珺将再度冲击<b>女子10米气手枪</b>的金牌。不过同时进行的<b>马术</b>比赛当中，中国小帅哥<b>华天</b>虽然不太可能创造历史夺得奖牌，不过他的表现也是值得大家关注的。</li>
  <li data-pid="_7yKtsGT">8日晚上11时，<b>游泳</b>会进行多项预赛，包括孙杨在内中国队许多明星选手可能都会参加，大家最好focus on央视的各项报道，说不定就有自己喜欢的运动员出场。</li>
  <li data-pid="pLakEbot">凌晨<b>2时</b>，老将吴敏霞将和搭档施廷懋一起冲击女子双人3米板的金牌，这是一场没有悬念的比赛，喜欢跳水的朋友还是值得熬夜欣赏的。对跳水没有太多兴趣的话可以转台看<b>男子花剑</b>的比赛，卫冕冠军<b>雷声</b>也是在凌晨3时出场的。（<b>CCTV1</b>现场直播）</li>
  <li data-pid="ab7P1eSr">凌晨<b>3时30分</b>，体操多项预赛的竞争将会展开，既然都熬到这个点了，体操迷还有睡觉的理由吗？</li>
  <li data-pid="uzTtIIfn">清晨6时，<b>男子56公斤级举重</b>，吴景彪、龙清泉都有夺金希望，早起的朋友还是可以看一看的。</li>
 </ul>
 <p data-pid="GEgX1-Pd">第三比赛日（国内8日晚上-9日上午）关键词：<b>射击神童</b>，体操男团</p>
 <ul>
  <li data-pid="V5E6lm60">8日<b>23时</b>，<b>曹逸飞</b>、<b>杨浩然</b>将一起冲击男子10米气步枪的冠军。96年出生的射击神童杨浩然过去两年取得了许多运动员一生都无法取得的成绩，仅仅20岁就已经拿到了奥运会以外所有能拿的大赛冠军（全运会、青奥会、亚运会、世界杯、世锦赛），只要他夺得金牌，就会实现个人的<b>大满贯</b>。老将曹逸飞成绩一直不差，只是每逢大赛就很难有太好的表现，这届奥运会有可能成为他证明自己的舞台。</li>
  <li data-pid="FimI9Vjx">9日<b>2时</b>，男子跳水的第一块金牌将在双人十米台的赛场上诞生，世锦赛冠军林跃、陈艾森这对搭档毫无疑问是最大热门，不过同时进行的还有一场重头戏，就是<b>体操男团</b>的决赛，作为中国代表团体操项目的夺金重点，男团有望实现三连冠的伟业。比赛大概会持续到北京时间6时。</li>
  <li data-pid="Sr3Y1b1G">9日<b>4时30分</b>，重返奥运大家庭的<b>橄榄球</b>将会决出女子金牌。弹丸小国<b>斐济</b>橄榄球实力一直不俗，是次有希望在这个项目上夺下本国在奥运会上的第一面金牌。</li>
  <li data-pid="1EMRnKAG"><b>6时</b>，紧接着男团比赛进行的会是<b>男子举重62公斤级</b>的赛事。小将<b>谌利军</b>去年在世锦赛上霸气打破世界纪录，来到奥运会的赛场，除了夺得金牌，相信小谌还有其他更大的野心。</li>
  <li data-pid="Bt8eMVfM"><b>9时</b>，孙杨将出战<b>200米自由泳</b>，<b>傅园慧</b>则要出征<b>100米仰泳</b>，两项赛事都有比较大的夺金希望，而且时间合适，很适合大家收看。</li>
 </ul>
 <p data-pid="5CFW35II">第四比赛日（国内9日晚上-10日上午）关键词：大力士，<b>菲尔普斯</b></p>
 <ul>
  <li data-pid="IVp6gXWp">今天是中国的传统节日<b>七夕节</b>，晚间黄金时段也没有什么值得关注的比赛，大家可以陪自己的爱人一起补补觉，准备开始凌晨的观赛之旅。</li>
  <li data-pid="mqa2KXYd">10日凌晨<b>2时</b>，<b>陈若琳、刘蕙瑕</b>的<b>跳水双人10米台</b>还是很具有观赏性的，虽然没有太多的悬念，不过还是值得看一看。</li>
  <li data-pid="zITy_4I5"><b>2时40分</b>，去年与今年都打破了世界纪录的女大力士<b>邓薇</b>就要出场争夺女子63公斤级的冠军了。<b>6时</b>，与她的福建老乡、雅典奥运冠军石智勇<b>同名</b>的小将<b>石智勇</b>也要再次挑战男子69公斤级的比赛。</li>
  <li data-pid="_IjAZXnI"><b>3时</b>，<b>体操女团</b>的中国小花们也要为奖牌而战了。如今的女团虽然没有了以往的统治力，不过依然是奖牌的有力竞争者，这次奥运会，姑娘们也有希望为我们带来惊喜。</li>
  <li data-pid="56yWtr7Y"><b>9时</b>，泳坛巨星<b>菲尔普斯</b>就要出场了。菲鱼有希望把自己的奥运金牌数再往上提高三枚，至于能不能如愿以偿就要看他是不是宝刀未老了。</li>
 </ul>
 <p data-pid="dHKRFZSM">第五比赛日（国内10日晚上-11日上午）关键词：赛艇，谁是一姐，<b>宁泽涛</b></p>
 <ul>
  <li data-pid="LcyFA2P9">10日晚上<b>18点30分</b>，新闻联播开始之前，<b>男子与女子赛艇</b>就要十分密集地决出多块金牌了。中国队一共参加了七个项目，至少有2个项目具备冲击金牌的实力，晚饭后的黄金时间，这场好戏不容错过。<b>20时</b>中国女排对波多黎各的比赛也值得期待。</li>
  <li data-pid="sxwD1Kvu">大约北京时间<b>20时</b>，有一场男子50米手枪的射击赛事，看完这场射击赛大家可以补补觉，定好2点的闹钟，因为接下来真的只能用四个字来形容，就是<b>好戏连台</b>。</li>
  <li data-pid="YaVj8mu5">大约北京时间<b>11日2时</b>，<b>秦凯、曹缘</b>有希望摘下男子双人三米板的冠军。几乎与此同时进行的还有<b>男子体操全能赛</b>，日本名将<b>内村航平</b>将受到来自<b>林超攀</b>的挑战，虽然胜算不大，不过依然是值得看的一场比赛。几乎和上面两场比赛同时开始的还有<b>女子69公斤级</b>的举重。饮恨伦敦的<b>向艳梅</b>有望夺下自己的第一枚奥运金牌。</li>
  <li data-pid="FTEUPW9h">这三场比赛结束之后，<b>乒乓球女单决赛</b>会让昏昏欲睡的观众们打起精神。<b>11日8时30分</b>，李晓霞与丁宁有望在决赛的赛场上上演直接对话，相信这也会是一场高水平的对决。</li>
  <li data-pid="eTCsqpFJ">最刺激的恐怕是<b>9时</b>开始的游泳各项决赛。万人瞩目的<b>宁泽涛</b>将在奥运会的赛场上冲击男子100米自由泳奖牌。由于时段不错，这可能会是这届奥运会收视率的一个高点。</li>
 </ul>
 <p data-pid="AdCtyQqi">第六比赛日（国内11日晚上-12日上午）关键词：高尔夫，<b>中华台北</b>，谁是一哥</p>
 <ul>
  <li data-pid="D8yHFhrO">傍晚<b>17时30分</b>，新项目高尔夫就要挥杆打响，除了男选手吴阿顺等人的表现之外，美女球手林希妤也有希望给国人带来惊喜，成为奥运会史上第一位高尔夫女冠军。</li>
  <li data-pid="aeOZk3lI"><b>20时</b>，<b>中国女子重剑队</b>有希望卫冕女子重剑团体冠军。和这场比赛同时开始的还有<b>杜丽</b>与<b>张彬彬</b>参加的<b>女子50米步枪比赛</b>。</li>
  <li data-pid="aSVaCQYD"><b>21时</b>，乒乓球男单半决赛的比赛应该是这个晚上的收视重点，不出意外马龙、张继科应该可以会师决赛。这两场比赛结束之后没多久，<b>皮划艇</b>就要密集地决出金牌了。作为中国水上项目最有可能夺金的一块，还是值得观众等到<b>23时</b>的。</li>
  <li data-pid="QtulM38o">12日<b>凌晨2时</b>，<b>女子射箭</b>个人赛将决出金牌。除了传统强国中韩之外，这届赛事<b>中华台北</b>队也不容忽视，女子射箭更是他们本届奥运会最有可能夺金的项目之一。届时的奖牌争夺很有可能会出现东亚德比或是海峡德比的盛况。</li>
  <li data-pid="VVyjMTuL">12日凌晨<b>3时</b><b>女子体操全能</b>，“黑珍珠”<b>拜尔斯</b>的卫冕之路没有太大的阻碍，来自中国的商春松，来自俄罗斯的图特哈良，还有英国新人唐尼，她们将为了奖牌而战。</li>
  <li data-pid="Lef00eE-">12日早晨<b>8时30分</b>，<b>马龙</b>与<b>张继科</b>很有可能在决赛赛场狭路相逢，对于乒乓迷来说绝对是一场不容错过的对决。</li>
 </ul>
 <p data-pid="WdVusGqL">第七比赛日（国内12日晚上-13日上午）关键词：蹦床，竞走，傅园慧</p>
 <ul>
  <li data-pid="EWvB0glx">12日晚上<b>20时30分</b>，<b>中国女排对阵塞尔维亚</b>，这一场比赛很有可能左右女排淘汰赛的形势。而从今天开始，第一大项田径的争夺也要拉开序幕了。女子10000米长跑将会决出本届奥运会第一块田径金牌。而在<b>晚间23时</b>，网球方面的<b>男子单打半决赛</b>与<b>双打决赛</b>也是网球迷不错的收看选择。</li>
  <li data-pid="0LHHy12d">13日<b>凌晨2时35分</b>，两位蹦床女将<b>何雯娜</b>、<b>李丹</b>就要上阵争夺冠军了。蹦床公主何雯娜作为北京奥运会冠军，在折戟伦敦之后，此番也将竭尽全力证明自己。与此同时进行的还有<b>男子20公里竞走决赛</b>，这可以说是中国田径队本届奥运会最有可能拿下的一块金牌，<b>陈定</b>与队友<b>王震</b>都将为冠军而来，陈定希望可以实现卫冕，而王震也在渴望属于他的第一块奥运金牌。</li>
  <li data-pid="UMSYsEB-">13日<b>凌晨3时30分</b>，<b>于颂</b>担当着延续中国在柔道大级别上的优势的任务，冲击女子78公斤级以上的金牌。一小时之后，<b>男子花剑</b>的四位剑客就要对花剑男团的金牌发起挑战。</li>
  <li data-pid="YIhFXef0">男子花剑的比赛结束不久之后，<b>男子85公斤级举重</b>的决赛场可能会出现大力士<b>田涛</b>的身影。而13日上午真正的主角可能是<b>傅园慧</b>，在<b>9时</b>她将参加<b>女子200米仰泳</b>金牌的争夺。几乎与此同时，<b>巩立姣</b>要向<b>女子铅球</b>的奖牌发起冲击。</li>
 </ul>
 <p data-pid="mArz-_Vr">第八比赛日（国内13日晚上-14日上午）关键词：中美女排，孙杨</p>
 <ul>
  <li data-pid="qkhcruNp">13日晚上<b>20时30分</b>，<b>男子100米</b>的小组赛就要正式开始竞争。<b>苏炳添</b>、<b>博尔特</b>等焦点选手都将在今晚一一亮相。<b>23时30分</b>则同时会有两场好戏，<b>网球女单</b>的决赛以及<b>男子25米手枪速射</b>决赛，世界纪录保持者<b>李越宏</b>势必要为冠军而战。</li>
