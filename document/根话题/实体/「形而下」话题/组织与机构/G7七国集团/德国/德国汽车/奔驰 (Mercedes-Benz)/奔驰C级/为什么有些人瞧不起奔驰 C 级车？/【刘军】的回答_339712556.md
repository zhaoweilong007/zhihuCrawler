# 为什么有些人瞧不起奔驰 C 级车？
- 点赞数：5998
- 更新时间：2018年04月18日16时49分02秒
- 回答url：https://www.zhihu.com/question/48598757/answer/339712556
<body>
 <p data-pid="1Unsxulu">我有个奥迪S3，基本处于奔驰C在奔驰里的地位或者不如。</p>
 <p data-pid="N2yL6Dua">新车上牌，旁边有个等上牌的A5，车主是个美女，白白的大长腿上套了个过膝靴，紧身短裙都快包不住屁股。正当我偷看人家大腿姑娘走了过来：帅哥，你这S3那家店提的，我当时也想买，全北京没现货啊。见到美女我打开全自动装逼模式：大望路那家店啊，我是他们的大客户，从外地给我调了一台。美女：这样啊，厉害，这是我名片，有时间一起出来玩儿呀。我心里默念：这大长腿是能玩一年，嘴上说：嗯嗯，你加我支付宝吧。</p>
 <p data-pid="v-kmJyxd">有次洗车旁边一个开G63得哥们搭话说：你这是真S3吧？快吗？我回：跟您这大号方盒子百公里加速差不多吧。哥们回：那可以啊，我这车开的不舒服，真想提个S3买菜用，随便刷个电脑四秒出头，谁都不惯着。我回：您内车卸四个轱辘就够买我这个啦，大哥一听高兴的合不拢腿，直接冲洗车小工喊：这奥迪洗车钱记我卡上啊。我们又愉快的互相臭捧了几个回合。</p>
 <p data-pid="_1JoILPC">有次在路边停车，刚要开走，冒出个人来收费，我说：你没有工作证，没有划线，路边也没有收费标志，我不会给你钱，要不你报警说我不给你钱，要不你别堵我车前面不让我走。他让出了车头位置，冲我喊：开A3牛什么牛？有钱买A8啊，你就是穷横！我把车窗摇上，猛踩了脚油门，让他呼吸一下穷横的新鲜尾气。</p>
