# 为什么有些人瞧不起奔驰 C 级车？
- 点赞数：7327
- 更新时间：2018年04月18日22时43分56秒
- 回答url：https://www.zhihu.com/question/48598757/answer/370278414
<body>
 <p data-pid="G4-EWXWd">坐出租车，看旁边的车，我随口说新款奔驰c银灰色还挺漂亮的，司机师傅回一句，奔驰c也能叫奔驰？</p>
 <p data-pid="2Uo5d1n0">我认真的跟司机师傅说，你说咱这地方上班一年收入20万算不算多，司机师傅说那算挺多的了，很多人挣不了这么多。</p>
 <p data-pid="WnURz1oi">我说，不吃不喝一年半才能买辆这车，这也不便宜啊。</p>
 <p data-pid="H9SxZCMX">司机师傅说，就是就是的。</p>
 <p data-pid="tvfJhwkE">很多人说话是从来不以自己为参照物的。</p>
 <p data-pid="9sB2w6vg">比如长得丑的人相亲也有很多外貌协会，因为他们眼里只选择看别人。</p>
 <p data-pid="YCg2vMgy">我大学上的二本师范学院，有次回高中，遇到了以前同班复读的同学，直问我，你们学校是不是很垃圾，我选择了无视，后来听说他复读后考了个大专，遇到这类有底气的傻缺，基本属于人生的概率。</p>
</body>
