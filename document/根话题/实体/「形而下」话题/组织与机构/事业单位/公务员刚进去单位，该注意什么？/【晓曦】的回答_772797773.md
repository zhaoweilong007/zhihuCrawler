# 公务员刚进去单位，该注意什么？
- 点赞数：9522
- 更新时间：2024年04月04日11时33分30秒
- 回答url：https://www.zhihu.com/question/319670351/answer/772797773
<body>
 <ol>
  <li data-pid="2E3biZpv">再好的同事在竞争和利益面前没有朋友，只要面子上客客气气一团和气就是正常。所以，不要刻意费心思搞关系，拉帮结派，现在的你会把自己玩进去。把精力都放在好好工作，提高业务能力，多学习其他技能上，让自己工作能力上立得住，比别人有优势，这是第一位的。让你自己成为无可替代的一个强人，这个时候，你会发现那些短暂脆弱的关系真的挺没用的。当然，平时一定要融入集体好好交流，多交朋友，这并不冲突，并不是说追求优秀就要自我孤立，只是摆在第一位的不是人际关系，而是提高自己能力。</li>
  <li data-pid="-1dkrQ47">做人沉稳不是说刻意地立人设沉默寡言，让别人看不透你，谁都不是傻子，你把自己围得像铁桶一样，日子久了谁看不出来？如果你本来就性格开朗热情。同事（观察后比较靠谱的同事）私下大可以真诚开心地谈天说地，成为生活上的朋友。但谈论的一定是自己家里和日常生活的平凡事，比如育儿经验，如何追妹，家里逼婚相亲，老公不爱干家务，最近新刷了哪部剧……切记不要谈论领导同事和工作上的是非，更不要说自己出轨离婚家里有关系走后门那些对你有负面影响的事。参考第一条，同事之间没朋友，你也无法保证同在一起共事多年，未来你和他就一直很好，他会一直替你保守秘密，更不知道他自己又有哪些可以说你的秘密的朋友。同时，条件好的务必低调，千万别炫富秀关系显摆刺激人。说话前要换位思考，你有孩子了，他不孕不育，你就别在他面前提自己孩子。</li>
  <li data-pid="5xtClIGE">切记同事之间不讨论工作领导同事是非长短。别人说了，你就听着，说一句，嗯！或者，实在骂得狠了，你不得不表态的时候，你就可以说一句，哎呀，听你这么一说他这事确实有点过分，你别生气，不和他一般见识。（但不要自以为巧妙地，帮他的对家说好话，以为自己是中立地劝人，抱怨的人会觉得你是不是在偏心他的对家）。如果你也同是受害者，你就听着别人抱怨，然后附和一下就好，不要被别人煽动当枪使。说话时的准绳就是，不要具体地去点评任何单位的人和事。可以相处真诚地说说家事，说说生活上的小困扰，比如孩子不好带，贷款太贵不好还，人家照样觉得你掏心掏肺。</li>
  <li data-pid="UH4whvQx">公务员群体是个娱乐圈相对的两个极端，踏实做事，不要自己上窜下跳把欲望都写在脸上地去刻意展示。比如，开会了没点你发言，领导问还有谁想说，你不要主动举手扯一堆。但你如果有机会的话，你可以正常申报或者委婉流露出你的特长，最高的境界是，你不主动毛遂自荐也不和人显摆，在工作交流与人相处中自然流露出来，有机会了不用你主动说，让你的同事领导自发自愿地推荐你。如果确实指派你去演讲发言等露脸的事，你要不就不上不说，一旦上了，就一定要充分准备一鸣惊人！</li>
  <li data-pid="LwtPIMtp">谁人背后不说人，只要不实际侵犯你的利益，那些口头是非，一笑而过，该怎么相处就怎么相处，不要最后本来没大事，一笑嘻嘻哈哈就过了，结果自己一屁股坐实了不和的传言，不关乎你实际利益的家长里短都不是事。</li>
  <li data-pid="hLZunVfZ">更不要去坏别人嫉妒别人，心态放平，不要见不得别人好，学会接受不公平。现在的现实是，纯靠自己当个科级干部还有可能，越往上走越不容易，绝对的权势关系面前再优秀也会被碾成渣子。但这并不是让你平庸的借口，你改变不了世界但可以改变自己，靠爹爹也会退休，这辈子你能掌握在手里的只有你自己。心有目标，没错过任何一个机会，自己尽力不后悔不留遗憾就好。</li>
  <li data-pid="vTJt0Gop">当有人一起抱团孤立讨伐某人时，如果这个人和你无冤无仇，千万不要胡乱站队地兴奋地参与其中。可以对被孤立的那个人也敬而远之，但不要落井下石。如果你此时跟风站队结仇，那个被孤立的人会更恨你，因为你俩本无冤无仇，你还去欺负他。三十年河东三十年河西，不求你冒着风险雪中送炭，但也千万别给自己多整个仇人。</li>
  <li data-pid="Bpi4Pgkg">工作上少说多做，与人相处上嘴严真诚。少说多做大家都知道。嘴严真诚是指：领导同事告诉你什么事，不要长舌妇一样地拿来当谈资和人说，你以为很秘密，其实传一传都知道了。领导很喜欢帮他干活还嘴严的人，但不要帮领导干违法违规的事，如果万不得已，你要留好当时的证据自己的日记那些，以防有朝一日。（具体不清楚了，也是听说曾经有人不得已按照领导指示做了不该做的，然后他留了个心眼，事后把卷宗的相关页设了夹层，把真正的情况证据夹在了里面，这个后来救了他。）</li>
  <li data-pid="ItXjKQVj">对同事，不要干那些特别low一眼就看出来的：说坏话，打小报告，互相耍心眼欺负人多干活的那种损人还不利己的事。不涉及晋升利益的情况下，平时愉快相处，有忙帮忙，可以成为生活上的朋友，但莫谈公事。更不做老好人。有学会婉转拒绝，留有余地，保护自己。</li>
  <li data-pid="bE57VqGd">如果不幸你和甲成为了仇家，但你同单位的朋友乙与甲是朋友，不要因此疏远乙，各人交各人的朋友，该咋相处还咋相处，和乙交流相处中忽略不提甲就得了。单位里的朋友，再怎么样都无法成为严格意义上的朋友。</li>
  <li data-pid="2QaN-j2G">不要刻意去装人设，人设崩塌了更可怕！很多东西是装不出来的，年轻人刚工作精神面貌充满朝气，积极热情善良，做好自己就很好。凭本心单纯直率点，比装模作样故作老成要好很多，你单位的领导和老同事们啥看不出来啊。发自内心的自信真诚善良，是最好的社交。但得保护好自己，不是让你做滥好人。</li>
  <li data-pid="mTdtQFKh">很多时候如果由于别人嫉妒，给你造成困扰甚至麻烦了，如果不涉及晋升，只是小人日常作祟。你要自我反思，如果你平时确实挺正常的，守礼谦虚，只是正常地在工作中展现了自己比较不错的能力或者领导表现出了对你的一些器重。多半是你还没完完全全地成为他们望尘莫及的强者。就像人们都嫉妒自己忽然中了彩票的邻居，但从来都觉得王思聪炫富理所应当。你要成为真正工作上缺你不可，各种证你全有，全单位立得住的人。这时，你的同事需要你的帮助，你的领导需要你干活，你还怕自己人际关系不好吗？当你有能力时，依然谦虚，愿意力所能及地帮助同事，大家都看见你能者多劳了，还怕大家不服你吗？</li>
  <li data-pid="k5qPlsLX">如果有小人坏你，背后拉小团体说你坏话，和别人说你的不是。你对于那些和你没冲突，但只是被人挑拨误会你的人，千万不要自己主动不搭理人家，一屁股坐实了那些挑拨，但也无需谄媚讨好或者刻意去解释什么。就当你啥也不知道，啥也没发生，还当好同事那样自然相处。</li>
  <li data-pid="JoV3AVEP">不要一上来就给领导同事送厚礼，升米恩斗米仇，以后你不送同等贵重的人家都会觉得不开心。家里只有钱没有关系的土大款，要小心领导吃上你了。</li>
  <li data-pid="0mAqugcs">和人交代事情前多换位思考一下，如果你是他希望听到的是什么，哪些是他可以接受的。一起共事也是，多换位思考，想想别人为什么这么做，哪些是合理的。但不是要你做活雷锋，不要把心理上的理解原谅和实际做事的大公无私划等号，升米恩斗米仇，做好份内的事，其他的自己把握。同事之间帮忙都是礼尚往来的，不要单方面傻付出。</li>
  <li data-pid="xnHZycn8">江湖人心叵测，世事变化莫测。你可以防卫，可以还击，像甄嬛一样成长。面对利益，大家各凭本事去争取，有关系的拼关系，有能力的拼能力。但你不要无中生有地去主动害人，我的家风永远教育我的是，走正道，才能持久。有关系更要有能力。至于最后的结果，尽力就好。</li>
  <li data-pid="12MsJOf6">看了一些评论，我这些针对的是新入职对自己的未来抱有期待的年轻人，毕竟人活一辈子，不能30岁不到就开始意志消沉地想混日子，前十年八年谁不想努努力看看运气。一个有能力的人看透不争，人家敬重你觉得那是老同志退隐山林人生看淡。一个没能力的人混日子，别人眼里那叫懦弱无能（除非富二代）。当然提升能力可能对你晋升没直接帮助，因为中国关系很重要。但不能因此就不去做了，积累能力不是为了千军万马去挤那一个晋升的独木桥，如果你把目的定得那么狭窄功利就真的走偏了。它是为了让你自己以后有更多可能，让你在拼家世已经落后的情况下不错过每一个机会，为了你自己成为一个更好的人，不虚度年华。你可以根据你的职位去考考司法考试，考考注册会计师……可能不一定哪次遴选选调就设定的就是相关条件，关系虽然占大部分，看知乎别人分享的经历大家也知道，也有靠自己和幸运上位的，你平时那些付出更多的是为了给自己一个交代不后悔。如果你对相关专业没兴趣，你也可以积累其他的，搞搞摄影，搞搞剪辑，写写书法……体制内的生活太闲了，也够你剑走偏锋沉默积累一鸣惊人了。刘慈欣原来是国家电网的，写明朝那些事的作者石悦，现在任山东省政府办公厅综合处处长。但无论怎么样，学习最好低调沉默地去积蓄，不要大张旗鼓让同事领导知道哦～</li>
  <li data-pid="-12dNJzw">当然，如果你早已经看透知足，就想开心快乐地混日子，那你就不用看这些经验了，真的。只要你对他没利益威胁，你当个谦虚的绿叶，总有出头鸟招人记恨。你甚至可以不管什么领导同事关我屁事，我就是去点到拿工资了，公务员是国家雇的，你真看开了爱干不干，三天打鱼两天晒网，最后领导都怕影响不好得哄着你来。我这些经验针对的是，目前对未来还是挺积极的年轻人，不是工作了十年八年，感觉未来不可期的老同志。</li>
 </ol>

 <p data-pid="44WaBW0m">随时更新的分界线</p>
 <p data-pid="TfoOiw1o">1千万不能轻视任何一个同事。那些工作越落后越没存在感的，也许背后不一定是哪个大领导的关系户。有时候靠山越大，人越低调。或者关系越硬，人越没本事。家庭条件差的，虽无法依靠父母，直观上你以为他家庭很一般，但你怎知他的其他亲戚就不行呢？所以，一定不要看表面轻易轻视任何一个人。</p>
 <p data-pid="CNryg-ev">2我感觉金庸在倚天屠龙记里的这句话非常适合套用在工作相处中：<b>他强由他强，清风拂山岗；他横由他横，明月照大江。</b></p>
 <p data-pid="VICI3APg">3.当你和人遇到冲突，或者遇到事情需要向领导或者第三者解释汇报时，在你不知道另一方是如何说的情况下，你可以忽略简化对你不利的，具体描述对你有利的，但给人的感觉一定是在陈述客观事实，让别人自己去判断。千万不要刻意去摘除解释自己，也不要在陈述中带主观色彩指责批评他人。</p>
 <p data-pid="Asp3zP6W">4.昨天看了我和我的祖国，深有感触。针对之前评论里面大家打趣说我说了这么多却没一条是全心全意为人民服务的建议，还有人说我这些做起来装起来很累。在此，我说几句，不想说什么假大空，就实实在在地讲讲人生。古往今来成大事者，就像我和我的祖国里开场的主角林致远，还有后面的女飞行员等等。这样的人内心必然是对自己这辈子有责任感和使命感的，所以在这种追求下，这样的人从内心深处就不会再有自私者的为人做事，压根就不存在装与不装。他们发自内心地对自己的工作一定是无比严谨认真的，对待周围的人也会是比较真诚而善良的（当然不等于傻付出不保护自己）。我们也一样，当你从所谓精致的利己主义里跳出来，真的有了愿望和使命，而且是发自内心的，就算愿望很难实现，但你整个人为人处事工作上都会有一个质的提高。不是让你当老黄牛，当雷锋，当郭明义……是你自己内心要有点事业追求，可以是为自己也可以是为社会，但绝对不只是自己的面子金钱荣誉，如果只是自己那点一亩三分地的利益得失斤斤计较一点责任感和社会追求也没有，你再努力再优秀，最终你的未来一定是有天花板的。这也是为什么当年被那么多人视为榜样的哈佛女孩刘亦婷最后泯然众人矣，她就是精致利己主义者。就算你突破了天花板，很多诱惑欲望，你也是控制不了最终会反受其害。其实领导岗位反而是承担更多东西，加班更多的。如果你只是为了当官而当官，为了提拔而提拔，你做的是升官发财的打算，那趁早算了。</p>
