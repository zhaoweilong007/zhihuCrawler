# 为什么有人说纯水是剧毒？
- 点赞数：24268
- 更新时间：2020年05月20日19时32分20秒
- 回答url：https://www.zhihu.com/question/348180117/answer/841081141
<body>
 <p data-pid="UnDsW6sE"><b><u>这是一氧化二氢的恶作剧</u></b></p>
 <p data-pid="FOjwTXOj">克莱格·杰克逊首次在网络上将此恶作剧发布的版本：</p>
 <p data-pid="K4DSzeVW">一氧化二氢的危险包括：</p>
 <ul>
  <li data-pid="JDZNZFVE">又叫做“氢氧基酸”，是酸雨的主要成分；</li>
  <li data-pid="wn5xHXHN">对泥土流失有促进作用；</li>
  <li data-pid="xg3I58xU">对<a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E6%25B8%25A9%25E5%25AE%25A4%25E6%2595%2588%25E5%25BA%2594" class=" wrap external" target="_blank" rel="nofollow noreferrer">温室效应</a>有推动作用；</li>
  <li data-pid="Z5QO3Wv_">它是腐蚀的成因；</li>
  <li data-pid="pMwxPykJ">过多的<a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E6%2591%2584%25E5%258F%2596" class=" wrap external" target="_blank" rel="nofollow noreferrer">摄取</a>可能导致各种不适；</li>
  <li data-pid="db-DWH2c">皮肤与其固体形式长时间的接触会导致严重的组织损伤；</li>
  <li data-pid="pkjz5yUM">吸入该物质容易引发窒息；</li>
  <li data-pid="sy_0M6lm">发生事故时吸入也有可能致命；</li>
  <li data-pid="ZTD_002w">处在气体状态时，它能引起严重灼伤；</li>
  <li data-pid="h_w011DB">在不可救治的癌症病人肿瘤中已经发现该物质；</li>
  <li data-pid="iesjUu_I">人类只要一接触一氧化二氢就会发生不可逆转的上瘾，对此物质上瘾的人离开它168小时便会<a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E6%25AD%25BB%25E4%25BA%25A1" class=" wrap external" target="_blank" rel="nofollow noreferrer">死亡</a>；</li>
  <li data-pid="i8CHV4mP">全球每年因一氧化二氢死亡者超过37万；</li>
  <li data-pid="NAIhCBlJ">是大多数有毒溶液的溶剂；</li>
 </ul>
 <p data-pid="3DVoEHHN">尽管有如此的危险，一氧化二氢常常被用于：</p>
 <ul>
  <li data-pid="_YEJHFaO">各式各样的<a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E6%25AE%258B%25E5%25BF%258D" class=" wrap external" target="_blank" rel="nofollow noreferrer">残忍</a>的动物研究；</li>
  <li data-pid="mmRSjTAI">美国海军有秘密的一氧化二氢的传播网；</li>
  <li data-pid="D3taGMMm">全世界的河流及湖泊都被一氧化二氢污染；</li>
  <li data-pid="byvxMqwM">常常配合杀虫剂使用；洗过以后，农产品仍然被这种物质污染；</li>
  <li data-pid="xO-oaBro">几乎所有食物都被其污染；</li>
  <li data-pid="w27GY2uU">地球上空气全部被其污染，含量高达0.01%-0.03%，某些地区高达1%；</li>
  <li data-pid="tBKb3oT-">在一些“<a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E5%259E%2583%25E5%259C%25BE%25E9%25A3%259F%25E5%2593%2581" class=" wrap external" target="_blank" rel="nofollow noreferrer">垃圾食品</a>”和其它食品中的添加剂；</li>
  <li data-pid="48rUcubP">已知的导致癌症的物质的一部分；</li>
 </ul>
 <p data-pid="wVKhbehq">然而，政府和众多企业仍然大量使用一氧化二氢，而不在乎其极其危险的特性。</p>
