# 特斯拉时不时就被爆出问题，国产电动车真的就没问题吗，还是国内媒体有意偏向报道/炒作特斯拉负面新闻？

 <p data-pid="7nOGuZer">怎么没有？</p>
 <p data-pid="yTU7NP5A">理想</p>
 <ul>
  <li data-pid="SYe3EjiX">增程式一直被吐槽过渡技术，1.2T发动机，号称新能源车百公里油耗8-9个油。</li>
  <li data-pid="V8bw_iDV">断轴一开始召回说是升级，都没被媒体和用户放过，最后理想道歉了。</li>
 </ul>
 <p data-pid="ombMdI3i">蔚来</p>
 <ul>
  <li data-pid="ZWEBNg6Q">38号车评人写了万字长文批评es8这款车不行。</li>
  <li data-pid="eWTvAaNj">被人吐槽补电要拖辆油车，续航短，老头乐。</li>
  <li data-pid="qfyq21gx">长安街升级趴窝。</li>
  <li data-pid="SpJ3H6aU">19年多辆车自燃，发公告说明原因，召回。</li>
  <li data-pid="9DEXxOWM">因为自燃和补贴滑坡，股价跌到剩1块多面临退市风险。</li>
  <li data-pid="pwwNHIhC">前段时间地下停车场刹车"失灵"，后续公布了行车记录仪，以及当时的详细数据。</li>
 </ul>
 <p data-pid="KRMijThH">小鹏</p>
 <ul>
  <li data-pid="HObvQyuH">G3降价引得老车主不满，把广州总部堵的水泄不通，何小鹏亲自写道歉信。</li>
  <li data-pid="qchnSC7T">各种说小鹏是小偷汽车，连马斯克本人都亲自下场diss小鹏偷代码。</li>
  <li data-pid="V9Cx_BSl">前段时间主动召回也被喷说技术不成熟。</li>
 </ul>
 <p data-pid="E77wP4Eq">国产车能喷，特斯拉喷不得？ 国产厂商起码遇到自己的车主该怂还是怂的，特斯拉是绝不向车主妥协的，但官方一出面，特斯拉就选择深夜道歉。</p>
 <p data-pid="r7a9oIWd">媒体就是喜欢搞事情，你只要用点心就能找出一堆国产新势力的负面报道，电动三傻，蔚来没有未来，小鹏没有朋友，理想没有理想。怎么到了特斯拉这里就想要特殊了，搞得媒体专门针对特斯拉一样。</p>
 <p data-pid="Sxj8LUay">声明：</p>
 <p data-pid="CMtX65Oq">我对特斯拉没什么意见，我也不知道这几次事件中特斯拉的车到底有没有问题。回答这个问题只是为了解答国产电动车有没有问题。有人黑其实也是好事，说明有热度，新事物肯定会受到很多质疑和指责的。</p>
 <p data-pid="r999jiJj">回答里没提到比亚迪是因为不怎么关注比亚迪，但是我觉得有点奇怪，作为一个深圳人，以前听到的都是大家喷比亚迪自燃，骗补，怎么现在大家提比亚迪都跟打鸡血一样。</p>
