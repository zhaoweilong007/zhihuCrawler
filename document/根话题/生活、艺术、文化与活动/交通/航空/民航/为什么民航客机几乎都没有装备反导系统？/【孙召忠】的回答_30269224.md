# 为什么民航客机几乎都没有装备反导系统？
- 点赞数：5805
- 更新时间：2014年12月19日10时23分47秒
- 回答url：https://www.zhihu.com/question/24525856/answer/30269224
<body>
 <p data-pid="Wbesj6KP">像民航客机这么大的目标，如果被导弹瞄上了还想不被打下来几乎是不可能的。</p>
 <p data-pid="GwmYRpc8">如果真想不被导弹打下来，要这样做：</p>
 <ul>
  <li data-pid="KFaMKNWZ">缩小飞机的体积，最长22米；</li>
  <li data-pid="dp4mq1zj">把双发动机挪到屁股上，再加上矢量喷嘴；</li>
  <li data-pid="X_op9iXT">驾驶室旁边加一对鸭翼；<br></li>
  <li data-pid="0A5xUVY0">改成双座，前面是驾驶员，后面是乘客；</li>
 </ul>
 <p data-pid="jTUe31M5">画面：<br></p>
 <ul>
  <li data-pid="jWpIJQx3">司机：您去哪儿啊？</li>
  <li data-pid="9VuRlUhg">乘客：基辅</li>
  <li data-pid="VagVZp9u">……几小时后……</li>
  <li data-pid="aGfqEfv7">乘客：我要喝饮料，怎么没看见空姐？</li>
  <li data-pid="gbmmk-Lg">司机：我早想配个空姐了，领导说就两个座位如果配了空姐旅客坐哪儿呢？我说我可以抱着空姐开飞机没问题的，领导让我立即滚粗去</li>
  <li data-pid="N5rpQtr6">乘客：我要尿尿</li>
  <li data-pid="1Ti3rPTR">司机：我给你打开玻璃罩子，你踩着板凳站起来向后尿，如果向前尿会吹得你满脸都是。风大你可要抓稳了，吹出去要被尾翼切成两半的<br></li>
  <li data-pid="WqTfc1Ho">乘客：我又饿了</li>
  <li data-pid="BOblMVxU">司机：你看见我转向灯的旁边那个红色按钮了吗？</li>
  <li data-pid="NGKIAhy3">乘客：看见了，是管食物的吗？</li>
  <li data-pid="yjJpEYKq">司机：是管弹射的，你再废话就给你弹出去<br></li>
  <li data-pid="XPJsp5jz">……又过了几个小时后……<br></li>
  <li data-pid="2lSGpm7P">乘客：怎么还没到呢？你绕路了吧？我投诉你</li>
  <li data-pid="GELB8SA9">司机：瞧您说的，这不顿涅斯克在打仗嘛，限行啦。坏了！雷达显示来了个山毛榉，红外干扰器又用光了，你再废话我把你泼上汽油点着了扔出去，让你当人肉红外干扰器……</li>
  <li data-pid="SVIBXPLx">司机：来不及了，你扶稳坐好，我要来个大迎角眼镜蛇机动，4、5个g的过载你没问题吧……</li>
 </ul>
