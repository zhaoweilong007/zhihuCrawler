 <p data-pid="wMJAV0zS">科目一是相对于其他那些来说最简单的，只要下载相关APP刷题即可，但不认真对待的话还是可能会挂掉的哦，只是说准备的时间不需要那么多罢了</p>

 <p data-pid="u0qTdPQI">不过大家不要担心，有准备的去考试，保持平常心就会过的。我在这里会跟大家分享自己刷题过程中总结的技巧和经验，<b>为了方便区分和总结记忆，做了提纲</b>，希望看过这篇文章的你们都一次过哦~</p>

 <ol>
  <li data-pid="NMACE_q5">十字路口<b>不可加速</b>；高速公路<b>不可减速</b></li>
  <li data-pid="rEVUOXSc">停靠前开<b>右转向灯</b>，后开<b>双闪</b></li>
  <li data-pid="j6x-9uQc">两车相隔不远，开<b>交替</b>，无交替选<b>近光</b></li>
  <li data-pid="umGqlRwc">都转弯：<b>右让左，转弯让直行，右方路来车先行</b></li>
  <li data-pid="Lb5UkkIb">科二科三道路驾驶技能，考试预约<b>不可超过5次</b></li>
  <li data-pid="puPBZSK-">雨天<b>不开雾灯</b>；无路灯夜晚，请开<b>近光灯</b></li>
  <li data-pid="zEbeX-uT">后有紧急车，可借<b>非机动车车道</b>让行</li>
  <li data-pid="bQ3hoBCN">初次领证者在<b>12个月</b>实习期内上高速，要有<b>三年以上</b>驾龄司机陪同，否则<b>罚200元</b></li>
  <li data-pid="vKvpYMd7">超车只能从左侧超，要<b>提前打左转向灯、变换远近灯、鸣喇叭</b>！！超车后要拉开距离，后开右转向灯开回原道；</li>
  <li data-pid="Ro8ML3Ke">牵引车要在<b>最右侧</b>车道开！牵引车与被牵引车距离：<b>4-10米</b></li>
  <li data-pid="WyYC6sFp">交叉路口堵塞，有无网状线都要等候，不可进入！！<b>交叉路口不可停车倒车</b>！！</li>
  <li data-pid="rlG3u4zx">初次申领驾驶证，要<b>暂住证</b>才可申请大型货车驾驶证！</li>
  <li data-pid="adcYbAhr"><b>辅路让主路</b>：主路车多速度快</li>
  <li data-pid="UtO2d0J0">肇事原因可能是：<b>酒后驾车；无证驾驶；严重超载</b></li>
  <li data-pid="yX_PUduV">开大、中型客货车，每年必审验身体</li>
  <li data-pid="d3yYTI-v">红灯时：车<b>完全</b>过停止线才可过；黄灯时：<b>越过</b>停止线即可过</li>

 <blockquote data-pid="pOQfZVDM">
  <b>part3 科二考试流程及心态调整...问题</b>
 </blockquote>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="zs0ETOph"><b>流程</b>：侧方 上坡 曲线 直转 倒库</li>
  <li data-pid="kK6l9bu7">有<b>两次机会</b>。假如在侧方不及格，则要在<b>30秒内</b>将车开到上坡场地的前面（就是离开侧方，准备进入上坡），然后从上坡开始考试，流程为：上坡 曲线 直转 倒库 侧方（意思是：你开完倒库，又要<b>开到起点的位置，再开一遍侧方</b>，当然如果开完侧方，你听到语音播报及格的话，就可以经过各个项目的场地，把车开回起点，这时候<b>系统不会记录</b>，因为你已经及格了。</li>
  <li data-pid="He43jnWg">另一方面，<b>如果你开到某个地方，又播报考试不及格 ，请注意！！这时你已经用完两次机会</b>，乖乖把车开回起点准备下次再来考吧 ......）以此类推，假如你在倒库挂掉，那前面全部作废，请将车在<b>30秒内</b>开到起点重新来一遍，所以说倒车入库挂掉是最最最悲催的哈哈哈</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="1Ufv0vud"><b>什么时候开始计时？</b>语音会播报的，我们这边是念：侧方停车...之类的<b>项目名称</b>，当车进入地上标记的项目名称时，语音一般就会播报了，不过有时候偏早或偏晚，反正我们认真开车就好了，不要紧张哦~</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="SgPL2V5b"><b>要不要花钱去模拟场？</b>如果<b>心态不好</b>的话我更加建议去，因为模拟场有<b>语音播报</b>什么的，而且开过几遍的话，<b>对考试场地会更加熟悉</b>。很多驾校的练习场地跟考试场地不同，就导致了我们在考试的时候会很陌生，很紧张。此外，我们一直练的是驾校的车，已经彻底熟悉如何控制它的<b>离合、刹车、方向盘等等</b>，这时候你突然接触另一辆不同的车，很可能不能快速学会控制它。因此，去模拟场可以让我们<b>学会随机应变</b>，<b>灵活驾驶动力不同的车</b>~</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="zF6A8CyO"><b>心态不好怎么办？怎么调整？：</b>一般人都是会紧张的，<b>为了使自己有信心考过，我们平时练车就要认真练，花多一些时间去练，这样你就会想：都练得很熟了，会过的吧...</b>嗯.....其实我考科二之前也是觉得自己大几率会挂，我连高考都挂了，老是在关键时刻死亡.....<b>但是！我居然没挂，总结一下原因：①练得很熟（前面说的）；②去了模拟场，面对新环境不会很慌；③了解场地哪条道容易开，为第二次考试做好准备....</b>大概就是这样子.......</li>
 </ul>
