# 历史上有哪两件事听起来不在同一个年代，实际上却同时发生的？
- 点赞数：6517
- 更新时间：2020年07月07日01时17分30秒
- 回答url：https://www.zhihu.com/question/333199024/answer/792150061
<body>
 <p data-pid="kTctintE">今年是公元2019年，而唐王朝建立于618年，两者相差1401年。</p>
 <p data-pid="u4YNS5k5">在历史的坐标轴上，假如我们把唐王朝的坐标往右移动1400年，以2019年为坐标轴原点。</p>
 <p data-pid="HKjfEUGg"><b>左半轴：</b></p>
 <ul>
  <li data-pid="tHfAC9MD">2019年，是唐高祖武德二年，唐朝刚建立一年，这一年，李渊53岁，李世民20岁</li>
  <li data-pid="I6xaJ6--">2018年五月，听闻表弟杨广被弑的李渊终于按耐不住决定称帝，建立唐朝</li>
  <li data-pid="r2BnDGjg">2018年三月，疯狂透支国力的隋炀帝被部下弑杀于江都，年50，隋朝就此凉凉</li>
  <li data-pid="E4g9fR_q">2004年，仁寿宫之变杨坚逝世，杨广（35岁）坐上期待已久的皇位</li>
  <li data-pid="cnIbWH8P">1999年十二月，唐太宗李世民出生</li>
  <li data-pid="zQUgL05f">1989年，隋军荡平南陈，三百年乱局终结，同年李建成出生</li>
  <li data-pid="DI0w3OPw">1981年，同样按耐不住的杨坚（40岁）篡了外孙的北周，建立隋朝</li>
  <li data-pid="1nS6vXor">1977年，北周灭北齐，北方统一</li>
  <li data-pid="d-scZnHU">1969年，杨广出生</li>
  <li data-pid="sdElmWPU">1966年，李渊出生</li>
  <li data-pid="aW74FFno">1957年，北周建立，南陈建立</li>
  <li data-pid="zgpazJki">1950年，北齐建立</li>
  <li data-pid="ylehwD_9">1948年，侯景之乱</li>
  <li data-pid="KUxveJ6e">1941年，杨坚出生，北齐兰陵王高长恭出生</li>
 </ul>
 <p data-pid="4IIqUJ1M"><b>右半轴：</b></p>
 <ul>
  <li data-pid="78QbvrfF">2021年，李世民击败窦建德和王世充，占据河南之地，同年萧铣被李靖、李孝恭率军平定</li>
  <li data-pid="51bBp2rg">2024年，武则天出生，唐朝平定刘黑闼，基本统一全国</li>
  <li data-pid="TvOXXr1D">2026年，玄武门之变，很快，李世民被立为皇太子，很快，李渊退位</li>
  <li data-pid="tspAj3bo">2028年，唐高宗李治出生</li>
  <li data-pid="tenlS8FM">2037年，武则天（14岁）召入宫中</li>
  <li data-pid="1KdPMvK2">2049年，李世民逝世，李治（19岁）即位，武则天出家</li>
  <li data-pid="YT8E86UZ">2083年，李治去世</li>
  <li data-pid="ZbodXBpt">2090年，武则天（67岁）即位为皇帝</li>
 </ul>

 <p data-pid="byzPZpZ9">如果以2019年为原点看隋唐历史会比较清晰得多，因为从北魏分裂开始到唐朝建立时间太短了，夹杂太多王朝建立和崩溃，所以看史料常常以为这些大事发生在不同年代，北魏，东西两魏，北周北齐，南梁南陈都挤在535~557年间消失或是建立，就如同从红军长征到建国十周年之间的历史距离；</p>
 <p data-pid="pk8SJ09i">杨坚出生41年，出生时是西魏人，长大变成北周人并且参与了消灭北齐的战争，后来自己亲手消灭了北周南陈统一天下；杨坚少年见证了：侯景大乱江南，梁武帝饿死，高欢正和宇文泰鏖战中原，北齐北周建立，南梁灭亡，陈霸先建立南陈。</p>
 <p data-pid="gvSAwVyv">杨坚是40后，李渊和杨广表兄弟都是60后，杨广比李渊小3岁；杨广出生，立为皇子，登基和被弑都在李渊的岁月中发生，杨广死后两个月李渊就即位为帝，这时李渊已经53岁了。</p>
 <p data-pid="R53rMsQR">隋朝81年建立，89年统一，而统一到崩溃不到二十年，唐初的人回忆南北朝就如同今天回忆苏联解体一样，讨论东西两魏大战像是今天讨论二战和解放战争，北周北齐南梁南陈像是冷战。</p>
 <p data-pid="o08tQrAC">北魏孝文帝改革和戊戌变法，陈庆之北伐和民国北伐，河阴之变和412政变时间也差不多重合。</p>
 <p data-pid="1_HuKHDi">李世民是90后，99年出生，太原起兵时才18岁，22岁就平定窦建德王世充两大集团，而兄长李建成是80后（89年出生），比李世民大整整10岁，唐朝建立时李建成29岁。</p>
 <p data-pid="qOrrfZhO">玄武门之变是26年，此时李世民26岁，而李建成已经37岁了。</p>
 <p data-pid="YV2eoo7k">武则天比李治大四岁，14岁就入宫中（这画面），25岁当尼姑，在皇宫中待了53年才修成正果，登上帝国之颠时已经67岁，小女生熬成才人，尼姑，昭仪，皇后，太后，大周皇帝，嗯，一节更比六节长。</p>
