# 大学生业余时间可以用来学什么技能？
- 点赞数：7901
- 更新时间：2023年09月11日00时36分22秒

<body>
 <h2>有朝一日我们都能成为六边形战士！</h2>
 <ol>
  <li data-pid="PdbXae25">急救包扎注射一类医疗技能，日常必备，有兴趣可以继续拓展中医西医，</li>
  <li data-pid="8Y95mBhU">烹饪，</li>
  <li data-pid="IWjJlUlx">office（Word，Excel，PPT），买一本《玩转office》去考两次全国计算机等级考试二级MS office就差不多熟了，</li>
  <li data-pid="IUQbCJwT">法律，尤其是劳动合同法，以及社保内容，工作必备，</li>
  <li data-pid="PK4rutRS">考专业证书（大学可以考哪些证请根据个人情况自行研究，比如常见的CET6、职称类、计算机二级MS office、教师资格证以及驾照可供参考）根据政策，考试资格每年都在变动，能考就尽快，</li>
  <li data-pid="vO9B3Rb-">专业课或辅修第二专业，毕竟不同专业有不同的用途，校内应该有渠道，也可以用网易公开课、中国大学MOOC什么的找找资源，</li>
  <li data-pid="1C0BawO1">理财，接触期权、股市、转债、利率、杠杆这些东西，可以下载个雪球先研究研究，这里面要学习的太多了，</li>
  <li data-pid="ZbFcTDTz">Ps Ae Pr等一系列Adobe设计软件使用，可以完成各种图片、视频、音频的后期，</li>
  <li data-pid="2Bm_V1nm">实习，很多公司只招在校生，好用的app平台有实习僧、BOSS、应届生求职等，还有校内宣讲会也可以多关注一下。</li>
  <li data-pid="mSCu1xj2">能力与成长，考资格证比如游泳教练资格证、轮滑教练资格证，周末去兼职一下玩玩，</li>
  <li data-pid="n0Te1FlB">多读书（这类app有kindle ，微信读书，网易蜗牛等），</li>
  <li data-pid="0eTTsHMA">运动①轮滑和冰刀（根据季节，一般是夏天轮滑，冬天冰刀）②球类像篮球排球足球乒乓球棒球网球羽毛球曲棍球棒球台球高尔夫球保龄球橄榄球垒球冰壶等等③瑜伽普拉提④武术散打剑道跆拳道柔道女子防身术等⑤滑板⑥其他像田径自行车跳绳游泳潜水健美操拳击蹦床射击射箭摔跤举重登山冲浪等等⑦找不到社团或者不愿意整以上这些的，就办个健身卡去健身房就完事，同样强身健体！⑧不想花钱去健身房的可以自学八段锦、金刚功、五禽戏、太极拳这类玄学五术-山。⑨跑酷（徒手爬墙、高处跃下这类，危险）PS：错误的跑步姿势伤膝盖，大家加油）</li>
  <li data-pid="yDI6xjtP">绘画，</li>
  <li data-pid="Q68WTqIB">编程，</li>
  <li data-pid="6RDLgn7W">烘焙甜点，</li>
  <li data-pid="V_iwKaQR">练字，</li>
  <li data-pid="I-DhSc7e">小语种，以及手语、盲文、摩斯电码等，方言如粤语、四川话、东北话、上海话、世界语、精灵语，丘丘语兰那罗语（皮一下），</li>
  <li data-pid="6H9Za37o">编织以及裁缝（手链，中国结，钥匙扣，十字绣，毛衣围巾等，家里有缝纫机的可以试着做衣服，还可以学习纺织、染色工艺，服装设计），</li>
  <li data-pid="Izxnr_gy">diy（uv胶，手工纸，橡皮章，陶泥，扎染，热熔胶，热缩片，流麻，造花液，黏土手办模型这些没玩过，别问我），</li>
  <li data-pid="DgsLlKdM">修自行车修电脑修手机修手表修水管接电路，</li>
  <li data-pid="I5m_9NOb">唱歌，</li>
  <li data-pid="iq_EjB0E">乐器（吉他钢琴古筝小提琴手风琴电子琴拇指琴竖笛空灵鼓琵琶古琴等），</li>
  <li data-pid="XG5fqplp">人际关系（实践多见闻），</li>
  <li data-pid="gk3BV1l9">舞蹈（街舞爵士舞民族舞拉丁舞交际舞宅舞芭蕾舞踢踏舞埃及手舞），</li>
  <li data-pid="Qy7GJyNH">美妆美甲美发设计以及穿搭技巧（这些资源太多了）</li>
  <li data-pid="UNVQtD_3">摄影，学会了可以给别人省一大笔写真成本，但是自己大概率不会出现在镜头里...，</li>
  <li data-pid="kMu4v1n0">魔术，n阶魔方魔板，</li>
  <li data-pid="pqi19um2">高中的知识以及扩充，</li>
  <li data-pid="FPhE9ga9">棋牌（中国象棋，国际象棋，围棋等），</li>
  <li data-pid="6bnw7ALI">玄学类（塔罗，风水，相面，易经，算命，回教符咒，祝由术...我说着玩的，大家别当真ψ(｀∇´)ψ ），</li>
  <li data-pid="GPQ5C2aX">可以备考考研考公，虽然有的不能报考，反正闲着也是闲着，</li>
  <li data-pid="_jHXptS6">参加各种竞赛，常见的像挑战杯、互联网+、数学建模，完成竞赛的过程也是能学到不少东西的，</li>
  <li data-pid="sztA1mMN">调香，调酒等</li>
  <li data-pid="TacFcCFR">雕刻及折纸，雕刻用地上的石头，树上的芒果，哪怕一块橡皮都可以试试：折纸资源太多了，能锻炼空间拓扑思维是最好的</li>
  <li data-pid="6BJRZtX2">房间设计，尝试按照各种户型给自己规划一个新家吧，包括房间的空间利用、地板和墙面选材等等，只有了解行业未来才能不被忽悠，</li>
  <li data-pid="uFnkey_5">非理工科类学生可以学习一下pcb板的电路绘制和焊接工艺，有专业人士人带领的情况下结合编程可以创造一些有意思的小东西，</li>
  <li data-pid="JQrG0FYD">创作，如编曲、写书、插画形式等，</li>
  <li data-pid="m_EPo99y">听说有人不会骑自行车，打响指，吹口哨，Bbox</li>
  <li data-pid="Ti9zMsYN">训犬，宠物美容</li>
  <li data-pid="QhEKCijH">飞针飞牌</li>
  <li data-pid="nZpV266n">奢侈品与珠宝鉴定</li>
  <li data-pid="CZvIpNeX">配音声优，在游戏、动画、媒体行业应该是个不错的就业方向，</li>
  <li data-pid="IIzwExHg">脑算速算，据说是在头脑中形成算盘的动态建模</li>
  <li data-pid="VuhH2LKS">品茶品酒花艺园艺</li>
 </ol>
