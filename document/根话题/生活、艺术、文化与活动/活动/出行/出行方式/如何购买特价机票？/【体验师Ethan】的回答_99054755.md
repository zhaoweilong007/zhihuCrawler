# 如何购买特价机票？
- 点赞数：6847
- 更新时间：2016年05月05日16时10分34秒
- 回答url：https://www.zhihu.com/question/28532142/answer/99054755
<body>
 <p data-pid="yR8binRp">想买到特价机票，一定要掌握好下面两个绝招，整理了一下放上来，应该是比较全的了。可能很多人发现我很久没有发知乎了，最近太忙，我也想你们。</p>
 <br>
 <p data-pid="7B_DRVZN"><b><u>大招一</u></b><b><u>根据目的地选择最适合廉航</u></b></p>
 <p data-pid="mCaoskm8">廉价航空相信大家都不陌生，这些航空公司通过取消一些传统服务来控制成本，因此可以长期大量提供便宜机票。根据不同目的地巧妙选择廉价航空，有时候可能便宜的出乎你的想象。下面就来历数一下，各大洲到底有哪些方便好用的廉价航空。</p>
 <ul>
  <li data-pid="cIx9PUVd"><b>东南亚</b><b>·</b><b>香港快运</b></li>
 </ul>
 <p data-pid="-8ssROTF">香港快运（HKexpress）是目前唯一一家以香港为基地的低成本航空公司，可达到目的地目前全部为亚洲地区，除了香港外，还有中国内地（昆明、宁波、兰州、无锡）、台湾、日本、韩国、泰国、越南、柬埔寨及缅甸的各大旅游热点城市，当然，从香港出发的价格是最好的。</p>
 <br>
 <p data-pid="F8JQLWxK">除了价格便宜之外，香港快运以84.5%的准点率成为香港最准时的航空公司，作为廉价航空，这一点真的是它很大的优势。还有件有趣的事，香港快运拥有的九架空中巴士A320中的五架以“点心”主题命名，别人都坐飞机你坐叉烧包去玩耍是什么感觉？</p>
 <ul>
  <li data-pid="XRbQaum1"><b>东南亚</b><b>·</b><b>亚洲航</b><b>空</b></li>
 </ul>
 <p data-pid="BTbTIhsY">亚航（AirAsia）几乎是所有人在东南亚旅行中最常用的廉价航空公司。从深圳、澳门、香港、广州、海口、杭州和天津出发，以吉隆坡和曼谷两地为主要中转地，可以连接马来西亚（16个大城市）、泰国（11个大城市）、印尼（14大城市）、越南（胡志明和河内）、柬埔寨（金边和吴哥）、缅甸（仰光）、老挝（万象）、菲律宾（马尼拉）、文莱等大部分东南亚地区，近期还开通了到澳大利亚（黄金海岸布里斯班和珀斯）的航线。</p>
 <br>
 <p data-pid="GGcCPyas"><u><b>除了本身就已经很便宜的机票之外，亚航一年有四次新航季大促销，通常是2月(有时候到3月)、5月、8月和11月，售卖半年以后至一年内的机票，和很多不定时的小促销。</b></u>在促销时能看到大量0元票或1元票（当然还是要付税费的），甚至能占到一半的航线。关注他家的微信和微博，就能很方便的得到促销信息。</p>
 <ul>
  <li data-pid="OWSIrtqg"><b>东南亚</b><b>·</b><b>捷星航</b><b>空</b></li>
 </ul>
 <p data-pid="VysMyzaD">其实我们在东南亚乘坐的捷星应该叫做捷星亚洲，主营业务为新加坡，除此之外还有澳大利亚和新西兰的捷星航空 ，越南的捷星太平洋和捷星日本，每天为澳洲、新西兰和亚太地区航线提供无数廉价机票。</p>
 <br>
 <p data-pid="J-RskHFv">有全球十大最安全航空公司首位的澳航做背后的老板，捷星航空在廉航中的客户满意度一向都最高，在享受低票价的同时，有个很好的旅行体验也是很重要的。关注捷星的微博或者官方微信可以查看促销信息。</p>
 <ul>
  <li data-pid="SRk-8NEN"><b>东南亚</b><b>·</b><b>酷</b><b>航</b></li>
 </ul>
 <p data-pid="DlVbD451">酷航（Scoot）是新加坡航空旗下的长途航线廉价航空公司，使用从母公司获得的波音777客机运营，初期主要运营往澳大利亚和中国的航线，现在已经扩展到亚太地区的14个国家。<b><u>2016年，酷航会进行一月一次的“闪人大促”，每次促销将会在每个月第一周固定的某一天，可以关注看看有没有你想去的目的地。</u></b></p>
 <ul>
  <li data-pid="O5Cxl2y5"><b>美洲</b><b>·</b><b>西捷航</b><b>空</b></li>
 </ul>
 <p data-pid="Sn9dYI1B">西捷航空（WestJet）是以加拿大卡加利市为总部的加拿大第二大航空公司，目的地共有89个，主要在北美洲及墨西哥。不久前还将机队中的一架波音737-800客机，改装成冰雪奇缘的主题，除了外表涂装，就连座椅枕巾、头顶置物柜也印上雪花与太阳花的图形。</p>
 <ul>
  <li data-pid="QrPsj5u8"><b>美洲</b><b>·</b><b>美国西南航</b><b>空</b></li>
 </ul>
 <p data-pid="vDLwnFT-">西南航空公司（Southwest Airlines ）是世界最大的廉价航空公司，成立于1971年6月18日，在总部设在美国德克萨斯州，是民航业“廉价航空公司”经营模式的鼻祖。在服务业早已将「顾客至上」作为口号时，西南航空却说：「员工第一，旅客第二」。西南航空的管理阶层都有一个共识：无论什麽情况，他们都相信员工有能力处理任何状况。因此，当一名行动不便的旅客滑着轮椅前来，地勤人员便决定开启已然关闭的机舱门。这反倒成就了西南航空最大的优势之一。</p>
 <ul>
  <li data-pid="KHCb667f"><b>欧洲</b><b>·</b><b>瑞安航</b><b>空</b></li>
 </ul>
 <p data-pid="oYHFO2We">瑞安航空（Ryanair）总部设在爱尔兰，是欧洲最大的廉价航空公司，拥有209条廉价航线，遍布欧洲17个国家的94个目的地，主打西欧南欧。<b><u>现在虽然不像前两年经常出现1欧、2欧或5欧的机票，但一般只要你提前做好功课、定好行程，不碰上大节日提前订，十欧到二十几欧就可以拿下机票。</u></b>如果是在某一个国家国内飞的话，对比一下就可以知道，甚至比火车便宜得多。</p>
 <ul>
  <li data-pid="0p5skGdk"><b>欧洲</b><b>·</b><b>易捷航</b><b>空</b></li>
 </ul>
 <p data-pid="6fvICTwx">易捷航空（easyJet）是一间英国的航空公司，总部位于英国伦敦卢顿机场，营运超过600条的国内、国际航线航线，航点遍布32个国家，载客数首屈一指，是英国最大的航空公司，欧洲第二大低成本航空公司，仅次于瑞安航空。</p>
 <br>
 <p data-pid="tTNrjT5z"><b><u>易捷航空的飞航点遍及全欧洲，只花50英镑就可以从伦敦的Gatewick机场或从北方的利物埔机场飞往欧洲其他国家，如西班牙、荷兰、法国、德国、瑞士及希腊等。</u></b>此外，Easy Jet还与欧洲许多国家的饭店有网际连接，提供便宜的住宿订房服务。</p>
 <ul>
  <li data-pid="KYhw-ohi"><b>大洋洲</b><b>·</b><b>维珍澳洲航</b><b>空</b></li>
 </ul>
 <p data-pid="pTzvdezH">维珍澳洲航空（Virgin Australia）是澳洲第二大的航空公司，也是英国维珍集团中拥有最庞大机队的航空公司。在澳大利亚境内有33个航空站，主要大城市的航班相当多，一天有多个时段可供选择，从澳洲往返新西兰和南太平洋群岛的班次也很密集，还有飞往英国，美国的航班可选择。</p>
 <br>
 <p data-pid="SO6t7_Gl"><b><u>每周四4:00pm-11:00pm（澳洲东岸时间）为减价时段，只要在网站上看到 “Happy Hour”出现，就有机票特价活动。</u></b>配合不同的节日，也会推出不同的特价方案，想买到便宜机票就必须勤快地经常上网页查看。</p>
 <ul>
  <li data-pid="dEyHtVgD"><b>大洋洲</b><b>·</b><b>虎</b><b>航</b></li>
 </ul>
 <p data-pid="rvZNvQ-c">虎航（Tigerair）澳大利亚在境内有12个航空站，主要大城市都有航班，大多在一百澳元左右。虎航便宜是便宜，不过设施就不要抱太大期望了，座位可能不会那么舒适，时有晚点的情况。曾经乘坐虎航澳大利亚的时候，发现check in柜台的小哥，后来在飞机降落后又为我们开舱门……虎航的员工应该真的是很少。</p>
 <p data-pid="-oS1-lvC"><br><b><u>大招二，换个</u></b><b><u>搜索引擎</u></b></p>
 <br>
 <p data-pid="hh7G2xuV">上面推荐的廉航更适合短途行李少的旅行，如果需要购买很多行李重量和餐食，廉航可能就不那么合算了，这时候，找到一家合适你的航空公司及合适价格就要靠搜索引擎了。</p>
 <ul>
  <li data-pid="ffwtX8hI"><b>天巡（</b><a href="https://link.zhihu.com/?target=http%3A//tianxun.com" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">http://</span><span class="visible">tianxun.com</span><span class="invisible"></span></a><b>）</b></li>
 </ul>
 <p data-pid="wUMKOBkP">skyscanner在中国叫做天巡，是预订国际机票必备搜索引擎之一。订票的时候会链接去代理商及航空公司官网，有很友好的中文界面并且能够直接显示人民币价格。可以在天巡关注某些航线的价格趋势，设置价格提醒，每天都会有邮件提醒你价格波动。优点是对于英文没那么好的网友比较适合。</p>
 <ul>
  <li data-pid="LbCyX6rQ"><b>PRICELINE</b>（<a href="https://link.zhihu.com/?target=http%3A//www.priceline.com/" class=" wrap external" target="_blank" rel="nofollow noreferrer">Priceline.com</a> ）</li>
 </ul>
 <p data-pid="PX-9Au8Z">PRICELINE是目前美国最大的在线旅游公司。在PRICELINE网站，最直观的可选项目就是机票、酒店、租车、旅游保险，机票价格偏贵。<b><u>但是如果临时决定出行，可以尝试他家的bid your price，这个功能支持你输入一个比较低的报价，系统会自动征询所有的航空公司。</u></b>如果有航空公司愿意接受这个报价 ，就赚到了。这个网站还有一个独特的功能，就是能帮你组合两个不同航空公司的航班，让你买到比正常情况便宜的转机机票。不过订这种票需要考虑好转机时间及行李提取的问题，避免因飞机延误造，转机时间不够等造成不必要的麻烦。</p>
 <ul>
  <li data-pid="D-xa6nvG"><b>STUDENTUNIVERSE</b>（<a href="https://link.zhihu.com/?target=http%3A//www.studentuniverse.com/" class=" wrap external" target="_blank" rel="nofollow noreferrer">Cheap student flights, hotels &amp; travel discounts</a> ）</li>
 </ul>
 <p data-pid="c_C5356K">这是一个针对学生机票的搜索引擎（要有.edu的邮箱并小于35岁）。大部分情况下它不会比下面要介绍的KAYAK便宜，但有时候推出的学生机票便宜的惊人。</p>
 <ul>
  <li data-pid="RSsDRwVW"><b>KAYAK</b>（ <a href="https://link.zhihu.com/?target=http%3A//www.kayak.com/buzz" class=" wrap external" target="_blank" rel="nofollow noreferrer">KAYAK Explore</a>）</li>
 </ul>
 <p data-pid="NoWX2_hU">在你只确定了大概出行时间却不知道去哪里的时候，KAYAK就体现出它的优势，只要输入出发地和大概时间，就会帮你按月份罗列出从你出发地出发的最便宜的机票。</p>
 <ul>
  <li data-pid="N43H8UfD"><b>LASTMINUTE</b>（<a href="https://link.zhihu.com/?target=http%3A//us.lastminute.com/" class=" wrap external" target="_blank" rel="nofollow noreferrer">lastminute.com</a>）</li>
 </ul>
 <p data-pid="V34c9bjF">就像它的名字，这个网站是那些不到起飞前不买机票的拖延症患者的救命稻草。但是早动手才是王道，毕竟这种说走就走的事还是有一定风险的。　　</p>
 <br>
 <p data-pid="CcAWZ3-m">好了，基本就是这些了。其实并不建议大家花费太多时间在订机票上，毕竟如果是出去玩，你更应该花心思在“玩什么”上面。大部分中国旅行者，还是停留在用几天时间比对机票酒店，然后看一堆长得很像的攻略就走的状态。</p>
 <br>
 <p data-pid="TsQsA02i"><b>与省机票钱相比，找到特别的旅行体验并提前完成预订（省去到当地比价和选择）在我看来更为实用和有价值。</b>这也是我为什么在做KLOOK旅行的原因，在路上通过参与户外、文化、美食、步行游等活动探索更多元的旅行体验。</p>
