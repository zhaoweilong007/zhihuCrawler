# 重庆未拴绳犬在闹市被八枪枪毙，我朋友认为相关人员操作有误，狗命和人命同样平等，我该怎么反驳？
- 点赞数：44442
- 更新时间：2021年05月31日09时39分26秒
- 回答url：https://www.zhihu.com/question/458955676/answer/1914671804
<body>
 <ol>
  <li data-pid="4_TRE-UH">狗命就是狗命，和鸡命、猪命、兔子命是一样的，和人命没有可比性。如果医学上发现杀一千条狗可以挽救一个绝症病人，医生就会去杀狗，政府和法律也绝对支持。相对的，如果兽医学发现杀一个人可以挽救一千条狗，兽医不会去杀人。谁敢杀政府和法律就杀谁。</li>
  <li data-pid="llHb7Y7o">如果某人认为他自己的命和狗命一样，那是他的自由。但是不能推广到其他人，除非那个人也自愿当狗。即便如此，法律也不会承认这种自降人格的行为。</li>
  <li data-pid="7Y2fma69">如果一条狗认为人命与狗命平等，那么它就是需要被消灭的刺头狗。如果一个人认为狗命和人命平等，这就是个不配当人的脑残。</li>
  <li data-pid="gn0Ax1Km">不要和脑残当朋友，这玩意儿可能传染。</li>
 </ol>
