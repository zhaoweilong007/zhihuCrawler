# 你见过的最漂亮的硬笔书法是什么样的？
- 点赞数：5814
- 更新时间：2020年04月10日23时00分07秒
- 回答url：https://www.zhihu.com/question/283091823/answer/980746543
<body>
 <p data-pid="XEI0ZBO9">这个和中国好声音一样，第一届全国硬笔书法大赛，能拿特等奖的，就是最强的。</p>
 <p data-pid="Gc3yQCJS">中国在80年代有多次大赛，一开始是青年大赛，那个时候就有任平、顾仲安的一等奖。</p>
 <p data-pid="i0EmaxuS">到了1985年，搞了一次不分年龄的全国大赛，全国十个特等奖，这是第一届不分年龄的全国大赛，是全国积累很多年的人才，评委是沙孟海这种大家。</p>
 <p data-pid="ni4Epi8p">全国范围能出来的，都有两把刷子。</p>
 <p data-pid="ndoNZSfO">这些人有十个， 中老年组（4人） <a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E9%25A9%25AC%25E6%2598%258E" class=" wrap external" target="_blank" rel="nofollow noreferrer">马明</a>（江苏） <a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E5%2588%2598%25E6%2583%25A0%25E6%25B5%25A6" class=" wrap external" target="_blank" rel="nofollow noreferrer">刘惠浦</a>（贵州） <a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E8%25B5%25B5%25E5%25BD%25A6%25E8%2589%25AF" class=" wrap external" target="_blank" rel="nofollow noreferrer">赵彦良</a>（新疆） <a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E6%259B%25B9%25E5%25AE%259D%25E9%25BA%259F" class=" wrap external" target="_blank" rel="nofollow noreferrer">曹宝麟</a>（安徽）</p>
 <p data-pid="7pcvPd4Y">青年组（6人） 任平（浙江） <a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E5%25BC%25A0%25E7%25A7%2580" class=" wrap external" target="_blank" rel="nofollow noreferrer">张秀</a>（女，湖北）<a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E4%25BB%2587%25E5%25AF%2585" class=" wrap external" target="_blank" rel="nofollow noreferrer">仇寅</a>（福建） 顾仲安（上海）葛全胜（北京） <a href="https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/%25E5%25BC%25A0%25E6%259D%25B0" class=" wrap external" target="_blank" rel="nofollow noreferrer">张杰</a>（江西）</p>
 <p data-pid="0zAyV36y">后来出字帖的有顾仲安，仇寅，任平，刘惠浦、张秀（唯一女性，当年23岁。）</p>
 <p data-pid="ijtVdzLT">从实用角度，行楷和行书，适合日常记笔记用。楷书张秀那种字体写起来会快一点，仇寅和刘惠浦的好看是好看，真写起来太慢。</p>
 <p data-pid="XQtYwe63">书法家也有各自擅长的，顾仲安的楷书也不错，但是他得奖是作品是行楷，学习要学其所长。</p>
 <p data-pid="irxTBWnw">多看看这种字，对硬笔字的汉字审美会有帮助。不至于拿着所谓神仙体，江湖体当宝。</p>
