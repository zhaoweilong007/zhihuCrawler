# 古代的通房丫头是什么意思？

<body>
 <p>少爷的通房丫头，从小就要陪少爷经历一些青春期的尴尬，长大后，也会继续发挥工具的作用。</p>
 <p>小姐的通房丫头，作为陪嫁，要在洞房之夜，守在床边进行圆房指导。</p>
 <p>一不小心，这一可怜的群体，就会被迫卷入更毁三观的纷争之中……</p>
 <p><em>*本文结合史料，以一个通房丫鬟的形象，讲述这个人群的真实处境。</em></p>
 <h2><strong>1.新婚</strong></h2>
 <p>民国初年，保定府。</p>
 <p>都说大宅门里腌臜事儿多，当地显贵张家，更是脏得连门口的石狮子都不干净。入夜，张家大院里外依旧张灯悬彩。</p>
 <p>参加婚礼的宾客亲友早已经散尽，正是新人春宵一度的时候。</p>

  <img alt="身着西服婚纱的民国婚礼合影" data-rawheight="720" data-rawwidth="990" src="https://picx.zhimg.com/50/v2-353ce30bb1c3b2995a90a9edc3cfc58b_720w.jpg?source=1940ef5c" data-original-token="v2-353ce30bb1c3b2995a90a9edc3cfc58b" class="origin_image zh-lightbox-thumb" width="990" data-original="https://pica.zhimg.com/v2-353ce30bb1c3b2995a90a9edc3cfc58b_r.jpg?source=1940ef5c">

 <p>新郎官张少爷刚过十八岁，他兴致勃勃地揭了盖头，喝了交杯酒，就要伸手去拉新娘高小姐。</p>
 <p>可高小姐却像吓坏了一样，丝毫不配合，一边抗拒着新郎官的动作一边躲，最后还哭了起来。</p>
 <p>张少爷费了半天劲，竟然毫无进展，不禁恼怒起来。</p>
 <p>高小姐从娘家带来的贴身丫头兰玉儿急得手足无措，赶紧上前安慰主子，想帮着新郎官圆房。</p>
 <p>可越是这样，新娘子哭得越厉害。</p>
 <p>旁边伺候洞房的张少爷家的奶妈吴妈却站着没动。</p>
 <p>她尚且年轻，面目姣好，望着眼前的情形一脸鄙夷地冷笑着。</p>
 <p>几个回合拉扯下来，张少爷仍未得手，气得跳下床来，一把拉住了吴妈。</p>
 <p>吴妈吓了一跳，赶紧说：「少爷，这可使不得，今天是您大喜的日子。」</p>
 <p>说完眼睛一直往边上新娘身上瞟。</p>
 <p>张少爷冷脸甩出几句话：「真是不懂事，坏我的兴致，要是不想洞房，趁早回家！」</p>
 <p>说完竟拉着吴妈占了原本新娘的位置，一时间把边上的高小姐和兰玉儿给看呆住了。</p>
 <p>尽兴之后，张少爷拿出了一贯的荒唐做派：「吴妈，还是你好，早知道我娶你不就好了。」</p>
 <p>吴妈面露得色，她原本是这家的奶妈，后来又借机上位，成了少爷的「枕边人」，已经无数次地幻着自己能够成为张家少奶奶了。</p>
 <p>可她毕竟是个下人，这种事根本不可能发生，却没想到竟在少爷的新婚之夜抢了新娘的风头，真是又惊又喜又隐约害怕。</p>
 <p>这还不算完，张少爷跳下床，竟拉着吴妈去了她的房间，把个新娘子晒在了洞房里。</p>
 <p>见两人走了，兰玉儿赶紧安慰满脸泪痕的高小姐。</p>
 <p>「小姐，到底怎么了？」</p>
 <p>高小姐「哇」地一声哭出来。</p>
 <p>「他为什么要这样羞辱我，让我死了吧！」</p>
 <p>兰玉儿一时哭笑不得不知说什么好。</p>
 <p>只因高府家教古板，高小姐父母早亡，是祖父一手带大，只学了一肚子孔孟之道，却浑然不晓得男女之事。</p>
 <p>兰玉儿叹了口气，只能悄声安慰小姐，「大喜的日子可不能哭呀，赶明儿就好了……」</p>
 <p>可连着几天，张少爷晚上都去吴妈的房里睡，早上才回来，对新娘子爱答不理。</p>
 <p>高小姐的脸被丢尽，自己的新婚之夜被别人当面抢了丈夫不说，对手竟然还是个下人。</p>
 <p>兰玉儿更是纳闷，这个吴妈跟我不都是低贱之人吗？究竟有什么了不得的本事，居然能让少爷在洞房之夜撇妻而去？</p>
 <p>细心一打听，才知道事情没这么简单。</p>
 <h2><strong>2.丫头</strong></h2>
 <p>张高两家都不是普通人，可算得上门当户对。</p>
 <p>张老爷做过海关监督的肥差，入股多家银行商号，家资巨万。</p>
 <p>高小姐的祖父则是前清翰林，曾给皇上讲过课，高家是全城有名的清贵宅第。</p>

 <p>一个有钱，一个有名，正凑成一个「好」字，可谓天作之合。</p>
 <p>可别看宅门大，这里头的龌龊事比寻常人家只多不少。</p>
 <p>张家三代单传，生下的其他男孩都夭折了，因此对这个儿子娇惯至极，十几岁才断的奶。</p>
 <p>吴妈就是张少爷最后一个奶妈。</p>
 <p>那时候吴妈也就十五六岁，长相出众。除了奶妈的工作，还帮着少爷的照顾日常起居。</p>
 <p>一来二去，俩人就凑到了一起。</p>
 <p>张家老爷太太一看，干脆让吴妈给少爷当了通房丫头，里里外外全都伺候。</p>
 <p>因为这层关系，吴妈的气势明显高过其他下人，甚至有自己单独的房间，只在老爷太太面前必恭必敬。</p>
 <p>兰玉儿看得特别明白，吴妈表面上对小姐客气，私底下根本不拿这位新主子当回事。</p>
 <p>一离开老爷太太的视线，她就和少爷打情骂俏，就差当众宣淫了。</p>
 <p>小姐是个老实人，急了就知道哭，兰玉儿可看不下去了。</p>
 <p>你一个奶妈兼通房丫头的身份，也配跟我们小姐争？</p>
 <p>我得帮小姐「开窍」，拢住男人的心。</p>
 <p>毕竟，大宅门里的人最势利眼，谁能拴得住爷们儿，谁说话才有底气。</p>
 <p>兰玉儿从箱子里找出一个小木匣，这是小姐出嫁前，高家派了个教习妇，秘密给她传授的。</p>
 <p>兰玉儿知道，自己该出手了。</p>
 <h2><strong>3.夺夫</strong></h2>
 <p>刚看了一眼兰玉儿拿出的东西，高小姐便红了脸，双手捂住眼睛。</p>
 <p>「看这种脏东西，真是羞死人了……」</p>
 <p>「我的好小姐呀，你再不长进，都被人家骑在头上了！」</p>
 <p>原来这竟是一套「秘戏图」。</p>
 <p>这种隐秘的册子，在汉代时便已出现雏形。</p>
 <p>在礼法严苛的古代，春宫图是最重要的性启蒙教材，通常会在女子陪嫁品里放置一册，供新婚夫妻参考学习。而陪嫁的通房丫头，往往要承担「教师」的角色。</p>
 <p>见高小姐双眼紧闭，兰玉儿急得直叹气。</p>
 <p>作为书香门第的陪嫁丫鬟，她有另一重身份和任务，就是帮助纯洁的小姐在夫家立足。</p>
 <p>至于用什么方法，用什么手段，都是教习妇仔仔细细叮嘱过的。</p>
 <p>她这种人，不但要伺候小姐，还要在小姐和夫君同房的时候成为「助力」。</p>
 <p>千说万劝，小姐总算被说动了。</p>
 <p>当天晚上，张少爷与高小姐顺利圆房。</p>
 <p>兰玉儿不时递个巾子，端盆温水，在一旁细心伺候。</p>
 <p>少爷一时兴起，抓住兰玉儿的胳膊，不料兰玉儿脸一板。</p>
 <p>「姑爷还是照看着点小姐，别饶上丫头。」</p>
 <p>她十分清楚，小姐是主子，自己是下人，主子绝容不得下人争宠。</p>
 <p>直到小姐点头，兰玉儿才顺服地任由主人处置。</p>
 <p>天还未亮，兰玉儿已经爬起来，准备伺候主人穿衣洗漱。</p>
 <p>看见卧床上一对鸳鸯沉沉酣睡，兰玉儿心中又是欣慰，又是苦涩，一时不知是喜是忧。</p>
 <p>小姐总算赢回局面，而自己又算是什么东西呢？</p>
 <h2><strong>4.工具人</strong></h2>
 <p>兰玉儿是个苦命人，她十岁就被赌鬼父亲卖到高家，给高小姐当贴身丫头。</p>
 <p>一开始因为手脚粗笨，没少被高家的管家收拾，挨打挨饿那只是家常便饭。</p>
 <p>她心里暗存了一个念头：伺候好小姐，等小姐出嫁时带自己走。</p>
 <p>她哪里知道，陪嫁出去的丫头还有「通房」的职责，如同苦瓜秧上结黄连，苦个没完！</p>
 <p>所谓通房丫头，就是过去大户人家特意买的幼女，让她们从小伺候少爷小姐长大，等主子结婚后继续贴身服务。这种恶俗一直流传到民国也没被禁止过。</p>
 <p>表面看，通房丫头跟主人关系亲近，近水楼台先得月，地位似乎比一般仆人高级。</p>
 <p>实际上她们的命运无比悲惨，说出来全是血泪。</p>
 <p>大户人家的少爷进入青春期时，欲望开始萌发，为了解决这个难题，通房丫头就派上用场。</p>
 <p>这样安排既满足了少爷的需求，又避免嫖妓沾染到脏性病，也不会败坏自家名声。吴妈就是因此从奶妈转成通房丫头的身份的。</p>
 <p>像兰玉儿这种随小姐嫁出去的通房丫头更惨，她们既要伺候主子一家起居，还要无条件满足男主人的需求。</p>
 <p>甚至有些名门大族，怕女儿嫁过去受委屈，就提前派个通房丫头过去男方家里「试婚」。</p>
 <p>「试婚」的内容包括考察男方的人品性格，以及健康状况。</p>
 <p>试得好，女方欢喜出嫁自不必说；万一试出男方是个银样镴枪头之类，那就干脆找个借口或赔钱退婚。</p>
 <p>因此，通房丫头没有私人空间，睡觉的地方与主人卧室相通，为的就是方便随时伺候主人。</p>
 <p>要在夫妻行事时，在旁守着、帮着。</p>
 <p>说白了，这个群体等于仆人加工具人，完全没有自主权，比桌椅板凳多喘一口气儿罢了。</p>
 <p>兰玉儿没想到，这次帮小姐夺回少爷的注意，只是暂时歇口气罢了，黑暗刚开始降临。</p>
 <p>更令人咋舌的荒唐事儿还在后面。</p>

 <h2><strong>5.儿子</strong></h2>
 <p>大户人家内部的争斗，就像潜伏在水面之下的漩涡，往往在暗中涌动。</p>
 <p>有了兰玉儿帮忙，高小姐终于在争宠大战中站稳脚跟，新的挑战接踵而至。</p>
 <p>要想彻底巩固地位，她必须尽快怀孕，为张家生下一个男孩。</p>
 <p>转眼间新婚已经过去一年多，高小姐的肚子依然没有动静，这成了家里的一块心病。</p>
 <p>张家三代单传，老爷又是晚来得子，在张府里，传宗接代是天字第一号大事。</p>
 <p>老爷请来不少名医为少奶奶检查身体，有德国洋大夫，也有前清的太医，药片药丸吃了一箩筐，全然不见效果。</p>
 <p>为了怀上孩子，高小姐不知上了多少炷高香，许了多少宏愿，背地里哭了多少回。</p>
 <p>她的担忧还有一个原因，自出嫁以来，娘家以肉眼可见的速度衰败，不过勉强维持个空架子罢了。</p>
 <p>张家太太好几次有意无意地埋怨老爷徒好虚名，净结交些有名无实的废物。</p>
 <p>高小姐心里清楚，这话就是说给自己听的。</p>
 <p>家门败落，自己再生不出儿子来，往后的日子她不敢想象。</p>
 <p>兰玉儿也替主子着急，但她能有什么办法呢，只能尽量宽慰小姐。</p>
 <p>没过多久，一个意外事件让少奶奶的焦虑直接爆表。</p>
