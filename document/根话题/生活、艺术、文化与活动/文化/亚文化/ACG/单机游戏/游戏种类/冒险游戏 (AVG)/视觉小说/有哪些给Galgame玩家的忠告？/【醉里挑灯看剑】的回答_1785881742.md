# 有哪些给Galgame玩家的忠告？
- 点赞数：17074
- 更新时间：2021年03月22日22时14分34秒
- 回答url：https://www.zhihu.com/question/449766701/answer/1785881742
<body>
 <ol>
  <li data-pid="RX_TsE3v">以下每句话都是各种血泪史</li>
  <li data-pid="PCBLEjJW">如果你认为你是个电脑盲，请你不要入坑。</li>
  <li data-pid="PYuaqVBb">如果你非要入坑，steam是你最好的选择，不要再从其它地方找资源了。</li>
  <li data-pid="-LR5dYdZ">还想入坑，认识个三次元打galgame的朋友是最好的。</li>
  <li data-pid="9X2oCSPZ">找资源的地方有一堆：弟弟花火绯月初音的青葱，不要随意给别人发私信要资源。注册个绯月的号以后用处多多。</li>
  <li data-pid="-7FOGj_k">别人给你资源，是情分；不给，是本分。至少说句谢谢行吗？</li>
  <li data-pid="ullWR1ho">拿到百度云资源后，不要在线解压，求你们了。</li>
  <li data-pid="8bYn3snj">文件后缀名是.part1，.part2……这些的，请全部下载下载，解压第一个文件即可。</li>
  <li data-pid="OWOmlubE">下载下来以后，解压软件请用winRAR。7zip也不错。</li>
  <li data-pid="TcL_lYJT">中文的解压密码，在解压时请勾选“显示密码”后再输入。</li>
  <li data-pid="3Zav3Wrv">我知道幼刀宁宁是你老婆，我知道nekopara天下第一，我知道Just Monika，请少刷一点梗。</li>
  <li data-pid="bptJuiEW">汉化组汉化作品很累，请别举报他们。</li>
  <li data-pid="j0ZQzpPE">不要买汉化资源。</li>
  <li data-pid="Wl99I6Xp">不要录播汉化游戏。</li>
  <li data-pid="tP6NcGS-">万华镜win10打不开，在setup\Package\GameData\Movie里面，把「ロゴ.ogv」改成「尽侨.ogv」，把「万华镜.ogv」改成「枩壶妫.ogv」。</li>
  <li data-pid="h0EH7mwb">学了日语之后，请不要仗着自己会日语看不起汉化玩家。</li>
  <li data-pid="z6VXpGMe">生肉打不开，安装目录改成英文重装一次。</li>
  <li data-pid="Pvf-5vwS">还打不开，请用win7并安装Locale Emulator转区后再运行。</li>
  <li data-pid="KG9witkw">还打不开，请用AppLocale、soraapp、pAppLocale转区后再运行。</li>
  <li data-pid="4YsNlJeq">还打不开，请用NTLEA转区后再运行。</li>
  <li data-pid="6mcVTOSu">还打不开，请安装日文字体包，将时区以及语言全部改成日本。</li>
  <li data-pid="RHJ_rlb5">还打不开，请用XP操作系统。</li>
  <li data-pid="OXdlvX5E">新作生肉打不开，你就知道之前注册了绯月的号的意义。</li>
 </ol>

 </ol>
