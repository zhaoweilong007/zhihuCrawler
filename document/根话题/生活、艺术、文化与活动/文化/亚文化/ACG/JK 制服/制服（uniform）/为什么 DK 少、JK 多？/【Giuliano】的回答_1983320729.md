# 为什么 DK 少、JK 多？
- 点赞数：14795
- 更新时间：2022年04月18日19时52分48秒

 <p data-pid="T6W3IbU-">JK触发器是数字电路触发器中的一种基本电路单元。JK触发器具有置0、置1、保持和翻转功能，在各类集成触发器中，JK触发器的功能最为齐全。在实际应用中，它不仅有很强的通用性，而且能灵活地转换其他类型的触发器。由JK触发器可以构成D触发器和T触发器。</p>

 <p data-pid="Vuww88ba">而JDK是 Java 语言的软件开发工具包，主要用于移动设备、嵌入式设备上的java应用程序。JDK是整个java开发的核心，它包含了JAVA的运行环境（JVM+Java系统类库）和JAVA工具。</p>
 <p data-pid="zNmtNDuD">没有JDK的话，无法编译Java程序（指java源码.java文件），如果想只运行Java程序（指class或jar或其它归档文件），要确保已安装相应的JRE。目前Oracle官网更新的jdk版本到了jdk16。</p>
 <p data-pid="_H5EVsX7">众所周知，java程序的运行过程如下：</p>
 <p data-pid="wHcYTgMA">①Java 源文件—-&gt;编译器—-&gt;字节码文件<br>
  ② 字节码文件—-&gt;JVM—-&gt;机器码</p>
 <p data-pid="wPLgjuEl">而机器码执行的过程，则需要无数数字逻辑电路的协调配合工作，这里面离不开各种时序电路，触发器，寄存器等等。</p>
 <p data-pid="FI1JKyra">jk触发器作为数字电路中应用最广功能最强大的一类触发器(CD4027、MC14027、74109（包括74LS109、74ALS109、74HC109、74HCT109、74AHC109、74AHCT109、74F109等管脚兼容器件），以上型号都是J-K触发器。)自然比jdk要常见的多。</p>
