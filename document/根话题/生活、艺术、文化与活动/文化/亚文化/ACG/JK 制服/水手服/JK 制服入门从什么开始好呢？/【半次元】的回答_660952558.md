 <p data-pid="ryY1_i8j">这是一篇科普+安利向的回答，希望可以帮助美少女们都能买到自己心爱的小裙子❤</p>
 <ul>
  <li data-pid="Bbi4WxZh"><b>什么是JK制服？</b></li>
 </ul>
 <p data-pid="ukGQh8vx">简单来说，JK制服就是<b>日本妹子的校服</b>。</p>
 <p data-pid="zVG7z0YC">JK为日语流行语，意为女高中生（じょしこうこうせい）。取假名音 jyoshikoukousei中的J和K。</p>
 <ul>
  <li data-pid="eSkMDhFc"><b>JK制服的分类？</b></li>
 </ul>
 <p data-pid="wv_GtNsL">JK制服的种类繁多。包括背心裙，夏服，中间服，马甲，西装，大衣，冬服，开衫，毛衣，衬衫等等。其实就跟中国的校服一样，每个地区和学校都会设计自己的校服，而日本高中生会根据温度进行不同的搭配，所以才诞生了这么多种类。</p>
 <ul>
  <li data-pid="EWHTd6Sz"><b>关于正统？</b></li>
 </ul>
 <p data-pid="VebA3eQx">这里的正统指的是“版型正统”，虽然JK制服种类繁多，但也有固定的制式。大多数正统的JK制服，布料挺括，襟线不反光，版型不收腰，裙子的褶子锋利不易皱。淘宝上一搜一大把的基本上都不是正统的JK制服。</p>

 <ul>
  <li data-pid="RJOEQDdD"><b>关于襟线反光？</b></li>
 </ul>
 <p data-pid="qO9_-5mX">正统制服的襟线不是一定不反光，因为襟线的材质（人造丝、涤纶、腈纶）以及织法的不同会使襟线具有一定的光泽感，</p>
 <p data-pid="rBh7CC6A">一般说襟线反光是指制作襟线的材料使用的是不反光，没有织纹的丝带或者螺纹织带，所以要辨别比较靠谱的方式是看襟线的<b>织纹，一般来说</b>织纹都是竖纹</p>
 <ul>
  <li data-pid="JPsd4NfW"><b>关于襟线的种类和颜色？</b></li>
 </ul>
 <p data-pid="Zb1KWNrR">襟线样式大致可以分为三种：蝮蛇线，山道线，编织襟线</p>
 <p data-pid="z3vvzVSs">几条襟线就是几本，比如无本、一本线、二本线、三本线，一粗一细的是亲子线</p>

  <li data-pid="YOjUJWAX"><b>水手服的种类？</b></li>
 </ul>
 <p data-pid="Ub-d21W4">水手服根据领子的不同可以分为札幌襟、关东襟、关西襟、名古屋襟等</p>

 <p data-pid="1P6dGgzK">常见搭配有：①水手服+百褶裙 ②衬衫+百褶裙or连衣裙，视季节增加马甲/背心+外套。③幼稚园装、体操服等其他制服。</p>
 <ul>
  <li data-pid="rsepoolJ"><b>关于制服裙?</b></li>
 </ul>
 <p data-pid="MvF1z9EI">主要可以分为<b>连衣裙，背带裙，半身裙</b>；裙褶主要可以分为①<b>轮褶/车轮褶</b>；②<b>前箱褶</b>；③<b>箱褶</b></p>
