# 你有没有因为有纹身被特殊对待过？
- 点赞数：5323
- 更新时间：2021年01月29日18时48分05秒

 <p data-pid="fu19wolH">因为我的纹身是在大臂，所以就算穿短袖也看不到，只有去健身房穿无袖背心时会露出来，健身的时候经常会有人因为好奇一直盯着我的纹身看，有一天有个男的盯着我手臂上的纹身看了很久，突然走过来问我这个是纹的还是帖的，我脑海里先是“啊?!”然后告诉他这是纹的，结果他说了一句“挺可爱的”。。。。挺可爱的。。。可爱的。。的。</p>
