# 有哪些比较小众的成语？
- 点赞数：21171
- 更新时间：2015年10月18日18时01分41秒
- 回答url：https://www.zhihu.com/question/21192202/answer/21400230
<body>
 <p data-pid="-fyxj1_u">既然是用来装逼，那就不能太生僻。最好是那种日常能用得上、但又不常见的。我对这类成语很感兴趣，一直留心收集。如下：</p>
 <ul>
  <li data-pid="HCF44v2V"><b>甘棠遗爱</b> 旧时颂扬离去的地方官。<br></li>
 </ul>
 <p data-pid="hhHld-EZ">《诗经·周南·甘棠》：“蔽芾甘棠，勿翦勿伐，召伯所茇；蔽芾甘棠，勿翦勿拜，召伯所说。”</p>
 <ul>
  <li data-pid="nRIUeuM4"><b>蚌病生珠</b> 比喻因不得志而写出好文章来。同“蚌病成珠”。</li>
 </ul>
 <p data-pid="JNLRWGqW">高燮《题蔡哲夫所绘沈孝则〈冰雪庐图〉即步哲夫韵》：“嗟哉蚌病乃生珠，诗渐可读消雄图。”</p>
 <ul>
  <li data-pid="wEYF2IHX"><b>鹤短凫长</b> 比喻颠倒是非，妄加评论。<br></li>
 </ul>
 <p data-pid="1mJV8TYF">《庄子·骈拇》：“凫胫虽短，续之则忧；鹤胫虽长，断之则悲。”</p>
 <ul>
  <li data-pid="tyvcNc9z"><b>诗无达诂</b> 诗词没有通达的解释。 <br></li>
 </ul>
 <p data-pid="PUEtl1aS">董仲舒《春秋繁露》卷三《精华》</p>
 <ul>
  <li data-pid="AtQ1iJC1"><b> 聊备一格</b> 姑且算作一种风格或格式。 <br></li>
 </ul>
 <p data-pid="dInGgYqE">清·陈廷焯《白雨斋词话》第五卷：“余于别集中求其措语无害大雅者择录一二，非赏其工也，聊备一格而已。”</p>
 <ul>
  <li data-pid="6eh777AS"><b>功不唐捐</b> 世界上的所有功德与努力,都是不会白白付出的,必然是有回报的. <br></li>
 </ul>
 <p data-pid="Y_Fx3IYA">《法华经·观世音菩萨普门品》:"若有众生、恭敬礼拜观世音菩萨，福不唐捐，是故众生皆应受持观世音菩萨名号"</p>
 <ul>
  <li data-pid="ENw7GAKO"><b>悔读南华</b> 《南华经》即《庄子》。比喻学识渊博而不为人所容。 <br></li>
 </ul>
 <p data-pid="PUsWgwy7">《唐诗纪事·温庭筠》：“因知此恨人多积，悔读南华第二篇。”</p>
 <p data-pid="2haQ8euL">陆游《怀镜中故庐》：从宦只思乘下泽，忤人常悔读南华。</p>
 <ul>
  <li data-pid="DKhcRDU0"><b>叩桥不渡</b> 行事小心谨慎。<br></li>
 </ul>
 <p data-pid="844L9bL3">1969年时，在第8期（旧）名人战结束后，日本著名的棋手高川格夸奖同样也是日本当时最顶尖的华裔棋手林海峰的话。</p>
 <ul>
  <li data-pid="2LFQiU4A"><b>野人献曝</b> 比喻贡献的不是珍贵的东西。 <br></li>
 </ul>
 <p data-pid="sodyUNrp">《列子·杨朱》:"昔者宋国有田夫，常衣缊黂，仅以过冬。暨春东作，自曝于日，不知天下之有广厦隩室，绵纩狐貉。顾谓其妻曰:'负日之暄，人莫知者。以献吾君，将有重赏。'"</p>
 <ul>
  <li data-pid="Wnrdnirm"><b>五日京兆</b> 比喻任职时间不会长，或凡事不作久长打算。 <br></li>
 </ul>
 <p data-pid="0kPR_754">《汉书》卷七十六〈赵尹韩张两王列传·张敞〉「吾为是公尽力多矣，今五日京兆耳，安能复案事？」「五日京兆竟何如？冬已尽，延命乎？」</p>
 <ul>
  <li data-pid="opzN3jDL"><b>见猎心喜</b> 看见别人在做的事正是自己过去喜好的，不由得心动，也想试一试。 <br></li>
 </ul>
 <p data-pid="-rD1dIyR">三国·魏·曹丕《典论自序》：“和风扇物，弓燥手柔，草浅兽肥，见猎心喜。”</p>
 <ul>
  <li data-pid="uKcuvVmM"><b>行不由径</b> 走路不抄小道。比喻为人正直，行动正大光明。 <br></li>
 </ul>
 <p data-pid="7-fxLQxc">《论语·雍也》：“有澹台灭明者，行不由径，非公事，未尝至于偃之室也。”</p>
 <ul>
  <li data-pid="lP50ZpjC"><b>觚不觚</b> 比喻事物名实不符。 <br></li>
 </ul>
 <p data-pid="3zVozo9v">《论语·雍也》：“觚不觚，觚哉！觚哉！”</p>
 <ul>
  <li data-pid="yPIp-dSd"><b>养寇自重</b> 故意留着敌对势力，用以牵扯另一方，以此使自己的地位变得重要。 <br></li>
  <li data-pid="TG9FgwA6"><b>被褐怀玉</b> 身穿粗布衣服而怀抱美玉。比喻虽是贫寒出身，但有真才实学。 <br></li>
 </ul>
 <p data-pid="bh3L0YI0">《老子·德经·七十章》吾言甚易知，甚易行。天下莫能知，莫能行。言有宗，事有君，夫唯无知，是以不我知。知我者希，则我者贵。是以圣人被褐而怀玉。</p>
 <ul>
  <li data-pid="RtLRnSR9"><b>玉友金昆</b> 友、昆：指兄弟。对他人兄弟的美称。 <br></li>
 </ul>
 <p data-pid="XwRVbWST">北朝·魏·崔鸿《十六国春秋·前凉录·辛攀》：“辛攀，字怀远，陇西狄道人也。兄鉴旷，弟宝迅，皆以才识著名。秦、雍为之谚曰：‘三龙一门，金友玉昆。’”</p>
 <ul>
  <li data-pid="CgStGvyo"><b>共挽鹿车</b> 挽：拉；鹿车：古时的一种小车。旧时称赞夫妻同心，安贫乐道。 <br></li>
 </ul>
 <p data-pid="Mnx962i4">《后汉书·鲍宣妻传》：“妻乃悉归侍御服饰，更著短布裳，与宣共挽鹿车归乡里。”</p>
 <ul>
  <li data-pid="5sUlZo96"><b>明扬侧陋</b> 侧陋指地位低下而才德兼备的贤人。明察荐举出身微贱而德才兼备的人。 <br></li>
 </ul>
 <p data-pid="cDsZ9nU-">《尚书·尧典》：“明明，扬侧陋。”</p>
 <ul>
  <li data-pid="XjSO_Fqh"><b>斗重山齐</b> 像北斗星一样令人敬仰，像泰山一般崇高。形容人才德兼备，享有崇高威望。 <br></li>
 </ul>
 <p data-pid="4AwbfoNU">明·胡文焕《群音类选·》：“痛追思旧日相随，德业文章，斗重山齐。”</p>
 <ul>
  <li data-pid="eEV9evDU"><b>口吻生花</b> 比喻谈吐文雅/比喻吟诗得意，兴趣极浓。 <br></li>
 </ul>
 <p data-pid="Td9GGs1H">唐 冯贽《云仙杂记》第五卷引《白氏金锁》：“张祜苦吟，妻孥唤之不应，以责祜。祜曰：吾方口吻生花，岂恤汝辈。”</p>
 <ul>
  <li data-pid="OAOtWhWA"><b>与世偃仰</b> 指没有一定的主张，随大流。<br></li>
 </ul>
 <p data-pid="XUf85Eg6">荀况《荀子·非相》：“与时迁徙，与世偃仰。”</p>
 <ul>
  <li data-pid="JhoiP4nD"><b>耳食之论</b> 耳食:耳朵吃饭。指听来的没有确凿根据的话。<br></li>
 </ul>
 <p data-pid="bDe0vCaT">司马迁《史记·六国年表序》：“学者牵于所闻，见秦在帝位日浅，不察其终始，因举而笑之，不敢道，此与以耳食无异。”</p>
