# 有个太帅的室友怎么办？
- 点赞数：7947
- 更新时间：2020年07月28日19时26分18秒
- 回答url：https://www.zhihu.com/question/402370540/answer/1297605071
<body>
 <p></p>
 <p data-pid="SkLOF1Js">我的室友很帅，帅的惊天动地，惨绝人寰。</p>
 <ol>
  <li data-pid="dzUqXs6e">第一眼我就被他迷住了，真的。比起像我们一样五大三粗的模样，这个室友简直跟个模特似的，精致的得不得了。</li>
  <li data-pid="ppDDeDJL">记得有一天，军训回来，我们几个都是臭汗淋漓，整个脸都是红彤彤的，就他一个人流的汗都是香的，我们几个还没混熟，不好意思凑那么近闻，但是已经无法忍受自己的臭味了，纷纷奔向了澡堂，没有问他那娘们唧唧的味道是怎么回事。后来才知道那是因为他用了他妈的防晒霜，香氛型。</li>
  <li data-pid="LUmK2LD4">帅哥在军训时就挺出名了，表白的小姐姐挺多的，但他都拒绝了，于是帅哥的高冷传出去，表白的人少了不少。但是据我观察他其实是一个内向的逗比。</li>
  <li data-pid="43tkYtFF">他的帅理所当然的成为了我们寝室的室草甚至是班草、院草，校草评选还没出来，但估计就是他了。嗯，骄傲自豪，哈哈哈。</li>
  <li data-pid="vr6b_Y7c">帅哥每天都会锻炼身体，有时候早上跑步，有时候晚上跑步，根据课程表来安排这一天的行程。每天不知道从哪知道帅哥课程表的小姐姐等在哪里邂逅他，我们几个哥们想去碰碰运气，看有没有看上他们的，就跟着跑了几天，结果都没坚持下来。</li>
  <li data-pid="NC4BanNO">但是我坚持下来了，因为我想减肥。</li>
  <li data-pid="Lh5_S-vY">帅哥于是每天下楼跑步都会叫上我。</li>
  <li data-pid="PWKLxK4L">又有小姐姐来给他送水了，但帅哥拒绝了，说自己带了水杯，帅哥真的话很少诶 ，没接上话，小姐姐尴尬的走了，帅哥还一脸懵逼的看着我。</li>
  <li data-pid="4-UQ6Ycv">我说：你这么帅应该会习惯女生搭讪吧。帅哥说：不习惯，我没觉得自己很帅，高中都没人和我表白过，因为我非常非常非常胖。</li>
  <li data-pid="YYBekHje">看不出来，这一个暑假咋减的 ，我得好好问问，说不定我也是个帅哥，但是被埋没了？</li>
  <li data-pid="bfXF8CeM">帅哥在前面跑，长腿迈得飞快，我在后面气喘吁吁，减肥明天再说吧，我先撤了。</li>
  <li data-pid="mqBMPsGD">又来了，我们几个室友跟着帅哥一起去上课，无数的目光盯着帅哥顺带扫了一下我们，面露不解，可能觉得为什么帅哥会和我们这种普通的人在一起玩，而不是和帅哥们在一起吧。习惯了，who care？</li>
  <li data-pid="UoAAsfFj">我们的座位经常在倒数第二排，好位置 感谢大家空出来，但是帅哥不知道是特意空出来的，还一直奇怪为什么老是坐在第二排。因此，逃不了课。</li>
  <li data-pid="mz5ziPrK">都怪老师也不是个傻子，我们这几个经常一起上课的，那么明显，突然没了不是明晃晃的告诉老师旷课了吗。帅哥和我们几个室友，从来都没有旷过课。嗯，好好学习天天向上，吧。</li>
  <li data-pid="lvEz1sao">帅哥太招风，他自己也慢慢觉得自己很招风，问我们怎么办。我们作为他的无敌好室友，当然得帮他像个法子。室友A说：要不你带个帽子或者口罩？室友B说：要不你办丑一点？我说:emmmmm</li>
  <li data-pid="UI8KCWza">帅哥毅然决然的（根本就没有选择）选了第一个。他带上口罩和帽子，我们几个莫名的觉得又tm的好帅。</li>
 </ol>
