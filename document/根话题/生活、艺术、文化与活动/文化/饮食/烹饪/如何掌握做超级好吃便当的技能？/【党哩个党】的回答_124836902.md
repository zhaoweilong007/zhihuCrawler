 <p data-pid="BECbCuPc">学生一枚，每天中午一个小时休息时间，是肯定没空回家吃饭的，于是每天都带一份便当。</p>
 <p data-pid="pvahoV7C">基本都是头一天晚上做好，耗时也控制在半小时以内。</p>
 <p data-pid="XVzKwO9P">很多菜谱经过简化，味道折扣不大，但是做起来要简单得多。</p>
 <p data-pid="IPLkGxUx">吃的时候会在学校的微波炉热一下，最好热透，毕竟隔夜的东西。</p>
 <p data-pid="e_HM820y">因为一个人住，一个人吃所以每次菜品相对简单，免得剩下的原料放在冰箱里一会儿就坏了。</p>
 <p data-pid="Ke7bXvGW">便当盒就是宜家最普通得玻璃便当盒，不算能装，觉得不够吃的同学可以买大号的便当盒和圆形的便当盒哦。质量各方面都过关，唯一缺点就是不能分隔食物，我一般就用米饭做隔断或者拿锡箔纸包住多汁的食物啦！</p>

 <ul>
  <li data-pid="YubvLdHs"><b>咖喱乌冬面便当</b><br></li>
 </ul>
 <ol>
  <li data-pid="IPbrtdWe">少许油将洋葱碎炒香，加入咖喱块融化，加水煮沸，加入胡萝卜块与土豆块煮至咖喱浓稠</li>
  <li data-pid="fn_ymSqG">乌冬面滚水烫熟，用凉水冲洗沥干，浇上咖喱即可</li>
 </ol>
 <figure>
  <img src="https://picx.zhimg.com/50/266776f2f4e38b100ae7e4c3062693ee_720w.jpg?source=1940ef5c" data-rawwidth="3000" data-rawheight="2250" data-original-token="266776f2f4e38b100ae7e4c3062693ee" class="origin_image zh-lightbox-thumb" width="3000" data-original="https://picx.zhimg.com/266776f2f4e38b100ae7e4c3062693ee_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="GxXp7QsV"><b>什锦咖喱炒饭</b></li>
 </ul>
 <p data-pid="e7D_otCC">（如果咖喱吃不完用来做炒饭也很棒！）</p>
 <ol>
  <li data-pid="OVZWdBrK">锅内少许热油炒香什锦丁，加入米饭翻炒</li>
  <li data-pid="lc9c8TUb">剩余的咖喱汁加入米饭翻炒均匀，加盐出锅</li>
 </ol>
 <figure>
  <img src="https://picx.zhimg.com/50/27c98d5e3da5a9b26541baed2ca9d754_720w.jpg?source=1940ef5c" data-rawwidth="1920" data-rawheight="1440" data-original-token="27c98d5e3da5a9b26541baed2ca9d754" class="origin_image zh-lightbox-thumb" width="1920" data-original="https://picx.zhimg.com/27c98d5e3da5a9b26541baed2ca9d754_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="_KODgvJ5"><b>三文鱼生饭</b></li>
 </ul>
 <p data-pid="h6xEDvwY">（我的最爱，吃完你会哭的）</p>
 <ol>
  <li data-pid="GaVkEe69">米饭蒸熟，加入寿司醋拌匀放凉（没有寿司醋用白醋，糖，盐按3:2:1比例调匀即可）</li>
  <li data-pid="CDYLuhcJ">新鲜三文鱼去皮切块</li>
  <li data-pid="GD3mO7vf">加入生抽，麻油，撒上葱花即可</li>
 </ol>
 <figure>
  <img src="https://picx.zhimg.com/50/ba4522480f1d6fcd36d3fe41b0ad8e7f_720w.jpg?source=1940ef5c" data-rawwidth="3000" data-rawheight="2250" data-original-token="ba4522480f1d6fcd36d3fe41b0ad8e7f" class="origin_image zh-lightbox-thumb" width="3000" data-original="https://picx.zhimg.com/ba4522480f1d6fcd36d3fe41b0ad8e7f_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="4FFTkfF-"><b>三文鱼炒饭</b></li>
 </ul>
 <p data-pid="CS_qODLe">如果前一天的三文鱼生饭没吃完怎么办呢，炒一炒也很好吃哦，毕竟生的三文鱼容易坏。</p>
 <ol>
  <li data-pid="inDcigaD">热锅下三文鱼生饭，不用添加任何东西，炒至鱼肉熟即可</li>
 </ol>
 <figure>
  <img src="https://pica.zhimg.com/50/ae97c18c16d85b6b7c7dc875f085225d_720w.jpg?source=1940ef5c" data-rawwidth="4128" data-rawheight="3096" data-original-token="ae97c18c16d85b6b7c7dc875f085225d" class="origin_image zh-lightbox-thumb" width="4128" data-original="https://pic1.zhimg.com/ae97c18c16d85b6b7c7dc875f085225d_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="oMN5uWHt"><b>黑椒牛排饭</b></li>
 </ul>
 <ol>
  <li data-pid="QnX3XaaW">牛排一块，撒上盐，黑胡椒，用刀背拍散</li>
  <li data-pid="cpwnkyRE">李锦记叉烧酱两勺，蚝油一勺，蜂蜜一勺，混合均匀后腌渍牛排十分钟</li>
  <li data-pid="7tFZULDz">锅内化开黄油，牛排两面个各煎20秒</li>
  <li data-pid="r8fzFqUV">烤箱预热150度，牛排入烤箱烤10分钟</li>
 </ol>
 <figure>
  <img src="https://picx.zhimg.com/50/534fb2385496779ddb1f6e9714503ca6_720w.jpg?source=1940ef5c" data-rawwidth="3000" data-rawheight="2250" data-original-token="534fb2385496779ddb1f6e9714503ca6" class="origin_image zh-lightbox-thumb" width="3000" data-original="https://picx.zhimg.com/534fb2385496779ddb1f6e9714503ca6_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="fmXZ2Aaw"><b>金枪鱼三明治＋照烧青花鱼</b></li>
 </ul>
 <ol>
  <li data-pid="DSFHjFer">锅内烧热少许油，打入鸡蛋迅速关火，盖上盖子焖一分钟</li>
  <li data-pid="fbqXJNV9">面包两片，加入金枪鱼罐头，蔬菜，番茄，煎鸡蛋，沙拉酱。切成两半</li>
  <li data-pid="g4OkS6WE">青花鱼去骨切块，锅内烧油煎青花鱼至微焦</li>
  <li data-pid="G__uYDej">加入生抽，味淋，水烧开。大火收汁</li>
 </ol>
 <figure>
  <img src="https://pic1.zhimg.com/50/7f3366882108c15f709d92d38af43e95_720w.jpg?source=1940ef5c" data-rawwidth="4128" data-rawheight="3096" data-original-token="7f3366882108c15f709d92d38af43e95" class="origin_image zh-lightbox-thumb" width="4128" data-original="https://picx.zhimg.com/7f3366882108c15f709d92d38af43e95_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="8PrXlrcX"><b>照烧鸡肉饭</b></li>
 </ul>
 <ol>
  <li data-pid="1v_o3HK8">鸡腿肉一块，用厨房纸擦干水分</li>
  <li data-pid="uEU_OJgv">锅内下少许油烧热，将鸡腿肉带皮面朝下，煎至鸡皮焦黄</li>
  <li data-pid="5t8-V83x">生抽，味淋（用料酒和糖混合即可代替），水调汁倒入锅内，没过鸡肉</li>
  <li data-pid="Ur8eN4AI">煮至汤汁较少时大火收汁（不停翻动避免糊锅），起锅切块即可</li>
 </ol>
 <figure>
  <img src="https://picx.zhimg.com/50/8ebeb6476cf6e353efdb8d7701b0be33_720w.jpg?source=1940ef5c" data-rawwidth="3000" data-rawheight="2250" data-original-token="8ebeb6476cf6e353efdb8d7701b0be33" class="origin_image zh-lightbox-thumb" width="3000" data-original="https://pic1.zhimg.com/8ebeb6476cf6e353efdb8d7701b0be33_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="w12P7Vg8"><b>鱼香茄子饭</b></li>
 </ul>
 <ol>
  <li data-pid="gIphpGg-">滚刀切茄子成块，表面裹上淀粉</li>
  <li data-pid="CE1rQhm_">锅内热油，下豆瓣酱，姜蒜末，糖炒出红油，加入茄子翻炒均匀。</li>
  <li data-pid="Lvzx0jFo">加入少许水，大火收汁，加醋，加盐起锅</li>
 </ol>
 <figure>
  <img src="https://pic1.zhimg.com/50/e97eee5a59e2299a20dc1f1626e423fb_720w.jpg?source=1940ef5c" data-rawwidth="4096" data-rawheight="3072" data-original-token="e97eee5a59e2299a20dc1f1626e423fb" class="origin_image zh-lightbox-thumb" width="4096" data-original="https://picx.zhimg.com/e97eee5a59e2299a20dc1f1626e423fb_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="HMPl4P72"><b>麻辣香锅饭</b></li>
 </ul>
 <ol>
  <li data-pid="amnPrHhS">土豆，西兰花，鱼丸，洋葱等食材用热水焯熟，肉用油炒熟盛出待用</li>
  <li data-pid="WR7h4D0v">锅内烧油，加入少许大料，姜片，蒜末，干辣椒段，干花椒煸香。加入豆瓣酱和大酱炒出红油</li>
  <li data-pid="6UiksAmw">食材倒入锅内，加入老抽，少许白糖翻炒均匀。起锅加盐。</li>
 </ol>
 <figure>
  <img src="https://picx.zhimg.com/50/2312a4d26922a467b49209254b6018b5_720w.jpg?source=1940ef5c" data-rawwidth="3000" data-rawheight="2250" data-original-token="2312a4d26922a467b49209254b6018b5" class="origin_image zh-lightbox-thumb" width="3000" data-original="https://picx.zhimg.com/2312a4d26922a467b49209254b6018b5_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="tBeevQAK"><b>鱼香烘蛋饭</b></li>
 </ul>
 <ol>
  <li data-pid="KLZe6KMW">鸡蛋两个打散。水淀粉按照水：粉＝1:1的比例调制。在蛋液中倒入调好的淀粉水搅拌均匀，留少许淀粉水备用</li>
  <li data-pid="QhQtb0sb">锅烧油至5成热，倒入蛋液。先开大火至蛋液边缘鼓起，调小火开始烘。一面基本凝固时翻面继续用小火烘</li>
  <li data-pid="RvLol8jR">。锅内留底油，将葱、姜、蒜末，少许豆瓣酱煸炒，加入醋，糖，水</li>
  <li data-pid="ijSJA7-B">大火烧开，勾入芡汁。淋在烘蛋上即可。</li>
 </ol>

 </figure>
 <ul>
  <li data-pid="0cbUgHzj"><b>卤肉饭</b></li>
 </ul>
 <ol>
  <li data-pid="aORYXw8Y">锅内用热油将洋葱碎炒至透明微焦，加入猪五花肉，少许糖煸炒，</li>
  <li data-pid="bP08h91V">加入大料（山奈，八角，香叶等），适量水，小火慢炖</li>
  <li data-pid="PczhJO0v">最后大火收汁即可</li>
  <li data-pid="NeYJVA5F">鸡蛋用滚水煮五分钟，捞起浸入冰水中。降温后剥皮放入卤水过夜即可。</li>
 </ol>

 </figure>
 <ul>
  <li data-pid="We1G4pbB"><b>水煮肉饭</b></li>
 </ul>
 <ol>
  <li data-pid="hjyraezb">蔬菜用少许油炒匀垫底</li>
  <li data-pid="JQK7utkV">猪里脊肉切成薄片，用少许盐，料酒，水淀粉腌制</li>
  <li data-pid="2I5T9Zh7">锅内烧油炒香豆瓣酱，加水，老抽，盐，大料煮开</li>
  <li data-pid="EQ69U0qt">下肉片煮熟</li>
  <li data-pid="zLnLVf9n">起锅撒上干辣椒，干花椒，姜蒜粒。烧滚油淋上即可</li>
 </ol>

 </ul>
 <ol>
  <li data-pid="jCnlbAo4">五花肉煮熟放凉，切片。黄瓜切片</li>
  <li data-pid="fOVeWVMp">生抽，醋，盐，糖，姜蒜，辣椒油，芝麻油调匀浇上即可</li>
 </ol>

 <ul>
  <li data-pid="r2apSPDw"><b>红烧肉饭</b></li>
 </ul>
 <ol>
  <li data-pid="e4K-Kp64">锅内烧热油，下白糖炒至焦糖色。加入五花肉块翻炒上色</li>
  <li data-pid="s_1fN78Q">加入大料，水烧开炖煮。大火收汁，起锅加盐</li>
 </ol>

 <ul>
  <li data-pid="GglsZjQZ"><b>回锅肉饭</b></li>
 </ul>
 <ol>
  <li data-pid="AtinP-OP">五花肉煮熟切片，锅内热油加入五花肉煸炒至打卷，盛出备用</li>

  <li data-pid="tJj24Euo">加入青椒，五花肉煸炒，少许酱油上色。起锅加盐</li>
 </ol>

 <ul>
  <li data-pid="mn6NVJmG"><b>糖醋排骨＋培根煎芦笋</b></li>
 </ul>
 <ol>
  <li data-pid="bQgnchdK">排骨焯熟沥水，锅内热油加糖炒糖色。加入排骨翻炒上色</li>
  <li data-pid="EVUFfHey">锅内加入水，醋炖煮至排骨软烂，大火收汁。</li>
  <li data-pid="4WKK_zmq">起锅加盐，再次加醋</li>
  <li data-pid="oCqCIXiY">芦笋切断用培根裹紧，用油煎香，撒少许盐即可</li>
 </ol>

 <ul>
  <li data-pid="bMJbRvU7"><b>口水鸡饭</b></li>
 </ul>
