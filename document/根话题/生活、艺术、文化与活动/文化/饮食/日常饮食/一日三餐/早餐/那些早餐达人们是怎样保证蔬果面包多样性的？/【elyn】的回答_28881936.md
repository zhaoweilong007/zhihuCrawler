# 那些早餐达人们是怎样保证蔬果面包多样性的？
- 点赞数：5375
- 更新时间：2015年03月23日18时30分03秒

<body>
 <p data-pid="dev_ptxb">谢邀... 具体很多早餐的做法都发在我的公众号<b>tianluo_hhhaze</b>了，欢迎关注。</p>
 <p data-pid="lqz3OHdT">题主标题说的蔬菜水果确实相对难处理一点，主食的面包实在是太方便了。。分开来说：</p>
 <ul>
  <li data-pid="PDQR647q"><b>面包的保存</b><br></li>
 </ul>
 <p data-pid="XkBI2RoP">我是自己做面包的，市售的面包也可以一样处理。但是这个办法不太适合带馅儿的尤其是什么奶酪馅肉松馅的面包，比较适合主食面包的保存。要点是：</p>
 <ol>
  <li data-pid="dyYZF3UX">烤好的面包如果确定能够在一两天之内吃完，那么用保鲜膜或者保鲜袋包裹<b><u>密封好，避免水分蒸发</u></b>就可以了。不包好的话面包容易变干，但是注意<u><b>要凉透</b></u>了才能做密封处理，否则容易长霉。<br></li>
  <li data-pid="4lkHF9k6">如果需要长时间保存的话，凉透密封后<b><u>放冷冻柜冻起来</u></b>。绝对不可以在冰箱冷藏柜！面包会老化，口感很差。<br></li>
  <li data-pid="eX6537ft">冷冻过的面包在吃之前如何处理？</li>
 </ol>
 <p data-pid="bQl76cfu">我一般是用最省事儿的办法，就是<u><b>提前一晚拿出来自然解冻</b></u>，早上直接吃。不过不管是直接吃还是复烤，这种冷冻过的面包最好都不要再次冷冻了，会影响口感。</p>
 <p data-pid="_BKNcJ9l">如果用烤箱的话，可以预热好150度加热个五分钟，温度和时间根据自己家烤箱的脾气调整一下。烤箱一定要先预热，否则预热过程中也会让面包丧失水分。</p>
 <p data-pid="Cl_ew3JY">如果是硬质欧包之类的，进烤箱加热之前最好在表面喷一点水，口感会更好。</p>

 <ul>
  <li data-pid="knursE3S"><b>保证蔬菜水果多样性的技巧</b><br></li>
 </ul>
 <p data-pid="-e3n1txY">我觉得有这么几个办法：</p>
 <ol>
  <li data-pid="2rar9kET"><b><u>少买点儿。</u></b>就买一个桃一个苹果，水果摊老板也不会把你怎么样啊。。。每种买一点点就可以多买几样啦。虽然菜市场的摊贩喊价都是多少钱一斤，但是你可以不买到一斤啊。。。比如豆芽是一两块钱一斤，好像价格很便宜了不买一斤说不过去？完全没关系，就买一块钱的，还嫌多的话买五毛钱的。。他要不肯卖总有人肯卖</li>
  <li data-pid="erSZfSaN"><b><u>让老板搭售。</u></b>比如买了两根丝瓜，说老板送两根葱吧，实在不行的话那我只买五毛钱的葱可以吗。。<br></li>
  <li data-pid="MO4j5ZBl"><u><b>买尺寸小一点的蔬菜。</b></u>不要小看这一点，一个大土豆和一个中型土豆的饱腹感还是差挺多的</li>
  <li data-pid="t4BrnapR"><b><u>搞清楚每种食材适合的保存方式和保存时间。</u></b>拿水果来说，苹果保存时间就比较长呀，桃的保存时间相对短一点，西瓜要是不切开的话也可以多放几天。就算一次多买了，也可以从保存时间短的吃起。拿蔬菜来说，土豆洋葱玉米保存时间就很长呀，西兰花这种就短一点，放久了容易变黄<br></li>
  <li data-pid="wobEFTdh"><b><u>注意保存方法。</u></b>随便举个例子，像小葱这种每次只用两三根但是一般都得买一把的东西来说，买回家之后晾干一下，然后用保鲜袋装起来放在零度保鲜那一层，放一周是没啥问题的。但是如果湿哒哒地放冰箱，就很容易烂掉。还有蘑菇这种蔬菜买回家最好马上放冰箱，一两天之内吃掉，否则非常容易长毛。</li>
  <li data-pid="fuLHXyB1"><b><u>多做几顿饭。</u></b>我除了早餐之外，中午也是带饭的。有时候晚上做准备工作的时候，会把早上需要的食材预留出来，其它的就拿来做便当<br></li>
 </ol>
 <p data-pid="4zTaFCoX">其实有些东西确实是摆一半自己默默吃掉另一半的，比如牛油果！！！太容易氧化，没办法分开两次用。如果摆盘只用半个的话，另外半个会自己开心地吃掉..</p>
