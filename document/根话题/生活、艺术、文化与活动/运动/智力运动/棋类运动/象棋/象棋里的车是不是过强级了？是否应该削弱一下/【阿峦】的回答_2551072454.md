# 象棋里的车是不是过强级了？是否应该削弱一下?
- 点赞数：8619
- 更新时间：2022年06月29日19时24分16秒
- 回答url：https://www.zhihu.com/question/426985535/answer/2551072454
<body>
 <p data-pid="AM4UxyjP">中国象棋 1.01b</p>
 <ul>
  <li data-pid="KWPi9u3K">降低了「車」的移动距离，每次移动「車」时，上限为 5 步。</li>
 </ul>
 <p data-pid="T4fDWczY">中国象棋 1.3</p>
 <ul>
  <li data-pid="9hwsf9XI">调整「車」的移动机制：「車」可以移动任意距离，但每次移动大于等于 6 步时，进入「过热」状态，3 回合内每次移动此「車」时上限为5步。</li>
 </ul>
 <p data-pid="FAVBUdEu">中国象棋 1.31</p>
 <ul>
  <li data-pid="YxfhqrEi">削弱「車」进入「过热」状态的条件，只要「車」不处于「过热」状态，那么只要移动「車」，此「車」就立即进入「过热」状态，持续 3 回合。</li>
 </ul>
 <p data-pid="xQFzbf8l">中国象棋 1.31c</p>
 <ul>
  <li data-pid="aTV1ZASW">调整「車」的过热状态持续回合数为 5 回合。</li>
 </ul>
 <p data-pid="Ll8Kalff">中国象棋 2.00</p>
 <ul>
  <li data-pid="3YZFa_dV"><b>移除「車」</b>。</li>
 </ul>
 <p data-pid="LEkS-F9H">中国象棋 2.01</p>
 <ul>
  <li data-pid="C0ia2OYl">「車」重新加入棋盘。</li>
  <li data-pid="ofupRn4w">新限制：开局前 10 回合不得移动「車」，「車」也无法被吃。</li>
 </ul>
 <p data-pid="Hx7kFTi9">中国象棋 2.10</p>
 <ul>
  <li data-pid="R1jOaiy2">调整「車」的机制：「車」后退时永远最多只能后退 5 步。</li>
 </ul>
 <p data-pid="1eBl0zlB">中国象棋 2.45</p>
 <ul>
  <li data-pid="qgjabb_T">调整「車」的机制：「車」横向移动时永远最多只能移动 5 步。</li>
  <li data-pid="BFd8tpNZ">取消与「过热」相关的规则。</li>
 </ul>
 <p data-pid="MKjHLoSY">中国象棋 3.0</p>
 <ul>
  <li data-pid="ybjTq72u">取消「車」在所有情况下移动步数的限制。</li>
  <li data-pid="gjIHzVn9">「車」吃子后，该方所有的「車」下回合都不得移动，但可以被吃，这个效果仅下回合生效。</li>
 </ul>
 <p data-pid="6KgQ1L8s">中国象棋 4.0</p>
 <ul>
  <li data-pid="7eNAjWCf">新增机制：「車」吃掉「車」时，自己也会被摧毁。</li>
 </ul>
 <p data-pid="uURDdGJK">中国象棋 5.0</p>
 <ul>
  <li data-pid="Fg25r2ZC">取消「車」吃「車」自毁机制。</li>
  <li data-pid="qp5uRwBH">取消「車」吃子后不能移动的机制。</li>
  <li data-pid="T5Aulp_h">新增机制：「車」吃子后该棋子获得「修车」debuff，下回合只能最多移动 3 格；若吃子后下回合连续吃子，则再下回合不能移动。</li>
  <li data-pid="43W5ATyx">若己方有棋子处于「修车」debuff 影响，则己方所有「兵」（卒）都不能移动。</li>
 </ul>
 <p data-pid="Iir3dSp9">中国象棋 5.01a</p>
 <ul>
  <li data-pid="K_eUcqz-">取消「修车」debuff 影响己方「兵」（卒）移动的机制。</li>
 </ul>
 <p data-pid="ciyQKHMt">中国象棋 5.1</p>
 <ul>
  <li data-pid="R4QHnI0u">调整「修车」debuff 的产生条件：现在开局的前 15 回合不会产生「修车」debuff 。</li>
 </ul>
 <p data-pid="xkmMG9f9">中国象棋 6.0</p>
 <ul>
  <li data-pid="vm3kOgk3">机制调整：「修车」debuff 第二阶段由不能移动调整为可以移动一格（可以吃子）。</li>
 </ul>
 <p data-pid="QDefytnP">中国象棋 8.0</p>
 <ul>
  <li data-pid="S_h_6ABn"><b>移除「車」。</b></li>
 </ul>
 <p data-pid="W8UU_fKe">中国象棋 8.01c</p>
 <ul>
  <li data-pid="x6KHOPdW">「車」重新加入棋盘。</li>
  <li data-pid="4d7f8uen">保留「車」的传统规则，去除所有和「車」有关的附加规则。</li>
  <li data-pid="va7ku4Vs">机制调整：「兵」（卒）可以后退，但若「兵」（卒）退回己方河道，则不拥有过河后左右移动的能力，但后退可以吃子。</li>
  <li data-pid="H0iaQ0zY">机制调整：「馬」可以选择走「日」（1*2格子）和「目」（1*3格子）的对角线格点，但仍然可以被蹩马腿。</li>
  <li data-pid="HAFAP6CK">机制调整：己方失去所有「車」「馬」「砲」，或失去所有兵（卒）时（满足以上两个条件任意一条即可），若相（象）在己方底线，则可以消耗一步将相（象）转化为兵（卒）。</li>
  <li data-pid="RgfeHQj5">机制调整：「砲」可以同时摧毁自己与隔2子的对方棋子。</li>
  <li data-pid="sDF32lfO">机制调整：「仕」（士）在己方失去所有相（象）时，可以在底线处移动至相邻的相（象）的位置，然后变为相（象），这步操作不可逆。</li>
 </ul>
