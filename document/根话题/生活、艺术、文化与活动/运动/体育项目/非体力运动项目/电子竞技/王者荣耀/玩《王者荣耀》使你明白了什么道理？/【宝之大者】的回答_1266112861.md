# 玩《王者荣耀》使你明白了什么道理？
- 点赞数：8917
- 更新时间：2020年06月08日11时22分26秒
- 回答url：https://www.zhihu.com/question/60612736/answer/1266112861
<body>
 <ol>
  <li data-pid="Y_IzM7fJ"><b>祸兮福之所倚，福兮祸之所伏</b><br>
   当你超神的时候，一旦被击杀，对面会大发横财。</li>
  <li data-pid="pI_eYBWm"><b>行不苟合，不要盲从权威</b><br>
   战绩不好的辅助，即便苦口婆心，也鲜少有人愿意听他一言；大杀四方的打野，哪怕贪功冒进，也常常有人情愿陪他浪死。听从正确的建议，而不要盲信权威。</li>
  <li data-pid="VxSikyCJ"><b>盛极必衰，否极泰来</b><br>
   没有人会单排一直连败或者连胜，ELO机制面前众生平等。</li>
  <li data-pid="_H-lbn25"><b>君子群而不党，小人党而不群</b><br>
   如果队友一意孤行，陪着他们一起送是无济于事的。不要为了合群而委曲求全。</li>
  <li data-pid="sw8A_yYu"><b>相马失之瘦，相士失之贫</b><br>
   牛魔很丑，所以很多人不喜欢他，埋没了他的才能。这是不可取的。</li>
  <li data-pid="5BKzwxhg"><b>祸患常积于忽微，智勇多困于所溺</b><br>
   高手过招，拼的就是细节。你的任何坏习惯，都会成为对方滚雪球的资本。</li>
  <li data-pid="zC0Ga-sE"><b>人生如棋，卒子过河当車使</b><br>
   小兵不会回头，只会傻傻地向前冲锋。但是若没有兵线，你将寸步难行。</li>
  <li data-pid="koofgNMi"><b>画虎画皮难画骨，知人知面不知心</b><br>
   对着你嘤嘤嘤的队友妲己，说不定掀起裙子比你大。所以逢人只说三分话，不可全抛一片心。</li>
  <li data-pid="2qZhi60H"><b>你看队友多傻逼，料队友看你应如是</b><br>
   静坐常思己过，闲谈莫论人非。常与同好争高下，不与傻瓜论短长。</li>
  <li data-pid="AMnEMmoV"><b>该出口时就出口，过了这村没这店</b><br>
   遇到真的烦人的队友，骂他丫的。星星可以再打，人以后就骂不着了。</li>
  <li data-pid="on3O73D1"><b>用发展的眼光看问题 ，透过现象看本质</b><br>
   一代版本一代神，没有一个英雄能持久imba，就像没有一朵花四时常新；这是个推塔游戏，不是个杀人游戏。</li>
  <li data-pid="hvj2m7B1"><b>高筑墙，广积粮，缓称王，闷声才能发大财</b> <br>
   懂得韬光养晦，才能一鸣惊人。与其耀武扬威，不如默默发育。莫令黄钟毁弃，却使瓦釜雷鸣。</li>
  <li data-pid="Zutnq2jz"><b>千金在手不如一技傍身</b><br>
   大后期，会秒换三装的人才能活下来。否则再多经济也是白给。</li>
  <li data-pid="UzHHSBCq"><b>你不理财，财不理你</b><br>
   我讨厌裸出大打野刀的队友，有那钱足够买陨星了。有限的经济，只有最大化利用，才能创造更多价值。</li>
  <li data-pid="MIUOwmbm"><b>要承认自己能力有限，但是不要忘记本职工作</b><br>
   百里守约，你是射手，狙不中不可怕，不会a才可怕。</li>
  <li data-pid="H4NWj1at"><b>力的作用是相对的，魅力除外</b><br>
   庄周无拘无束，不会被任何人控住，却也控不住任何人。</li>
  <li data-pid="BN1oSK4n"><b>梅须逊雪三分白，雪却输梅一段香</b><br>
   典韦比曜伤害高，曜比典韦更灵活。英雄各有特点。没有人尽善尽美，也没有人一无所长。</li>
  <li data-pid="u-_xZYLe"><b>决定木桶容量的，是最短的木板</b><br>
   最菜的一人，最容易成为团队的缺口。所以要均衡发展，不可偏科。</li>
  <li data-pid="WX8dlkgc"><b>人靠衣装马靠鞍，狗配铃铛跑得欢</b><br>
   凤求凰看不起范海辛，圣斗士看不起精灵王。社会是看脸的，所以请你整衣郑容正行，自尊自强自爱。</li>
  <li data-pid="Nnj4ZRYZ"><b>人不可貌相，海水不可斗量</b><br>
   比起看脸，社会更看重实力。所以虚有其表，只能愚人一时。</li>
  <li data-pid="8koFeIgp"><b>人外有人，山外有山</b><br>
   弱小不是生存的障碍，傲慢才是！永远不要感到自满。求知若饥，虚心若愚。</li>
  <li data-pid="uZXYiB-S"><b>心有翼，行无疆</b><br>
   不想拿五杀的蔡文姬不是好蔡姬。不要画地为牢，自困囹圄。</li>
  <li data-pid="04pQPhO0"><b>心美一切皆美，境深万象皆深</b><br>
   你不快乐不是因为输了一局游戏，而是因为你本来就不快乐。</li>
  <li data-pid="p3n84mVL"><b>一个几把游戏而已，里面能有个屁道理？</b><br>
   少打游戏，道理都在书里面！<br><br></li>
 </ol>
