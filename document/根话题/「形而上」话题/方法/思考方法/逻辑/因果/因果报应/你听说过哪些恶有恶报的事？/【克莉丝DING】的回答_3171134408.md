# 你听说过哪些恶有恶报的事？
- 点赞数：5459
- 更新时间：2024年03月19日16时48分41秒
- 回答url：https://www.zhihu.com/question/330524973/answer/3171134408
<body>
 <blockquote>
  我悄悄地举报贫困室友买了苹果笔记本。
  <br>
   然后班里唯一的国家励志奖学金就落入了我手中。
  <br>
   这件事情我谋划已久了。
 </blockquote>
 <p>1</p>
 <p>开学我就知道，班里一共就两个国家级奖学金名额。</p>
 <p>一个是国家奖学金，不限家庭条件，只看综合成绩。</p>
 <p>一个是国家励志奖学金，既要成绩好又要家庭困难。</p>
 <p>国家奖学金我是不指望了，因为班里的 A 君每天几乎住在图书馆里学习。</p>
 <p>大学过的像高三，这谁能比的过他，让我像他一样，我选择狗带。</p>
 <p>我另辟蹊径，开学竞选了上了班里的班长，还经常参加院级和校级的各种比赛，从中获个小奖，这样也有不少加分。</p>
 <p>期末各项成绩出来后，我发现我综测第二，室友 C 综测第三。</p>
 <p>综测略略胜她一点。</p>
 <p>但架不住我和她家庭条件有差距。</p>
 <p>我爸妈都是豆奶厂的退休职工。</p>
 <p>她爸爸在她两岁的时候在工地从脚手架上摔下来死了。</p>
 <p>妈妈在超市做收银员，每月就两千八的工资。</p>
 <p>她家是社区认定的低收入家庭，住的是政府补贴的公租房。</p>
 <p>这么一比我完全处于劣势，心里真是不好受。</p>
 <p>我努力了一学年，还是差一点点。</p>
 <p>虽然有校级奖学金保底，但如果能获得国家级奖学金，将来毕业写在简历上也是十分亮眼的。</p>
 <p>来上大学不就为了找个好工作嘛，能从众多竞争者中脱颖而出，就是要有国家级的奖项当底气。</p>
 <p>但我惊喜地发现，室友 C 新买了一台苹果笔记本。</p>
 <p>她说是她暑期打了几份工赚来的钱，犒劳一下自己。</p>
 <p>我心开始痒了，如果举报她，是不是我就可以得奖了。</p>
 <p>几乎没有犹豫，我偷偷将她使用苹果电脑的照片拍下来，火速新注册了一个 QQ。</p>
 <p>通过新 QQ 邮箱写了一封匿名举报信给导员。</p>
 <p>过了几天，综合评定的结果下来了，果不其然，我得到了国家励志奖学金。</p>
 <p>室友 C 看着通知栏奖学金的告示，面色阴沉。</p>
 <p>那一段时间她在寝室里都没怎么说过话，眼神空洞，行动僵硬，像一块木头一样毫无生气。</p>
 <p>2</p>
 <p>国家励志奖学金的证书和奖金发下来后。</p>
 <p>我取出现金放进信封里，递给了她。</p>
 <p>我说：「名我要，钱你拿着吧。」</p>
 <p>她开始是一愣，后来眼眶红了，对我说：「谢谢。」</p>
 <p>接过信封的时候，她甚至有几滴泪落了下来，泪水滴在我的指尖。</p>
 <p>我像被火燎了一样收回手，心里觉得有点不是滋味。</p>
 <p>下午上专业课的时候，她主动坐到我旁边，拿出一杯奶茶放在桌上，说：「给你买的快喝吧。」</p>
 <p>她这一举动让我很有点不得劲，但我还是礼貌地接过奶茶，对她说：「谢谢。」</p>
 <p>之后，她对我越来越热络，不管是上课还是去图书馆，她都喜欢约着我一起。</p>
 <p>我真不习惯她这样，我是班长，看上去脾气好，性格开朗和每个人关系都很好。</p>
 <p>但我对于选择朋友，还是有一套自己的标准的。</p>
 <p>她和我不是一路人，玩不到一块去，我不想和她每天同进同出。</p>
 <p>当然，我不能强硬地拒绝她的各种邀约，只能找各种看上去合理的理由来拒绝她。</p>
 <p>她算是聪明人，渐渐有所悟，也不像之前那样每天对我发出邀请。</p>
 <p>我们的关系恢复到这件事发生之前。</p>
 <p>我对这样的现状感到很满意。</p>
 <p>3</p>
 <p>但人算不如天算，意外发生了。</p>
 <p>导员的一句话打乱了我和她之间的平衡。</p>
 <p>一次班会上，导员宣布本市知名的企业家，我们的校友张先生捐了一笔款给学校。</p>
 <p>张先生希望学校领导综合评估后，将这笔钱发放给家庭贫困且品学兼优的学子。</p>
 <p>「我综合考虑了你们班每个人的综测成绩和家庭情况，这笔奖金我决定发给小 C。」导员看向小 C。</p>
 <p>「小 C 我还要提醒你一句，今后不要购买一些昂贵的个人物品，比如苹果笔记本，会影响我们对你的奖项评估。」指导员严肃地说。</p>
 <p>室友 C 听到导员这么说，脸色煞白。</p>
 <p>我知道这个时候，自己的脸色也好看不到哪去，心里骂了导员八百遍大嘴巴。</p>
 <p>开完会，回到寝室，室友 C 就开始摔摔打打，骂骂咧咧了。</p>
 <p>她重重地把水杯撞在桌子上，粗鲁的打开衣柜门，换好睡衣，再一脚把衣柜门踹关。</p>
 <p>嘴里念叨各种脏话，她说：「怪不得，我说自己怎么没有选上国家励志奖学金，原来是有人暗中摆了我一道。」</p>
 <p>我缩了缩脖子，一声不敢吭。</p>
 <p>知道她购买苹果笔记本电脑的人，也就我们寝室这几个了。</p>
 <p>能拍到她在座位上用苹果笔记本的人，那必定是我们之一。</p>
 <p>我们寝室共四人，除了我和室友 C，就是绿茶和佛子了。</p>
 <p>绿茶顾名思义，她的精力都放在和男人周旋上。</p>
 <p>她高中就有个男友，目前两人异地恋，但在本校她还同时和两个男人暧昧纠缠。</p>
 <p>佛子更是能坐着绝不站着，能躺着绝不坐着。</p>
 <p>不重要的课就逃掉，在寝室基本处于躺平追剧磕 cp 状态，成绩在挂科边缘疯狂试探。</p>
 <p>所以，这两人志不在此，绝不可能是告密者。</p>
 <p>绿茶和佛子也很清楚，两人对视一眼，露出等着看好戏的八卦表情。</p>
 <p>这一段时间，室友 C 稍一不顺心就在寝室乱摔乱打，指桑骂槐。</p>
 <p>我的祖宗十八代被她问候了无数遍，掘坟鞭尸。</p>
 <p>我自己也被她形容成乌龟的孩子，在垃圾桶里和老鼠抢饭吃。</p>
 <p>我忍。</p>
 <p>小不忍则乱大谋，确实是我做错了，她撒撒气这事也就过了。</p>
 <p>4</p>
 <p>这天，我正在图书馆做作业，辅导员打了个电话来。</p>
 <p>我心生一种不妙的预感。</p>
 <p>「小晞，你来趟我的办公室。」导员说。</p>
 <p>「好的。」我挂了电话，急急地走向办公楼。</p>
 <p>辅导员办公室里，室友 C，佛子，绿茶站成一排。</p>
 <p>我来了后，导员开口说：「你们寝室怎么回事，学校三令五申不让用大功率电器，你们怎么还用。」</p>
 <p>「啊？！」我忍不住喊了一声，这是怎么回事，我们寝室哪有什么大功率电器呀。</p>
 <p>「我们寝室没有大功率电器，辅导员您是不是误会了。」我试图解释。</p>
 <p>「喏，明明知道今天有卫生检查，还在寝室做火锅吃。」导员朝办公桌上努嘴。</p>
 <p>我看到一个电火锅正摆在上面。</p>
 <p>我盯着那个电火锅，心中一片茫然。</p>
 <p>这是怎么回事？我第一次见这个火锅，这确定是我们寝室的？</p>
 <p>「张导，对不起，我错了，今天是我的生日，所以我买了一些食材，想煮一顿火锅和室友们分享一下。」室友 C 开口道。</p>
 <p>我怔怔地看着室友 C，脑子有点转不过来。</p>
 <p>作为班长我搜集有全班同学的身份信息，我知道她的生日是五月份，可现在可是十月份。</p>
 <p>退一万步讲，她邀请室友吃火锅，却独独选在我不在且院学生会卫生检查的时候。</p>
 <p>「根据校规，使用大功率电器，取消你们寝室这个学期评优评先资格，引以为戒。」导员说。</p>
 <p>我心里飞快地搜索着这个学期还有什么评优的项目，咯噔一下，心凉了半截。</p>
 <p>我要竞聘院学生会宣传部长的事情要黄了。</p>
 <p>本来作为班长，我在院里的口碑人缘都不错。</p>
 <p>这次竞聘本来十拿九稳，宣传部长的身份正在向我招手。</p>
 <p>这一闹，黄了。</p>
 <p>气愤无奈的情绪顿时充满我的胸腔。</p>
 <p>这种感觉就像驾考科目三路边停车后还没完成最后一套动作，后排的人就直接打开车门，被判不合格一样。</p>
 <p>我窝火极了，忍不住恶狠狠地看向室友 C，她迎着我的目光，挑衅似的弯弯嘴角。</p>
 <p>这一刻，我有想冲上去薅她头发的冲动。</p>
 <p>导员又训斥了我们几句，他看了看时间，说自己待会还要开会，要我们离开。</p>
 <p>告别导员，我们从办公室出来，四人一路无言，回到寝室。</p>
 <p>寝室门刚关上，我就质问 C：「你是故意的吗？」</p>
 <p>她将一个信封甩在了我的脸上。</p>
 <p>我一摸，是我当时给她的励志奖学金信封，里面鼓鼓的，钱应该都还在。</p>
 <p>她说：「我是故意的，怎么样，被人阴的滋味不好受吧。你的臭钱太脏了，我不要，还给你。」</p>
 <p>佛子和绿茶面面相觑，摸不清是咋回事，劝也不好劝，就都没吱声。</p>
 <p>我拿起信封，装进抽屉里，说：「不要就不要呗，拿钱撒什么气。」</p>
 <p>「下次我不会再让你逮着机会阴我，明年的励志奖学金一定会是我的。」室友 C 恨恨地说。</p>
 <p>「话别说太早，小心到时候打脸。」我讥诮道。</p>
 <p>「我就把话撂着，咱们走着瞧。」室友 C 白了我一眼。</p>
 <p>5</p>
 <p>好得很，这样最好不过了。</p>
 <p>我收回所有愤怒，没有回嘴，像什么都没有发生一样淡定处之。</p>
 <p>室友 C 骂了我十来天。</p>
 <p>我忍了，算我欠她的，这下我们两清了。</p>
 <p>至于她说明年励志国奖一定会是她的，相当于向我下了战书。</p>
 <p>我这个人一向遇强则强，喜欢接受挑战，特别爱看对手吃瘪的样子。</p>
 <p>这下我又有事可干了，充满了斗志，瞬间支棱了起来。</p>
 <p>自从这场交恶后，我和室友 C 就不再说话了。</p>
 <p>一个寝室里统共就四个人，一旦其中两人互不搭理，气氛就变得十分尴尬。</p>
 <p>一天，绿茶实在看不下去了，对我们说：</p>
 <p>「这周末我和一个男生寝室组了一个联谊趴，我们整个寝室都去。」</p>
 <p>「我周末要去打工，去不了。」室友 C 立即生硬地回绝。</p>
 <p>绿茶背对着室友 C，嘴角很不开心地撇了一下。</p>
 <p>我朝绿茶笑了一下，说：「怎么你又要纳妃啊。」</p>
 <p>绿茶嗔怪地瞥了我一眼说；「别胡说八道。」</p>
 <p>佛子正在床上追剧，把零食嚼得嘎嘣响，边吃边说：</p>
 <p>「我是个随波逐流的人，你们商量，去就都去，不去就都不去。」</p>
 <p>我说：「茶茶，对不住，智者不入爱河，这种事情我不感兴趣，不打算去。」</p>
 <p>绿茶对我说；「你看看你和 C，为了几个臭钱就开撕，把寝室氛围搞得这么尴尬，我为了你们组个局，你们还都不卖我面子，你俩整天就知道整一些没用的事。」</p>
 <p>室友 C 听到绿茶提起她，朝我们这边看了一眼。</p>
 <p>我觉得有些好笑，问：「茶茶，什么是有用的事，什么是没用的事呀，你说我听听呗。」</p>
 <p>「你呀。」绿茶指指我，「还有你们俩。」她又指了指室友 C 和佛子。</p>
 <p>「你们仨都是光棍，二十岁的大好年龄，正是人生中最青春靓丽的时候，两个人为一个奖学金争，一个人整天追剧磕别人的 cp，你们知不知道，女人最后还是要选一个好男人结婚过日子，趁着年轻的时候为什么不多挑挑拣拣，为自己争取最好的呢。」</p>
 <p>绿茶一口气说完，悲悯地看着我们仨。</p>
 <p>我愣了，她说得还真有几分道理。</p>
 <p>佛子手中的零食顿时不香了，咀嚼声戛然而止。</p>
 <p>室友 C 也若有所思地低下了头。</p>
 <p>「我为你们争取到的是计算机科学专业 M 君的寝室联谊趴。」绿茶说。</p>
 <p>「M 君呀，知道吗，计科系草，他们寝室其他人也各个都是精神小伙。」绿茶看我们几个呆愣住了，便又强调了一下。</p>
 <p>M 君，我的大脑电光火石间闪过一道惊雷。</p>
 <p>哈哈哈~~~老天待我不薄，M 君就是那个通过高中数模竞赛全国一等奖保送来本校的人。</p>
 <p>很久前，我就想找机会认识他，但苦于找不到一个即不刻意又丝滑的理由来接触他。</p>
 <p>这不，机会来了。</p>
 <p>「你说得有道理，这次联谊我加入。」我说。</p>
 <p>「那我也去。」佛子说。</p>
 <p>室友 C 沉默了一会，开口说：「我和老板请个假，也去。」</p>
 <p>绿茶终于笑逐颜开，嘘了一口气，说：「哎，不枉我苦口婆心说道，你们仨终于开窍了。」</p>
 <p>6</p>
 <p>我考虑了下一步棋该怎么走，最好的办法就是通过竞赛上大分。</p>
 <p>我盯上了大学生数模竞赛，也打听到了本校的数模高手 M 君。</p>
 <p>我想和强者组队，参与竞赛就冲着全国一等奖去搏，有整整十分的加分呢。</p>
 <p>有了这十分，纵使班里这次的综测第一 A 君每天不吃不喝地在图书馆里学习，眼镜度数增加个两百度，也比不过我。</p>
 <p>我下次想要的可不是什么国家励志奖学金，我要的是国家奖学金。</p>
 <p>就算室友 C 获得了励志奖学金，她也还是矮我一头。</p>
 <p>很快到了周末，绿茶，室友 C，佛子都化了淡妆，穿上了小裙子。</p>
 <p>而我还是上身白 T 恤，下身牛仔裤的打扮，还背上了我平时上课用的书包。</p>
 <p>绿茶从上到下打量我一番，脸上露出恨铁不成钢的表情，张了张嘴想要数落我。</p>
 <p>但她想了想，今天好容易集齐寝室所有人一起出去。</p>
 <p>为了保持大家的好心情，不想拂面子我的面子，终究啥也没说，闭上了嘴。</p>
 <p>联谊趴见面地点选在一个咖啡厅，环境优美，暖光灯照，烘托出一种暧昧的气氛。</p>
 <p>男生们早就等候在此，看到我们前来，朝着我们打招呼。</p>
 <p>我轻声问绿茶：「哪位是 M 君。」</p>
 <p>绿茶说最左边那个，只见一个身穿白衬衫的男孩正在低头把玩着手里的咖啡杯。</p>
 <p>坐在他旁边的男生见我们看向他，用胳膊肘顶了他一下。</p>
 <p>他抬起头来，好看的眉眼舒展开来，鼻梁高挺，嘴唇微微上翘，朝我们微笑点头示意。</p>
 <p>我不禁感叹，绿茶不愧是绿茶，目光狠辣，这等姿容竟比女孩子还好看。</p>
 <p>不多想，我要赶紧干正事。</p>
 <p>我快步上前来到 M 君身旁，用只有我和他能听到的声音说：「我要和你聊聊数学建模竞赛的事，等会选我。」</p>
 <p>他愣了一下，疑惑地看了我一眼，并没有回复。</p>
 <p>我们模仿《雪滴花》的情节，男寝女寝相对而坐。</p>
 <p>佛子突然诡异的笑了一声，我知道她又沉浸在自己的世界里了，估计是想起了电视剧中的 cp。</p>
 <p>我们彼此自我介绍了一番，然后绿茶在桌上摆了四样东西，分别代表我们四个女生。</p>
 <p>男生选择其中的一样物品就可以和相应的女生单独约会。</p>
 <p>从左到右，由 M 君先选。</p>
 <p>我看了一眼 M 君，又用眼神指了指钢笔，他也正看着我，将我的表情尽收眼里。</p>
 <p>他犹豫了一瞬，还是从桌子上拿起了钢笔。</p>
 <p>成了，我欣喜极了。</p>
 <p>但余光看到绿茶的脸瞬间垮了下来，看来她是真动怒了，我回去还得和她好好解释一番。</p>
 <p>7</p>
 <p>离开了大部队，我和 M 君来到了一家书店，找了个书咖坐了下来。</p>
 <p>「我想和你组队参加这次的数模竞赛。」我直截了当的说。</p>
 <p>「你的专业是生物科学，怎么会想要参加数模竞赛呢。」M 君问。</p>
 <p>我没有着急回答，而是打开书包，取出我的简历和一叠资料递给 M 君。</p>
 <p>「虽然我是生物科学专业，但是我从小到大一直在接受机器人编程的课外培训，曾经也获得过机器人编程大赛的全国奖项。」</p>
 <p>我用手指点点我的简历。</p>
 <p>「我熟悉编程语言，所以参与数模竞赛不成问题，你可以看看我的个人简历。」</p>
 <p>M 君拿起我的简历看了起来。</p>
 <p>等他看完，我接着说：「我收集了近五年来所有省份及全国数模竞赛试题，自己尝试做了一遍。」</p>
 <p>我又指了指那叠资料。</p>
 <p>「这就是我写解题思路及答案，你可以看后再考虑要不要和我组队。」</p>
 <p>M 君拿起资料看了起来，时间一分一秒的过去了。</p>
 <p>看了大概约三分之二，他抬起头来，笑着对我点点头，说：</p>
 <p>「你的基本功很扎实，解题思路清晰，答案也接近最优值。」</p>
 <p>得到他的肯定后，我放心了大半，接着说：「我想和你组队，目的是冲着全国一等奖去的，我需要这个竞赛的加分。」</p>
 <p>M 君略微思索了一下，说：「嗯，明白了，我和你组队一起参加这次的数模竞赛。」</p>
 <p>我心里顿时放起了烟花，组队的事情终于办妥了。</p>
 <p>8</p>
 <p>我和 M 君在一起吃了顿晚饭后才分开。</p>
 <p>回到寝室，我拿钥匙开门，却发现门被反锁了，怎么也打不开。</p>
 <p>敲门也无人应答。</p>
 <p>我想了想，顿时明白了，觉得有点好笑。</p>
 <p>里面的仨人，岁数加在一起也六十多了，怎么还这么幼稚呢。</p>
 <p>我清了清嗓子，边敲门边喊：「茶茶，你误会了，我需要的是 M 君的脑子，不是 M 君的身子。」</p>
 <p>喊了两遍，门里传来急急的脚步声，绿茶把门打开了。</p>
 <p>她一脸怒气地对我说：「你瞎喊什么呀，什么脑子身子的，把我们说的像两个女妖精要吃掉 M 君一样。」</p>
 <p>我扑哧一声笑出来，扯了扯绿茶的衣角说：「寝室里面说话不方便，我们去一个没人的地方说。」</p>
 <p>我带绿茶来到女寝楼后的小树林。</p>
 <p>她一脸怒意地看着我，不发一言。</p>
