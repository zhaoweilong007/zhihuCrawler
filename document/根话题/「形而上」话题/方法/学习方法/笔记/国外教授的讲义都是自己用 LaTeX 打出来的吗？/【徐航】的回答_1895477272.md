# 国外教授的讲义都是自己用 LaTeX 打出来的吗？
- 点赞数：5555
- 更新时间：2021年05月20日05时20分03秒
- 回答url：https://www.zhihu.com/question/29227449/answer/1895477272
<body>
 <p data-pid="BZEMsRR6">我校数学系之前有一传奇叫Dexter Chua，此君每节课边听课边用latex记笔记，并且把整个大学期间上过的数学课所有笔记分类整理成册，造福了不知道多少人。据他自己统计一共有4266页，204847行代码，现在都上传到了他的主页上，并且还包含他用的编译器和配置。</p><a href="https://link.zhihu.com/?target=http%3A//dec41.user.srcf.net/notes/" data-draft-node="block" data-draft-type="link-card" class=" wrap external" target="_blank" rel="nofollow noreferrer">Dexter Notes</a>
 <p data-pid="hayPNUKT">这说明只要够熟练，latex比手写快多了。</p>
 <p data-pid="L3g-HuS2">而且大学教授的讲义也不需要年年换新的，新三年旧三年缝缝补补又三年，每年还有学生帮忙纠错。</p>
 <p data-pid="arNLEdv4">我个人的经验，VSCode + Vim + snippets自定义快捷输入 + 自动补全 + 各种需要的latex package和插件，熟练过后还是很快的。（感谢给我传道的 <a class="member_mention" href="https://www.zhihu.com/people/e0a019f3ecf59ea8b5a2e98354b85ed2" data-hash="e0a019f3ecf59ea8b5a2e98354b85ed2" data-hovercard="p$b$e0a019f3ecf59ea8b5a2e98354b85ed2">@物理猫</a> 。）一开始总会觉得慢，但一个学科常用的符号和式子也就那么多，边写边发掘自己常用的表达式，根据需求随时增加新的快捷输入，只会越来越快的。</p>
