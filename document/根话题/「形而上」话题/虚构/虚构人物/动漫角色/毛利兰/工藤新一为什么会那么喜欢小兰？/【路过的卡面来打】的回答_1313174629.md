# 工藤新一为什么会那么喜欢小兰？
- 点赞数：7784
- 更新时间：2020年07月30日23时42分17秒

<body>
 <h2>*好了，我来一句句反驳你的观点来了</h2>
 <blockquote data-pid="LVjLtXKe">
  工藤新一在各方面都是非常出众的，而小兰虽然也很优秀，但是和工藤也确实是不在一个水平的
 </blockquote>
 <p data-pid="GZ16gpoP">为什么你会觉得兰酱和新一不在一个水平？</p>
 <ul>
  <li data-pid="5fdo2PYs">颜值</li>
 </ul>
 <p data-pid="LVc3dfY3">新一是帅哥这点我不做详细阐述。少年侦探团第一次见到兰酱，就说“小兰姐姐真的是一个美人诶”还有每次外出，经常有男人找兰酱搭讪，从小学到高中喜欢她的男生不断，甚至男女通吃。这对小情侣颜值不可能不在一个水平吧。</p>
 <ul>
  <li data-pid="9-jpQE4H">能力</li>
 </ul>
 <p data-pid="uy3e63ea">新一是有名的高中生侦探，推理能力强足球踢得好，会拉小提琴，兰酱似乎也不差吧。料理好吃，空手道关东地区冠军，潜水，滑雪，保龄球，网球，陶艺，编织，钢琴，唱歌样样都能露一手，成绩也是学霸水准。</p>
 <p data-pid="OMu-otLI">PS.新一在夏威夷技校(bushi)学到的开游艇开飞机射击等技能因为是属于剧场版，不是73的原著剧情，所以不包含在内。</p>
 <ul>
  <li data-pid="Mts14RXz">性格</li>
 </ul>
 <p data-pid="0mHGbUOg">兰酱每次去买菜卖菜的伯伯阿姨都会很亲切的和她打招呼，送她一些东西，在帝丹高中欢迎程度排名前十，少年侦探团的小朋友都非常愿意和她相处，贝姐称她为angel，说兰酱和新一是她在世上唯二的珍宝，这些都可以从侧面体现出兰酱性格有多好多受欢迎了吧</p>
 <ul>
  <li data-pid="KtlOkJK2">家境</li>
 </ul>
 <p data-pid="f47tKNPg">虽然说兰酱的家境肯定没有新一好，但是她也不差啊。事务所这一幢楼是她家的，一楼出租二楼事务所三楼住人。父亲警校传奇，母亲律政界女王。如果她家家境不好，她也没多余的钱去学空手道钢琴什么的吧，大叔追星赌马玩小钢珠喝酒也是要砸钱的事情啊【笑】只不过我觉得大叔赚得多花的也多就是了。</p>
 <p data-pid="6SMXk5gr">另外大叔一个月的酒钱换算成人民币要两万来着(漫画108话)。这如果家庭条件不好哪花的起两万买酒啊(笑)</p>
 <blockquote data-pid="if1d4643">
  而且两个人的兴趣爱好也不一样
 </blockquote>
 <p data-pid="zBxVikDT">不是我嘴毒哈，你这句话真的挺幼稚的。两个人在一起从来不需要什么兴趣爱好完全一致，而是彼此之间的理解支持和三观一致。(经过评论区小可爱的提醒，对这句话做一个补充。兴趣爱好的一致绝对绝对是一个加分项，但是不代表兴趣不一样这一对cp就不适合在一起)不管是情侣还是夫妻，在热恋过后必定会经过一段时间的磨合，作为夫妻更可能是需要一辈子去磨合。兴趣爱好一致，但是不能支持对方的工作，不能包容彼此的小缺点日后必定争吵不断。你提到福尔摩斯时我认真听，把你的所有话都记在心里，我去练空手道时你在一旁默默看着我 静静等着我，这何尝不是一种美好？三观的一致也是灵魂上的高度契合，这就是人们常说的灵魂伴侣啊。</p>
 <h2>先谈他们相互理解与支持吧</h2>
 <ul>
  <li data-pid="IwVyA0fe">初恋情人回忆事件里，兰酱要冲进火海救人，柯南不但没有阻止反而递给她头盔，和她一起去救人。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="ISaEFoCo">新一vs冲矢昴里，兰酱要去解救人质，虽然刚开始新一阻止了她，但是马上选择支持她的做法，告诉她如何实施救援。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="KxEgcpos">兰酱不可能不明白做侦探是有危险的，但是她从来没有劝过新一不去当侦探，也没有阻止过他办案。就算非常期待他的告白却仍然选择以大局为重，给予新一足够的理解与支持“我就在这里，不躲也不逃，你快去吧，大侦探”</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="yu2B4IqJ">虽然表面上不喜欢新一谈福尔摩斯，但是遇到难以解决的事情还是会想到福尔摩斯的名言“如果可以确实的让你毁灭，为了公众的利益，我很乐意迎接死亡”去伦敦的时候查资料攒钱买礼物规划路线，《贝克街的亡灵》里面感叹 如果新一看到了一定会很高兴。足以说明兰对新一爱好的支持尊重吧？</li>
 </ul>
 <h2>再说一下他们的三观</h2>
 <p data-pid="ULzET4tC">这个确实太明显了，比如说纽约篇救贝姐，“救人是不需要理由的”</p>
 <p data-pid="X-uLF50g">再比如满月篇宁愿牺牲自己也要救哀</p>
 <p data-pid="1FdXvjQw">再再比如饱受争议的拉窗帘事件，这个我挺想说一下，可能有点跑题，不想看请略过。</p>
 <p data-pid="vq39Svj_">当时兰酱说“不希望有人死在她面前”所表达的内涵其实和新一认为的“侦探不应该把犯人逼上绝路”是一致的。虽然说很多人说兰这么做是那所有人的生命在堵，但是</p>
 <ol>
  <li data-pid="i26a8jkm">当时狙击手也说明了，要狙击炸弹男是很困难的。</li>
  <li data-pid="I-ncdbvM">除非是极其恶性无法缓解的案件，警方是不能不经过谈判就击毙嫌疑人的，因为这样是违反程序规定，如果直接狙击后续警方会很难办</li>
  <li data-pid="pCD96GWz">炸弹男走到窗户边找充电器的时候狙击手申请开枪，但是指挥官一直犹豫不决，一直没有批准。为什么？很明显是因为就算炸弹男走到了窗户边狙击犯人还是有一定困难的，<i>所以说狙击手和警方也是拿所有人生命赌</i>。</li>
  <li data-pid="ktscDRAL">当时炸弹男是用的需要按才会爆炸的炸弹(奇怪的描述2333)，如果狙击手真能一击毙命那万一那玩意儿撞到桌角？撞到地上？照样爆炸。也就是说在击毙犯人的条件下，还需要在不到一秒钟的时间里确保稳住那个按钮。这难度真的不是一般大，还要有运气成分。</li>
  <li data-pid="IcuwXYRf">兰拉上窗帘一方面是不希望犯人s，另一方面是她非常信任新一。这种信任不是盲目的，因为警方当时也是选择相信新一，让他试着劝阻炸弹男。</li>
  <li data-pid="fgBIGddk">而且炸弹男罪不至死，在警方和她都必须赌一把的情况下，拉上窗帘劝阻他到是最保险的办法。</li>
  <li data-pid="XCOaNJ5b">名柯这部作品所传递的价值观就是，生命可贵，生命没有贵贱之分，救人是不需要理由的，体现一部作品的价值观靠的就是角色的表达。</li>
 </ol>
 <p data-pid="BeMbjjLX">好了回归正题，这起事件的ED之后有一个彩蛋，劫持事件过后新一对兰说了谢谢。</p>
 <p data-pid="fW9qWMP-">谢谢她懂他的意图，谢谢她能理解她，谢谢她明白“侦探如果把犯人逼上绝路，那么和杀人犯有什么区别”</p>
 <blockquote data-pid="q6dPU3ND">
  真的是樱花班那次就一见钟情了吗，那才几岁啊，就真的能一直喜欢吗
 </blockquote>
 <p data-pid="PW1xaZlk">嘿，我告诉你，那还真是一见钟情。《变小的名侦探》里兰酱问他的初恋是不是有希子，新一在心里想着初恋就是你啊。樱花班更不用说吧“从遇见你的那一天起，你的那张笑脸就让我心动不已。”</p>
 <p data-pid="2G68ywru">至于是不是一直喜欢，答案是肯定的。这个要结合日文原文看。</p>
 <p data-pid="aua4-tBP">“从遇见你的那一天起，你的那张笑脸就让我心动不已”在这里原台词用了から。这个词是用在过去式之后，表示在……之后一直怎样。也就是说新一对兰酱一见钟情 感情十三年间从来没有变过，一直一直喜欢兰酱。</p>
 <p data-pid="IjlZnzXI">还有一个，“心动不已”原台词用的是ぞっこん 这个词的意思是 打心眼里喜欢，比一般的喜欢程度更深。从初遇的那一刻起，小新一就打心眼里喜欢小小兰，就算单恋了十二年就算告白还没得到回应他还是一直这么喜欢她，从未改变，从不动摇。</p>
