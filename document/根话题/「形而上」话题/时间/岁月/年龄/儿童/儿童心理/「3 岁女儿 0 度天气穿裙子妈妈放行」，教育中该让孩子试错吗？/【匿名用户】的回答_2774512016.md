# 「3 岁女儿 0 度天气穿裙子妈妈放行」，教育中该让孩子试错吗？
- 点赞数：5055
- 更新时间：2022年11月25日18时15分58秒
- 回答url：https://www.zhihu.com/question/568359757/answer/2774512016
<body>
 <p data-pid="JSyc-DxS">我儿子差不多两三岁的时候，零下几度要求穿短裤，我也让穿了。他在寒风中问我腿为什么会自己动……</p>
 <p data-pid="80TC0z-1">后续就是我带他跑了两圈浑身出汗暖和了以后回屋了，啥事没有。后来也不要求穿短裤了。我告诉他腿自己动那就叫——冷。</p>
