# 如何看待人教版教材疑似出现低级错误，用爱因斯坦相对论证明勾股定理？
- 点赞数：13273
- 更新时间：2020年06月18日13时51分45秒
- 回答url：https://www.zhihu.com/question/401988398/answer/1288700416
<body>
 <p data-pid="UnMiARk2">这个错误的荒谬程度大约相当于</p>
 <ul>
  <li data-pid="aqPmuQFu">在地理课本上写瑞士在南半球；</li>
  <li data-pid="H8Y7L0Za">在生物课本上写仙人掌是哺乳动物;</li>
  <li data-pid="swBAKN3L">在历史课本上写项羽是拿破仑他二舅;</li>
  <li data-pid="E8tVMSby">....</li>
 </ul>
 <p data-pid="XLBxiN0h">注意这仅仅是就<b>荒谬程度</b>而言，就其<b>危害性</b>来说，比以上三个谬误大得多。因为以上三者是简单的事实谬误，比较容易纠正。但原问题中的这个“证明”的荒谬性体现在方法层面，很难纠正。</p>
 <p data-pid="EaIxL_gG">（我读初中时）初中数学的两个重要任务就是让学生</p>
 <ul>
  <li data-pid="w2_GxdOA">读懂简单的数学表达式和命题；</li>
  <li data-pid="NsW0hYqJ">领会到什么是数学证明。</li>
 </ul>
 <p data-pid="haujlk5Q">对于天资普通的学生，达成这两点需要老师煞费苦心的引导，学生自己也需要投入相当的精力。即使如此，有些人直到学生生涯结束仍然是一知半解。而我们所讨论的这个“证明”恰恰是这两项任务失败的典型例子：</p>
 <ul>
  <li data-pid="-zHzTySE">胡乱解释表达式 <img src="https://www.zhihu.com/equation?tex=E%3Dmc%5E2" alt="E=mc^2" eeimg="1"> 中各个字母的含义；</li>
  <li data-pid="wInYc8zh">表达式都是胡诌的还谈什么证明。</li>
 </ul>
 <p data-pid="J2H7MBgN"><b>如果这个谬误真的出现在人教的出版物上</b>，那就是在这个原有教学任务内在的困难上，人为地加上一个权威的假“证明”的误导，可以说贻害无穷。</p>
