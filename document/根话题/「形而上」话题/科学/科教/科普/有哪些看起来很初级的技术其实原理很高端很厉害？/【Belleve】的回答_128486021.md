# 有哪些看起来很初级的技术其实原理很高端很厉害？
- 点赞数：6610
- 更新时间：2016年10月26日19时14分21秒
- 回答url：https://www.zhihu.com/question/51452831/answer/128486021
<body>
 <p data-pid="uVIwH9lJ"><b>你在 word 里面打一行字，然后它出现。如果是在中间插入的话，把后面的内容挤下去。</b></p>
 <p data-pid="XqCZ2uG0">这个非常自然的操作背后是数十位顶尖工程师（将近一半是数学家）的杰作。</p>
 <p data-pid="gAjaCcDB">因为这个排版要做到：</p>
 <ul>
  <li data-pid="sE8adkC_">可以处理所有已知的人类语言，<b>以及它们的任意组合。</b><br></li>
  <li data-pid="IPuBXG9m">可以处理绝大多数已知的人类排版规则（比如某些标点之间不能断开），<b>以及它们的任意组合。</b><br></li>
  <li data-pid="gIgAtZnJ">可以处理图片、表格、公式、各种嵌入的组件，<b>以及它们的任意组合。</b><br></li>
  <li data-pid="X_5UFdSM">可以处理「反常」、「损坏」的输入，不能疯掉。<br></li>
 </ul>
 <p data-pid="kCcCCdIx">然后呢，因为是实时编辑的所以：</p>
 <ul>
  <li data-pid="5VCtMHIN">所以排版必须快，编辑不能卡（TeX 不是实时的）。<br></li>
  <li data-pid="gcnzTVFb">要支持撤销重做剪切粘贴等你们看上去稀松平常的功能（每一个都不好做）。<br></li>
  <li data-pid="p9DPF6U_">而且 Word 是带界面的！这个界面不能太复杂，否则没人会用你家软件。也就是说，所有这些复杂的功能都必须藏起来，然后以最自然的方式提供。</li>
 </ul>
 <p data-pid="b2q6FcDY">哦对了：</p>
 <ul>
  <li data-pid="o7whjUdI">这个是通用的排版库，所以要可以用在多个软件里面，要有通用性。还要能跨平台。<br></li>
 </ul>
