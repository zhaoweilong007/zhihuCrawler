# 对于医生来说，什么样的病人属于“优质病人”？

 <p data-pid="ataEPq5P">SSS级“优质病人”：</p>
 <ul>
  <li data-pid="Zq40Ay97">自己办入院、开医嘱、写病程记录、办出院，最好连护士的活也干了；</li>
  <li data-pid="FNyze3_q">点名只要实习生操作，甘为练功桩；</li>
  <li data-pid="hUTNb8jg">能像苏联的Leonid Rogozov那样自己动手术就更好了！我也不值班了，给你写小说。</li>
 </ul>

 <p data-pid="I-qt_tUT">S级“优质病人”：</p>
 <ul>
  <li data-pid="AboAqbi5"><b>疾病所致的经济负担不高</b>：<b>不一定得经济收入高，但需有充足的降低风险措施</b>，像购买商业保险、长期良好的健康习惯（降低各种疾病风险）、定期体检（早期发现病变，及时治疗）、结交医生朋友、一定的社会地位。</li>
  <li data-pid="8zV_U41s"><b>会自我教育</b>：有批判性思考的能力，积极学习疾病相关知识，更新以往的疾病偏见，对自己病情有更深入的了解，消除未知所造成的恐惧。</li>
  <li data-pid="P_SFEaIi"><b>自我管理能力强</b>：遵医嘱、留意自身病情变化并客观记录、及时反馈；比如：唐先生自糖尿病后，参考“中国2型糖尿病自我管理处方专家共识”，记录自己的基本情况（BMI、年龄、作息时间、饮食、并发症...）、血糖变化、胰岛素及口服药使用状况，不仅客观地认识和主观掌控自身病情的进展，还大大方便复诊时医生调整治疗方案。</li>
 </ul>
