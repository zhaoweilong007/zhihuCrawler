 <p data-pid="nWLn74UG">哥白尼：虽然我不能确定那个世界是否真的存在，但我知道它不是为我们服务的。</p>
 <p data-pid="zmTzQt-4">伽利略：我等发现了一个全新的世界，我观察到了它的一些规律，但还不知道怎么回事儿。</p>
 <ul>
  <li data-pid="JnFfkjvn">开普勒：我和我老师第谷不配拥有姓名？</li>
  <li data-pid="rZE7or1K">伽利略：没办法我会磨望远镜，看得更远，你不会。</li>
 </ul>
 <p data-pid="w8h1e8ft"><b>牛顿：我打开了这个世界的大门，并描述了这个世界的基本规律。</b></p>
 <ul>
  <li data-pid="qogH1VMn">牛顿：不好意思，我不仅会磨望远镜，还会打造望远镜，并还有非凡的洞察力和计算力。</li>
  <li data-pid="qt_cU6N5">格雷戈里：我真是谢谢您。</li>
  <li data-pid="khamsOvx">胡克：牛顿！又是你？？？？（胡克在1674年造出格雷戈里望远镜，牛顿在1668年造出世界第一架反射式望远镜）</li>
 </ul>
 <p data-pid="3DqCr4mu">欧拉：我等构建了数学体系，才让那群物理学家站得更高，看得更远。</p>
 <ul>
  <li data-pid="G4o5Bvuf">伯努利：我只是业余随便搞搞数学的。</li>
  <li data-pid="VGRA0P5r">富兰克林：我是兼职搞物理的，不小心起草了一个《独立宣言》。</li>
  <li data-pid="zl-CrCyK">拉格朗日：那我就兼职物理学家，搭个桥吧。</li>
 </ul>
 <p data-pid="tAUENKZK">麦克斯韦：俺把物理学和数学之间的桥变成了一条大道，并预言了电磁波的存在，且认为光是一种电磁波。</p>
 <ul>
  <li data-pid="rgRFNBqV">赫兹：我证明了电磁波的存在，不小心把名字变成了频率的单位，并和焦耳他们整出了热力学第一定律。</li>
  <li data-pid="ZzE-Aj7N">法拉第：好巧，我也是。发现了电磁感应现象，然后名字成了单位。</li>
  <li data-pid="5121iAbR">欧姆：我也复议。</li>
  <li data-pid="fvZOHcnM">库伦、瓦特、瓦特、安培、焦耳……：还有我们</li>
  <li data-pid="eXRuZM3v">卡文迪许：为什么我没有留下姓名？算球了，我给地球称个重吧。</li>
 </ul>
 <p data-pid="g2lG5O9l">克劳修斯：我引入熵，和开尔文一起建立了热力学第二定律，他好像和上面那群人一样也是单位怪。</p>
 <ul>
  <li data-pid="_WgfE2HB">阿伏伽德罗：我提出了分子的概念。</li>
  <li data-pid="ax6vblmE">道尔顿：我建立了原子理论。</li>
  <li data-pid="vyXklnsS">玻尔兹曼：我提出了玻尔兹曼熵公式，从统计学的角度阐述了热力学第二定律。</li>
 </ul>
 <p data-pid="7jSSy3ji"><b>开尔文：我等几乎发现了世界上所有的东西，这个世界几乎完美了。最多只需要再修修补补，例如咋们头顶上的两个破洞。</b></p>
 <blockquote data-pid="fvgv-mvy">
  1900年4月27日，开尔文发表了题为“在热和光动力理论上空的十九世纪的乌云”的演讲。他说，物理大厦已经落成，所剩只是一些修饰工作……现在，它的美丽而晴朗的天空却被两朵乌云笼罩了。
 </blockquote>
 <p data-pid="b6PdZeS8">爱因斯坦：我预言了乌云后的东西，并踩着洛伦兹搭的桥，扒开了一朵乌云，发现里面竟然真的藏着这个世界宏观的秘密，并解释了它，甩出了引力场方程。虽然我也把另外一朵乌云扒拉了一半，发现它可能是微观世界的秘密，但由于和宏观相悖，所以我并不想承认他。</p>
 <p data-pid="pWpDApR7">波尔：你不承认也得承认，就让我们哥本哈根学派解释另外一朵乌云背后的秘密吧。</p>
 <ul>
  <li data-pid="F5hRq4Au">波尔：我发现，<i>一</i>个实验可以展示出物质的粒子行为或波动行为，但不能同时展示出两种行为。</li>
  <li data-pid="tO_SoAO9">海森堡：我来总结一下，这就是不确定性原理：你不可能同时知道一个粒子的位置和它的速度。</li>
  <li data-pid="eIsJYC2c">薛定谔：这个很简单，我用我的波函数就能表达！（甩出薛定谔方程式）</li>
  <li data-pid="IzRD0J2C">波尔、海森堡：但你依然无法解释测量动作带来的粒子变化，我们坚定地认为，测量的动作造成了波函数坍缩，原本的量子态概率地坍缩成一个测量所允许的量子态。（哥本哈根诠释）</li>
 </ul>
 <p data-pid="8VlyPrcF">薛定谔：我完美的薛定谔方程式怎么可能坍塌！绝对不可能！不对！你们肯定是不对的！（物质不可能同时处在是和非的两种状态，然后再经过观察坍缩成一种状态。）</p>
 <p data-pid="yh-k-x2j">爱因斯坦：是的，上帝不会掷骰子！（怎么可能出现坍塌，一定有一个未发现的连续过程）</p>
 <p data-pid="ZvTPPjp2">薛定谔：我支持老爱，我用薛定谔的猫来反驳你们。</p>
 <blockquote data-pid="nxmD1AEZ">
  密闭的铁匣子里放着一个盖革计数器和少量的铀，因为量非常少，所以很可能一个小时内有一个原子衰变的概率和没有衰变的概率一样。当第一次衰变发生时，通过继电器，装置会释放锤子砸碎一瓶氢氰酸。另外残忍的是，还有一只猫设被关在这个铁匣子里 。
 </blockquote>
 <ul>
  <li data-pid="9PBwQ5B-">泡利：真太妙了薛定谔，你的猫就是处在即是生又是死的叠加状态。</li>
  <li data-pid="SUM3U4B6">狄拉克：你的猫正好拿来让我们宣传量子力学了，这不，效果相当的惊人。</li>
  <li data-pid="zfR3Haf4">薛定谔：？？？</li>
 </ul>
 <p data-pid="ym1qsA8L">诺特：在一群物理学家争论不休时，我发现了一个秘密。爱因斯坦形容我是现代数学之母，其实我只是发明了诺特定理：<b>连续对称性和守恒定律的一一对应。</b></p>
 <blockquote data-pid="yYbiCC2n">
  简而言之，我们的世界是宇称守恒的，时间上的连续对称变化（时间平移对称性），导致能量守恒定律。空间旋转对称性（空间各向同性）导致角动量守恒定律。从微观来说，一个粒子的镜像与其本身性质完全相同。
 </blockquote>
 <p data-pid="LWSLMeLg"><b>一众物理学家们（纷纷点赞）：赞同，迄今发现的所有物理定律，没有不宇称守恒的。</b></p>
 <p data-pid="5sMJqtfm">费米：我用费米理论初步解释了弱相互作用，并顺手指导了原子弹的建造。</p>
 <p data-pid="Ug1r7EoJ">杨振宁：老师的理论已经很好了，但宇宙可能并不守恒。θ介子和和τ介子是两种质量、寿命、电荷都相同的粒子，但θ介子却衰变生成两个π介子，τ介子却衰变生成三个π介子。θ和τ会不会是同一种粒子，只不过宇称不守恒？</p>
 <p data-pid="obUKOZfW">李政道：我们进一步计算发现<b>，</b>β衰变也并不符合宇称守恒，我们想要找人帮忙做个实验来证明它。</p>
 <p data-pid="G6gvoC50">费曼：我已经完善了量子电动力学，四大相互作用力解释着宇宙现象，看这物理学的世界已经多么的完美，我不相信宇称不守恒。</p>
 <ul>
  <li data-pid="QSa01h3b">拉姆齐：老师，我也认为宇称不守恒，我想用实验证明。</li>
  <li data-pid="Ow5fY115">费曼：那是一个疯狂的实验，不要在上面浪费时间。</li>
 </ul>
 <p data-pid="W_2dRMtn">杨振宁、李政道：实验已经设计好了，谁来做实验？</p>
 <p data-pid="dTlA-3l6">吴健雄：我来吧。</p>
 <p data-pid="MWyKHfU6">泡利：我愿意用任何东西赌，宇称一定是守恒的。</p>
 <ul>
  <li data-pid="RggzvYy9">朗道：宇称不可能不守恒。</li>
  <li data-pid="18EBPptS">沙皮罗：老师，我也觉得θ-τ介子衰变宇称不守恒，这是我的论文。</li>
  <li data-pid="KC8OZ3c6">朗道：一边去！</li>
 </ul>
 <p data-pid="2RqgP2M8">吴健雄：试验成功了。</p>
 <p data-pid="7Pvg48RI">费曼、泡利、朗道：？？？？？</p>
 <ul>
  <li data-pid="9-jfm3Kp">拉姆齐、沙皮罗：……</li>
 </ul>
 <p data-pid="Tz3bnY_x">众多物理学家：物理学的整个大厦都倾斜了。</p>
 <p data-pid="lJrswiYv">杨振宁：没事儿，我把这个大厦扶住了！</p>
 <ul>
  <li data-pid="fMqNyFz2">爱因斯坦完成广义相对论后，曾试图统一电磁力和引力，但并没有成功。</li>
 </ul>
 <p data-pid="tz17OGQi">杨振宁：我和米尔斯提出了杨-米尔斯理论，为电磁力、弱相互作用、强相互作用提供了一个统一的数学模型。后来我顺手提出的杨-巴克斯特方程，也成为了数学物理的重要方程之一。</p>
 <p data-pid="uhOVZwzz">韦尔切克等众人：在杨-米尔斯理论的基础上，我们引入对称性自发破缺与渐进自由的概念，于是描述强、弱、电磁三力的标准模型诞生了。</p>
 <ul>
  <li data-pid="2Arm6Rlk">至此，只有引力尚未被统一。</li>
 </ul>
 <p data-pid="hFXmi_Df">霍金：我认为大统一理论迟早会被发现，我提出了霍金辐射，并写了几本书。</p>
 <p data-pid="UYrM-b9U">其它众人：我们提出了弦理论等等，试图建立大统一理论，但这些理论本质还是规范场论。</p>
 <p data-pid="oBFAGT6u">丘成桐：我的卡拉比-丘流形是弦理论中的重要概念。</p>
 <p data-pid="vYOVnmlZ">……</p>
 <ul>
  <li data-pid="fWMWPE6k">注：上面的主要内容来源于科学家的生平事迹以及说过的话，少部分合理编撰。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="NB3JAtMH">自1954年杨振宁提出杨-米尔斯理论以来，就已经奠定了他在未来几十年不可动摇的地位。</p>
 <p data-pid="qOky97Qm">甚至有可能在未来数百年，杨振宁都可能是后人教科书上最后的一位物理学泰斗。</p>
 <p data-pid="80w9lGIi">唯有最后建立真正大统一理论的人，其理论框架超越标准模型之外，才有可能超越杨振宁的地位。</p>
 <p data-pid="-_gOsPBy">杨振宁也是最长寿的物理学家，几乎没有之一。</p>
 <p data-pid="XbQjT6o-">他的长寿，或许也是近现代物理学的美好注脚。</p>
