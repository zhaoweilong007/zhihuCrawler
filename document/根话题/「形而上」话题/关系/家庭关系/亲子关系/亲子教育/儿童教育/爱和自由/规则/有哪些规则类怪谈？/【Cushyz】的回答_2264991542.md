 <h2>向量组的线性相关性：</h2>
 <ol>
  <li data-pid="QMinqxef"><img src="https://www.zhihu.com/equation?tex=S%3D%5C%7B%5Calpha_1%2C%5Calpha_2%2C%5Ccdots%2C%5Calpha_m%5C%7D" alt="S=\{\alpha_1,\alpha_2,\cdots,\alpha_m\}" eeimg="1"> 是 <img src="https://www.zhihu.com/equation?tex=F%5En" alt="F^n" eeimg="1"> 中的一个非零向量组。</li>
  <li data-pid="p0EaJH--">S中<b>一定</b>有一个不是零向量的向量。</li>
  <li data-pid="1gQlfFcf">如果你在S中发现了一个元素都是零的向量，请立刻查看下一个向量，<b>不要</b>回顾你已经考察过的向量，否则你有可能<b>永远无法</b>找到非零向量。</li>
  <li data-pid="7V9Mj3mA">如果S有一个线性无关的部分组，可以线性表示S中其余的任何向量，那么这个部分组就是极大线性无关组。</li>
  <li data-pid="zpIdMxAa">S的极大线性无关组是<b>存在</b>的，如果您找不到，请不要惊慌，并且尽快联系您的代数老师。</li>
  <li data-pid="q5O071Iu">S与其任一极大线性组无关组等价。</li>
  <li data-pid="O2kJE-jW">S的任意两个极大线性无关组等价。</li>
  <li data-pid="LjiHOV9T">S的任意两个极大线性无关组所含向量个数<b>一定</b>相等。</li>
  <li data-pid="GBD2tgkY">如果你不慎发现了S的两个拥有不同的向量数的极大线性无关组，请立刻重新阅读第2条和第3条，<b>不要</b>违它们。</li>
  <li data-pid="44iWr9ni">向量组S<b>没有</b>所含向量数量超过n的极大线性组。</li>
  <li data-pid="sRVfnE4u">如果 <img src="https://www.zhihu.com/equation?tex=m%3En" alt="m>n" eeimg="1"> ，那么向量组S<b>一定不是</b>线性无关的，无论m再大也不行。</li>
  <li data-pid="uAJMfIwO">如果有人说自己可以找到S的一个含有n+1个向量的极大线性组，请不要理会他；如果发现他从数学系走出来，并且你无法理解他的行为，请<b>立刻</b>联系数学系相关老师。</li>
  <li data-pid="aaRWv6D5">零向量组的秩是零；非零向量组的秩是其任一极大线性无关组所含向量数。</li>
  <li data-pid="9UajrMjF">S的秩可能大于13，也可能小于666，但是<b>绝不</b>会大于n，也<b>绝不</b>会大于m。</li>
  <li data-pid="4BvVuBOG">m和n都是整数！m和n都是整数！m和n都是整数！</li>
  <li data-pid="guRnmtls">如果你不慎违背了2到14的任意一条，请尽量<b>不要</b>再找您的数学老师，特别是<b>代数老师</b>。</li>
 </ol>
