# 本来只是想装逼，结果装逼太成功让人深信不疑是种什么样的体验？
- 点赞数：6967
- 更新时间：2021年08月23日20时35分40秒
- 回答url：https://www.zhihu.com/question/66645796/answer/246947667
<body>
 <p data-pid="FDVrN4I8">说一个，应该大概三四年前的事了，那时候因为经常混 GitHub，算是国内很早混 GitHub 的，然后经常写博客、写公众号分享，甚至还写了一本免费的 GitHub 电子书，跟所有程序员普及 GitHub 的好处，然后 GitHub 上很多人都关注我，很多国外的大公司那时候不知道谁想到的，就是招人的时候，去 GitHub 上找一些有自己开源项目， star 较多，同时关注人数较多的人发邮件，现在这招都被用烂了，但在当时非常新鲜，有一天我打开邮件，一看惊呆了，竟然收到了来自 Facebook 的邮件，我用我那蹩脚的英语水平竟然还看懂了邮件内容，妈呀，这是邀请我去 FB 面试啊，卧槽啊，当时惊了个呆啊，哪见过这阵仗啊，第一件事就想着，这必须得装个逼啊，于是发了个微博。</p>
 <p data-pid="V2Aajv71">然后在圈内震惊了好吧，那个火啊，被羡慕的，那时候圈内哪见过这种操作，不知道哪个好事的 @ 了我的老板，还说让老板给我加工资，这事被老板知道了，你们猜怎么着？</p>
 <p data-pid="0Mw3u-A3">老板还特么真给我加薪了，而且还在公司内部公开表扬我，说我被 Facebook 挖都没去？</p>
 <p data-pid="EwseXkIv">纳尼？只是被邀请面试好么？老子也还没决定要不要去面试体验下呢好吧？</p>
 <p data-pid="-6ZsR1IO">后来看在给加的工资不少，再加上我那半吊子实力去了也肯定面试不上，就没去。</p>
 <p data-pid="5XT_kU0K">但是这事没完，老板不仅在内部说，还尼玛在融资的时候，经常跟投资人说，我们有个被 Facebook 挖角的工程师，卧槽，关键那些投资人还吃这套啊，公司还真的融资一步步走到现在，活的还挺不错。</p>
 <p data-pid="Nt9NqIeJ">当初的一个装逼，老板不仅深信不疑，还经过他的演变，我在公司一直都挺出名的，成了公司唯一被 Facebook 挖角的工程师，而且投资人也深信不疑，最关键的是，我特么在这家公司混的还不错！</p>
 <p data-pid="EjdF6OHu">所以，奉劝大家，没事多装装逼吧，凭本事装的逼没啥坏处。</p>
 <p data-pid="HtrQDk-0">我有时候经常开玩笑说，老子靠一己之力，装逼让公司融资成功的！逃）</p>
