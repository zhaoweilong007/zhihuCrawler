# 有哪些你从相亲中悟出的道理？

- 更新时间：2019年01月04日13时11分17秒

<body>
 <p data-pid="szDnrD7k">作为过来人，给90/95后的一些建议：</p>
 <ol>
  <li data-pid="Mh7mYtiC">第一次见面，女生选择非常高档的场所，说明她不是冲着你来的。</li>
  <li data-pid="bsM4xffx">第一次见面，男生主动送小礼物，绝非轻慢浅薄，这么做只有好处没有坏处。</li>
  <li data-pid="GGN3MZNg">第一次见面后，女生连续两次回你微信的间隔超过10min，则代表对方无感。</li>
  <li data-pid="_lPS-WuB">能聊下去不代表能好上，80%的相亲死于第一次与第二次Date之间。</li>
  <li data-pid="UbRvifQ4">肉体权重排序：颜值＞性能力＞身材，放之男女皆准。</li>
  <li data-pid="Ys8npuIt">以宽勉的诚意面对父母的介绍，糟糕的心情不会忍耐很久。以抵触的态度应对父母的介绍，会让你既没有好心情，又错过很多优质对象。</li>
  <li data-pid="rPJXQ7o-">大部分相亲对象没有网友说的那么奇葩，因为多数网友只会吐槽糟糕的经历并添油加醋，成功者都蒙在被窝里偷着乐呢。</li>
  <li data-pid="23O3hsp-">见面前一定先看照片，面部没有光影明暗变化的照片，无参考价值。</li>
  <li data-pid="awyjz-wj">不要妄图从第一次见面中就看出男生的性格、心态、财富。</li>
  <li data-pid="xln0H7AS">如果男生相亲时一边扒拉饭一边抬眼皮瞟你，则表明对你有意，哪怕只察觉到一次。</li>
  <li data-pid="C5ocnbLl">相亲时，男生不要找原由碰杯。不要给女生夹菜，非要夹，用公筷。</li>
  <li data-pid="srHaGTk3">如果男生对女生有感觉，单手托腮；如果女生对男生有感觉，趁他大段讲话的时候双手托腮。这两种都是很自然的暗示，注意同时要看着对方的眼睛，无需刻意微笑。</li>
  <li data-pid="d84qNMIf">男生不要穿西服，谨慎穿纯白衬衫，这只会凸显你脸上的缺陷；女生最好不要穿长裙。</li>
  <li data-pid="sUty0szl">迟到不要找原因。</li>
  <li data-pid="_PrFi1m0">赚得多的，聊工资时说税后净入；赚得少的，说税前收入。如果你俩能成，这可以减少很多日后的烦恼。</li>
  <li data-pid="mKS488Qf">不要上来就聊天气，我保证聊完了还是冷场。女生打破尴尬可以直接问男生的公司、工作性质，因为大多数男生都可以通过工作打开话匣子。</li>
  <li data-pid="e63Bl498">男生主动提AA，要么他情商趋近于0，要么他对你不满意。</li>
  <li data-pid="PnQeqzIF">见面后，当天微信无话，女生就不要再问男生“对我什么感觉”了。</li>
  <li data-pid="3HG0jUDp">面对极度木讷的男生，你不要抱着“也许建立关系后他就是个无敌大暖男”这样的想法。</li>
  <li data-pid="k9Tx9rIt">永远不要低估任何一个女生的经历，她只是掩饰起来不给你知道而已。</li>
  <li data-pid="Wn7YA7IJ">不要揪着对方的爱好找话题，大多数人所谓的爱好就是随便说说罢了。真是爱好的，ta说了你也接不上话。聊工作、抱怨公司、骂领导永远比扯爱好更能延续交流，也更切实际。</li>
  <li data-pid="r-DFAZjM">要么只吃饭，要么看电影+吃饭，绝对不要只看电影。</li>
  <li data-pid="NkzvZknZ">绝大多数男生无法控制看到相亲对象丑时的各种微表情微动作，而且绝大多数女生都能察觉到你对她颜值的微反应。</li>
  <li data-pid="OkSHe2fF">相亲完觉得可以聊，晚上睡前一句<i>“今天过得很开心 晚安 ”</i>，看似很书面很呆板，却很能表达自己的态度，很妥当，很礼貌。</li>
  <li data-pid="ogtHemtF">男生如何稳妥介绍自己的情况：<b>家事略说，私事详说，高收低说，低收实说，大面坦诚，小处谦虚，多聊心情，少表观点。</b></li>
  <li data-pid="cLgm_prq">女生如何稳妥介绍自己的情况：<b>家富莫提，家贫略讲，多聊父母，少提兄弟，笑谈困苦，严诉未来，多看对方，少盯手机。</b></li>
 </ol>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="98oC7g4Y">貌似很多人对第5条和第13条有疑问，这里补充解释一下：</p>
 <p data-pid="cJMElopw">第5条实际上是两人确定关系后的情况，也就是上过床了，严格讲性能力不算相亲问题。</p>
 <p data-pid="AJ0D7u20">第13条女生最好不要穿长裙，主要是长裙对于女生身高和气质要求很高，穿好了加分，穿不好在男生眼里简直灾难……个人观点，160以下的女生避免长裙，165以上的女生可以考虑。大部分情况下，女生穿牛仔裤、牛仔短裤配合T恤或运动装就OK了，男生眼睛没那么毒，只要别自己踩坑就行。当然，俩人熟了以后就随便穿吧，他要是敢说个不字，抽他！</p>
