# 大龄剩男该做点什么？
- 点赞数：6179
- 更新时间：2019年10月16日20时44分30秒
- 回答url：https://www.zhihu.com/question/278850286/answer/499319218
<body>
 <p data-pid="1sCQkmEi">这篇应该是最后一次写相亲的文章。我已经脱单了，而且我要把脱单的方法介绍给诸位男同胞，这一定是本问题下最直接、最有效、最可执行的方法，绝非“摆正思想、放松心态、努力向前”这种没用的屁话。</p>
 <p data-pid="U2Eg8ZHF">我的方法需要你具备一点点条件：</p>
 <ol>
  <li data-pid="EK-pwBKG">你是个三观正常、心态正常的人</li>
  <li data-pid="MJinXoBn">你有收入</li>
  <li data-pid="cWOsp5IU">父母、朋友、同事在给你介绍相亲对象</li>
 </ol>
 <p data-pid="s4TgOkJF">怎么样，这条件一点也不苛刻吧？我敢说，只要你满足这三个条件，按我的方法来，保准能成。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="JQKLuepM">我的方法就是：<b>送给每个和你相亲的姑娘一支200元以上的口红。</b></p>
 <figure data-size="small">
  <img src="https://picx.zhimg.com/50/v2-fd4110826bd11e160eb20e34597eae10_720w.jpg?source=1940ef5c" data-caption="" data-size="small" data-rawwidth="1122" data-rawheight="1096" data-original-token="v2-fd4110826bd11e160eb20e34597eae10" data-default-watermark-src="https://picx.zhimg.com/50/v2-55bad92b9821defb655d749637c2a9a3_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1122" data-original="https://picx.zhimg.com/v2-fd4110826bd11e160eb20e34597eae10_r.jpg?source=1940ef5c">
 </figure>
 <p data-pid="Udcjzll9">没错，就这么简单。下面讲原因。</p>
 <ul>
  <li data-pid="9cWLm5en"><b>为什么要送？</b></li>
 </ul>
 <p data-pid="gfyijaex">因为在相亲中，光靠聊天完全不足以让对方判断你是怎样的人。特别是对于年龄稍大的女生，阅历比较丰富，谁不知道在微信上聊天是可以随意刻画人设的？行动比语言更能表明自己的立场。你送一支200多元的口红，这个行为在姑娘看来：</p>
 <ol>
  <li data-pid="eJeK2pRP">你有一定经济能力，才送得起；</li>
  <li data-pid="8ScA5iYA">你不是个吝啬鬼，相反很大方；</li>
  <li data-pid="yePIK8_n">你很重视这次的见面，有诚意。</li>
 </ol>
 <p data-pid="YPpoi3Tp">经济能力、有诚意、不抠门，这是相识之初女生眼中最重要的特质。我就问你一句：要表达出这些信息，光靠微信聊天得聊多久、得费多少心思！三个对女生非常重要的信息，一支口红搞定。</p>
 <ul>
  <li data-pid="tWFLleOb"><b>为什么是200元以上的口红？</b></li>
 </ul>
 <p data-pid="DQ0WZ2PI">首先，200多元的口红一般都是YSL、迪奥这种品牌货，买了绝对不会错。</p>
 <p data-pid="WAGd0iW7">其次，在口红圈，300元已经是很拿得出手的价位了，不像其他化妆品那样没有价格天花板，你不会花冤枉钱。</p>
 <p data-pid="vtFVnhrA">第三，一支口红能用很久，而且口红是很多女生每天都要涂的，每次拿起来都能想到你。</p>
 <ul>
  <li data-pid="MIqShcud"><b>怎么送？</b></li>
 </ul>
 <p data-pid="s_XYz90X">随便找个理由就行。比如：</p>
 <p data-pid="Ve0R-25K">“客户送我的口红，他还以为我结婚了呢，也没问就给我了”</p>
 <p data-pid="UnreaOXU">我是怎么做的？我前后送过三支口红，YSL和迪奥都有，没找理由，直接问“我买了一只**的口红，可以送给你吗？”客客气气的。事实上，不找理由最能体现出“为什么送口红”里面的重要三点。</p>
 <ul>
  <li data-pid="cpCLdmGJ"><b>女生会接受吗？</b></li>
 </ul>
 <p data-pid="QcOktw4s">开局送口红，这操作女生能接受吗？我用亲身经历告诉你——会接受的。第一个是直接欣然接受；第二个是客气了两句，然后接受了；第三个是很高兴地接受了，然后回请了我一顿饭。结局嘛，当然是和第三个女生好上了。</p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-1e767257544cd93c5224bceacaf882c5_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="1095" data-rawheight="1857" data-original-token="v2-1e767257544cd93c5224bceacaf882c5" data-default-watermark-src="https://picx.zhimg.com/50/v2-4659ca3f7b466cbe0b32d8e66b13c3e7_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1095" data-original="https://pic1.zhimg.com/v2-1e767257544cd93c5224bceacaf882c5_r.jpg?source=1940ef5c">
 </figure>
 <p data-pid="Ymw0BenL"><b>“没有女生会拒绝一支迪奥口红”</b>，这句话真是至理名言。她们的接受度之高，以至于男生都不用事先问色号，比如我就直接买的999正红。另外，为了确保正品，我是从狗东自营和天猫旗舰店订的。</p>
 <ul>
  <li data-pid="agasqVZw"><b>送口红之后会有什么变化？</b></li>
 </ul>
 <p data-pid="zFU0JhGV">这么跟你说吧，我的条件也一般，以前不送的时候，相亲都是姑娘挑我；后来开始送口红，立马变成我挑姑娘了——变化就这么大，局势从被动变为主动。口红递过去你真的能明显感到姑娘发自内心的高兴，说实话，送人玫瑰手留余香，那一刻我都觉得值了。其中一个姑娘当天晚上回家后就试妆，还发自拍给我。而且之后的沟通也顺利多了，两个人都能主动聊天，而不是男生挖空心思找话题，女生不冷不热地回复。</p>
 <p data-pid="N0ic2Wlh">如此一来，两个人很快就能走近对方。换句话说，女生多少都是腼腆害羞一些的，口红就像一把钥匙，一旦她接受了，就会打开大门，不会和你隔墙交流。</p>
 <ul>
  <li data-pid="Cfw3vEBK"><b>会不会被姑娘利用？</b></li>
 </ul>
 <p data-pid="LCp3WDje">这就看你的阅历了，我之所以和第一个送口红的姑娘没成，就是看到对方只接受而没有回馈，小脾气也比较多，说白了就是作。所以，送口红能让你掌握主动权，你也得擦亮眼睛，不能姑娘一热心你就晕了。</p>
 <ul>
  <li data-pid="NYaTQ9F0"><b>凭什么上来我就花好几百块钱？</b></li>
 </ul>
 <p data-pid="y3lQS4I0">要是抱着这种思想，那你趁早洗洗睡吧。能一眼让女生看上的，要么气质绝伦，要么颜值靠谱，要么事业有成，要么谈吐有致。这四样但凡有一样归你，你TM能成剩男？咱都这地步了，还不说多付出一下，自然界野鸟求偶还得洗干净羽毛撅着屁股给雌鸟跳一段舞呢吧？现在吃顿像样的饭不也得好几百，相比之下两三百的大牌口红算得了什么？</p>
 <p data-pid="XcjLaPU8">最关键的，你本人得是真心想脱单。你要是个吝啬鬼，我劝你别用这个方法，假装大方，相处下来没三天就得现原形。<b>我的方法是帮助有困难真心想找到合适伴侣的人，不是教渣男骗婚。</b></p>
