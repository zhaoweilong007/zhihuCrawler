# 怎样告诉三岁的小孩爸爸去世？

- 更新时间：2024年03月11日19时44分22秒

 <p>死亡很沉重<span><span>。</span></span>和一个年幼的孩子谈死亡更加沉重<span><span>。</span></span></p>
 <p>曾有一位妈妈向我咨询<span><span>，</span></span>丈夫因为意外突然离世已经 2 个月了<span><span>。</span></span>她和家人一直告诉 3 岁的儿子<span><span>，</span></span>爸爸去很远的地方出差了<span><span>，</span></span>要过很久才能回来<span><span>。</span></span>孩子每天追问爸爸去哪里了<span><span>，</span></span>什么时候才能回来<span><span>，</span></span>爸爸是不是不喜欢他了<span><span>，</span></span>为什么不回来看他<span><span>。</span></span></p>
 <p>她说<span><span>，</span></span>为了孩子<span><span>，</span></span>她必须坚强<span><span>，</span></span>但是孩子不停追问的时候<span><span>，</span></span>她真的觉得坚持不下去<span><span>，</span></span>心都要裂开了<span><span>。</span></span></p>
 <p>她原本打算等孩子大一些再告诉他爸爸去世了<span><span>。</span></span>可是<span><span>，</span></span>现在这个借口可能编不下去了<span><span>。</span></span>孩子虽然才 3 岁多<span><span>，</span></span>但似乎是知道了些什么<span><span>。</span></span>最近追问得越来越少了<span><span>，</span></span>但是情绪越来越低落<span><span>，</span></span>变得不太爱说话<span><span>，</span></span>经常会在纸上画爸爸<span><span>。</span></span></p>
 <p>这位妈妈问我<span><span>，</span></span><strong>究竟要不要告诉孩子<span><span>，</span></span>爸爸去世了<span><span>，</span></span>不会再回来了<span><span>？</span></span>该怎么和孩子说死亡这件事<span><span>？</span></span></strong></p>
 <p>随着年龄的成长<span><span>，</span></span>孩子对死亡的概念逐渐地演化<span><span>：</span></span></p>
 <ul>
  <li><strong>第一个阶段<span><span>：</span></span>2～5</strong> <strong>岁<span><span>。</span></span></strong></li>
 </ul>
 <p>对孩子来说<span><span>，</span></span>死亡很难理解<span><span>，</span></span>尤其是<strong>死亡的永久性</strong><span><span>。</span></span>这个年纪的孩子通常认为死亡是一个暂时<span><span>、</span></span>可逆<span><span>、</span></span>非个人化的概念<span><span>。</span></span>他们在故事里经常看到各种各样的角色<span><span>、</span></span>动物<span><span>、</span></span>花草<span><span>，</span></span>死亡平静得就像是睡着了一样<span><span>，</span></span>或者<span><span>，</span></span>在某种食物<span><span>、</span></span>药水<span><span>、</span></span>魔法的帮助下<span><span>，</span></span>死去的东西又复活了<span><span>。</span></span></p>
 <p>所以<span><span>，</span></span>当说到家人去世时<span><span>，</span></span>如果孩子没有多少反应<span><span>，</span></span>不代表孩子没有心<span><span>、</span></span>不爱自己的亲人<span><span>，</span></span>这可能只是因为孩子对死亡的理解还很懵懂<span><span>，</span></span>觉得家人的离开只是暂时的<span><span>，</span></span>以后还会回来的<span><span>。</span></span></p>
 <ul>
  <li><strong>第二个阶段<span><span>：</span></span>5～9</strong> <strong>岁<span><span>。</span></span></strong></li>
 </ul>
 <p>这个年纪的孩子基本上都能理解<span><span>：</span></span>每一个活着的生物最终都会死亡<span><span>；</span></span>死亡是生命的终止并且不可逆转<span><span>（</span></span>具有某些宗教观念的家庭也许会有不同的看法<span><span>）</span></span><span><span>。</span></span>他们逐渐理解<strong>死亡的非功能性</strong> —— 死去的人无法再做一个活人可以做的事情了<span><span>。</span></span>孩子可能会问你<span><span>：</span></span>死去的人有感觉吗<span><span>？</span></span>人死了<span><span>，</span></span>还能吃东西吗<span><span>？</span></span>还能动吗<span><span>？</span></span>还会做梦吗<span><span>？</span></span></p>
 <p>他们可能会产生对死亡的恐惧<span><span>。</span></span>他们可能知道有些事物意味着死亡<span><span>，</span></span>比如骷髅头<span><span>，</span></span>害怕见到和死亡相关的事物<span><span>。</span></span>他们可能害怕自己会死掉<span><span>，</span></span>害怕自己睡着了以后就醒不来了<span><span>，</span></span>晚上不愿意去睡觉<span><span>，</span></span>或者睡觉的时候做噩梦<span><span>。</span></span>他们也可能害怕家人死亡<span><span>，</span></span>不想让家人离开自己身边<span><span>，</span></span>总要反复确认家人的安全<span><span>。</span></span></p>
 <ul>
  <li><strong>第三个阶段<span><span>：</span></span>9</strong> <strong>岁～青春期<span><span>。</span></span></strong></li>
 </ul>
 <p>到了这个年纪<span><span>，</span></span>孩子很清楚死亡的本质<span><span>，</span></span>他们知道自己和家人终有一天也将死去<span><span>。</span></span></p>
 <p>所以<span><span>，</span></span><strong>在孩子年幼的时候<span><span>（</span></span>比如学龄前<span><span>）</span></span><span><span>，</span></span>就可以借助绘本故事<span><span>、</span></span>逝去的花草等等<span><span>，</span></span>帮助他们理解死亡<span><span>，</span></span>对死亡形成一个正确的认知</strong><span><span>。</span></span>这样<span><span>，</span></span>当孩子逐渐成长的时候<span><span>，</span></span>就不会对死亡那么困惑和恐惧<span><span>。</span></span>当身边的亲人去世时<span><span>，</span></span>孩子在心理上也能有所准备<span><span>。</span></span><span><span>（</span></span>在文末<span><span>，</span></span>我给大家准备了一个关于死亡教育的绘本清单<span><span>。</span></span><span><span>）</span></span></p>
