# 怎么看待恋人的恋爱史？
- 点赞数：10201
- 更新时间：2015年04月03日13时24分07秒
- 回答url：https://www.zhihu.com/question/19556576/answer/12215879
<body>
 <p data-pid="Bxis8HXf">如果你的恋人没有过恋爱史，那么——</p>
 <ul>
  <li data-pid="yEbKg5NR">你的恋人有可能在某些方面不太正常。除非他/她真的很年轻。<br></li>
  <li data-pid="ICjoE1xs">你的恋人的情感成熟度可能值得担心。</li>
  <li data-pid="kswDRv14">你的恋人有可能在某些时候会变得很讨厌。</li>
 </ul>
 <p data-pid="kCzirM4S">如果你自己没有过恋爱史，那么——</p>
 <ul>
  <li data-pid="N6wXp_OP">你自己的情感成熟度是可能值得担心。</li>
  <li data-pid="STTvxC85">请让你自己在和他/她相处的时候，注意不要变得很讨人厌。</li>
 </ul>
 <p data-pid="_yuFmuN1">如过你的恋人有过恋爱史，但你没有过，那么——</p>
 <ul>
  <li data-pid="RcQtZEXp">要感谢他/她之前的恋人帮你调教好了他/她。</li>
  <li data-pid="NH-brnt4">初恋都是用来练手的，你应该庆幸你的恋人和他/她的初恋分手了。<br></li>
  <li data-pid="kvJ_Mn1y">请不要以为恋爱就应该是从一而终的，否则这个世界的离婚率会比现在更高。</li>
 </ul>
 <p data-pid="RuGezuim">如果你自己有过恋爱史，你的爱人也有过，那么——</p>
 <ul>
  <li data-pid="Yejz_J2U">你们是公平的。请尊重对方。</li>
 </ul>
 <p data-pid="7xEzDC6g">如果你们两人都有过恋爱史，但你的恋人的恋爱史比你的更长、更丰富，那么——</p>
 <ul>
  <li data-pid="PtTFafyO">他/她的情感成熟度有可能比你高，请向他/她学习，并感谢有一个比你成熟的人包容你、照顾你。</li>
  <li data-pid="YeBNAlda">如果他/她真的确定了是在你这里安定下来，请为自己感到骄傲。</li>
  <li data-pid="jHnzXdBc">如果他/她并不想安定下来，你也不想，那么请抓住机会，拿他/她练手，这远远强过你和情窦初开的小年轻练习感情。</li>
 </ul>
 <p data-pid="fD1CJnA2">如果你自己有过恋爱史，但比你的恋人的恋爱史更长、更丰富，那么——</p>
 <ul>
  <li data-pid="W9n2DhGz">请让他/她来知乎看我上面的回答。</li>
 </ul>
 <p data-pid="_g0iHygl">如果你很想知道他/她的恋爱史，那么——</p>
 <ul>
  <li data-pid="_IKUCehU">请确保你有足够的情感成熟度去倾听他/她的历史而不往心里去。</li>
  <li data-pid="66KXHmrl">如果你的情感成熟度不够，请确保你有乐观、练达、热情和积极的心态或性格。</li>
  <li data-pid="nWwXebDT">如果你的情感成熟度足够高，或者性格足够好，他/她会主动告诉你他/她的情感史的。</li>
  <li data-pid="UQ8cRRf6">如果你的情感成熟度不够高，或者性格不够好，就不要去主动问他/她，否则难受的是你自己。告诉自己，这不是他/她的问题，而是你自己。</li>
 </ul>
 <p data-pid="MzgIcYnQ">如果你的恋人和前任还保持很好的朋友关系，那么——</p>
 <ul>
  <li data-pid="3XfDhjeX">这很正常。</li>
  <li data-pid="fgmi7n3c">无法和前任保持良好关系的人，有可能情感成熟度不够高。</li>
  <li data-pid="LPbCfGNy">能和前任保持良好关系的人，有可能有着更高的控制力和人格魅力，请庆幸你现在拥有他/她。</li>
  <li data-pid="RzWEn-pZ">请尊重和包容他/她的一切朋友，不管他们是否他/她的前任。如果他/她的朋友是人品极其糟糕的人，那么你的恋人也是糟糕的，与是否前任无关。</li>
 </ul>
 <p data-pid="NfDipNe0">最后送两句压箱底的忠告——</p>
 <p data-pid="1iOHwPR5"><b>无论如何，请信任他/她，否则你这辈子都不安宁。</b></p>
 <p data-pid="kDWr47rb"><b>如果他/她不值得信任，那么请等到你发现的时候再说。</b></p>
