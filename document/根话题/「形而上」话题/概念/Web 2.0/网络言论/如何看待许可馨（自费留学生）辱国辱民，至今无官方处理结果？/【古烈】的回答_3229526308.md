# 如何看待许可馨（自费留学生）辱国辱民，至今无官方处理结果？
- 点赞数：5805
- 更新时间：2023年09月28日08时53分07秒
- 回答url：https://www.zhihu.com/question/391680162/answer/3229526308
<body>
 <p data-pid="KzJMxruQ">许可馨挺无辜的，她说的每句都是大实话，属于你品一下都挺有道理的那种，反映了很多社会现实。尤其是她说跟父母的人脉和苏州的经济比起来，自己的努力一文不值，这简直是人间清醒的发言，但当时甚至就连这也成了她的一条“罪状”。</p>
 <p data-pid="ul3oSw1B">只不过因为2020年那段特殊时期，利用人们对留学生、海外群体的普遍仇视，许可馨才被全网讨伐，但是讨伐到最后什么也没得到。就像50年前，群情激昂的小将们试图从“走资派”家里抄出罪证却演变成打砸抢一样，他们的愤怒、自诩的“正义”，便成了一样最可笑的物事。</p>
 <p data-pid="T5x436Ht">她跟鲶鱼姐、大G姐的性质完全不同。鲶鱼姐是公然以“上等人”的身份，对底层民众进行人格羞辱，而大G姐真正让我们见识到什么叫有权可以为所欲为。然而讽刺的是，针对这两人的惩戒力度、网暴力度，似乎并不比许可馨更大，这才是最令人感到讽刺的事情。</p>
