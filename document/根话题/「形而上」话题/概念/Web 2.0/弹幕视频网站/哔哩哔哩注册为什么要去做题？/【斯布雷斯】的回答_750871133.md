# 哔哩哔哩注册为什么要去做题？
- 点赞数：6244
- 更新时间：2019年07月16日15时10分36秒
- 回答url：https://www.zhihu.com/question/334066856/answer/750871133
<body>
 <p data-pid="DrxzsyYb">这是b站能想到的筛选用户的最好办法了。</p>
 <p data-pid="g-qBspsc">多说一句，以前的题是真的难。像我这种深度死宅都是勉强过关。要是不难不至于淘宝上买会员。</p>
 <p data-pid="Qvaxiwke">为什么呢？因为当时b站还是一个主打二次元作品的弹幕视频网站。很多弹幕视频网站都是成也弹幕，败也弹幕。所以控制弹幕质量非常重要，筛选用户是必然选择。</p>
 <p data-pid="HLW_0oRT">相比之下，现在的题不要太简单。</p>
 <p data-pid="Tlg03CeG">实际上说明b站正在从一个二次元社区转变为集内容创作、直播平台、电商、游戏于一体的综合生态社区。</p>
 <p data-pid="phz_TFPm">即使是弹幕方面，以b站如今的体量和运行模式，即使不筛选用户，也有一套相对成熟的弹幕审查机制，自然可以打开大门迎客了。</p>
</body>
