# 哔哩哔哩注册为什么要去做题？
- 点赞数：8024
- 更新时间：2019年08月06日14时56分30秒
- 回答url：https://www.zhihu.com/question/334066856/answer/777300412
<body>
 <p data-pid="WFItoTQb">这是开卷考试，而且还允许上网搜索，因而只是测量用户拥有基本的<b>文字阅读能力</b>以及基本的<b>网上信息检索能力</b>而已。</p>
 <p data-pid="KmPA33qo">对于允许上网搜索的开卷考试，考题本来就不可能过于简单。</p>
 <p data-pid="jK2xb4Yy">但是目前这些，有一定文化水平的都可以完成。</p>
 <p data-pid="c6RBB1Uf">不能完成的，要么是文化水平严重不足（有胡乱起哄跟风的潜质），要么是耐心严重不足（有成为无脑喷子的潜质），要么是机器人（不是真人）。这些群体就是B站需要筛选出去的群体。</p>
</body>
