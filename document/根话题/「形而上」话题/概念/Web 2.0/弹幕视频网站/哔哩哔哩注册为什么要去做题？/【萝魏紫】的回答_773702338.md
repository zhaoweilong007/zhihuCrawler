# 哔哩哔哩注册为什么要去做题？
- 点赞数：5067
- 更新时间：2019年08月03日18时24分57秒
- 回答url：https://www.zhihu.com/question/334066856/answer/773702338
<body>
 <p data-pid="Vc7vX6lC">猫扑知道吗？</p>
 <p data-pid="PBnWLnls">本世纪初最大互联网论坛，现代中文互联网有一半以上年龄超过两年的经典梗都是猫扑出来的。人肉搜索的发源地，凭一句话没图就能帮人找到十几年的朋友，当年连帝吧之类都是小弟弟，全网三巨头之一，最早的几次网络混战全是猫扑的人搞出来的，号称猫扑的汉子天涯的妹，晋江的姑娘画漫画。</p>
 <p data-pid="3P6ycyIX">猫扑当时，新人每半小时刷新一次页面得到一mp,10mp才可以发帖100mp才可以回帖，你想帖子置顶还要花mp买救生圈，加上各种限制，发图，悬赏之类。内容质量多少高，经典频出。真正的中文互联网半壁江山。</p>
 <p data-pid="GfghmhaG">后来放开注册了，一下子进来几百万小学生，生生把一个最经典的网站搞成今天这种半死不活的样子。</p>
