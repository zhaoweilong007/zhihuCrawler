# 哔哩哔哩注册为什么要去做题？
- 点赞数：8215
- 更新时间：2019年11月18日15时20分29秒
- 回答url：https://www.zhihu.com/question/334066856/answer/899126540
<body>
 <p data-pid="9vJXLLjT">我至今仍记得，多少年前，辛辛苦苦做完100道题（涵盖天文地理二次元中各种你能想到和想不到的） 怀着激动的心情，开开心心准备提交的时候，手抖把页面关掉了…</p>
 <p data-pid="vGKHXEVM">那天，风和日丽，从四楼阳台却传出了一阵鬼哭狼嚎，声音之犀利，之惨痛，前所未有，闻所未闻…就连桌子也在为此悲痛愤恨到颤抖！（绝对不是因为我对桌子做了什么）</p>
 <p data-pid="u62YJOum">大概沉寂了一周，我才从情绪中恢复过来…遂又重新做了一次题…而这套题的难度远远高于我第一次做的那套…</p>
 <p data-pid="fvN2S8oZ">印象最深的是考了一道化学题…对不起，画面太过美丽，考完就不想再回忆！</p>
 <p data-pid="BV0H1rym">所以懂了吧？哔哩哔哩是为了锻炼我们学习持之以恒的能力，是为了让我们在面对高考时能够做到微笑面对，是为了让我们不被社会的挫折所打败，提前模拟情景而已。</p>
 <p data-pid="uvRetNDA">用心良苦，哔哩哔哩！干杯！</p>
</body>
