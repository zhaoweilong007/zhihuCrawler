# 如何评价郭杰瑞？
- 点赞数：7452
- 更新时间：2018年01月20日11时19分01秒
- 回答url：https://www.zhihu.com/question/264893682/answer/299707394
<body>
 <p data-pid="SNFiLrW8">一直都要看郭杰瑞的视频。刚开始看的时候，经常被他结尾那句我是郭杰瑞吓到。（这人该不会是个傻子吧）视频的标题和字幕也明显看得出不是他本人做的，看他视频可以看得出来，做标题字幕的人明显中文水平比他说话的口语水平高出不是一个层次，刚开始我以为是有团队卖他憨厚老实的人设来吸粉，看了几期才发现这人是真的憨厚。从语言表达可以看出来。慢慢看着，就发现卧槽你家房子这么大的，卧槽你家冰块机还有这么多功能，卧槽刷卡还用黑卡。卧槽视频基本每一期都不同的外景（大多数up主都是在自己家录或者校园）。卧槽做了这么久还是没有广告。你好歹搞个公众号二维码提醒粉丝视频更新啊。如果你注意力会发现你要拍他那样的视频。你的物质和文化基础条件成本太高了，只不过他刚好拥有，他很多视频可以看得出，他对食物或者文化方面，他对美国的总结，对中国方面的总结都很到位。而不是像别的up主放大优缺点来对比。请一两个美国人试试食物，问问几个问题就能代表每个美国人。可以参照老干妈的相关视频。（我现在被很多up主引导以为每个美国人都爱吃老干妈）</p>
 <p data-pid="ypB1WtQL">总结：看郭杰瑞视频可以看到中美大众文化总体差异。看他视频，给人的感觉就像你的朋友跟你交流知识，分享他的见识。不是嘻嘻哈哈取悦你，看完后除了笑什么都不记得。</p>
</body>
