# 为什么你慢慢开始不那么喜欢党妹了？
- 点赞数：6958
- 更新时间：2020年11月27日10时13分22秒
- 回答url：https://www.zhihu.com/question/345280206/answer/1597522758
<body>
 <p data-pid="FpSyY7eo">一个是感觉她生活条件变好了离我越来越远了吧。前期她买洛丽塔，虽然已经是我买不起的程度了，但可以看出她真的有点肉疼，就是大家咬咬牙买自己喜欢的东西的那种状态，就觉得她就是和我们一样的人。然后她小心翼翼地拆开、化妆、穿上，是那种花了钱就要郑重对待的感觉。但现在，花多少多少万搭个棚、买个裙，对她来说好像都是不值一提的事了。</p>
 <p data-pid="Patvku1E">还有一个是视频内容千篇一律。不论是什么主题，都美得很一致，就像一个模特娃娃在换发型和换衣服而已。我之前看她视频也不是看她化妆之类，我只是喜欢她开箱介绍化妆品或者裙子的时候，“哇哇哇”那种reaction，真的很有代入感。我记得毛戈平化妆品那一期，我就真的在和她一起哇哇哇。</p>
 <p data-pid="M03FrRCg">b站up主大多与我们同龄，本来我们把他们视作一个有趣的朋友，现在却好像在看着一个商业化的工作者。</p>
