# 如何评价 B 站 UP 主小约翰可汗？
- 点赞数：7878
- 更新时间：2022年10月06日17时44分19秒
- 回答url：https://www.zhihu.com/question/434587590/answer/2174987206
<body>
 <p data-pid="_zksETF_">可汗是宣传口的天才。</p>
 <p data-pid="-p_PQApJ">智利首都圣地亚哥几乎每年都会爆发怀念阿连德反对皮诺切特宪法的游行，但就在两年前，当时刚学了点新词的网民们把智利的游行称为“美丽风景线”（虽说也不是什么大问题，但总给人一种莫名不舒服的感觉）</p>
 <p data-pid="DLdEKq_v">但今年智利游行的视频在国内互联网上变了，“团结的人民永远不会被击溃”成为了新的刷屏词条，尽管不能把功劳全部都归结给可汗，但可汗的科普与宣传毫无疑问让更多的人了解了阿连德在智利所做的斗争。</p>
 <p data-pid="b5hINSN5">同样的还有桑卡拉，在可汗做科普之前，我也不知道，在布基纳法索这个名字拗口的非洲边角料国家，有着桑卡拉这样一位为公为民的同志在做着斗争。还有巴拿马，尽管托里霍斯称不上是我们的同志，但他用自己的生命捍卫了巴拿马的国家主权，仍然是一位可歌可泣的民族英雄。</p>
 <p data-pid="0cHMAg06">虽然小约翰有时候为了兼顾诙谐性，不得不放弃一些东西，但不能因此去责怪可汗，毕竟一个科普类型的视频要同时兼顾趣味性和严肃性并不是一件容易的事情。</p>
 <p data-pid="ZA9Nh9Pr">而可汗最难能可贵的一点在于从不输出情绪，输出观点，他只是把博卡萨、蒙博托所做的荒唐事讲述给观众，只是把美苏英法这些帝国主义国家犯下的恶摆在观众面前，而观众自会有自己的判断和价值取向。</p>
 <p data-pid="dJ6D22s2">从某种意义上讲，小约翰可汗只是一个说书人，可这样一个说书人所讲的故事，远比一百次的批判，一千次的诡辩和一万次的喊口号更为有用。</p>
