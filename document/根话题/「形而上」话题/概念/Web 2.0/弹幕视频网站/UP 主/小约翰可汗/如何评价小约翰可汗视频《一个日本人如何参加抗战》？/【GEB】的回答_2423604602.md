# 如何评价小约翰可汗视频《一个日本人如何参加抗战》？
- 点赞数：5549
- 更新时间：2022年04月05日01时45分40秒
- 回答url：https://www.zhihu.com/question/525760802/answer/2423604602
<body>
 <p data-pid="9VN-2OZB">可以看出可汗把三碗水端平(左壬、兔友、神友)的本事确实是高，这期视频显然是服务于左壬和兔友，他们的对抗烈度不亚于神兔对立。</p>
 <p data-pid="Y1Pe0O19">和中央党校合作，着重介绍野坂参三在抗日战争中的贡献，又在结束时隐晦地暗示他在国际共运的角度中其实是个争议人物。既让兔友满意，也照顾了左壬的情绪。</p>
 <p data-pid="_gzLbzxX">小约翰的粉丝群体少见得横跨左、神、兔三大对立群体，同时还能保持和睦，这在舆论环境日渐极化的当下尤为难得。作为政治学科班出身，他本人的学术水平未见得有多高，但是至少是入了门的，基本方法论还是懂的。他的视频基本以介绍为主，不太添加甚至极力避免自己的观点，靠视频风格和娱乐效果吸引粉丝，同时极力避免争议，这是他非常高明的地方。有这样的本事，能吃上这口饭一点也不奇怪。</p>
</body>
