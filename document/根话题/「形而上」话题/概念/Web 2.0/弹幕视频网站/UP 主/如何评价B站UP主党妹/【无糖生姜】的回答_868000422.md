# 如何评价B站UP主党妹?
- 点赞数：6600
- 更新时间：2021年07月08日08时59分14秒

 <p data-pid="frapim4X">2021年7月8日更新一下吧。</p>
 <p data-pid="a_XQ_iVE">最近看她出事了，我就想起来我好像有这么个回答还是来更新一下吧。</p>
 <p data-pid="3EZ-jmp-">我取关党妹已经有一年多了，原因挺简单，因为我对跳舞是真的没有兴趣。</p>
 <p data-pid="BMN1QGeq">----以下是2019年的回答---</p>
 <p data-pid="fHlueuFY">她是我难得见到的靠自己努力把口碑翻盘的人。</p>
 <p data-pid="stplODqM">两年前？知乎等地对她的评价负面的居多吧，就说只是脸好看，技术糟糕，内容全靠相声撑着，微博上脾气不好爱掐架等等。</p>
 <p data-pid="FR0INwM7">后来我在B站看到她砸钱去系统学化妆了，再后来看到首页推荐了她的妆教视频，整个造型完成后是真的挺漂亮的，最近看到她的欧洲大片系列就果断关注了。累的时候看看漂亮小姑娘还是挺开心的。</p>
 <p data-pid="G-ZUSeMU">坚持做自己喜欢的事情的人真的会发光啊。</p>
