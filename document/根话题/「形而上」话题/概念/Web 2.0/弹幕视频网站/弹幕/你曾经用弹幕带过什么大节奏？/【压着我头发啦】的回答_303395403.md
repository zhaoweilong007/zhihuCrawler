# 你曾经用弹幕带过什么大节奏？
- 点赞数：5691
- 更新时间：2021年07月17日17时59分20秒
- 回答url：https://www.zhihu.com/question/266010668/answer/303395403
<body>
 <p data-pid="QFAVkHlC">弹幕不知道算不算。之前在QQ炫舞里，进一个直播间，然后某主播在社区，被粉丝追。刚好那个区有号，就顺便去玩玩，然后主播来了一句，“谁有坐骑带我飞一下。”</p>
 <p data-pid="2cIlUiYE">我就站在他面前开了个坐骑，带他去了跑不到的地方。（那个地方只能飞行出去，玩过炫舞的都知道社区海上有块儿石头，只有飞行能过去。）</p>
 <p data-pid="pk-MGeMF">然后我点了一下退出游戏......</p>
 <p data-pid="sY31PKmR">回去看直播，满屏幕哈哈哈哈哈哈哈哈哈哈哈哈哈哈</p>
 <p data-pid="bv6PTC4Q">后面那个主播还说了一句，</p>
 <p data-pid="sQWQxpxd">“这个人肯定在我直播间里。”</p>
