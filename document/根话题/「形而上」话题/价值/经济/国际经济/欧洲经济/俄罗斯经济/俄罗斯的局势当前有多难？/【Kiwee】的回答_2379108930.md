# 俄罗斯的局势当前有多难？

<body>
 <p data-pid="bU1aPTMK">====3月11日更新========<br>
  最新进展是，俄罗斯政府有效屏蔽了推特和脸书，有效的意思是真的上不去了，毕竟当年他们也封过telegram，然而失败了。Youtube，Instagram，Tiktok，Whatsapp等还能正常使用(这几个受众面更广一些)。看新闻麦当劳和可口可乐加入要暂停在俄罗斯的经营，然后我赶紧去买了顿麦当劳吃了下，还好价钱都还没变，店也还开着。</p>
 <p data-pid="OA3_vyz9">莫斯科这边目前房价已经起飞了，按卢布计算的话很多都是翻了一倍的涨幅；还有化妆品也是，所以如果有俄罗斯代购说卢布汇率跌了，化妆品便宜，都是骗人的；目前俄罗斯基础物价，肉蛋奶食品价格稳定。</p>
 <p data-pid="m9U9kYMI">评论区指责我“对侵略战争都没立场，这就是立场。” 我还是想解释下，乌克兰俄罗斯历史上就纠结的很，这次战争也是错综复杂，我们中国人就别站队了，俄罗斯乌克兰内部都不缺反思党。不走宏大叙事，从国家角度思考，那就从自己角度想想，你去炒炒期货，粮食，趁卢布贬值，去远东多买点地，粮食，木材。有资源的说服资本来俄罗斯抄底俄罗斯人才市场，你看欧美制裁的欢，美国科技大厂在俄罗斯的研发部门可一个没关，前几天苹果还从华为莫斯科研究院的眼皮子底下挖了一个GPU大佬。俄罗斯的人才资源是真的性价比高啊，莫大，MIPT计算机专业本科毕业生批发价8-10万卢布/月，博士20-30万卢布，签一个莫大的教授当顾问15-25万卢布，你们看看现在的汇率，多好的机会啊！！！乌克兰那边，找个翻译，去挖一挖逃难的哈尔科夫国立大学，基辅国立大学，柴可夫斯基音乐学院这些学校的教授，请他们去中国访问访问，有邻近国家大使馆的帮助，这真的不难。我们公司之前的挖卡巴斯基几个实验室的员工就是他们在中国比赛的时候，找懂俄罗斯翻译去接洽，聊下来的。这不是发战争财，这是帮助当地提振经济和就业！<br>
  ====3月8日原回答=======<br>
  本来不想写回答，看这个造谣的答主 <a class="member_mention" href="https://www.zhihu.com/people/0954e00a332cc1bdd6087ef69eacc2c7" data-hash="0954e00a332cc1bdd6087ef69eacc2c7" data-hovercard="p$b$0954e00a332cc1bdd6087ef69eacc2c7">@留学僧</a> 的赞同还是蹭蹭涨，来一条一条辟个谣，最后放上一些图片证明。</p>
 <ol>
  <li data-pid="WrQcaSFV">Whatsapp无需挂VPN可以正常使用，在俄罗斯我只碰到过LinkedIn无法使用的情况，不过领英被封是很多年前的事了，原因是不同意隐私数据本地存储。</li>
  <li data-pid="QvVDo-UZ">Youtube，Facebook，Instagram等均可正常使用，谷歌服务除Google Pay以外，均可正常使用，且访问速度跟之前无区别。</li>
  <li data-pid="JyhT4mkt">并非所有欧美品牌下架，只有少量品牌下架。本着实事求是的精神，我今天去公司楼下商场看了下，Zara确实关了，但是阿迪还在，Nike没注意有没有关。</li>
  <li data-pid="x65ra3nl">莫斯科居民疯抢宜家产品这条算是半真半假。宜家说因为供应链关系会很快关闭，所以当天有很多市民去抢购，但是车直接停在大马路上造成严重交通堵塞这个闻所未闻，因为我家离宜家并不远，这几天我的出行并没有碰到什么严重交通堵塞。</li>
  <li data-pid="POkegcNT">禁飞所有国际航班这条，我前几天想回国处理融资的事情，看了下机票，大概5万人民币一张，价格劝退。即使现在在去哪儿上查的话，5月初的机票还是在卖的，6万一张了已经。我之前看到的消息是，俄欧互相禁飞了，然后俄航的国际除飞明斯克外也停飞了，其他消息不清楚。</li>
  <li data-pid="ewpyoCOi">工作上与欧洲有合作的项目全部暂停或取消，这条应该是谣言，别的不知道，莫斯科大学，MIPT等和欧洲高校的交换项目的那些学生都还在欧洲，之前说在欧洲留学的俄罗斯学生要被遣送回国也只是炒作，并没有施行。</li>
  <li data-pid="ZA_6YYGT">涉及到汇款的这种，都是谣言，因为俄罗斯目前只有几个银行被移除SWIFT，甚至最大的银行Sberbank因为欧洲需要用其进行能源交易，都没有被禁。我在俄罗斯做AI相关的IT创业，公司开户在Sberbank，目前SWIFT使用是正常的，从香港汇款到莫斯科完全没有问题。而且，这波制裁是利好我这样在俄罗斯进行创业的外国公司的（市场不在俄罗斯本土），因为汇率原因，目测这个月的开销，甚至只有上个月的一半了，国内公司完全可以来俄罗斯市场进行一波人才抄底。</li>
  <li data-pid="yHDDpsMe">广播，电视循环24小时播放战争新闻和政府的演说。啥年代了，还有广播？指的出租车上的新闻么？还有多少人看电视？之前在群里和朋友们开玩笑，我关于俄乌战局的所有信息，都是群里他们告诉我的，因为那时候我的同事们还在讨论哪家的薄林饼好吃（最终теремок胜出）。</li>
  <li data-pid="OPDjXd_i">最后那句感叹的话，我倒是也想到一些最近发生的事情。我助理的哥哥也上战场了，她是布里亚特人（跟中国人长得没什么区别），她们都不敢告诉父母，哥哥去前线了，也是怕家里人担心。她哥哥会隔几天打一个电话回来报平安，但是不会透露部队所在位置。他哥哥是恰克图（远东地区）的росгвардия，也就是联邦近卫军部队的，有别于俄罗斯联邦武装力量，只能算准军事组织。如果把俄罗斯近卫军等准军事组织算上的话，俄罗斯目前能动员的军队应该在200万人左右，我也不知道这个仗能打多久，但是从动员层级上看的话，俄罗斯应该是做好了大打的准备的。另一个例子是，助理的表妹目前在医学院大三，她们大四的学姐学长们被告知，随时准备上战场，这个我不知道是真的连没毕业的医学生都要准备动员还是只是一手准备，只能说俄罗斯为这次行动做的准备很深远了。其他同事聊的不多，助理一开始是非常反战的，因为听说要把她妈妈的部分养老金拿去支援顿巴斯，她吐槽了好久，但是仗打到现在，她看到那些被抓的俄军俘虏的视频，看到乌东之前屠杀的新闻，也开始动摇，希望前线顺利了。印象最深的就是她说的“我们俄罗斯人的命就不是命了么”，可能大家对这句话没什么感觉，但是她之前，经常说的都是“他们俄罗斯人怎么怎么样”，这里的俄罗斯人指的是俄族人，因为她是俄罗斯的少数民族。可以说，这一战以后，俄罗斯的凝聚力都上升了。</li>
 </ol>
