 <p data-pid="mDOc2xmh">有两位知友提到安塞尔亚当斯的构图很普通。亚当斯对摄影界最大的贡献的确是遵循F64小组所推崇的理念，进而产生的「区域曝光法」。但不能表明他的构图就很一般了啊，忘记在哪里看到的一句话，说可能是因为安塞尔亚当斯学习过乐器的缘故，在其的照片之中我们似乎能感受到画面的节奏和律动感，我想构图也是这种律动感产生的因素之一吧。不过在这里提到他完全是出于个人对他的喜爱。</p>

 <p data-pid="-Ofc56j_">-------------------------------------------------------------分割线-------------------------------------------------------------------------</p>
 <ul>
  <li data-pid="mgUvFb8i"><strong>何为构图？</strong></li>
 </ul>
 <p data-pid="xAsDd6kw">构图，就是把构成整体的那些部分统一起来，在有限的空间里或平面上对作者所表现的形象进行组织，形成画面的特定结构，借以实现摄影者的表现意图。</p>
 <ul>
  <li data-pid="XLQFgix2"><strong>构图的基本要求是什么？</strong></li>
 </ul>
 <p data-pid="TGW37OwS">多样，统一，简洁，有表现力。</p>
 <ol>
  <li data-pid="iOATHohR">多样：就是要求画面富有变化而不呆板，符合视觉上的美的要求。</li>
  <li data-pid="ulpYqkiu">统一：是要有统一目标，不杂乱。</li>
 </ol>
 <p data-pid="SCwykKcY">多样统一构图法则上的遵循，首先是形状的多变化，其次是光线影调色彩背景等因素的运用。</p>
 <ul>
  <li data-pid="H3gagjvW"><strong>如何在构图中做到「多样统一」？</strong></li>
 </ul>
 <ol>
  <li data-pid="mbf3eqwd">寻找被摄体内在的联系。</li>
  <li data-pid="H3SsKgmZ">用全局的眼光把握取舍。</li>
  <li data-pid="DeeBHh3U">自觉培养框取意识。</li>
  <li data-pid="hIsvDYaF">寻找并构建活的框架。</li>
 </ol>
 <br>
 <ul>
  <li data-pid="L4trljLd"><strong>构图的大概分类。</strong></li>
 </ul>
 <ol>
  <li data-pid="VLQWGkGv"><strong>传统式构图「封闭式构图」</strong></li>
  <li data-pid="6NSOcEke"><strong>现代式构图「开放式构图」</strong></li>
 </ol>
 <br>

 <p data-pid="mwQhVdlX">顾名思义，传统式构图画面严谨，画面感十足，严谨范，严谨美。</p>

 <p data-pid="2AUoMhm0">下面是开放式构图。</p>
 <ul>
  <li data-pid="kdYUvPvW"><b>什么是开放式构图？</b><br></li>
 </ul>
 <p data-pid="nRE2ep6M">开放式构图是现代构图的主要方式。拍摄「美国人」系列的罗伯特弗兰克是开放式构图的鼻祖，后面会说到。开放式构图打破了边框的界限，画面，结构具有向外延伸和扩展的趋势，着重表现画面向外部的扩张力，强调画面的内外间联系。</p>
