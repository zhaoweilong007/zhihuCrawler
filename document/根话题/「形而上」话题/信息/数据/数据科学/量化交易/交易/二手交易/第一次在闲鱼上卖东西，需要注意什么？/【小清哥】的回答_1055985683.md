 <p data-pid="ZQBdjQwY">大家好，我是小清哥，一个5年资深闲鱼玩家。</p>
 <p data-pid="C5iDPeSI">新手玩家闲鱼卖货，看这一篇就够了！绝对干货！</p>

 <p data-pid="eSFXJ_dp">我会结合我自己同时作为【买家】和【卖家】的视角</p>
 <p data-pid="6vjKvUsb">用最简单易懂的语言，非常清晰的告诉你，第一次在闲鱼卖东西，需要注意什么！</p>
 <h2>一、一定要远离事逼</h2>
 <blockquote data-pid="Ji4bs0Zg">
  <b>请记住这句话：事逼是万恶之源</b>
 </blockquote>
 <p data-pid="zzcFHIot"><b>1、事逼的危害</b></p>
 <ul>
  <li data-pid="AFJxHXLk">他们会恶意找茬；</li>
  <li data-pid="DB6QvHtr">他们说话很难听，会惹你生气；</li>
  <li data-pid="jhggkuOC">他们会有恶意退款、退货的风险；</li>
  <li data-pid="e55xeCTo">他们会有收货后不确认，反而挑刺砍价的风险；</li>
  <li data-pid="057nhjZq">他们会有恶意差评风险；</li>
 </ul>
 <p data-pid="zlj2673x"><b>2、如何辨别事逼</b></p>
 <ul>
  <li data-pid="HCziTcyd">他们不看商品介绍，反复咨询提问，耽误你的时间；</li>
  <li data-pid="b8A8MsJx">他们会提出各种理由，想占便宜，而且要求特别多；</li>
  <li data-pid="yiFyGdAR">他们说话态度很差，就像你欠他钱一样；</li>
  <li data-pid="i1hE7g0J">他们会想尽办法挑毛病砍价；</li>
  <li data-pid="ir8XCSwq">他们会跟你装可怜，博同情；</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-4269fbde28a7761785555913c0b88d0a_720w.jpg?source=1940ef5c" data-rawwidth="240" data-rawheight="240" data-size="normal" data-caption="" data-original-token="v2-4269fbde28a7761785555913c0b88d0a" data-default-watermark-src="https://pica.zhimg.com/50/v2-1fe209fa3fb3ec4f1916322b4b2eda66_720w.jpg?source=1940ef5c" class="content_image" width="240">
 </figure>
 <p data-pid="-rOJdqjU"><b>3、如何应对事逼</b></p>
 <ul>
  <li data-pid="zc0Wcf-4">直接拉黑！直接拉黑！直接拉黑！；</li>
  <li data-pid="ug9ImwwZ">直接在商品描述里写上：不接刀，事逼勿扰；</li>
  <li data-pid="yFcY1aV3">如果你想找虐，硬要卖给他，那请做足准备；</li>
 </ul>
 <h2>二、一定要能对付砍价</h2>
 <p data-pid="yIWQY2VV"><b>1、如何定价比较好</b></p>
 <ul>
  <li data-pid="43uktSgO">第一步到淘宝、京东了解全新价格，不能卖得比全新的还贵吧；</li>
  <li data-pid="vrdHADlW">第二步到闲鱼搜索，对比同行们的价格，不能比同行便宜太多吧，不然就亏了；</li>
  <li data-pid="aDgTXkks">第三步根据全新的价格和同行的价格，以及商品的新旧程度，评估出自己心里价格；</li>
  <li data-pid="mkgxz4cH">第四步先确定自己的底价，低于XXX元不卖；</li>
  <li data-pid="N1pWpcXH">第五步在底价的基础上增加5%-10%，作为自己的售价，原则上不要超过闲鱼平均水平太多；</li>
  <li data-pid="qapG-M8R">第六步要合理设置运费，如果售价高于平均水平，顺丰包邮可以加快售出的速度（包邮需排除新疆、西藏等偏远地区）。如果售价低于平均水平，最好就不包邮，用闲鱼的自动运费模板。</li>
 </ul>
 <p data-pid="9ddrrrkn"><b>2、如何对付砍价</b></p>
 <ul>
  <li data-pid="yyMksIdG">首先确定是着急出呢？还是慢慢出？；</li>
  <li data-pid="bxyuUToR">但不管自己急不急着出，都要说是慢慢出，不要给暴露心理上的漏洞；</li>
  <li data-pid="2_1CLOfR">商品描述里，永远都要写“不接刀”；</li>
  <li data-pid="i7qiVn23">商品定价上要留有被砍价的空间，不要直接挂最低价，万一出不掉后面可以降价；</li>
  <li data-pid="BVfbJ4pL">一定要守住自己的底价，内心不要动摇！如果你接第一刀，会有更大的刀等你；</li>
  <li data-pid="m3sdt18W">如果着急出，对方出价高于底价就接受；</li>
  <li data-pid="RRxbO8HK">如果遇到“屠龙宝刀”，别生气，直接回复“出不了”，别墨迹，远离事逼；</li>
  <li data-pid="OK7sXv9c">如果遇到砍价带嘲讽、侮辱的，请先长按信息举报后，再拉黑；</li>
 </ul>
 <h2>三、一定要懂的销售话术</h2>
 <ul>
  <li data-pid="xCBnQj_O">想要快速出手，不会销售可不行！</li>
  <li data-pid="1bSjkoOZ">尽量不要主动推销自己的宝贝，给别人很想卖的感觉，这样谈判自己会处于弱势；</li>
  <li data-pid="gO9t4T61">适时的营造非常火爆感觉，让对方着急想要买你的宝贝；</li>
 </ul>
 <blockquote data-pid="jsWOeiOl">
  “不好意思，想要的人有点多，回复比较慢”
  <br>
  “别人出价比你高，你考虑一下”
  <br>
  “想要的话，直接拍就可以，我尽快发货”
  <br>
  “错过以后就很难再碰到这么合适的宝贝了”
 </blockquote>

 <h2>四、一定要会优化曝光</h2>
 <p data-pid="mqPqSVnp"><b>1、优化曝光的好处</b></p>
 <ul>
  <li data-pid="jJbi37bN">让宝贝被更多人看到，甚至获得官方首页推荐；</li>
  <li data-pid="lgFfzioj">咨询的人会变多，所以是我们挑买家，而不是买家挑我们，不怕出不掉；</li>
  <li data-pid="p1AoYe-z">加快出手的速度，卖得更快；</li>
 </ul>
 <p data-pid="tVwcrELd"><b>2、如何优化曝光</b></p>
 <ul>
  <li data-pid="EB3egGg0">图片要拍好：记住①背景纯粹干净②主体突出完整③画面明亮清晰④多角度展示；</li>
  <li data-pid="_bcz_8LB">标题优化好：别人会用什么词语搜索你的宝贝，你就用什么词语做标题，不同词语都加上；</li>
  <li data-pid="toEVODpX">标签要贴准：在最后一张图片贴标签，标签就是把标题的搜索词，分别拆开贴上；</li>
  <li data-pid="3eU6MvZI">提高账号权重：新手卖东西，平常多点赞、留言、咨询互动可以提高账号权重；</li>
 </ul>

 <h2>五、一定要注意防骗</h2>
 <p data-pid="DD3YJPK-"><b>1、交易保证金（开通买卖服务）骗局</b></p>
 <ul>
  <li data-pid="TjsyyXKE">闲鱼上卖东西，是不需要缴纳任何保证金的；</li>
  <li data-pid="g7DA2sWg">也不需要开通任何什么《买卖服务》；</li>
  <li data-pid="_vp8qOXS">闲鱼官方是不会给你发任何私信的；</li>
  <li data-pid="aoUzi4bT">骗子会假冒官方头像，通过私信给你发一些莫名其妙的通知，其中就包括缴纳1000元交易保证金，千万别信啊；</li>
 </ul>
 <p data-pid="niPqlF4M"><b>2、脱离闲鱼平台交易骗局</b></p>
 <ul>
  <li data-pid="k9bRp-uD">凡是让你去其他平台沟通、交易的人99%是骗子；</li>
  <li data-pid="0AqO6FW9">一般借口就是支付宝没钱，可不可以微信转你；</li>
  <li data-pid="JMw6-RZh">遇到这种情况，跟他说必须在闲鱼交易，或者直接不鸟他就好了；</li>
 </ul>
 <p data-pid="SeKMCf9a"><b>3、收货后掉包骗局</b></p>
 <ul>
  <li data-pid="ChsvVtiv">买家在收到货后，以“假货”、“货不对板”、“质量问题用不了”等理由申请退款，并且会拍一堆照片和视频给你看；</li>
  <li data-pid="aapZPAOh">如果你确认自己没坑他，那么就一定是他在掉包；</li>
  <li data-pid="cJ1OOGaR">遇到这种情况，如果事先留有证据，证明他手上的不是你发过去的产品，你完全无需担心；</li>
  <li data-pid="6L4EvjDK">如果你没有证据，那么你自求多福吧，即使他退回给你，也是把掉包的假货推给你；</li>
  <li data-pid="3l2-9-HC">所以售前一定要说清楚，发货要留证据，避免掉包；</li>
  <li data-pid="pkc-8Md1">其次贵重物品，一定要发顺丰到付，到付一定要当面确认，没有问题才可以签收的，他也不好掉包；</li>
 </ul>
 <p data-pid="3FZs4Nee"><b>4、发货后修改地址骗局</b></p>
 <ul>
  <li data-pid="DZUA-cS-">买家在你发货后，突然跟你说地址填错了，让你把货物中转到其他地址；</li>
  <li data-pid="FN9Jha6K">到时候他就会申请【未收到货退款】，说东西没拿到；</li>
  <li data-pid="mI7Xe7PS">所以交易前确认地址，一旦发货就不要修改地址，地址是买家填的，要他自己负责；</li>
 </ul>
 <p data-pid="AsLc-o-r"><b>5、当面交易，事后退款骗局</b></p>
 <ul>
  <li data-pid="5R77Gztl">在当面交易时，买家拿到东西后，没有当着你的面，在APP点确认收货；</li>
  <li data-pid="Wd7rGMcs">等你们各自回家后，买家会申请退款，说自己没有拿到东西，闲鱼平台一般会同意退款，而且不会帮助你维权；</li>
  <li data-pid="q_PG5uDk">如果发生在你身上，你能做的就是自己收集他拿到货的证据，比如交易地点的监控录像，然后报警，警察一般也不会立案，只会帮忙协商；</li>
  <li data-pid="9nCfP_9v">所以当面交易时，一手交钱，一手交货。一定要买家当面确认收货；</li>
 </ul>
 <p data-pid="TjInXXk7"><b>6、非全款交易</b></p>
 <ul>
  <li data-pid="nm8BBhMQ">买家会跟你说，现在没那么多钱，可不可以分期付款，或者先付个定金，让你先发货；</li>
  <li data-pid="4JU59elv">闲鱼是不支持任何形式的分期付款的；</li>
  <li data-pid="2sZsqJBE">如果你改了价，发了货，只让买家付了一部分钱，他收到货之后，立马确认收货，你剩下的钱也不会给你了；</li>
 </ul>
 <p data-pid="LCnDgPu8">大家还遇到什么奇葩的骗术，欢迎评论分享</p>
 <h2>六、一定要避免违规</h2>
 <p data-pid="gGE5WLOO"><b>1、常见要避免的违规类型</b></p>
 <ul>
  <li data-pid="loaWFjdT">不要发布违规商品：闲鱼不允许卖药品、医疗器械、食品(新鲜水果除外)；动物或动物制品；违法服务(在线招嫖、代考作弊、风水算命等等)；</li>
  <li data-pid="vDWJug4h">不要发布假冒商品：卖高仿、A货等各种形式的冒牌货，被发现直接封号；</li>
  <li data-pid="tdb-Ewe-">不要发布虚拟商品：闲鱼上不允许卖任何虚拟资源商品(网盘资源、盗版课程、影视资源等等)</li>
  <li data-pid="oyZ7LndX">不要频繁切换登录手机：不要在不同手机上来回登录自己的闲鱼账号，会被封；</li>
  <li data-pid="l4SYKJ3r">不要有违规行为：不要人身攻击别人，不要发布垃圾引流广告；</li>
  <li data-pid="azp8kSVR">不要发布违规信息：比如微信、V、VX、WX等各种变种引流广告都不能发，具体很多，这里篇幅有有限，不详细写；</li>
  <li data-pid="p-TVJBLK">不要借账号给别人：账号是最宝贵的财富，一旦别人做违规的事情被封，得不偿失；</li>
 </ul>
 <p data-pid="sXoMxYnC"><b>2、违规后的惩罚</b></p>
 <ul>
  <li data-pid="S-5Jo8IK">一般发布信息违规是禁言7天，7天内不能有任何互动，发布任何信息，包括商品；</li>
  <li data-pid="NfTQmueW">其次发布违规商品，商品会被直接下架；</li>
  <li data-pid="IWK3rE1T">当账户违反了一般/严重/违规行为，单项累计达十二分节点即被系统处以店铺屏蔽、限制发布商品及公示警告七天等节点处罚。</li>
  <li data-pid="kvJGkhci"><b>举例</b>：某卖家因滥发信息被扣十二分，同时又因违背承诺被扣十二分，达到了二十四分的处罚节点，该会员应被执行店铺屏蔽、限制发布商品及公示警告十四天的处罚。</li>
 </ul>

 <p data-pid="At-fvb8e"><b>3、违规了怎么办</b></p>
 <ul>
  <li data-pid="4OkuB6JL">被禁言7天，如果不是真的误判，申诉没用，直接老老实实等7天，自动解封；</li>
  <li data-pid="IDJcwg7O">账号被永久封禁，系统是无法解封的，需要联系人工电话客服，真诚道歉，请求解封，有一定的几率成功；</li>
 </ul>
