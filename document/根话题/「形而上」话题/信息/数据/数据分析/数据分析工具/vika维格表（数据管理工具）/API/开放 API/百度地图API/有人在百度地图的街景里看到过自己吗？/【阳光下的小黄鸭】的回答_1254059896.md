 <p data-pid="GMMel6c1">大黄陪伴之---------睡觉篇章</p>
 <p data-pid="PXhgSbfG">反正就是从那个时候开始她就成了我第一个跨种族的小伙伴，刚回来小小一只，拳头大小，我小时候对她很照顾，老是担心她吃不习惯，适应不了新环境，经常做做作业发呆就想她，或者在床上眼睛一闭上会想起她，然后就去看她在干嘛。</p>
 <p data-pid="tkOUeAqc">狗的成长是很快的，几个月时间就长大了。</p>
 <p data-pid="zrwPOv_0">大黄非常的成熟，给我的感觉就是定心，小时候我爸妈开的小吃店，晚上回来会很晚，我又特别怕鬼（那时候爷爷去我大伯家住了一阵），睡觉前都要看看橱柜，看看床底的那种，有的时候很怕，都在床上发抖，然后屁股尿流的跑出去。月光很亮，照在大地上，每一处我都看的非常清楚，大黄就趴在铁门口守着大门，看到我回过头来跟我摇尾巴，我就特别的安心，我就想，这景色多么美啊，然后就着我们家大院就躺在地上了，旁边是大黄，我能听到她的呼吸声，特舒心。那时候没有那么多污染，星星太多了，铺满整个天空，正值夏季，身边蛙鸣蝈蝈叫，我就想这么美为什么我会去想到鬼，我就不想进房间睡觉了，想着索性就大院躺一晚上吧，结果没多久被蚊子叮的收不了，才回去睡觉。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="AfXvPndu">大黄陪伴之------------吃饭篇</p>
 <p data-pid="-3U2k3Z8">现在想起来最后悔的就是没有给大黄一个非常非常大的牛腿骨作为奖励。直到看到了网上有人买了大腿骨给自家狗子盘了后，再想实现却再没有办法实现了。</p>
 <p data-pid="DOXj-JN8">小时候吃饭，大黄就在旁边坐着。</p>
 <p data-pid="RWcHrS6J">我们家的大黄吃饭，和我们自己吃饭，我们非常尊重彼此。我在吃饭的时候大黄不会离得特别近，她会坐在一米左右的距离看着你，这是她5岁前会做的，5岁后基本很少这样了。那时候吃饭不像现在大家坐在桌子上，大夏天的，也没有空调，一家人坐在室外大院子里，拼几个板凳，围着坐着吃，我都会心急火燎的吃，迫不及待的吧吃掉的骨头吐出来给大黄，一开始的大黄会直接叼了就走，趴在一遍用两只前爪搭在骨头上慢慢啃，看着她如愿以偿的啃骨头，我才感觉我这顿饭吃的香，吃的值得。</p>
 <p data-pid="zSjkU4iO">大黄很好养活，正宗中华土狗，并且啥都能吃，我们一般是将剩菜拌着给她吃，鱼刺什么的她都能吃，并且不卡，真是神了，我一直想学她的这个技能，因为我吃鱼，经常被卡，哈哈。如果吃了肉，我们会用肉汤拌饭，炖了鸡，会用鸡汤拌饭，有时候我爷爷会去菜市场买一些别人不用的淋巴肉，鸡屁股等，回来和米一起煮一锅肉汤饭给大黄吃，偶尔爸妈开的小店也会带一些剩菜剩饭给大黄。反正一句话，大黄真的好养活。</p>
 <p data-pid="G3Bl6xfQ">大黄也有一个癖好，离我家70米远，过一会拐角有一个小型垃圾站，附近住在大楼里的会把一些生活垃圾扔在那里，于是，垃圾站变成了大黄搜寻宝藏的藏宝阁。而且从垃圾站里翻出来的骨头等，感觉她吃的更香，哈哈。当然，毕竟这片区域不止她一只狗，为了这片宝地，一公里范围内的狗经常彼此之间相互斗争，这些我会在后面的篇章中给大家介绍。</p>
