# 有人在百度地图的街景里看到过自己吗？
- 点赞数：59218
- 更新时间：2020年05月31日01时41分16秒
- 回答url：https://www.zhihu.com/question/285331801/answer/1225534555
<body>
 <p data-pid="7zA7FRvF">没有看到自己，但看到了在院子门口和路人聊天的，已经去世两年多的爷爷。</p>
 <p data-pid="CAa34bbs">晚上怎么都睡不着，开始在百度地图里搜自己呆过的一些地方，想重拾一些过去的回忆。百度全景有一个功能叫做时光机，点击一下可以看到几年前的街景，我找到了自己小时候生活的老街，点了下时光机，发现最早可以回溯到2015年3月，四处转动视角，果然看到了那些已经被拆掉的老房子。当我想放大照片多回味一些细节时，突然发现一个熟悉的身影正坐在凳子上晒太阳，和别人聊天。起初有点不相信自己的眼睛，往前移动了几米再看，眼泪一下子就下来了。那个消瘦的驼背的身形我太熟悉了，就是已经去世两年多的爷爷。</p>

 <p data-pid="dWQxSMGN">我把妈妈叫了过来，她也认出了这是爷爷，那个时候爷爷还能自己走路，经常坐在这里晒太阳。我开始疯狂点击这条街上的每一个位置，然后再回溯到15年，从里面找不同角度的爷爷的身影，直到来到街的拐角，爷爷已经完全被车给挡住了，看不到了，我的眼睛也早被眼泪模糊了。</p>
 <p data-pid="tuC_QqbM">我从小跟着爷爷长大，童年记忆里全是他带着我玩的场景，在河边抓知了，捉蚂蚱，捉鱼捉虾，爷爷退休前是公司经理，对谁都特别严厉，唯独对我很溺爱。小时候不懂事，记得有一次想捉弄爷爷，假装触电趴在桌子上，爷爷进来拍了拍我见没反应，抱着我就往外冲，后来才发现是我恶作剧，狠狠地收拾了我一顿，还为此生了场病，因为这件事我一直都很愧疚。小时候我常对爷爷说，一定要好好读书，等长大了去大城市买套大房子，然后把他和奶奶接过去享福，每当这时爷爷就特别开心，夸我懂事。但后来去外地上了大学，便很少回家了，每次回去总能看到爷爷坐在院门口等我，我能感觉到他很开心，但他总让我快走，不要耽误学习。</p>
 <p data-pid="emc8VP21">再后来，我读了研，爷爷也慢慢开始糊涂了，有时连爸爸和伯伯都不认得了，但关于我的事他都记得一清二楚，总是问我现在在哪儿上学，然后和我讲小时候的故事，有次我和爷爷说了很久的悄悄话，他特别高兴，但第二天就又都不记得了，问我大学毕业了没有。最后几次见爷爷时他已经不能讲话了，我坐在他旁边拉着他的手，过了一会看到他眼角有泪淌了下来。</p>
 <p data-pid="wRMMQKl0">爷爷去世那天我正考博复试，接到妈妈电话说让我快回去，我买的晚上的票，赶到家时已经是凌晨了，我爸给我发了个定位，打车到那以后看到门口挂着一个电子标牌，上面的红字滚动着 深切悼念XXX同志，我站在门口盯着那标牌挪不开步，楞了有十多分钟。</p>
 <p data-pid="bg52WGW6">已经两年多了，到现在我潜意识里还是接受不了爷爷已经去世的事实，总是梦到小时候他骑车带我到处逛，总觉得他一直都还在那里晒着太阳，等着我回去。而我，也还有机会再好好孝敬他。每想起这个，都心如刀绞。</p>
 <p data-pid="AvG1jo_u">我没有买到大房子，也没有机会再孝敬您。对不起，爷爷，我食言了，</p>

 <p data-pid="-6w5fx-K">一开始只是想找个没人的地方倾诉下对爷爷的思念，没想到收到了这么多条回复和鼓励，看着大家相似的经历，真的很感动。</p>
 <p data-pid="psmK6IeR">小时候梦到亲人不在了，醒来后发现是场梦，心里总是格外地有安全感，而这一天真的到来时，自己就像是只被丢在天台上的猫，再也无处躲藏。这种感觉，失去至亲的人应该都懂。 如果你还有机会再陪陪长辈，请一定一定要珍惜。能这样无条件爱你的人，以后可能真的碰不到了。</p>
 <p data-pid="VTAx6Hw5">有小伙伴留言说也通过百度全景找到了逝去的亲人，由衷的为你开心。如果没有，也请不要难过，就像寻梦环游记里说的那样，每一个我们怀念的人都会活在我们心中，陪伴我们成长直到老去，只要我们活着，还记着他们，他们就不算真正离去。长辈的温存犹在，我们也要余生幸福。</p>
 <p data-pid="13MpvtQv">我开始相信人是有灵魂的，不然为什么会在那么失落的一天，以这种方式再次见到了爷爷。</p>
 <p data-pid="gQfSVPQV">而你们，大概就是他找来的天使吧。</p>
