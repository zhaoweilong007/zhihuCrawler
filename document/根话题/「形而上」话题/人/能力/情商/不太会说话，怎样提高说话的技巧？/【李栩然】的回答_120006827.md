# 不太会说话，怎样提高说话的技巧？
- 点赞数：11108
- 更新时间：2024年03月11日16时13分08秒

<body>
 <p>我觉得不太会说话<span><span>，</span></span>可能很多时候就是明明没话说<span><span>，</span></span>却偏要找话说<span><span>，</span></span>结果说的全是废话<span><span>，</span></span>自然不受待见<span><span>。</span></span></p>
 <p>所以<span><span>，</span></span>废话不多说<span><span>，</span></span>直接上干货<span><span>，</span></span>一共九条<span><span>，</span></span>就称为<span><span>「</span></span>独孤九言<span><span>」</span></span>吧<span><span>。</span></span><br></p>
 <p><strong>1. </strong>发言<span><span>、</span></span>讲话<span><span>、</span></span>研讨之类分享型的说话场合<span><span>，</span></span>最好抢在前面来说<span><span>，</span></span>这是因为人的注意力差不多也就能集中一两个小时的样子<span><span>，</span></span>越靠后<span><span>，</span></span>大家越没有精力去听你讲了什么<span><span>，</span></span>前面的人偶尔超时发言大家还能接受<span><span>，</span></span>但后面的人可能都没时间说什么了<span><span>，</span></span>所以只要你认真准备了<span><span>，</span></span>靠前说比靠后说要好<span><span>。</span></span></p>
 <p><strong>2.</strong> 演讲<span><span>、</span></span>面试类需要打分的说话场合<span><span>，</span></span>则恰恰相反<span><span>，</span></span>最好是在后面来说<span><span>，</span></span>一方面<span><span>，</span></span>前面的人没有参照系容易被打低分<span><span>，</span></span>人的心理也容易期待后面有更好的<span><span>，</span></span><span><span>「</span></span>我是歌手<span><span>」</span></span>这样的节目里<span><span>，</span></span>再大的牌都希望自己能在第六<span><span>、</span></span>第七位出场<span><span>，</span></span>就是这个道理<span><span>；</span></span>另一方面<span><span>，</span></span>就像前面提到的<span><span>，</span></span>人到后面注意力不会那么集中<span><span>，</span></span>这时候你的发言只要能唤起他们一点注意力<span><span>，</span></span>就很容易出彩<span><span>。</span></span></p>
 <p><strong>3. </strong> 即兴发言时<span><span>，</span></span>有两种情况最让人绝望<span><span>：</span></span>一是说不出完整的句子<span><span>，</span></span>全程结结巴巴<span><span>、</span></span>断断续续<span><span>，</span></span>不停的<span><span>「</span></span>恩<span><span>，</span></span>恩<span><span>，</span></span>恩<span><span>，</span></span>呃<span><span>，</span></span>呃<span><span>，</span></span>呃<span><span>」</span></span><span><span>，</span></span>还有一种就是离题万里<span><span>，</span></span>洋洋洒洒不知所云<span><span>。</span></span>其实这两种都是因为说话时逻辑性不够<span><span>，</span></span>说出来的话全是断章残句<span><span>，</span></span>不知道你想要表达什么意思<span><span>，</span></span>所以开口前请先用三句话的方式<span><span>，</span></span>帮自己整理和概括下思路<span><span>：</span></span><strong>我要说什么<span><span>（</span></span>发现了什么问题<span><span>）</span></span></strong><span><span>，</span></span><strong>为什么这么说<span><span>（</span></span>什么原因导致问题<span><span>）</span></span></strong><span><span>，</span></span><strong>说这些是为了做什么<span><span>（</span></span>有什么好的建议<span><span>）</span></span></strong><span><span>。</span></span></p>
 <p><strong>4. </strong> 要想提高说话的逻辑性<span><span>，</span></span>就要在平时说话时<span><span>，</span></span>多关注下自己常用的连词<span><span>。</span></span>别只会不停地<span><span>「</span></span>然后……然后……然后……<span><span>」</span></span><span><span>，</span></span><span><span>「</span></span>其实……其实……其实……<span><span>」</span></span><span><span>，</span></span><span><span>「</span></span>但是……但是……但是……<span><span>」</span></span><span><span>，</span></span><span><span>「</span></span>而且……而且……而且……<span><span>」</span></span><span><span>。</span></span>学会在合适的地方用合适的连词<span><span>，</span></span>比如表并列的时候用<span><span>「</span></span>同时<span><span>」</span></span><span><span>「</span></span>况且<span><span>」</span></span><span><span>「</span></span>何况<span><span>」</span></span><span><span>，</span></span>表承接的时候用<span><span>「</span></span>于是<span><span>」</span></span><span><span>「</span></span>说到<span><span>」</span></span><span><span>「</span></span>此外<span><span>」</span></span><span><span>，</span></span>表转折的时候用<span><span>「</span></span>然而<span><span>」</span></span><span><span>「</span></span>偏偏<span><span>」</span></span><span><span>「</span></span>不过<span><span>」</span></span><span><span>，</span></span>表因果的时候用<span><span>「</span></span>原来<span><span>」</span></span><span><span>「</span></span>由此<span><span>」</span></span><span><span>「</span></span>以致<span><span>」</span></span>等等<span><span>，</span></span>多掌握几个总没错<span><span>。</span></span>重要的是<span><span>，</span></span>当你发现自己没话说或者离题万里的时候<span><span>，</span></span>用一个合适的连词往往能圆回来——我就见过很多发言人说了一长段不知所云后<span><span>，</span></span>来一句<span><span>「</span></span><strong>所以</strong><span><span>，</span></span>我才要强调巴拉巴拉<span><span>」</span></span><span><span>，</span></span>至少让人乍一听<span><span>，</span></span>好像还挺有逻辑<span><span>，</span></span>能注意到你接下来要说什么<span><span>。</span></span></p>
 <p><strong>5. </strong> 要想在人多的时候发言有<span><span>「</span></span>一鸣惊人<span><span>」</span></span>的效果<span><span>，</span></span>一定要在平常就准备一些小套路<span><span>，</span></span>熟能生巧后<span><span>，</span></span>什么样的场合都能套着就用<span><span>，</span></span>百试不爽<span><span>。</span></span>介绍几种常用的<span><span>：</span></span><span><span>「</span></span>故事导入<span><span>」</span></span><span><span>，</span></span>准备几个常用小故事或者自己独特的经历<span><span>（</span></span>不超过 300 字<span><span>）</span></span><span><span>，</span></span>表达观点前先讲故事<span><span>；</span></span> <span><span>「</span></span>自我总结<span><span>」</span></span><span><span>，</span></span>开口先对自己要说的话做个简短的总结<span><span>，</span></span>比如<span><span>「</span></span>我想了想<span><span>，</span></span>只有三句话要对大家说<span><span>」</span></span><span><span>「</span></span>关于这个问题<span><span>，</span></span>我有四个层次的思路<span><span>」</span></span>等<span><span>；</span></span>准备几个逼格高一点的诗句或对联<span><span>，</span></span>在说话的时候点缀一下<span><span>。</span></span></p>
 <p><strong>6. </strong> 陌生场合<span><span>，</span></span>开口的第一句话很重要</p>
