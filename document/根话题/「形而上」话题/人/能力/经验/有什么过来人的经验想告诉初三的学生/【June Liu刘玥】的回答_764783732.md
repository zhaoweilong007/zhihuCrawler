 <p data-pid="iIZLiOdM">2.20更，现在大概还有100多天，很多同学们问我现在很差还还来得及吗?我的答案是:当然了！光想和担忧是没用的，大家谁不想去重高，可能最后只有几万人考上了，所以干才有用啊！每每我收到这样的消息，我总是为你们感到焦急，因为我知道考不上高中会怎样，在社会上总“低人一等”的状态（抱歉是事实）。实在静不下来，你不妨想想你爱的人，你爱做的事，你憧憬的高中......且不说努力了就会成功，但不努力就一定不会成功。坚持下去就可以不再纠结，彻底放过自己。</p>

 <p data-pid="KM655eT1">坐标江苏 2019届中考生 现高中在读</p>
 <p data-pid="LgAhrPmi">中考也算超常发挥了，足足比一二模高了40多分!考上了重高!然而两个月前我还是个被老师警告考不上4星高中的臭妹妹(´×ω×`)</p>
 <p data-pid="fgPx5mrv">(本文逻辑较乱......但通俗易懂。)</p>
 <p data-pid="5t1qKtNL">小二，上干货! 来喽!</p>
 <p data-pid="HSTD-rYQ"><b>语文</b>：</p>
 <ol>
  <li data-pid="O7ixpeut">趁着暑假赶紧找学长学姐借书!把书上要求的必读<b>名著</b>读完，中考名著题真考的超级细!有时间一定要多读几遍，网上找历年中考题练练。初三的时间基本上是没有时间读的，老师只能给你发无数套卷子练，中考还不一定考到</li>
  <li data-pid="6x9_NHD0"><b>练字</b>真最基础，也最重要了，别练啥网红字体!糊的很!正正规规的正楷字不挺好的嘛?记住字在你的卷面及作文中是大头，字好看的同学老师默认为是好生，简直是老师的小宝贝，小心尖!</li>
  <li data-pid="MiZ6HjZx"><b>阅读，文言文</b>一定要连续练，考试没手感就完蛋。我当时是做《黑马阅读》难度有点大了，刚上初三还是不建议了，辅导书还是自己去图书店看，适合自己的最好</li>
  <li data-pid="97FD8Fw-">初三一定要精打细磨<b>10</b>篇<b>作文</b>，不同体裁都得有，亲情、友情、托物言志、人生哲理、事物百态、自然道理......，初中大多是记叙文，大考的时候千万别搞啥创新!冒险必失败!议论文你写不来的，论据都不足咋写?（看个人水平）</li>
  <li data-pid="FT-4Qv1t">看书时<b>好词好句</b>都别愣着，拿笔记并背下来，终有一天你会用到，提高作文逼格!多做摘抄，积累素材</li>
  <li data-pid="Tobce2gK">准备一个小本子，总结<b>阅读题型</b>（记叙文，小说，散文，说明文，议论文），用处会非常大</li>
  <li data-pid="ZHNZAH2l"><b>阅读</b>也很重要!疯狂阅读，作文与考试，作文素材等。也可以研究名家作品，如林徽因，龙应台，余秋雨等。写出自己风格的作业也会让阅卷老师眼前一亮的!</li>
 </ol>
 <p data-pid="eHWE4n9k"><b>数学</b></p>
 <ol>
  <li data-pid="vVHWEtkV">书上<b>公式</b>啥的都先背好，<b>函数（一次函数，二次函数，三角函数，反比例函数.....）</b>也是一大巨头,赶快消灭它们，高中也得继续来，现在我就搞到头秃，www~</li>
  <li data-pid="o7dyl0CR">如果<b>基础</b>不好呢，就买基础。然后在搞啥压轴题。<b>自己要知道自己是什么水平~</b></li>
  <li data-pid="Pk5086Yd">数学上课听真不够，自己还得<b>实践</b>，比如啥公式自己推一推，说不定就发现新世界了</li>
  <li data-pid="gt67EDem"><b>作业力所能及的一定要写，</b>不会的可以求助老师，同学，作业帮等等，但要在弄懂的前提下在独自在写一遍，也求求你不要把这个当借口抄作业，无底线的搜题......你全对又能怎样，中考全对才是真牛逼!</li>
  <li data-pid="KWTznn_1"><b>错题集</b>一定要有!（原题＋答案＋反思＋总结）</li>
  <li data-pid="aqf3Mlmh"><b>不要盲目刷题，选择自己薄弱环节</b>。</li>
 </ol>
 <p data-pid="uSVljkzv"><b>英语</b></p>
 <ol>
  <li data-pid="hV30IIuZ">在中考<b>口语</b>考完之前都别松懈，一天来篇听力训练，单词都得会读还得不读错。实在不会可以学音标，B站和网课平台一般都会有（拜托~口语练好简直是装逼神器好嘛!）</li>
  <li data-pid="ksMsz_Pp"><b>阅读</b>也得练，增加词汇量（不知道大家理解阅读看不懂着急的心情嘛?）</li>
  <li data-pid="JX-8zi68"><b>作文</b>就背背啦，都会有一定的套路，<b>字</b>也很重要!</li>
  <li data-pid="VczAJ-4j"><b>上课一定要记笔记</b>，听与记结合才能无敌</li>
  <li data-pid="hl9xq8NM">当个勤奋的人儿，没事背背单词，短语</li>
  <li data-pid="1mi7AVy9"><b>语法</b>经常性总结，复习（不知道大家知道艾宾浩斯遗忘曲线吗?）</li>
 </ol>
 <p data-pid="plY5Kgyt"><b>物理</b></p>
 <ul>
  <li data-pid="O4JZiLOY">当时初三电学真的搞死我了呜呜呜，到初三下学期我还40多分（满分80），后来良心发现补了1v1 物理老师是个帅小伙 很喜欢老师（很单纯的那种）(≧∇≦)/ 每次上物理课都是快乐时间，中考74分!</li>
 </ul>
 <ol>
  <li data-pid="PCJeMouI"><b>公式</b>得背，计算题得用!</li>
  <li data-pid="q25W_aKr"><b>实验题</b>熟记于心，细枝末节不能忘</li>
  <li data-pid="nfCNLATe"><b>刷题</b>也是好方法，细心的小伙伴刷中考卷会发现有的题型是多次出现，是必考考点，要着重复习</li>
  <li data-pid="7VK14hJx"><b>新题型</b>别怕，都只是换了个马甲</li>
  <li data-pid="A9Jpk6-5"><b>上课认真听＋记笔记＋练习</b>，不然就会眼高手低了~</li>
  <li data-pid="CfALA46d"><b>错题集</b> 参上数学5</li>
 </ol>
 <p data-pid="HcoyjEyO"><b>化学</b></p>
 <ul>
  <li data-pid="lcZ2I06p">因为我不是很擅长，就不做过多回答了，我当时就是天天被知识点背到吐＋刷题刷到恶心。还是后悔当时上化学习题课没好好听，现在高一衔接都忘差不多了。<b>衷心告诉大家化学方程式好好背，化学课一节都别落下，多刷题见多识广!</b></li>
 </ul>
 <p data-pid="-QHiTI40"><b>政治历史</b></p>
 <ol>
  <li data-pid="g6UWofVa"><b>都有套路!</b>经常性总结，老师也应该会讲</li>
  <li data-pid="WIaJj9KA"><b>书都得啃透</b>小小的考点点可能就是你重来都不放在眼里的知识点</li>
  <li data-pid="4WpzSnkc"><b>关注时政</b>政治历史的考题都有关联，我这里2019年中考卷基本每一门都有五四运动和建国70周年相关题目......</li>
  <li data-pid="f96RmdFs"><b>理解性记忆，</b>比如记关键词</li>
 </ol>
 <p data-pid="RxGxxbQD"><b>小科</b></p>
 <p data-pid="-a46CTqe">体育，美术，音乐，信息，口语一定要满分!!!这些不能是你的累赘。</p>
 <p data-pid="cxowgIfD"><i>最后谈谈中考那几天吧，几乎考前今天心慌慌的，感觉有好多都没复习，当我走进考场才发现心静下来了，监考老师也不是很严，可能是因为我已经做好当个loser的思想准备，考试时还抖腿，考完就没敢怎么对答案，结果让我惊了（*/∇＼*）哈哈哈哈哈</i></p>
 <p data-pid="Z5vflLFn"><i>所以学弟学妹不要怕，中考是面对全市的学生，总会有垫后的人，算是过关性考试，试卷难度不会很大，我们这里是7＋2+1，也就是说只有那么一题你可能会没思路，别的都是基础题和稍难题，你完全没问题的OK?</i></p>
 <p data-pid="9nclr3Ml">语文老师曾说过：“不能踏进高考考场，无非是莫大的遗憾!”相信大家也想体会青春的激情四射，在最美的年华奋斗!同学们，你们的现在的目标仅仅只是中考，请把它K.O</p>

  <li data-pid="YWMZzMgM">少刷知乎，有些东西真的能改变你的思想，毕竟大名鼎鼎的杠呼〈手动狗头〉</li>

  <li data-pid="Lzh6w9Fe">要找能振奋自己的东西，当时把idol打印贴在桌上，也挺管用的，就怕自己看的入迷哈哈（彭于晏让我反复去世!!!）</li>

 <p data-pid="QHYPmMVi">2.5更新</p>
 <p data-pid="PycQ_a5g">很很抱歉没有回复一些宝贝的私信，简单概括我所看到私信里的一些问题。</p>
 <p data-pid="e_kx7JNJ">1.名著问题。现在看到这篇答案可能已经来不及了，那么我建议大家将精力放在刷题总结和概括中。把历年中考题翻出来刷，并且总结归纳。每章段落大意也要熟记 ，当然老师上课时的知识点也要记牢。</p>
 <p data-pid="S6jFfjmh">2.理科问题。这篇文章中我已经做了很明确的答案。如果实在急于求成，可以像语文阅读一样进行同类型题目的概括。重中之重还是要把知识点熟记于心，并且刷题。</p>
 <p data-pid="4A0v09zH">3.体育问题。有些宝贝身体有特殊情况，可以不强求。毕竟还有其他科目可以弥补。跑步可以用目标法：没跑一小段就定一个目标，比如我要跑到前面那棵树那里，等到跑到那棵树那里时再定一个目标，循序渐进，你就会发现很有趣。这是日本某一运动员的方法，大家可以百度。跳远实心球这些技巧性的运动可以请教体育老师，掌握技巧后并不难。</p>
