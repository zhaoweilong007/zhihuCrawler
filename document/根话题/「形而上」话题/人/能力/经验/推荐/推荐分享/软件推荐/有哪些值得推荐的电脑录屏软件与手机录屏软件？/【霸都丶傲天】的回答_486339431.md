# 有哪些值得推荐的电脑录屏软件与手机录屏软件？

<body>
 <p data-pid="wsdWaQ67">我看了这个提问下面的很多回答, 很多不是广告就是已经下线。</p>
 <p data-pid="nxf0vqUm">总结一句：</p>
 <ol>
  <li data-pid="9Q8Uq3Oy"><b>免费好用的录屏软件只有开源世界才有! </b></li>
  <li data-pid="JVNtTh0v"><b>任何有商业行为的软件都将收费或者降低画质录制时长强制你交费。</b></li>
 </ol>
 <p data-pid="UGRoXi4U">直接进入主题：</p>
 <p data-pid="3D7NHmSL">目前好用的免费的非常优秀的录屏软件我知道的用过的<b>只有下面3个</b>：</p>
 <ol>
  <li data-pid="meHgAxtK"><b>ScreenToGif 这个缺点是录制视频没有声音，主要是为了录制Gif</b>！它的Github开源项目地址<a href="https://link.zhihu.com/?target=https%3A//github.com/NickeManarin/ScreenToGif" class=" wrap external" target="_blank" rel="nofollow noreferrer">在这里</a>，下载地址<a href="https://link.zhihu.com/?target=https%3A//github.com/NickeManarin/ScreenToGif/releases/download/2.14.1/ScreenToGif.2.14.1.Setup.msi" class=" wrap external" target="_blank" rel="nofollow noreferrer">在这里</a>。完全免费，代码完全开放，不用担心病毒。（<a href="https://link.zhihu.com/?target=https%3A//pan.baidu.com/s/1Ej9nbL90XWbc9uW7z4pfXw" class=" wrap external" target="_blank" rel="nofollow noreferrer">百度云</a> 提取码 xgg5）</li>
  <li data-pid="TN5q956M"><b>Captura 这个录制视频有声音，主要面向电脑视频录制，摄像头录制</b>，同样代码完全开源，完全免费，不用担心病毒。Github开源项目地址<a href="https://link.zhihu.com/?target=https%3A//github.com/MathewSachin/Captura" class=" wrap external" target="_blank" rel="nofollow noreferrer">在这里</a><b>， </b><a href="https://link.zhihu.com/?target=https%3A//github.com/MathewSachin/Captura/releases/download/v8.0.0-beta3/Captura-Setup.exe" class=" wrap external" target="_blank" rel="nofollow noreferrer">EXE下载地址在这里</a> captura录制视频默认不带声音，如果需要声音，软件界面有下拉列表选项开启的。（<a href="https://link.zhihu.com/?target=https%3A//pan.baidu.com/s/1HV7zSx_eC4-PezsDzXmPww" class=" wrap external" target="_blank" rel="nofollow noreferrer">百度云</a> 提取码nhyn）</li>
  <li data-pid="Zd47ywkV"><b>obs-studio(</b>2019/01/14补充<b>)</b>, 一款非常强大的免费开源无广告国外开发的软件(录屏只是其功能之一, 并且对于某些高端玩家<b>既要录制屏幕又要录制摄像头选择这款</b>), <a href="https://link.zhihu.com/?target=https%3A//github.com/obsproject/obs-studio" class=" wrap external" target="_blank" rel="nofollow noreferrer">Github开源地址戳这里</a>. <a href="https://link.zhihu.com/?target=https%3A//obsproject.com/download" class=" wrap external" target="_blank" rel="nofollow noreferrer">官方下载地址戳这里</a>. 如果不能访问, <a href="https://link.zhihu.com/?target=https%3A//pan.baidu.com/s/1Ss9HPl4Q2oNBT9a-FQsQFQ" class=" wrap external" target="_blank" rel="nofollow noreferrer">百度云下载地址在这里</a>. 这款功能高级所以难上手建议使用前百度相关指南. 很多老铁不知道OBS录屏如何使用，我录了一个Gif图，大家参考一下吧 <a href="https://link.zhihu.com/?target=https%3A//github.com/AJLoveChina/hejie2.0/blob/master/obs-usage.gif" class=" wrap external" target="_blank" rel="nofollow noreferrer">obs录屏使用教程GIF图</a></li>
 </ol>
 <p data-pid="Ayb1wD4T">上面我推荐的三款产品可以满足99.9%的人的需求, 并且<b>全部免费无任何广告Github开源国外开发, 如果我说的不对请在评论下方狠狠的捶我 ~~</b></p>

 <p data-pid="ZofglB34">作为一个开发过录屏软件的人，我来说说录屏软件到底是怎么回事：</p>
 <p data-pid="7dtz1m8g">基本所有的录屏软件的核心步骤都是如下 ：</p>
 <p data-pid="3n80PkQS">从系统底层api或者第三方库装好的库读取视频音频流 =&gt;通过主流的FFmpeg编码压缩=&gt;完成</p>
 <p data-pid="vryOBeZ3"><a href="https://link.zhihu.com/?target=http%3A//github.com/" class=" wrap external" target="_blank" rel="nofollow noreferrer">Github.com</a>社区有大量的免费开源巨TM好用的录屏软件（我只是列了2个我用过的非常好用的），而且连相应二次开发的库都应有尽有：Java，C#，nodejs的都有，但是我不知道为什么国内的软件氛围这么差，把人家免费的东西包装一下然后收费或者加广告给国人使用，真的烦鸭鸭</p>
