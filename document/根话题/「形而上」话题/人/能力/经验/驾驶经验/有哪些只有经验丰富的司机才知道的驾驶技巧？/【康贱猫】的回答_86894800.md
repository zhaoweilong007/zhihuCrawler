# 有哪些只有经验丰富的司机才知道的驾驶技巧？
- 点赞数：13339
- 更新时间：2017年02月24日09时26分06秒
- 回答url：https://www.zhihu.com/question/40494621/answer/86894800
<body>
 <p data-pid="kgrmJdB5">由于东西太多太杂，强迫症的我又来整理了。</p>
 <ul>
  <li data-pid="wB3jDQVe"><i><b>高速行车篇</b></i><br></li>
 </ul>
 <ol>
  <li data-pid="1o2wQov7"><b><i>永远不要</i></b>跟其他车并排行驶，就算<b>实在没办法</b>必须并排时，主动退后半个车身或者前进半个车身（推荐前者），可有效避免因盲区产生的误判。而且即使对方车辆失控，也不会导致连环车祸，殃及自己。这里解释一下为什么我推荐后退半个车身。的确，后退以后可能正好落入对方盲区，但这里也分情况讨论。盲区是位于车辆左后方距离一米左右的范围，如果贴得很近，反光镜里也是看得见的。 而如果落在盲区里，你跟对方车辆也相隔了一米多的距离。这一米的距离也就足够在对方躲避障碍物或者强行变道的时候你果断踩下刹车进行避让，并且，你能更清楚的观察对方车辆的动向。总之，高速上离其他车越远越好，实在不行必须要并行的时候，尽可能把主动权握在自己手里。另外城市道路同样适用本条规则。</li>
  <li data-pid="zW3tLhG6">急刹车的时候扫一眼中央后视镜，如果后面的车贴你很近，在前方距离允许的情况下先重踩一脚刹车，提示后车，再放松刹车，为后车留出反应时间和距离，尽力避免追尾。如果后方没车，尽早打开双闪提示后方车辆。（这里说的急刹车不是一脚跺死的那种，一脚跺死的紧急情况你也没啥可做的了，听天由命吧）<br></li>
  <li data-pid="BOiTMpKp">别听导航瞎bb，自己看一眼导航的屏幕，该哪个路口拐或者哪个道口下一目了然。提前知道行车路线，提早并线，避免发生事故。另外高速或者高架上如果错过路口，<b><i>千万别刹车或猛打方向企图挽救</i></b>，你是在玩命。</li>
  <li data-pid="hDJ3bKtt">当前环境下最高车速=可视距离-反应距离-刹车距离-安全余量。换言之，就是当你可以看见前方有状况，反应过来，踩下刹车，直到停下，前方还有一段你自己定义的安全余量。有了这个公式，同样限速90的乡间道路，白天和黑夜的安全行车速度肯定就是不同的，雨天和晴天也是不同的。</li>
  <li data-pid="tSajyQyY">同理，跟车时候保持的车距=反应距离-刹车距离-安全余量。</li>
  <li data-pid="g2-_0Qiz">碰到有呆子跟你保持0距离，如果在超车道，或许对方很急，避让到行车道给他超车。如果在行车道，缓缓减速让他超过去。如果咬定你了，怎么也不肯超车，那你跟你的前车就保持2倍安全车距，帮那个呆子也留一份。你要照顾人家，毕竟，他呆啊。。<br></li>
  <li data-pid="T5ZR0PDb">如果想超速（当然我并没有推荐大家超速的意思，但万一真的很急），找一台超速的当地牌照的车，跟着他，他知道哪里有摄像头。外地牌的别跟。为了确保是老司机不是老油条，跟一段路看看路过摄像头时是不是有明显的减速（违法本来就是要承担相应后果的，既然你想超速，就不要妄想有什么0成本的方法，该被罚的还是会被罚，尽可能减少一些罢了）。</li>
  <li data-pid="4qanMZnE">跟卡车并排的时候由于中央气体流速快，压强小，你会被“吸”向卡车。超越瞬间，这个气流消失，你会有一种被弹开的感觉，如果在超车道，小心别“弹”到隔离带上</li>
  <li data-pid="UI_JlRKi">不要跟卡车并行，如果空间不够，宁愿在后面跟着，有机会的时候加速超越。总之并行时间越短越好。<br></li>
  <li data-pid="CXDs4fDh">跑长途之前检查胎压，冷却液，机油等，后备箱常备半桶机油（本人目前奥迪车主），不仅仅所谓“油”老虎的VAG，其他品牌的车最好也备一些（以前我开福特，也常备），万一机油缺了，你又找不到你以前加的那款，混着加不是什么好事情。当然如果你有钱，全换成新的我也不拦着</li>
  <li data-pid="9zaKFDUQ"><b>高速上车坏了或者出事之后，停到紧急停车带，车上所有人（注意，是所有人）赶紧跑到隔离带外，车子周围半径10米内不要呆。并在后方150米外放置三角警示牌</b></li>
  <li data-pid="xLKaQMB8">上匝道后<b>迅速加速</b>（欧洲流行地板油加速）至限速，同理，进入下匝道后再开始减速，不要阻碍主路上的其他车辆。</li>
  <li data-pid="6DBKPMDS">在上匝道后需要并线进入主路时，如果刚巧旁边有台挂车，而你前方辅路即将消失的时候，立即减速，让卡车过去，再立刻加速并进去。不要傻乎乎一路并行，然后你就死了。</li>
  <li data-pid="M1nXFJd-">不要呆在两辆卡车之间。前面的卡车一脚刹车，后面的如果没有踩，那你就没有了。<br></li>
  <li data-pid="3F7vaNVo">在超车道上打左转向灯的意思是告诉前面的车，我要超车，你让一下。这招也可以用在感觉右边的车可能会突然变道的情况下。警告他，你现在正在飞速靠近，别变道过来。（然而在中国并没有人会让你）德国高速飚过240，全程最左车道并且左转向灯，根本没有人会变到你前面，因为他们也知道你根本刹不住。大家都不想死。</li>
  <li data-pid="EuwOAtcf">下雨天在高速上行驶几公里之后，轻轻点一下刹车，把制动盘上的水刮掉。以防万一前方出现紧急状况第一脚刹车距离因为制动盘上有水而变长。（现在的新车有自动除水功能）</li>
  <li data-pid="RGOSFP7L">让速不让道，遇到紧急情况第一反应踩下刹车而不是去掰方向盘</li>
  <li data-pid="hSC8GSq1">转向灯的意思是，我想要变道，而不全是我正在变道。所以请在变道前两三秒提前打开转向灯。我看到太多人都是边掰方向边打转向灯的。</li>
  <li data-pid="Fe1sYkSS">眼光要放远，不要仅仅停留在前车。可以露出半个车头或者透过前车的车窗看前车的前车。这样你就相当于变向给自己腾出了2倍的车距。发现前车的前车刹车了，你也立马刹车。因为你知道，你的前车肯定也会刹车的。</li>
  <li data-pid="RSlus0FO"><b>此条国外道路环境适用。</b>当高速行驶在行车道上想变道去超车道时，如果此时后方有车正在超车道上，你打开转向灯，对方对你闪了一下大灯（注意不是狂闪），意思是他会减速让你完成变道，此时你就可以放心大胆地变道了。记得变道完成之后打开双闪表示谢意，闪一两下即可。同理，遇到前方有车想要变道时，你也应该这么做。</li>
 </ol>
 <br>
 <br>
 <ul>
  <li data-pid="5DQM3zKw"><i><b>城市行车篇</b></i><br></li>
 </ul>
 <ol>
  <li data-pid="Ug8_AJg-">看到有车走位飘忽，离他远点。同样如果有车刹车很猛，跟车的时候也远点。</li>
  <li data-pid="eSI6G0EQ">如果对方车拿大灯闪你一下，而此时你又啥事都没干的话，多半是前方有事故或者警察在偷偷测速之类。相同的，看到对方车道有事故的时候不妨在经过之后用大灯提示对方车辆。</li>
  <li data-pid="KMTwIagS">如果有车卡在你前面，不停踩刹车或者闪某个灯（比如雾灯），有可能是你车上的这个灯坏了或者忘了关。（夜间情况下示宽灯不亮简直要命，反正我是肯定会上去提醒的）</li>
  <li data-pid="i0hrobk4">不要轻易冒头公交车，因为可能有人从车头窜出横穿马路。可以从公交底下看有没有人脚，但我推荐还是踩一脚刹车，谨慎通过。<br></li>
  <li data-pid="1mOXb7u0">路口起步同理，不要做第一个蹿出去的。过了路口再加速也不迟</li>
  <li data-pid="Bdb-BUfB">去路不熟的地方，可以跟着当地车开，大幅度降低驾驶紧张感，避免过早疲劳。</li>
  <li data-pid="fUoH672E">虽然现在后视镜都是复合曲面镜，但还是要养成回头检查盲区的习惯，当然也不是每次都转头，只有在需要的时候。</li>
  <li data-pid="-4ZR1qRG">那么正好解释一下什么情况是需要转头的。<b>平时开车多观察中央后视镜</b>，后面有哪些车，前面有哪些车，脑子里有个印象。当你想变道的时候，发现后面左侧有台车不见了，那么请问它去哪了？你的盲区里嘛。。。所以直线行车，如果有经常看后方车辆的习惯，是可以告别左右后视镜的。但是为了确认，还是看一眼比较好。</li>
  <li data-pid="FyY0hK23">脑中有个大概印象，你的周围前后50米范围内大概都有哪些车，他们的速度大概都是多少等等。这样相当于脑中模拟了接下来5秒可能发生的情况。对变道，刹车，超车，都有帮助。</li>
  <li data-pid="mE1GYyWN">猜测路上其他所有人的意图，比如有台车逐渐向你这道并过来，多半是想变道。又比如有台车开得乎快乎慢，多半是路不熟，在找路。离远点，随时可能急刹车</li>
  <li data-pid="n_tGSh6P">如果不想被加塞，可以侧出小半个车身。比如在最右侧车道行使，就稍稍靠左一些，但记住不要压线。这样可以避免加塞者获取跟你并排行使并伺机挤入车流的位置。就算被挤了，由于右侧还有很大空间，果断避让后再次用车头卡回去即可。切记，只要你占住位置，他就永远压着线，撞了也是对方全则。</li>
  <li data-pid="GD4czNM2"><b>不要开斗气车，心态要放平。</b>他快让他去，他乱开也是他素质不好，你跟他较劲只会拉低自己的层次，完全没这个必要。但是如果严重妨碍公共交通的驾驶是需要坚决鄙视的（只是鄙视，别真的上去直接干）。要相信，城市开快车是建立在危险的基础上，是极不负责任的行为，心中默念要飙车请上赛道。</li>
  <li data-pid="blrJLuST">能开快就尽量开快，120的高速上开80简直就是浪费道路资源。争取做到限速多少开多少，大家一起努力就能大幅提升道路运力。另外，高速最左车道为超车道，不超车的时候别一直占着，哪怕你开120。（然而现在太多80开在超车道上的，还怎么都不肯让，只能从右侧超）</li>
 </ol>
 <br>
 <br>
 <ul>
  <li data-pid="TDaxCSE1"><i><b>山路行车篇</b></i><br></li>
 </ul>
 <ol>
  <li data-pid="_KfINV6l">老生常谈的话题，山路下坡尽量别用刹车，发动机制动拯救你的人生。老司机从山顶到山底刹车灯都是不带亮一下的，而车速却错落有致，该快的地方照样快，该慢的地方依旧慢。也别问我自动档该怎么办了，有手动模式的用手动模式，没有的就用限制档。具体怎么用请自行翻阅使用说明书谢谢。</li>
  <li data-pid="jHihzvmx">大下坡或者高速上，看到后视镜里面大货车狂闪灯，多半是刹车失灵，逃命吧！！！</li>
  <li data-pid="39-7b8Np">山路盲弯，入弯前按喇叭</li>
 </ol>
 <br>
 <br>
 <ul>
  <li data-pid="59EDSEvA"><i><b>突发状况篇</b></i><br></li>
 </ul>
 <ol>
  <li data-pid="ezR_5B8E">下雪天或者路面结冰时，在<b>安全的</b>直道上猛踩一脚刹车，直到abs启动。感受当前加速度，可以估算当前路面的极限抓地力。通过这个感受调整车速和转向角度，从此不再打滑。</li>
  <li data-pid="7P46m1md">视线不好的时候（黄昏，清晨，夜间，雾天等等），尽早打开大灯（不是远光）。<b>具体衡量标准可以是你看不清对方没开灯的车。为的不是照明，而是提示别的车你在那里。</b></li>
  <li data-pid="8w7_ntgF">没事别开双闪，特别是什么大雨大雾的。一来双闪的穿透效果不及雾灯，二来开了双闪会和转向灯冲突，变道危险性上升。觉得看不清前车示宽灯说明你后面的车也看不清你，果断打开雾灯即可。</li>
  <li data-pid="OZc03afg">出现甩尾（没有esp）的时候不要慌，反打方向一脚油就能救回来。如果不能克服一脚油的恐惧，不给也没事，但别乱踩刹车，只会加剧甩尾（此处针对前驱车型）</li>
  <li data-pid="i-lONRxC">雾天不要开远光，晃瞎的只有你自己。另外能见度极低的情况下沿着路上的线开。对面开远光的时候也可以用这招</li>
  <li data-pid="d7XXFLM2">上坡或者大负荷运行之后，冷却液温度可能升高，此时如果冷却效果不够，可以考虑打开外循环，风量调最大，温度调最高，关闭压缩机，猛吹。如果太热可以打开车窗散热。这样可以有效抑制发动机过热。当然，如果这样做还拯救不了，多半是节温器或者水泵坏了，赶紧停车吧。</li>
  <li data-pid="fRqMhra8">感觉刹车踏板行程变短，检查一下手刹是不是还有一格没松完。拉得越多，行程越短（不过脚感的差别或许只有老司机能体会出来）。</li>
 </ol>
 <br>
 <br>
 <ul>
  <li data-pid="_P0m50hS"><b><i>养车护车篇</i></b><br></li>
 </ul>
 <ol>
  <li data-pid="Z-4-Z3k-">当转向灯闪烁频率很快的时候，多半是其中一个转向灯的灯泡坏了，赶紧去检查吧~</li>
  <li data-pid="xBkpIfG7">机械液压助力车型方向盘不要打到底，留5度左右可以保护转向机</li>
  <li data-pid="sChQoJPr">坡道停车的时候，把前轮打向马路牙子并卡住，可以防止手刹失灵的时候车子溜走。</li>
  <li data-pid="hfnDhnS0">小雨就别整天开着雨刮器，不是有间歇档么，赶紧用起来。干刮的话对雨刮条不好，而且影响视线。</li>
  <li data-pid="1C-BUh-L">发现冷却液温度维持在较高温度，但又不至于开锅，去检查一下冷却液，可能过多或者过少。</li>
  <li data-pid="vuqrKMI6">每2000公里把机油尺拔出来看一眼。不要每次只会在小保养的时候让4s店给你看。一万公里正常机油消耗就能到1升，很多时候都是缺了却自己不知道。</li>
  <li data-pid="QZ62p8PB">学会换备胎，刹车灯，大灯等常见损耗件，并在后备箱里常备一些常用备件和工具，关键时刻拯救人生。</li>
  <li data-pid="o4P1YMEo">手动车型点火时踩离合，点火时踩离合，点火时踩离合。重要的事说三遍。除非你能做到每次上车就检查档位是否为空档，不然就乖乖踩离合吧，抬之前记得摸一下档杆就成。还能减轻启动机负担。</li>
 </ol>
