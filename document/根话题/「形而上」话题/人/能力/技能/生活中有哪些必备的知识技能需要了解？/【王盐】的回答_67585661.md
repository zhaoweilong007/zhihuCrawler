# 生活中有哪些必备的知识技能需要了解？
- 点赞数：6268
- 更新时间：2020年08月18日10时41分01秒
- 回答url：https://www.zhihu.com/question/35112627/answer/67585661
<body>
 <p data-pid="c2j5KbtO"><b>沟通。</b></p>
 <p data-pid="XaV7A3RG">沟通这个技能的学习从来没有出现在我们的教育中，进入工作阶段后，这方面的技巧和经验也是全靠个人自行体悟，但这个技能绝对是从生到死工作生活泡妞方方面面全都能用得到的技能，非常有必要专门的学习。</p>
 <p data-pid="nbSvN7EW">大家每天都在和人打交道，沟通是我们每天都会做的事情，但我们做的足够好吗？我们对沟通本身，又了解多少呢？</p>
 <p data-pid="lgew4F8h">坐下来，好好的思考一下，你是否曾遇到过关于沟通的问题：<br><br></p>
 <ol>
  <li data-pid="UAe9-RtC">和女朋友吵架，明明心里想和好如初，却又什么都说不出来。</li>
  <li data-pid="wK9shren">上司批评你某件事做的不够周全，你很委屈，之前明明把风险和顾虑都告诉了上司，他不记得了，现在反而来批评你。</li>
  <li data-pid="Y-ZrHuEQ">你和朋友自驾游之前发信息提醒他带帐篷，结果他以为你带了帐篷，自己就没带。</li>
  <li data-pid="WNPl-n3g">女朋友带你去见父母，你本想多说些好话博伯父伯母一笑，结果弄巧成拙搞得场面十分尴尬。</li>
  <li data-pid="OFeQ8zsU">你和朋友就一个话题争论了半天，最后发现大家说的不是一码事。</li>
  <li data-pid="xj-SFq9u">常常有人对你说：你想要表达什么呢？</li>
 </ol>
 <p data-pid="9A9-ES3y"><br>
  如果你曾遇到过相似的情况，建议你读读这篇文章。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="qyq96b6e"><b>（一）经常被人遗忘的沟通关键：认知差异</b></p>
 <p data-pid="2udHueK7">沟通时常见的误区就是把注意力过度的放在自己的表达上，认为自己只要“认真的表达”，沟通就完成了，对方肯定会理解自己想要传递的信息。</p>
 <p data-pid="Ndi03lcm">但每个人的思维方式、知识体系、个人经历等诸多不同的地方，对方并不一定能准确的接受并理解你想要传递的信息。</p>
 <p data-pid="_hsqdoqn">一个沟通的完整过程是这样的：<br><br></p>
 <ol>
  <li data-pid="QWAEM58q">大脑产生想要表达的信息</li>
  <li data-pid="oHyisRN-">将脑中产生的想法转换成对方可理解的信息（如语言、动作、图形等）</li>
  <li data-pid="C7IssmlR">表达给对方</li>
  <li data-pid="qHoqylZ0">对方接受信息，并将接受到的信息进行分析理解</li>
  <li data-pid="v6j70yze">转换后的信息进入大脑，成为对方理解的最终信息</li>
 </ol>
 <figure data-size="normal">
  <img src="https://pica.zhimg.com/50/20c7bc1c59c940c1bf671494d8eb0092_720w.jpg?source=1940ef5c" data-rawwidth="1280" data-rawheight="720" data-size="normal" data-original-token="20c7bc1c59c940c1bf671494d8eb0092" class="origin_image zh-lightbox-thumb" width="1280" data-original="https://pic1.zhimg.com/20c7bc1c59c940c1bf671494d8eb0092_r.jpg?source=1940ef5c">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="XkIxvQYZ">想法——&gt;转化成信息——&gt;表述 接受——&gt;转化成信息——&gt;理解</p>
 <p data-pid="FSMEtR_c">对沟通过程认识不足，就很容忽略认知差异带来的影响。</p>
 <p data-pid="zdjOb3dh">常常是自己费了半天口舌，对方还是一头雾水。</p>
 <p data-pid="dAsJe61P">当我们表达出自己的想法后，并不确定对方是否正确的理解我们的意图。</p>
 <p data-pid="ltC0kazk">这是可以采用“反馈修正”的方法来提高信息传达的准确率。</p>
 <p data-pid="HD6KCbOj">反馈修正：<br><br></p>
 <ol>
  <li data-pid="wweNEfxZ">表达自己的建议后，请对方复述一遍</li>
  <li data-pid="0yAZa5pr">略过大家理解一致的部分</li>
  <li data-pid="sjxoR16d">将对方复述过程中产生的差异部分抽取出来，再次表达</li>
  <li data-pid="PVaLme2t">请对方复述二次传递的信息</li>
  <li data-pid="yB80AiHv">如果对方理解还存在差异，换种表达方式再进行尝试</li>
  <li data-pid="uTj53oc3">重复以上步骤，直至最终消除误差</li>
 </ol>
 <p data-pid="BahL3buZ"><br>
  必须经过不断的沟通，才能尽可能的消除认知差异，所以沟通时要保持耐心。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="-bpIUJkp"><b>（二）如何更成功的表达</b></p>
 <p data-pid="tKq2hTlq">获得他人的理解是人类源于本能的渴望，但如果我们的表达能力有限，就很难让他人正确的理解我们想要表达的想法与感情。</p>
 <p data-pid="xryRvnjr">如何能更好的表达？</p>
 <p data-pid="8jzQjDY7">（1）选择精准的表达工具</p>
 <p data-pid="_VtX5ceG">常用的表达工具有口语、文字、符号、图形、肢体语言等，适当的选择表达工具会有助于提高信息传递的准确性。</p>
 <p data-pid="Y4vvwBDB">如果你想向恋人表达自己的爱意，那比起给她发一条信息，面对面的拥抱亲吻，说出爱她的情话，后者显然效果更好。</p>
 <p data-pid="_zsDbMix">当你打算汇报本年度销售业绩时，图形显然会比数字看起来更加直观有力。<br><br></p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/9c6406896d20adcabdedae835387ab54_720w.jpg?source=1940ef5c" data-rawwidth="1280" data-rawheight="720" data-size="normal" data-original-token="9c6406896d20adcabdedae835387ab54" class="origin_image zh-lightbox-thumb" width="1280" data-original="https://pic1.zhimg.com/9c6406896d20adcabdedae835387ab54_r.jpg?source=1940ef5c">
 </figure>
 <p><br><br></p>
 <ul>
  <li data-pid="7Q1EqkO4">如果想要表达的信息偏感性，尽量采取口语、肢体动作、图像等方式进行沟通。</li>
  <ul>
   <li data-pid="oiKtutIH">这些表达方式富含了丰富的人类感情，易于对方接受与理解。</li>
  </ul>
  <li data-pid="saLSq3V4">如果想表达的信息偏理性，尽量选择科学性的语言，如专业术语、有精准定义的概念词汇，数字等。</li>
  <ul>
   <li data-pid="xQNx7lIC">这类沟通方式，语义精准简练，不容易产生误差，也避免混杂无用或冗余的信息。</li>
  </ul>
 </ul>
 <p data-pid="3pnw6Qie"><br>
  （2）严谨的思考方法</p>
 <p data-pid="y_Q6rNZm">在沟通时，我们也要注意遵循一些基本的思考方法，这样才能缜密有效的传达信息。</p>
 <p data-pid="YzcAUicy">a.选择适当的切入角度</p>
 <p data-pid="HPdE2fN-">每个人对同一事件的观察角度是不同的，站在不同角度理解到信息可能天差地别，你必须选择一个便于他人理解，并符合你想传达信息的角度。</p>
 <p data-pid="7-WiELch">比如仓库起火，涉及此事库房管理人员第一反应肯定是哪里出了疏漏，商人会想财产有没有受到损失；而当地政府官员更加关心人员有没有出现伤亡；这就是不同角度带来的差异。</p>
 <p data-pid="vEzt38EB">在沟通之前，务必要考虑对方的自身情况、立场，选择一个便于他人理解，并符合你想传达信息的角度。</p>
 <p data-pid="Ba4sJ7Kr">b.安排适当的信息传递顺序，便于接受信息者理解</p>
 <p data-pid="_hM6UUdl">即使是相同的信息，采用不同的叙述顺序也会给人不同的感受。</p>
 <p data-pid="m-TWmmG-">根据不同的目的，采取不同的表达循序：<br><br></p>
 <ul>
  <li data-pid="BJS4txwR">通知</li>
  <ul>
   <li data-pid="u1PIy4sM">先阐述结果，再补充原因</li>
  </ul>
  <li data-pid="Zy9CNSPW">说服</li>
  <ul>
   <li data-pid="fXJgkItM">先从延伸话题开始，引导对方理解与赞同，取得共识后，再通过感情相似或逻辑相似的方式回归正题。</li>
  </ul>
  <li data-pid="xTUPThcE">吸引</li>
  <ul>
   <li data-pid="PCJ9FyHo">先抛出完整信息中引人注意的片段，再从细节引申到整体。</li>
  </ul>
  <li data-pid="j00nMzRO">安慰</li>
  <ul>
   <li data-pid="1qLe16bW">先复述对方的想法或观点，安抚情绪后，再用准备好的具有情绪相似性的信息转移对方注意力，最终引导对方自行表达出要传达的核心信息。</li>
  </ul>
 </ul>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/05747df21493d242613bd6c25ea173b9_720w.jpg?source=1940ef5c" data-rawwidth="1280" data-rawheight="720" data-size="normal" data-original-token="05747df21493d242613bd6c25ea173b9" class="origin_image zh-lightbox-thumb" width="1280" data-original="https://picx.zhimg.com/05747df21493d242613bd6c25ea173b9_r.jpg?source=1940ef5c">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="FjsHwu1O">c.区分客观事实与个人感受<br><br></p>
 <ul>
  <li data-pid="cQjMQgmz">保持客观、真实，避免夸大或扭曲信息</li>
  <li data-pid="R0qheN5H">剥离个人的主观情绪，只阐述事实</li>
  <li data-pid="Lh6PwJa1">避免形容词的使用</li>
  <li data-pid="1qX08wtt">表达方式尽可能的简洁、清晰、直观</li>
 </ul>
 <blockquote data-pid="DZEHSDi7">
  某地仓库着火，库管人员在回复上级询问情况的电话中说道：
  <br>
  <br>
  “1号仓库火势逐渐减小，我们搬走了附近堆放的一些体积较大的易燃物，应该没大事。”
  <br>
  <br>
  但事实上，“火势逐渐减小”是库管人员自己做出的判断，并不一定正确，“附近堆放的一些体积较大的易燃物”则模糊不清，这个“附近”是多大面积的一个区域？“一些”到底是多少数量？“体积较大”到底体积是多少？“易燃物”有很多种，具体是哪种？至于“应该没大事”也属于库管人员的个人判断，不属于真实有效信息。
 </blockquote>
 <p data-pid="Mzm5_lVZ"><br>
  d.保持对词汇的正确理解<br><br></p>
 <ul>
  <li data-pid="aLJig83w">对于无标准翻译的外来概念，去查原始语言的涵义</li>
  <li data-pid="o477Zjoj">对专用术语的概念，要认真、正确的理解，保持其精确的定义</li>
 </ul>
 <blockquote data-pid="90p-yfBf">
  小红和小明聊天时，提到了“裸妆”。小明并不清楚“裸妆”的概念，他按照字面意思进行理解，认为“裸妆”是“没化妆”的意思。
  <br>
  <br>
  于是小米在发表自己对女孩子化妆这个问题上的想法时说：“我觉得女生不用每天辛苦花那么长时间的妆容，不是重要场合的话，裸妆就很好啊。”
  <br>
  <br>
  小红表示赞同。
  <br>
  <br>
  但小明真正想要表达的信息是——他认为女孩子平时可以不化妆。
 </blockquote>
 <p data-pid="JUSZvT72"><br>
  e.真诚的态度<br><br></p>
 <ul>
  <li data-pid="cpQJPzuO">真诚是沟通的关键态度，发自认同自己所表达的想法，对方才会接纳、理解。</li>
  <li data-pid="ADfrw7vT">如果自己所想与所说并不一致，会出现不自然的肢体语言，让对方产生质疑。</li>
 </ul>
 <p data-pid="kmybRicT"><br>
  （3）商业叙述的原则</p>
 <p data-pid="zwbV1O1s">商业叙述和日常沟通存在一定差异，商业叙述强调传递信息的效率，要求信息可快速理解、便于传递、定义精确，在内容上，注重利弊分析、因果关系，表达方式一定要精简。</p>
 <p data-pid="VaL2FA69">商业叙述的大致顺序：<br><br></p>
 <ol>
  <li data-pid="EejAMnnk">核心主题</li>
  <li data-pid="23SAW_uK">内容简要</li>
  <li data-pid="j0alJuBH">利弊分析</li>
  <li data-pid="eNYM8S_K">其他补充</li>
 </ol>
 <p data-pid="vP4ekmLe"><br>
  以下是两个范例。</p>
 <p data-pid="iv6TSQbO">即时信息</p>
 <blockquote data-pid="tl7OsDSj">
  陈总，10月4日的会议后的入住酒店更改为柏悦酒店。
  <br>
  <br>
  因第二天会议地点临时更改，为避免在路程上耗费太多时间，重新为您预定了酒店。
  <br>
  <br>
  您到达酒店后，张秘书会接待并安排入住相关事宜，如有其他问题，随时与我联系。
 </blockquote>
 <p data-pid="A63QDAGx"><br>
  会议通知邮件</p>
 <blockquote data-pid="0G6LUpiv">
  正海，下午3点（10月2日）请来大会议室参加会议。
  <br>
  <br>
  会议主要讨论网站的用户注册流程优化。
  <br>
  <br>
   BOSS认为目前的用户注册流程过于繁琐，应进行简化。我已准备好初步方案，并邀请了技术一部、用户运营部门、客服部门的负责人参加。
  <br>
  <br>
  会议上需要你分享目前用户注册流程的运行逻辑，为了让其他非技术类部门的同事方便理解，请提前做好准备。
  <br>
  <br>
  PS：BOSS会来参加，记得不要迟到。
 </blockquote>
 <p data-pid="tMQqNibe"><br>
  人物、时间、地点、事件，这些信息一定要准确、直接。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="JemNDwN5"><br><b>（三）肢体语言</b></p>
 <p data-pid="dxxnsYly">之所以单独把肢体语言单独拿出来说，是因为我们在肢体语言方面的训练与经验实在是太少太差。</p>
 <p data-pid="gvZmNWn2">语言从某种意义上，是一种低效率的沟通方式，而肢体语言则包含了更丰富的信息传达。</p>
 <p data-pid="F2Dl3ntZ">这也正是我们看电影时，仅仅是角色的一个眼神，我们却能在那个瞬间感受到非常丰富的难以用语言形容的信息。</p>
 <p data-pid="rl1towp9">文字是可以伪装的，但肢体语言却很难伪装的出来。<br><br></p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/7265e185224e7355361805affefe3131_720w.jpg?source=1940ef5c" data-rawwidth="1280" data-rawheight="720" data-size="normal" data-original-token="7265e185224e7355361805affefe3131" class="origin_image zh-lightbox-thumb" width="1280" data-original="https://pic1.zhimg.com/7265e185224e7355361805affefe3131_r.jpg?source=1940ef5c">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="QCJgtogm">网上文笔激扬才华横溢让我们觉得很厉害的人，在现实中见面时却和网络上的形象差距甚远。</p>
 <p data-pid="_LYjskd4">肢体语言是即时性的，不会有过多的时间思考，能更多的反应一个人的真实状态。</p>
 <p data-pid="MbGzl60_">如果没有好的肢体语言，口中所说出的句子再优雅考究，也不会给人好的印象。</p>
 <p data-pid="UuI6lKk8">大家可以回忆一下，当你会想起某个人的时候，最先想到的是这个人的形象，还是他所说出的话语？</p>
 <p data-pid="z2nwDFRM">也正因为此，在互联网高度发展的今天，我们还是需要通过面试、会议、约会这样的方式来更深入的了解彼此。</p>
 <p data-pid="KrQdQWZq">另外，肢体语言会令人形象更加立体丰满，可以拉近彼此间的关系，无论双方发过多少短信，打过多少电话，也不如真正见面对双方关系的增进更有好处。<br><br></p>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/adaf94f629dd5aee3e769dd331e03e85_720w.jpg?source=1940ef5c" data-rawwidth="1920" data-rawheight="1080" data-size="normal" data-original-token="adaf94f629dd5aee3e769dd331e03e85" class="origin_image zh-lightbox-thumb" width="1920" data-original="https://pic1.zhimg.com/adaf94f629dd5aee3e769dd331e03e85_r.jpg?source=1940ef5c">
 </figure>
 <p data-pid="mWf6zHQk"><br><i>在著名电影《低俗小说》中有一个片段，讲述的是男女主人公去餐厅吃饭跳舞的故事，其中人物的对白单独评论，真的是非常乏味、无聊的对话。但由于演员出色的肢体语言，令观众百看不厌。</i></p>
 <p data-pid="M3KTacOr">肢体语言需要注重细节，每一个环节没有做好，即使是细微之处，也很容易被他人察觉出来。</p>
 <p data-pid="cU8q_CfC">我们就用常见的社交礼仪“握手”来做例子。</p>
 <p data-pid="QWHSzvyB">握手时需注意的要领：<br><br></p>
 <ul>
  <li data-pid="4aqtUlmV">自然的伸出手，动作缓慢从容</li>
  <li data-pid="EbeX_1bU">身体前倾，背身保持挺直状态</li>
  <li data-pid="O1XhHa5m">双眼与对方平视，面带微笑</li>
  <li data-pid="de25UwVJ">握手时力度适中，贴合对方手掌即可</li>
  <li data-pid="oSMcZtVQ">握手时间不宜太长，尤其是与异性握手</li>
  <li data-pid="MePuXf0Q">握手时要专心，集中精神，切记左顾右盼，心不在焉</li>
 </ul>
 <p data-pid="sc7obi1o"><br>
  说起来很容易，但没经过训练，大部分人做的都不到位，你可以在镜子面前，试着做出握手的动作，观察一下哪里存在不足。</p>
 <p data-pid="XVnubXIF">a.塑造一个良好的沟通形象</p>
 <p data-pid="Zx5SfJYy">社交场合的基本沟通原则<br><br></p>
 <ul>
  <li data-pid="7mlTUbRY">保持微笑、维持从容</li>
  <li data-pid="uEsLspPo">不将文化、信仰差异作为话题</li>
  <li data-pid="KHCa2_Fn">不随意探测个人隐私</li>
  <li data-pid="77qkdZpF">戒除口头禅、避免口音</li>
  <ul>
   <li data-pid="Dlzpdc9E">口头禅是没有经过思考的，习惯的反应用语</li>
   <li data-pid="Teo4BDgO">容易被人认为是在搪塞、应付，增加彼此误解的可能</li>
   <li data-pid="tImZVIAs">语言的驾驭能力也是一个人素质水准的表现</li>
  </ul>
  <li data-pid="34mjfcYZ">谨慎谈及你的消遣爱好</li>
  <ul>
   <li data-pid="c_irQ8ug">消遣爱好会反映一个人的知识水平与品味</li>
  </ul>
 </ul>
 <p data-pid="iGX1D9at"><br>
  需要注意避免以下行为<br><br></p>
 <ul>
  <li data-pid="nzW5P-0u">音量过大，或掌控不好</li>
  <li data-pid="OqMSUvxh">句尾语调上扬或过度低沉</li>
  <li data-pid="CGoRb_Jl">声音太尖锐或阴阳怪气</li>
  <li data-pid="iGuXElLb">过多口头禅或语助词</li>
  <li data-pid="wDijB3XR">中气不足或语调欠变化</li>
 </ul>
 <p data-pid="tLpsON0j"><br>
  b.肢体语言的训练</p>
 <p data-pid="ClZTOvmL">1）对着镜子练习<br><br></p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/72d3180c49de0536af521da9d8c41c26_720w.jpg?source=1940ef5c" data-rawwidth="900" data-rawheight="643" data-size="normal" data-original-token="72d3180c49de0536af521da9d8c41c26" class="origin_image zh-lightbox-thumb" width="900" data-original="https://pic1.zhimg.com/72d3180c49de0536af521da9d8c41c26_r.jpg?source=1940ef5c">
 </figure>
 <p><br><br></p>
 <ul>
  <li data-pid="FtojBf1z">对着镜子说话，观察自己的面部表情，并加以调整</li>
  <li data-pid="aF-z9REj">尝试在镜子前，表现你在不同情绪不同场景下会做出的言行</li>
  <ul>
   <li data-pid="g8U23Oly">非常生气的指责</li>
   <li data-pid="hz6zn6m8">委屈的解释</li>
   <li data-pid="qyCCgU9m">不耐烦的抗拒</li>
   <li data-pid="rrDsvVUQ">在高层会议上向老板汇报工作</li>
   <li data-pid="4f21zJGs">在讲台上授课</li>
   <li data-pid="EswMcc-z">试图哄正在生气的女朋友</li>
   <li data-pid="DPDtVN38">向餐厅经理投诉把汤洒在你衣服上的服务员</li>
  </ul>
  <li data-pid="8qu65yVS">在练习过程中，不要试图隐藏自己最真实的反应。</li>
 </ul>
 <p data-pid="ZA2V8Po2"><br>
  2）学习讲故事<br><br></p>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/e60a8d89fcc8ee78ad0c3d7251e946c3_720w.jpg?source=1940ef5c" data-rawwidth="2000" data-rawheight="1274" data-size="normal" data-original-token="e60a8d89fcc8ee78ad0c3d7251e946c3" class="origin_image zh-lightbox-thumb" width="2000" data-original="https://pic1.zhimg.com/e60a8d89fcc8ee78ad0c3d7251e946c3_r.jpg?source=1940ef5c">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="m7FLIp7e"><br>
  人生就是由一个又一个的故事所组成的，我们会对精彩的故事深深着迷，这点从未改变过。</p>
 <p data-pid="Y_YvT7z3">为什么TED演讲者的分享故事时总是讲的那么引人入胜？</p>
 <p data-pid="WmeB8vFa">为什么同样的笑话，我和郭德纲讲出来，效果却相差甚远？</p>
 <p data-pid="176VoNxU">如果讲好一个故事，这就是我们要学习的目标。<br><br></p>
 <ul>
  <li data-pid="VaoPeJaE">目标</li>
  <ul>
   <li data-pid="-7-gf1J4">如何将一个故事讲的引人入胜</li>
   <li data-pid="zHkxbG9n">如何把身边的一件小事描述的生动有趣</li>
   <li data-pid="rrEq2RJb">如何描述一个场景令听众如临其境</li>
  </ul>
  <li data-pid="j0-E8Ti2">需要关注的方面</li>
  <ul>
   <li data-pid="cRudHvSP">音域、音量、语速、停顿</li>
   <ul>
    <li data-pid="RhloqY4S">音域：低沉的声音会令人信服</li>
    <li data-pid="x2Qe_lFy">音量：声音的大小变化可以提高听众的注意力</li>
    <li data-pid="QgKG7es2">语速：放慢语速，便于听众理解消化信息</li>
    <li data-pid="fLfA6FHt">停顿：适当的沉默</li>
   </ul>
   <li data-pid="9XtURqsY">适度声调、和情境的口吻</li>
   <ul>
    <li data-pid="kb1Kp4Jn">不同的音调会产生不同的含义</li>
    <li data-pid="DscGi-53">在不同的情景下，声音要传达出符合情景的状态</li>
   </ul>
   <li data-pid="73Bdc1l6">准确精炼的语言</li>
   <ul>
    <li data-pid="GRLrocsi">避免口头禅</li>
    <li data-pid="Tk5k7lm8">减少重复性词汇</li>
   </ul>
  </ul>
 </ul>
 <p data-pid="vhmWYdR9"><br>
  3）挺直身体<br><br></p>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/80a063304ee1816c9495815d1eb76857_720w.jpg?source=1940ef5c" data-rawwidth="485" data-rawheight="678" data-size="normal" data-original-token="80a063304ee1816c9495815d1eb76857" class="origin_image zh-lightbox-thumb" width="485" data-original="https://pica.zhimg.com/80a063304ee1816c9495815d1eb76857_r.jpg?source=1940ef5c">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="miR7TLpr">现在每天坐在电脑面前，很容易引起驼背，可以采用上图的方法训练一下，简单实用。</p>
 <p data-pid="Y5cFMign">4）眼神交流<br><br></p>
 <ul>
  <li data-pid="rTHBMkfq">说话时要看着对方的眼睛</li>
  <li data-pid="xpTvw6sH">对公众演讲时，注意适当的用目光扫过全场，而不是只对着一个方向</li>
  <li data-pid="4ouoXcN3">眼神柔和，保持真诚、开放的态度</li>
  <li data-pid="4tI29r6i">长时间的对视会显得不礼貌，不要一直僵硬的盯着对方</li>
  <li data-pid="RmBeUyAz">如果面临谈判场合，可以尝试将目光集中至对方的某一只眼睛上，会增强压迫感</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="4BSR3h4B"><br><b>（四）提问与倾听</b><br><br></p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/570cbfc2e84ec625fd7ac1b93034ee66_720w.jpg?source=1940ef5c" data-rawwidth="415" data-rawheight="292" data-size="normal" data-original-token="570cbfc2e84ec625fd7ac1b93034ee66" class="content_image" width="415">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="SVf1ZMkA">提问与倾听是沟通中重要的部分，它们是你与他人建立主动建立关系、交换信息的主要途径。</p>
 <p data-pid="QhrrVjV6">Ⅰ.提问</p>
 <p data-pid="XlgOxPva">（1）我们为什么要提问？<br><br></p>
 <ol>
  <li data-pid="5kWk_lhD">有助于锻炼自身的思考能力</li>
  <li data-pid="7qUFffNY">简单有效的主动沟通的方式</li>
  <li data-pid="bs1yzbfo">可以建立彼此间的关系，拉近距离</li>
  <li data-pid="jZWTpm-f">好的问题，其价值不亚于好的答案</li>
  <li data-pid="S8IMpw9O">可以提升解决问题的能力</li>
 </ol>
 <p data-pid="N9_DfzgL"><br>
  （2）提问之前要做的准备<br><br></p>
 <ol>
  <li data-pid="djCrkf82">先尝试自己自行解决问题，若确定无法依靠自己解决再去请教</li>
  <li data-pid="a3UBCGW_">针对这个问题提前做好功课，避免问一些基础、低效的问题</li>
  <li data-pid="BVWM8Pn2">确定自己想要问的问题核心是什么，避免提问时思路不清</li>
  <li data-pid="rowNECY8">避免开放性、抽象、模糊不清的问题</li>
  <li data-pid="FjZdunPG">保持谦逊、友善、真诚求知的态度，避免过于强势、或炫耀性的提问</li>
  <li data-pid="SLxZqBkN">切忌贸然打算别人正在进行的谈话</li>
 </ol>
 <p data-pid="RjT-J5iJ"><br>
  （3）提问应避免的误区<br><br></p>
 <ul>
  <li data-pid="np3nDXql">提问不等于好奇</li>
  <ul>
   <li data-pid="LK2FuQth">尽量少问以下这几类问题</li>
   <ul>
    <li data-pid="rVJ8uTey">如何看待……？</li>
    <li data-pid="QvjMI095">如何评价……？</li>
    <li data-pid="cC1HWs06">如何……的体验？</li>
    <li data-pid="2t8umuCa">你觉得……？</li>
   </ul>
   <li data-pid="RpSfnAv0">这类问题本质上属于猎奇，而非求知解惑，提问者更多在于“想知道别人的所想所见”，并非源于对问题本身的思考。</li>
  </ul>
  <li data-pid="uJQm_sdc">对提问这件事存在心理负担</li>
  <ul>
   <li data-pid="u51OrV0g">认为自己提问的同时暴露自己的无知与不足，害怕别人耻笑。</li>
   <li data-pid="01mC-tqX">觉得提问是不尊重别人的表现</li>
   <li data-pid="sXZDZLzy">害怕别人认为自己是在抢风头</li>
   <li data-pid="ndOlTVMN">自认为“想法不成熟”没有信心提出问题</li>
  </ul>
  <li data-pid="pPAMLUod">经常提出不加思考的问题</li>
 </ul>
 <p data-pid="E0eBYNr7"><br>
  Ⅱ.倾听<br><br></p>
 <figure data-size="normal">
  <img src="https://pica.zhimg.com/50/567fb5d14340789fdc6bd12b078e019f_720w.jpg?source=1940ef5c" data-rawwidth="1280" data-rawheight="720" data-size="normal" data-original-token="567fb5d14340789fdc6bd12b078e019f" class="origin_image zh-lightbox-thumb" width="1280" data-original="https://pica.zhimg.com/567fb5d14340789fdc6bd12b078e019f_r.jpg?source=1940ef5c">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="ogIDF7MP">每个人都有表达欲，但很少有人懂得倾听。</p>
 <p data-pid="KtNdPWri"><br>
  a.一个好的倾听者需具备的素养<br><br></p>
 <ul>
  <li data-pid="1Lf7ax8W">换位思考</li>
  <ul>
   <li data-pid="QdqnVibW">自我意识是阻碍倾听的一大因素，当你总是站在以自我为中心的角度上，很难理解对方想表达的信息。</li>
  </ul>
  <li data-pid="W6FzuqzT">共情能力</li>
  <ul>
   <li data-pid="6vIz-nUz">用心去感受对方的情绪、思想，去尝试理解、包容、接纳</li>
  </ul>
  <li data-pid="R1D5WqTD">专注</li>
  <ul>
   <li data-pid="m0Z4g6nr">专注于交谈的对象及其所表达的内容</li>
  </ul>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="Y67AUNDj">b.倾听者需避免的误区<br><br></p>
 <ul>
  <li data-pid="YUGH-Nn2">自以为是，过早对沟通目标做出判断。</li>
  <li data-pid="BhDyvpY6">偏见，因自身主观因素，对沟通目标抱有先入为主的成见</li>
  <li data-pid="RUoAgxEG">频繁打断对方，不予以适当的尊重</li>
  <li data-pid="Kv8neEaz">试图主导谈话的节奏</li>
  <li data-pid="EmC0b3-n">总忍不住想要发表自己的看法</li>
  <li data-pid="f_gaha2E">情绪过于激动</li>
 </ul>
 <p data-pid="wsAbpUXw"><br>
  c.倾听技巧<br><br></p>
 <ul>
  <li data-pid="9blz_vdI">在开始沟通前，令交谈的对象放松下来</li>
  <ul>
   <li data-pid="z-g03e04">不要急切生硬的进入正题，可以先从聊家常或令人心情愉快的话题</li>
  </ul>
  <li data-pid="WZE-T1O_">在交谈中，看着对方的眼睛</li>
  <li data-pid="-QvkZ9X9">配合对方的节奏，并给出回应</li>
  <ul>
   <li data-pid="D_R6t7W5">当对方讲到一个段落时，可以询问“然后怎么样了？”</li>
   <li data-pid="xj_PyRe8">对一些内容给出相应的反应，“这样你也真的是很辛苦”，“当时一定很开心吧？”</li>
  </ul>
  <li data-pid="DRHPUTUP">注意体会对方的细微的情绪变化，并做出相应的关心</li>
  <li data-pid="HsDdq0mL">表达自己的想法时，使用疑问句</li>
  <li data-pid="ifK_rFFK">如果对方情绪激动，暂时中断交谈，以安慰对方为优先</li>
  <li data-pid="CANuRjfz">如需进行长时间的交谈，可以提前准备一些饮料和水</li>
  <li data-pid="WAyspOfl">在倾听时，不要反驳对方，不做价值判断</li>
  <li data-pid="jJhKQyuD">保持耐心、善意</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="giR7F38X">d.训练方法<br><br></p>
 <ol>
  <li data-pid="wqyGUko8">在公众场合，留意听身边人的谈话，将接收到的信息，整理成一个完整的故事。</li>
  <li data-pid="FWbq4oti">尝试和陌生人聊天，引导对方分享自己的故事。</li>
 </ol>
 <p class="ztext-empty-paragraph"><br></p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="osyXruM1"><b>（五）信任</b></p>
 <p data-pid="JuSAo4xl">沟通的基础，只有信任。</p>
 <p data-pid="0sEdbeN4">如何获取他人的信任？<br><br></p>
 <ol>
  <li data-pid="Vo9HD7ke">一诺千金，言出必行</li>
  <li data-pid="kZOIHJ21">对自己的原则保持坚定态度</li>
  <li data-pid="e8bADXnA">坦诚、真挚，发自内心的去沟通</li>
  <li data-pid="mx4_6AZU">陪伴</li>
 </ol>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="Mnh8ysXr">沟通是一项对人生意义重大的技能，大家一定要有意识的去学习与训练。</p>
