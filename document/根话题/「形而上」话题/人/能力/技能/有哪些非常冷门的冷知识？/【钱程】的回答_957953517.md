# 有哪些非常冷门的冷知识？
- 点赞数：6404
- 更新时间：2020年01月02日23时13分07秒
- 回答url：https://www.zhihu.com/question/286733591/answer/957953517
<body>
 <p data-pid="jVVgW57I">我来说说关于坚果的冷知识吧。</p>
 <h2><b> 花生</b></h2>
 <ul>
  <li data-pid="l5XIfj_D">花生不是中国本土植物，它和番茄、马铃薯、辣椒一样，源自<b>南美阿根廷和玻利维亚</b>，后来在秘鲁、巴西一带大量种植，之后由哥伦布带入西班牙，然后传遍世界。</li>
 </ul>
 <p data-pid="vAkDtcZ9">（关于这一点，有很多读者提到了汉阳陵出土了炭化的疑似花生的植物标本。但这似乎没有得到学术界的公认。大部分文章还是认同南美起源，比如这篇2016年的：<a href="https://link.zhihu.com/?target=https%3A//www.sciencedirect.com/science/article/pii/B9781630670382000010" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">https://www.</span><span class="visible">sciencedirect.com/scien</span><span class="invisible">ce/article/pii/B9781630670382000010</span><span class="ellipsis"></span></a>关于这方面的详细信息也希望有植物学或者考古学大佬帮忙解答一下。）</p>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="KG9wb31H">由于花生在国内太受欢迎，导致很多人以为花生是本土原产的，于是上至三国时期，下至明朝早期，放浪不羁的电视剧主角们都在吃花生米，这其实是严重不符合史实的。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-7267076824fd5780d014b80d4d9f643d_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="1024" data-rawheight="653" data-original-token="v2-7267076824fd5780d014b80d4d9f643d" data-default-watermark-src="https://pic1.zhimg.com/50/v2-69c0a687b462560db3093cc0942b517a_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1024" data-original="https://picx.zhimg.com/v2-7267076824fd5780d014b80d4d9f643d_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="7l10aTpH">闽南语的某些片区把花生叫做「土豆」 （土里的豆，嗯，没毛病）；而欧洲的一些国家把花生叫做「中国豆」 （毕竟中国大爷天天都在吃它）</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="h1mZw2qw">花生是最常见的食物致死因素之一——当然，是因为过敏。<b>据统计，美国有千分之六的人对花生严重过敏。</b>每年世界上大约有150-200人因为花生过敏而死亡。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="Slizd0Mm">花生油中可以提炼出甘油，而甘油是制作硝酸甘油炸药的重要原料（所以花生油厂可能算是军工企业——认真脸）</li>
 </ul>
 <h2><b>腰果</b></h2>
 <ul>
  <li data-pid="4JZlAG_y">跟花生一样，腰果也来自美洲，<b>腰果原产于巴西</b>，16世纪的时候被欧洲殖民者发现并带回到了欧洲。目前腰果产量最高的国家是越南，印度紧随其后。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="01SIagom">我们平常吃的腰果来源于腰果的最末端的<b>真果</b>部分，在真果上端有<b>花托膨大发育成的假果</b>（腰果苹果 cashew apple）。腰果苹果也可以吃，还可以酿酒，但极端易腐烂。所以很少在市场见到。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-76d5bb3f21b529ea6fa6a16789c1b8a0_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="1080" data-rawheight="845" data-original-token="v2-76d5bb3f21b529ea6fa6a16789c1b8a0" data-default-watermark-src="https://pica.zhimg.com/50/v2-3f13232bee3d23b5043b80626cca65a9_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1080" data-original="https://picx.zhimg.com/v2-76d5bb3f21b529ea6fa6a16789c1b8a0_r.jpg?source=1940ef5c">
  <figcaption>
   上端是腰果苹果（假果部分），下端是真·腰果
  </figcaption>
 </figure>
 <ul>
  <li data-pid="b7a4bfFD"><b>腰果的外皮有皮肤刺激性</b>，在采收、处理时，工作人员必须很小心。否则会把手弄伤。这也是为什么我们买到的所有腰果都是去皮的原因。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <h2><b>核桃</b></h2>
 <ul>
  <li data-pid="LCpy-eh6">目前市场上主要的核桃品种有两种：<b>英国核桃</b>（其实是伊朗的）和<b>黑色核桃</b>（这是美国的）。不过目前核桃年产量最高的国家是中国，毕竟中国发明了「6」这个核桃食用新单位。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="RwUsBZrH">核桃其实是核桃的种子。绿油油的果实熟透后会从顶部裂开，里面的核桃显露出来。</li>
 </ul>

 <ul>
  <li data-pid="nvslzybl">核桃剥开后，里面还有一层内种皮，<b>内种皮富含天然抗氧化物质，可以保护里面的核桃肉不被氧化。</b>（核桃肉含有65%的脂肪，在空气中特别容易被氧化）在制作核桃奶时，要把内种皮尽量去掉，否则核桃奶颜色会变深。</li>
 </ul>
 <figure data-size="normal">

 </figure>
 <ul>
  <li data-pid="jyorOnIe">「大爷的最爱」之文玩核桃其实用的是<b>麻核桃（J. hopeiensis）</b>，跟吃的核桃不是同一个种。最贵的文玩核桃价值可以在20万以上。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-b0f73ce663c622c5b82a4b49a87c3c2a_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="800" data-rawheight="533" data-original-token="v2-b0f73ce663c622c5b82a4b49a87c3c2a" data-default-watermark-src="https://picx.zhimg.com/50/v2-1c5b68d3ad0448ad96aa4032a800425a_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="800" data-original="https://pic1.zhimg.com/v2-b0f73ce663c622c5b82a4b49a87c3c2a_r.jpg?source=1940ef5c">
  <figcaption>
   文玩核桃
  </figcaption>
 </figure>
 <p data-pid="GYSyd3Y1">所以快快对你家的纸皮核桃收回“一夜暴富”的眼神——<b>不是所有核桃都能盘得出来的！</b></p>
 <p class="ztext-empty-paragraph"><br></p>
 <h2><b>山核桃 &amp; 碧根果 </b></h2>
 <ul>
  <li data-pid="XdMEKQlF"><b>山核桃</b>（小核桃）和<b>碧根果</b>同属山核桃属（<i>Carya</i>），它们俩亲缘关系很近；而核桃属于胡桃属（<i>Juglans</i>），与山核桃、碧根果亲缘关系远很多。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-dab76b78cf6d2b3990a93df7f5546867_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="1080" data-rawheight="720" data-original-token="v2-dab76b78cf6d2b3990a93df7f5546867" data-default-watermark-src="https://pic1.zhimg.com/50/v2-9f588ef4dc600389336033387ddcadb5_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1080" data-original="https://pica.zhimg.com/v2-dab76b78cf6d2b3990a93df7f5546867_r.jpg?source=1940ef5c">
  <figcaption>
   碧根果
  </figcaption>
 </figure>
 <ul>
  <li data-pid="HGn0wKZS">山核桃是我国的原生种，而碧根果来源于北美洲。山核桃种植的历史不长，从明朝后期才开始种植，成为中国本土的坚果之一。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="hE9hI3ew">山核桃成熟之前，采收人员会爬上很高的山核桃树，用一根长竿子（4米左右）把山核桃打下来。<b>这个工作很危险，每年都会有人不慎掉下来摔死。</b>目前政府正在推广安全带以防止这种情况的发生。</li>
 </ul>

 <h2><b>夏威夷果 </b></h2>
 <ul>
  <li data-pid="pFlVzNjW">澳大利亚坚果原产自澳大利亚，是为数不多的不是产自美洲的常见坚果。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="B64tuFwh"><b>但大家还是叫它夏威夷果。</b></li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="GrNjmvUz">19世纪中期被欧洲人发现，但当时大部分人都认为这种坚果有毒，不能吃。直到有一个小男孩偶然尝试之后竟然安然无恙，从而这种坚果才正式成为人类的食物。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://pica.zhimg.com/50/v2-38c805e9a42fc1da1d73f2cd8136e605_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="1080" data-rawheight="720" data-original-token="v2-38c805e9a42fc1da1d73f2cd8136e605" data-default-watermark-src="https://picx.zhimg.com/50/v2-ca40535b51567a2bfa20296c5316c540_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1080" data-original="https://pic1.zhimg.com/v2-38c805e9a42fc1da1d73f2cd8136e605_r.jpg?source=1940ef5c">
  <figcaption>
   夏威夷果
  </figcaption>
 </figure>
 <ul>
  <li data-pid="LIhgP9pg">19世纪末，夏威夷的甘蔗园主将其引种到夏威夷，从此开始以「Hawaii nut」的名义大量种植，于是「夏威夷果」被大众熟知。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="o2JBsAtl">夏威夷果对狗有毒。<b>狗摄入后会有四肢无力，后肢瘫痪，无法站立的情况，严重会导致死亡。</b></li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <h2><b>扁桃仁&amp;杏仁</b></h2>
 <ul>
  <li data-pid="Uh_yI5nf">扁桃仁（又称巴旦木）最初被叫做「美国大杏仁」，但其实原产于伊朗。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="ORhqi4Cm">美国所产的扁桃仁初次引进中国时，采用<b>「美国大杏仁」</b>作为商品名，对消费者造成了误导。美国加州杏仁商会（Almond Board of California）于是<b>在2012年发表声明，宣布在中国市场的「美国大杏仁」更名为「巴旦木」</b>。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-49ce2ede5af161796f526bfcc967f3d0_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="1080" data-rawheight="720" data-original-token="v2-49ce2ede5af161796f526bfcc967f3d0" data-default-watermark-src="https://picx.zhimg.com/50/v2-7e8f74d628bfd6917277bfc01f2f118a_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1080" data-original="https://picx.zhimg.com/v2-49ce2ede5af161796f526bfcc967f3d0_r.jpg?source=1940ef5c">
  <figcaption>
   扁桃仁（巴旦木）
  </figcaption>
 </figure>
 <ul>
  <li data-pid="B3HWaVwm">「巴旦木」这个名字来源于扁桃仁的波斯语 Badam 音译。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="xqiryJC7">美国扁桃仁收获时通常会用特制的<b>摇树机</b>把果实摇下来。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-9cbc90b271eea7b44adc49ba60abe9ae_720w.gif?source=1940ef5c" data-size="normal" data-rawwidth="561" data-rawheight="375" data-original-token="v2-9cbc90b271eea7b44adc49ba60abe9ae" data-thumbnail="https://picx.zhimg.com/50/v2-9cbc90b271eea7b44adc49ba60abe9ae_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="561" data-original="https://picx.zhimg.com/v2-9cbc90b271eea7b44adc49ba60abe9ae_r.jpg?source=1940ef5c">
  <figcaption>
   非常壮观的摇树操作
  </figcaption>
 </figure>
 <ul>
  <li data-pid="uEmpKHyR">中国杏仁是蔷薇科杏属植物<b>杏（<i>Prunus armeniaca L.</i>）</b>或者<b>山杏（<i>Armeniaca sibirica (L.) Lam</i>）</b>的种子。而扁桃仁是蔷薇科桃属植物<b>扁桃（<i>Amygdalus communis L.</i>）</b>的种子。两者不是同一个属的物种。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="ctu90v8v">未经处理的中国杏仁可能含有苦杏仁苷，水解会释放氰化氢，过量食用会中毒。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <h2><b>开心果</b></h2>
 <ul>
  <li data-pid="H-PmCGY-">开心果起源于西亚，栽培历史3500年左右。伊朗目前是开心果最大产地，占了年开心果生产量的50%以上。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="wfDYI29E">开心果的正式名称叫做「阿月浑子」。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-7c432216f2e9ebdc06f47f7942785cf8_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="1080" data-rawheight="720" data-original-token="v2-7c432216f2e9ebdc06f47f7942785cf8" data-default-watermark-src="https://picx.zhimg.com/50/v2-6cecbc5311be035031badcf08a4293e2_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1080" data-original="https://picx.zhimg.com/v2-7c432216f2e9ebdc06f47f7942785cf8_r.jpg?source=1940ef5c">

 <ul>
  <li data-pid="Mr73w3V5">开心果脂肪含量很高，如果保存不当，甚至会发生自燃现象。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <h2><b>巴西坚果</b></h2>
 <ul>
  <li data-pid="uZy0kIJ_">巴西坚果真的起源于南美亚马逊热带雨林，巴西和玻利维亚是目前巴西坚果的主产区。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="eBpwcpIE">巴西坚果树是亚马逊雨林里最高大的树之一。<b>高达50米以上。</b></li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="H_z2s0jk">虽然收获巴西坚果不需要摇树也不需要用竹竿（这不还是因为摇不动么），但<b>一定要带上护盾！</b>因为巴西坚果的果实重 2 公斤以上，从几十米的树上掉下来，重力势能非常大。在采收时需要带护盾，不然有可能被砸伤甚至砸死。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://pica.zhimg.com/50/v2-1ad5b6a1e4c2e092e91dc0f2f938902a_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="500" data-rawheight="750" data-original-token="v2-1ad5b6a1e4c2e092e91dc0f2f938902a" data-default-watermark-src="https://picx.zhimg.com/50/v2-15b21924ba9593837534c452b1a8eb5e_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="500" data-original="https://pic1.zhimg.com/v2-1ad5b6a1e4c2e092e91dc0f2f938902a_r.jpg?source=1940ef5c">
  <figcaption>
   雨林中高耸入云的巴西坚果树
  </figcaption>
 </figure>
 <ul>
  <li data-pid="S5R6pxm4">切开1厘米厚的木质果壳后可以看到像橘子一样瓣状的巴西坚果粒。把这些坚果粒烘干后就是我们买到的巴西坚果了。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://pica.zhimg.com/50/v2-4bbc91a888778499acbc1729ace88fff_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="550" data-rawheight="432" data-original-token="v2-4bbc91a888778499acbc1729ace88fff" data-default-watermark-src="https://picx.zhimg.com/50/v2-d8a866a8d46d7287c373d8f331c1e512_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="550" data-original="https://picx.zhimg.com/v2-4bbc91a888778499acbc1729ace88fff_r.jpg?source=1940ef5c">
  <figcaption>
   巴西坚果的果壳和果仁
  </figcaption>
 </figure>
 <ul>
  <li data-pid="Gm9gfW8R">巴西坚果中硒元素的含量非常高，有一些关于硒的营养学研究会选用巴西坚果作为实验组食物来进行研究。只要每天吃超过14克巴西坚果，硒就会摄入超过最高限量。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <h2><b>榛子</b></h2>
 <ul>
  <li data-pid="_PsFroql">科学家和吃货们还没搞清楚榛子是地中海起源还是中亚起源。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="ekoZ1OUn">不过科学家搞清楚了榛子特殊的香气来自一种叫做「庚烯酮」的物质，并且<b>加热后能散发出几百倍的香味</b>。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="ftgwqpl9">榛子和巧克力是绝配。著名的榛子巧克力酱Nutella的诞生是因为当时意大利西北部的可可豆供应不足，费列罗公司决定用榛子来代替一部分可可，接着发明了Nutella的前身“榛子棒”，之后才改成榛子酱。发明这种酱的人一定是个天才。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-9871f02d4fa7b5799b5273076c8ebc97_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="1080" data-rawheight="600" data-original-token="v2-9871f02d4fa7b5799b5273076c8ebc97" data-default-watermark-src="https://pica.zhimg.com/50/v2-f238ced451569ba4b33e91999796ff04_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="1080" data-original="https://picx.zhimg.com/v2-9871f02d4fa7b5799b5273076c8ebc97_r.jpg?source=1940ef5c">
  <figcaption>
   榛子巧克力酱
  </figcaption>
 </figure>
 <ul>
  <li data-pid="a5bAHi84">费列罗巧克力内层就涂的是Nutella巧克力酱。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="due77KyR">意大利这么需要榛子，旁边的土耳其就多种种——<b>土耳其垄断了全球70%的榛子产量</b>。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <h2><b>松子</b></h2>
 <ul>
  <li data-pid="-2DOZkiQ">松子真的是松树上长出来的！东北亚（中国，朝鲜，韩国一带）的红松和西藏的白松都可以长出好吃的松子。</li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-06bcaaa0b01ac80c14262f8dac08f8be_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="600" data-rawheight="375" data-original-token="v2-06bcaaa0b01ac80c14262f8dac08f8be" data-default-watermark-src="https://pic1.zhimg.com/50/v2-cb09918e4eaefb64c3db9a21acb66099_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="600" data-original="https://pica.zhimg.com/v2-06bcaaa0b01ac80c14262f8dac08f8be_r.jpg?source=1940ef5c">
  <figcaption>
   松树和松果
  </figcaption>
 </figure>
 <ul>
  <li data-pid="MW-DAq-Z">带壳松子反而比去壳松子更容易坏。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="i9blMyfp">某些品种的生的松子可以对你的舌头施短暂的小魔法，让人一直感觉嘴里有股金属的味道，好像时时刻刻都在玩东北舔铁门的游戏。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="binVEma9">但不用怕，这种味觉紊乱不需要特殊治疗，一般几天之后症状会自行缓解。</li>
 </ul>

 <h2><b>最后再说说那些关于「坚果」本身的冷知识</b></h2>
 <ul>
  <li data-pid="9HmRhC3W">坚果的油脂含量非常高，100g 坚果平均热量能达到 600kcal。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="FTs110sn">作为对比，100g 肥牛的热量为 200kcal。</li>
 </ul>
 <h2><b>吃不完的坚果如何保存？</b></h2>
 <ul>
  <li data-pid="DTK8_GRx">坚果一般需要低温干燥（32-38℃）后才经得起长期存放。否则由于水分含量高，容易长霉。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="gHJk7qt_">实在吃不掉的话，大部分坚果都可以冷冻长期保存，所以放到冰箱就好啦！</li>
 </ul>
