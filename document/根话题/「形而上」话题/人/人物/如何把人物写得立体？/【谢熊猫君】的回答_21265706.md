# 如何把人物写得立体？
- 点赞数：15375
- 更新时间：2017年10月03日10时49分14秒
- 回答url：https://www.zhihu.com/question/22411485/answer/21265706
<body>
 <p data-pid="UOnfZTgT">我自己的写作水平很差，我只是来节选一下著名作家Chuck Palahniuk（《搏击俱乐部》作者）的建议(<a href="https://link.zhihu.com/?target=http%3A//litreactor.com/essays/chuck-palahniuk/nuts-and-bolts-%25E2%2580%259Cthought%25E2%2580%259D-verbs" class=" wrap external" target="_blank" rel="nofollow noreferrer">Nuts and Bolts: “Thought” Verbs</a>)：</p>
 <blockquote data-pid="9vWwJPOe">
  <b>从现在开始，在接下来最少半年内，你不可以使用“思想动词”。</b>
  <br>
  思想动词包括：想，知道，理解，意识到，相信，想要，记住，想象，渴望等等等等你喜欢用的动词。
  <br>
  思想动词还包括：爱和恨。
  <br>
  还有些无趣的动词，比如“是”和“有”，也要尽量避免。
  <br>
  <br>
  <br>
  <br>
  在接下来的半年内，你不可以写出这样的句子
 </blockquote>
 <ul>
  <li data-pid="KHnK7vNx">李雷想<u><b>知道</b></u>韩梅梅是否<u><b>愿意</b></u>晚上和他出去约会。<br></li>
 </ul>
 <p data-pid="VSFKntT_">你必须写这样的句子<br></p>
 <ul>
  <li data-pid="uL9bnsgL">这是一个早上，李雷错过了昨晚的最后一班列车，所以只能支付了高昂的打车钱回家。回家后他发现韩梅梅在装睡，因为韩梅梅从来不曾睡得这么安静过。以往，韩梅梅只会把自己的那杯咖啡放进微波炉里加热，这一天，两个人的咖啡都加热好了。<br></li>
 </ul>
 <p data-pid="FpF_zmjC"><b>你的角色不可以“知道”事情</b>，你必须把细节展现给读者看，让读者自己“知道”到这些事情。<br><b>你的角色不可以“想要”一件东西</b>，你必须把这件东西描述给读者听，让读者自己“想要”这件东西。<br><br><br><br>
  你不可以写<br></p>
 <ul>
  <li data-pid="93zKFL6q">李雷<u><b>知道</b></u>韩梅梅<u><b>喜欢</b></u>他。<br></li>
 </ul>
 <p data-pid="1v8MyAn0">你要这样写<br></p>
 <ul>
  <li data-pid="MqwhT3Pu">课间的时候，韩梅梅总是会紧紧地靠在李雷经常打开的储物柜上。她单脚站着，另一只脚的高跟鞋则顶在储物柜的门上，留下一个高跟鞋底的印记，也留下她的香味。这样当李雷来使用储物柜的时候，密码锁上就会有她的体温和香味。到了下一个课间的时候，韩梅梅又会靠在那里。<br></li>
 </ul>
 <p data-pid="qThweirO">也就是说，<b>你在描写人物的时候不可以走捷径，只能描写感官细节——动作、气味、味道、声音和触觉。<br></b> 通常来说，写作的人把“思想动词”用在段落开始，先用这些思想动词陈述了段落的骨架，然后再来描绘。例如：<br></p>
 <ul>
  <li data-pid="Wd3Fg-tw">凯特<u><b>知道</b></u>她这次赶不及了。车辆从远方的桥那边就开始堵塞，挡住了八九个公路出口；她的手机电池用尽了；家里的狗还没有人带出去溜，这下肯定要把家里弄得一团糟；她之前还答应了邻居帮忙给花浇水……<br></li>
 </ul>
 <p data-pid="GCuRZvHf">你看，开头那一句“知道”把后面的那么多描述都给剧透了。不要这样写，如果你真的想写“知道”，那你可以把这句话放到段落的最后面，或者干脆改写成<br></p>
 <ul>
  <li data-pid="UFxW-j1R">凯特这次肯定是赶不及了。<br></li>
 </ul>
 <p data-pid="BTw-wZvZ"><b>思考是抽象的，知道和相信是无形的。你只需要用有形的动作和细节来描述你的角色，然后让读者来“思考”和“知道”</b>，你的故事写出来就更好了。</p>
 <blockquote data-pid="U8xvSArv">
  爱与恨也是。
  <br>
  不要直接告诉读者
 </blockquote>
 <ul>
  <li data-pid="5kbgChne">露西<b><u>讨厌</u></b>吉姆。<br></li>
 </ul>
 <p data-pid="bvRTM1Yo">你应该像个法庭上的律师一样，一个细节一个细节的讲，把“讨厌”的证据一个一个列出来。</p>
 <ul>
  <li data-pid="ZY0-URwU">早上点名的时候，老师刚念完吉姆的名字，在吉姆刚要答到的时候，露西轻声的说了句‘呆逼’。<br></li>
 </ul>
 <p data-pid="WzD74KHm">刚开始写作的人常犯的一个错误就是把他们写作的人物孤立起来。作者可能在写作的时候是一个人，读者在读书的时候可能是一个人，但是<b>你笔下的人物只可以在很少的时候是一个人的，因为一个被孤立的人物会开始“思想”。</b></p>
 <ul>
  <li data-pid="8s0ptb1b">马克开始<b><u>担心</u></b>这趟出门会花太久的时间。<br></li>
 </ul>
 <p data-pid="NQTf0bhD">更生动的写法是这样的</p>
 <ul>
  <li data-pid="S5hxWBUk">公车时间表说车12点的时候回来，马克看了下表，已经11点57了。这条路一路看到头，都没有公车的影子。司机肯定是在很多站之外的地方偷懒停车睡午觉呢。司机在会周公，马克却会因此而迟到。当然这还不是最糟糕的，司机可能还喝了点小酒，最后载着马克开着开着就撞了……<br></li>
 </ul>
 <p data-pid="Lj3xkdNi">一个被孤立的人物会进入想象和回忆中，但是即使这样，你也不可以用”思想动词“。<br></p>
 <blockquote data-pid="MQ4vPjTL">
  而且，<b>你也不可以用”忘记“和”记得“。</b>你不可以写
 </blockquote>
 <ul>
  <li data-pid="LDYZ5ui9">莉莉还<b><u>记得</u></b>吉姆是怎样给她梳头的。<br></li>
 </ul>
 <p data-pid="kq4OQkSD">要写成</p>
 <ul>
  <li data-pid="yC_Ipl7j">大二那年，吉姆会用自己的手温柔的给莉莉梳理长发。<br></li>
 </ul>
 <p data-pid="ZgbsQ3RQ">不能走捷径，要写细节。当然，<b>尽量不要让人物孤立，让人物互动起来，让他们的动作和语言和展现他们的思想，你作为作者不要去干预你的人物想什么</b>。</p>
 <blockquote data-pid="49AhXYBG">
  另外，在你努力避免使用“思想动词”的时候，<b>尽量减少“是”和“有”这样单调的动词</b>。
  <br>
  不要写
 </blockquote>
 <ul>
  <li data-pid="07V0BrTv">“安的眼睛<u><b>是</b></u>蓝色的”或者“安<u><b>有</b></u>蓝色的眼睛”。<br></li>
 </ul>
 <p data-pid="bcaofmLf">要写成</p>
 <ul>
  <li data-pid="rUJ2WOqb">安轻咳了一下，用左手轻轻的拂过脸庞，把烟从她蓝色的眼睛旁边拍散，然后她微笑着说……<br></li>
 </ul>
 <p data-pid="oZZA45Mr">尽量少用“是”和“有”，试着把这些细节掩藏在人物的动作后面。这样，你就是在展现你的故事，而不是简单的说故事。<br></p>
 <blockquote data-pid="Six11gPr">
  你如果真的按我说的在写作时候给自己这些约束，你一开始会很讨厌我，但是过了半年之后，你就可以不再纠结这些约束了，到时你就习惯了这样的写作方法。
 </blockquote>
