 <ol>
  <li data-pid="HdJsTg71">从此对星座之类的巴纳姆效应嗤之以鼻，两个人八竿子打不到一块的星座，就这么在一起了。<br></li>
  <li data-pid="pBR6bzUy">谈恋爱以前觉得自己很幼稚，不会哄女孩子，但是追她的人那么多，竟然就答应了。于是有了哄妹子的自信。</li>
  <li data-pid="L-Mg1SiY">我以前只喜欢一见钟情的姑娘，她不属于这类，遇到她以后觉得一见钟情是不靠谱的。</li>
  <li data-pid="ooIpX_ov">以前喜欢不认真，真心有多难得自己清楚，后来才捡回来真心。</li>
  <li data-pid="qqxXOIfh">虽然生活基本靠嘴，但是自己是个胆怯的人，尤其不愿表达情绪和感情，从她的一句话起才开始勇敢。她让我觉得不要因为过去而丢了说爱的勇气，于是从此对她春情荡漾一发不可收拾。</li>
  <li data-pid="AQDzUC49">谈恋爱以前我有个特意功能，几乎天天都做梦，尤其多的是战争片灾难片，更厉害的是梦里还带字幕。在一起后再做梦，都不敢做的太过寥廓，生怕梦里走得远了小姑娘就不见了。我怕她怕，所以梦里也要陪她。从此多了无数的温柔梦。</li>
  <li data-pid="RrN0BPMl">哎呀谈个恋爱说话不要太文艺。我的表常年调慢一分钟，这样我的以后就可以永远亦步亦趋追随她的时间。</li>
  <li data-pid="M42Vf6tZ">在我和她在一起之前，我犯了不少错误，本来以为生活不会有 fairy tales 一样的好日子，就那么一塌糊涂一直乱下去，她帮我终结了这一切。</li>
  <li data-pid="mC-4iNNQ">我学的是Marketing，我会做很多模型分析客户群需求，但是这些都比不上谈恋爱以后，多了一个同类，我猜得到她的心思，她猜得到我的心思。</li>
  <li data-pid="TKEotJnf">我可以给她起无数的外号，茫茫多茫茫多的外号。我都不知道我这么厉害。</li>
  <li data-pid="1WaNPIxe">春，曙为最。可以约她起早，看白雾未消，泛紫细云。</li>
  <li data-pid="UuaxMFmd">夏则夜。两人相伴，有月自不待言，无月也有群萤交飞，下点小雨也是情味。</li>
  <li data-pid="m_zZrrrI">秋则黄昏。找条康庄大道，牵手前后摇摇摇摇摇摇走，身后还有夕阳照耀，风声虫鸣。</li>
  <li data-pid="cU2GSQnc">冬则晨朝。有时白雪皑皑，两个人走出的脚印都是成双成对；有时无雪亦无霜，寒气凌冽，也有一双温暖的手。</li>
  <li data-pid="e9_3KBqR">上面四句是清少纳言的冒领句，我反正是喜欢躺在她的软肚子上看书的。</li>
  <li data-pid="wQNvYTjd">可以体味心动的感觉，不要问我心动的感觉是什么样子，这个我真回答不了。我只记得是看电影，我在身边偷偷瞄她，反正就觉得好看好看好看好看。</li>
  <li data-pid="E_f141vE">可以培养恶趣味，自从谈恋爱收心，两个人都相信忠贞，但是吃醋不要太好玩。</li>
  <li data-pid="DYDiyGhj">我经常说漏嘴说错话，她记性好得不行，搞得我现在圆谎的本事厉害得飞起来。</li>
  <li data-pid="CWM8nHIu">姑娘身上有好香的味道，我反正是喜欢闻，贴着闻——就是你们想的，怎么猥琐怎么闻。</li>

  <li data-pid="o93saNW5">和吃鸭舌猪舌诸多口条比起来，还是这个姑娘的嘴最软，啃起来感觉最好。</li>

  <li data-pid="ewZZosLT">谈恋爱以后会发现自己的脸皮无比的厚，可以说肉麻的情话，可以不刮胡子去扎人，还能抱着大腿撒娇！</li>
  <li data-pid="Fe9mGZan">又多了个特异功能，哪怕她不在身边，闭上眼连她鼻毛在哪个位置都浮现在眼前！</li>
  <li data-pid="FXK-IDtE">恩。。除了对她没有抵御力之外，会发现自己对别的姑娘都是城堡，怎么调戏都不怕。</li>
  <li data-pid="n9reFgmC">再也不用去看美女图片，她就有世界上最好看的眉眼。</li>
  <li data-pid="HJLmEu26">相信我，熊抱这种词你没抱过你是不理解的。每次抱着都感觉要把她抱断了，好小只。</li>
  <li data-pid="4_Q2dnFb">谈恋爱以后重色轻友是常事，生活重心除了工作就只有她。所以现在别嘲笑别人重色轻友。</li>
  <li data-pid="rjuuYJRA">除了真理之外，大有愿为她背叛世间一切的勇气，我之所以对真理心怀前程，是因为我对她的爱也寓涵其中，天经地义。</li>
  <li data-pid="WN1dIFdw">没女朋友（男朋友），你跟谁去表白呀~这事有多好玩多惊悚你不做过不知道的呀。</li>
  <li data-pid="1lDCqf4C">烦闷的时候有安慰，孤独的时候有陪伴。</li>
  <li data-pid="6yqQzsZG">我是一个懒到从来不吃带壳食物的人，她给我剥。</li>
  <li data-pid="qyg_np5d">她不会做饭，我说我会做，虽然不好看，吃还是能吃的。她不吃芹菜，但是她说，如果是我做的，哪怕是芹菜宴她也吃。（题主你感不感动！感不感动！）</li>
  <li data-pid="lq4VMbh_">她挂电话的时候说拜拜的声音特别好听。</li>
  <li data-pid="VQ8ibBM2">她喜欢我亲她脖子和耳朵，但是一直都躲，然后扑过来亲我，但是我每次都不躲，因为我也喜欢。</li>
  <li data-pid="7qSzhVPa">我的大臂现在接近40cm ,有时候有个小胳膊小细腿捏捏也感觉蛮好的。。</li>
  <li data-pid="ZwOtcdCC">小姑娘肯定是比大糙汉子轻的，可以把她抱起来甩，甩到以为屁股都掉了。</li>
  <li data-pid="rkIe1Py7">有女朋友了，就可以试试把她哄得笑成120度，多有成就感呀。</li>
  <li data-pid="J0b9oGyG">每个月到她能吃冰激凌的时候了，都会莫名其妙的开心得不行~</li>
  <li data-pid="zVGBGo6_">她会拿嘴巴喂水给我喝，凉凉的，那个感觉特别好。</li>
  <li data-pid="oMLLEONq">我可以和她说我的任何事，哪怕扣鼻屎的姿势。那个感觉特别好。</li>
  <li data-pid="lcZjypEC">谈恋爱以后一天胖半斤，这是真的，八块腹肌都看不到了。不过她不会嫌弃我，照样捏我的小肚子。</li>
  <li data-pid="9dR2or70">她教我怎么拒绝别人，别人没教过。</li>
  <li data-pid="aU_N43jW">不觉得两个都觉得自己吵架特别厉害的人在一起从来不吵架很好玩吗~</li>
  <li data-pid="amtE5UAa">她在我面前每天都好蠢好二，也是故意满足我的小心思。傻一点没人要我反正要的哇。</li>
  <li data-pid="Zbdidepi">我妈反正是夸我谈了恋爱嘴上甜腻得。。哄妈妈都是张口就来。</li>
  <li data-pid="3ldzokBp">我长得不好看，但是她夸我特别英俊！</li>
  <li data-pid="5ASr98xE">还没在一起的时候，其实她一直损我脸大，后来偷偷告诉我，她是外貌协会，挑的人肯定不会差。</li>
  <li data-pid="UEtjVpRt">坐在一起的时候她会很自然的靠过来，彻底解决椅子没有扶手手没地放的问题。</li>
  <li data-pid="OegdCGls">她觉得每次调戏我都特别有成就感。恩。。我反正也心甘情愿。</li>
  <li data-pid="2twQd21h">谈恋爱以后才明白什么是打情骂俏，她打我的感觉也特别甜蜜。</li>
  <li data-pid="qiyCMeMN">有女朋友了，相处久了，两个人的语气会越来越像，口头禅也一样。</li>
  <li data-pid="Zk47mhC5">我想看什么她就穿什么，模拟人生不用玩了~</li>
  <li data-pid="c3rSronN">她总是笨手笨脚，然后我帮她善后，成就感爆棚。脏活累活也是甘心。</li>
  <li data-pid="lLXh5YkE">在书里剧里看到好多所谓的轰轰烈烈的爱情，但是和她在一起生活再平淡也觉得是轰轰烈烈。</li>
  <li data-pid="KkJmvWM-">经历过以前觉得异地恋只是养电子宠物，后来发现是养活的电子宠物！</li>
  <li data-pid="wJPDAhPJ">可以用情侣头像！比如我用酒桶，她用大胸索拉卡！</li>
  <li data-pid="XJc_6SW8">手牵手走路时，两个人的步伐是一致的。都不用军训了。</li>
  <li data-pid="lPMZ2ftn">我反正是觉得，爱这东西就是个倒叙体的传记诗，这是我的爱恋最炽热的年代，然后此刻相识。</li>
  <li data-pid="a36r9vrI">她喜欢把头枕在我腿上睡觉，两个人都会觉得安心。</li>
  <li data-pid="NGl1mZuC">她不喜欢化妆，我就嘲笑她是第二眼美女，但是其实我一直就觉得会和她有故事。</li>
  <li data-pid="-2ZvPxSo">和我不同，她不怎么会撒娇和哄人，但是很有爱。</li>
  <li data-pid="OSW8VLcs">于是有了一个人，她在我面前就是特别傻，小把戏都能被我轻易看穿，隐秘的心事都能明白。</li>
  <li data-pid="6XspWcmE">感觉就是凑巧啊，经常是短信刚按发送就收到她的短信。</li>
  <li data-pid="dsQ97Jo3">她拥有最无与伦比的挑烂片本事，于是后来我看过了电影院上过的几乎所有烂片。</li>
  <li data-pid="V88Cx0Hr">有一次她神秘的问我知不知道今天是什么日子，我说不知道，她说是我们在一起三个月的日子，后来我告诉他，她记错日子了。</li>
  <li data-pid="ptq3Cezk">因为故事多，所以不怎么记得日子。但是因为她，我记得所有的纪念日。</li>
  <li data-pid="FJzFS1P3">她很喜欢在我面前炫耀她的腿很长很细很漂亮，我就会炫耀我的腿有110公分。</li>
