# 一人说一个冷知识，看谁的更无聊！?
- 点赞数：5875
- 更新时间：2023年10月04日23时47分55秒
- 回答url：https://www.zhihu.com/question/619125180/answer/3232761502
<body>
 <p data-pid="wMthNjjn">多分享几个：</p>
 <ol>
  <li data-pid="TvvPBSPs">把 xlsx 文件后缀改成 zip ，打开有惊喜，其实就是sharedStrings.xml保存了数据</li>
  <li data-pid="aM3p75vb">win + v 显示粘贴板，不需要第三方软件，不过有数量限制</li>
  <li data-pid="8reRSCO7">ctrl + shift + esc 快速打开Windows10任务管理器<br></li>
  <li data-pid="Tnx1uEK2">win +. 可输入emoji 表情</li>
  <li data-pid="Bj_twa7q">Win+l 锁屏</li>
  <li data-pid="3MYk3Wnh">Win+d切回桌面</li>
  <li data-pid="WaNLYO2K">QQ就能长截图，还能ocr识别图片中的文字</li>
  <li data-pid="woZYNJgO">按住 ctrl +单击可新标签页打开网页链接，比右键打开方便</li>
  <li data-pid="w8VWWjC_">安装Windows 11 系统时使用用户名 no@thankyou.com 就能跳过联网账户，创建本地账户</li>
  <li data-pid="yP60epPm">windows任务管理器里选中某个进程，按下Ctrl键就不会跳了，或者按照内存排序。</li>
  <li data-pid="Duzhiekw">有些pdf文件加了密码限制编辑操作，从pdf的文件格式标准来看，实际上限制编辑是不可能的，需要编辑软件和pdf文件一起完成，只需要一个不支持此特性的pdf工具就能完美绕过，比如用打印保存为pdf，这个新pdf就可以编辑</li>
  <li data-pid="t8VgrZQr">在文件资源管理器输入 shell:AppsFolder 即可管理所有已安装的电脑软件</li>
  <li data-pid="BVNmIJtP">将微博图片地址中的尺寸改为oslarge即可查看无水印图片。</li>
  <li data-pid="tAI5ItaY">微博设置了半年可见，如果对方有百度百家号，可以看半年前的微博，百家号会同步。</li>
  <li data-pid="gBLtiuTk">身份证号码前6位为所在地区（市、镇、区）的行政区划代码，接着后面8位是年月日生日 ，顺序码 3位，防止同一个地方同一天出生的人重号，第17位奇数代表男偶数代表女，校验码1位（0-9和 x），用来校验身份证号码正确，通过前17位计算模11得出来的，就是0-10 这11个数字,其中10用x表示，读作shí。</li>
  <li data-pid="4ce4aAVO">微信转账测试好友是否把你删除了</li>
  <li data-pid="CJUSglq-">浏览器地址栏中输入 data:text/html, &lt;html contenteditable&gt;回车直接变编辑器</li>
  <li data-pid="nKGFqKRO">@标识符前面的字符会被浏览器忽略，所以这个地址 <a href="https://link.zhihu.com/?target=https%3A//www.baidu.com" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">https://www.</span><span class="visible">baidu.com</span><span class="invisible"></span></a>@google.com 实际是打开了谷歌，不要看域名就以为链接安全</li>
  <li data-pid="o67dmde_">浏览器控制台输入document.body.contentEditable=true直接编辑网页，都不用ps了</li>
  <li data-pid="dUPChKV8">某些网页禁止了复制，浏览器控制台输入document.body.outerHTML=document.body.outerHTML就可以解除复制限制</li>
  <li data-pid="gxCZvTq2">写个bat脚本即可多开电脑微信，比如start "" "D:\wechat\WeChat.exe"</li>
  <li data-pid="zP3L37iX">archive能查看网页快照，也许能找到删除的网页</li>
  <li data-pid="0V45b-cn">输入法通过u模式输入ushuishuishui就能打出生僻字 淼</li>
  <li data-pid="nsoEhkPh">没有搜索功能的论坛网站可以在搜索引擎用site试试</li>
  <li data-pid="ugmIuvks">以前内事不决问百度，外事不决问谷歌，现在先问chatgpt <a href="https://link.zhihu.com/?target=https%3A//mp.weixin.qq.com/s/5N7qwG-kuXlVAkCQcybzDw" class=" wrap external" target="_blank" rel="nofollow noreferrer">2023 年最简单的方式免费玩 ChatGPT</a></li>
 </ol>
 <p data-pid="lOohn9xE">看了下评论，更新下： 鼠标中键点击链接就可以，微信转账测试不用真的转账，如果删除了会有提示，Win+Ctrl+Shift+B 重启显卡驱动。</p>
