# 村民因道路不通自掏十多万修建浮桥并收过路费，被司法部门认定「寻衅滋事」而获刑，这样的判罚是否合理？

 <h2><b>没有了浮桥，去对岸种地只能绕行。</b></h2>
 <ul>
  <li data-pid="NrZSXP8w"><b>开农用车得走3个多小时，来回就得7个小时。</b></li>
 </ul>
 <blockquote data-pid="Dvc67n0k">
  <b>5年了，当地没有做出在涉事河流上建桥的规划〔有更新〕。</b>
 </blockquote>

 <ul>
  <li data-pid="0ldCOQwW"><b>主角「黄德义」，生于1967年6月，吉林省白城市洮南市（县级市）瓦房镇「振林村」人，在「洮儿河」边长大，</b></li>
 </ul>
 <blockquote data-pid="lSt-mUni">
  <b>洮[táo]儿河是「<span class="nolink">嫩江</span>」右岸的一条重要支流。</b>
 </blockquote>
 <ul>
  <li data-pid="m7cmOUwQ"><b>嫩江是「<span class="nolink">松花江</span>」的最大支流。</b></li>
 </ul>
 <blockquote data-pid="hDOVS5ir">
  <b>松花江是「黑龙江」在中国境内的最大支流。</b>
 </blockquote>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-8ec3592d401103c60c67141eb676a1e4_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="1920" data-rawheight="1440" data-original-token="v2-8ec3592d401103c60c67141eb676a1e4" class="origin_image zh-lightbox-thumb" width="1920" data-original="https://picx.zhimg.com/v2-8ec3592d401103c60c67141eb676a1e4_r.jpg?source=1940ef5c">
  <figcaption>
   洮儿河（阿尔山沿途）
  </figcaption>
 </figure>
 <ul>
  <li data-pid="p95Lp1vu"><b><span class="nolink">振林村</span>、安全村以及附近其他许多村落，都位于洮儿河两岸。</b></li>
 </ul>
 <blockquote data-pid="tof9Mnns">
  <b>去河对岸种地，以及亲戚之间的走动，仿佛「一场修行」。</b>
 </blockquote>
 <ul>
  <li data-pid="5sqkoaCj"><b>由于落差大，水流急，常有村民因过河而溺亡。</b></li>
 </ul>
 <blockquote data-pid="l1j6jh5U">
  <b>据不完全统计，六十年代末到现在至少22名村民溺亡。</b>
 </blockquote>
 <ul>
  <li data-pid="6k_V-30r"><b>黄德义自己就有两次涉水过河差点丧命的经历。</b></li>
 </ul>
 <blockquote data-pid="A3g6mVeD">
  <b>黄德义的儿时伙伴<span class="nolink">邢某权</span>的妈妈，黄德义的舅舅<span class="nolink">隋某禄</span>，黄德义的同学陈某，黄德义的学生张某，都被这条河夺去生命。</b>
 </blockquote>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-3d7867c7ab1aa7c12ad89909b438e9c4_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="1309" data-rawheight="116" data-original-token="v2-3d7867c7ab1aa7c12ad89909b438e9c4" class="origin_image zh-lightbox-thumb" width="1309" data-original="https://picx.zhimg.com/v2-3d7867c7ab1aa7c12ad89909b438e9c4_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="Iv5_lPZw"><b>黄德义兄弟5人，祖上都以摆渡为业。</b></li>
 </ul>
 <blockquote data-pid="Z-NqihFT">
  <b>上世纪70年代，黄德义的父亲曾造了一条木船，用来在洮儿河上摆渡。</b>
 </blockquote>
 <ul>
  <li data-pid="-fUfV4ZW"><b>“回想1975年，我父亲「黄镇」是小队队长，乡亲过河困难，就去洮南水利局找时任局长「<a href="https://www.zhihu.com/search?q=%E6%9D%A8%E5%AE%B6%E7%94%9F&amp;search_source=Entity&amp;hybrid_search_source=Entity&amp;hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A3104586285%7D" class="internal">杨家生</a>」。”</b></li>
 </ul>
 <blockquote data-pid="lAAnv9lM">
  <b>“杨局长给我们队拿1000元钱，让我们自己造船过河。”黄德义说。</b>
 </blockquote>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-1bdb11db1d06cd5d50901e4763ae623d_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="453" data-rawheight="603" data-original-token="v2-1bdb11db1d06cd5d50901e4763ae623d" class="origin_image zh-lightbox-thumb" width="453" data-original="https://pic1.zhimg.com/v2-1bdb11db1d06cd5d50901e4763ae623d_r.jpg?source=1940ef5c">
  <figcaption>
   黄德义在工地打工。
  </figcaption>
 </figure>
 <ul>
  <li data-pid="c-NVYlbz"><b>90年代，黄德义的三哥想继续在洮儿河上摆渡。</b></li>
 </ul>
 <blockquote data-pid="AZIJ41RL">
  <b>那时候，黄德义已经是一名乡村教师。</b>
 </blockquote>
 <ul>
  <li data-pid="OCmIVU9s"><b>黄德义出了4000元，造了3条铁皮船，连接在一起，和三哥一起摆渡。</b></li>
 </ul>
 <blockquote data-pid="vMI2rM5T">
  <b>水少河窄的时候，就直接横在河里，两边搭上木板连接河岸供人车通行，河面宽的时候就通过人力用绳子在两岸拽。</b>
 </blockquote>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-7b77ddbe9b2c44644ed8b313437b2c1b_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="600" data-rawheight="410" data-original-token="v2-7b77ddbe9b2c44644ed8b313437b2c1b" class="origin_image zh-lightbox-thumb" width="600" data-original="https://pica.zhimg.com/v2-7b77ddbe9b2c44644ed8b313437b2c1b_r.jpg?source=1940ef5c">
  <figcaption>
   移动浮桥
  </figcaption>
 </figure>
 <ul>
  <li data-pid="Tkbtbp5t"><b>彼时摆渡船收取的费用并没有标准。</b></li>
 </ul>
 <blockquote data-pid="XPi6yzBq">
  <b>“2000年之前用牲口，马用的多，如果水浅赶马车都能过去，趟地都能过去。等2000年以后马都不用了，开始用拖拉机等机动车，但水深一点就过不去了。”</b>
 </blockquote>
 <ul>
  <li data-pid="4nKcYEzF"><b>3条船组成的摆渡船，能摆渡的人和车有限。</b></li>
 </ul>
 <blockquote data-pid="jd7EUu7l">
  <b>2014年，黄德义又焊了13条铁皮船。</b>
 </blockquote>
 <ul>
  <li data-pid="BSB1whfx"><b>船宽1.5米，长7.5米，横着用木板连起来，搭成了一个固定浮桥。</b></li>
 </ul>
 <blockquote data-pid="hsxz5Cdg">
  <b>天堑变通途。</b>
 </blockquote>
 <ul>
  <li data-pid="Z5C_9jSc"><b>浮桥跟着水面宽窄调整。</b></li>
 </ul>
 <blockquote data-pid="96RlC5YX">
  <b>一旦汛期水太大，或者冬天上冻了，桥就从水面上撤下来。</b>
 </blockquote>
 <ul>
  <li data-pid="djaRa_qR"><b>黄德义称，因为修桥有很多人参与，便安排大家轮流去桥上收费，收费标准并无硬性规定。</b></li>
 </ul>
 <blockquote data-pid="jVy0kcdv">
  <b>过路的村民和路人根据车辆大小、人数多少以及个人意愿支付费用，少则一两元，最多给过10元，很多附近熟悉的村民过桥也不用给钱。</b>
 </blockquote>
 <ul>
  <li data-pid="ZytLXm_i"><b>“修桥拆桥需要人，我哥、我侄儿等都参与，我也给不了工钱，就一个月就给他一天。”</b></li>
 </ul>
 <blockquote data-pid="KRXUQxW1">
  <b>“那有水就得点，没水就拉倒。有钱没钱都过河，愿意给就给，不愿给就拉倒。”</b>
 </blockquote>
 <ul>
  <li data-pid="2uL42geO"><b>黄德义的说法，得到了附近村民的验证。</b></li>
 </ul>
 <blockquote data-pid="RUUCvhhp">
  <b>“有给3块的，有给5块的，也有2块、1块的，这没准儿。”</b>
 </blockquote>
 <ul>
  <li data-pid="Kyy2fllB"><b>2008年5月22日，黄德义的父亲离世。</b></li>
 </ul>
 <blockquote data-pid="yXclKA9I">
  <b>黄镇万万没想到，因为这桥，他的一个儿子、两个孙子被关，儿子教了35年书的工作也给丢了。</b>
 </blockquote>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-3d7867c7ab1aa7c12ad89909b438e9c4_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="1309" data-rawheight="116" data-original-token="v2-3d7867c7ab1aa7c12ad89909b438e9c4" class="origin_image zh-lightbox-thumb" width="1309" data-original="https://pic1.zhimg.com/v2-3d7867c7ab1aa7c12ad89909b438e9c4_r.jpg?source=1940ef5c">
 </figure>
 <ul>
  <li data-pid="FhUrImCp"><b>桥修好的第二年，当地水利部门就开始对桥进行罚款。</b></li>
 </ul>
 <blockquote data-pid="F5khGkHO">
  <b>“2015年交了1万块钱，说交钱就不用拆了。16年交1万，都是我哥他交的，17年我哥他们都不交了，因为这桥也不咋挣钱。”</b>
 </blockquote>
 <ul>
  <li data-pid="wIdiTrjN"><b>2018年10月，「洮南市水利局」以「非法建桥」为由处罚并强制拆除浮桥。</b></li>
 </ul>
 <blockquote data-pid="nGV4oYZr">
  <b>“18年，我问交钱还行不行了，他说交钱也不行了，让我拆。我们自己拆的，不是他们给拆的。”</b>
 </blockquote>
 <ul>
  <li data-pid="tXvuIJG5"><b>桥拆了，黄德义以为就没事了。</b></li>
 </ul>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-ecff169e7c6101de5e0bd9bfbf9545bc_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="590" data-rawheight="1280" data-original-token="v2-ecff169e7c6101de5e0bd9bfbf9545bc" class="origin_image zh-lightbox-thumb" width="590" data-original="https://picx.zhimg.com/v2-ecff169e7c6101de5e0bd9bfbf9545bc_r.jpg?source=1940ef5c">

 <ul>
  <li data-pid="TUciYqdm"><b>2019年2月1日，黄德义被「洮南市公安局」以涉嫌「寻衅滋事」罪刑事拘留。</b></li>
 </ul>
 <blockquote data-pid="561bzgPB">
  <b>黄德义的三哥以及多位家人也被采取「刑事措施」。</b>
 </blockquote>
 <ul>
  <li data-pid="oLhUbFQ4"><b>2019年7月，「洮南市检察院」以涉嫌寻衅滋事罪将黄德义等多人公诉至洮南市法院。</b></li>
 </ul>
 <blockquote data-pid="XI8ipAPj">
  <b>洮南市检察院指控，2005年至2014年，被告人黄德义伙同其他人私自建船体浮桥拦截过往车辆收取过桥费。</b>
 </blockquote>
 <ul>
  <li data-pid="ub_TpjBa"><b>洮南市检察院指控，2014年冬，黄德义又出资并组织在该处河道私建浮桥。</b></li>
 </ul>
 <blockquote data-pid="Fd3oVNfx">
  <b>至该桥被强制拆除，黄德义共收取过路费44000多元。</b>
 </blockquote>

 </blockquote>

 <blockquote data-pid="0RhV_I21">
