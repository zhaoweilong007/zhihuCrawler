# 为什么都说碎片化信息无用，我却感觉刷手机也可以学到很多东西？

<body>
 <p>你所接受的一切信息<span><span>，</span></span>构成了你的思维方式<span><span>。</span></span></p>
 <p>所以长期接受碎片信息的后果就是<span><span>：</span></span></p>
 <p><strong>让你的思维变得狭隘<span><span>，</span></span>难以进行复杂的思考</strong><span><span>。</span></span></p>
 <p>碎片信息通常具备这样的特征<span><span>：</span></span></p>
 <ul>
  <li>它们往往是一些事实的集合而非逻辑</li>
  <li>它们往往大量简化了推演过程</li>
  <li>它们往往将多路径简化为单一路径</li>
  <li>它们往往不够严谨<span><span>、</span></span>全面</li>
 </ul>
 <p>简而言之<span><span>，</span></span>碎片信息为了达到易于习得的目的<span><span>，</span></span>通常会显著降低认知成本<span><span>，</span></span>最明显的方式就是<span><span>：</span></span>将复杂的事物简单化<span><span>。</span></span></p>
 <p>它们往往只告诉你表面上的东西<span><span>，</span></span>却不会告诉你背后的原理<span><span>，</span></span>以及它与其他事物之间的联系<span><span>。</span></span></p>
 <p>我们所说的<span><span>「</span></span>知识<span><span>」</span></span><span><span>，</span></span>由两部分组成<span><span>：</span></span>一是<span><span>「</span></span>事实<span><span>」</span></span><span><span>（</span></span>或曰<span><span>「</span></span>观念<span><span>」</span></span><span><span>）</span></span><span><span>，</span></span>二是<span><span>「</span></span>联系<span><span>」</span></span><span><span>。</span></span></p>
 <p>事实就是一个个点<span><span>，</span></span>联系则是把点连接起来的线<span><span>，</span></span>它们所构成的网络<span><span>，</span></span>就是我们的知识结构<span><span>。</span></span></p>
 <p><span><span>「</span></span>事实<span><span>」</span></span>决定了你的知识广度<span><span>，</span></span><span><span>「</span></span>联系<span><span>」</span></span>决定了你的知识深度<span><span>。</span></span>如果你了解事物之间的联系<span><span>，</span></span>即使你只知道 ABC<span><span>，</span></span>你也可以根据这三者的内在逻辑<span><span>，</span></span>得出 DE<span><span>，</span></span>甚至 F<span><span>，</span></span>这个过程就叫作思考<span><span>。</span></span></p>
 <p>但如果你不了解它们的内在逻辑<span><span>，</span></span>即使你知道 ABCDE<span><span>，</span></span>你也是没办法得出 F 的——你不知道需要把它们放在一起<span><span>，</span></span>更不知道放在一起之后它们能够呈现出怎样的内在逻辑关系<span><span>。</span></span></p>
 <p>这就是碎片化信息的弊端<span><span>。</span></span>当我们接受碎片信息时<span><span>，</span></span>我们实际上是在扩充<span><span>「</span></span>事实<span><span>」</span></span><span><span>，</span></span>但并没有增加<span><span>「</span></span>联系<span><span>」</span></span><span><span>。</span></span>长此以往<span><span>，</span></span>会使我们的知识结构变成一张浮点图<span><span>：</span></span>孤零零的知识点漂浮在各个位置<span><span>，</span></span>却缺乏一个将它们有序串联起来的网络<span><span>。</span></span></p>
 <p>这样的结果是<span><span>：</span></span></p>
 <p>1. 碎片化知识通过连续的新鲜内容<span><span>，</span></span>不断刺激你的大脑<span><span>，</span></span>让你始终处于<span><span>「</span></span>啊<span><span>！</span></span>又知道了新的东西<span><span>」</span></span>的喜悦中<span><span>，</span></span>从而难以自拔<span><span>，</span></span>这也就是我们难以抑制刷微博<span><span>、</span></span>刷朋友圈的缘故<span><span>，</span></span>因为我们只需要付出很少<span><span>，</span></span>就可以沉浸在<span><span>「</span></span>获得了新东西<span><span>」</span></span>的刺激里面<span><span>。</span></span><br>
   但是<span><span>，</span></span>这些获得的信息<span><span>，</span></span>因为它们缺少跟其他信息的<span><span>「</span></span>联系<span><span>」</span></span><span><span>，</span></span>因此难以被我们<span><span>「</span></span>提取<span><span>」</span></span><span><span>，</span></span>而<span><span>「</span></span>提取<span><span>」</span></span>得越少的内容<span><span>，</span></span>会被提取得多的内容挤压在记忆的底部——因此<span><span>，</span></span>这些碎片化的信息极其容易被我们遗忘<span><span>。</span></span>你以为你得到了很多<span><span>，</span></span>但其实你什么都没有得到<span><span>。</span></span></p>
 <p>2. 我们的知识网络<span><span>，</span></span>决定了我们如何思考<span><span>。</span></span>那么<span><span>，</span></span>长期接受碎片信息<span><span>，</span></span>对你的思考能力并没有提升——你的<span><span>「</span></span>网络<span><span>」</span></span>并没有扩大<span><span>；</span></span>甚至是有害的——你已经习惯了用孤立的知识点去看待问题<span><span>，</span></span>习惯了一层<span><span>、</span></span>两层的思维<span><span>，</span></span>难以对事物进行五层<span><span>、</span></span>六层乃至更深入的剖析<span><span>。</span></span>长此以往<span><span>，</span></span>你将弱化对于复杂事物的思考能力<span><span>。</span></span></p>
 <p>这里要注意一点<span><span>：</span></span>碎片化信息<span><span>，</span></span>其实跟来源没有太大关系<span><span>。</span></span>如果你习惯了接受<span><span>「</span></span>A 是 B<span><span>」</span></span><span><span>、</span></span><span><span>「</span></span>C 是坏的<span><span>」</span></span><span><span>、</span></span><span><span>「</span></span>因为 A 所以 B<span><span>」</span></span>的简单观念<span><span>，</span></span>那么<span><span>，</span></span>无论你是在刷微博<span><span>，</span></span>还是在读书<span><span>、</span></span>看公开课<span><span>，</span></span>其实接受的都是碎片化知识<span><span>。</span></span></p>
 <p>聪明人的思考方式<span><span>，</span></span>其实很简单<span><span>，</span></span>就是三个字<span><span>：</span></span>元认知<span><span>。</span></span>同样一个问题<span><span>，</span></span>较为聪明的人<span><span>，</span></span>会去思考这个问题产生的背景<span><span>、</span></span>原因<span><span>、</span></span>合理性<span><span>、</span></span>必要性<span><span>、</span></span>可能性<span><span>，</span></span>等等<span><span>。</span></span>他们会拔高一层去看待这个问题<span><span>，</span></span>从而更容易找到一根线条<span><span>，</span></span>将它和远处的某个固有的<span><span>「</span></span>观念<span><span>」</span></span>联系在一起<span><span>，</span></span>从而拓展自己的思维网络<span><span>。</span></span></p>
 <p>所以<span><span>，</span></span>如果你有这样的意识<span><span>，</span></span>那么其实无论刷微博还是读书<span><span>，</span></span>都可以避免碎片化认知<span><span>。</span></span></p>

 <p><strong>2.</strong> <strong>找到知识网络的触点</strong><span><span>。</span></span>亦即自己感兴趣的<span><span>、</span></span>但尚未进行探索和了解的知识点<span><span>。</span></span><br>
   阅读<span><span>、</span></span>学习的时候<span><span>，</span></span>有意识地去接触这些触点的知识<span><span>，</span></span>延展自己的知识网络<span><span>。</span></span></p>
