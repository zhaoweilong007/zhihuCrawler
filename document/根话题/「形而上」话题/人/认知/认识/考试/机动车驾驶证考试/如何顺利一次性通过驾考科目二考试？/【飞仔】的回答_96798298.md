 <h2>掌握理论知识更让你知道 why ，实际操作的时候更快知道 how</h2>
 <p data-pid="bHSCUCq5">驾校教练的目的是让你以最快速度通过考试，但其往往忽略了理论知识的重要性，这会导致学员在操作的时候容错率很低，降低实际考试时的通过率。</p>
 <p data-pid="IZyJL3lZ"><b>汽车基本知识</b></p>
 <p data-pid="TB9CuPp2">汽车所有的东西，分成三个部分：速度控制、方向控制、交流辅助设备。</p>
 <ul>
  <li data-pid="Oz5-F-dc"><b>速度控制</b></li>
  <ul>
   <li data-pid="FeAT7LXJ"><b>离合器</b></li>
   <ul>
    <li data-pid="dyJMG5sp">离合器的本质是一个动力开关。完全踩下离合，动力被切断，此时车辆的动力为 0；离合完全松开，动力被完全接上，此时车辆的动力为 1。</li>
    <li data-pid="TM4KesZj">为什么教练老让你踩一半离合来控制测速呢，既然是“动力开关”，不应该是要么关闭为 0，要么开启为 1 的么？</li>
    <ul>
     <li data-pid="HygLgVqh">其实离合的设计者设计了一种叫“半联动”的状态。如果用数字来代表开关的状态，那么在你完全踩死到完全松开离合的整个过程中开关状态（动力）是这样“0-0-0-0-0.4-0.5-0.6-1-1-1-1”。就是说你送离合的刚开始一段都是0，中间（不一定是中间，这个点一般叫做“结合点”）开始后慢慢增加，过了那段区间后就直接跟松开差别不大了。我们用来控制车速的，刚好是“0.4-0.5-0.6”（数字我乱写的，大概这个意思）这段。</li>
    </ul>
    <li data-pid="-3KStnk8">为什么在1档或者1档切2档时离合松太快会熄火？</li>
    <ul>
     <li data-pid="uJAzWMFH">因为这个时候发动机没有火力全开，因为你没踩油门。突然给它太大的负载下，会导致发动机在“排气和吸气”过程中停止掉。那为什么慢松离合发动机就不会熄火呢？因为聪明的人类给离合设计了一个结构，把整个过程变得平滑了：本质上是设计一个缓冲结构，它可以让档不是从0-1，而是从0-到0.4、0.5、0.6慢慢到1的。相当于你骑车的时候，给你把坡度变小了，你骑起来就轻松点。</li>
    </ul>
    <li data-pid="bGcvy0FU">为什么新手前三次练车左脚踩离合踩得抽筋？</li>
    <ul>
     <li data-pid="aPGuny0j">姿势不对，请看图</li>
    </ul>
   </ul>
  </ul>
 </ul>

 <ul>
  <li data-pid="i8c86x10"><b>刹车</b></li>
  <ul>
   <li data-pid="SwD2bvEA"><b>刹车是右脚边的左边。所有脚踏板从左到右是：离合（自动挡没有）、刹车、油门。</b></li>
   <li data-pid="yJP8eiy-">刹车踩的越深，车辆刹车停的越快。刹车并不是只有踩到底和不踩两者状态，刹车可以踩 0.5 、0.4 等等。</li>
   <li data-pid="UoPqW6do">一般汽车的刹车性能比自行车、电动车和摩托车要好，比如在50km/h的速度下，你用力踩到底，车几乎会当场停下，放心，也不会翻车。120公里速度下，只要不乱动方向盘，一脚刹车踩到底，不存在翻车的可能。</li>
   <li data-pid="LvsbPuWJ"><b>油门</b></li>
   <ul>
    <li data-pid="29QwJ-tu"><b>油门是用来控制加速的快慢的。</b></li>
    <li data-pid="UCc7wOYu">慎重油门踩到底，在考科目二时候几乎不存在需要重踩油门的情景。换挡时要先松油门。</li>
   </ul>
  </ul>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="yQcjeiyp"><b>转向控制</b></li>
  <li data-pid="2ygMvW9v">汽车有4个轮子，但方向盘一般都只能控制2个前轮的方向，2个后轮是不能转向的。</li>
  <li data-pid="C8ad9bJO">方向盘转1度，前轮角度也转1度，转向比就是1：1，一般用在赛车上。</li>
  <li data-pid="hOxDPnKS">一般的普通汽车按照驾校的车1圈半打死（540度）来算，打死的时候车轮的转向角度在35度的样子。如图</li>
 </ul>

 <ul>
  <li data-pid="M4oTs3er"><b>辅助设备</b></li>
  <li data-pid="LKQIYePU">雨刮、灯光、喇叭、后视镜</li>
  <li data-pid="2THOd3kk">雨刮用来下雨时刷掉雨水，如果不知道如何使用，那一定要问下教练如何开，毕竟你不能保证你不在一个下雨天考试。</li>
  <li data-pid="hjyTFS09">灯光主要是用来车和车之间进行沟通，以及晚上看不见路的时候照亮。科目二里面需要使用灯光的场景都是用来提醒别人的。</li>

  <li data-pid="IiAHUbLT">后视镜其实是一个凸面镜，用来增大视角的，会产生一些形变，主要的作用是向后看。</li>

 <h2>考试实操：停车入库</h2>
 <p data-pid="Mbk5A42E">其实所有的科目二考试项目有个特别反人类的地方，而且也是跟实际路面操作不符的地方：为什么要一次到位，且不能停车啊。实际道路上停车，我一般都是两把的。</p>
 <ul>
  <li data-pid="cIr5BjaX"><b>停车入库</b></li>
 </ul>
 <p data-pid="9mAM0d6V">秘诀：因为前轮可以转向，后轮不能主动转向。所以只要车后轮在恰当的位置，前轮可以随便调。如下图。</p>

 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="2Cqg56La">步骤如下</p>
 <ul>
  <li data-pid="BBISbrvG"><b>1.在合适的时候方向盘向右转到底</b></li>
  <ul>
   <li data-pid="t83Ecp-u">为什么是向右转到底</li>
   <ul>
    <li data-pid="6mtYGqTk">为了利用最大限度的将前轮向右转向，让后轮尽可能的向右库脚移动。</li>
   </ul>
  </ul>
  <li data-pid="V650De68">何时开始将方向盘向右转到底</li>

 <p data-pid="cs9Cus19"><b>2. 在合适的时间调整后轮的角度</b></p>
 <ul>
  <li data-pid="dAqg4lmC">当后轮在右库脚的延长线上时，可以开始调整后轮的角度。这样后轮在接下来的运动中才可能刚好摆在与左库脚和右库脚相同距离的中间完美位置。如图。</li>
 </ul>

   调整后轮角度的目的
  </figcaption>

 <ul>
  <li data-pid="Tq_9ysJL">怎么调整呢？靠观察右后视镜中右轮和右库脚的距离，这个距离可以预测出在不调整方向盘的情况下，右库脚和车右后轮的距离。如果这个距离太窄，则需要适当向左回下方向（向左回方向可以减小车后轮，往右库脚的偏向程度），回方向的时候注意观察调节（如果觉得自己向左回多了，则可以继续向右打方向），预测出车后轮接下来的运动轨迹。</li>
  <li data-pid="YUZ56AQN">直到车达到图中的状态 3 ，这一阶段算成功了。</li>
 </ul>
 <p data-pid="lCpoFl0M"><b>3. 把车摆直</b></p>

  <figcaption>
   调直车身
  </figcaption>
 </figure>
 <p data-pid="JOnQuBkx">怎么调？保持方向盘，看左后视镜中车身与左边库线平行时回正方向盘，车就会是上图状态。期间可以微调方向盘，诀窍就是保持车后轮在距离库两边的中间位置。</p>
 <p data-pid="aX9IxxLL"><b>4. 让车头超过线</b><br>
  这一段是最简单的：看标记。每次调整好座椅后让教练帮你停一个刚好过线的位置，然后你坐上去找一个死标记，记住就行了。</p>
 <p data-pid="xkrpigH0"><b>5. 出库</b></p>
 <p data-pid="1GAg8kBp">出库的时候，左后轮不要压倒左库脚。</p>
 <p data-pid="vcxX0cX4"><b>6. 从左边入库和从右边一样，入库再出库，出库向右拐的时候记住车头不要压到车头的那条线。</b></p>

 <h2><b>考试实操：侧边停车</b></h2>
 <p data-pid="AJr06par">侧边停车对比倒车入库来说要简单很多，车轮的方向和车的运动轨迹如下图。</p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/a340b73094edde20e75af820c427f7d7_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="269" data-rawheight="748" data-original-token="a340b73094edde20e75af820c427f7d7" class="content_image" width="269">
 </figure>
 <p class="ztext-empty-paragraph"><br></p>
 <p class="ztext-empty-paragraph"><br></p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="UPPwZdCn">分为几个阶段：<br>
  1. 找准时机向右打满方向</p>
 <p data-pid="oNXjScav">一般教练会让你从右后视镜中看不到库角1时开始右打死方向，目的也是让你让车身在“位置2”时就打方向。</p>
 <p data-pid="l0PMV0c5"><br>
  2. 车身在“位置3”时回正方向，回正后继续让车移动到“位置4”，此时右后车轮已经入库，左后车轮将要进库。位置3时车身左侧正好对着库角3 ，回正方向后让车身从 3位置 继续移动到4位置。这么做就是为了最大程度利用车库的纵向空间，同时又不压到右边线。</p>
 <p data-pid="gH7rU8cD"><br>
  3. 车身在位置4时向左打死方向盘，车会继续运动，左后轮不断入库，右后轮会向左移动些，直到将车停在6位置。</p>
 <p data-pid="lXXRwKdu"><br>
  4. 出库就很简单了，不说了。</p>

 <h2><b>考试实操：曲线行驶</b></h2>
 <ul>
  <li data-pid="OT0StI1y"><b>时间太久，具体的操作有点记不清了，但原则没变。</b></li>
  <li data-pid="IolP-qWD">曲线行驶的关键是判断车前轮和边线的距离，来让车后轮远离边线。</li>
  <li data-pid="PA39C0Ne">一般车的引擎盖的中间位置与弯外线重合的时候，车前轮都还没压到线。</li>
  <li data-pid="-N6I6s9P">重点：保证两个后轮不压线就行，前轮是可以随便调的。</li>
  <li data-pid="Cu4iR9Pq">重点：在遇到弯道的时候，让车的引擎盖前端尽量靠近边线，然后前车轮向弯心转弯。</li>
  <li data-pid="jFE5nd4a">重点：2、4、5，即遇到弯道则保证车头不出线情况下，让车前轮尽量贴近弯外，然后方向盘向弯心转弯。</li>
 </ul>
