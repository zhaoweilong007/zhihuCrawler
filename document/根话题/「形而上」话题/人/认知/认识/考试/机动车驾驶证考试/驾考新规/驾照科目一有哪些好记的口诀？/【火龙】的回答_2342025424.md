 <h2><b>二、违规知识</b></h2>
 <h2><b>1、口诀：【假一吊二撤三醉五逃终生】</b></h2>
 <p data-pid="_n5R_def"><b>假一</b>：提供虚假材料申领驾驶证，贿赂舞弊行为，一年内不得再次申领<br><b>吊二</b>：被吊销驾照后，二年内不得重新申领；<br><b>撤三</b>：以不正当手段取得驾驶证，公安机关撤销后，三年不得再次申领<br><b>醉五</b>：醉酒肇事，五年内不得重新申领；<br><b>逃终生</b>：肇事逃逸者，终生不得重新申领。<br><b>补充</b>：饮酒后驾驶机动车，暂扣驾驶证<b>6个月</b>；<b>3年内</b>有注射毒品行为不得申请机动车驾驶证</p>
 <h3>2、常考违规小知识点总结</h3>
 <ul>
  <li data-pid="yL4m_WUO"><b>15日</b>内交罚款</li>
  <li data-pid="TkFDNfyC">记分达到12分，<b>15日内</b>考科目一（不考科目三）</li>
  <li data-pid="0GUXGKTh">记分达到24分，<b>15日内</b>考科目一，<b>10天内</b>加考科目三</li>
  <li data-pid="E0VPPDbd">未按照国家规定投保交强险，<b>扣留机动车+</b>应缴纳保费的<b>2倍罚款</b></li>
  <li data-pid="ji_r-JxV">补领驾驶证后，依然使用原来的驾驶证，<b>处20-200的罚款</b>（注意：<b>不会</b>吊销和拘留！）</li>
  <li data-pid="De4FsWWu">驾驶证丢失、超过有效期、扣留、积分12分，<b>不得驾驶机动车！</b></li>
  <li data-pid="ByJfEvDg">吊销驾驶证的几种情况：1）超速50%；2）醉酒驾驶；3）驾驶拼装车或达到报废标准的机动车；4）将车交给驾驶证被吊销人员【<b>口诀：醉酒超速开破车，还把车借给无证人员，罚款200-2000+吊销</b>】</li>
  <li data-pid="3XMae_OX">拼装车上路要<b>强制报废+处200-2000罚款</b></li>
  <li data-pid="I9S7Ds2H">需要撤离不撤离导致交通堵塞的，对驾驶人员处以<b>200元</b>罚款。<b>【口诀：该撤不撤罚你200】</b></li>
  <li data-pid="Ae_lwPXH">证被扣仍然开车，200-2000罚款+拘留<b>【相当于无证驾驶，很严重，所以罚款+拘留！】</b></li>
  <li data-pid="8jYZsRUn">交通肇事后逃逸，尚不构成犯罪的，<b>罚款200-2000+可拘留</b></li>
 </ul>
 <h2><b>三、</b>初次申领驾驶证</h2>
 <ul>
  <li data-pid="CcvxQ95p">准考证明有效期<b>3年，</b>科目二、三考试预约次数不得超过<b>5次【记忆：2+3=5】</b></li>
  <li data-pid="k1vqDrwS"><b>18岁</b>以上不设上限（70以上也可以了！）</li>
  <li data-pid="Oh1QQjWk"><b>70岁</b>以上驾驶人<b>1年提交一次</b>身体条件证明，未提供<b>注销驾驶证</b>。</li>
  <li data-pid="3RoitmTr">视力要求<b>4.9</b>以上</li>
  <li data-pid="uQDguX0D">实习期、记分周期是<b>12个月</b>，驾驶证有效期是<b>6年</b></li>
  <li data-pid="_t8i8zQj"><b>3年</b>内有注射毒品的，不得申请机动车驾驶证。</li>
  <li data-pid="06lCx90k"><b>1年内</b>无累计积分的机动车驾驶人，可以延长机动车驾驶证审验期。</li>
  <li data-pid="kZNOGOEt">初次申领机动车，可以申请很多车（城市公交车、大型货车、小型汽车、小型自动挡汽车、低速载货汽车、三轮车、摩托车、有轨电车、无轨电车），反向记忆不能申请【<b>❌客车、牵引车】</b></li>
  <li data-pid="CVG9R3Em"><b>注意：持C1可以驾驶C2、C3、C4✅，但C2不能驾驶C3❌</b>【C1：小型汽车驾驶证；C2：小型／微型自动挡汽车；C3：低速载货汽车；C4：三轮车】</li>
 </ul>

  <li data-pid="ZDlMlIw2">驾驶证<b>补办【核发地】</b>；<b>换证【居住地】；</b>行驶证补办<b>【登记地】</b></li>

 <h2><b>六、</b>开车易错小知识点【汇总】</h2>
 <ul>
  <li data-pid="5aXl6d-9">靠路边停车要提前开启<b>右转向灯</b></li>
  <li data-pid="PC6zXBEq">驾驶车辆进入铁道路口<b>不能❌变换档位</b>，容易熄火</li>
  <li data-pid="4Y3AcBOS">雨雪沙尘暴等能见度低的情况下要使用【<b>近光灯】</b></li>
  <li data-pid="50MVasXU">冰雪天气起步开启<b>近光灯</b>。</li>
  <li data-pid="225RanDT">夜间车辆通过有路灯的路段使用<b>【近光灯】</b></li>
  <li data-pid="NlG5bYSd">超车后，应在与被超车车辆拉开必要距离后，<b>开启右转向驶入原车道</b>【<b>不能</b>立即驶入！】</li>
  <li data-pid="cHpRa7fn">道路上有中心圆，应<b>靠近中心圆左侧转弯</b></li>
  <li data-pid="Th6BgVFl"><b>不能掉头的标志</b>：人行道、实线、禁令标志、隧道</li>
  <li data-pid="3bcizqxo">马路牙子边黄漆，<b>漆满【禁止停车】，漆黄色虚线【禁止长期停车】</b></li>
  <li data-pid="KR-mkxG3">交叉路口、铁道路口、急转弯、宽度不足4m窄桥、桥梁、坡道、隧道以及距离上述地点<b>50m内路段不得停车【口5站3，不得停车</b>：交叉路口50米，加油站30米】</li>
  <li data-pid="PW97OZ7-">机动车变更车道要【<b>提前</b>】开启转向灯【“同时”开启是错误的❌】</li>
  <li data-pid="a1bEodEK"><b>轮胎气不足、胎压低，</b>与路面接触多，摩擦力大、会造成温度过高，造成爆胎</li>
  <li data-pid="kuuisoko">车辆下坡可以<b>减档行驶✅</b>，但不可以为了省油<b>空档滑行</b>❌</li>
  <li data-pid="EuGO6Rvc">搬运昏迷伤员首先<b>放开气道</b>，再采取仰卧位</li>
  <li data-pid="KBDDj_PW">停车位 ：<b>蓝色免费、白色收费、黄色专用</b></li>
  <li data-pid="Bi2FTbdv">狭窄坡路，<b>上坡先行</b>；下坡一方已经到中途了，另一方还没上车，就下坡先行。</li>
  <li data-pid="X9mxkC99">停车视距包括：<b>反应距离、制动距离、安全距离【口诀：反治安✅】，注意：</b>没有【启动距离❌】</li>
  <li data-pid="x0Hhww_v">雨天行车地面湿滑，<b>阻力小</b></li>
  <li data-pid="y93zjFm3">校车停靠时后方车辆停车等待，<b>不得</b>鸣笛催促</li>
  <li data-pid="N2mR_l5Z"><b>临时停车</b>：车身距道路边缘不超过<b>30厘米</b></li>
  <li data-pid="0ZPGJEAe">逆时针2个知识点：1）上车前<b>逆时针</b>绕车一周进行安全检查；2）进入环岛<b>逆时针</b>行驶</li>
  <li data-pid="aSNDTujn">夜间通过<b>急转弯、坡路、拱桥、人行横道、没有交通信号灯控制的路口</b>，应当交替使用<b>远近光灯。</b></li>
  <li data-pid="2yOFPemP">侧滑：<b>车尾</b>向那一侧滑动，就向哪一侧适当调整方向盘。转动时动作要敏捷轻盈。</li>
  <li data-pid="z-8Ze1hv">机动车道上不得从机动车【左侧❌】上下车，右侧可以</li>
  <li data-pid="HMZ4CEWV">驾驶证吊销后，使用伪造驾驶证，处<b>15日以下拘留+罚金</b>（无证驾驶）</li>
  <li data-pid="9yozdtLK">机动车<b>不能非法安装报警器、标志灯具</b>，公安强制拆除收缴+罚款200-2000.</li>
  <li data-pid="jLFOv2m-">绿光在左<b>【前雾灯】</b>、黄光在右<b>【后雾灯】</b></li>
  <li data-pid="W0VjcMPW">光束直射，蓝色向左是【<b>远光灯】</b>；光束向下，绿色在左是【<b>近光灯】</b></li>
 </ul>

 <h3><b>1、开车三个先行原则</b></h3>
 <ul>
  <li data-pid="IaxwQLtW">转弯车辆让<b>直行车</b>辆先行</li>
  <li data-pid="2XVs6zs_"><b>右方道路</b>来车先行</li>
  <li data-pid="ABHKZLjv">右转弯车让<b>左转弯</b>车辆先行</li>
 </ul>
 <p data-pid="A6KL-DVf"><b>【口诀】 直行车&gt;左转弯&gt;右转弯</b></p>
 <h3><b>2、开车必备</b></h3>
 <p data-pid="iZe0zlWs">随车携带机动车两证两标一牌</p>
 <ul>
  <li data-pid="8j4xVL_h"><b>两证</b>：行驶证、驾驶证</li>
  <li data-pid="wciC06XO"><b>两标</b>：检验合格标志、保险标志</li>
  <li data-pid="EmDv6ffn"><b>一牌</b>：悬挂机动车牌</li>
 </ul>
 <h3><b>3、限速30的情形</b>：</h3>
 <ul>
  <li data-pid="NigRlG4B">车辆驶入<b>匝道</b></li>
  <li data-pid="7LJLJ-4h">进出<b>非机动车道、铁路路口、急转弯、窄路、窄桥；掉头、转弯、下陡坡</b>；</li>
  <li data-pid="Jki-RRnj">遇雾雨雪沙尘冰雹<b>能见度&lt;50米</b>；</li>
  <li data-pid="_t2MHITn"><b>冰雪泥泞</b>道路行驶；</li>
  <li data-pid="Jbq2zZnz"><b>牵引发生故障的机动车</b>时。</li>
 </ul>
 <h3><b>4、没有限速标志、标线的道路上，最高时速</b></h3>
 <ul>
  <li data-pid="pyZLUMxt">没有道路中心线的道路、城市道路30km／时；公路40km／时</li>
  <li data-pid="1QPUm2ME">同方向只有一条机动车道的道路，城市道路50km/时，公路70km／时</li>
 </ul>

  <br>

  <br>

 <h2>七、高速路相关知识</h2>
 <ol>
  <li data-pid="45kJcnvr">高速路能见度<b>&lt;200米</b>：开启雾灯、近光灯、示廓灯、前后位灯，<b>车速&lt;60km/时，车距&gt;100米。</b></li>
  <li data-pid="slxadJIT">高速路能见度<b>&lt;100米</b>：开启雾灯、近光灯、示廓灯、前后位灯+<b>危险报警灯</b>，<b>车速&lt;40km/时，车距&gt;50米。</b></li>
  <li data-pid="xWG-9VnX">高速路能见度<b>&lt;50米</b>：开启雾灯、近光灯、示廓灯、前后位灯+<b>危险报警灯，车速&lt;20km/时</b>，尽快驶离高速公路。</li>
 </ol>

  <li data-pid="8-UyOgNl">3）能见度 50，车速20，车距看不清，开危险报警灯，快点离开！！！</li>

 <p data-pid="1veupZh4">1.车辆驶入<b>匝道后，限速30km</b>，等驶入加速车道后再提速<b>60km+</b>。</p>
 <p data-pid="hjiDo6PM">2.高速公路匝道<b>不得</b>倒车、掉头、加速</p>
 <p data-pid="LuNZLSH6">3.高速公路上左侧-<b>超车道</b>，中间-<b>主车道</b>，右侧-<b>应急车道</b>，右侧路肩-<b>供紧急情况临时停车</b>（路肩不可以行驶）</p>
 <p data-pid="ZZCA5FAX">4.高速公路上不能停车，感到疲劳去服务站休息</p>
 <p data-pid="9NsONq71">5.高速路车速&gt;100km，保持100米车距，&lt;100km，保持50米车距。</p>
 <p data-pid="CgjL0PAB">6.高速路车辆故障处理步骤：<b>1.开启危险报警闪光灯；2.将车辆移至不妨碍交通的位置；3.放警告标志（高速150米，普通道路50-100米），乘车人员转移至安全处后迅速报警；4.等待救援。</b></p>
 <p data-pid="wIierquM"><b>7.高速路驾车，车速&gt;100km,车距&gt;100m；车速&lt;100km，车距&gt;50m</b></p>

 <h3>1、扣1分</h3>

  <li data-pid="uTmEHtx7">人行道不避让行人。</li>

  <li data-pid="Loc-T23R">闯红灯</li>

 <ul>
  <li data-pid="TUUE_j9o">驾驶的车辆与准驾车型不符</li>
  <li data-pid="a8N-mZXP">没有悬挂牌照或者是牌照存在污损</li>

  <li data-pid="OyrTPljh">无证驾驶校车</li>

 <ul>
  <li data-pid="0Yx3zGCV">酒后驾驶车辆</li>

 <h2>十、交警手势</h2>
 <ul>
  <li data-pid="sRu5baQY">【停车】左手高举，掌心向前</li>

  <li data-pid="1Q5gQpGO">【左转弯】右手平举，掌心向前，左手斜向下</li>
  <li data-pid="E1IyO-eR">【右转弯】左手平举，掌心向前，右手斜向下</li>
  <li data-pid="N-2wVRDS">【等待转弯】单手向身体一侧（左边或是右边）上下摆动</li>
  <li data-pid="FFAacLOv">【变道】右手单手在胸前左右摆动</li>

 </ul>
