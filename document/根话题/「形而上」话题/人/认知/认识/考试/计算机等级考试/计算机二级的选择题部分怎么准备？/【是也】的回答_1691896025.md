 <p data-pid="a86ixGwA"><b>选择题不可以放弃！！！</b></p>
 <p data-pid="mwraf41M">但也不能死背，我第一次考的时候，前一个星期开始背计算机的选择题，几乎是背一遍忘一遍，而且还特别痛苦。拿着知识点背，什么都不知道，对着真题去背，背完还选错ಥ_ಥ，那就是人间小迷糊。</p>
 <p data-pid="8qL2zYFX">所以推荐大家在考试前两个星期就开始刷选择题，1000多道选择题，无论是在手机版还是纸质版，一个星期都能刷完，不过可不是简单刷一遍这么简单，把自己容易错的题收藏标记出来。</p>
 <p data-pid="m7LPvXAT">我也给大家整理了一部分容易弄混、弄错的选择题，<b><i>结合真题使用，效果更佳。</i></b></p>
 <p data-pid="NHekDsbv"><i>这部分只是针对</i><b><i>公共基础(也就是前10道题</i></b><i>)，后10道相对比较简单，在操作中有体现。</i></p>
 <p data-pid="43Pi9utU">大家也不用全背，把结合这些知识点把全部选择题做一遍，再看一遍易错的题，就可以了。</p>

 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="V3YaK1vO"><b><i>一、计算机系统</i></b></p>
 <ul>
  <li data-pid="lprBRIcr">世界上第一台电子计算机问世的时间：1946年2月15号(20世纪40年代)</li>
  <li data-pid="tq3VAPPp">地点：美国</li>
  <li data-pid="WKwDbFZM">名称：埃尼阿可(ENIAC)</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="Y-Bc62M8">计算机网络的功能包括三个方面:</li>
  <li data-pid="toUanatz">①资源共享</li>
  <li data-pid="c3Un-yeY">②信息传送</li>
  <li data-pid="b9pb4dFz">③分布处理</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="m0lZ380v">计算机具有主要的特点</li>
  <li data-pid="8F7Ltq_C">①高速、精确的运算能力</li>
  <li data-pid="bGf0KNyD">②准确的逻辑判断能力</li>
  <li data-pid="fE9OjoNo">③强大的存储能力</li>
  <li data-pid="NTQdj3w-">④自动功能</li>
  <li data-pid="w1xrrsR_">⑤网络与通信功能</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="8pfSIqHF">计算机系统由计算机<b>硬件和软件</b>两部分组成</li>
  <li data-pid="TsMqw5bR">硬件包括中央处理器、存储器和外部设备等</li>
  <li data-pid="8s90Itap">软件是计算机的运行程序和相应的文档</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="w_4d6d1t">计算机低级语言：机器语言、汇编语言、符号语言</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="v74X4PPZ">计算机高级语言</p>
 <ul>
  <li data-pid="0U3FmNwN">①高级语言的数据结构要比汇编和机器语言丰富</li>
  <li data-pid="lFxfPRyp">②高级语言与具体机器结构的关联没有汇编及机器语言密切</li>
  <li data-pid="YM-PNIu-">③高级语言更接近自然语言(算法语言)更容易被掌握</li>
  <li data-pid="JZQ7ssNN">④高级语言编写的程序要经过编译或解释计算机才能执行</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="Jujlwm6m">计算机采用的电子器件为:</li>
  <li data-pid="Vs9QbE_9">第一代是电子管</li>
  <li data-pid="z3AZNqm-">第二代是晶体管</li>
  <li data-pid="ov8HjJJC">第三代是中、小规模集成电路</li>
  <li data-pid="BfVXTR9z">第四代(当代)是大规模、超大规模集成电路</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="ngRZ57Tw">计算机辅助教育:它的总称是CBE(Computer Based Education)，包含CAI、CAD和CMI。</li>
  <li data-pid="nDe0Yshi">CAI:指计算机辅助教学(Computer Aided Instruction)，它是利用计算机以向学习者提供图文并茂、生动形象的图像的方式进行教学，提高学习兴趣和效果。</li>
  <li data-pid="gPdIBcO0">CMI指计算机管理教学(ComputerManaged Instruction)</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="1W-p1Ft8">操作系统是是管理、控制和监督计算机硬件与软件资源的计算机程序，是<b>系统软件的核心，它是直接运行在计算机硬件上的，最基本的系统软件</b>。其主要功能包括:</li>
  <li data-pid="k_D9Z-DB">①CPU管理</li>
  <li data-pid="8de35Etz">②存储管理</li>
  <li data-pid="WKMaJIp5">③文件管理</li>
  <li data-pid="2dPbWi9S">④设备管理</li>
  <li data-pid="oz5E_auq">⑤作业管理</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="V7iTGLdU">只读型光盘英文缩写是CD-ROM</li>
  <li data-pid="AIny_LR9">一次写入光盘英文缩写是CD-R</li>
  <li data-pid="GxRkQ82I">可擦写型光盘英文缩写是CD-RW</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="6SPeUOv1">ROM，即只读存储器,其特性是一旦储存资料就<b>无法再将之改变或删除</b>。电源关闭后存储在其中的数据<b>不会消失</b>。</li>
  <li data-pid="h486RTl_">RAM是计算机内存储器中的一部分，是一种可读可写存储器，可以细分为SRAM和DRAM，用来存储当前正在运行的应用程序和其相应数据。存储内容<b>可以修改</b>，电源关闭后存储在其中的数据<b>会消失</b>。其特点为:</li>
  <li data-pid="dLjatx3c">①随机存取</li>
  <li data-pid="Zh4Tt50v">②易失性</li>
  <li data-pid="bQmDODWT">③对静电敏感</li>
  <li data-pid="hkc1np3F">④访问速度快</li>
  <li data-pid="xMAFN6c8">⑤需要刷新</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="JLuNdiRN">常见的视频文件格式有：</li>
  <li data-pid="ginzw44_">AVI</li>
  <li data-pid="f5nUVJhc">MOV</li>
  <li data-pid="FEMLkkeu">ASF</li>
  <li data-pid="_NDcsXU-">WMV</li>
  <li data-pid="F04gAXa4">RM</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="9OogpN7R">输入设备:输入设备是向计算机输入数据和信息的装置，用于向计算机输入原始数据和处理数据的程序。</li>
  <li data-pid="OXshFugS">常用的输入设备有：键盘、鼠标、触摸屏、摄像头、扫描仪、光笔、手写输入板、游戏杆、语音输入装置，还有脚踏鼠标、手触输入、传感等。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="afv1VntQ">计算机病毒(ComputerVirus)是编制者在计算机程序中插入的破坏计算机功能或者数据的代码，能影响计算机使用，能自我复制的一组计算机指令或者程序代码。</li>
  <li data-pid="yyJyfcR6">计算机病毒具有传播性、隐蔽性、感染性、潜伏性、可激发性、表现性或破坏性。</li>
  <li data-pid="wI3jfuD3">计算机病毒是一个<b>程序</b>，<b>一段可执行码</b>。就像生物病毒一-样， 具有自我繁殖、互相传染以及激活再生等生物病毒特征。计算机病毒有独特的复制能力,它们能够快速蔓延，又常常难以根除。</li>
  <li data-pid="AM6Pndv3">它们能把自身附着在各种类型的文件上，当文件被复制或从一个用户传送到另一个用户时，它们就随同文件一起蔓延开来。</li>
  <li data-pid="8ynqM1Mx">计算机染上病毒后可能出现</li>
  <li data-pid="fSR9o4tp">1.经常死机</li>
  <li data-pid="aHK8cNrp">2.文件打不开</li>
  <li data-pid="9iGMq18o">3.系统无法启动</li>
  <li data-pid="xAKSoDfk">4经常报告内存不够</li>
  <li data-pid="_8g9UHtw">5.U盘等设备未访问时出读写信号</li>
  <li data-pid="qt0ZIu8p">6.提示硬盘空间不够</li>
  <li data-pid="KuLHRLJO">7.出现大量来历不明的文件</li>
  <li data-pid="23GKG0pH">8.启动黑屏.数据丢失</li>
  <li data-pid="tt6JJaPO">9.键盘或鼠标无端地锁死</li>
  <li data-pid="SVk5xAHj">10.系统运行速度慢</li>
  <li data-pid="1Oy8nAm0">11.系统自动执行操作</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="14urY28m">网络拓扑是指网络形状，常见的网络拓扑结构有:总线型、星型、环型、树型、网状型拓扑结构等。</li>
  <li data-pid="ravKq83_">局域网常用的网络拓扑结构是总线型、星型、环型。</li>
  <li data-pid="sfQsTF3H"><b>星型</b>拓扑结构是中央节点和通过点到点链路连接到中央节点的各节点组成。</li>
  <li data-pid="RYOjpJON"><b>环型</b>拓扑结构各个节点通过中继器连接成一个闭合环路。</li>
  <li data-pid="vGbegMFK"><b>总线型</b>拓扑结构是采用一根传输总线作为传输介质，各个节点都通过网络连接器连接在总线上</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="43haqKOL">LAN局域网</li>
  <li data-pid="__i-vO1z">WAN广域网</li>
  <li data-pid="F0Zjmw_s">MAN城域网</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="FnSD7GNU">操作系统：win7 ios unix dos</li>
  <li data-pid="kT_NcnS7">办公软件：office excel ppt</li>
  <li data-pid="zRff5L0g">数据库系统：Access SQL Oracle</li>
  <li data-pid="PLDXOXoV">计算机语言：JAVA Python C语言</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>

  <li data-pid="SRm9Ui-h">应用软件：人事管理 财务管理 财务报表</li>

 <ul>

 <p data-pid="UMSyHz5a">ASCII码代表的数</p>

 </ul>

 <ul>
  <li data-pid="WfrMNU9o">冒泡排序需要比较0(n^2)次 (n(n-1)/2次)</li>
  <li data-pid="em7M_G1S">简单选择排序,无论是否最坏都需要0(n^2)次(n(n-1)/2次)</li>
  <li data-pid="gctEqGjI">直接插入排序,最坏情况需要比较0(n^2)次(n(n-1)/2次)</li>
  <li data-pid="rMwJeUmH">在最坏情况下快速排序、冒泡排序和直接插入排序需要的比较次数都是n(n-1)/2</li>
  <li data-pid="j87trfWv">堆排序需要比较的次数为nlog2 n</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="d0VEhf0W">遍历序列排序口诀：前中一样，后序倒着写 中后一样，前序倒着写</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="Jo1fPH9b">顺序储存——一定是连续的</li>
  <li data-pid="KKZ-e1Lk">链式存储——不一定是连续的</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="ffg_brmq">循环队列的队头指针与队尾指针都不是固定的,随着入队与出队操作进行变化。因为是循环利用的队列结构，所以对头指针有时可能大于队尾指针，有时也可能小于队尾指针。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="BAnzFbst">线性表的顺序存储结构具备如下两个基本特征:</li>
  <li data-pid="pV4QiEL_">①线性表中的所有元素所占的存储空间是连续的</li>
  <li data-pid="L_UtLn4Z">②线性表中各数据元素在存储空问中是按逻辑顺序依次存放的。用一-组任意的存储单元来依次存放线性表的结点，这组存储单元既可以是连续的，也可以是不连续的，甚至是零散分布在内存中的任意位置上的。</li>
 </ul>

 <p data-pid="Ac52gxsj"><b><i>四、数据库设计基础</i></b></p>
 <ul>
  <li data-pid="STSa9gjf">数据库管理系统是指位于用户与操作系统之间的数据管理软件。</li>
  <li data-pid="90HcuS_i">数据库管理系统是为数据库建立、使用和维护而配置的软件。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="bZo3-rdx">数据管理技术的发展经历了3个阶段:人工管理阶段、文件系统阶段和数据库系统阶段</li>
  <li data-pid="K99XN26j">人工管理阶段无共享，冗余度大</li>
  <li data-pid="3UZbvDNt">文件管理阶段共享性差，冗余度大</li>
  <li data-pid="mNiSId3H">数据库系统管理阶段共享性大，兀余度小</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="V5RAt0xq">数据库管理系统的特点:</li>
  <li data-pid="QfeQtUdu">共享性高，冗余度小;</li>
  <li data-pid="UAYdF9wR">具有高度的物理独立性和逻辑独立性;</li>
  <li data-pid="6ruB3L6w">整体结构化，用数据模型描述;</li>
  <li data-pid="PvDuHGEV">由数据库管理系统提供数据安全性、完整性、并发控制和恢复能力</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="bRoY9SHo">数据库管理系统的功能包括</li>
  <li data-pid="eJvmP1Gi">数据模式定义</li>
  <li data-pid="JVXWo8aP">数据存取的物理构建</li>
  <li data-pid="3yHa9GNr">数据操纵</li>
  <li data-pid="WR-MS9bN">数据完整性安全性的定义与检查</li>
  <li data-pid="oWihLaHH">数据库的并发控制与故障恢复、</li>
  <li data-pid="7hlXnlrV">数据的服务</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="JCdr0xKW">范式NF</li>
  <li data-pid="AKAGPx8h">1NF：列的原子性，每列可再分 一般是一张表</li>
  <li data-pid="WF3xwK_S">2NF：在一范式的基础上，消除部分函数依赖</li>
  <li data-pid="IZ5A9dHC">3NF：在二范式的基础上，消除传递函数依赖 一般是三张表</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="EOcnbH6D"><b>自然连接</b>是一种特殊的等值连接，它要求两个关系中进行比较的分量必须是相同的属性组，并且在结果中把重复的属性列去掉</li>
  <li data-pid="78gkhfCb"><b>并</b>是指将两个关系中的元素全部合并到一个新的关系之中;(两表等值连接)</li>
  <li data-pid="WPqo1TC6"><b>交</b>是指将那些同时属于两个关系的元素组成一个新的关系;差是指将属于前一个集合但不属于后一个集合的元素组成新的关系。</li>
  <li data-pid="IGnFe7P6"><b>差</b>是两表等差连接</li>
  <li data-pid="3k-V-uny"><b>选择</b>运算又称为限制。它是指从一个关系.(表)中找出满足一定条件的所有元组(记录),即在二维表中选取若干行。选择运算是根据某些条件对关系做水平分割，即选取符合条件的元组。</li>
 </ul>
