 <h2>三、自学技巧</h2>
 <p data-pid="epScckxo">1、<b>解决选择题的灵丹妙计—看视频</b></p>
 <p data-pid="_Fxdu6J_">在你操作题做到天昏地暗的时候，我再告诉你个方法来缓解下你的情绪。<b>停下来去看选择题公共基础的视频。</b></p>
 <p data-pid="5yWcw5Ir">选择题分为两大块：<b>公共基础和操作应用题</b></p>
 <p data-pid="1CjxQmEy">操作应用不用担心，几套卷子一做你就会了，最头疼的是公共基础，对于那时没有一点计算机基础的我来说是块特别难啃的硬骨头。</p>
 <p data-pid="cu2vvTsu">这是从B站上找的资源，分享给大家：</p><a data-draft-node="block" data-draft-type="link-card" href="https://link.zhihu.com/?target=https%3A//www.bilibili.com/video/av10270113%3Ffrom%3Dsearch%26seid%3D1003155080261894516" data-image="https://pic1.zhimg.com/v2-8c6c7cc3d5432554a8a6605e1201e2dc_180x120.jpg" data-image-width="1920" data-image-height="1080" class=" wrap external" target="_blank" rel="nofollow noreferrer">计算机二级office全套视频【真题】&amp;【实时更新】_哔哩哔哩 (゜-゜)つロ 干杯~-bilibili</a>
 <p data-pid="aJOUANbh">看视频不用动脑子，她列出来的重点你就记下来，记住你看视频的目的不是搞懂每一个知识点，而是梳理出框架，在头脑中把各个知识点串联起来。看完视频再去做题会得心应手很多。</p>
 <p data-pid="Q_eonxmW"><b>2、Word重难点汇总</b></p>
 <ul>
  <li data-pid="CXzUzGE3">字体的设置（如果题目提供样图片样张让你自行设置，修改的尽量相似即可）</li>
  <li data-pid="5o5PlPmh">段落设置（注意首行缩进，段落间距的单位有行和磅，单位可以手动输入，行距中的固定值和多倍行距）</li>
  <li data-pid="wuYRF08K">样式的新建，修改，复制，赋予，样式集的使用（考试频率高，题量大）</li>
  <li data-pid="gbjdZNj-">替换是考试的重点（批量修改内容、样式，批量删除内容）</li>
  <li data-pid="kZ6_oRtX">选择中的矩形选择和选择格式相似的文本</li>
  <li data-pid="XQCJvZdB">选择性粘贴（粘贴Excel表格到文档中，并随着数据源改变而自动更新）</li>
  <li data-pid="PrbtgrLw">插入系统自带封面（运动型封面要先将日期域控件和时间控件删除）</li>
  <li data-pid="pSAKPih3">文档中插入表格，并进行格式设置（文本转换成表格，重复标题行是重点）</li>
  <li data-pid="zjaMwAaP">图表部分中复合图表（注意主次坐标轴的设置）是重点也是难点。</li>
  <li data-pid="OQoPSy2n">利用图片形状和smartart做一份个人简历</li>
  <li data-pid="eUAfsmBq">插入超链接（链接到文件、网址，链接到本文档中的位置），快捷键是ctrl+k。</li>
  <li data-pid="6CDUJaFW">页眉页脚部分，绝对是word文档中最重要的考点也是最难的考点。</li>
  <li data-pid="s91kbLJc">初级版：统一的页眉页脚，为页眉添加一条上宽下窄的线条，添加公司logo。</li>
  <li data-pid="PXPyzTZ5">升级版：奇偶页不同，首页不同，设置页码格式。</li>
  <li data-pid="YAE_DNNC">终极版：不同的章节显示不同的页眉，多种页码格式，结合奇偶页不同和首页不同。</li>
  <li data-pid="gO7vBq17">插入文本框模板，绘制文本框，文本框格式的设置，（利用文本框制作一个论文封面）</li>
  <li data-pid="3wNcuKbG">插入文档属性，插入域（类型为文档信息的域，styleref域），将所选内容保存到文档部件库，域代码的修改。</li>
  <li data-pid="YtIOuJdn">插入艺术字（上弯弧的文本效果），首字下沉的设置，系统日期的插入（自动更新），符号的插入（对号和五角星等）</li>
  <li data-pid="VPXuG1El">为word文档应用一个主题。</li>
  <li data-pid="bygZwl_n">页面设置：页边距、装订线、纸张方向/大小、页码范围（对称页边距，拼页）、页眉页脚距边界的距离的设置，指定行网络，设置行数。</li>
  <li data-pid="UvkEqRuL">分栏部分：设置栏数、分隔线，栏间距，分栏符，表格和图表跨栏居中。</li>
  <li data-pid="JTW7avsp">分隔符部分，重点是分节符（下一页，连续，偶数页，奇数页）</li>
  <li data-pid="L0TrI-yJ">分清水印和页面背景的不同。</li>
  <li data-pid="eiXP_E_j">页面边框的设置，艺术型边框经常考，请注意比较页面边框，段落边框，字符边框的不同。</li>
  <li data-pid="WcN0xBLJ">自动生成目录，插入特定级别的目录，目录格式的设置，更新目录。</li>
  <li data-pid="4f87RHRv">脚注部分，插入脚注尾注，设置脚注格式，脚注和尾注的相互转化。</li>
  <li data-pid="DGCZg4rT">题注部分，本知识点经常和多级列表同时考核，难度系数比较高，题注的插入，题注样式的修改，交叉引用，表目录的插入。</li>
  <li data-pid="q73HsURI">索引部分，标记索引项，插入索引，批量删除索引</li>
  <li data-pid="BUtBuK2E">邮件合并部分（注意原文档和结果文档的区别，注意编辑收件人列表的设置，规则的设置）</li>
  <li data-pid="_MxC-2fF">中文的简繁转换，新建批注，文档的保护。</li>
  <li data-pid="dt3QGxIT">大纲视图的使用，导航窗格的使用，显示比例的调整。</li>
 </ul>
 <p data-pid="YDmedJks"><b>3、Excel重难点汇总</b></p>
 <ul>
  <li data-pid="JJOIITU-">表格格式的调整（行高、列宽，边框、底纹）</li>
  <li data-pid="sH61Ue1T">单元格格式的调整（对齐和数字，其中要注意的是跨列合并居中，自定义格式的设置）</li>
  <li data-pid="MG_77ZME">数据的选择性粘贴（转置粘贴）</li>
  <li data-pid="TaZ8mjCg">绝对引用，相对引用，混合引用。</li>
  <li data-pid="v7fVdWPw">条件格式的设置（难点是利用公式确定的条件格式）</li>
  <li data-pid="1tuI3MHi">套用表格格式</li>
  <li data-pid="br-OxZCl">为数据区域定义名称，将表格转换为区域。</li>
  <li data-pid="NjbwvOLV">自定义排序（需要注意的是按笔画降序排序，按自定义序列排序、按颜色排序）</li>
  <li data-pid="GkAZO2ig">筛选（自动筛选和高级筛选，自动筛选用得比较多，需要特别注意高级筛选）</li>
  <li data-pid="9AHARQiw">插入图表（复合图表依然是重点，迷你图也是一大考点）</li>
  <li data-pid="0unTW0Hw">页面设置（纸张大小，纸张方向，页边距，其中缩放打印是重点）</li>
  <li data-pid="E6b2oidR">获取外部数据（自网站、自文本重点）</li>
  <li data-pid="CoJeorts">分列的使用</li>
  <li data-pid="Atnys5-m">数据有效性的设置（整数 小数 日期 序列 文本长度 公式）</li>
  <li data-pid="eUHDrCYp">合并计算</li>
  <li data-pid="PptZk94o">分类汇总（重点，需要特别注意的就是在分类汇总前应该按分类字段进行排序）</li>
  <li data-pid="pGSiQNhg">插入数据透视表（注意分组和报表筛选页功能的使用），插入数据透视图。</li>
  <li data-pid="50UvOEuL">函数公式专题</li>
  <li data-pid="_HdJor33">五大计算函数：sum( )求和 ， average( )求平均， max( )求最大， min( ) 求最小 count( )求个数</li>
  <li data-pid="YWKUmwUr">条件判断函数：if( ) 注意多层嵌套的使用</li>
  <li data-pid="p3ok3gfy">排名函数：rank( )请注意绝对引用问题</li>
  <li data-pid="DTi88YeS">条件计算函数：sumif( ) 单条件求和，sumifs( ) 多条件求和，countif( )单条件求个数，countifs( )多条件求个数。</li>
  <li data-pid="rBFE7Mpz">日期函数：date( ) ， today( ) ，datedif( ) ，year( )，month( )，day()， weekday( )</li>
  <li data-pid="VK9ivAr4">数值函数：mod( )求余数，int( )求整，round( )四舍五入函数，roundup( )，rounddown( )，large( )</li>
  <li data-pid="-7bn_Zhq">文本函数：Left( )， Right( )， Mid( )，Len( )，Text( )，&amp;</li>
  <li data-pid="rhFP9Fg7">查询函数：vlookup( ) ，lookup( )</li>
  <li data-pid="jxwbR00t">数组函数是一大难点，注意运行数组函数的快捷键是ctrl+shift+enter</li>
 </ul>
 <p data-pid="WeqMpw03"><b>4、PPT重难点汇总</b></p>
 <ul>
  <li data-pid="0Z76lgoG">新建幻灯片（从大纲，重用幻灯片）</li>
  <li data-pid="xl2Dam-v">设置版式，分节，重命名节。</li>
  <li data-pid="WQo1N7p3">文本转换成smartart</li>
  <li data-pid="EsMuWs0p">统一替换字体</li>
  <li data-pid="3w0VEc5V">给不同对象插入超链接</li>
  <li data-pid="Bq9q83TB">插入日期时间，幻灯片编号，页脚，设置标题幻灯片不显示。</li>
  <li data-pid="gm8WU_G1">插入音频（设置背景音乐，考试时最好带个小耳机，有些电脑会出现无音频播放设备无法插入音频的情况。）</li>
  <li data-pid="KsuOgqgG">设置幻灯片大小，方向，应用幻灯片主题，设置背景样式。</li>
  <li data-pid="BKToV5XK">设置幻灯片切换效果、自动换片时间。</li>
  <li data-pid="2TcahgHp">为不同对象设置动画，设置动画顺序、开始方式。（尤其需要注意图表动画和smartart对象的动画设置。）</li>
  <li data-pid="jbF8he49">创建自定义幻灯片放映方案</li>
  <li data-pid="Pkc9BJj_">隐藏幻灯片</li>
  <li data-pid="O68zPkh-">设置幻灯片放映方式、放映选项。</li>
  <li data-pid="CVO_UgzO">母版视图的设置，一次性删除所有备注</li>
 </ul>
