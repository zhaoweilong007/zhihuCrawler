 <p data-pid="whxX-12g">========================================================</p>
 <p data-pid="wk6I5HrP">你说的是凯利·麦格尼格尔的《自控力》吧。个人认为是一本教会你管理自己的好书，虽然它也不是万能的，但对于年轻人管理自己的工作、学习和压力还是很有用的！</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="JFgipxTG">喜欢它的原因如下：</p>
 <ol>
  <li data-pid="2I7T_G3F"><b>有大量实验例证，读起来更有启发，而且不会枯燥</b><br></li>
  <li data-pid="DRxX1wTv"><b>用比较浅显的语言帮助我了解自己的情绪和大脑，纠正了很多我以前不知道的观念。</b><br></li>
  <li data-pid="aG_ILlFA"><b>最重要的一点，有实际操作战略，日常生活中很实用！</b><br></li>
 </ol>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="mPkBUKJg">缺点嘛，作者有点啰嗦，实际上有用的信息不多。</p>
 <p data-pid="kPJqyeeY"><b>所以我总结出了读书笔记，时不时的拿出来翻翻就可以了！分享一下吧，不谢！</b></p>
 <p data-pid="EQfZwmbv"><b>有用的话记得回来点个赞~</b></p>

 <p data-pid="eFMydlzi"><b>前言</b></p>
 <p data-pid="861VuVdx">自知之明是自控的基础。认识到自己的意志力存在问题，则是自控的关键。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="ZJC-UpPE"><b>第一章：我要做，我不要，我想要：什么是意志力？为什么意志力至关重要？</b></p>
 <p data-pid="ncRm7n4S">理论：</p>
 <ul>
  <ul>
   <li data-pid="GhyBPwOB">如果没有自我意识，大脑总会默认选择最简单的。</li>
   <li data-pid="FCxVP2j-">注意力分散的人更容易向诱惑屈服。</li>
  </ul>
 </ul>
 <p data-pid="EiR6OKAD">战略：</p>
 <ul>
  <ul>
   <li data-pid="9s0umRly">当你面对诱惑和拖延症时，你得想清楚，你真正想要的，其实是XXX。只有想到这些，才能遏制你的一时冲动。这就是“我想要”的力量。</li>
   <li data-pid="yeWNPvYy">你可以给冲动的自我起个名字，比如把及时行乐的人叫做“饼干怪兽”，把总是<br>
     不想开工的人叫做“ 拖延者”。在它们占上风的时候，你就能意识到他们的存在，帮你唤醒那个明智的自己，唤醒意志力。</li>
   <li data-pid="CreJrnte">花时间观察自己是怎样屈服于冲动的，什么样的想法、感受和情况最容易让你有冲动，想些什么或者暗示能让你放弃冲动？</li>
   <li data-pid="-XCByJ06">冥想可以提升自控力，提升你集中注意力、管理压力、克服冲动和认识自我的能力。一段时间后，你的前额皮质和影响自我意识的区域里，大脑灰质都会增多。</li>
   <ul>
    <li data-pid="ijH6fpdd">原地不动，安静坐好，背挺直，双手放在膝盖上。</li>
    <li data-pid="7JHY_bsR">注意你的呼吸。闭上眼睛，吸气的时候在脑中默念“吸”，呼气时再脑中默念“呼”。走神的时候，重新将注意力集中到呼吸上。</li>
    <li data-pid="TZIXn1Vr">感受呼吸，弄清自己是怎么走神的。你会注意到空气从鼻子和嘴巴进入和呼出的感觉，感觉的吸气时胸腹部的扩张和呼气时胸腹部的收缩。</li>
    <li data-pid="p3-RwoWq">每天做比较短的训练，也比把较长的训练拖到明天好。</li>
    <li data-pid="j6R4p0wR">冥想时感觉越糟糕，它在现实生活中的作用就越明显</li>
   </ul>
  </ul>
 </ul>
 <p data-pid="3QuOiSwb"><b> 第二章：意志力的本能：人生来就能抵制奶酪蛋糕的诱惑</b></p>
 <p data-pid="xtk-0ufv">理论：</p>
 <ul>
  <ul>
   <li data-pid="7gx3Fp-R">应激反应：当人们感到压力时，交感神经系统会控制身体，心率升高，心率变异度会降低，这种生理现象让你能够战斗或者逃跑。</li>
   <li data-pid="EbIyeF6B">三思而后行：当人们成功自控的时候，副交感神经系统会发挥主要作用，心率降低，心率变异度会升高，帮助缓解压力，控制冲动行为。</li>
   <li data-pid="kjhjuJV9">压力是意志力的死敌。任何给身心带来压力的东西都会影响自控力的生理基础，摧毁意志力。如焦虑、愤怒、抑郁、孤独、慢性疼痛和疾病。</li>
   <li data-pid="iOFYxFVT">睡眠不足会影响身体和大脑吸收葡萄糖，当细胞没能获得足够的能量，你就会感到疲惫，想吃甜食，想摄入咖啡因，身体一直处于应激状态，释放大量压力荷尔蒙，使心率变异度大大降低，从而压力越来越大，自控力越来越差。好消息是，这些反应都是可逆的。只要补上一个好觉，前额皮质就会恢复如初。</li>
  </ul>
 </ul>
 <p data-pid="4NXkNL73">战略：</p>
 <ul>
  <ul>
   <li data-pid="8RvGqQ6X">减压方法（这个是答主个人的解压方法，仅供参考）：</li>
   <ul>
    <li data-pid="z_7RS3KH">不要把最难和最不想做的东西留到最后，会越拖压力越大！早点开始动手！</li>
    <li data-pid="ST5WdDLd">做家务</li>
    <li data-pid="vaiy3z93">精油+关灯+泡脚+大提琴音乐 = relax<br></li>
    <li data-pid="pgnZGg_r">冥想</li>
    <li data-pid="i3cAeBd2">健身</li>
    <li data-pid="w2UZTxEQ">保证良好睡眠</li>
    <li data-pid="FbJC2IXc">购物</li>
    <li data-pid="Klmxw87Q">散步</li>
   </ul>
   <li data-pid="bMg6yxrI">快速提高意志力的方法：将呼吸频率降低到每分钟4-6次，也就是每次呼吸用10-15秒的时间，这样训练几分钟后，你会感到平静，有控制感。放慢呼吸可以激活前额皮质，提高心率变异度。</li>
  </ul>
 </ul>
 <p data-pid="XWvAQX-2"><b> 第三章：累到无力抵抗：为什么自控力和肌肉一样有极限？</b></p>
 <p data-pid="Q9oVeTHc">理论：</p>
 <ul>
  <ul>
   <li data-pid="D2ySUOY2">低血糖能解释很多意志力时效的情况，因为自控是大脑活动中耗能最高的一项，为了保存能量，大脑不愿意给你充足的能量去抵抗诱惑、集中注意力、控制情绪。大脑的首要任务是获得更多能量，而不是保证你做出明智的决定，实现你长远的目标。</li>
   <li data-pid="EvLi9VIw">现代社会中出现的失控实际上是大脑战略性冒险本能的延续。</li>
   <li data-pid="jgOD2mA6">从长远来说，过度依赖糖分并不是自控的好方法，就像美国数百万2型糖尿病患者一样。</li>
   <li data-pid="uSrXkemz">意志力就像肌肉一样，要么使用，要么消失。</li>
  </ul>
 </ul>
 <p data-pid="NBF-KF7Y">战略：</p>
 <ul>
  <ul>
   <li data-pid="u_t2cKGY">如果你觉得自己没有时间和精力去处理“我想要”做的事，那就把它安排在你意志力最强的时候做。</li>
   <li data-pid="Pow0Sl_g">保证你的身体有足够的食物供应，这样能给你更持久的能量。</li>
   <ul>
    <li data-pid="q_m0M7pl">少吃多餐</li>
    <li data-pid="WNcA2Eeu">低GI食物</li>
   </ul>
   <li data-pid="xXHfExil">不要用代糖。因为甜味剂会刺激身体从血液中吸收葡萄糖，当你缺少能量没有自控力的时候，你的身体和大脑会想“血糖补充在哪呢？”，从而导致饥饿，暴饮暴食，和体重增加。</li>
  </ul>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <ul>
   <li data-pid="RbaRwbGG">训练“意志力肌肉”</li>
   <ul>
    <li data-pid="GpBzDrWt">增强“我不要”的力量：不驼背（这是da主从小养成的坏习惯）</li>
    <li data-pid="V6Vgl1dz">增强“我想要”的力量：每天冥想5分钟</li>
    <li data-pid="f168wPpE">目的：养成习惯，关注自己正在做的事情、选择更难的事情，通过练习，大脑开始习惯三思而后行</li>
   </ul>
   <li data-pid="JRX55fPa">觉得自己“疲惫”的没办法自控的时候，试着挑战一下自己，挺过第一波疲惫感。</li>
   <li data-pid="4AZL5Dki">挖掘“我想要”的力量，让自己恢复能量。</li>
   <ul>
    <li data-pid="WfQPsCG4">如果挑战成功，你会有什么回报？</li>
    <li data-pid="z1mwt39z">如果挑战成功，还有谁会获益？我身边的人会如何被影响？</li>
    <li data-pid="Yg_hpmKs">如果现在愿意做困难的事，那么一段时间后，这个挑战将会变得容易。如果取得进步，你的生活会是什么样子？现在的不适是不是值得？</li>
   </ul>
  </ul>
 </ul>
 <p data-pid="1iY7zvJJ"><b>第四章：容忍罪恶：为何善行之后会有恶行？</b></p>
 <p data-pid="5qdHTH71">理论：</p>
 <ul>
  <ul>
   <li data-pid="FXz8UEEw">『道德许可效应』：好的行为总是允许我们做一点坏事：</li>
   <ul>
    <li data-pid="4wOKgmE1">如果锻炼了就说自己很『好』，没去锻炼就说自己很『坏』，那么你很可能因为今天去锻炼了，那么你可能因为今天去锻炼了，明天就不去了。</li>
    <li data-pid="Pzrgy-2d">如果处理了一个重要项目就说自己很『好』，拖延着不去处理就说自己很『坏』，那么你可能因为早上取得了进步，下午就变得懒散了。</li>
   </ul>
   <li data-pid="XtGFW-Cn">我们容易人为，纵容自己就是对美德最好的奖励，我们忘记了自己真正的目标，向诱惑屈服了。</li>
   <li data-pid="z9l8GUYc">『目标释放』：当你取得进步的时候，大脑就停止了思维进程，而这个进程正是推动你追求长远目标的关键。然后，那个放纵自我的声音就会想起来，你就会转而关注那些还没有得到满足的目标。你曾努力克制的目标会变得更强大，诱惑也更难以抵挡。</li>
   <li data-pid="qlJLZ9BJ">列出 to do list 真的让人如释重负。我们把需要做的事当成了自己已经付出的努力，这会给我们错误的满足感。这很好的解释了，我们想到未来的选择时，就会很容易犯下大错。</li>
   <li data-pid="m3zg982o">道德许可就是一种身份危机。我们之所以会奖励自己的良好行为，是因为我们内心深处人为，真正的自己想做坏事。于是，每次自控都是一种惩罚，只有放纵自我才是奖励。</li>
  </ul>
 </ul>
 <p data-pid="m5tGVh4s">战略：</p>
 <ul>
  <ul>
   <li data-pid="2ywoAMAf">不应该想着“我做到了，好了，现在我可以做点我真正想做的事了！”。应该想着“我做这件事是因为我想要……”</li>
   <li data-pid="_W4hxoH2">记住我们为什么会拒绝诱惑。</li>
   <li data-pid="6hnip3Hd">把今天做的每个决定都看成是对今后每天的承诺。</li>
   <ul>
    <li data-pid="o5cLvCYk">“我想不想在一年里每天下午都吃一块糖？”</li>
    <li data-pid="IyunkgTD">“我是不是想承担永远拖延下去的恶果？”</li>
   </ul>
   <li data-pid="adRvdRmE">走出“道德许可”的陷阱，那个想要变好的自己才是真正的自己，想按核心价值观生活的自己。</li>
  </ul>
 </ul>
 <p data-pid="14r4pvIo"><b>第五章：大脑的弥天大谎：为什么我们误把渴望当幸福？</b></p>
 <p data-pid="KvaN-xOg">理论：</p>
 <ul>
  <ul>
   <li data-pid="eoPssxvs">奖励系统：当大脑发现获得奖励的机会时，它就释放出叫做多巴胺的神经递质，多巴胺会告诉大脑的其他部分要注意什么，怎么才能让贪婪的我们得手。当大量分泌的多巴胺劫持了你的注意力时，大脑只会想如何获得或者重复那个触发它的东西，这时我们就展现了自己最敢于冒险、最冲动、最失控的一面。</li>
   <li data-pid="rHqErqLm">多巴胺控制的是行动，而不是快乐。</li>
   <li data-pid="UxupKe0P">我们感受到的是期待，而不是快乐。</li>
   <li data-pid="k053zHjl">进化根本不关心你快乐与否，但它会利用对快乐的承诺，让我们不停地为生计奔忙。</li>
   <li data-pid="RR_FAmsH">微博微信、电子邮件、短信、电子游戏：都是自我刺激的现代版。</li>
  </ul>
 </ul>
 <p data-pid="Mr1FEOLO">战略：</p>
 <ul>
  <ul>
   <li data-pid="kHa0jUZk">找一些常常让你放纵自己的诱惑因素</li>
   <li data-pid="uP5AJlDZ">关注你放纵的过程，注意奖励承诺给你什么感觉？</li>
   <li data-pid="_KxTfUN3">而真正接受了诱惑之后是什么感觉？</li>
   <li data-pid="kTd6EX7L">和期望比起来，体验如何？</li>
   <li data-pid="_5qLC3Hh">奖励的承诺有没有消失？</li>
   <li data-pid="OvpCaD5b">什么时候你会感到满足？</li>
   <li data-pid="_oIO9Z78">如果有什么事让你觉得不愉快，所以总是拖延着不去做，你能不能把它和能让多巴胺神经元燃烧的事联系在一起，从而促使自己去做呢？</li>
   <li data-pid="Xp1EqZ-4">如果我们想拥有自控力，就需要区分让我们生活有意义的真实奖励，和让我们分散精力、上瘾的虚假奖励。</li>
  </ul>
 </ul>
 <p data-pid="OYy9N8yn"><b>第六章：“那又如何”：情绪低落为何会使人屈服于诱惑？</b></p>
 <p data-pid="YvduRTlP">理论：</p>
 <ul>
  <ul>
   <li data-pid="Kn9vqcW_">压力（愤怒、悲伤、自我怀疑、焦虑）➡️ 情绪低落 ➡️ 大脑想维持人的心情 ➡️ 大脑选择奖励的承诺（上一章）➡️ 向诱惑屈服 ➡️ 我们缓解压力的方法反而会让我们更有压力</li>
   <li data-pid="M5uafYh-">“那又如何”效应：破罐子破摔</li>
   <ul>
    <li data-pid="eN5dvLff">拖延：<br></li>
    <ul>
     <li data-pid="6E2azcU9">拖到了deadline前几天感觉自己已经没办法完美达成目标了，罪恶</li>
     <li data-pid="tRYHlYDs">越拖越久，压力越来越大</li>
    </ul>
    <li data-pid="Evri3Pre">任何挫折都会引起这样的恶性循环</li>
   </ul>
   <li data-pid="is1CxojZ">自我批评会降低积极性和自控力，它不仅耗尽了“我要做”的力量，还耗尽了“我想要”的力量。自我谅解和自我同情则会提升积极性和自控力，比如，在压力和挫折面前支持自己、对自己好一些。</li>
   <li data-pid="BgcQuBIo">改变的承诺：发誓改变会让我们充满希望，在什么都没做之前，就感觉良好了。它只能给我们一时的快乐，但接下真正做出改变时面临的挑战却会给你当头一棒，让你感到失落。</li>
  </ul>
 </ul>
 <p data-pid="nIp46gO2">战略：</p>
 <ul>
  <ul>
   <li data-pid="tllVH_X_">远离那些会让你产生恐惧的电视新闻、访谈节目、杂志或网页。</li>
   <li data-pid="NyMtL7LX">尝试更有效的解压方法，增加大脑中改善情绪的化学物质，如血清素、gama-氨基丁酸和让人感觉良好的催产素，它们不像释放多巴胺的物质那样让人兴奋，所以我们往往低估了它们的作用。</li>
   <li data-pid="YC1OfsNd">每个人都会犯错误，都会遇到挫折。用以下方法，让我们面对失败时同情自己：</li>
   <ul>
    <li data-pid="0YTp2lK5">你感觉如何？你对自己说了什么？是不是自责？</li>
    <li data-pid="g-3HUt8G">你只是个凡人，每个人都有失去自控的时候，这是人性的一部分，挫折并不意味着你本身有问题。想想其他你尊敬/关心的人也经历过同样的抗争和挫折吗？</li>
    <li data-pid="YnTF6AQ-">如果你的好朋友经历了同样的挫折，你会怎么安慰他？你会如何鼓励他继续追求自己的目标？</li>
   </ul>
   <li data-pid="iXxIXG1-">做个乐观的悲观主义者：这种方法预见失败其实是一种自我同情的方式，不是自我怀疑。</li>
   <ul>
    <li data-pid="HCpcFbaO">想一想自己的意志力挑战，扪心自问：</li>
    <ul>
     <li data-pid="w0HvXQjH">我什么时候最可能受到诱惑并放弃抗争？</li>
     <li data-pid="3t_fmtzD">什么东西最可能分散我的注意力？</li>
     <li data-pid="Kj2T0qsa">当我允许自己拖延/暴饮暴食的时候，我会怎么劝自己？</li>
     <li data-pid="RCQ3dOO7">你需要采取行动回忆自己的动力吗？</li>
     <li data-pid="V7N8m-w7">需要远离诱惑吗？</li>
     <li data-pid="xSjRzMFp">需要找朋友帮忙吗？</li>
     <li data-pid="aZDu4mbQ">需要用你学过的其他意志力策略吗？</li>
    </ul>
   </ul>
  </ul>
 </ul>
 <p data-pid="-1KqTJB4"><b>第七章：出售未来：及时享乐的经济学</b></p>
 <p data-pid="dyt8dHVC">理论：</p>
 <ul>
  <ul>
   <li data-pid="TXuWak2D">“延时折扣”：等待奖励的时间越长，奖励对你来说价值越低。很小的延迟就能大幅降低感知到的价值。</li>
   <li data-pid="WWjEk_-Z">“延时折扣”解释了为什么我们宁愿放弃未来的幸福，也要选择即刻的快感。为了明天做准备，或许还有可能。但是为了十几年以后做准备，那可就太久了。</li>
   <li data-pid="lTpDtRFL">抽象、不能直接看到的奖励也会让奖励系统的刺激作用减少。</li>
  </ul>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <ul>
   <li data-pid="9UT38x9x">“未来的自己”：人们会把未来的自己理想化，希望未来的自己可以做到现在的自己做不了的事，让他们承担现在的自己犯下的错误。</li>
  </ul>
 </ul>
 <p data-pid="W5_IPl73">战略：</p>
 <ul>
  <ul>
   <li data-pid="72auFnxK">降低折扣率：</li>
   <ul>
    <li data-pid="Iqcybtsi">当你受到诱惑时，想象一下，这个选择就意味着你为了即时的满足放弃了更好的长期奖励</li>
   </ul>
  </ul>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <ul>
   <ul>
    <li data-pid="--eVJrEI">想象你已经得到了长期的奖励。想象你正在享受自控的成果</li>
   </ul>
  </ul>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <ul>
   <ul>
    <li data-pid="z66IKOco">扪心自问：你愿意放弃它，来换取正在诱惑你的短暂快感吗？</li>
   </ul>
   <li data-pid="AAAemrk_">10分钟：在10分钟期间你一定要时刻想象着长远的奖励，以此抵制诱惑。10分钟就能在很大程度上改变大脑处理奖励的方式。</li>
   <ul>
    <li data-pid="NL5IUB3K">『我不要』：坚持10分钟等待，然后就可以得到它。</li>
    <li data-pid="VYBCK4SJ">『我要做』：坚持做10分钟，然后就可以放弃</li>
   </ul>
   <li data-pid="0ojf5lF0">对诱惑创造一点距离，让拒绝变得更容易。</li>
   <li data-pid="8MvLLDak">想象未来的自己可以增强现在的意志力，未来的自己会对我们现在的付出感激不尽。</li>
   <ul>
    <li data-pid="WAbzS0Eo">想象希望成为的自己</li>
    <li data-pid="_6VgNJjv">想象害怕成为的自己</li>
   </ul>
  </ul>
 </ul>
 <p data-pid="5eAkmjtv"><b>第八章：传染：为什么意志力会传染？</b></p>
 <p data-pid="TLDXF5hX">理论</p>
 <ul>
  <ul>
   <li data-pid="DABrEtHm">镜像意志力传染</li>
   <ul>
    <li data-pid="jO9arPVx">第一种：无意识模仿。当镜像神经元探测到其他人的行动时，它会让你的身体也准备做同样的动作</li>
    <li data-pid="LsPezdaA">第二种：传染情绪。自己的镜像神经元会对别人的情绪产生反应。别人的欲望会引发我们的欲望。</li>
   </ul>
   <li data-pid="r3_yF-3s">“社会认同”：当群体里其他人都在做某件事时，我们容易人为这件事是应该做的聪明事。</li>
   <li data-pid="9FUUl-H4">”自豪“的力量：自豪和羞愧是属于大脑皮层的情绪区，能更迅速更直接的印象我们的选择。自豪可以引起好的情绪，让自己安然度过诱惑；而羞愧则会让人感觉糟糕，让人放弃抵抗破罐破摔，而非自我控制。</li>
  </ul>
 </ul>
 <p data-pid="RGCa9JBq">战略</p>
 <ul>
  <ul>
   <li data-pid="FGiNhjg_">防止传染坏的：少看不好的影响，远离给你不好影响的人，找到新的和你有同样渴望的『群体』并加入进去，可以是支援小组、俱乐部、网络社区、杂志……</li>
   <li data-pid="6RlFyDBK">利用传染好的：当你需要多一点意志力的时候，想想你的榜样。问问自己：那个人会怎么做？</li>
   <li data-pid="SuhNf9kM">利用社会认同：只要我们相信社会规范就是正确的事，社会认同就会增强我们的自制力。</li>
   <li data-pid="l7RmdxxF">利用周围的人：在考虑作何选择时，可以想象自己是别人评估的对象，为自控提供强大的精神支持。</li>
   <li data-pid="nEXEaY4S">利用自豪的力量：让自己感觉别人都在检视自己，让我们有机会向别人报告自己的成功。</li>
  </ul>
 </ul>
 <p data-pid="Yu89lBOk"><b>第九章：别读这章：“我不要”力量的局限性</b></p>
 <p data-pid="GBnMrOj0">理论：</p>
 <ul>
  <ul>
   <li data-pid="A23MWfgH">『讽刺性反弹』：压抑人的本能时，越是不让我们想一件事，我们就越去想它。</li>
   <li data-pid="blMPuew7">节食：限制卡路里减肥法不仅对减轻体重或身体健康没什么好处，而且被越来越多的证据证明有害身心。</li>
   <ul>
    <li data-pid="iOMT12bz">禁食某种食物会增加人对这种食物的欲望</li>
    <li data-pid="lVjV5mEa">循环反复式的节食会使血压和胆固醇含量上升，抑制人体免疫系统，增加心脏病、中锋、糖尿病等风险。</li>
   </ul>
  </ul>
 </ul>
 <p data-pid="Mc7GHro_">战略：</p>
 <ul>
  <ul>
   <li data-pid="rO3uDzNM">对抗『讽刺性反弹』：放弃自控</li>
   <ul>
    <li data-pid="1a_RB2Hu">承认自己脑海中的欲望，去想自己所想，追随自己的感觉（你不必相信它是真的，不要觉得必须采取行动）</li>
    <ul>
     <li data-pid="k2aeboSv">类似佛教中“无念”，有感知，无反应</li>
    </ul>
    <li data-pid="UJUgw5Jt">接受这种想法，提醒自己『白熊现象』和『反弹』理论</li>
    <li data-pid="DLzWflgk">把注意力转移到呼吸上，有时候，这些翻新的想法会打断你对呼吸的关注，那就把这些想法想象成飘过脑海和身体的浮云，想象这些浮云不断消散或者飘过，把呼吸想象成一阵风，毫不费力地将浮云吹散，吹走。</li>
    <li data-pid="tmMAbCNX">无需让这些想法离开，只需保持呼吸的感觉。这些想法总是来来去去，你无法控制会出现什么内容，但你不必接受它的内容，这就是思维运作的方式，它不一定意味着什么，但你可以选择自己相信什么，选择自己要做什么。</li>
    <li data-pid="0u6_EEFU">记住你的目标，提醒自己『我想要』的承诺。</li>
   </ul>
  </ul>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <ul>
   <ul>
    <li data-pid="Z6v-ddvj">无论你是否满足了冲动，它最终都会消失。</li>
   </ul>
   <li data-pid="kavLe8Nd">把『我不要』变成『我想要』的挑战：关注自己想做什么，而不是不想做什么，避免反弹效应。</li>
   <ul>
    <li data-pid="D-f4Xgib">比如：把『不要迟到』重新定义为『做第一个到的人』</li>
   </ul>
  </ul>
 </ul>
 <p data-pid="MdF1y191"><b>结语</b></p>
 <p data-pid="E0cG2Jln">理论：</p>
 <ul>
  <ul>
   <li data-pid="ElXBSzM0">真正的自控力的秘诀：集中注意力。</li>
  </ul>
 </ul>
 <p data-pid="H7HPt4ny">战略：</p>
 <ul>
  <ul>
   <li data-pid="88xoHO4R">『自我意识』：当注意力分散的时候，或者即将向诱惑投降的时候，你需要静下心来，弄清楚自己的欲望。记住自己真正想要的是什么，什么才能让你真正的快乐。</li>
  </ul>
