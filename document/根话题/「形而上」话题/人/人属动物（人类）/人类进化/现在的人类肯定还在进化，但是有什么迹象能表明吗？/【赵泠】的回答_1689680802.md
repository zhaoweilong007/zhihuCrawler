# 现在的人类肯定还在进化，但是有什么迹象能表明吗？
- 点赞数：6265
- 更新时间：2021年01月21日14时13分51秒
- 回答url：https://www.zhihu.com/question/440336198/answer/1689680802
<body>
 <p data-pid="BR_ndWTu">工业革命以来<sup data-text="工业革命以来，人类对环境造成的影响明显增加，气候变化和生活方式的改变都可以带来性状的变化（其中一部分可以表观遗传），原子时代以来释放到环境中的放射性物质、化学药物和微塑料等等都可以影响演化，而且稳定的环境和食物来源允许人类的各种单核苷酸突变在几代人间迅速扩散。" data-url="" data-draft-node="inline" data-draft-type="reference" data-numero="1">[1]</sup>，人类生物学身体的演化趋势有：</p>
 <ul>
  <li data-pid="xUPdmKOx">人类站立时的<b>平均重心位置在向后移动</b>；</li>
  <li data-pid="zzMuZQRR">人类数万年来<b>脑容量略微减少</b>的趋势还在继续；</li>
  <li data-pid="8rp7X6W8">冰岛等地的人群<b>与受教育程度相关的等位基因突变减少</b><sup data-text="https://doi.org/10.1073/pnas.1612113114" data-url="" data-draft-node="inline" data-draft-type="reference" data-numero="2">[2]</sup>；</li>
  <li data-pid="CKTnxZzk">对阿尔茨海默症、哮喘、多种过敏、高血糖、高血压、高血脂、心血管疾病等易感的基因突变在一些人群中所占的比例<b>徐徐下降</b><sup data-text="https://www.ncbi.nlm.nih.gov/pmc/articles/PMC2868295" data-url="" data-draft-node="inline" data-draft-type="reference" data-numero="3">[3]</sup>；</li>
  <li data-pid="XwhJ3BA0">对骨质疏松和另一些疾病易感的基因突变在一些人群中所占的比例<b>有所增加</b>；</li>
  <li data-pid="OazPY8oy">一些女性人群的<b>更年期有所推迟、月经初潮年龄和首次生育年龄有所提前</b>，女性适于首次生育的年龄是一项显著可遗传的性状<sup data-text="https://doi.org/10.1073%2Fpnas.1104210108" data-url="" data-draft-node="inline" data-draft-type="reference" data-numero="4">[4]</sup>，一项研究显示女性更年期每200年可能推迟1岁；</li>
  <li data-pid="ROpQvt3G">剖腹产在许多地区的滥用引起当地新生儿头围的增加，且当地产妇的骨盆宽度并未增加；</li>
  <li data-pid="S_fI04t_">尼德兰等地的人群<b>平均身高</b>在过去两百年里急剧增加；</li>
  <li data-pid="RNJHQqHV">人类的面部继续了之前数千年间的缩短和狭窄化趋势，使得<b>阻生智齿</b>的发生率比数百年前有所升高，但同时<b>缺失智齿</b>的基因在人群中的占比延续了数千年来缓慢增加的趋势；</li>
  <li data-pid="Lc-HLw_H">2017年，一项研究发现18世纪初移居到美国的阿米什人群体里出现了<b>对抗糖尿病并帮助维持端粒</b>的基因突变，目前已经覆盖了他们人数的24%，携带该突变的人平均寿命延长10年；</li>
  <li data-pid="dhBd0Wim">南非人群中<b>对抗HIV</b>的基因突变在我们眼前出现并扩张。如果HIV不能通过更高的基础突变率盖过人类的对策，数百年后南非人可能对HIV免疫；</li>
  <li data-pid="V9_B0zOh"><b>平均体温下降</b>——美国一项研究发现，从十九世纪六十年代以来，美国人的平均体温每10年下降0.03摄氏度；</li>
  <li data-pid="iHXZjbqY">“遗存正中动脉”之类<b>解剖变异</b>也是人类演化的表现，尽管不能据此说演化加速。</li>
 </ul>
 <p data-pid="XsOoHCUv">在不同的现代人群中，解剖显示的遗存正中动脉发生率在1.1%~35%不等。</p>
 <ul>
  <li data-pid="TwN96BTQ">2012年一项印度的研究解剖了60条胳膊，显示遗存正中动脉在当地的发生率是6.6%，</li>
  <li data-pid="fom1H0EN">2019年波兰的一项研究<sup data-text="https://dx.doi.org/10.12659%2FMSM.912269" data-url="" data-draft-node="inline" data-draft-type="reference" data-numero="5">[5]</sup>解剖了125条胳膊，显示遗存正中动脉在当地的发生率是4%，</li>
  <li data-pid="A-xOTjFE">以上两项研究中，双臂都有遗存正中动脉现象的案例极少。</li>

 </ul>

 <p data-pid="mMieCM1T">此外，2020年，研究人员通过正电子发射计算机断层扫描发现咽鼓管圆枕附近有一组密集排列的唾液腺<sup data-text="https://doi.org/10.1016/j.radonc.2020.09.034" data-url="" data-draft-node="inline" data-draft-type="reference" data-numero="6">[6]</sup>，这玩意到底是本来就有还是最近多少年里新出现的都不知道。</p>
 <p data-pid="tmt7Mb04">在当前条件下，可遗传突变的最主要来源是精子的突变。随着年龄的增长，男性的精子突变越来越多。这意味着<b>男性平均寿命的增加、平均生育年龄的推迟可以增加人类的基因突变</b>——2012年的一项研究显示，男性每延迟生育1年，平均可以多产生2个单核苷酸突变。</p>
