 <p><strong>一<span><span>、</span></span>我走过的弯路<span><span>，</span></span>也可能是你会出错的地方</strong></p>
 <p>2018 年 4<strong> 月<span><span>，</span></span>当我决定认</strong>真减脂<span><span>，</span></span>搜索减脂饮食<span><span>，</span></span>一帮人告<strong>诉我要低 </strong>GI 饮食<span><span>，</span></span>米饭的 GI 值高<span><span>，</span></span>会阻挡我的减脂大业<span><span>；</span></span>另<strong>一帮人告诉我要用粗粮甚至</strong>是魔芋代替主食<span><span>，</span></span>因为这样热量会非常低……</p>
 <p>被<span><span>「</span></span>干货<span><span>」</span></span>塞满大脑的我立马关掉了网页<span><span>，</span></span>下单了电蒸锅<span><span>。</span></span></p>
 <p><strong>从此紫薯<span><span>、</span></span>土豆<span><span>、</span></span>玉米等粗粮成了我餐盘里的新宠</strong><span><span>，</span></span>为了色彩鲜艳有食欲<span><span>，</span></span>每一餐中我还增加了各种水果<span><span>。</span></span></p>
 <p>如此清淡的饮食让我<strong>以肉眼可见的速度瘦下好几斤</strong><span><span>，</span></span>在指导了多位同事的饮食后<span><span>，</span></span>我为自己<span><span>「</span></span>健康饮食<span><span>」</span></span>的成果感到欣喜<span><span>。</span></span></p>
 <p>在坚持<span><span>「</span></span>健康饮食<span><span>」</span></span>的第二个月<span><span>，</span></span>我有了马甲线<strong>雏形</strong><span><span>，</span></span>一度以为自己是天选之子……</p>
 <p>但很快<span><span>，</span></span>追求<span><span>「</span></span>健康饮食<span><span>」</span></span>的<strong>我开始饱受胃病的折磨<span><span>，</span></span></strong>粗粮和魔芋中含有大量不能被人体完全消化的膳食纤维<span><span>，</span></span>一日三餐都用粗粮当主食终于使我的肠胃受损<span><span>。</span></span></p>
 <p>作为一个铁打的湖北妹子<span><span>，</span></span>不能吃辣也就算了<span><span>！</span></span></p>
 <p>我竟然<strong>边走路边打嗝<span><span>？</span></span></strong></p>
 <p>肚子每天都胀鼓鼓的<span><span>，</span></span>吃了几口食物<span><span>，</span></span>就迫不及待<strong>解开拉链</strong>的感受<span><span>，</span></span>你能体会吗<span><span>？</span></span></p>
 <p><strong>衣柜里的高腰裤再也不敢穿</strong>了<span><span>，</span></span>马甲线自然也变成了<strong>缘分到了才会有</strong>的东西<span><span>。</span></span></p>
 <p>后来我花了一段时间调理<span><span>，</span></span>已经改善很多了<span><span>，</span></span>过程中<strong>意外地发现自己也能够长期拥有马甲线</strong>了<span><span>。</span></span></p>
 <p>我们知道减脂需要<strong>摄入<span><span>，</span></span>身体的能量来源有近 50% 来自碳水化合物<span><span>，</span></span>而主食就是碳水化合物最大的来源<span><span>，</span></span>长期用粗粮代替所有主食<span><span>（</span></span>粗粮的热量往往很低<span><span>）</span></span><span><span>，</span></span>我们的日常能量摄入就再也满足不了身体的需要了<span><span>，</span></span>这种情况下体重是一定会减轻的<span><span>，</span></span>但减轻的都是水分和肌肉<span><span>，</span></span>这也说明了为什么我明明吃得很少腹部还有脂肪囤积<span><span>。</span></span></strong></p>
 <p>说了这么多<span><span>，</span></span>是真心想告诉迈出减脂第一步的朋友们<span><span>，</span></span><span><span>「</span></span>摄入<span>」</span>确实没什么毛病<span><span>，</span></span>但<strong>只在意热量<span><span>，</span></span>而忽视了营养密度<span><span>，</span></span>会让我们陷入一种极端</strong><span><span>，</span></span>就算瘦下来也很<strong>容易反弹</strong><span><span>。</span></span></p>
 <p><strong>二<span><span>、</span></span>减脂小白容易陷入的误区</strong></p>
 <p><strong>误区一<span><span>：</span></span>只吃复合碳水<span><span>，</span></span>不吃任何简单碳水</strong></p>
 <p>不知道有多少人刚迈出减脂第一步就是停止吃米饭<span><span>，</span></span>一下子就完成了从主食到全粗粮的转变<span><span>，</span></span>袁隆平爷爷都哭了……</p>
 <p><strong><span><span>（</span></span>1<span><span>）</span></span> GI 值<span><span>、</span></span>胰岛素与肥胖的关系</strong></p>
 <p>不同的食物对血糖的影响不一<span><span>，</span></span>我们常用 GI 值来判定食物对血糖的升高程度<span><span>。</span></span></p>
 <p><strong>高 GI</strong> 值<span><span>（</span></span>GI&gt;75<span><span>）</span></span>的碳水被称<strong>为 简单</strong>碳水<span><span>，</span></span>容易被人体消化吸收<span><span>，</span></span>对血糖的影响大<span><span>，</span></span>容易囤积脂肪<em><span><span>。</span></span>如水果<span><span>、</span></span>果汁<span><span>、</span></span>饼干<span><span>、</span></span>蛋糕<span><span>、</span></span>蜂蜜<span><span>、</span></span>白米饭<span><span>、</span></span>精面粉制成的面条<span><span>、</span></span>馒头<span><span>、</span></span>面包</em>等<span><span>。</span></span></p>
 <p><strong>为使血糖浓度降低</strong><span><span>，</span></span><strong>胰岛素</strong>就要发挥作用<span><span>，</span></span>促进全身组织对糖的摄取<span><span>、</span></span>储存和利用<span><span>，</span></span>将吃进体内的热量转化为脂肪<span><span>，</span></span>所以长期摄入高 GI 值的食物会使人发胖<span><span>。</span></span></p>

 <p><strong>低 GI</strong> 值<span><span>（</span></span>GI≤55<span><span>）</span></span>的碳水被称<strong>为 复合</strong>碳水 <span><span>，</span></span>含有大量不易被人体消化的膳食纤维<span><span>，</span></span>延缓胃的排空时间<span><span>，</span></span>减缓血糖的上升<span><span>，</span></span>又能增加饱腹感<span><span>，</span></span>得到很多减脂人群的喜爱<span><span>。</span></span><em> 如糙米饭<span><span>、</span></span>燕麦<span><span>、</span></span>红薯<span><span>、</span></span>薏米<span><span>、</span></span>荞麦<span><span>、</span></span>玉米<span><span>、</span></span>藜麦<span><span>、</span></span>乳制品<span><span>、</span></span>荞麦<span><span>、</span></span>豆类等</em>等<span><span>。</span></span></p>
 <p><strong>我们不难得出一个结论<span><span>：</span></span>只吃复合碳水<span><span>，</span></span>不吃任何简单碳水就能瘦<span><span>，</span></span></strong>这个结论并没有错<span><span>，</span></span>但显然是存在健康隐患的<span><span>。</span></span></p>

 <p><strong><span><span>（</span></span>2<span><span>）</span></span></strong> <strong>长期以粗粮当主食的危害</strong></p>
 <ul>
  <li><strong>导致胀气</strong></li>
 </ul>
 <p>粗粮中含有较多难以被人体消化的膳食纤维<span><span>，</span></span><strong>大量摄入膳食纤维容易导致胃胀气<span><span>。</span></span></strong></p>
 <ul>
  <li><strong>导致胃酸返流</strong></li>
 </ul>
 <p>粗粮食用过多会明显延缓胃排空<span><span>，</span></span>可能导致食物积存<span><span>，</span></span>这些食物就可能带着胃里的胃酸<span><span>，</span></span>反流到食管里<span><span>，</span></span>损坏食管黏膜<span><span>。</span></span></p>
 <p>我受胃病困扰的那段时间就经常<strong>胃酸返流</strong><span><span>，</span></span>这是一种非常难受的体验<span><span>。</span></span><strong>导致营养不良</strong></p>
 <p>长期过量食用粗粮会<strong>影响人体对蛋白质<span><span>、</span></span>无机盐以及一些微量元素的吸收</strong><span><span>，</span></span>导致营养不良<span><span>，</span></span>成为一个<span><span>「</span></span>瘦胖子<span><span>」</span></span><span><span>。</span></span></p>
 <ul>
  <li><strong>导致暴饮暴食</strong></li>
 </ul>
 <p>长期摄入大量的膳食纤维会使人体<strong>失去对饥饱的感知</strong><span><span>，</span></span>最终克制不住食欲<span><span>，</span></span>体重也随之反弹<span><span>。</span></span></p>
 <p>长期不吃精细主食的我<span><span>，</span></span><strong>摄入不能满足身体所需<span><span>，</span></span>身体需要在别处获得能量</strong><span><span>，</span></span>于是食欲变得不可控制<span><span>，</span></span>每个周末我都会将自己关在房间<span><span>，</span></span>吃掉大量的零食<span><span>。</span></span></p>
 <p><span><span>（</span></span><strong>3<span><span>）</span></span></strong> <strong>主食的正确打开方式</strong></p>
 <ul>
  <li><strong>粗细结合<span><span>，</span></span>循序渐进</strong></li>
 </ul>
 <p>将平时饭量的 50% 替换成糙米<span><span>，</span></span><strong>或是替换部分</strong>主食为薯类<span><span>、</span></span>豆类等粗粮<span><span>，</span></span>一定要遵循循序渐进的原则<span><span>！</span></span></p>
 <ul>
  <li><strong>早午餐<span><span>，</span></span>吃复合碳水</strong></li>
 </ul>
 <p>复合碳水消化过程复杂且缓慢<span><span>，</span></span>为人体持续供能<strong>高达 3 </strong>小时<span><span>，</span></span>这类<strong>碳水不易被人体转化成</strong>脂肪<span><span>，</span></span>适合作为工作日的早午餐<span><span>。</span></span></p>
 <ul>
  <li><strong>运动前后<span><span>，</span></span>吃简单碳水</strong></li>
 </ul>
 <p>健美运动员会在运动前喝一杯低聚糖饮料<span><span>，</span></span><strong>运动前补糖可使肌糖原达到最大填充</strong><span><span>，</span></span>延长运动至疲劳的时间<span><span>。</span></span></p>
 <p><strong>运动后<span><span>，</span></span>简单碳水能快速补充能量<span><span>（</span></span>糖原<span><span>）</span></span></strong><span><span>，</span></span>帮助肌肉组织尽快修复<span><span>，</span></span>且不易转化成脂肪<span><span>。</span></span></p>
 <ul>
  <li><strong>肠胃不适时<span><span>，</span></span>吃简单碳水</strong></li>
 </ul>
 <p>粗粮再好<span><span>，</span></span>在肠胃不适时<span><span>，</span></span>也要听从身体发出的信号<span><span>，</span></span>吃好消化的简单碳水<span><span>。</span></span></p>
