# 减肥有什么冷知识？
- 点赞数：5604
- 更新时间：2020年08月21日18时51分37秒
- 回答url：https://www.zhihu.com/question/334673162/answer/1415327024
<body>
 <ol>
  <li data-pid="iGg5ufuO"><b>你以为油脂会让人发胖，但是不吃油人会死，而且减肥慢</b></li>
  <li data-pid="Sm7H8LDu"><b> 光吃水果减肥，却不知越吃越肥，内脏脂肪越来越多，肚子始终不消失，水果一天最好控制在500g左右(具体因人而异)</b></li>
  <li data-pid="_iA2NG-L"><b>减肥不是就吃鸡胸肉，西兰花，减肥也可以吃你任何想吃的，你只需要每天有一点点热量缺口就好</b></li>
  <li data-pid="W6pmozSs"><b>不是我运动就能减肥，最终还是取决于你的热量缺口</b></li>
  <li data-pid="lSSIYcmG"><b>节食减肥越来越胖，因为基础代谢下降</b></li>
  <li data-pid="0bFLMDO6"><b>肌肉增长给你提升基础代谢有限而且很少，比如一个很壮肌肉男他可能基础代谢提升就有半包牛奶热量(或者提升10%基础代谢）</b></li>
  <li data-pid="4pj6KAtp"><b>有局部减脂取决你的基因，比如有些人的基因一瘦先瘦胸，有些人就瘦腰</b></li>
  <li data-pid="L6EJsgnK"><b>压力会让你变胖，而且是腹部肥胖，所以持久的腹部肥胖考虑下是否压力太大(这种持久是指你已经尝试过很多减脂方法而且热量缺口有）</b></li>
  <li data-pid="M0rkwOWs"><b>洗冷水澡可以帮助减肥，因为提升了你的基础代谢</b></li>
  <li data-pid="ANdMAhfm"><b>喝咖啡帮助减肥因人而异，有些人可以有些人喝完只会心慌意乱，而且咖啡随着时间你体内产生耐受性</b></li>
  <li data-pid="NbjiDCu2"><b>你以为的你吃的少，其实巨多无比</b></li>
  <li data-pid="BQmPFwle"><b>很多时候体重大部分是基因决定的，就像同样的热量下，有些人吃了胖有些人瘦</b></li>
  <li data-pid="YhFcTLtG"><b>减脂不单纯的是热量的计算，还有一些肠道菌群以及体内激素的影响</b></li>
  <li data-pid="hgA5OAjk"><b>你觉得小番茄减肥，但是同样一碗小番茄和西瓜的热量差不多，所以为什么不直接吃西瓜，而且小番茄凝集素，吃多了对人体不好</b></li>
  <li data-pid="jf2ewIwR"><b>你觉得水煮鸡胸肉是健康的减脂的，但是鸡胸肉加点黄瓜胡萝卜一点油就是宫保鸡丁，热量可能多个50-100大卡，你觉得那个吃的爽</b></li>
  <li data-pid="QnUYRrlv"><b>女生大腿肌肉脂肪多，也就是肌肉中含有的甘油三酯多，理论上应该多做一些高强度训练</b></li>
  <li data-pid="-66IUTWo"><b>总是想着瘦小腿，岂不知除了体态调整，很多就只是天生了(不是后天肥胖导致的）</b></li>
  <li data-pid="ckAjlf6n"><b>凡是你觉得减肥有效果的补剂大部分是让你兴奋增加了你的基础代谢，但是效果也就一点点</b></li>
  <li data-pid="SH-Fd4T-"><b>当你想着要1个月瘦10斤或是想吃的东西以后都不能吃了，那么你越你的目的越来越远了</b></li>
  <li data-pid="zCn16Jae"><b>真正保持一个好的身材，或者减脂成功，也需要很多心理战术，比如你如何打发无聊的时光，你如何应对你自己的压力，你如何看待自己，到底是你没吃够还是你麻痹了自己对食物的感觉</b></li>
  <li data-pid="8WCrBDDU"><b>我们不断的想着快速减脂，却连基本的饱腹感以及饥饿感都分辨不出来，你想要减脂多多少少前期会经过一些饥饿感</b></li>
  <li data-pid="_ZKB9BLp"><b>大部分人觉得运动30min'以上脂肪才开始燃烧，却不知道你休息时候60%的能量来源你的脂肪，当你同样时间非常低强度训练可能和你高强度训练脂肪消耗的量一样，但是高强度总体消耗的热量多，更利于你的减脂</b></li>
  <li data-pid="cGlkC2Ry"><b>减脂不在乎你的脂肪燃烧，只在乎热量缺口，不管你的热量缺口来源哪里，比如我吃肯德基每天控制在1600大卡，但是我消耗2000，我一样处于减脂状态</b></li>
  <li data-pid="H564kjPG"><b>当你真的吃的很少，训练又很多还没有瘦下去，不是你的基础代谢损伤，就是你的内分泌出问题，这时候你需要去医院</b></li>
  <li data-pid="nIT2hM2k"><b>以为腹部训练就可以减肚子，其实可能减不了肥，最主要的是你需要制造热量缺口</b></li>
  <li data-pid="91i5DHwj"><b>有时候正确的呼吸就可以让你马上拥有马甲线</b></li>
  <li data-pid="zzzwwVCW"><b>女生不是瘦才好看，可能有时候体态比瘦更重要，这是很多人都会忽略地方</b></li>
  <li data-pid="b2ymnyl0"><b>大基数减肥越快皮肤越松</b></li>
  <li data-pid="PJfYThbf"><b>当你胖的你的脂肪细胞数量都开始增长时候，你会很容易又会变胖，特别是小孩子小时候很胖长大很难减</b></li>
  <li data-pid="yxUN3KdA"><b>刚开始运动一段时间你觉得你的双腿粗了或者你身体肿了，只是你体内的水分和糖原增加</b></li>
  <li data-pid="JVChFhxO"><b>同样热量碳水比例越大你越水肿</b></li>
  <li data-pid="WfG1eKWI"><b>短时间内的体重变化都是水分变化，所以你会感觉一段时间的水肿</b></li>
  <li data-pid="EhVurfaS"><b>减肥需要多喝水40ml*体重的水都没喝到太影响你的减肥了</b></li>
  <li data-pid="505jneFc"><b>还有就是保持正常的作息，饮食睡觉都比较规律有助于你的减肥</b></li>
  <li data-pid="OrZ72lJC"><b>饭前喝一杯热水可以减少你的吸收，帮助减肥</b></li>
  <li data-pid="jpjob3BL"><b>饭前吃水果是开胃的，会使得你食欲上升</b></li>
  <li data-pid="DvNV1857"><b>一般高GI食物容易引起回返行低血糖，比如你吃蛋糕是吃不饱的，越吃越饿，过段时间又觉得撑</b></li>
  <li data-pid="wjqYxiY9"><b>脂肪是最难吸收消化的，需要很长时间，而且饱腹感强，相比碳水，高脂更容易减肥(比如生酮饮食）</b></li>
  <li data-pid="ahyvrdpD"><b>蛋白质的食物热效应最高，所以高蛋白饮食减脂效果较好，但是肝肾有问题慎用</b></li>
  <li data-pid="seFfImUw"><b>每日蛋白质摄入久坐不动0.8-1.2g/kg，有运动1.2-1.6g/kg，饮食蛋白质足够就不需要蛋白粉了(因人而异）</b></li>
  <li data-pid="_tiqMzqQ"><b>抗性淀粉冷却后不会被人体吸收，有利于减脂，含直链淀粉支多的大米</b></li>
  <li data-pid="IrQrkgin"><b>还有关于高脂，高蛋白，低碳，轻断食我觉得各有利弊，效果因人而异</b></li>
 </ol>
