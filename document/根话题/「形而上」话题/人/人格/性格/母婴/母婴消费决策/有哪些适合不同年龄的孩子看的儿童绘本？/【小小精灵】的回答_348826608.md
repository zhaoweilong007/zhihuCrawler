# 有哪些适合不同年龄的孩子看的儿童绘本？
- 点赞数：5386
- 更新时间：2019年05月13日13时08分01秒
- 回答url：https://www.zhihu.com/question/25815002/answer/348826608
<body>
 <p data-pid="l6rzzWt1">我呢，当了十几年的幼儿园老师，同时也是两个孩子的妈妈，这些年看过的绘本少说也有五六百本吧，这几天不太忙，就花了点时间把之前看过的，我觉得质量较好的绘本挑选出来，整理了一个清单，分享给大家，供各位爸爸妈妈参考，希望对你们有帮助。</p>
 <p data-pid="p-sCMhCR"><b>不想看绘本介绍的，可以直接拉到文末，文末列了清单。</b></p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="f1vddACM"><b>国内 </b></p>
 <ul>
  <li data-pid="U1bIwwdV"><b>百岁童谣 </b>山曼 编著 明天出版社</li>
 </ul>
 <p data-pid="IDUA-qfx">丛书共五册，包括描述幼儿游戏的《花巴掌》、刻画年节喜庆活动的《正月正》、汇集民间摇篮曲的《外婆桥》，以及表现田园风光、农家生活的《小巴狗》和《大槐树》，由著名民俗学家、散文家山曼先生编选、导读，配以童趣、乡野的彩图，渲染出浓郁的民俗韵味。</p>
 <ul>
  <li data-pid="-SefouFC"><b>“嘟嘟熊系列”丛书</b> 葛冰 著 吴带生 绘 中国少年儿童出版社</li>
 </ul>
 <p data-pid="RypjkzBQ">这是一套培养幼儿良好行为习惯的图画书。“嘟嘟熊”是一个好孩子，形象可爱，大家都喜欢他。在小读者中，他有很高的威信，孩子们纷纷以他为学习的榜样。</p>
 <ul>
  <li data-pid="X-gj_M0p"><b>儿歌365 首</b> 圣野 吴少山 选编李全华等 绘 浙江少年儿童出版社</li>
 </ul>
 <p data-pid="L_xhIUu9">精选的365首儿歌，每一首都简短上口、易于吟诵。儿歌的主题也十分丰富，十分贴近儿童的生活经验和心理经验。</p>
 <ul>
  <li data-pid="RAEQ1ION"><b>一园青菜成了精</b> 周翔 明天出版社</li>
 </ul>
 <p data-pid="4WWVAxyu">这本书是首届丰子恺儿童图画书奖的获奖作品，巧妙地蕴藏了青菜们的特性。小葱青秆像一根银杆枪；韭莱的叶片狭长如刃锋，辣椒的浑身红通通，茄子的紫涨圆滚，让人读出意料之外却又不得不信服的荒诞。</p>
 <ul>
  <li data-pid="Kh7t1uq7"><b>牛年的礼物：剪纸中国 </b>赵镇琬 著于平 任凭 图 新世界出版社</li>
 </ul>
 <p data-pid="CTYgmO6f">一本绝美原创剪纸图画书。在天地之初，为人类造福的神农炎帝原来长了个可爱的牛牛头。他先是教人种五谷，又为人寻找草药，尝尽百草......</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="A6gsEmPj"><b>日本</b></p>
 <ul>
  <li data-pid="8CsF9yoC"><b>鳄鱼怕怕牙医怕怕</b> (日 ) 五味太郎 著 ( 日 ) 五味太郎 图台北上谊文化编辑部 译</li>
 </ul>
 <p data-pid="ivgftBvV">我们害怕去看牙医，原来牙医也害怕给人看病，五味太郎以诙谐的对比文字，在书中幽默的刻画出病人和牙医对立的矛盾心理，让人会心一笑。</p>
 <ul>
  <li data-pid="RdDLqAia"><b>鲸鱼</b> (日 ) 五味太郎 著 ( 日 ) 五味太郎 图余治莹 译 河北教育出版社</li>
 </ul>
 <p data-pid="_UoINLg-">一只候鸟飞在天上，大喊着“鲸鱼”！人们找来找去都没找到，其实是村边湖的形状跟鲸鱼一样，候鸟并没有骗人。看到的东西不一定就是事实，你眼里的事实别人也不一定认同。一个看似简单的故事留给读者的是无穷的回味。</p>
 <ul>
  <li data-pid="mdzab-GX"><b>“可爱的鼠小弟”系列</b> (日 ) 中江嘉男 文 ( 日 ) 上野纪子 图 赵静 文纪子 译</li>
 </ul>
 <p data-pid="rk2VP2jV">鼠小妹织了一件小背心，送给鼠小弟做生日礼物。她想把礼品盒包得漂漂亮亮的，动物们热心地来帮忙，礼物终于包好了。可是后来，鼠小妹惊异地发现，小背心根本就没被放进盒子里……</p>
 <ul>
  <li data-pid="tO7VswRS"><b>喵喵</b> (日 ) 长野英子 文 / 图蒲蒲兰 译 二十一世纪出版社</li>
 </ul>
 <p data-pid="t62ZphDO">这是一个剪贴绘本，绘本中的婴儿、衣服、玩具、猫，甚至是大地，都是用纸剪出来贴在白纸上，然后再根据故事为主人公画上表情，营造出一个又一个亲切、可爱的图案。</p>
 <ul>
  <li data-pid="-GBFe-xq"><b>噼里啪啦系列丛书</b> (日 ) 佐佐木洋子 编绘 二十一世纪出版社</li>
 </ul>
 <p data-pid="9xhxW1K1">共七册，分别描绘孩子在刷牙、洗澡、游玩、吃点心等各种时候所碰到的问题，以风趣的方式教会他们人生的最初的知识。</p>
 <ul>
  <li data-pid="5u5rPID1"><b>首先有一个苹果</b> (日 ) 伊东宽 著蒲蒲兰 译 二十一世纪出版社</li>
 </ul>
 <p data-pid="3kc5uLx4">首先有一个苹果，两条虫子从里面钻出来，三只鸟想吃虫子……作者很巧妙地把原本枯燥乏味的数字融入了形象生动的故事中，以一种既好玩又有趣的方式表达了数字的概念。</p>
 <ul>
  <li data-pid="xDzpt30Q"><b>小酷和小玛的认知绘本</b> (日 ) 秦好史郎 著杨文 译 北京少年儿童出版社</li>
 </ul>
 <p data-pid="6S70J8zq">有各种各样的颜色，真好玩！熊宝宝小酷和小玛起床了。它们穿上衬衣，衬衣是绿色的，和树叶一样。它们又穿上了和桃子，野草，大海一样颜色的衣服……骨碌，骨碌地转了起来。</p>
 <ul>
  <li data-pid="jj6Eo2w0"><b>圆白菜小弟</b> (日 ) 长 新太 著 ( 日 ) 长 新太 图彭懿 译 南海出版公司</li>
 </ul>
 <p data-pid="rKS7RBmi">猪山大哥说他饿了，要吃圆白菜小弟。可圆白菜小弟指着天上说，你要是吃了我，你就会变成这个样子。“嘣咔——”猪山大哥飘到了天上，鼻子变成了圆白菜.....</p>
 <ul>
  <li data-pid="uzawWmBG"><b>古利和古拉 </b>(日 ) 中川李枝子 著 ( 日 ) 大村百合子 图季颖 译 南海出版公司</li>
 </ul>
 <p data-pid="14uOZzWW">古利和古拉发现一个大鸡蛋，可是鸡蛋太大没法搬走啊！这可难不倒古利和古拉，拉着大锅，背着大碗……古利和古拉在林子里烤起了蛋糕。</p>
 <ul>
  <li data-pid="S6UsrR3A"><b>好饿的小蛇 </b>(日 ) 宫西达也 著彭懿 译 二十一世纪出版社</li>
 </ul>
 <p data-pid="kRT9e8M3">好饿的小蛇散步的时候，发现了一个圆圆的苹果，“啊呜”真好吃；第二天，小蛇发现了一根黄色的香蕉，“啊呜”真好吃……第六天，小蛇发现了一棵结满红苹果的树，你猜猜好饿的小蛇会怎么样呢？</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="I28xSzXD"><b>韩国</b></p>
 <ul>
  <li data-pid="B-3n7XWQ"><b>黄雨伞</b> (韩 ) 柳在守 著 / 绘 ( 韩 ) 申东一 作曲 接力出版社</li>
 </ul>
 <p data-pid="srabiFRv">雨中，一把黄色的雨伞飘动在小路上。一会儿，蓝雨伞、红雨伞、绿雨伞、粉红雨伞也飘来了，它们飘过路口，飘过游乐场……看上去像是万紫千红的花朵。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="EGy9kzDH"><b>美国 </b></p>
 <ul>
  <li data-pid="d5NaSLxg"><b>好饿的毛毛虫</b> (美 ) 艾瑞克·卡尔 著郑明进 译 明天出版社</li>
 </ul>
 <p data-pid="uChN3GTZ">一条毛毛虫饿了，他开始去找吃的。星期一吃了一个苹果，可还是好饿。星期二吃了两个梨子，可还是好饿。星期三吃了三个李子，可还是好饿......</p>
 <ul>
  <li data-pid="EZzWATb2"><b>大卫, 不可以</b> (美 ) 大卫·香农 著 ( 美 ) 大卫·香农 图余治莹 译 河北教育出版社</li>
 </ul>
 <p data-pid="YsXxOzl7">《纽约时报》年度最佳图画书。大卫是一个把家里搞得一团糟的小男孩，世界上哪一个孩子不渴望像大卫一样随心所欲地在墙壁上乱写乱画、把浴室变成一个沼泽地、头戴铁锅敲得叮当乱响……</p>
 <ul>
  <li data-pid="UC7OS_o-"><b>家有恐龙系列</b> (美 ) 简·约伦 文 ( 美 ) 马克·蒂格 绘任溶溶 译 接力出版社</li>
 </ul>
 <p data-pid="FuRahcE6">作品以十只可爱的大恐龙宝宝为主角，根据幼儿在生活中遇到的不爱睡觉、不好好吃饭等生活细节，创作了童趣盎然、有趣好玩的童话。</p>
 <ul>
  <li data-pid="VeVWwcfy"><b>母鸡罗丝去散步 </b>(美 ) 佩特·哈群斯 著上谊编辑部 译 明天出版社</li>
 </ul>
 <p data-pid="dH-6eXgX">这是一本很简单的绘本经典，短短的几句，却有很丰富的场景铺排，洋溢着欢乐悠闲的氛围，散步的目的有了更多的想象，小孩子看了自会会意的咯咯笑。</p>
 <ul>
  <li data-pid="Q47e-aYq"><b>逃家小兔</b> (美 ) 玛格莉特·怀兹·布朗 著 ( 美 ) 克雷门·赫德 图黄迺毓 译</li>
 </ul>
 <p data-pid="3SPvZS57">小兔子对妈妈说：“我要跑走啦！”“如果你跑走了，”妈妈说，“我就去追你，因为你是我的小宝贝呀！”一场爱的捉迷藏就此展开……</p>
 <ul>
  <li data-pid="9-sW0rL0"><b>小蓝和小黄</b> (美 ) 李欧·李奥尼 著 ( 美 ) 李欧·李奥尼 图彭懿 译 明天出版社</li>
 </ul>
 <p data-pid="CmClKcjo">讲述了小蓝和小黄是一对好朋友，一起游戏，一起上课。然后小蓝和小黄融合在一起变成了“绿”的故事。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="-XxYAOS6"><b>欧洲国家 </b></p>
 <ul>
  <li data-pid="GhUTrmY4"><b>我爸爸</b> (英 ) 安东尼·布朗 著 ( 英 ) 安东尼·布朗 图余治莹 译 河北教育出版社</li>
 </ul>
 <p data-pid="E7GpoW6J">通过简单朴实的语言和精心设计的排比句式，用孩子的口吻和眼光来描绘一位既强壮又温柔的爸爸，不仅样样事情都在行、给孩子十足的安全感，还温暖得像太阳一样</p>
 <ul>
  <li data-pid="M2oP0wWZ"><b>我喜欢书 </b>(英 ) 安东尼·布朗 著 ( 英 ) 安东尼·布朗 图余治莹 译 河北教育出版</li>
 </ul>
 <p data-pid="xzglO6qC">向孩子展示了一个有趣的书的世界，让他们知道，书里有童话，有恐龙，有怪物，有知识，有强盗...有一切他们想到的和想不到的事情……撩动他们对阅读的好奇和渴望。</p>
 <ul>
  <li data-pid="d3TWdWEF"><b>小鸭黛西系列</b> (英 ) 简·西蒙斯 著曹敏等 译 接力出版社</li>
 </ul>
 <p data-pid="FRF5Qw2r">讲的是一只富有好奇心的小鸭子黛西，跟着鸭妈妈一起探索世界的故事。</p>
 <ul>
  <li data-pid="4OfE67-v"><b>小玻系列翻翻书</b> (英 ) 希尔 编绘 彭懿 译 接力出版社</li>
 </ul>
 <p data-pid="H1RUoxUX">以一只名为小玻的天真无邪的顽皮小狗为主角，根据幼儿生活选取主题，演出温馨可爱的家庭生活和精彩有趣的故事.</p>
 <ul>
  <li data-pid="1llKji1B"><b>我也可以飞</b> (法 ) 让罗姆·瑞利亚 著麦小燕 译 湖北美术出版社</li>
 </ul>
 <p data-pid="17H1PTzl">鸟宝宝想当一只飞鸟，可爸爸妈妈告诉他，还太小了。鸟宝宝不服气，当不成飞鸟那么就去当蚂蚁。可当蚂蚁就得在树干上走路，鸟宝宝刚一起步走，就直往下落，幸好有片叶子接住了他……</p>
 <ul>
  <li data-pid="gUOYewhQ"><b>小熊不刷牙</b> (瑞士 ) 斯伐拉纳·提欧利那 著 湖北美术出版社</li>
 </ul>
 <p data-pid="6boyJm1e">一本讲刷牙的重要性，教导孩子养成良好的卫生习惯，适用于所有家长和孩子的优秀图画书。</p>
 <ul>
  <li data-pid="K__GLIxK"><b>“米菲”绘本系列</b>（荷兰）迪克·布鲁纳 著 童趣出版有限公司</li>
 </ul>
 <p data-pid="PoksfPBN">讲述了小兔米菲的日常生活故事——去海边、去动物园、去做客、过生日、生病了等等。故事很简单，很温馨，有欢乐，有伤心，有困难，有胜利……</p>
 <ul>
  <li data-pid="eme7jaFz"><b>彩虹色的花</b> (波兰 ) 麦克·格雷涅茨 著蒲蒲兰 译 二十一世纪出版社</li>
 </ul>
 <p data-pid="tvkdifOQ">一朵彩虹色的花，将自己的花瓣都用来帮助有困难的小动物了，最后，自己却被覆盖在白雪下面，可是，它的希望和梦想还在继续，当春天来到时，新的花朵又在阳光下绽放开来……</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="ZDCaxOCS"><b>澳大利亚</b></p>
 <p data-pid="hr5qAhFQ"><b>给爸爸的吻</b> (澳 ) 弗朗西斯·沃茨 著 ( 英 ) 戴维·利格 图熊怡然 译</p>
 <p data-pid="VB0UiD6Q">小熊爸爸对小熊的爱轻轻地地流露出来，滋润着小熊的生命和心灵。而小熊献给爸爸的甜甜的吻，还有大大的拥抱，就是对这温暖的、无私的舔犊之爱的最好的回报。</p>
