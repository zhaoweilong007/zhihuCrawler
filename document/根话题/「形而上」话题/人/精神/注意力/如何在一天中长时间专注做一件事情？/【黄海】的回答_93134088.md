# 如何在一天中长时间专注做一件事情？
- 点赞数：6546
- 更新时间：2017年02月19日01时55分36秒
- 回答url：https://www.zhihu.com/question/19610991/answer/93134088
<body>
 <p data-pid="owoPBjEK">泻药。</p>
 <p data-pid="VLXQhzIS">我是按照以下的办法来保持精神集中的。</p>
 <ol>
  <li data-pid="QIn1UFT2"><b>遇见你最好的朋友——Pareto定律。</b>Pareto定律也就是所谓的80/20定律，它认为20%的原因能造成80%的结果。所以，在你真正想要集中精力做一件事情之前，应该在清楚什么时候开始着手，而不是一时兴起，毫无准备。</li>
  <li data-pid="hkzx0WGs"><b>30分钟的马拉松。</b>以30分钟为周期，只做好一件事情，禁止多任务操作。</li>
  <li data-pid="Wd297XWf"><b>保证高质量的睡眠。</b>根据自己的生理习惯，每天保持充足的睡眠，而不是为了其他工作任务，牺牲睡眠时间。在自己的能力范围内，买最好的床垫、枕头、被单等床上用品。</li>
  <li data-pid="aVNcTztw"><b>列清单。</b>我知道你有很多事情要做，但我希望你能列出3件每天必须做的事情，无论每天怎么样，这3件事情一定要完成。</li>
  <li data-pid="rD0-FwYY"><b>用运动来代替咖啡因子。</b>以前我习惯喝咖啡来提神，自从看完<a href="https://link.zhihu.com/?target=http%3A//open.163.com/movie/2013/11/4/0/M9C55AKKH_M9C55E340.html" class=" wrap external" target="_blank" rel="nofollow noreferrer">罗素・福斯特: 我们为什么要睡觉？</a>这个TED演讲之后，我通过锻炼身体，来让自己的身体更加充满活力，而不是一味地用咖啡来使自己感到振奋。</li>
  <li data-pid="5eGzfZA5"><b>远离时间黑洞。</b>回复无关紧要的信息、手机即使短讯、无谓的吹水闲扯，这些都会极大地消耗了时间，挫伤你有限的精力。<b>一句话，关掉必要的电子产品，杀掉通知。</b></li>
  <li data-pid="-wbxy4-4">最后，尼采说过：<b> "there are no actors, only actions."不要只深思熟虑，最后却止于幻想，什么都不做。</b></li>
 </ol>
