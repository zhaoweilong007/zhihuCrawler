# 据报告显示北京平均月薪 13930 元，第四季度全国平均招聘薪酬 10558 元，如何分析此数据？
- 点赞数：7397
- 更新时间：2023年02月06日23时02分54秒

<body>
 <p data-pid="hv7tyMgB">我讲几句吧，可能颠覆60%以上人的三观。</p>
 <p data-pid="NmwI-84K">这个报告中，北京的平均月薪是13930元，其中IT行业近3万元。大家知道所谓的一次分配、二次分配吧，具体到这个薪酬报告的案例中：</p>
 <ul>
  <li data-pid="IsV9ThuG">一次分配：就是自己通过给公司打工实际赚的钱；</li>
  <li data-pid="C3FcsZHh">二次分配：自己赚的钱再经过税收、社保等进行重新分配。</li>
 </ul>
 <p data-pid="ghqa0wjN">好了，我们就分别按照月薪1万和3万的两个人来算，他俩真正靠给公司打工获得的收入是多少呢（一次分配）？如下所示：</p>

 <p data-pid="XPAF0Hmp">也就是说：</p>
 <ul>
  <li data-pid="uLsQqyK-">所谓月薪1万的人，实际赚的是13850，实际到手7665，二次分配后剩余 55.3%；</li>
  <li data-pid="ThXZ8JY7">所谓月薪3万的人，实际赚的是41336，实际到手21328，二次分配后剩余51.5%。</li>
 </ul>
 <h2>公积金修正</h2>
 <p data-pid="RqqEtwTw">上述算法是把公积金当作税收看待的，但其实，公积金相当于强制储蓄 + 税收的结合态，也就是说：</p>
 <ul>
  <li data-pid="cvS-OiAB">当公积金不消费的时候，它是强制储蓄，和退休前的你无关；</li>
  <li data-pid="EXbQeBTc">当公积金消费来买房的时候，基本上70%用来支付的是各种广义税收（土地出让金等）。</li>
 </ul>
 <p data-pid="OODjMh7F">这个时候你可能会意识到，为啥公积金一定要强制储蓄了吧，其实就是让你强制消费房地产，进而可以征收70%左右的广义税收。</p>
 <p data-pid="K1hx-3e7">因此，可以把你二次分配后的收入 上调每月公积金总额*30%。也可以通过房租/房价的PE值进行校正，殊途同归。</p>

 <h2>消费环节修正</h2>
 <p data-pid="_z75zm_x">你打工换来的钱只是一般等价物，最终的目的肯定是要通过一般等价物换他人的劳动成果。也就是说，前面章节只是你用劳动换进一般等价物过程中征收的税，当你再换出一般等价物的时候，还有逃不掉的消费税：</p>
 <ul>
  <li data-pid="RMT1LVQ0">狭义消费税：就是有15类商品有消费税；</li>
  <li data-pid="lCdMDxmf">商品增值税：商品基本上都有；</li>
  <li data-pid="Cl5XOWFJ">广义消费税：狭义消费税 + 商品增值税 + 企业所得税。</li>
 </ul>
 <p data-pid="uyOiXBRG">这个广义消费税，就按照商品价格的25%算（不严谨，但量级差不多）。因此，假设你挣得钱全部花掉，完成了 劳动成果 ——&gt; 一般等价物 ——&gt; 劳动成果 的端到端生产消费后：</p>
