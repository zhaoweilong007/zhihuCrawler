# 高考历史选择题应该怎么做？

<body>
 <p data-pid="-vn3IWXq">全国一卷<br>
  2017高考 历史选择全对 预估历史80+</p>
 <p data-pid="GQvXRyB3">一点感想 希望能给学弟学妹们一点帮助～</p>
 <ul>
  <li data-pid="FVOi9BGg"><b>【一】敢于放弃</b><br>
   我们学校从高三一开学就开始进行文综练习，个人认为尽早的接触文综整卷是非常有帮助的，不仅训练题感，更是训练对时间的把控程度。我把放弃放在首位，是基于对自己的认识，虽然我练习这么多文综卷，但完完整整做完的，只有5份。也就是说，我对我自己的速度的把控之下，一定要舍弃某些题目，来确保大局的完整度。所以，权衡利弊，我最终选择放弃41题小论文。毕竟每年能拿高分的人数屈指可数（6分以上）这是我们老师告诉我们的，但是各个省份改卷不同，也要留意，有的省份有写有分，这就要看情况定夺了。<br>
   这样你就有更多时间去完成其他有把握的科目啦</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="PMEAWxpJ"><b>【二】不要盲目背诵</b><br>
   全国卷考的是素养，而非背诵，从高一开始，我就没背过啥背景，意义，这类毫无用处的课文了，生搬硬套是很可怕的，与其有时间有精力背诵这些，不如多整理几遍时间表</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <ul>
  <li data-pid="UNiXZFlN"><b>【三】要善于整理</b><br>
   给大家看几张我整理的历史笔记<br>
   多翻参考书，有些见解真的，独到又有趣<br>
   善于记到小本本上，多读读</li>
 </ul>
