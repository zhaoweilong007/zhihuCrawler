 <p data-pid="8fslnnPJ">不说教，不灌鸡汤，说说掏心窝子的话，<b><i>内容简洁，但干货较多，建议收藏！</i></b></p>
 <h3><b>1、开学篇</b></h3>
 <ul>
  <li data-pid="Ul_zoH4a">认真阅读录取通知书上的入学须知！</li>
  <li data-pid="3r1tv0L6">早点加入新生群，提前熟悉大学生活。</li>
  <li data-pid="9vI_Gm-Z">不要带太多东西去学校。</li>
 </ul>
 <h3><b>2、寝室篇</b></h3>
 <ul>
  <li data-pid="G91K0YWm"><b>寝室选床位小技巧</b></li>
  <ul>
   <li data-pid="JOuXoX8t">上铺，避免别人坐你床上。</li>
   <li data-pid="Ts3_Aa2U">靠窗，光线和通风比较好。</li>
   <li data-pid="6-foRaNN">离灯远，不用担负晚上关灯的职责。</li>
  </ul>
  <li data-pid="gtQHhZiG"><b>寝室社交原则</b></li>
  <ul>
   <li data-pid="g7nMirGP">少待宿舍，过于强调合群，也是走向平凡的开始</li>
   <li data-pid="Qz4ASCIc">拿别人的东西，务必要说一声</li>
   <li data-pid="76jbz6yw">不要在背后议论他人，不加入任何八卦团</li>
   <li data-pid="ic9yoBYz">刷剧玩游戏听音乐请带上耳机</li>
   <li data-pid="-jujllIy">尽量在宿舍搞好关系，但不要认为室友就是朋友</li>
  </ul>
 </ul>
 <h3>3、学习篇</h3>
 <ul>
  <li data-pid="tIZjW14t">不求名列前茅，但绝对不能挂科！</li>
  <li data-pid="m0begZfj">即使挂科，也一定不要考试作弊！</li>
  <li data-pid="40M6kTDV">学好本专业知识，这是毕业后的安身立命之本！</li>
  <li data-pid="V-YYUU9N">英语学习很重要，早点通过四六级！</li>
  <li data-pid="QDF76z1r">宿舍不是学习的地方，图书馆才是知识的海洋！</li>
  <li data-pid="DSsBqDV5">一定要学会一门技能！比如office、photoshop或者Python…</li>
 </ul>
 <p data-pid="aFOoq2zc"><b>4、考试篇</b></p>
 <ul>
  <li data-pid="iEcajEYR"><b>本科期间「必考」证书</b></li>
  <ul>
   <li data-pid="9jWrn27p">CET-4、CET-6</li>
   <li data-pid="boKvZpN0">普通话等级考试（文科）</li>
   <li data-pid="Qhki5F7J">全国计算机二级</li>
   <li data-pid="hFv7-Xnt">驾驶证</li>
  </ul>
  <li data-pid="yyEK7_Sh"><b>本科期间「选考」证书</b></li>
  <ul>
   <li data-pid="fgyeY8dd">导游证</li>
   <li data-pid="b4ccNytr">教师资格证</li>
   <li data-pid="6x17WwmJ">托福&amp;雅思</li>
   <li data-pid="UdsmwqsZ">注册会计师（CPA）</li>
   <li data-pid="wyeh4iBu">法律从业资格证（法考）</li>
  </ul>
  <li data-pid="knO6vTfr"><b>大学生各类考试的报名渠道，基本就是这4个网站：</b></li>
  <ul>
   <li data-pid="vZHhoo3I"><b>①</b>中国教育考试网</li>
   <li data-pid="XtLML28A"><b>②</b>中国人事考试网</li>
   <li data-pid="X6xuC8GU"><b>③</b>全国翻译专业资格考试官网</li>

  </ul>

 <h3>6、实习篇</h3>
 <ol>
  <li data-pid="lMRUhq4W">实习是性价比最高的职业试错手段，早点规划。</li>
  <li data-pid="aWyZ5uPg">就业形势越来越严峻，大一大二多出去实习，不要做一张白纸。</li>
  <li data-pid="8MJOJHLQ">尽量选择大平台实习，他们经常招聘暑期实习生，多关注实习类微信公众号、boss直聘等。</li>
  <li data-pid="50pZqWzn">实习主管(老师)的人品比实习薪资重要。</li>
  <li data-pid="WdaQVrgZ">与组里同事和睦相处，但也要学会拒绝重复性的工作。</li>
  <li data-pid="UJ9X0ctP">实习在精不在多——看看这段经历能否丰富简历，为自己赋能。尽量少做发传单、服务员、淘宝客服、送外卖这种体力活。</li>
 </ol>
 <h3>7、恋爱篇</h3>
 <ol>
  <li data-pid="Cgmfp3Dp">能在大学谈恋爱，就一定要在大学谈；不能谈的要创造机会谈；</li>
  <li data-pid="rUDbI1Tg">树立正确的恋爱观：有他／她活得更好，没他／她也能活；</li>
  <li data-pid="_fZ5hrK1">掌握必要的生理知识～～～</li>
  <li data-pid="8PQnC9kd"><b>不要因为寂寞而</b><a href="https://www.zhihu.com/search?q=%E6%81%8B%E7%88%B1&amp;search_source=Entity&amp;hybrid_search_source=Entity&amp;hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2565198333%7D" class="internal">恋爱</a><b>，不要因为无聊而恋爱，不要因为别人都在恋爱而恋爱；如果你正处于恋爱阶段，请认真去爱，负责任的去爱，男生一定要有保护女生的决心。</b></li>
  <li data-pid="mao9Z2w-">如果你家里不富裕，或者有学业上的追求，不妨把恋爱的时间花在学习，毕竟本科期间的恋爱，只有极少数能走到<a href="https://www.zhihu.com/search?q=%E5%A9%9A%E5%A7%BB&amp;search_source=Entity&amp;hybrid_search_source=Entity&amp;hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2565198333%7D" class="internal">婚姻</a>（但是研究生期间的恋爱会稳定的多，我对家里即将读研的妹妹说，一定要在校找到男朋友，不然工作后很难找）。</li>
 </ol>
 <p class="ztext-empty-paragraph"><br></p>
 <h3>8、健康篇</h3>
 <ol>
  <li data-pid="9C6ofhKX">有条件的话可以办张健身卡，学校周围的都不贵。</li>
  <li data-pid="9EQn5SF1">千万不要习惯去熬夜，头发真的会掉。</li>
  <li data-pid="8TY1YRd5">有一个自己喜欢的体育运动，比如游泳、足球，跑步也算。</li>
  <li data-pid="BkelGW0M">校医院很便宜，有什么小毛病建议走几步去看看，比如智齿拔牙，外面上千，校内报销后几十搞定。</li>
  <li data-pid="0bqP4FCp">心理健康更重要，不要太为难自己。</li>
 </ol>
 <h3>9、自律篇</h3>
 <ol>
  <li data-pid="YLPLzK__">优秀真的是一种习惯！</li>
  <li data-pid="dM-7br4W">留够维持生计的费，把多余的钱省下存起来，学会理财！赚不了多少也亏不了多少，要的就是经历。</li>
  <li data-pid="JVFOcvVK">远离那些能让你短时间内获得短期满足的东西。如：小视频、脑残小说电视剧、垃圾游戏。卸载掉抖音，快手，今日头条等极为浪费时间的APP，你每天可以多出很多时间做有意义的事情。</li>
 </ol>
 <h3>10、人际关系篇</h3>
 <ol>
  <li data-pid="Ob9i4K_S">和宿管大妈打好关系，你会体会到什么叫做特殊待遇</li>
  <li data-pid="rBhCbvcC">平时学会多夸夸人，会让你的人际关系变好很多。<b>比如，看到这篇文章花1秒点个赞鼓励我～</b></li>
  <li data-pid="SzVFVf6x">有机会一定要入党</li>
  <li data-pid="xKavrDGH">脸皮厚一点，多去参与比赛和活动</li>
  <li data-pid="sIrYo3vv"><b>培养自己的分享精神</b>。有好的资料、整理的笔记、辅导员的通知，完全可以分享给室友和同学，不用藏着掖着。分享出去不仅会增加你在同学对你的好感，而且有利于养成有条理、按时按点完成任务的学习习惯。</li>
  <li data-pid="247pCeAp"><b>多与一些厉害的师兄师姐交流。</b>举个例子，你想考研，那你必须要知道<a href="https://www.zhihu.com/search?q=%E7%A0%94%E7%A9%B6%E7%94%9F&amp;search_source=Entity&amp;hybrid_search_source=Entity&amp;hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2565198333%7D" class="internal">研究生</a>的报名时间，如何复习？如何找资料？如何面试？如果这个时候有个帮助你的师兄师姐，那么这些问题都会迎刃而解。当然大家有问题也可以在这个答案下面留言。</li>
  <li data-pid="wmTdbn6r">大一要警惕学长学姐以及教官<a href="https://www.zhihu.com/search?q=%E5%A5%97%E8%BF%91%E4%B9%8E&amp;search_source=Entity&amp;hybrid_search_source=Entity&amp;hybrid_search_extra=%7B%22sourceType%22%3A%22answer%22%2C%22sourceId%22%3A2565198333%7D" class="internal">套近乎</a>，尤其是想处对象的那种。</li>
  <li data-pid="86xf0Tnh"><b>别在同学面前搞任何虚的东西。</b>大学四年时间，大家有足够的机会去读懂你，家里是否真有钱，学习好不好，能力强不强。前期把自己吹捧的多高，后期在同学心里的印象就有多差。</li>
 </ol>
