# 大学计算机系最努力的同学都是如何学习的？

- 更新时间：2023年05月24日23时36分02秒

<body>
 <p data-pid="96vBHZmD"><b>那当然是疯狂卷啊。</b></p>
 <p data-pid="2JSdWlon"><b>不是学院最努力的同学，但是自认为还是算在正确的道路上持续付出。</b></p>
 <p data-pid="M6HqUMsH">在高中，我完全没有想过自己会学计算机，成为程序员。</p>
 <p data-pid="n-X9hwEv">我在大一之前完全没有编程经验，高考后，填的专业前几个也是清一色的传统工科，被软件工程录取纯碎是意外。</p><a href="https://link.zhihu.com/?target=https%3A//csguide.cn" data-draft-node="block" data-draft-type="link-card" data-image="https://pic1.zhimg.com/v2-2dbd0d1dd19a062134782bb94f052abc_ipico.jpg" data-image-width="200" data-image-height="200" class=" wrap external" target="_blank" rel="nofollow noreferrer">计算机系统学习网-编程指北</a>
 <p data-pid="xOehZ2JR">大一开学，第一门编程语言就是学的 C 语言，怎么说呢，老师讲的，书上写的我都能看懂，但是让我写一个大的程序，我就摸不到头脑。不知从何下手，学了半学期，也只能写写课后习题，算算水仙花数什么的。</p>
 <p data-pid="HoM8QHe8">大一上考完 C 语言后，我便在心里默默念想，这辈子再也不写 C 了，但没想到的是，后来这成了我最爱的一门语言。</p>
 <p data-pid="ZDJFuyVB"><a href="https://link.zhihu.com/?target=https%3A//www.yuque.com/csguide/go" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">https://www.</span><span class="visible">yuque.com/csguide/go</span><span class="invisible"></span></a></p>
 <p data-pid="NJ805gWy"><b>整个大一，一个词语就是“迷茫”，一个字就是“混”</b></p>
 <p data-pid="AEPob57L">混到什么程度呢，王者荣耀打了几千场，线代考试前一晚才在 mooc 上刷完所有章节，因为平时上课我基本都没听的，这也是我最后悔的事情。</p>
 <p data-pid="vmqBU6vZ">转变大概从大一下开始吧，那时候好几个室友都拿到了奖学金，于是，我自己也开始慢慢反思，觉得这样下去，大学可能就废了。</p>
 <p data-pid="4xWnaHGx">默默的把王者荣耀卸载（当然，后面又下回来了，反反复复十几次。。。）</p>
 <p data-pid="8jmdWHPN">大一下，开始学习数据结构与算法、数字逻辑这些计算机核心课了。</p>
 <p data-pid="uoeftJGY">此外，还学了一门当时我看来非常牛逼的语言：Java。</p>
 <p data-pid="06zqATPT">为啥说我当时觉得它牛逼呢？</p>
 <p data-pid="AoiyXT-B">因为刚经历过 C 语言的摧残，发现 Java有丰富的类库，各种方便的工具类。</p>
 <p data-pid="ZnwO2_Zh">再也没有指针了，变成了对象引用，简直是新大陆。</p>
 <p data-pid="JnqzCxCx">所以我学得很认真，还记得当时看过高琪的 Java 300集 教程，也是从那里算是正是开始了我编程学习之路。</p>
 <p data-pid="wgCh0FvD">自己也用 Java 写了不少好玩的东西，比如联机五子棋、HTTP Server 等等。 类似这种：</p>
 <p data-pid="Vl-uNyz3">这让我第一次感受到编程这么有趣，俗话说兴趣是最好的老师，Java 也算是帮我解锁了对于编程的兴趣。</p>
 <p data-pid="vMzGhVYZ">在学 Java 的时候喜欢在网上搜资料，但总觉得各种博客上的知识过于零散，为了“精通” Java，我在知乎搜索各种书单，先后看了《Java核心技术卷》《Java并发编程》好几本书，自觉 Java 基础还算可以，不过现在已经快两年没写了，成功转入C++阵营。</p>
 <p data-pid="vmZJrESH">而且我有个习惯，就是每天早上基本都是六七点起床，八点多对我已经算赖床了，所以和我室友显得有点“格格不入”。</p>

 <p data-pid="hieio0Yw">早上起太早的话，他们还在睡觉，我在寝室写代码的话，难免会发出各种声音吵到他们，尽管我做什么都是蹑手蹑脚，生怕发出椅子摩地，或者碰撞的声音，这让我也觉得不太方便。</p>
 <p data-pid="3AknVX3w">所以，后面没有课的早上我一般都去图书馆五楼自习室了，在那看书，看书觉得烦了，就写会代码或者刷刷知乎，看看博客。</p>

 <p data-pid="Nw8eh82e">一般我都是赶在中午下课高峰之前去吃完，然后去学校绿荫道散步</p>
 <p data-pid="xQy2S9zN">大二时，在上一些无聊的课时，我总把这本书带着，每节课可能会翻个几页，那时候看到里面的汇编、内存管理、存储器层次我是真的云里雾里，因为还没学操作系统，C 语言也学得很差。</p>
 <p data-pid="nSniQuhr">所以断断续续的看了几章，说不上学到了什么，但让我对这种枯燥的基础知识第一次产生了兴趣，有一种想彻底搞懂它们的想法。</p>
 <p data-pid="o_qSs6Qs">后来在知乎大佬的指引下，为了看懂这本书，去看了王爽那本《汇编语言》，也在上图。</p>
 <p data-pid="Y5X1rvqL">重学了 C 语言（主要是指针和内存），由于已经学了汇编，指针对我再也不是问题了，同时看完了《C和指针》、《C专家编程》、《C陷阱与缺陷》，对 C 语言使用得也愈加的熟练。</p>

 <p data-pid="LH7x9CMu">大二那段时间，学院里突然刮起全员学机器学习的风气，虽然不至于全部，但是真的是很多人都在学，明明 16 年大一时客户端安卓开发还挺火的，到了 17 年各种机器学习、神经网络就席卷而来。</p>
 <p data-pid="R2AM1A0O">不能不感叹，IT 行业日新月异呀，不学习就可能被淘汰。</p>
 <p data-pid="EbEHpwer">当然了，我也不例外的加入了，先后看了些吴恩达的视频、西瓜书。</p>
 <p data-pid="tCY-bnCG">很多学弟学妹在问怎么咨询、提问，最近我开放了我的编程学习圈子，欢迎加入，不管是计算机转专业，还是大二面临分流、大三找实习、校招、写简历、Offer 选择等都可以直接问。加入星球后，还可以找小北提问1V1定制学习路线、学习规划、修改简历、做项目等等，点击下方了解：</p>

 <p data-pid="X7hmc594">跟着做了手写数字识别这种入门必做的项目，但是后来总觉得对机器学习、深度学习提不起兴趣，原因之一就是我觉得无法完全掌握每一步，了解每一步背后的原理，再加上自己数学也学得不太好，这让我学得很没底气。</p>
 <p data-pid="eZBZVt0W">于是机器学习之路也止于大二。</p>
 <p data-pid="E5uXoaNz">大二是学习内容最丰富的时间，当时室友用 Python 写了抢课脚本，感觉很好玩。</p>
 <p data-pid="u9nkg5jE">然后自己又跑去学 Python，学爬虫。</p>
 <p data-pid="i63dmECe">写了一堆爬虫，什么爬豆瓣电影、知乎回答、抢课脚本，玩了一段时间也没太大兴趣了。</p>
 <p data-pid="CF1UQSIj">时间来到了大二下，按照课程安排，开始上操作系统了。</p>
 <p data-pid="SvZf-3Zk">于是我又在知乎搜索「操作系统怎么学」，不得不说知乎上各位前辈大佬的回答对我的指引作用极其的大。</p>
 <p data-pid="RIRBJrTu">顺着一堆回答，我发现原来还能跟着别人写一个 mini OS，这对我吸引力太大了。</p>
 <p data-pid="xq7LYFM8">于是我买了一本《操作系统真象还原》，这本书真的非常仔细，把写 OS 需要用到的前置理论知识、Intel手册、汇编、内联汇编全都给你讲清了。</p>
 <p data-pid="Dju_QAuh">我一路跟着书写，写到了汇编开机打印出我的名字，再到后来进入 C 语言写内核各个模块，写到内存管理我就中断了，没有继续写下去，这也算个遗憾吧。</p>

 <p data-pid="6xIK2Orz"><b>专注的大三</b></p>
 <p data-pid="Jpm4AiW4">于是在大三左右开始了非常标准的 Linux C/C++学习之路：</p>
 <p data-pid="F1hL4GN_">前前后后看了《C++ Primer》、《APUE》、《UNP》、《深度探索C++对象模型》、《STL源码剖析》、《C++设计与演化》、《Linux内核设计与实现》、《Linux内核完全注释》、《effecitve C++》 、《effective modern C++》、《程序员自我修养》、《Linux高性能服务端编程》、《Linux多线程服务端编程》(知乎陈硕大佬写的)......</p>
 <p data-pid="o8ufBpfr">顺便还看了点 Muduo 源码，自己模仿着基于 Epoll 写一个 Reactor模型的 Web 静态服务器，跟着知乎 Milo 大佬写 Json 库，总之，这段时期是写了一些小”轮子“的。</p>
 <p data-pid="zXkpjSMY">（PS：知乎上有朋友问我是如何看完这么多书的，其实在之前的文章中就写了，看书我会有选择的挑章节看，不会从头看到尾的，而且大三开始课就少了，我基本就在刷这些书，其实时间还蛮多的？毕竟我也没参加什么社团、学生会之类，也没女朋友。。。</p>

 <p data-pid="zK4N23bK">书看得越多，其实你越能找到自己的方向，大三上我就给自己学习定位：</p>
 <p data-pid="3iqk6NlE">深入C++语言（多线程）+ 存储（学习 Redis、leveldb）+ 网络编程（学习muduo，各种网络编程模型）+ 学习分布式（MIT 6.824、ddia、google 三大论文等）</p>
 <p data-pid="bRbKxMej">基本上是沿着上面几个方向去展开学习的，有的看书，有的论文，也有的看博客看源码。</p>
 <p data-pid="wbBgJos-">总之，知识面在大三不断扩充，自己开始关注学习 C10K、C100K 这样的后台服务器开发高性能、高并发的解决方法。</p>
 <p data-pid="0E5Lwdn1">当然了，还有些到现在还没开始，比如 leveldb 的源码、6.824 的 lab，可能是变懒了......</p>
 <p data-pid="fEK4qAxG">也可能是学习变得不那么纯碎面向技术了，变成面向工作、面向赚钱了哈哈哈。</p>
 <p data-pid="S0yh6ZzS">值得一提的是，大三上我们学院有门 System Programing 的课，教材就是使用的 CSAPP，因此我之前大二草草看完的书，得以再次翻开，这一次我把附带的是个实验也挨着挨着做了，收获良多，有位运算各种奇技淫巧，有 Bomb lab 刺激的打怪通关、也知道了buffer overflow attack.... 还有实现基本的内存管理等等。</p>
 <p data-pid="uwM35Yyn">印象最深的便是做 bomb lab，一个个 phase 的通过真的很让人兴奋，我几乎是熬夜连着把这几个 phase 过掉的，当然头发也没少掉。</p>

 <p data-pid="2q91Nopj">在这里列举一些我做过的觉得有意思的 lab（之前有篇文章写过）：</p>
 <p data-pid="qY0mRP8i"><b>第一个是文件系统</b></p>
 <p data-pid="BCOVeegS">这个玩意挺有意思的，就是要让你自己去设计一个文件系统，比如文件、数据块、目录等等，而且你要提供创建、读写、删除、随机读、随机写等等接口，然后通过 Linux 的 VFS（虚拟文件系统） 机制挂载到 Linux 内核中，然后就可以用系统调用 open、read、write 这些来在你的文件系统上进行读写。VFS 是Linux一个机制，它规定了 IO 接口，然后你去实现这些接口，你就可以挂载在上面。通过这个东西我深入的学习和了解了文件系统，报酬也是非常值的。</p>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-3bb18b22491c665c8bb3fd9fd53e5fdc_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-original-token="v2-3bb18b22491c665c8bb3fd9fd53e5fdc" class="content_image">
 </figure>
 <p data-pid="YXU9J0T_"><b>第二个是SVC</b></p>
 <p data-pid="2fQca52G">就是版本管理系统，类似 SVN 和 GIT的，但是相对 GIT 做了很多简化，核心的就是创建分支、分支合并、回滚、commit 这些，通过这个我又去详细了解了 GIT 的实现原理，并且自己写了一个简单版本的 GIT，受益匪浅，也赚到了几千大洋。</p>
 <figure data-size="normal">
  <img src="https://pica.zhimg.com/50/v2-b78851f49339614046809318282185d9_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-original-token="v2-b78851f49339614046809318282185d9" class="content_image">
 </figure>
 <p data-pid="EyxSfm9N"><b>第三个是操作系统的锁实现</b></p>
 <p data-pid="8-xKR410">这个就是让你基于硬件的原子指令 xchg、cmpxchg 这些实现一个锁，这个需要内联汇编、汇编等知识，简直是硬核！！！做完这个也解开了我对锁实现的一些疑惑。</p>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-8ac3dcc1381733761d128b142beb8e69_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-original-token="v2-8ac3dcc1381733761d128b142beb8e69" class="content_image">
 </figure>
 <p data-pid="vSgm803a"><b>Web Proxy</b></p>
 <p data-pid="oD1Gv1sJ">这个就更有意思了，要求用 C++ 写一个代理服务器，类似 Nginx 的代理功能，当然，相比 Nginx 弱鸡很多，</p>
 <p data-pid="gVpXMyg5">但是麻雀虽小，五张俱全，而且老外最有意思的是要求代理服务器对一些图片和 HTML 恶作剧，就是代理服务器随机的在原始服务器，返回的响应里 插入一些图片和文字，因为那天是愚人节。。。 不得不说老外还是很有趣的。</p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-585da1a27e987b8ef311c8ed754ad7d9_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-original-token="v2-585da1a27e987b8ef311c8ed754ad7d9" class="content_image">
 </figure>
 <p data-pid="hf7l3hAF"><b>还有模拟实现 TCP 可靠传输的</b></p>
 <p data-pid="SAlcOFAx">这个就更硬核了，人家老外老师就是给力，直接做了一个网站，这个网站你可以通过 TCP 连接上去，然后你需要运行几个节点，互相发消息，他们之间的消息都会经过老师的网站，所以通过网站上是可以控制丢包率的，也能控制节点的拓扑结构，要求就是让你基于这种不可靠的信道，做出可靠数据传输！这简直就是翻版 TCP 嘛，超时重传、ACK、滑动窗口啥的都给我上！就是像下面图中一样，圆圈就表示你可达的范围，不同节点形成各种网络拓扑，可以调节网络 丢包率 Loss chance。</p>
