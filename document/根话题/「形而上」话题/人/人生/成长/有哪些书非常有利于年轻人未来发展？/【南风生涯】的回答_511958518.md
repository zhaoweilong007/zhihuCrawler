# 有哪些书非常有利于年轻人未来发展？
- 点赞数：79322
- 更新时间：2022年11月29日08时06分28秒
- 回答url：https://www.zhihu.com/question/275351176/answer/511958518
<body>
 <p data-pid="LzqbM2IY">分享一个被过万人收藏的私人书单吧：</p>
 <blockquote data-pid="cpNdNb7p">
  以下书籍都是我亲自看过，并且觉得还不错，值得推荐的！ 我还特意筛选了下，尽量形成一个合理的知识体系，从宏观到微观，从理论到实用。
 </blockquote>
 <p data-pid="8hjlRj4U"><b>一、社会、人类</b></p>
 <ol>
  <li data-pid="MsSVXvdJ">《人类简史》[以色列]尤瓦尔·赫拉利 （一部世界史竟然能够畅销，确实不容易，注意认清作者，有一本名字一样的）</li>
  <li data-pid="GO-96uXm">《人类学透镜》詹姆斯·皮科克（入门书籍）</li>
  <li data-pid="b8L6ISQN">《社会学入门》詹姆斯·汉斯林（都说是入门了，给你一个更宽广的视角吧）</li>
  <li data-pid="mwUFaFpv">《论人类不平等的起源》卢梭（不介绍了，经典书籍）</li>
  <li data-pid="SIr9DU90">《国富论》亚当斯密</li>
  <li data-pid="uAe8IvMr">《功利主义》穆勒（这里的功利主义真和我们生活中常说的功利不一样）</li>
  <li data-pid="oyNRdeV3">《1984》、《动物庄园》不解释，细思极恐。</li>
  <li data-pid="_l66k5Nb">《自私的美德》安.兰德（ 作者力倡个人主义，认为不能使个人利益得到最大伸张的社会，就不是理想社会，我觉得在我们国情下尤其具有启发意义。 ）</li>
  <li data-pid="M6DMJodR">《人生的智慧》叔本华（有人评论“ 如果一生只读一本心灵鸡汤，就要读这一本。”）</li>
  <li data-pid="SDD7zip6">《帕斯卡尔思想录》（ 12岁独自发现了“三角形的内角和等于180度” ，17岁时写成《圆锥曲线论》的神人 ）</li>
 </ol>
 <p data-pid="AM5Bt6Xp"><b>二、心理</b></p>
 <ol>
  <li data-pid="21aLM-8D">《社会心理学》戴维迈尔斯（社会心理学最好的作品，我觉得没有之一）</li>
  <li data-pid="gfQjBA38">《爱的艺术》弗洛姆（网友评价：书薄，但是分量重）</li>
  <li data-pid="jig6EETs">《生命之爱》弗洛姆（很喜欢这个心理学家的书）</li>
  <li data-pid="tH3OESYN">《心的出路》 伊莉莎白•库伯勒•罗斯 （其中的核心理念挺认可的）</li>
  <li data-pid="lh96QI5J">《心灵简史》[英]乔治·汉弗瑞等（心灵成长的通俗诠释，可读，易读）</li>
  <li data-pid="qs6zFLON">《幸福之路》罗素（听书名就知道了）</li>
  <li data-pid="fNtIhNfK">《进化心理学》巴斯（读完对婚姻和择偶有更深的认知）</li>
  <li data-pid="4AV5CDAh">《亲密关系》罗兰.米勒（和上一本搭配着看，有意思）</li>
  <li data-pid="bA1_wMA-">《常识》潘恩（李玉冰译版）应知友<a href="https://www.zhihu.com/people/15fec5c0033ef42c6feffa1d8cda7444" class="internal">@骑鲸者</a>建议，替换《乌合之众》</li>
 </ol>
 <p data-pid="Qy0eUcUU"><b>三、文化</b></p>
 <ol>
  <li data-pid="ZOnkdhU5">《中国文化的深层结构》孙隆基（可能不是很好读，我读了几遍）</li>
  <li data-pid="Jd8uTnps">《素书》（据传是黄石公所做，没做考证，但我觉得很好，耐心品味）</li>
  <li data-pid="UxjuI9Ph">《菜根谭》洪应明（中国传统文化还是有很多不错的，不过少部分内容辩证看待吧）</li>
  <li data-pid="cyHM5ZyD">《智慧书》巴尔塔沙·葛拉西安（一本小册子，据说曾经风靡欧洲，还是要自己悟吧）</li>
  <li data-pid="c5bqrA3p">《人情与面子》黄光国（对于中国人际关系的划分是经典）</li>
  <li data-pid="RbmlVGok">《权经》冯道（ 五代十国的传奇人物,一生历经五朝十二帝,可谓真正的官场不倒翁 ）</li>
 </ol>
 <p data-pid="hpbKf1qw"><b>四、思维</b></p>
 <ol>
  <li data-pid="-rrjO_wu">《超越感觉：批判性思考指南》文森特•鲁吉罗（批判性思考，个人最提倡的）</li>
  <li data-pid="rC2kHDpa">《尼采的锤子》[英]尼古拉斯·费恩（小故事中有哲理，我觉得挺有趣的）</li>
  <li data-pid="Dtr1j2Q1">《思考的力量》理查德保罗（个人非常喜欢的一本，但是有点晦涩，需要耐心）</li>
  <li data-pid="RjFYUQY9">《学会提问：批判性思维指南》这本更加简明</li>
  <li data-pid="L3T1jUNe">《像心理学家一样思考》[美] 唐纳德·麦克伯尼（相对浅显，易读）</li>
  <li data-pid="7Fe880gD">《我们如何思维》约翰杜威（实用主义的典型代表杜威，理论和干货多）</li>
  <li data-pid="tvgJMSJJ">《有效思维》[英] L. S. 斯泰宾（读的人应该不多，我印象最深的是有关自我中心主义）</li>
  <li data-pid="lUS-3yVU">《身边的逻辑学》伯纳·派顿（前面几章看的很爽，后面好像印象较浅）</li>
 </ol>
 <p data-pid="J5j6UPLw"><b>五、教育</b></p>
 <ol>
  <li data-pid="FqT8TNZd">《如何成为卓越的大学教师》肯·贝恩（推荐中国的大学老师好好读，好多大学老师是不合格的）</li>
  <li data-pid="4tctfS0V">《给教师的建议》[苏] B.A. 苏霍姆林斯基（教育界有人说话必称“苏霍姆林斯基说过…所以，不解释，推荐所有老师看）</li>
  <li data-pid="7jS8cJRk">《光有爱还不够》克洛德·阿尔莫（真正的父母之爱，是帮助孩子构建自我。）</li>
  <li data-pid="gg1Ct1RU">《教学设计原理》加涅（老师、培训师建议看，课程设计与开发经典之作）</li>
  <li data-pid="0HCQtzUH">《民主·经验·教育》约翰杜威（杜威，大牛，不解释）</li>
 </ol>
 <p data-pid="d9pCJvMY"><b>六、职业发展和求职</b></p>
 <ol>
  <li data-pid="OgSkpQ_1">《适合比成功更重要》[美] 布朗温•卢埃林（有测评，不过仅供参考）</li>
  <li data-pid="fswd8nmu">《富爸爸，穷爸爸》</li>
  <li data-pid="KZ9Cfwlu">《快乐富有过一生》汉斯·雅各比（个人认为观念上的引导还是不错的）</li>
  <li data-pid="tPRytbe1">《聘之有道面试读心术》王新宇（看看企业招人的逻辑、逆向思考）</li>
  <li data-pid="52-RnozU">《金字塔原理：思考、写作和解决问题的逻辑》[美] 巴巴拉·明托</li>
  <li data-pid="QU3YEA18">《首先，打破一切常规》（识别优势，准确自我定位）</li>
  <li data-pid="Rd7nq949"><b>偶的知乎专栏和公众号，</b>里头有大量职业发展相关文章（职业探索、面试官要求、求职技巧等）</li>
 </ol>
 <blockquote data-pid="nCQfGgf8">
  <b>吾生有涯而知无涯，然则怕什么真理无穷，进一寸有一寸的欢喜。</b>
 </blockquote>
 <p data-pid="vigO8o69"><b>诚心建议有心人收藏此文，并且里头的书有时间都找来看看。</b></p>
