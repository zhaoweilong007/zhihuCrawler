# 人间清醒是什么样子？
- 点赞数：6582
- 更新时间：2024年01月15日23时06分37秒
- 回答url：https://www.zhihu.com/question/494108902/answer/2727035098
<body>
 <p>管家今天早上告诉我，陈清从贵州采样回来了，还带回来一个姑娘。</p>
 <p>并对外称是他的救命恩人，他要以身相许。</p>
 <p>一个十八线小明星劣迹斑斑，竟然能哄得陈清带她回家见我？</p>
 <p>送他进入国内最好的医科大学，培养他成为医学界翘楚，他竟然敢？</p>
 <p>看来我真的是对他太宽容了。</p>
 <p>「停掉他的信用卡，收回他名下的所有房产，另外给温老爷子打电话，去 C 国交流学习的名额不用给陈清留了。</p>
 <p>「学王宝钏恋爱脑？想挖野菜是吧，好，我成全你！」</p>
 <p>（1）</p>
 <p>还没等我开始查那个小明星的消息，手机就一直振个不停。</p>
 <p>陈清的电话不停地打过来。</p>
 <p>接了以后听着陈清理所应当的口气，发现他还丝毫没意识到事情的严重性。</p>
 <p>「妈，爷爷的生日快到了，到时候我带着文婷一起来，你们也见见面，婷婷性格很好，你们一定能相处得很好。」</p>
 <p>我的亲儿子让我跟一个上不得台面的小明星好好相处。</p>
 <p>好样的，我教给他的东西都学到狗肚子里了。</p>
 <p>我冷哼一声：「你还记得你的未婚妻温软吗？」</p>
 <p>「妈，那都是你们老一辈定下的，现在都什么年代了，我要自由恋爱！再说了婷婷……」</p>
 <p>恋爱脑上头太可怕了，感觉听他多说一秒都会降智。</p>
 <p>没等他说完我就挂掉电话。</p>
 <p>手机里管家发过来的文件。</p>
 <p>两个小时，足以将那个小明星的底细查个明明白白。</p>
 <p>因长相跟当红演员李梦有点相像被星探发现，出道就以「小李梦」自称，圈儿里粉丝好坏参半，多数都是冲着那张脸。</p>
 <p>【她为了某个角色频繁出入导演房门。】</p>
 <p>【某整容医生曝光她的整脸细节】。</p>
 <p>网上随便一搜就能看到她打了胎流了产，在片场耍大牌的新闻。</p>
 <p>就这样的也敢肖想陈家儿媳妇的身份？？</p>
 <p>我冷哼一声，真是林子大了什么鸟都有。</p>
 <p>转手将这些消息发给陈清。</p>
 <p>下一秒钟就看到陈清发来的消息。</p>
 <p>「妈，婷婷真的是个好女孩，网上这些都是造谣的，要不是她救我，您现在说不定都见不到我了，我从小都听您的，现在我长大了，我的婚姻我想自己做主，我已经可以独立了，希望您能支持我。」</p>
 <p>好啊我支持你，孩子长大了都会要挟亲妈了，我冷笑。</p>
 <p>我倒是宁愿你死在贵州的大山里，也好过给我整出这点子糟心事。</p>
 <p>「好的，你可以独立了，爷爷生日宴你不用来了，我会把你的名字从家谱删除，你知道的，从小到大，你想要的妈妈都会答应你，这次也如你所愿。」</p>
 <p>（2）</p>
 <p>那个妄想进入陈家的小明星底细不用我动手，自是有人把它送到老爷子桌前。</p>
 <p>我想老爷子应该跟我一样生气。</p>
 <p>毕竟他跟我一样，对陈清寄予厚望。</p>
 <p>陈家世代从医，老爷子是上个世纪战争下来的，退下来以后就在家里养花喝茶，年轻的时候跟我妈妈感情颇深，因为妈妈身体不好，所以就只要了我一个，还纵容地让我跟妈妈姓。</p>
 <p>我是陈家嫡亲的女儿，陈清的父亲是入赘进我们陈家的，他是缉毒警察，在陈清五岁的时候身份暴露，最后连一具完整的尸体都没留下。</p>
 <p>陈清学医，也是我为了他接手陈家的医药公司做准备。</p>
 <p>也正是因为陈清是我们家的独苗，他才敢这么有恃无恐地跟我公开叫板。</p>
 <p>我来到老宅子的时候，老佣人示意我老爷子在书房。</p>
 <p>看他们紧张的表情，我就知道老爷子一定很生气。</p>
 <p>推门进去就看到一地的碎片。</p>
 <p>我一眼看出这是去年老爷子生日的时候，陈清在一个老收藏家手中淘来的紫砂壶。</p>
 <p>不算贵，可平时老爷子喜欢得不得了，别人碰一下他都跟人急。</p>
 <p>我叫来佣人收拾碎片。</p>
 <p>一边给老爷子顺着气，一边搀扶着他走下楼，来到他的小花园。</p>
 <p>「爸，这件事我会处理好，您消消气。」</p>
 <p>「咱们老陈家，用资源不是为了培养一个只知道恋爱的孽障的！还有陈清和温家小姑娘的亲事儿……我这张老脸还怎么面对温老弟！」</p>
 <p>老爷子拄了拄拐杖，语气是又气又急。</p>
 <p>「爸您放心，温家那边我会去说，若云会理解的，我想好了，成不了亲家我就收温软做干女儿。」</p>
 <p>当晚我就看到小明星文婷的社交账号上发了一张牵手照并配文：</p>
 <p>余生请多多指教！</p>
 <p>背景是我给陈清买的大平层，墙上的画是我高价买来的名家名画。</p>
 <p>名家定制的茶几上，倒映着两个依偎着的身影。</p>
 <p>——正是我那个蠢儿子和他的救命恩人女朋友。</p>
 <p>我的耐心到达了极限，打电话给私人助理，立刻收回这套房子。</p>
 <p>十分钟后我就收到了陈清的电话。</p>
 <p>「妈，你为什么收回我的房子，你这样做太过分了！」</p>
 <p>我不耐地揉了揉眉心，一股火气怎么也压不下来：</p>
 <p>「这套房子是我在去年买的，购房合同、房产证还在保险箱里，你要看看房本上写的谁的名字吗？」</p>
 <p>「可我是你儿子，是陈家唯一的继承人！你当初也说过这是给我准备的婚房。」</p>
 <p>陈清还是没有认清事实。</p>
 <p>「你也可以不是。」</p>
 <p>我真的是一分钟都不想再忍受这份天真，顺口补充：</p>
 <p>「对了，房子里的所有东西都是我高价买回来的，每一样都是有记录的，少一样东西都能判个几年，我这么说你明白吗？」</p>
 <p>说完就挂断电话，想我柳歆自持理智，在他爸爸去世的时候都没掉过眼泪，竟然在今天数次破功想打人。</p>
 <p>真是白养的狗东西。</p>
 <p>早知道长大的时候是个恋爱脑，出生之前就应该打掉他。</p>
 <p>（3）</p>
 <p>助理的动作很快，第二天就告诉我陈清名下的财产已经清点完毕，除了他自己这几年在制药研究所的每月工资，其他陈家给他的全部冻结。</p>
 <p>他现在应该是除了那张工资卡一无所有。</p>
 <p>卡是上午冻结的。</p>
 <p>下午陈清就在庄园的门口，堵住我的车。</p>
 <p>家里的房子很多，经常住的就是这座位于半山腰的庄园别墅。</p>
 <p>早在陈清说自己要独立开始，这座庄园他就进不去了。</p>
 <p>滑下车窗，我看到陈清牵着一个女孩的手向我走过来。</p>
 <p>陈清身材修长，常年待在实验室，皮肤比一般女孩子都白。女孩看着文文弱弱的，很白净，嘴角噙着笑，如果不考虑我是陈清的妈，这副画面倒也是挺赏心悦目的。</p>
 <p>「妈，婷婷想来看看你。」</p>
 <p>「她想来看我我就必须看她？你知道我的时间多贵吗？还有，你们到底是为什么来找我，以为我不清楚吗？」</p>
 <p>「还不是为了钱。」这句话成功将陈清脸气得通红。</p>
 <p>也是，从小到大都没为钱发过愁的人，怎么能受得了别人用钱侮辱他。</p>
 <p>说完我示意司机摇上车窗，进入庄园，思考片刻，还是让保安给他们开门。</p>
 <p>别墅距离大门口有将近 800 米的距离，他们两个没有开车过来，小明星还穿了高跟鞋，走过来需要费一番力气。</p>
 <p>站在 2 楼落地窗前，刚好能看到陈清和小明星文婷走到别墅前。</p>
 <p>我清楚地看到了那个小明星看到这栋金碧辉煌的住宅时眼前一亮。</p>
 <p>真实又有意思。</p>
 <p>「吴妈，给婷婷洗一盘草莓，我记得前几天比利时运来了几箱。」</p>
 <p>陈清一进门就很自觉地指挥人伺候他的小心肝儿。</p>
 <p>说着扭头对文婷说：「婷婷，你不是爱吃草莓吗？待会儿我们带走一些。」</p>
 <p>小明星矜持地坐在沙发上，没说话，只是微微一笑，完全一副任由你做主的娇弱模样。</p>
 <p>有些人就是这样，什么都不用做，就会有舔狗把最好的送到她面前。</p>
 <p>我一下楼就看到吴妈在为难，招手示意她下去。</p>
 <p>坐在大厅的主位，佣人极有眼色地倒了一杯水递上来。</p>
 <p>陈清看懂了我的意思，脸色极为难看地坐下。</p>
 <p>「妈，我希望我们能好好谈谈，婷婷真的是很好的女孩，要不是她我可能就死在贵州的山里了，我希望你能接受她，她是我的救命恩人，也是我的福气。」</p>
 <p>在心中翻了个白眼，有你我是真的服气。</p>
 <p>「这么算的话，我生你的时候难产，九死一生，医生都劝我放弃你，可我还是咬着牙生了下来，让你在最好的条件下成长，这是多少人求而不得的，你算一下，你要怎么回报我？」</p>
 <p>这些话还真不是骗人，要是没有我，陈清绝对不能好好地活到现在。</p>
 <p>作为豪门唯一的公子，每年都会有大大小小的「意外」发生。</p>
 <p>陈清脸色有点难看，文婷温柔地握了握他的手。</p>
 <p>「阿姨，您是陈清的母亲，母亲为孩子做的任何事不都是应该的吗？您怎么能用金钱来衡量呢？」</p>
 <p>她有点儿瘦，皮肤很白，说话温温柔柔的，脸上化着淡妆，看样子比陈清小很多。</p>
 <p>她的身上带着些廉价刺鼻的香水味，仔细看脸上还是能看出来一些端倪。</p>
 <p>——科技的痕迹。</p>
 <p>「您也不要因为我，伤害了你们之间的感情。」</p>
 <p>几句话挑拨得陈清感动得差点稀里哗啦，我不禁多看了她几眼，冷笑。</p>
 <p>有点段位，但是不高。</p>
 <p>陈清倒是很护着她，立马表明立场：</p>
 <p>「妈，我觉得你肯定是误会婷婷了，她网上的那些都是——」</p>
 <p>「网上那些都是谣言，她从来没有进过导演房间，每次进去都是在做头发。」</p>
 <p>陈清大概是没想到我会这么说，继续说：「对，还有她脾气很好——」</p>
 <p>「从不耍大牌，网上那些欺负人的都是谣言，是看不惯她太火？」</p>
 <p>陈清脸色有些发黑，似是对我有些生气：「妈，婷婷是一个很真诚的人，她——」</p>
 <p>「她没有打过胎，从不发脾气，人也很温柔。」我故意停顿了一下，假装仔细看了看文婷的脸，「她的脸也是纯天然的。」</p>
 <p>说完好笑地看着陈清脸色又红又黑。</p>
 <p>当事人的脸色倒是没多少变化，丝毫不慌。</p>
 <p>太好笑了，这样的故事我只在年轻时看的小说里见过。</p>
 <p>当时看的时候都觉得脑残。</p>
 <p>当真实发生在我身边后，我才发觉，这已经不是脑残这么简单了。</p>
 <p>小明星垂着眼，好似被欺负了一样，轻声道：「阿姨，娱乐圈确实鱼龙混杂，可我对陈清的真心是无价的，您反对我，看不起我，没关系，可是，陈清是个独立的人，他需要被尊重。」</p>
 <p>我放下手中的水杯：</p>
 <p>「吴妈，给我换一杯清水，这杯茶味有点浓，我快要吐出来了。」</p>
 <p>（4）</p>
 <p>我笑了，茶到我头上了。</p>
 <p>陈清则是宝贝似的看着文婷，生怕我把她怎么样。</p>
 <p>冷笑之余，我的心头泛上丝丝寒意。</p>
 <p>看吧，这就是我养了二十多年的亲儿子。</p>
 <p>辛苦这么多年把他养大，就被这样一个女人给迷得神魂颠倒。</p>
 <p>「说吧，你要多少钱？」</p>
 <p>「阿姨，你怎么能侮辱我们的感情，我对陈清是认真的，我只图这个人，跟他有没有钱没有关系，你怎么能这么看我？」</p>
 <p>小绿茶说着含情脉脉地看着陈清。</p>
 <p>陈清一下子就炸了：</p>
 <p>「妈，不管你接不接受，我都要跟婷婷在一起，我明天就去研究所请假，准备结婚，等结了婚安定下来，我带着婷婷去 C 国交流学习，就当作度蜜月了。」</p>
 <p>「都可以，你开心就好。」</p>
 <p>我爽快地回应道，他们两个都愣住了。</p>
 <p>「那我的银行卡还有房子——」</p>
 <p>「你的？写你名字了吗？那不是我的吗？你不是已经成年了吗？可以独立了，我们陈家虽然现在只有你一个，但是我还年轻，回头我会把你的名字从陈家家谱划去，属于你的你尽管拿走，从现在开始，你独立了。」</p>
