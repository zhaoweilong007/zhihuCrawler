# 如何看待海淀某互联网公司鼓励员工尽量住在公司，在互联网公司上班真的这么拼吗？
- 点赞数：6323
- 更新时间：2020年07月20日07时56分56秒
- 回答url：https://www.zhihu.com/question/407829485/answer/1348953614
<body>
 <p data-pid="7pID-i-6">996盛行的时候，007只是一句夸张的话，007就是一周7天每天24小时的工作，怎么可能实行呢？没想到，现在007还真的被有些公司做到了，现实总是超越我们的想象力。</p>
 <p data-pid="OpHWDu7J">996都不能保证生产效率，007更无法提高生产效率，我敢肯定，这家公司必定有一个焦虑而且狠毒的老板，这个老板提出所谓『不畏加班，不念下班』的口号，其实对于员工的要求就是『不要当人，甘心当狗』，它要的是员工的绝对服从，通过压榨员工所有的时间缓解自己的焦虑，其实它也根本不知道如何经营管理好一个公司，从社会达尔文主角度，这种公司、这种领导难道不应该就被被淘汰吗？</p>
 <p data-pid="W-HdtF6x">我对于996的态度一向是这样：</p>
 <ul>
  <li data-pid="bxP8ZeIQ">老子再也不会去做996的工作。</li>
  <li data-pid="OHyEOEIb">谁要是真的家里穷，要去做996的工作，那么请保证自己赚到更多钱，不要把行业薪水拉低，也算为社会做贡献了。</li>
  <li data-pid="yVzut5ta">愿意接受低薪996的，那是工贼。</li>
 </ul>
 <p data-pid="xggiUBp_">现在对于007的，我的态度是这样：</p>
 <ul>
  <li data-pid="eZKvqVJG">老子绝不会接受007。</li>
  <li data-pid="cq5kw4qc">谁要是真的家里穷，要去做007的工作，那么请保证自己的公司倒闭，毁灭这些公司，也算为社会做贡献了。</li>
  <li data-pid="tSlX8CWs">愿意接受低薪007的，那是奴隶。</li>
 </ul>
