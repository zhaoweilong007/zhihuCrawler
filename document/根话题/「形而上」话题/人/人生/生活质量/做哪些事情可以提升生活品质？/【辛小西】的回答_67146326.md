# 做哪些事情可以提升生活品质？
- 点赞数：8482
- 更新时间：2016年05月23日17时29分56秒
- 回答url：https://www.zhihu.com/question/22347626/answer/67146326
<body>
 <p data-pid="dWDJTVqr">首先，最重要的一点是，<u><b>你希望有所改变，并下决心要好好爱自己。</b></u></p>
 <p data-pid="-1sUVgg-"><b>健康饮食</b></p>
 <ul>
  <li data-pid="3DBooXTe">三餐固定时间吃饭、细嚼慢咽，学会享受美食。我发现慢慢吃饭的人都比较瘦。我自己本身吃饭一直很快，对食物没什么追求，管饱就行。看了《深夜食堂》之后，慢慢体会吃货的幸福，他们很容易从美食上得到满足。</li>
  <li data-pid="3nbOAvI5">少吃多餐。每餐不要吃太饱，胃是可以被撑大的。我就非常容易饿，每次吃很多，吃饭就变成了完成任务。</li>
  <li data-pid="V-1DhvfZ">每天一定量的水果，多喝水。吃水果也有很多美好的方式，可榨汁、可摆盘，把这个过程变成一种享受你自然会坚持下去。我喜欢吃红枣，把枣子放到微波炉里30秒后开水冲泡，为了吃到枣子只能把水都喝完。</li>
  <li data-pid="Rn0x-tRp">尝试学做菜。做菜是很考验耐心的事儿。可以在做菜的过程中磨磨暴躁不耐烦的性子，做一道让自己满意的菜，都是能提高幸福感的。</li>
 </ul>
 <p data-pid="fLKbOV6W"><b>保持运动</b></p>
 <ul>
  <li data-pid="6pPnq-aJ">晨跑。我个人倾向于晨跑，想要做一个能早起的人。下午跑步会因为各种不可控的计划原因被打消，不能坚持。跑步要用好的装备，跑步要用好的装备，跑步要用好的装备。重要事情说三遍！注意跑步姿势！不然会适得其反。</li>
  <li data-pid="jSC9Ny-i">饭后散步。和父母出去走走逛逛。发现美好的事物拍下来，保持对生活的热爱。</li>
  <li data-pid="Kz6GZnio">每天早起拉伸。</li>
 </ul>
 <p data-pid="HWMGFlfk"><b>拿起笔：记录</b></p>
 <ul>
  <li data-pid="RlmP-NNX">买一本好本子，用自己写顺手的钢笔，记下打动自己的文字。高中开始我就有记录文字的习惯，看电影、看书记下精彩的一段。时常翻看，每每都能眼眶发热。（这是自己感动自己）还能练字！（每次看以前的字都有想要好好练字的冲动）</li>
  <li data-pid="0r-5ef45">记录每天的开支。看自己在什么地方花钱最多，才能节省下不必要的开支。</li>
 </ul>
 <br>
 <p data-pid="3WFk8IL9"><b>理性购物：<u>少买东西！</u></b></p>
 <ul>
  <li data-pid="VKdnO83d">整理衣柜，定期淘汰不常穿的+廉价的衣物，多备些百搭款。这过程会心疼！但是，你会明白，现在的痛都是你当初脑子进的水！</li>
  <li data-pid="Q3rRfByx">多逛逛奢侈品区。想想，也就是几个月不买淘宝货可以省下的。当然，你去看看导购的冷脸也能让你小小的下个决心。</li>
  <li data-pid="5s342pdW">护肤品用好的，化妆品最简。化妆品真的是用不完，所以不需要多，也不要买一堆囤货，把对化妆品的热爱放在其他兴趣上。不要盲目崇拜护肤品护理的方案，好皮肤要靠正常的作息和食补，护肤品不要盲目崇拜大牌，适合就好！！我前几年作息不规律，脸爆豆，喝了一个月的杂粮粥，加上用了合适的护肤品，奇迹般的好了。</li>
 </ul>
 <p data-pid="LMqlEFVB"><b>你的问题主要在于：读书太少，而想得太多</b></p>
 <ul>
  <li data-pid="9CJQ_vFI">控制脾气，学会感恩。这是很基本的一点。只有你对世界心存善意，用美好的心态去面对，你才能感受到世界的美好。我本身很容易发脾气，易怒。我有个闺蜜乐天派，我喜欢她不管受了多大伤都还是坚信总会好起来的。这样的心态才能感受美好，不是么？</li>
  <li data-pid="OqGyfgkU">少说话，多干事。是不是话走的比你脑子快？一出口就后悔？那就少说话。有话慢慢说。我的语速很快，前几天去闺蜜家，把她的几岁的弟弟吓哭了。我痛下决心要慢慢说话。语速一快，声音再好听也是巫婆了么 T T.</li>
  <li data-pid="CSJR1Ft8">看好书，观佳影。对象从小看奥斯卡，一直鄙视看连续剧的我，我也觉得他可以在智商和情商上秒杀我。不说有没有必然联系，总之他看起来逼格高多了。</li>
  <li data-pid="TQvvj0Vj">有节制。不要放纵自己的欲望。狂吃、狂购物、狂笑、狂哭。事情一过度就变味了。</li>
 </ul>
 <p data-pid="dHzJaqMg"><u><b>做一个干净的人。</b></u>这是我一直努力的方向。</p>
 <ul>
  <li data-pid="dODLYdx1">一回家就换居家服，卸妆。<br></li>
  <li data-pid="vZQip8ev">一周大扫除一次：擦桌子擦柜子、清理冰箱、整理衣柜。<br></li>
  <li data-pid="K0zergQ3">定期清洗化妆刷和海绵蛋。我一般刷子一两星期一次，海绵蛋一两天。看到有颜色就难受。</li>
  <li data-pid="G7KG9nzU">学色彩搭配。我喜欢莫兰迪色调，慢慢搭配衣服。穿纯色有质感的衣服。<br></li>
  <li data-pid="EYcKfdH2">统一家具的颜色。现在家里都是原木的家具，纯色的床单。回家感觉很幸福呢。<br></li>
  <li data-pid="v9imsMSY">换了白色的床单，一星期换一次。（感谢 <a data-hash="1c1c6809820d8e5c9d39ef520d3ab93b" href="https://www.zhihu.com/people/1c1c6809820d8e5c9d39ef520d3ab93b" class="member_mention" data-tip="p$t$1c1c6809820d8e5c9d39ef520d3ab93b" data-hovercard="p$b$1c1c6809820d8e5c9d39ef520d3ab93b">@月月酱</a> 提醒，要一个星期换一次床单。。很早以前不知道哪本书上看到枕套是必须一个星期换一次的，对于床单好像没有那么严格。当然，能做到勤洗是最好的。还有，被子多晒太阳！）<br></li>
  <li data-pid="nRjKI5hW">尽量要眼里的东西少，东西收纳到柜子里。这样才会看起来干净简洁。东西一多我就懒得整理了，这样只会恶性循环。<br></li>
  <li data-pid="Hab0EutT">一起床就整理床。放上好看的抱枕，铺好盖毯。想想每天躺在好看的床上心情就美哒哒。当然，整理床也是防止你再睡个回笼觉。</li>
  <li data-pid="GgmZJ3ul">对待好你的头发和指甲，我不擦指甲油，不染发。以前一直烫发，太伤头发，去剪了中发，出门用博朗负离子卷发棒，吹头发喷防烫喷雾，头发无伤。<br></li>
 </ul>
 <p data-pid="4DtIZbzU"><b><br><br>
   享受生活</b></p>
 <ul>
  <li data-pid="LRvqTYa2">泡澡、闻香。美好的气味也能使人的心情变好。现在的天气，我用L'Artisan Mure Et Musc,黑莓甜而不腻的味道加上黑醋粟，很干净的气味。雨天用Stella Mccartney 同名香水，抑郁的玫瑰味。在家，阳光好的时候，靠窗沙发上看书，喷Bvlgari的红茶，酸酸的味道很有意思。当然我最爱的，治愈我的是Diptyque Philosykos，无花果的奶味一出来，瞬间美好了。</li>
  <li data-pid="YiQ4F5_W">我的书桌上只放着笔记本和一个香薰机（插usb的），平时开电脑的时候会点，根据自己喜好滴精油，我比较喜欢柠檬、杜松。</li>
  <li data-pid="4jTAG5Im">享受阳光。我常坐在靠窗的小沙发上，开着音响，听着音乐（例如：Greg Laswell的And Then You、james hoffman的cut your losses这种类型），让阳光只晒到下半身，好像一伸手就能抓到。感觉音乐慢慢会侵入你的大脑，然后直抵你的心里，和阳光一样温暖。这时候我经常什么也不想，就是这样静静的享受这样的时间。很幸福，真的，觉得自己更容易满足了，仿佛只是为了星期日午后的阳光而活！这样也够了不是么？</li>
  <li data-pid="jEqMa-3U">种植物。我在正对床的柜子上放了一盆植物。每天早上醒来就看到它，对新的一天有了不一样的期待。<br></li>
  <li data-pid="1cbeWXab">擦身体乳、穿袜子。一洗完澡就擦身体乳，亲测身体乳真的能白！！！一白遮百丑。天气冷了，穿袜子能保暖，女生这点很重要，我一回家就换上软软的袜子，只在睡觉脱掉。</li>
  <li data-pid="6q8NOxuV">冥想。准备要学的技能。冥想的好处很多不赘述了。自省。</li>
  <li data-pid="9qEaxKUj">如果你不能照顾好小动物，就不要养了。养动物不只是想要一个玩具陪着玩，你要确定自己是否真的有能力去照顾。</li>
  <li data-pid="Pe6wSp23">有一个知冷暖的爱人。谈一场很久的恋爱，胜过不断的分分合合。我和对象在一起六年了，每天保持视频，等他回国。</li>
  <li data-pid="wlHx8RML">有不同类型的朋友，保证一点：你在她们面前不用伪装。定期和她们聚聚，给她们正能量，你会感觉自己被需要。我有一个学音乐的闺蜜，教我如何听古典乐；一个学园林设计的闺蜜，欣赏她的设计和想法；学编导的闺蜜，喜欢给我拍照……你旁边的人会给你带来快乐，你也是。</li>
  <li data-pid="TCBMcXPb">读诗、听古典乐。好的文字和经典的音乐之所以经典不是没有道理的。我床头柜上放着一本诗经和博尔赫斯的诗集，睡前会看一篇。当然你可以选择一本自己喜欢的书（千万不要是小说！看了就不睡觉了）。坏心情不带到床上。</li>
 </ul>
 <p data-pid="PGXBQXZJ">最爱博尔赫斯的诗：</p>
 <p data-pid="NTY0aV_H">『我给你一个从未有过信仰的人的忠诚。</p>
 <p data-pid="5B7xQFDS">我给你我设法保全的我自己的核心——不营字造句，不和梦交易，不被时间、欢乐和逆境触动的核心。</p>
 <p data-pid="wr85RzfM">我给你早在你出生前多年的一个傍晚看到的一朵黄玫瑰的记忆。</p>
 <p data-pid="QXqoEo4V">我给你关于你生命的诠释，关于你自己的理论，你的真实而惊人的存在。</p>
 <p data-pid="mXJFuwbE"><u> 我给你我的寂寞、我的黑暗、我心的饥渴；<br>
   我试图用困惑、危险、失败来打动你。</u> 』</p>
 <br>
 <p data-pid="ovPZLVw3">暂时这么多。最后强调一点：<u><b>热爱你每一次的改变。</b></u></p>

 <ul>
  <li data-pid="i5Gpjp3W">跑步姿势</li>
 </ul>
 <p data-pid="YcOKKnWP">我是参考这篇：</p><a href="http://www.zhihu.com/question/20693323/answer/44093009" class="internal">正确的慢跑姿势是怎样的？ - 谭舟匀的回答</a>
 <br>
 <p data-pid="aSY-SvPY">总结起来就是：大腿发力，腹肌发力。开始会找不到发力点，可以多观察下旁边跑步的人，会发现基本都是错误的姿势。</p>
 <p data-pid="_d7WCjpi">PS：刚晨跑回来，天气舒服的想哭</p>
 <ul>
  <li data-pid="ckk4Z-pl">身体乳。强调：只要用身体乳都会白，而且皮肤会变得超好。冬天必须用。我皮肤很容易过敏，坚持身体乳过敏好了很多。我最爱的是body drench深层滋润那款，海淘的，是我用过皮肤最滋润的身体乳。夏天用asdm那款美白身体乳。<b><u>要想白，就要注意防晒！</u></b></li>
  <li data-pid="Lcc7FlfO">杂粮粥：高粱、玉米碎、小米、芸豆、薏仁米、红米、黑豆、芝麻、核桃、燕麦、麦仁、百合、芡实，然后凉些了加蜂蜜。痘皮建议用用ipsa自律乳液和流金水，皮肤很快稳定下来。（谢谢 <a data-hash="863f5857371c079a48cbafc96c56f0a6" href="https://www.zhihu.com/people/863f5857371c079a48cbafc96c56f0a6" class="member_mention" data-editable="true" data-title="@starfish岚" data-tip="p$b$863f5857371c079a48cbafc96c56f0a6" data-hovercard="p$b$863f5857371c079a48cbafc96c56f0a6">@starfish岚</a> 提醒：女性为了乳腺健康，乳腺专家建议蜂蜜和鸡都要尽量少吃。因为雌激素含量较多。 ）</li>
  <li data-pid="lNBXL8Cm">头发护理。这个有点多了。发质一定要好。宁愿短发无分叉，也不要干枯的一头长发。</li>
 </ul>
 <p data-pid="3VJm9Gct"><b><u>护发步骤：</u></b></p>
 <ol>
  <li data-pid="LNGsmDNw">aveda梳子梳顺</li>
  <li data-pid="qKR6qTBt">无硅洗发水洗头</li>
  <li data-pid="DBrohp6V">发膜当护发素用（根据你头发好坏程度适当调整。现在我的头发没有分叉的问题，觉得头发干了才会用发膜）</li>
  <li data-pid="1FaHZWut">擦干头发用防烫喷雾，我用tresemme美国红色版。用松下水离子吹风机吹头</li>
  <li data-pid="D0jk-n1i">7成干擦发油</li>
  <li data-pid="iYwKfceX">特殊护理：一周用一次露得清的去残留洗发水</li>
 </ol>
 <p data-pid="FUVDtUs-">这样的护理让我半个月，头发从稻草变好，一点都不夸张。</p>
 <p data-pid="5_B3LDzx">（具体产品看另一个回答）</p>
 <ul>
  <li data-pid="BX4bvT69">关于坚持。早上醒来在床上挣扎了5分钟，今天不跑了？后来还是起来了。动力？</li>
 </ul>
 <p data-pid="XWHOOQXJ">你想不想瘦了？你想不想变美了？说好的坚持呢？反正都跑了那么多天不要功亏一篑！你还在打卡呢。</p>
 <p data-pid="olMEPR2m"><u><b>私以为能坚持下去最大的动力是，前面的努力让你有改变了。</b></u></p>
 <p data-pid="qZAgbhtE"><u>所以，记录自己每一个改变的细节：</u>变白了？皮肤好了？心态好了？更容易开心了？</p>
 <p data-pid="TqQ_Pa_p">不要着急，给自己21天，看看其中的一个好习惯有没有养成。</p>
 <p data-pid="dfAfw6P2">我这样告诉自己：已经浪费了二十几年，要好好想想自己想过什么样的生活，什么品质的生活了。还不晚，不是么？</p>
