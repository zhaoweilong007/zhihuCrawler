# 什么是「极简主义」的生活方式？
- 点赞数：6883
- 更新时间：2014年06月14日13时13分19秒
- 回答url：https://www.zhihu.com/question/23703464/answer/26551904
<body>
 <p data-pid="5R5vW0fu"><b>（本文将持续补充、更新）</b></p>
 <p data-pid="NJQdyzoY"><b><br><br>
   极简主义生活方式，是对自身的再认识，对自由的再定义。</b></p>
 <ul>
  <li data-pid="yYCTAnD6">深入分析自己，首先了解什么对自己最重要，然后用有限的时间和精力，专注地追求，从而获得最大幸福。<br></li>
  <li data-pid="MVXFtrLW">放弃不能带来效用的物品，控制徒增烦恼的精神活动，简单生活，从而获得最大的精神自由。</li>
 </ul>
 <br>
 <p data-pid="OoGFlQf_"><b>欲望极简</b></p>
 <ul>
  <li data-pid="6zTG0jvU">了解自己的真实欲望，不受外在潮流的影响，不盲从，不跟风。</li>
  <li data-pid="GSJf_vVA">把自己的精力全部用在自己最迫切的欲望上，如提升专业素养、照顾家庭、关心朋友、追求美食等。</li>
 </ul>
 <p data-pid="gonjYMNF"><b>精神极简</b></p>
 <ul>
  <li data-pid="nhhP_kLl">了解、选择、专注于1-3项自己真正想从事的精神活动，充分学习、提高。</li>
  <li data-pid="0NvHZbIl">不盲目浪费自己的时间与精力。</li>
 </ul>
 <p data-pid="9Rqc4s8I"><b>物质极简</b></p>
 <ul>
  <li data-pid="2OMzuYpg">将家中超过一年不用的物品丢弃、送人、出售或捐赠。比如看过的杂志、书，不再穿的衣服，早先收到的各种礼物或装饰品。<br></li>
  <li data-pid="83Hrhnch">明确自己的欲望和需求后，不买不需要的物品。</li>
  <li data-pid="PIoa7a41">确有必要的物品，买最好的，充分使用它。</li>
  <li data-pid="y1_jZBpa">不囤东西，不用便宜货、次品。</li>
  <li data-pid="HJyvJghW">用布袋，代替塑料袋和纸袋。</li>
  <li data-pid="K6CgH9Fv">用一支好用的钢笔，替代堆积如山的中性笔。</li>
  <li data-pid="46oYK8j9">用瓷杯、钢杯代替纸杯。</li>
  <li data-pid="gytxSe5u">用电脑写东西，少用纸。养成纸质文件扫描、存档的习惯。</li>
  <li data-pid="1Mzhu9Z8">整合、精简电源线、充电设备。不重复购买电子产品。</li>
  <li data-pid="eWzJrRYn">精简出门行头，只带“身手钥纸钱”。</li>
  <li data-pid="NN-87pNV">精简银行卡，仅保留一张借记卡，一张信用卡。</li>
 </ul>
 <p data-pid="9Cb1c_eN"><b>信息极简</b></p>
 <ul>
  <li data-pid="7CvhCXmS">精简信息输入源头，减少使用社交网络、即时通讯。少看微博、朋友圈。</li>
  <li data-pid="Yq5bX22y">定期远离互联网、远离手机，避免信息骚扰。</li>
  <li data-pid="lmAbAAqd">不关注与己无关的娱乐、社会新闻。</li>
  <li data-pid="SuLYZTdl">精简电子邮箱数量。</li>
  <li data-pid="XWJ4qHc7">关注的知乎用户少而精，宁缺毋滥。时间线干净。</li>
  <li data-pid="5_Jp1Ojw">APP使用少而精，删除长期不使用的应用。</li>
 </ul>
 <p data-pid="i-_67d6O"><b>表达极简</b></p>
 <ul>
  <li data-pid="669ymZ4r">写东西、说话，尽可能简单、直接、清楚。</li>
  <li data-pid="8MVRwz86">多用名词、动词。少用形容词、副词。</li>
 </ul>
 <p data-pid="9F6fUxA4"><b>工作极简</b></p>
 <ul>
  <li data-pid="Isngaoma">使用有效的GTD方法，不拖延。</li>
  <li data-pid="uHS6It0J">及时清理电子邮件，不要让它们堆积起来。</li>
  <li data-pid="2DmXgCM6">一次只专注做一件事，尽可能不做Multi-task。</li>
 </ul>
 <p data-pid="bJlmuxe9"><b>生活极简</b></p>
 <ul>
  <li data-pid="fb0kVAD0">慢生活。</li>
  <li data-pid="q3LUeWzE">不做无效社交。</li>
  <li data-pid="jyu5lQa9">锻炼。</li>
  <li data-pid="ZkeYCtdi">穿着简洁、不花哨。</li>
  <li data-pid="c1IhSBsc">少吃含有添加剂的食品。</li>
  <li data-pid="MFN1cP02">喝白水和纯果汁，不喝添加了大量化学成分的碳酸饮料和果汁。<br></li>
 </ul>
 <p data-pid="seUro5AC"><b>实践极简主义的方法、角度有很多，关键是要行动起来。</b></p>
