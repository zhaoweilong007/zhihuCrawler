 <h2><b>超级万字长文，从护肤-健身-饮食-穿衣-化妆-收纳-清洁-学习-自由职业……方方面面历时六年的经验，看完可以改变你对极简的概念</b></h2>
 <h2>极简是一种思维模式：遇到任何事情都想如何系统化，在搭建好高质量系统之后，用最少的精力去维护和执行它。</h2>
 <p data-pid="-UMkaMLr">护肤极简阶段一</p>
 <ul>
  <li data-pid="6MHgYA-P">从高中时我就踏上了科学护肤之路，我开始关注一些果壳的护肤小组和互联网的一些论坛，当时消费主义还并没有大行其道，大概是一四年左右。我当时做了非常多的功课，了解了果酸A酸烟酰胺等基本的成分，并且对防晒有了深刻的认知。</li>
  <li data-pid="6cG3xV2G"><br>
   成为成分党后我不再关心任何吹得天花乱坠的护肤品，因为我知道对我而言最重要的，是他背后的成分表，我对所谓的海洋之谜、鱼子酱精华也毫不动心，因为我知道可能简单刷酸带的效果可能远比这些所谓的贵妇护肤要好。而当消费主义大行其道，伊美也成了各大博主的重要鼓吹之一，但实际上，我在查任何医疗美容之前我都会在谷歌学术上调查相关的论文实验，马上就可以判断到底是噱头还是安慰剂甚至是毁容产品。</li>
  <li data-pid="sycKxKKH"><br>
   我做了各种功课之后发现目前最有效的抗老医美只有单极射频，我便买了家用仪器tripollar，因为我的皮肤除了肤色问题，没有任何瑕疵，我开始尝试各种美白猛药，完全没有耐心的我使用过最有效的是黑色素阻断剂对苯二酚霜</li>
  <li data-pid="ZxkbpGbd"><br>
   实际上，如果建立了这种科学观念。真的会变得非常简单，起码你对市面上95%的产品不再动心，而你每用一样护肤品，你都知道会带来什么直接效果。</li>
  <li data-pid="L3KKssmA"><br>
   在各种猛药的冲击下，我的皮肤的确变好了。这种改变虽然肉眼可见，但是因为我三天打鱼两天晒网，总是不能长期维持，我又陷入了新的纠结当中。</li>
  <li data-pid="GtiMPb4f"><b><a href="https://link.zhihu.com/?target=https%3A//www.douban.com/group//topic/30217090/" class=" wrap external" target="_blank" rel="nofollow noreferrer">启蒙我的，至少看过十几遍的学术派护肤帖子</a></b></li>
 </ul>

 <p data-pid="ekEzivBb">护肤极简阶段二</p>
 <ul>
  <li data-pid="73czanxr">虽然比很多女孩子的护肤品都少，但是仍旧还是有四五个步骤，每天晚上光是护肤就要40分钟，总希望把所有有效的成分都堆到脸上。连防晒霜都不能坚持涂总是三天打鱼两天晒网，哪怕用了猛药，皮肤也并没有什么革命性的改善。</li>
  <li data-pid="ZHkc0Vip"><br>
   一直到今年夏天我才改变了自己的观念。我意识到了无论再怎么极简，体验感必须要好，每次做必须要是‘享受’，而不是‘坚持’。这样才会有长期的可能性。如果每次都要提前一小时上床耗时耗力，在种繁琐的情况下是无法享受的。</li>
  <li data-pid="ptsq-5iY"><br>
   所以我便砍掉所有的东西，只剩下之前提过的对苯二酚霜和麦色滤专利的防晒霜（好多人想知道我全部的护肤品，这就是我全部的护肤品我已经写出来了，除了卸妆膏用来卸防晒之外，我没有任何其他的护肤品，甚至没有保湿霜的）。又砍掉了早已搁置许久的家用美容仪，采用每年去医院做正规医美的方式。都是单极射频原理，家用美容仪tripollar换成了医用美容thermage.</li>
  <li data-pid="6sgdiZai"><br>
   这次我真的做得很到位，因为砍掉90%的护肤步骤，操作起来简单太多了。所以我可以每天都执行而不是断断续续了，效果一下子就出来了。</li>
  <li data-pid="N4x0arIo"><br>
   每天起床第一件事情就是防晒。冬天时，妈妈来看我的时候被我的肤色给震惊到了，这大概是我活了20年唯一一次这么白和嫩吧，我妈在路边的时候都说我皮肤晃得刺眼，和小孩子的皮肤一样。</li>
  <li data-pid="7yH1VZWM"><b><a href="https://link.zhihu.com/?target=https%3A//www.douban.com/note/154595195/" class=" wrap external" target="_blank" rel="nofollow noreferrer">麦色滤的防晒霜原理和品牌推荐</a></b></li>
 </ul>

 <p data-pid="oS3aoHx7">健身极简阶段一</p>
 <ul>
  <li data-pid="hLDQ77gw">一开始接触健身，其实我没有想健康，只是单纯的想要减肥而已。青春期发育，我的身高没有怎么变高，体型却开始变宽。一四年开始我成了知乎重度用户，在它的洗脑下，我对健身有了深入的概念。</li>
  <li data-pid="IRuOcHn1"><br>
   当时接触的第一个理念不是有氧运动，而是HIIT。这种高强度间歇训练的概念非常诱人，我第一次接触到的相关操作是跳大名鼎鼎的健身操：insanity.传说跳完地狱级难度的Insanity可以两个月内瘦30斤。</li>
  <li data-pid="Vy2Ihu_p"><br>
   当时还是个高中生求生，妈妈买了各种运动裤运动内衣瑜伽垫，并且在家里的墙上贴了镜子跳操。</li>
  <li data-pid="trYt69QV"><br>
   hiit让我知道了，原来不光是护肤，任何事情都有最直接简洁的方法，策略永远要比努力重要，同样的努力，跑步和各种有氧运动还有市面上流行的女士健美操对比HIIT而言不堪一击，所以HIIT现在在世界范围内渐渐走向主流的心肺锻炼运动。<br>
   但是因为没有任何健身基础，体能非常差劲，每次跳一会儿，累得不行无法坚持。Hiit是为数不多的好健身方式之一，但是对心肺能力的要求太高了，普通人没有运动习惯的根本无法跟上，甚至会损伤膝盖。没有意志力的我又开始寻求新的方式</li>
  <li data-pid="hABRZSyP"><b><a href="https://link.zhihu.com/?target=https%3A//www.glofox.com/blog/insanity-vs-hiit-whats-the-difference/" class=" wrap external" target="_blank" rel="nofollow noreferrer">Insanity vs. HIIT: What’s the Difference?</a></b></li>
 </ul>

 <p data-pid="zNFCHRZO">健身极简阶段二</p>
 <ul>
  <li data-pid="UVmgU0Ma">后来又开始下载软件做一些温和的徒手动作，比如卷腹之类的，嗯，大家都知道，这只是另一种三天打鱼两天晒网，因为见效缓慢，而且没有配合饮食，很快就放弃了。说的就是你keep我觉得没有多少人可以坚持的下来。包括其他任何手机健身软件其实都是一个性质，除非意志过人，不然很难让人真正改变。</li>
  <li data-pid="jIhZYO2N">在受到打击后，我觉得自己之所以不能坚持，是因为仪式感不够，没有健身氛围。来了美国之后一直想要去健身房，好好锻炼一下自己。</li>
  <li data-pid="ivF-taNn">一开始的确非常高兴，因为连锁健身房人声鼎沸一大堆大妈大爷过来健身了，甚至还有些臭。在大家都这么努力的情况下，我也要努力呀，于是开开心心办了卡。</li>
  <li data-pid="9q1cIuZQ">没想到一个月里面也只去了四五次和想象中的天天去健身房完全不是一个概念。因为启动成本太高，每次去健身房都必须要穿上各种专业的设备跑鞋，因为没有车，还需要打车到那里，白天有课的时候只能晚上去，晚上天黑又冷。每次去健身房都不怎么情愿，因为去的太晚，有些时候他们还要关门。</li>
  <li data-pid="E2cfsw0N">所以我咬了咬牙，继续我的健身探索。</li>
 </ul>
 <p data-pid="asz7YwBG">健身极简阶段三</p>
 <ul>
  <li data-pid="iAdS-ODB">在微博上有人晒了一个50岁的女人健身的照片 ，她拥有少女一样纤细的背部和漂亮的肌理，这时候我意识到，健身神奇的让人留住了青春，后来看了很多健身大神的视频、照片之后，意识到这不是少数情况，而是非常普遍的。我已经意识到健身不只是为了减肥，而是为了抗老，无论是外表上的还是精力上的，健身往往能让人维持在身体机能最好的状态，我不想人到中年变得虚肥无力。贪婪的我想要永远少女。</li>
  <li data-pid="bUhNL4-1">我意识到自己也许只是缺一个专业的私人教练而已，我幻想着，只要有个专业人员来指导，我必定能够合理健身一日千里。</li>
  <li data-pid="gz6Z-fcY">但是我却踏上了一条被坑的道路。我兴冲冲去了美国大型连锁店LA Fitness，雄赳赳气昂昂地对着店员说，我要私人培训课程。一米九超级帅的店员两眼放光像看到羊一样。把我拉到了一边，跟我口吐唾沫讲了两个小时。我都快听到不耐烦了，终于结束了，离开时我很开心地打电话告诉男朋友说我最近变漂亮了，所以店员都要跟我搭讪。回到家后还是美滋滋的，</li>
  <li data-pid="GOMGGF4f">顺手查了这个健身房的私人教练一查才知道原来像这种大型连锁店，他们压榨私人教练，所以工资非常非常低挣的都是血汗钱。好的健身教练早就跑了，剩下那些滥竽充数的一点也不值。我傻眼了，扭扭捏捏发了短信说明天的预约取消。我以为对方是贪图我的美色，但其实是贪图我口袋里的钱。</li>
  <li data-pid="xSuFw5Aj">这时候的教训是，如果一定要私人教练的话，一定要找那种小的私人的健身房，而不是大型连锁的。价格其实是一样的，一般在美国这里是40-60美元一节，但是前者是物有所值。</li>
 </ul>
 <p data-pid="OiRzUGJC">健身极简阶段四</p>
 <ul>
  <li data-pid="nCytLUmh">我的私教梦破碎一半之后，我暂时不敢再找私教了，于是我在youtube上面找了很多健身博主。</li>
  <li data-pid="OwEC8eMJ">但是意外惊喜是这次我终于找到了一个超级大神级别的博主。ATHLEAN-X，他总是把每个动作都拆分出很多科学原理来，深入浅出，对于小白和高手都非常友好。他的很多视频国内有搬运，并且翻译成了中文。非常全面而免费。他在youtube上拥有全世界最多的粉丝，拥有无数信徒。试问哪个私人教练能有他这样的能力呢？这个时代，利用互联网才是实现极简的最好途径，但是一定要注意甄别，找到心目中最好的一位就紧跟他一个人系统学习，不要乱花渐欲迷人眼。</li>
  <li data-pid="qmGdHH4U">我这次不再去健身房，而是去了超市买了用来负重的弹力带、哑铃和健腹轮，在家跟着Jeff的视频练习。</li>
  <li data-pid="GdFSdPYZ">非常诚恳地说，这是我见过最容易坚持的方法，而且因为有合理的负重，不是徒手，可以达到70%的健身房的效果。</li>
  <li data-pid="9PtvjQPw">因为我最近才开始，这样目前的收获就是耐力心肺力量变强了，并且运动之后肌肉不那么酸痛了，腰围小幅度缩水了。最关键的是非常好坚持，减少了大量的启动成本，并且循序渐进。</li>

 <p data-pid="I6tXiNo1">饮食极简阶段一</p>
 <ul>
  <li data-pid="ykepGl5j">刚来美国的这段时间天天都吃披萨和速冻的半成品，比如意大利面或者是肉酱，这些半成品在沃尔玛非常便宜，但这也是美国胖子的根本源头，这些加工食品里面放了大量的玉米糖浆来代替普通的白糖，所以热量非常的高，脂肪往往也过度。所以我在第一阶段胖了10斤。</li>
  <li data-pid="cp-UA1Yd">开始点外卖和吃餐馆维生，但是，美国物价人工方面极度高昂像我还没有独立生活之前，确实无法承担这样高的支出，何况并不健康，种类也少。体重一直维持不下。只能挣扎着自己做饭，因为胖了已经不敢再继续虐待自己的身体。</li>
  <li data-pid="Y8iZwjsU">但是一个人住每天做饭非常痛苦，光是买菜到切菜做菜洗碗就让人崩溃。我突发奇想没几天做一顿才这样吃的剩饭总比吃垃圾食品健康。</li>
  <li data-pid="ErgoKWmz">可问题是实在没有做菜天赋做的东西，千篇一律货又是黑暗料理厨艺非常不稳定，哪怕吃新鲜的都非常折磨人，何况是剩菜。</li>
  <li data-pid="-7YfRj35">又开始偷懒，每次都去买一大堆中国零食不好好吃正餐，因为正餐实在难以下咽。不过这段时间在断了美国饮食后，我迅速地瘦回了原来的体重。只是和健康、快乐毫无关系，甚至得了胃病，有了口臭，虽然男朋友不嫌弃我，但是我简直是奔溃的。</li>

 <p data-pid="zqfQ_arY">饮食极简阶段二</p>
 <ul>
  <li data-pid="rDRE48Pw">我开始对饮食120%的注重，我去上了一学期的营养学课，并且看了无数关于食品的纪录片。简单而言，食品的比例非常重要，同样卡路里的食物，不同的比例导致的体型是非常不一样的。食物有三种类型，一种是淀粉类，一种是纤维类，一种是蛋白质（脂肪）类，增肌的话要控制蛋白质50% 纤维25% 淀粉25%的比例，减脂的话是50%纤维 25%淀粉 25%蛋白质比例。</li>
  <li data-pid="KehuoSio">厨艺苦手的我不得已开始研究吃冷冻食品，你可能会问冷冻食品，不又回了第一阶段的超级不健康状态吗？实际上，这次我对冷冻食品的挑选有了一些不同。首先，我只会选择纯原料的，而不是过度加工的冷冻食品，比如说已经处理好煮熟了的的手撕鸡胸肉、白虾、午餐罐头，中国超市卖的急冻雪里红、白笋、拨好煮熟的毛豆，淀粉类，我也只买一些纤维粗粮馒头、或者一些稍微煮个3、4分钟就可以吃的馄饨饺子，每看一个包装食物之前，我都会好好的看背后的成分表，标签确认糖分（尤其添加糖）和反式脂肪是低的，纤维和蛋白质含量是高的。</li>

 <p data-pid="yKJeqiFM">清洁极简阶段一</p>
 <ul>
  <li data-pid="Z5EeRqlG">清洁实际上是建立在收纳的基础上，如果你拥有的东西足够少，那么明亮简洁是一定会有的。如果你居住的场合只有一点点家具而不是被杂物堆积的话。其实你根本就不需要花什么精力去清理它。这个只要极简进行到一定程度的人肯定都会深有体会。</li>
  <li data-pid="pyeRyTbn">曾经的我因为知乎影响对电子家居生活非常痴迷总希望能建立起一套家务系统解决80%的杂物，但是在拥有自己的一套房子之前，不可能大规模的买这些电子家具，但是到了美国之后我才发现原来大部分美国人根本就不做家务。因为他们哪怕是租的房子里，都会在铺满地毯的房间，烘干机，洗衣机，洗碗机，小到电动牙刷都是人人普及的。</li>

  <li data-pid="eiNYo48o">第二是烘干机，你可以把所有衣服中洗衣机里抱出来，然后直接扔到烘干机里节省了大量时间，在1小时之后就可以全部拿出来：没有任何褶皱，同时高温消毒后，带着一股温暖的感觉。因为没有了晾晒的需求，所以可以省下整个阳台的空间，将阳台做成封闭式的，连接客厅增大客厅面积。同时也不需要熨斗或者挂烫机类产品，因为从烘干机中拿出来的衣服没有褶皱。</li>

 <p data-pid="TWbIFjAd">自由职业极简阶段<b>一</b></p>
 <ul>
  <li data-pid="bR0Az4Tn">我男朋友毕业后进了五百强后，一年后却和一家小的创业公司签订合同，为他们远程遥控工作，他告诉我他那年被朝九晚五和滞后的人际关系给压倒了，他终于意识到原来为什么那么多大人都活得那么沉闷。他不想自己慢慢成为他曾经瞧不起的人。而我的规划是在毕业之后也做自由职业。这是我从初中时开始的梦想，我总是羞愧告诉人，我想做一个网文小说作者，我知道很多人对这个职业会有偏见。</li>
  <li data-pid="J2WWHO8b">我本来大学入学时想选的是计算机专业，但是我男朋友听到以后却眼睛亮晶晶的，他说，如果我爱它，就应该把所有精力都投入到它上面，而不是做自己不喜欢的事情。我的男朋友极度热爱自己的职业，他做的技术美术，主要工作内容是编程和3d美术，他经常说他的工作就像他的小宝宝一样，他随时随地带着电脑，离开二小时都会开始焦虑。他一直都很喜欢挑战，也不害怕困难，因为热爱，所以他能坚持到现在。</li>
  <li data-pid="oGmemUwU">我在他的洗脑下大学放弃了竞争激烈的计算机专业，而选择了写作。</li>
  <li data-pid="ELZQJPRT">不知道多年后的我会不会后悔，但是我想，我是一个成长型的人，每一年我的智力和能力都会提升。只要我一直学习、尝试、总结、找策略、建立高效工作流，我总能够靠网络小说养活自己，直到有一天实现财务自由。</li>
  <li data-pid="nbxpaUZj">男朋友的阿姨是大学文学系的教授，她听到我的想法后第一反应是我最后99%会成为一个编辑结束。她的断言让我更加憋了一股劲，但是在每天的课堂上的写作锻炼下，我渐渐地没有那么迷茫了，因为我知道只要每天去做，总有一天厚积薄发。如果我现在学的是计算机专业，无论是压力超大的课程还是毕业后996的福报，大概都会让我崩溃吧。</li>

 <p data-pid="8RThVteB">自由职业极简阶段二</p>
 <ul>
  <li data-pid="_YUwes1Q">首先做自由职业最关键的就是要出门。</li>
  <li data-pid="sTTj-FAd">虽然只是一句轻飘飘的话，但是很多自由职业者大概都有沉痛的经历，日夜颠倒、作息混乱，然后逐渐非人化蓬头垢面，不想再出门。到后来恨不得还是找回朝九晚五的工作，起码有正常的生活和社交。我并没有做自由职业长久经历，起码到现在还没有，可是我曾经休学过一年，我现在仍旧有很重的出门恐惧症。</li>
  <li data-pid="iUmXBP0T">对于一个自由职业者而言，唯一能让你有社交和正常作息的就是在白天去咖啡馆像一个普通上班族一样工作。</li>
  <li data-pid="uLv8SQdy">为什么是咖啡馆，而不是图书馆？因为每个区的图书馆往往只有一到两个，而咖啡馆却遍布整个区域。每天都可以去不同的咖啡馆转换心情，而且总有离家近的，不像图书馆很可能需要很多交通。咖啡馆往往人声鼎沸，有种生活的烟火气息，而且会有免费Wi-Fi空调，工作的时候不会像在图书馆里那么孤单。</li>
  <li data-pid="nNkH-Qiy">咖啡馆点一个咖啡，可以坐四个小时，不过尽量去连锁店而非私人小店。等我假期没有课的时候或者是白天没有课的时候，我也会经常跑到星巴克拿着电脑坐一下午。不过，自由职业如何接触新的人群，如何建立友谊不至于那么孤单。是我仍旧在探索的。</li>
 </ul>

  <li data-pid="n5a9uNIu">我最终的改变是两方面的，一是建立自己的工作流，所有工作都高效化效率化。另一方面一是建立起可以反复使用的每日日程。<br>
   我的工作流：记录工作时间的timing→故事细纲scapple→语音输入正文 →根据读者反应修改正文</li>
  <li data-pid="LsY_Zlur">timing的好处就在于它可以输入关键词，随时随刻自动统计你关键词下使用电脑的每一分钟每一秒。对于一个所有工作都在一台笔记本电脑上完成的人，它可以非常精准的，将我所有有效的工作时间都统计起来。我每天给自己规定了自由职业的时间，每天无论是做什么内容的写作，一定要够有效四个小时的量。我不需要再投入任何精力去计算和记录，只要打开软件就知道自己今天是否达成目标。</li>
  <li data-pid="YbTJTErI">建立反复使用的每日日程后，我发现文字的日程并不如图片的有效，用图片做的日程表是非常直观的，并且图片做的日程表，还可以安排时间结构。</li>
  <li data-pid="hgLZicZU">在工作流和日程的帮助下，我现在每天非常轻松的，可以解决一切，同时我意识到了自动记录的重要性，而精准是自动记录的前提。自动记录加精准数据，可以解决人生很多的麻烦。我今天刚下单订了一个睡眠的戒指，因为我的失眠非常可怕，我之后过一两个月会再来更新这一部分的体验。</li>
