 <h2>从思考的深度来看，我把思考分成3个层次：</h2>
 <h2>☆ 第一层：逻辑思维</h2>
 <p>就是你思考至少是得符合逻辑的，但就算是在这一层，也依然有绝大部分人不能达标。</p>
 <p>听着好像挺简单，不就是逻辑思维吗？</p>
 <p>我们来测试一下：</p>
 <p>比如，你看如下的言论有无逻辑？</p>
 <ul>
  <li>“你别看 XXX 捐了多少钱，他只是为了逃税罢了...”</li>
  <li>“她穿得那么暴露，活该被色狼盯上...”</li>
  <li>“你说他打球不行，你行你上啊...”</li>
 </ul>
 <p>我们生活中，经常会听到这类的神逻辑，乍一听，好像是那么回事，可是细细一想，这逻辑好像有点不太对劲，那么究竟是哪里不对劲呢？</p>
 <p><b>要回答这个问题，让自己的逻辑变得无懈可击，你就需要学习逻辑思考中的核心思维方式：三段论。</b></p>
 <h2><b>什么是三段论？</b></h2>
 <p>简单来说，这是一种「大前提 → 小前提 → 结论」式的推理过程。</p>
 <p>其基本逻辑是：如果大前提是什么，且小前提是大前提的一部分，那么小前提也是什么；</p>
 <p>比如，著名的「苏格拉底三段论」：</p>
 <ul>
  <li>（大前提）所有的人都是要死的</li>
  <li>（小前提）苏格拉底是人</li>
  <li>（结论）所以苏格拉底是要死的</li>
 </ul>
 <p>嗯，无可辩驳…</p>
 <p>也许你以前已经听过三段论，但感觉这种说话方式好费劲，平时我们也不是这样说话的啊，只有科学研究，学术论文需要用到这种文体吧？</p>
 <p>其实三段论的应用范围很广，大到治理公司、设计产品；小到说一句话，写一段文字，其实都需要用到三段论，它是你逻辑的基础。</p>
 <p><b>只是，在实际应用中，它并不是那样标准的三段形态</b>，或者是隐去了大前提，或者是隐去了小前提，或者是隐去了结论，因而，才让你忽略到了它的存在。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p><b>我们回到前面的三句神逻辑：</b></p>
 <p><b>“你别看 XXX 捐了多少钱，他只是为了逃税罢了…”</b></p>
 <p>之所以这位同学有这样的言论，是因为他的大脑中，可能有一个这样的价值论断：做好事的人，都有自私的目的。</p>
 <p>这个就是他的大前提。</p>
 <p>我们用三段论的方式，拆解一下他的逻辑推断：</p>
 <ul>
  <li>（大前提）做好事的人，都有自私的目的；</li>
  <li>（小前提）XXX捐钱了，他在做好事；</li>
  <li>（结论）XXX一定有自私的目的；</li>
 </ul>
 <p>所以，他得出了XXX的捐钱是为了逃税的结论。</p>
 <p>只不过，他在表达的时候，把这个大前提给隐藏了，因此我们才会觉得这个逻辑听着有些不对劲，这个不对劲，就是指对那个没露脸的错误的大前提...</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p><b>我们再来看另外一句：</b></p>
 <p><b>“她穿得那么暴露，活该被色狼盯上…”</b></p>
 <p>这句话可能隐藏了什么大前提？</p>
 <p>我猜，他脑海中的大前提可能是：受害者必有罪过。</p>
 <ul>
  <li>（大前提）受害者必有罪过</li>
  <li>（小前提）她是受害者（被色狼盯上）</li>
  <li>（结论）她一定有罪过（穿着暴露）</li>
 </ul>
 <p>所以，他得出了这样的结论：“她穿得那么暴露，活该被色狼盯上…”</p>
 <p>可是，这个大前提是对的吗？所有的受害者都是有罪吗？穿着暴露也是一种罪过吗？</p>
 <p>他的逻辑千疮百孔...</p>
 <p><b>我们可以看到，这些因果推论，其实都符合三段论的形态，只是隐去了大前提，而错误恰恰就发生在这个大前提上。</b></p>
 <p>当你能熟练运用三段论的眼光去看待这些推论的时候，就能很快的找到对方逻辑的谬误点。所以，想要让自己的逻辑变的严密，第一步，就是要学会使用三段论。</p>
 <h2><b>那么，我们该如何使用「三段论」呢？</b></h2>
 <ol>
  <li><b>用于逻辑推理</b></li>
 </ol>
 <p>第一次世界大战期间，德军向法军猛烈进攻，法军为了避开德军锐气，便将自己的部队隐藏了起来，德军一时失去了攻击目标。</p>
 <p>德军指挥官下令侦察敌情。有一天，德军一名军官用望远镜搜索时，突然发现了前方慢慢地爬出了一只名贵的波斯猫，懒洋洋地躺在那里晒太阳。</p>
 <p>于是德军军官根据波斯猫的出入地点，找到了法军指挥所，并一举摧毁！</p>
 <p><b>一只波斯猫竟然毁了一支部队，这是如何推理出来的？</b></p>
 <ul>
  <li>（大前提）法军高级指挥官喜欢养名贵的波斯猫；</li>
  <li>（小前提）前方阵地有名贵的波斯猫；</li>
  <li>（结论）所以，前方阵地可能有法军高级指挥官。</li>
  <li>（大前提）法军高级指挥官住在法军高级指挥所内；</li>
  <li>（小前提）前方阵地可能有法军高级指挥官；</li>
  <li>（结论）所以，前方阵地也可能有法军高级指挥所！</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <p><b>2. 用于逻辑验证</b></p>
 <p>说出「神逻辑」的人并不会被抓，但是会影响你的思维逻辑，当不该连接在一起的两个要素连接在一起，而你又没有察觉，便会进一步推出更多的逻辑谬误，最终导致你的思维一片混乱…</p>
 <p>比如鲁迅在《论辩的魂灵》中，有一段反讽顽固派的诡辩手法时是这么说的：“你说谎，卖国贼是说谎的，所以你是卖国贼。我骂卖国贼，所以我是爱国者。爱国者的话是最有价值的，所以我的话是不错的。我的话既然不错，你就是卖国贼无疑了。”</p>
 <p>&amp;!~&amp;%@^*！...</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p>那么，我们除了找到隐藏掉的错误大前提，还有没有其他方法来识别谬误呢？</p>
 <p><b>三段论中有5项基本原则，分别是：</b></p>
 <ul>
  <li>第一，四项错误；</li>
  <li>第二、中项两不周延；</li>
  <li>第三、大项扩大，小项扩大；</li>
  <li>第四，前提都为否，结论不必然；</li>
  <li>第五，前提有一否，结论必为否。</li>
 </ul>
 <p>关于这5点的介绍，网上已有很多了，你可以自行百度。</p>
 <p class="ztext-empty-paragraph"><br></p>
 <p><b>3. 用于预测未来</b></p>
 <p>当我们掌握的是一些<b>基本规律</b>的时候，我们就可以以此作为大前提，做一系列的推演，进而对未来事物的发展进行预测...</p>
 <p>比如，我们抬头看天，发现阴云密布，然后回家拿了把伞...</p>
 <p>这是一个生活中常见的情形，也是我们解决问题的一种最基本思考方式，叫做：空雨伞。什么意思？</p>
 <ul>
  <li>空：（抬头看天空）把握事实和现状</li>
  <li>雨：（可能要下雨）解释、预测</li>
  <li>伞：（回去拿把伞）行动、提案</li>
 </ul>
 <p><b>但这个过程，依然省略一个大前提：阴云密布的时候，有70%的概率会下雨。这是一条规律。</b></p>
 <blockquote>
  <b>注：</b>70%这个数字为了便于说明，而非精确比例
 </blockquote>
 <p>因此，如果用三段论的方式，把上面的这个推理过程翻译一下就是：</p>
 <ul>
  <li>（大前提）阴云密布的时候，有70%的概率会下雨</li>
  <li>（小前提）现在阴云密布</li>
  <li>（结论）未来，有70%的可能性会下雨</li>
  <li>（解决方案）带把伞</li>
 </ul>
 <p>三段论的这种思考方式在逻辑思维中，叫做：演绎法。演绎法是逻辑思维的基础，如果这个没掌握好，所有想法都是扯淡，整个言论就会像豆腐渣工程，经不起推敲，一推就倒...</p>
