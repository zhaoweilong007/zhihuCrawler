# 国内码农和国外码农区别有哪些，会趋于一致吗？
- 点赞数：7753
- 更新时间：2021年11月10日17时32分36秒
- 回答url：https://www.zhihu.com/question/497793332/answer/2216734220
<body>
 <p data-pid="SgfkL5WG">我是在美国工作过两年，回国经历了逆文化冲击，现在勉强算是适应了国内互联网公司的节奏。随便聊聊，没有崇洋媚外的意图，只是刚好最近被剥削得很不爽，趁机吐槽一下。</p>
 <h2>1.年龄</h2>
 <h3>美国公司：</h3>
 <ul>
  <li data-pid="RTDMU249">同事里20多到70多岁的都有，众数是三四十的中年人，大部分工作目标都是为了早日退休，攒够钱就随时办退休party。也有些纯粹因为热爱工作、热爱写代码选择不退休的。</li>
  <li data-pid="diQ8vdXv">我们组的核心成员之一，是位72岁的老头，他每天4点多起床到公司写一会儿代码，等天全亮就戴上头盔去骑山地车锻炼，9点多回公司继续工作。对这老头印象深刻，是因为他逻辑清晰、思路锐利，他是code review小组的成员，经常在邮件里破口大骂其他人写的代码写得有多烂，被投诉，只好在邮件里道歉，过几天继续骂，在我工作的两年里一直循环。</li>
  <li data-pid="5VrmohpV">我的另一位资深同事，是位68岁的架构师，热爱工作，每天都乐呵呵的，对我这种新毕业生也很友好，有人问他什么时候退休，他回答说他死的那天。</li>
 </ul>
 <h3>我国公司：</h3>
 <ul>
  <li data-pid="VGrnM1yO">回国之后我现在工作的公司，员工平均年龄在30岁以下。年纪大的都去哪里了呢？极少数在管理层。</li>
 </ul>
 <h2>2. 加班</h2>
 <h3>美国公司：</h3>
 <ul>
  <li data-pid="FrILK3Cr">从没加过班，晚上发版除外（会默认第二天调休）。</li>
  <li data-pid="0xugTqqK">经常正开着会，时间到了5点半，产品打断领导说到点了他要回去喂狗（他是一个50岁的不婚族，养了一院子狗），然后就散会下班了。</li>
  <li data-pid="Gpy5rGPp">加班需要申请，有次我申请工作日晚上加班，没批准只好回家了。因为加班费会比较高，需要从项目预算走，领导控制预算不给批。</li>
  <li data-pid="00dW41xV">偶尔周末去办公室取东西，几层停车场只有两三辆车。</li>
 </ul>
 <h3>我国公司：</h3>
 <ul>
  <li data-pid="EMuCrgV7">996是常事了。</li>
  <li data-pid="hSvU225e">印象比较深的是我司之前有个清华本科+美国硕士的小伙子，每天7点半准时下班，结果试用期被辞退了，原因是工作态度不积极，据说后来还和公司打了官司，不知输赢。</li>
 </ul>
 <h2>3. 代码质量</h2>
 <h3>美国公司：</h3>
 <ul>
  <li data-pid="g3ZVZ5sg">项目在前期花的时间是最多的，比如说需求分析、架构讨论、技术讨论。</li>
  <li data-pid="_eVmWKGb">写代码会考虑得比较长远，比较有时间去考虑开发原则、维护成本，领导也会乐意去安排版本来解决技术债务。</li>
 </ul>
 <h3>我国公司：</h3>
 <ul>
  <li data-pid="gm4nWNgk">国内互联网节奏会要快得多，讲究小步快跑，就几天的开发时间，不管三七二十一先上线再说，刚开始我都惊呆了。</li>
 </ul>
 <h2>4.工作氛围</h2>
 <h3>美国公司：</h3>
 <ul>
  <li data-pid="VsBt4-p-">老美的公司确实比较尊重员工，在员工关怀上做得比较好。我可以感受到，和领导职位不同，但是我们人格是平等的，彼此尊重。</li>
  <li data-pid="pBx49cSt">记得有一次发版前几天，组里程序员说他压力太大，领导给他假期让他放松调整，版本被延迟上线。</li>
  <li data-pid="viFYvRXs">美国有family first的文化。有个老印同事，家里老人身体不好，公司同意他回印度工作照顾家人，远程跨国工作。经常有同事因为要看孩子比赛请假。领导自己也会偶尔周五请假，因为要去和女儿一起参加学校的公益活动。</li>
  <li data-pid="QX7OgH5o">对差异性接受度也比较高。同事有变性人、残疾人，大家相处得都很好。</li>
 </ul>
 <h3>我国公司：</h3>
 <ul>
  <li data-pid="9OUffIKp">领导高高在上，官威很大。请个假，和求他借钱似的，组长还提醒我让我请假原因不要写“旅游”不然可能会不给批假。</li>
  <li data-pid="QteH5q8o">记得有个需求，大家都认为不合理没必要，我去找领导沟通，刚提了一句还没展开，领导直接甩脸色“你是领导还是我是领导”。</li>
  <li data-pid="YKJryqvY">有个同事因为耿直，和领导不和，被各种排挤冷暴力，逼他自己辞职拒给赔偿金。</li>
  <li data-pid="SwB_kCBD">开个线上事故复盘会，做root cause分析，就像要把人钉在耻辱柱一样，我不理解这对解决问题有什么帮助。</li>
 </ul>
 <h2>5.工作之外</h2>
 <h3>美国公司：</h3>
 <ul>
  <li data-pid="PiwcvWVj">很注重对健康的投资。至少1/3同事有每天早上去健身房的习惯。公司很多球场，晚上下班能看到很多同事在楼下踢足球、打排球。健身不只是为了锻炼，还是很多同事的爱好。看起来平平无奇的程序员，可能都是隐藏的运动高手，多年马拉松选手、山地车骑手遍地都是，还有不少极限运动爱好者。</li>
  <li data-pid="zhMSr2SG">喜欢看牙医。喜欢看各种体育比赛。喜欢旅游，基本上每年至少一次家庭旅游，游轮是热门项目。</li>
  <li data-pid="VuwglT9Z">一部分同事热衷慈善回馈社会，小到捐血捐钱做公益，大到组织慈善拍卖会。</li>
  <li data-pid="6MLOLJcW">据我观察都没啥夜生活，下了班就开车直接回家两点一线，偶尔聚餐也是和同事朋友。可能是我自己的感觉，人和人之间的链接比较淡薄，所以华人码农也会经常吐槽空虚无聊。</li>
  <li data-pid="NA-ITi2E">已婚同事的其他时间和我国的一样，花在养孩子和投资上。</li>
 </ul>
 <h3>我国公司：</h3>
 <ul>
  <li data-pid="Nc5l9Bk2">办公室的好多同事，不敢看体检报告。都是20多岁的年轻人，检查出来啥的都有，胆囊炎、结石、痛风。。。前几天还有一个要好的同事请假去做痔疮手术的（捂脸），据他说是因为久坐，加班经常吃小龙虾。</li>
  <li data-pid="Up2Pfmvb">相比之下离职率高太多了，每个月都有几个认识的同事离职，跳槽的、转行的、回老家躺平的。</li>
  <li data-pid="5I_So5uX">除了领导们，几乎每个人看起来都很焦虑，都想着退路，想着搞点什么副业。</li>
 </ul>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="y0uhp-sQ">至于会不会趋于一致，个人认为是时间的问题，但是我们这一代还能不能等到，要看身体好不好了。无论怎样，一定保重好健康。<br><br></p>
