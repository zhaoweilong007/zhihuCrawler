# 有哪些东西是你读博士以后才懂的？
- 点赞数：6483
- 更新时间：2018年05月09日03时17分44秒
- 回答url：https://www.zhihu.com/question/49608607/answer/382682726
<body>
 <ol>
  <li data-pid="NDcMGwTp">永远要有 plan B. Research 不可能一帆风顺. 最终在 academia "生存"下来的未必是资质最出色的, 而是那些最输得起的人.</li>
  <li data-pid="Hh_qj48B">目标要远大. Think big. 但具体操作起来要细致. 很多时候文章发在top还是field top, 取决于一些细微环节的把控.</li>
  <li data-pid="nJhnkATf">远离所有的 gossip. 每个专业领域, 每个系都会有 politics. 但 politics 归根结底是资源互换.作为PhD, 唯一可以交换资源的是自己的能力. 把自己做大做强, 自然有大佬赏识. 建立有效的 network 是发了paper之后再需要考虑的事.</li>
  <li data-pid="LMp4Pdhk">Research 是自己的事. Dissertation 是自己的工作. 所有事情都要立足于自己. 没有导师不喜欢独立能力强的学生 (前提当然是在一个decent 的 research school).</li>
  <li data-pid="w34vIV1y">怎样的才是值得选择的导师? 能把你的 publication 提升一个档次, 并且能够帮你搞定好的 placement 的人才是. 其他都是餐后甜点. Academia 僧多粥少已是常态. 即便你的导师人很 nice, 学术水平也不错, 但只要他不是手握资源的领域牛人 (比如是 Journal 的 editor, funding committee 的成员, recruiting committee 说话算的成员), 你将来在 academia 生存的难度指数上升. 很残酷, 但是这就是现实 (或者说是很多人的血泪教训).</li>
  <li data-pid="PtLPlosH">有的大牛是 control freak. 在选择导师时规避他们. 这样的导师不仅会让你的博士生涯痛苦不堪, 更重要的是, 因为他们在学术圈内的人缘通常不好, 也很难帮你搞定好的 placement (同样是很多人的血泪教训).</li>
  <li data-pid="EFJV2ck8">不要怕麻烦导师 (这点对于导师是领域大牛的PhD尤其重要). 每个导师拥有的资源都是有限的. 你不争取自然会给更努力争取的人. 帮你的次数多了你就成他的嫡系了.</li>
  <li data-pid="ikSWJmsX">除非原则问题, 不要向任何同行抱怨你的导师/曾经的导师, 或者你所在领域的大牛. 在竞争激烈的学术圈, 你所说的任何一句关于他们的非正面的话, 最终都会以添油加醋的形式传到他们那里. 可以多培养一些爱好, 结交一些非学术圈的朋友. 毕竟吐槽有益身心健康.</li>
  <li data-pid="en1uVm9U">保持良好的竞技状态和积极的心态. Stay strong. Stay positive. 在竞争激烈的学术圈没有弱者的位置. When the going gets tough, the tough gets going.</li>
 </ol>

 <p data-pid="tcJ2-Bmq">0508 补充说明</p>
 <p data-pid="is1z5Nn5">当初写这个帖子, 是因为学弟的事一时义愤, 同时也是耳闻目睹了很多有才华的PhD 的遭遇, 觉得我这个"生存"下来的"幸运儿," 有责任把一些需要规避的问题如实地讲出来, 帮助正在或即将要读博的学弟学妹. 感谢诸位同道的共鸣和谬赞. 借我 coauthor 的一句话, 良好的科研环境, 需要"接地气"的理想主义者. 与大家共勉, 并敬祝研安!</p>
 <p data-pid="HjulZ-La">下面简单回复一下评论中提到的具体问题:</p>
 <ul>
  <li data-pid="bPgIaZ9K">关于 Plan B: 可以理解为广义上的两手准备. 比如刚开始做 PhD 时, 容易偏好更具挑战性的大 topic. 但现实是 top journal 的 acceptance rate 已经低到了一个荒谬的程度. 风控起见, 最好同时做可以保 field top 的稳妥性 topic. 这点在 tenure-track 阶段也同样适用. 推而广之, 其他关键事项也需要做万一事有不协的准备. 举个极端的例子, 答主就曾经看到过传说中活久见的 negative reference letter.</li>
  <li data-pid="fm9S8mXj">关于文章细节的把控: Top 和 field top 的区别通常不像我们想象的那么大. 但如何操作, 需要分专业分领域具体分析. 一个相对普适的方法是多读领域大牛出道时写的 paper, 从各个维度仔细体会学习.</li>
  <li data-pid="3JRuxDQi">关于如何规避 research 上不再 active 的大牛: 可以看他的 recent pub list.</li>
  <li data-pid="TMTMGMwH">关于如何规避人品不甚靠谱的大牛: 这个本身比较难. 可以参考该大牛的往届 placement record 和他的学术界地位是否相符, 但这点并不能作为一个硬性标准. 或许这个问题可以换一个角度考虑, 领域内学术人品有口皆碑的大牛名单, 还是比较容易打听到的. 投到他们门下即可.</li>
  <li data-pid="P_G7KxlV">关于如何确知某位可能的导师是否"funding committee 成员/ recruitment committee 的关键成员": 向该校关系好的学长打听.</li>
 </ul>
