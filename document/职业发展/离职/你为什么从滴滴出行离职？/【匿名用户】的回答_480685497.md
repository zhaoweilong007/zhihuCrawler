# 你为什么从滴滴出行离职？
- 点赞数：5084
- 更新时间：2018年08月29日10时23分53秒
- 回答url：https://www.zhihu.com/question/22385673/answer/480685497
<body>
 <p data-pid="MOjtvPDB">6月离职的我来说一波。。。先匿了。。。</p>
 <p data-pid="CysL7mNP">从MBB离职，加入滴滴战略部。因为柳小姐和其他高层的背景，所以投行和咨询的人选特别受欢迎。满心欢喜的我顶着冬日的寒风，从上海跑到了马连洼这个鸟不拉屎的地方。从动了从滴滴离职的念头到真正离开，就两周，主要可以总结成一下几点吧：</p>
 <ol>
  <li data-pid="Kq4wqBKm">工作效率。至少在战略部我看到的效率没这么高，很多人早上10点多才到办公室，洗洗杯子，买个咖啡，然后准备吃午饭，吃完休息一会儿，快两点才开始上班，工作没多少会儿，6:30去拿饭（公司有晚餐），然后回位置上坐一会儿，下班走人。</li>
  <li data-pid="a17WqWV8">加班问题。诚如上面所说，有划水度过一天的人，也有我这样苦逼的人。我的工作时间基本上早10晚11。甚至开会到12点也是有的。周六日老大也会不分时间的在钉钉群里说工作。我的老大是个背景相当不错的人，之前在大的投行和PE都做过，他每天的日程和我第一条写的差不多，但是我们下面的人呢，就比较苦逼了，比如我有一次周五约了饭三里屯，不到7点走掉了，然后在钉钉上给我喷了一通，嫌我走得早，OK，我理解，大概西二旗不理解Friday Night这种东西。有好几次手里的活已经交付完毕了，8点多走了，又说“你应该给自己找点活干”“你就不会学习一下别的业务”，此人在晋升答辩里，甚至让团队里其他小伙伴帮他做了他的晋升答辩PPT，辣鸡。我是一个效率至上的人，非常反对无(bu)意(gei)义（qian）的加班</li>
  <li data-pid="Sn12aA3v">公司内部团队的协同。滴滴内部和阿里是差不多的（我听说的，不确定），很多团队之间是抢项目的，因为滴滴的业务除了按产品线分为大家熟知的快车，优步，优享，专车，豪华车，企业级，顺风车，小巴，还会分司机端（B端）和乘客端（C端），城市运营，大区什么的，我到走也没弄清楚。这就导致了很多团队有天然的利益冲突。往往一个项目还没有立起来的时候，太多的团队就已经牵扯进来了，然后开始各种无意义的漫长讨（si）论（bi）。我甚至见到过A团队认为项目可行性太低，否掉了B团队的想法，然后自己另起炉灶重新做同一个项目，为此，B组leader在将近100人的钉钉群里大骂脏话。</li>
  <li data-pid="IG9Kil-U">人才训练和培养。我从来不敢说自己是个人才哈，但我认为在滴滴，很多team leader包括我自己的leader根本不懂怎么带新人。我刚去的时候就被莫名奇妙的拉到一个项目上，然后丢给你一份PPT，让你照猫画虎，那时候公司自己每天出的数据里我很多参数还都看不懂，就开始做项目了。第一个项目做得我是一头雾水，跟leader小小的反映了一下，他说我觉得以你的背景，应该一下就能弄明白，我很忙没时间带你，你多向其他同事请教吧，汗~</li>
  <li data-pid="cdqhIdOf">司机保障体系。不详细聊这个体系的具体内容，怕涉及商业机密，大家有兴趣去了解去一下这个体系到底都是干嘛的，我只能说这个项目是让我离职的最后一根稻草，因为这个东西是一个非常unethical的事情，这个项目从立项到落地，我从头到尾都在，但是我无力阻止。这个保障体系对现在的滴滴而言，无异于饮鸩止渴。项目刚上的时候，每天开日会，很多人看着增长的数字兴奋到面部肌肉变形，司机在线时长得到增长，很多人完成了自己的绩效，但是这种对司机毫无保留的压榨，让我看到了，多少商业体系，多少受过高等教育的人，为了自己，出卖自己的人性，露出了可恶的嘴脸，我要对滴滴所有参加这个项目的人（包括我自己）说一句：shame on you</li>
  <li data-pid="zLQJHAp3">滴滴的企业文化说不唯上，大家每天说着不唯上，做得都是什么事情，溜须拍马的人有多少。Wj咖啡的名字就是对不唯上最大的打脸。WJ=Will and Jean，Will是谁，Jean是谁，自己心里都清楚吧</li>
  <li data-pid="Phrg6_Av">对本公司的蜜汁自信和同业的大肆抹黑。在我受到的教育里，我认为对竞争对手的无端打压是一件非常不齿的事情，可是谁不知道，MT开城的时候，公司有多少人提起MT就破口大骂。是啊，MT可是你市场份额的竞争者。面对竞品的烧钱一般的补贴，居然说别人的数据是烧钱少出来的。我的妈，真是颠覆了我的三观，数据是烧出来的没错，难道本公司当年的数据是自己的口碑挣回来的吗？难道优步是真的自己做不下去才卖身的吗？当年的补贴大战你没有参与吗？你这样和robin li有什么区别？</li>
  <li data-pid="TpKCrayR">郑州空姐出事的时候我还在，可以告诉你们，公司根本就没有人真正的关心，大家该吃吃，该喝喝，动不动就是反正我们估值高，有钱，出了事有PR和安全事务部摆平。等PR稿出来了，象征性的发发朋友圈，顺便看看哪位同事因为此事受了牵连，八卦一下，done。</li>
 </ol>
 <p data-pid="5pr7I3LX">有点抱歉，没有为改变全球出行作出什么实质性的贡献。曾经也梦想仗剑走天涯，但是也只是曾经了。来过滴滴，爱过滴滴，曾经也是充满梦想的小桔子，再见了。</p>
