# 中了五千万你会立马离职吗？
- 点赞数：29067
- 更新时间：2022年07月18日15时25分12秒
- 回答url：https://www.zhihu.com/question/333864736/answer/2580489081
<body>
 <ol>
  <li data-pid="Vz61S3Zf">5000万，既然是中奖的话，应该是买彩票得来的，所以需要交20%的意外所得税，所以到手应该是<b>4000万</b>。</li>
  <li data-pid="zW3W3M2r">拿到这样一笔钱，为了一些玄学，还是会拿出来50万捐款，大概率会捐给一些帮助留守女孩的组织，做做功课，捐给靠谱的地方。<b>还剩3950万</b>。</li>
  <li data-pid="9pgAMEJB">然后这个时候，会找个借口把年假什么的都请了，老板要不批，我就直接走，反正我在不在公司其实也完全没有影响，假期到手，就开始安排下面的事情。</li>
  <li data-pid="hBp6Y6e5">作为一个没房没车的人，第一选择买房。挑一个长沙、重庆、成都这样宜居好吃的城市，买套房。偏爱平层，一个人住也不用太大，150平左右，买一个贵一些的小区，加上装修600万搞定。<b>还剩3350万。</b></li>
  <li data-pid="OE4S1S4i">自己是有驾照的，但对车没有什么太大的要求，安全可以代步就行，喜欢SUV，大概率买一辆50万左右的开开。<b>还剩3300万。</b></li>
  <li data-pid="-OJbUTlK">这些都弄好了，开始告诉爸妈自己中奖了，为了避免爸妈说出去被亲戚借钱，就说中了1000万，能让自己衣食无忧，但是借钱，没有。然后给我爸50万换车，我妈是全职在家所以给妈妈100万零花，不够再找我要，同时多多照顾外公外婆和爷爷奶奶。到这里，<b>还剩3150万</b>。</li>
  <li data-pid="3_zz4yig">固定资产差不多搞定了，开始考虑可持续发展，现在的行情买房不是最好的投资，最稳健的还是基本的理财，所以找几家银行谈一下利息，大额争取到4.2%，把鸡蛋放在不同的篮子里面，1500万基础投资，每年利息63万。<b>还剩1650万</b>。</li>
  <li data-pid="D3aQw8WS">然后找几个理财的经理，500万给他们，不要求赚大钱，分开一年能有10万的收益就行。现在<b>还剩1150万。</b></li>
  <li data-pid="rtmq4IeA">回公司辞职，不想工作的话，找个机构出国读研，正好出去见见世面，玩耍。不想读太久，争取好好学习申请英国一年的master。费用200万内搞定。<b>还剩950万</b>。</li>
  <li data-pid="v7bJsOrz">这一切都结束了，我就拿着每年大概70万的利息，和一套房一辆车，950万灵活的资金开始旅游，学习，丰富自己。不求轰轰烈烈钱滚钱，只求简单快乐身体健康。</li>
 </ol>
