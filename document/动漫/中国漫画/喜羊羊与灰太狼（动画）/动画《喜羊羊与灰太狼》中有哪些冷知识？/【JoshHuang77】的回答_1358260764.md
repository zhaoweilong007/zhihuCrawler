# 动画《喜羊羊与灰太狼》中有哪些冷知识？
- 点赞数：16954
- 更新时间：2020年09月03日07时50分35秒
- 回答url：https://www.zhihu.com/question/408113026/answer/1358260764
<body>
 <ul>
  <li data-pid="HJPXfw1K">灰太狼对红太狼的称呼早期为“老婆”。<b>但由于审核原因</b>，在最近的剧集中全部改为了“红红”，而这个称呼最早在初代作品中出现过。</li>
 </ul>

 <ul>
  <li data-pid="1Y9rtnxR">初代作品中经常出现<b>刀、叉、绳子、大锅</b>等道具，甚至还有过<b>机关枪</b>。<b>但由于审核原因</b>，官方在YouTube频道上传的<b>重置后的初代剧集（注意，不是原版）</b>中将刀叉等改成了勺子（所以有时候能看到<u>灰太狼在磨勺子而不是磨刀</u>，详见下面的<u>第353集片段</u>）；在重置版的初代作品以及后面的新剧集中，绳子改成了果冻状的物质，大锅则在做饭的时候从来不会点火，机关枪则全部被水枪或玩具枪外形的武器取代。</li>
 </ul>

  <li data-pid="fCf1Vb_6">初代片头曲《别看我只是一只羊》的第一句是“喜羊羊、美羊羊、懒羊羊、沸羊羊、慢羊羊、<b>软绵绵、</b>红太狼、灰太狼”，而事实上除了初代作品的片头曲之外，之后每一部《喜羊羊与灰太狼》作品的片头曲在唱到“软绵绵”的时候出现的都是<b>暖羊羊</b>（详情见下面第一个链接）。这是因为<u>暖羊羊在初代作品第55集“灰太狼之死”才登场</u>，是后来才添加的主要人物，所以片头曲制作的时候并没有考虑到她（关于暖羊羊登场的场景，详见下面第二个链接）。</li>
