# 如果让你重新开始学计算机，你的学习路线会怎么选择?
- 点赞数：6032
- 更新时间：2022年08月03日09时33分58秒
- 回答url：https://www.zhihu.com/question/492545174/answer/2607142730
<body>
 <p data-pid="HpOvOjCR">国外有一位小哥开源了一个学习路线，目前已经累计在 Github 上收获了 204k star，可以说是全球最火的学习路线了。</p>
 <figure data-size="normal">
  <img src="https://pic1.zhimg.com/50/v2-f79ae32c3bb201ccac29d9b9af5856ef_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="650" data-rawheight="438" data-original-token="v2-f79ae32c3bb201ccac29d9b9af5856ef" class="origin_image zh-lightbox-thumb" width="650" data-original="https://pic1.zhimg.com/v2-f79ae32c3bb201ccac29d9b9af5856ef_r.jpg?source=1940ef5c">
 </figure>
 <p data-pid="rFdquX12">这仓库的名字叫做“<b>developer-roadmap</b>”，其包括的学习路线涵盖：<b>前端</b>、<b>后端</b>、<b>DevOps</b>、<b>Android</b>、<b>React</b>、<b>DBA</b>、<b>Java</b>、<b>Go</b>、<b>Python</b>，非常全面。另外，<b>测试</b>、<b>AWS</b> 的学习路线还在路上。</p>
 <figure data-size="normal">
  <img src="https://pica.zhimg.com/50/v2-9e944036465fee118fc8b23713bea244_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="650" data-rawheight="524" data-original-token="v2-9e944036465fee118fc8b23713bea244" class="origin_image zh-lightbox-thumb" width="650" data-original="https://picx.zhimg.com/v2-9e944036465fee118fc8b23713bea244_r.jpg?source=1940ef5c">
 </figure>
 <blockquote data-pid="uYCyHYNv">
  题外话：国外使用 PostgraSQL 还是比较多的，用法和 MySQL 差不多，但是PostgraSQL很多方面确实比 MySQL 好用一些。这几年在国内也越来越流行。
  <br>
 </blockquote>
 <p data-pid="sQhQs5pb">作为程序员，如果你还不知道这个仓库的话，就真的太亏了。几年前，我就曾经在公众号推荐过这个项目，当时 star 数量不过也才 90k+ 。</p>
 <p data-pid="Ld2-esQR">网上有非常多的技术大佬都是参考这个仓库写的学习路线。</p>
 <p data-pid="uMxTLHuk">Github 地址 ：<a href="https://link.zhihu.com/?target=https%3A//github.com/kamranahmedse/developer-roadmap" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">https://</span><span class="visible">github.com/kamranahmeds</span><span class="invisible">e/developer-roadmap</span><span class="ellipsis"></span></a></p>
 <h2><b>效果预览</b></h2>
 <h3><b>前端学习路线</b></h3>
 <p data-pid="_JY2xhIJ">下图就是这个仓库关于前端的学习路线的部分内容，非常全面细致。</p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-40587e33cc80154e0e77b1f77dda55d3_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="650" data-rawheight="647" data-original-token="v2-40587e33cc80154e0e77b1f77dda55d3" class="origin_image zh-lightbox-thumb" width="650" data-original="https://picx.zhimg.com/v2-40587e33cc80154e0e77b1f77dda55d3_r.jpg?source=1940ef5c">
 </figure>
 <p data-pid="5VBnwBu7">一步一步按照这个路线去网上找对应的教程看，你完全可以把前端自学的非常好，循序渐进。</p>
 <h3><b>后端学习路线</b></h3>
 <p data-pid="oDfYXbT0">下图就是这个仓库关于后端的学习路线的，内容总体还是非常赞的。基本涵盖了一个后端程序员必须要掌握的一些知识，非常直观！ 你可以将其作为自己后端学习之路的重要参考。</p>
 <figure data-size="normal">
  <img src="https://pica.zhimg.com/50/v2-365cb853dfb97359e0b4d2fdf74d0e83_720w.jpg?source=1940ef5c" data-caption="" data-size="normal" data-rawwidth="650" data-rawheight="670" data-original-token="v2-365cb853dfb97359e0b4d2fdf74d0e83" class="origin_image zh-lightbox-thumb" width="650" data-original="https://picx.zhimg.com/v2-365cb853dfb97359e0b4d2fdf74d0e83_r.jpg?source=1940ef5c">
 </figure>
 <p data-pid="Vgb22CyJ">需要注意的是：<b>对于可以有多个选择的地方，作者都给出了自己的推荐（紫色勾）。但是， 你不需要完全按照作者的推荐来，毕竟国内外的技术流行度还是有区别的。</b> 比如在编程语言这里的话，作者推荐了 JS，但是，在国内的话，显然是 Java 或者 Go更适用一些。再比如关系型数据库这里，作者推荐的 PostgreSQL，但是，在国内肯定是 MySQL 更加通用一些，PostgreSQL 或许是未来的趋势。</p>

 <h2><b>总结</b></h2>
 <p data-pid="3zahPQD3">我推荐这个仓库的原因并不是因为这个学习路线有多么详细，主要是因为它提供的学习路线是真的非常直观。这种用来展示学习路线的方式，大家也可以学起来！</p>
