# 你有哪些用计算机技能解决生活问题的经历？
- 点赞数：5912
- 更新时间：2016年06月08日12时43分48秒
- 回答url：https://www.zhihu.com/question/28361295/answer/104716780
<body>
 <p data-pid="cbD0vWtl">谢邀......</p>
 <p data-pid="_yxd-WJn">在电脑上常常一边干活儿一边听网易云音乐。每次想切歌的时候，我都是要先找到网易云音乐的那个桌面，然后按一下快捷键command + →，切换下一首，然后再找到我刚刚干活儿的那个桌面。</p>
 <p data-pid="GeRAh-H-">嗯，我得打断我当前的事情，花费起码三秒钟，去切一首歌。</p>
 <p data-pid="3rqvMos8">我忍了很久了。。。</p>
 <p data-pid="XZ3TXLJV">终于有一天，我决定自己写一个Alfred workflow，一次性终结这个难题。</p>
 <p data-pid="d8YowImB">于是...我查遍网上的文档，花了一个小时…</p>
 <figure>
  <img src="https://picx.zhimg.com/50/db0ad810693685d6768ee288cf0a677b_720w.jpg?source=1940ef5c" data-rawwidth="1560" data-rawheight="2048" data-original-token="db0ad810693685d6768ee288cf0a677b" class="origin_image zh-lightbox-thumb" width="1560" data-original="https://picx.zhimg.com/db0ad810693685d6768ee288cf0a677b_r.jpg?source=1940ef5c">
 </figure>
 <br>
 <p data-pid="bvjoiASk">终于写出了七行代码。</p>
 <div class="highlight">
  <pre><code class="language-applescript"><span class="k">set</span> <span class="nv">old</span> <span class="k">to</span> <span class="p">(</span><span class="nb">path to</span> <span class="na">frontmost</span> <span class="nb">application</span> <span class="k">as </span><span class="nc">text</span><span class="p">)</span>
<span class="k">tell</span> <span class="nb">application</span> <span class="s2">"System Events"</span>
	<span class="k">tell</span> <span class="nb">application</span> <span class="s2">"NeteaseMusic"</span> <span class="k">to</span> <span class="nb">activate</span>
	<span class="na">key code</span> <span class="mi">124</span> <span class="nv">using</span> <span class="nv">command</span> <span class="nv">down</span>
<span class="k">end</span> <span class="k">tell</span>
<span class="nb">delay</span> <span class="mi">1</span>
<span class="nb">activate</span> <span class="nb">application</span> <span class="nv">old</span>
</code></pre>
 </div>
 <p data-pid="g5CXMiEU">它的工作流程很容易理解，就是把当前的app记录下来，然后切换到云音乐，触发command + →快捷键，然后等一秒钟，切换回刚刚的app。</p>
 <p data-pid="Iw9le5uT">我终于可以在调用workflow的时候，默默欣赏桌面开始切换来切换去了。</p>
 <p data-pid="mxUIRyXz">可是！为什么不能直接后台切歌呢？！还得看这它跳来跳去，又浪费我一秒钟！</p>
 <p data-pid="BTi6nyrw">于是...我查遍网上的文档，又花了一个小时…</p>
 <figure>
  <img src="https://picx.zhimg.com/50/db0ad810693685d6768ee288cf0a677b_720w.jpg?source=1940ef5c" data-rawwidth="1560" data-rawheight="2048" data-original-token="db0ad810693685d6768ee288cf0a677b" class="origin_image zh-lightbox-thumb" width="1560" data-original="https://picx.zhimg.com/db0ad810693685d6768ee288cf0a677b_r.jpg?source=1940ef5c">
 </figure>
 <br>
 <p data-pid="umdPHlt4">改成了三行代码。</p>
 <div class="highlight">
  <pre><code class="language-applescript"><span class="k">tell</span> <span class="nb">application</span> <span class="s2">"System Events"</span> <span class="k">to</span> <span class="k">tell</span> <span class="nv">process</span> <span class="s2">"NeteaseMusic"</span>
	<span class="nv">click</span> <span class="na">menu</span> <span class="nb">item</span> <span class="s2">"下一个"</span> <span class="k">of</span> <span class="na">menu</span> <span class="mi">1</span> <span class="k">of</span> <span class="na">menu</span> <span class="nv">bar</span> <span class="nb">item</span> <span class="s2">"控制"</span> <span class="k">of</span> <span class="na">menu</span> <span class="nv">bar</span> <span class="mi">1</span>
<span class="k">end</span> <span class="k">tell</span>
</code></pre>
 </div>
 <p data-pid="SaEwtl2H">这个更容易理解了，就是在后台按了一下网易云音乐菜单栏里的『控制/下一个』。问题解决。</p>
 <p data-pid="_V6TI8h1">我非常欣慰。虽然花了两个小时，可是以后切歌再也不用来回找窗口了，切一次可以省3秒钟，我只要切两千多次歌，这个时间我就赚回来了！</p>

 <p data-pid="eBjEkGCw">------ 关于F9 update: 6.8 ------</p>
 <p data-pid="DxmzPePl">评论区关于F9的问题太多了...统一回复一下...</p>
 <p data-pid="NoFkP-dH">1. 其实可以注意到，答主是mac党。command键、F9键，均为mac键盘按键。F9能干嘛，上面那张键盘图里圈出来了。</p>
 <p data-pid="wlW8HM4w">2. 如果你是mac，F9没有用，可能原因：官网下载的网易云音乐才可以用F9，App Store里下载的不可以（未验证）。或者偏好设置里把F系列功能键设置成了标准功能键。或者F9跟别的东西冲突了。</p>
 <p data-pid="QlfG1PDh">3. 如果你是PC...那我也不知道了，评论区办法挺多的，可以去找。</p>
 <p data-pid="FbAd1pU4">4.不要忘了除了F9下一首，还有F7上一首，F8暂停/播放。F10-F12是调音量的。</p>
 <p data-pid="OWks_UIf">5. 网易云音乐都是可以设置全局快捷键的。不过，全局快捷键容易跟别的应用冲突。</p>
 <p data-pid="K8P9SYqe">6. 不要问答主为什么不设置全局快捷键...我要是知道这种东西...我会不知道F9么...</p>
