# 为什么别选计算机专业？
- 点赞数：5079
- 更新时间：2021年11月18日00时59分24秒
- 回答url：https://www.zhihu.com/question/465369002/answer/2213759239
<body>
 <p data-pid="ciFmMNVP">不明白现在鼓吹计算机是什么意思。</p>
 <p data-pid="UIwckd7W">985计算机毕业，刷LeetCode无数，背诵了无数的八股文，面试造火箭，好不容易卷进了大厂。最近开始相亲了，真的是处处碰壁啊。<b>个人观察，女生找对象的时候，并不是看你的收入的，一个年入30万的律师，一个年入20万的医生，年入20万的公务员，年入20万的男老师，年入40万的金融男，在相亲市场上要比年入50万码农抢手的多。那些年入不过15万的中学老师就没有一个看得上码农的。</b></p>
 <p data-pid="0fpV8LTo">我问了很多相亲女生的看法，总结起来无外乎2点：</p>
 <p data-pid="Wrw86FxP">1，码农工作不够稳定。翻译成人话就是，码农吃青春饭，35岁就要被淘汰。和律师，医生，公务员相比差远了</p>
 <p data-pid="tner46ZY">2，码农工作不够体面。翻译成人话就是，没有任何社会资源，孩子上学帮不了，父母看病帮不了</p>
 <p data-pid="DdSDLjC5">其实个人条件还算不错，身高183，在高中的时候就有很多人追的。结果到了大学，计算机系女生只有1/4，到了工作中，天天面对电脑敲代码，几乎见不到多少女性。不过个人还是不担心的，毕竟身高在哪里摆着，实在不行就找一个条件低一点的，最主要的是我不脱发。反观很多同事同学，他们就不那么乐观了，基本上30岁左右就秃头了，而且还驼背，他们找对象是真的难啊。很多人连女生的手都没有牵过。</p>
 <p data-pid="8LCw_UCZ">这里要说一下，码农可以说是优质的对象了。因为我们每天面对电脑写代码，课余时间也是要刷题研究技术，是不可能出轨的。像现在热捧的律师、医生、金融，出轨的几率是很高的。等到他们发达了，外面各种小三秘书真的很常见！给我们码农一个机会吧</p>
 <p data-pid="BdfjosZ0">总体来说一下我对计算机专业的看法：</p>
 <p data-pid="_jZDaZ1t"><b>1，时代不一样了。</b>码农之所以被吹捧，必然是有原因的。08年的时候，计算机真的是鄙视链的最低端了，我表哥就是那时候毕业的，那时候计算机都是分不够被调剂过去的。但是呢，移动互联网的爆发，外加人工智能，给计算机续了2次命。我的很多领导，也就是10年到15年加入互联网公司的，字面意义的财务自由了。年薪百万，房产增值若干，天天上班欺上瞒下玩弄女下属就是这一帮人（说实话不少人素质很低）</p>
 <p data-pid="Vo9qu9YG">阿里巴巴是14年上市的，14年前你在大街上真的没有几个路人知道马云，那时候真的是，二本进大厂都不难。我们18年已经很卷了，我也是凭借着985才进的大厂。而现在呢，我们招聘，非985硕士连看都不看的。是的，我们一年给新生就可以给30万，但是我们一般要刷掉十几个人才招一个，你确定你能卷进了我们大厂？</p>
 <p data-pid="sAXIeQKx"><b>2，码农工资没有这么高。</b>另外，40万的包裹是包含股票的，必须干满4年才能给你，离职跳槽全部没有了，其实你的工资也就是20万出头，扣完税真的没多少明白吗？不像公务员，基本工资一个月一万，但是各种年终奖，补贴，精神文明奖，五险一金。真的不一样。考虑到程序员基本上是35岁失业，那时候孩子估计还在上小学，一家老小，吃什么喝什么。</p>
 <p data-pid="BCndfrCo">另外，收入分为明面收入和暗面收入，人家医生分钱的时候，是不会给你说的。人家拿回扣收红包也是不会给你讲的。</p>
 <p data-pid="Nkz9naP2"><b>3，找对象很重要。</b>我们计算机专业女生很少，大学四年下来真的连和女生说话的机会都没有。毕业了就去大厂卷了，更没有机会和女生接触了。</p>
 <p data-pid="mih2sxtT"><b>4，码农是真的苦啊。</b>大学的时候，通宵赶项目是常态，各种新语言，各种框架，各种配置环境，能把你逼疯你信吗？其他的工科为了考试作业发愁，我们看了就是好笑。考试算个啥，代码你有bug，调试一周都找不到原因，问题是代码跑不出结果，就是零分，别管你花了多少时间。这才是真正的绝望。我们的大作业，1~2周让你写一个大作业真的是常态，在外面的培训班至少要学一个月，真的不知道那群老师的脑子是不是进水了</p>
 <p data-pid="_TWGwpB_"><b>5，最后说一下35岁退休，为什么会35岁退休？</b>本质上就是因为你35岁的竞争力已经比不过27的小年轻了，年老色衰，人家NBA球员35岁就算老将了，我们同理。我们一辈子也就只能挣10年的钱，现在不多拿一点，以后老了怎么办。评论区有人说算法不失业的笑死我了。要知道算法岗是15年以后才出现的，以前根本没有算法岗这个东西，从业者就没有超过35岁的，何谈35岁危机？</p>
 <p data-pid="h3xxxRUq">总结：</p>
 <p data-pid="p_-Dgv5e">计算机并没有想象中的美好。最重要的就是互联网的增长能不能持续的问题，之前计算机也是劝退专业，后来是移动端人工智能给计算机续命了。现在互联网的风口早就过了，资本的宠儿已经是自动驾驶和新能源了，如果还是以为互联网的高薪可以持续，那么你真的很幼稚了。</p>
 <p data-pid="jKxjVpGL">更不要说计算机由于是工科，自带硬核属性，大学4年真的很辛苦。到了工作也是硬核模式，代码写的不好真的是被末尾淘汰。而且就像所以的工科一样，女生很少，不好找对象。由于计算机是对着电脑的，眼睛近视，弯腰驼背，秃头真的就是职业病了。</p>
 <p data-pid="e2VBKGL8">考虑到计算机是35岁失业，综合来看真的没有这么香。个人建议，以下几种人可以学习计算机：</p>
 <ol>
  <li data-pid="0MGtlRSy"><b>女生。</b>因为计算机行业女生很少，上大学老师一般会给你放水，不会了可以找男同学舔狗帮你写作业，上班了有机会认识年薪数百万的互联网精英（而不是底层赚几十万的码农哈），性别优势巨大。</li>
  <li data-pid="wrB-b6nC"><b>家境极其贫寒的孩子。</b>就是说上大学需要助学金，需要自己做家教赚零花钱，去食堂连肉都不舍得吃的那种。计算机最大的优势是，本科就可以就业。985毕业，去科大讯飞深信服，找一个年薪10万的工作不难</li>
  <li data-pid="FnZa7A-M"><b>二次元、书呆子、宅男。</b>不喜欢出去玩，不喜欢和人说话，对女生没有兴趣，就喜欢捣鼓电脑，就喜欢打游戏。实际上计算机在没火之前，绝大部分是这种人</li>
  <li data-pid="Lh19Ho24"><b>想移民的人。</b>发达国家的人普遍保障好，脏活累活不想干，所以码农就有了移民的机会了。当然了，移民并不一定好，首先是英语这一关，另外国外种族歧视严重，疫情反复，没法照顾父母。</li>
  <li data-pid="eOxt5yPj"><b>二本三本专科。</b>像这种层次的学生，基本上就是毕业就要进厂打工，考公务员由于学历太差基本上没有机会，临床的分数也不够，考金融研究生本科学历太差，基本上面试就被刷下去了。计算机能让他们在办公室里面，写程序拿几千的工资，他们心中只有感恩</li>
 </ol>
 <p data-pid="LzX49Svo">下面列一下时间线：</p>
 <p data-pid="GPfkrBDv">1，高考结束。</p>
 <ul>
  <li data-pid="5y4yTfPD">A想学计算机，由于计算机分数线非常高，A为了选择计算机，不得不降低一个档次择校</li>
  <li data-pid="aeHLRXKc">B想学医学，他发现很多医学院的分数线并不高。B甚至可以选择临床八年，毕业就是博士。他去了北京某985</li>
  <li data-pid="WNr7NFqp">C想学金融，像央财上财，分数线已经是中后985的水平了</li>
 </ul>
 <p data-pid="EuJa1mEj">2，大学期间</p>
 <ul>
  <li data-pid="nBWfI9jr">A天天写代码，改BUG，时不时需要通宵赶项目，几乎没有任何时间出来玩，也几乎找不到女朋友</li>
  <li data-pid="Q9ee0Dyi">B由于直接临床八年，大学只是保持了平均的GPA，其他时间参加了很多社团。由于他是临床的，所以在学校里面，面子很足</li>
  <li data-pid="RahmpPvh">C学金融。金融实在是简单，基本上就是一个文科专业，不需要动脑子，而且女生非常多。C在大学谈了2个女朋友</li>
 </ul>
 <p data-pid="0xPOjWoc">3，大学毕业</p>
 <ul>
  <li data-pid="BDwPEfnC">A想要保研，但是发现保研均分已经卷到87分了，毕竟计算机专业都是精英中的精英，卷不过他们也是正常。想要找工作，发现自己连笔试题都过不了。不得不只能考研了，408专业课真的难，而且还要和很多转专业的竞争</li>
  <li data-pid="q4VceT48">B上的是临八，不需要考研</li>
  <li data-pid="DSTzH87V">C由于大学天天玩，不得不要考研，他发现金融考研不难，毕竟只需要准备数学就行了，专业课基本上可以算是没有门槛</li>
 </ul>
 <p data-pid="1JT396aP">4，找实习</p>
 <ul>
  <li data-pid="FgOnJfqj">A想要找实习，腾讯阿里根本没有机会。为了找工作，他每天刷LeetCode 100多个小时，刷了100多道题目。此外，他还需要背Java八股文，还需要做项目，难度很大</li>
  <li data-pid="1bR0xFs-">B的实习都是学校安排的，自己不需要操心</li>
  <li data-pid="NUy8BciK">C发现实习也很难找，毕竟金融这一行特别看学历，有些公司甚至是非清北不收。不过凭借着上财的学历，他也轻松去了四大</li>
 </ul>
 <p data-pid="kaMHE96L">5，研究生</p>
 <ul>
  <li data-pid="ZhW66nRL">A的导师人不好，天天压榨他们做项目，不让他们出去实习。A为了毕业忍气吞声，为了毕业还要卷深度学习，每天忙到11点，虽然老师P都不懂。</li>
  <li data-pid="mmsllzCG">B已经开始跟着导师做手术了。如果那个病人没有事先打好招呼，B的导师一般喜欢让B上去练练手，虽然B已经搞砸好几次了，不过呢，病人也不知道啊</li>
  <li data-pid="v94Vjs7N">C研究生基本上就是在实习了，还能拿很高的实习工资。由于自己是金融的，也靠这个头衔骗了几个妹子上床，不小心做了渣男</li>
 </ul>
 <p data-pid="Ex4CTh9j">6，毕业</p>
 <ul>
  <li data-pid="bhZXLnuw">A研究生3年被导师搞得差点抑郁，不过计算机还算高薪。由于没有好的实习，腾讯阿里去不了，只能去二线大厂。也许是上天有灵，A被华为捞了，A感激涕零，终于可以去深圳享受福报了</li>
  <li data-pid="LkShi7I0">B比A晚一年毕业，他发现还需要规培，而且工资只有几千，感觉自己被欺骗了，不禁大喊“劝人学医，天打雷劈”，B顺利进入北京的三甲</li>
  <li data-pid="IRoJhT1C">C找工作也算顺利，去了上海陆家嘴，虽然只是基层员工，但是挺光鲜</li>
 </ul>
 <p data-pid="g4MEG6kt">7，工作3年后</p>
 <ul>
  <li data-pid="rHgEPi6U">A的工资已经到了40多万，但是身体已经卷不动了，毕竟是每天12点下班。不过A还是上进的，考评也是A或者B。由于有了一些积蓄，A也打算看房子了</li>
  <li data-pid="mmBeoBDu">B终于结束了规培，工资也不到1万多了，买房子是遥遥无期了，不过由于B是体制内，所以医院给解决上海户口</li>
  <li data-pid="yW-gQC2V">C升迁了一次，工资有时候可以到底50万，C也打算买房了，另外，由于工作便利，C投资了很多股票，据说收益不错</li>
 </ul>
 <p data-pid="vln8N8r3">8，结婚</p>
 <ul>
  <li data-pid="tTEIP2-N">A由于接触不到女生，不得不走向相亲。由于码农不收待见，而且A已经秃头了，所以被很多女生嘲笑。另外深圳房价很高，A勉强上车，压力很大。最终A找了一个一本的女生结婚了</li>
  <li data-pid="ywhnl3QP">B也去相亲了。虽然他没有房子，工资很低，但是非常受到欢迎。有很多土著想要找一个医生的，他们家产至少也是2000万往上的</li>
  <li data-pid="EXd0T_zs">C找了一个自己的师妹，年轻漂亮，幸福美满</li>
 </ul>
 <p data-pid="PMgX25a4">9，工作7年</p>
 <ul>
  <li data-pid="yOJLNRVL">A已经32岁了，常年的加班让他得了很多疾病，不得不经常去医院。有一次，在工作中被背了黑锅，A气愤不过，辞职了。后来A知道了新工作，工资只有不到40万，不过不加班。</li>
  <li data-pid="mJvBFyer">B已经成为主治了。工资也是有30多万了，另外，还有很多灰色收入，不过不好明说</li>
  <li data-pid="GryjVsVP">C又升职了，工资也有60万了，C已经打算买好车了</li>
 </ul>
 <p data-pid="xV4eUBes">10，工作12年</p>
 <ul>
  <li data-pid="evcbe-JE">A硬撑了几年，在自己37岁的时候，终于被强制“退休”了。不过房贷也不多了，A打算自己自己开一个码农培训班。此时他在知乎等平台上，大肆宣传码农的高薪，为自己招揽顾客。此时，孩子孩子上小学</li>
  <li data-pid="UKkcUkf4">B离主任只有一步之遥。工资也到了大几十万，学区房什么的也不需要操心，毕竟老婆就是土著。另外小学老师也是一直巴结B，对B的孩子很好。小学的班主任甚至想让B做家委会主席，不过B还是拒绝了</li>
  <li data-pid="qWGj2VlC">C的工资已经接近百万，第二套房子还在还贷款。小孩也去了私立双语名校，英语说的比美国人还流</li>
 </ul>
 <p data-pid="9EzaOG1k">11，工作20年</p>
 <ul>
  <li data-pid="5WthA60k">A最后还是找了一个私企混，签的也是劳务合同，工资一年可以30万以上，不过在深圳不够看。看到有同学上市拿到几百万的期权，自己真的羡慕，留下了无能了泪水。同时，A利用自己码农的优势，从小培养自己的孩子写代码！</li>
  <li data-pid="Ho9DUdVe">B成为副主任了。工资虽然明面上50万，但是背地里谁知道呢。现在打击课外培训，但是很多名师还是愿意背地里给B的小孩辅导的，毕竟很多人还是想巴结医生的。还有一个北京的教授，想要把成果给B的小孩，据说自主招生有用。还有人想给B的小孩写推荐信，据说出国有用。不过北京高考容易啊，B让孩子自己决定</li>
  <li data-pid="P0Jlwx6a">C成为金融高管，有时候也去潜规则一下实习生。C还掺和了各种投资，貌似收益客观。孩子在国际高中，反正出国的钱也拿得起</li>
 </ul>
 <p data-pid="U3Q0mUSO">12，工作30年</p>
 <ul>
  <li data-pid="8EVHpyZt">A已经写不动代码了，不过小生活也算滋润。B的小孩也要大学毕业找工作了，虽然由于广东高考比较难，但是小B也去了211。B对孩子说，父母不认识什么人，一切都要靠你自己。</li>
  <li data-pid="uZfvs66p">B已经成为主任了。交往的也都是非富即贵。钱什么的都是小意思了。他的孩子想去当读博士，到大学当教授，B全力支持，还找到小B的院长给小B写出国的推荐信</li>
  <li data-pid="Zh5ITLXf">C的孩子打算回国了，C此时房产已经3套了。C打算让小C接班，还是去金融行业吧，毕竟工资高，而且C还有人脉关系</li>
 </ul>
 <p data-pid="qb3L18n5">30年后，A起码在深圳有一套房子，孩子也211毕业了，小日子也不过。B认识很多达官显贵，房子买了一套，不过丈母娘有N套房子，最后全是自己的，也不知道怎么花钱。C已经有了3套房子，此外炒股也是收益不菲，算是在上海站稳了脚跟。</p>
