# 哪个瞬间让你觉得一入计算机深似海?
- 点赞数：8110
- 更新时间：2022年04月02日16时40分22秒
- 回答url：https://www.zhihu.com/question/66800857/answer/247119678
<body>
 <ol>
  <li data-pid="Wxl5LMaR">写代码遇到解决不了的bug翻墙出去搜google搜sof搜github，换了七八个关键词，发现在犄角旮旯里的一个页面里2006年有一个苦逼跟我一样的问题，然后十多年无人回复。那时候感觉跨越时空两颗孤独的灵魂相通。</li>
  <li data-pid="2ZKW0TIt">分析目标网站页面爬的时候发现对方发现我在爬的时候。你出一招我出一招，瞎比划。那时候感觉生命就是这么被浪费的。</li>
  <li data-pid="qMI9KcU5">给国外的服务器运维的时候，延迟太高导致得记住打了几个字符，Tab与Delete用了几回，大脑变成栈了。那时候感觉生命又TM被浪费了。</li>
  <li data-pid="NVYBmT9N">手贱更新库报出一堆error的时候。那时候感觉红色的Error鲜艳如吐出的血。</li>
  <li data-pid="RLiSSa6E">给亲戚朋友修电脑修手机修电视修导航的时候。搞的定那是你应该的，搞不定你学这个干嘛吃的。那时候感觉我真不如去筛沙，还能锻炼身体。</li>
  <li data-pid="1az0VXYn">女儿出生了，想发朋友圈，脑子里首先蹦出的是『Hello World！』。那时候感觉自己真的没救了。</li>
  <li data-pid="Gj2d__V6">别人问我互联网IT，以为能跟我聊人工智能区块链无人驾驶VR，我发现我了解的不比他多的时候。那时候仿佛曾经写过的代码都成了乱码。</li>
  <li data-pid="-gl-RpRt">学弟学妹刚毕业拿我的两倍工资时。那时候感觉，还有个屁感觉。</li>
 </ol>
