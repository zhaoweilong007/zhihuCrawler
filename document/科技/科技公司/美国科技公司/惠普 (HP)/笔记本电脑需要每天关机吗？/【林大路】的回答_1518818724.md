 <h2>如果你用Win10的电脑，经常放包里</h2>
 <h2>拿出来的时候机器发烫</h2>
 <p data-pid="rjnLgY1p">那么是温十的bug起作用了（bug表现就是不能正常休眠，或者笔记本放包里因为震动关系激活了系统，导致合盖运行，放包里又是闷罐，能不发烫吗？）</p>
 <p data-pid="5KsaUcuV">这种情况建议准备放包里的时候</p>
 <h2>手动关机</h2>
 <p data-pid="FLXHOpkZ">如果不存在该问题</p>
 <h2>直接盖上就好</h2>
 <p data-pid="nIHJi8Ma">如果盖上以后可能经常不用，比如</p>
 <ol>
  <li data-pid="8vUTQe_E">三天</li>
  <li data-pid="vU97VIZs">一周</li>
  <li data-pid="nZO9FK4l">半个月</li>
 </ol>
 <h2>建议关机</h2>
 <p data-pid="dQmEwIzx">因为睡眠或者休眠都会持续掉电</p>
 <p data-pid="hLpULvkd">虽然电源管理会在电池电力低下的时候</p>
 <h2>自动关机</h2>
 <p data-pid="_4zV5e8c">但是，电源管理默认是10%-6%自动关机</p>
 <p data-pid="YGCCzbml">这样会损伤电池，寿命会降低</p>
 <h2>一般保持电量在20%以上</h2>
 <p data-pid="PQz4AWHJ">电池寿命理论上在三到五年都可以零损耗</p>
 <h2>所以，没事就把电插着，即便盖上</h2>
 <p data-pid="PNETtJb3">即便不用，但这是最好的保护电池的方法</p>
 <p data-pid="DJInu_an">毕竟，目前笔记本主流电池还是锂电池</p>
 <p data-pid="IIpDWbI0">ʕ•̫͡•ʕ*̫͡*ʕ•͓͡•ʔ-̫͡-ʕ•̫͡•ʔ*̫͡*ʔ-̫͡-ʔ</p>
 <p data-pid="MzJaokSo">锂电池，就是这么操作的</p>
 <h2>虽然锂电池有充放电次数</h2>
 <p data-pid="fjXWZcqU">但是你插上是使用外部供电（电源适配器供电，这时候电池变成了UPS），并不会损耗电池。</p>

 <ol>
  <li data-pid="7AsS1zqR">待机（即睡眠）</li>

 <p data-pid="mPOx0_YG"><b>大路哥就分别来讲讲这4种的区别</b>，而待机（睡眠）的S1/S2/S3等模式请自行百度。</p>
 <h2>待机（睡眠）其实比较简单</h2>
 <ul>
  <li data-pid="_9EJwnFI">待机就是关闭了CPU、磁盘等供电，只保留内存的供电，耗电很低，约几瓦。</li>
  <li data-pid="XSRl3VBv">因为内存的特性是断电后内存中的数据消失，不可恢复。</li>
  <li data-pid="vi0qEjF_">待机可以在敲击键盘或者鼠标的时候唤醒，并快速恢复到原来状态。</li>
  <li data-pid="VomH1pB-">但是笔记本电池消耗完了，原来状态下的临时数据就没了，就得重新启动了。</li>
  <li data-pid="_e3fP1q4">当然台式机如果遇到停电，结果也是一样。</li>
  <li data-pid="9Zpz2ILQ">所以笔记本推荐使用这种模式，他是最快速的恢复方式。</li>
 </ul>
 <h2>休眠是一种可以快速恢复的关机模式</h2>
 <ul>
  <li data-pid="kpcR_cD9">休眠会把内存中的临时数据镜像到系统分区的硬盘中，然后关闭所有供电。</li>
  <li data-pid="0MqudMuU">恢复的时候按键盘和鼠标是不能唤醒的，需要按电源开机键。</li>
  <li data-pid="VLVzj-PH">他的保障是停电了断电了也不怕，一样可以恢复之前的状态。</li>
  <li data-pid="fwv1alz4">恢复的时候会把硬盘中的镜像临时数据重新写入内存。</li>
  <li data-pid="PL12y8i9">如果是机械硬盘，恢复会比较慢，如果是固态硬盘，一般读取速度在1G/s左右，其实几秒钟就可以恢复了，如果你是台式机，推荐采用这种方式，毕竟你没有电池包，哈哈。</li>
 </ul>
 <h2>混合睡眠是一种新模式（XP不支持）</h2>
 <ul>
  <li data-pid="QG5zJiW_">既然是混合就是<b>混合了待机和休眠的特性</b>，<b>目前Windows 10默认就是这个模式</b>。</li>
  <li data-pid="SCWWMpf8">他<b>同样保持内存供电</b>，内存的<b>临时数据也会写入硬盘</b>。然后其他硬件关闭供电，只保留内存供电。</li>
  <li data-pid="vyKjpVxp">如果没有断电，可以通过键盘鼠标唤醒，并从内存中即刻恢复。</li>
  <li data-pid="Oa8nzjet">如果断电，需要用电源开机按钮换新，并从硬盘中恢复数据到内存，需要几秒到几十秒恢复。</li>
  <li data-pid="IdnmafSb">那个几十秒是机械硬盘模式，哈哈，这个模式比较保险。</li>
  <li data-pid="jIUWTm1x">唯一的缺点就是关机比较慢，尤其是一些笔记本的固态硬盘读取速度快，但是写入只有300M/s的时候，关闭时间会老长，像大路的笔记本就是这种固态，关机需要20多秒，唤醒倒是秒开。</li>
 </ul>
 <h2>快速启动</h2>
 <p data-pid="m1VMyj-O">上次更新是10月15日，今天都21日了，实在太忙，继续补充快速启动。</p>
 <p data-pid="pY3r25zC">快速启动可以说是UEFI主板的标配，快速启动本身实质上和混合睡眠没什么区别。</p>
 <p data-pid="Ea8Xe5B0">说白了就是启用该功能后，当你关机的时候，并没有真正的关机。</p>
 <p data-pid="brfCEil9">他的效果和混合睡眠/休眠一样，关机的时候可能会比较慢，尤其是部分固态硬盘。</p>
 <p data-pid="ek-Pyg4b">哪种固态硬盘呢？读取速度2GB/s,但是写入速度只有300MB/s的固态硬盘。</p>
 <figure data-size="normal">
  <img src="https://picx.zhimg.com/50/v2-5ce6b6795bdcfe9f95d3d666150ae58c_720w.jpg?source=1940ef5c" data-size="normal" data-rawwidth="601" data-rawheight="432" data-original-token="v2-99958012fce73e5e4ab859d58479899b" data-default-watermark-src="https://pic1.zhimg.com/50/v2-1f3834a275b08402294d03a29d3279b5_720w.jpg?source=1940ef5c" class="origin_image zh-lightbox-thumb" width="601" data-original="https://pica.zhimg.com/v2-5ce6b6795bdcfe9f95d3d666150ae58c_r.jpg?source=1940ef5c">
  <figcaption>
   这是真实存在的，而且用久了写入速度更慢
  </figcaption>
 </figure>
 <p data-pid="62lwb5Pm">因为现在的Windows 10的电脑基本标配都是8G内存，至少一般有3~5GB数据要写入。</p>
 <p data-pid="Bm_27mHu">理论上15秒左右也能关机，但是关机还有其他任务要处理，所以这种状态关机30秒都很正常。</p>
 <p data-pid="_tQEBnQe">所以很多朋友新笔记本买到，就会先测试固态硬盘读写速度，如果遇到这种的。</p>
 <p data-pid="VQf7bhQB">一般就拆了换掉，自己买根好的升级，这样快速启动模式，关机也很快，10秒内搞定。</p>

 <p data-pid="AILCeZiH">我们即便不用或者经常不用的手机、数码相机、笔记本，都需要经常拿出充电，否则电池可能就废了，你倒省事，不知道你见过锂电池流水（液体）的情况么？</p>

 <p data-pid="ZKR7-00P">个人猜测这位大哥有点强迫症，还有断章取义，我相信一百个人看这张图中的概述，可能有二三十种不同的理解。大路是这么理解的，其他就不再做解释：</p>
 <p data-pid="C6UGRI55"><b>在建议如下之前<a href="https://link.zhihu.com/?target=http%3A//apple.com" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">http://</span><span class="visible">apple.com</span><span class="invisible"></span></a>讲了影响电池的两个因素：</b></p>
 <ol>
  <li data-pid="lI0CrT0T">环境温度</li>
  <li data-pid="26Hd6KZV">电池在断电存放前的百分比</li>
 </ol>
 <p data-pid="j_ibf2Ta"><b>关于环境温度</b><a href="https://link.zhihu.com/?target=http%3A//apple.com" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">http://</span><span class="visible">apple.com</span><span class="invisible"></span></a>在下面的4段内容里面举了个例子或者说是建议，不要把笔记本存放于高于32°的环境，除了大部分地区的夏天，其他三个季节基本不会超过这个温度。其实关于这个温度是有几个原因的：</p>
 <ul>
  <li data-pid="-mmhqUXO">apple的笔记本和手机高度集成，整机密度非常大，在散热方面可以说是公认的相对失败。容易发热，其实大多数厂商的轻薄本也有类似的情况。</li>
  <li data-pid="J0FRs9fd">别说把笔记本用来存放了，就是工作的时候大部分也是高于32°的内部空间温度，你用的时候都部讲究，存放的时候你要讲究？是不是断章取义，是不是强迫症？</li>
  <li data-pid="LMgD2jT6">如果你用苹果手机，开车喜欢用手机导航，夏天的时候如果不开空调出风口之一不对着手机吹，苹果手机就会提示温度过高，将停止充电，如果温度继续保持下去，苹果手机将提示，老子要关机了，不然要热死了。</li>
  <li data-pid="wWnZTF3U">其实经常这样，不管是原装电池还是后面换装的电池，很容易就鼓包了。</li>
  <li data-pid="PDUAjOym">你会发现手机后背翘起来了，笔记本电池鼓包外壳也会翘起来，笔记本电池一般在触摸板下面。</li>
 </ul>
 <p data-pid="lRDoxUkV"><b>关于电池百分比，<a href="https://link.zhihu.com/?target=http%3A//apple.com" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">http://</span><span class="visible">apple.com</span><span class="invisible"></span></a>讲了三段来说明：</b></p>
 <ol>
  <li data-pid="r73dF87P">电量应该保持在50%左右</li>
  <li data-pid="b53F4WVB">如果长期不用，应该半年为笔记本/手机充一次电，让他继续保持在50%左右</li>
  <li data-pid="oAQZsQIi">长期存放一定要（绝对）关闭电源，避免电池额外的损耗过快，导致低于50%甚至到我说的20%以下。</li>
 </ol>
 <p data-pid="l9Af7GWl"><a href="https://link.zhihu.com/?target=http%3A//apple.com" class=" external" target="_blank" rel="nofollow noreferrer"><span class="invisible">http://</span><span class="visible">apple.com</span><span class="invisible"></span></a>也说了<b>如果不这样做的后果，和我前面讲的意思大概也差不多</b>：</p>
 <ol>
  <li data-pid="CJgEslc_">导致深度放电，<b>最终无法给电池充电</b></li>
  <li data-pid="ouD98M5w"><b>长期存放完全充电，将导致失去部分电池容量</b></li>
 </ol>
 <h2>我们比较下这两种后果：</h2>
 <ul>
  <li data-pid="mkpmENld">深度<b>放电=电池直接挂了</b></li>
  <li data-pid="iJggw7TO">完全<b>充电=最大电量会减少</b></li>
 </ul>
 <p data-pid="pAlPTqGj">如果只能（大部分真实情况是如此）<b>二选一，你选哪个</b>？</p>
 <p data-pid="W-19bk2n">最后我们再综上所述，大家谁没事把电脑放那么久？几乎没有。</p>
 <h2>我们更多的情况可能是电源长期保持插上</h2>
 <ol>
  <li data-pid="cZIHjMGy">而且在长期插上的情况下，大部人会移动或者携带笔记本，所以电池不会长期保持满电。</li>
  <li data-pid="2q-HiKyX">有极个别的情况，笔记本从买来就一直插着，除了坏掉了，这种情况可能有万分之一。</li>
  <li data-pid="7Yu8x397">而且干这事的一般是机关单位，还特么外接了UPS，停电都不会断开。</li>
 </ol>
 <h2>但是聪明的厂商帮我们解决了这个问题</h2>
 <ul>
  <li data-pid="YI6izSL7">比如大路的Yoga有这样的设置，大路的暗影精灵也有这样的设置。</li>
  <li data-pid="JZk4MoIO">有部分老款笔记本BIOS中没这样的设置，那么是什么设置呢？</li>
  <li data-pid="evsYC_wb">BIOS的电池管理里面有：<b>最长电池寿命模式</b></li>
  <li data-pid="8SbjcJEP">这个设置<b>开启后一般笔记本在电池充入80%~90%就会停止充电</b></li>
  <li data-pid="ZvMjcqYa">并且在那个电池小图标中<b>提示：电源已接通，未充电</b>。</li>
 </ul>
 <h2>如果你的BIOS里面没有这样的设置怎么办？</h2>
 <p data-pid="laloIUsO"><b>如果你希望有这个功能！！！</b></p>
 <ul>
  <li data-pid="z-YSV3wn">其实一般笔记本<b>出厂自带一个电源管理软件</b>，里面也<b>可以设置“最长电池寿命模式”</b></li>
  <li data-pid="4O-4PA8c">你<b>启用这个模式</b>就可以了</li>
 </ul>
 <ol>
  <li data-pid="jHFfyO83">但是<b>如果你卸载了</b>，你可能会发现<b>官方没有单独适配的这个软件</b>的<b>下载</b></li>
  <li data-pid="Qm_pM-0i">请给笔记本原厂售后<b>400服务电话联系，告知型号，让他们给你发邮箱里</b>。</li>
 </ol>
 <p data-pid="hW1tcLxb">希望能给你 <a class="member_mention" href="https://www.zhihu.com/people/b3a2a19978eca28677956270b8672de5" data-hash="b3a2a19978eca28677956270b8672de5" data-hovercard="p$b$b3a2a19978eca28677956270b8672de5">@李云山</a> 和其他有兴趣的网友解惑。</p>
