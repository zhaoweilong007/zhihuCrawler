# 和华为公司合作是什么感受？
- 点赞数：6065
- 更新时间：2024年01月09日13时43分49秒
- 回答url：https://www.zhihu.com/question/398157192/answer/3355597734
<body>
 <p data-pid="tsm2PH3b">我们是一家OEM 厂,给HW做一个X86 Server 的项目, 后来因为众所周知的原因, 项目黄了. 按之前和其他大厂的合作惯例,我们公司都已经准备转卖那些多余的物料和设备了. 结果华为在通知我们项目取消后,还直接联系PL, 把这个案子所有设备和相关物料全部买回去. 这是我从业这么久首次碰到这种良心企业. 也是首次项目失败后项目核算居然没有亏.</p>
</body>
