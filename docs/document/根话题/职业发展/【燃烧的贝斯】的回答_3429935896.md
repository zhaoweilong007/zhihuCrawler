# 为什么都在劝退航空航天？
- 点赞数：12070
- 更新时间：2024年04月09日14时09分16秒
- 回答url：https://www.zhihu.com/question/454466945/answer/3429935896
<body>
 <p data-pid="MYgdyLpb">之前世界各国都把航空航天包装的太完美，什么必须无菌环境，什么不允许0.01毫米误差，什么人类顶级技术的聚合，看之前央视采访的纪录片感觉和过去去机房似的，得穿防护衣穿鞋套全身就只有眼睛还得小声说话生怕声音的震动影响零件稳定性！？</p>
 <p data-pid="KjrlscMz">后来看了《火星时代·深入SpaceX》，没有统一的工服就戴了个口罩，扳子随便扔，一边和镜头说话一边拧螺丝，至于什么无菌环境啥的也就和奔驰4S店维修区差不多。就这样人家的猎鹰发射几次了？星链发射多少卫星了？</p>
 <p data-pid="9rWdlNmg">现在查查所有航空研究所真正的工程师能有多少？领导有多少？工程后台服务又有多少？Space X最伟大的一点不是给到了人类未来去火星的机会而是打了包括NASA等所有国有研究所的脸，星际探索的中心是技术本身而不是维持技术的神圣性。</p>

 <p data-pid="E94qvvR-">第三、神化高尚化某个行业无论怎么辩解我都不认可，任何行业都是为人民服务为人类服务，把任何行业或者个人捧的太高从而达到其他目的这都是愧对人民嘱托的表现，不接受任何反驳。</p>
