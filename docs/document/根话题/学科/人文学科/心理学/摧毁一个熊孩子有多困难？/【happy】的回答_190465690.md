# 摧毁一个熊孩子有多困难？
- 点赞数：12518
- 更新时间：2017年06月28日13时42分59秒
- 回答url：https://www.zhihu.com/question/50996567/answer/190465690
<body>
 <p data-pid="oes7dqEv">刚刚在大连罗斯福地下一层麦当劳发现的事情，一个男生，大概十四五的样子，和朋友一起，旁边有个独身的女生趴在桌子上睡觉，穿的牛仔裤露出了屁股。然后那个男生用手机拍了女生的屁股。我刚好端着盘子路过，看到了，我跟他说，把照片删了，这样不好，他笑着无动于衷看我，一句话不说。我又说你这样做不道德，赶紧删了好不好。他还是不讲话。</p>
 <p data-pid="essyK_bm">我就在他背后找到一个座位坐着，一直在想要怎么才能治他，看到他们快要走了，就拿手机打开照相机对着他，叫了他一声同学，他把脸转过来，我拍了一张他的脸，威胁他说，你删了那张照片我就删了这张照片。他还是不说话也不删，和他朋友站起来走了。我几乎要放弃了。</p>
 <p data-pid="Fol3kpUt">我正打完这篇回答的第一行字，他朋友回来跟我说，他已经删了照片了，于是我选择相信，删了他的照片。</p>
 <p data-pid="7X5vhRfL">可能一开始那个男生觉得在朋友面前丢面子没有删，也有可能已经发朋友圈，现在删了也没关系。我不知道当时我为什么要管这件事，只觉得一个女生困到在麦当劳睡着一定很辛苦，虽然没有整理好仪表，但这也不是别人冒犯取笑她的理由。</p>
