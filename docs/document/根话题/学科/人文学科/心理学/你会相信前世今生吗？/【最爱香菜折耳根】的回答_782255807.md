# 你会相信前世今生吗？
- 点赞数：22194
- 更新时间：2020年12月13日20时03分39秒

<body>
 <p data-pid="jyKriXIi">天晚上看见宝宝从爬爬垫一下子翻身过来差点撞到茶几，她看我那个眼神，让我突然想起来我以前做的梦。（之前忘记了这个梦，醒来觉得很遗憾，回想了好久好久无果）</p>
 <p data-pid="LjFacLQL">怀孕前一个月吧，做梦梦见了在逛商场，一个小孩子抱住了我的腿，我说你这孩子咋瞎抱人呢？他说我没有啊，就是抱你。我说我不认识你呀，他说现在不是认识了吗。因为那孩子眼睛小，穿得不太好，我就心里想这孩子咋这么丑兮兮呢。结果他就撅起嘴说：我现在丑，以后长大了就好看了，你别嫌弃我呀。我问这个崽怎么知道我想啥，他说我就是知道。中间还说了啥我忘了。有一句记得清楚，他就问我不喜欢男孩子吗？我说没有啊，只是更喜欢小女孩，觉得更可爱。他就又撅嘴，然后倔强的说男孩子也很可爱啊。</p>
 <p data-pid="AMHtDMPX">最后他就蹦蹦跳跳走了，边走还回头跟我说了句：以后要喜欢我。</p>
 <p data-pid="MZdwYrEi">当时那个眼神就是有点弱小，可怜又无助。跟我宝宝昨晚那眼神一致，所以一下就想起来了。</p>
 <p data-pid="YT6QLDfB">我本身是个爱喝酒的人，但是怀孕前期我还不知道的时候，我爸爸过生日，好朋友来找我玩，我都莫名其妙居然一口酒都没有喝，竟然我的姐姐，哥哥，朋友都没有怎么劝我喝。要知道以前这些人聚在一起不喝一口酒是绝对不可能的。</p>
 <p data-pid="4-sGjnTE">后来孕晚期元旦节，在老家广场上去放孔明灯，灯上面写了保佑孩子健康，最好是个女儿。当晚又做了个梦，这个梦一直记得。还是那个孩子，在一个山上，他问我，你准备好了吗，我说准备啥。没回答我，然后问我，你真的喜欢女孩子吗？我说喜欢啊，然后他脸红了立在那儿不说话，我准备去拉他，他羞涩的说那你看我穿裙子好看吗，我才发现他穿的裙子，我忙点头说好看好看，丑萌丑萌的。他说，哼～那好吧，我为了你就做个女孩子吧。然后梦结束了。</p>
 <p data-pid="3QWXd4d4">孕期一直没有去查孩子性别，但是每个看到我的人都说我怀的儿子，因为肚子比较紧，还有身体一些特征，基本都说我怀儿子，在产房生了哭的声音特别大，婆婆和孩子姑姑，姨妈外面的医护人员都以为是儿子，因为哭声大，结果是个小公主。</p>
 <p data-pid="oz0tipm_">哈哈哈，昨天晚上突然想起前一个梦，感觉浑身舒坦。一下子就打开记忆的阀门了。</p>
