# 古代屠城是每个人都要找出来处死吗？

- 更新时间：2021年03月07日10时17分11秒

 <p data-pid="5rsABBQZ">清军在大同屠城几乎杀光了所有人，</p>
 <p data-pid="CMunHXV3">除了之前投降满清的叛徒及其家属，</p>

 <p class="ztext-empty-paragraph"><br></p>
