# 哪个瞬间你突然觉得读书真有用？
- 点赞数：9812
- 更新时间：2017年04月24日21时30分38秒
- 回答url：https://www.zhihu.com/question/35720340/answer/158163271
<body>
 <p data-pid="bUv4gp2P">我弟被别人诬陷反将之打脸的时候。</p>
 <p data-pid="_sz7zKmG">过年前某一天，我弟跟胡同口的小孩一起去他们家另一个院子去拿他妈妈（按辈分我们叫婶婶）的手机，回来的路上手机摔了。</p>
 <p data-pid="SjtKs86t">据我弟说，他们拿上手机之后那小孩把手机装外套口袋里，回来路上他们俩蹲路边玩被谁吓到了，然后他们跳起来就要跑。结果那小孩一跳起来手机也甩出来了，手机壳和手机就分了家。刚好我弟往前冲，然后就一脚踩到手机壳上。壳是硅胶的软壳，踩上去可能脏但是没有损坏。估计是摔这一下手机屏幕摔坏了。</p>
 <p data-pid="44iofZxy">那会儿好多人聚在她们家打牌聊天，当着大家面跟婶婶解释了，她说那既然是摔坏的就是摔坏的吧，我又没让你们赔。然后我妈跟她客套两句就带着我弟走了，我还在。结果我妈带着我弟刚走她就问她儿子:你跟我说，手机是不是XX踩坏的?XX是我弟，问是不是我弟“踩坏”的，这个问题指向性很明显啊。他们家孩子说是。明明刚说清楚了现在又扣屎盆子，忍不了。我说婶，要不然给我看看你手机。不情不愿掏给我了。看到手机屏幕我更加坚定相信我弟，不是踩坏的，就是摔的。手机屏幕上的裂纹类似于这样↓</p>

 <p data-pid="_QYuQGul">不过没有花这么厉害。一个五六岁的小孩，身高大概在110cm±，冬天的大外套口袋离地面最多四五十公分，那么高的位置摔一下肯定不会这么严重。</p>
 <p data-pid="BgBPMmUP">为什么肯定是摔坏的呢？简单来说，摔坏的屏幕裂纹是纤维状的，裂纹密集程度随着手机被摔的程度略有不同，但形状是一样的。但是如果是被踩坏或者被重物击打损坏的屏幕，因为着力点集中，屏幕裂纹会呈放射状或者说类似蜘蛛网的形状，着力点损坏严重，向周围逐渐减轻，甚至着力点会是粉碎的。如下图↓</p>

 <p data-pid="fbNDwG3L">我说婶，你这手机一看就是摔的，踩坏的裂纹形状根本就不是这样。然后我当着大家的面拿出我的手机抠下了钢化膜，一脚跺下去。拿着钢化膜和手机比较，给她指出来二者的区别，她也是尴了个大尬，打着哈哈说:说清楚不就可以了吗，你看你还又这样。我说不这样你信我弟了吗婶，你心里不痛快搞得我们也不痛快，现在大家不是都痛快了?</p>
 <p data-pid="F_gaW0dc">然后她就闭嘴了。</p>
 <p data-pid="3dck3W3a">今天回想起来，依旧心疼我那张钢化膜。十块钱呢。</p>
