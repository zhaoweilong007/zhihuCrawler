# 你最喜欢哪本小说的哪个片段？
- 点赞数：5529
- 更新时间：2018年03月14日22时29分09秒
- 回答url：https://www.zhihu.com/question/268052080/answer/341463711
<body>
 <blockquote data-pid="xgnsOAn6">
  天下只有两种人。譬如一串葡萄到手，一种人挑最好的先吃，另一种人把最好的留在最后吃。照例第一种人应该乐观，因为他每吃一颗都是吃剩的葡萄里最好的；第二种应该悲观，因为他每吃一颗都是吃剩的葡萄里最坏的。不过事实上适得其反，缘故是第二种人还有希望，第一种人只有回忆。
 </blockquote>
 <p class="ztext-empty-paragraph"><br></p>
 <p class="ztext-empty-paragraph"><br></p>
 <p data-pid="dURpxb82">——钱锺书《围城》</p>
